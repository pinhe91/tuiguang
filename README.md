### 公园里两个点痣人的不同境况

文: 中国大陆大法弟子 

【明慧网二零二二年八月二十六日】我市有一个小公园，散步聊天的人很多，是我与同修经常去讲真相的地方。
公园里有排椅，运动器械，地面也很平坦，公园的围墙很矮，是石头砌成的，正适合当座椅，是夏天乘凉的好地方，一年四季都有人在这里游玩。我们在这里讲真相，有时坐在石头墙上，这个人听完真相走了，又来个人坐下；有时坐在排椅上，张三听完真相，或者有的人不愿听走了，一会儿又来人坐下，很多人就这样明白了真相。

有时我们围着小公园的围墙走一圈寻找有缘人，看到有摆摊叫卖的，有游医给人看病的，有点痣的……公园有两个点痣的摊位，都是老太太。来的时间长了，我们发现公园北墙外一家的老太太周围总是有几个人围着，而公园南墙外那家的老太太那儿没有人或者偶尔有人和她聊天。我们明显的知道这两人的生意不一样。
有一天，我们看南墙老太太那儿没人，就试着给她讲真相，可是只讲了几句，这老太太就大吵大嚷的说些不好的话，她旁边坐着的人也随她一起反对。于是，我们向北墙老太太那个方向走去。

还没到那儿，北墙老太太就热情地招呼我们，并拿出座椅让我们坐下。我们便说起话来，给她讲大法的美好，讲三退保平安等，她非常接受。她说她什么都没入过，可是她很喜欢听真相，很相信大法，边说边喊出了：“法轮大法好”。

我们聊的很投机。她给我们讲了一件事情：有一天，她摊位的附近，有一位女大法弟子发真相材料，被一位警察盯上了，警察抓住大法弟子要去派出所，这时老太太趁警察不注意，心生一计，眼疾手快一下子就夺下了大法弟子的挎包藏起来了，一会儿警察过来问老太太挎包的事，老太太说不知道没看见，就这样，老太太的善举保护了大法弟子躲过了一难。

试想，一个人在这样恐怖的形势下，在警察眼皮子底下不计自己的安危，保护一个不认识的人，这是多大的善举，多大的福德，对周围的人也是无声的影响。

从此，北墙老太太的生意非常红火，一顺百顺，我们什么时候路过这里，什么时候看到这里总是围着一些人，我们每次和她打招呼，她都会笑嘻嘻的对我们说：法轮大法好。相比而言，南墙老太太的摊位总是冷清清的少人或无人。

通过这两位老太太的对比，明眼人一看就明白了，对大法的态度就是给自己选择了未来。
