### 丈夫说：“哎呀！我的眼睛好了”

【明慧网二零二二年十一月二十七日】

#### 丈夫说：“哎呀！我的眼睛好了”

文：辽宁大法弟子

我修炼法轮大法不到一个月时，全身的病就都好了。二十多年了，丈夫看到了我身体的变化，非常支持我修炼，也非常相信大法，经常给人讲法轮大法好。可是让他学，他就说“我放不下执著，放不下这、放不下那”，一直没有走進大法修炼。
二零一七年的一天，丈夫开着电动车去工地干活。快到工地的时候，他突然两眼模糊、头晕；两眼不能同时睁开，只能睁一只眼闭一只眼。他以为自己没有休息好，到工地休息一会就好了。他在工地躺了一上午也没好，心里有点不稳了。下午一点的时候，丈夫给我打电话，说了他的情况后要回家，我说：“你慢点开，回家吧。”

从工地到家四十分钟的路程，他开了六个小时，到家已经是晚上七点了。电动车车胎碰到马路边的石头被刮坏了，是找朋友帮忙拖回家的。丈夫说看到的我都是重影的。我说，我在网上查了一下，有几个象你这种情况的人，去医院也没治好。今天有点晚了，明天去医院吧。今晚咱俩念“法轮大法好，真善忍好”，他说：“行。”

看的出丈夫的情绪很低落。他躺在床上，我坐在床边，我俩不停的念“法轮大法好，真善忍好”。念着、念着，他说要去卫生间，就下了床。刚站起来，丈夫说：“哎呀！我的眼睛好了，就是还有点晕。”我说：“你快去，回来咱俩炼动功。”

这时我一看表，我俩整整念了一个小时的“法轮大法好，真善忍好”。炼完动功，丈夫的症状完全消失了。

从此，丈夫走進了大法修炼。

#### 谁信谁就得福报

文：江苏大法弟子

我以前工作厂里同事的弟弟得了食道癌，我想到他家去给他讲大法真相。如果他要是相信的话，那他就有救了。

我到同事弟弟家之后，得知他已经去了医院，我就把带的真相小册子《希望》给了他妻子，让她去医院时给他看。他妻子说：“他文化程度低，能看懂吗？”我说：“这和文化程度没关系，只要认识字，相信就管用。”

后来同事弟弟的妻子跟我讲，她把小册子给他后，他一直在认真的看，并按照上面写的“法轮大法好，真善忍好”九字真言默默的念。

过了几天，在手术之前，医生给同事的弟弟检查身体，指标一切正常。医生说：“怎么回事？”同事弟弟说他当时不敢说是念九字真言念的。过了几天，他就出院回家了。

到现在，已经六、七个年头过去了。同事弟弟每年去医院体检，结果都很正常，这就是法轮大法的神奇！谁信，谁就得福报。
