### 疫情中　民众觉醒

文: 大陆大法弟子  

【明慧网二零二三年二月一日】初四这天，我和一个七十多岁的陌生阿姨聊天，准备给她讲真相，她说：“你知道吗，我住的小区这一个多月因染疫已死了37个人了。”
我接她话说：共产党执政以后搞各种运动就人为害死八千万中国人，人不治天治，贵州有块藏字石，上面写着“中国共产党亡”，咱们顺天意而行，你就叫安康，把共青团、少先队退了，神佛保佑好人。

她说好的。她又告诉我，她在上大学的孙子说了，自己坚决不入党，入团都后悔死了。她很开心的样子，因为孙子不愿入党。看来民众自己都知道远离邪党就有福报。然后我又给她讲了法轮功的真相，让她记着“真善忍好，法轮大法好”。她说：是的，人一定要善良。

过年前几天，一个六十几岁的大姐坐着在晒太阳，我想给她讲真相的，刚坐下来她就说：“这个东西（指病毒）不能得，得了以后你看我力气也没有了，原来买这点菜直接就可以拿到家的，现在一路上要停几次，这是第二次停下来休息了，到家之前至少还要停一次。”我再一看，在她脚边放着几个装有菜的马夹袋。

她继续诉苦：“上个月放开后，我先生发热，以为是感冒，自己按常规吃点药，三天后没改善，到医院去看，一查氧饱和度很低（多少我记不得了，比95还要低），医院里人山人海，床位也没有，着急啊！还好，家里有个侄子在其他医院里工作，联系后搞了一个床位，买了一个氧气包，救护车也叫不到，就叫了出租车转院了。从進医院到转院这点时间，死了三个，有的人救护车到医院抬下来就没气了，恐怖啊！转院后，先生住了22天，不能再住了（规定住两个星期）出院了，现在在家里慢慢调养。要是没救过来我怎么能接受得了！怎么能这样管理呢？我姐原来一直讲一些与官方不一致的消息，我就讲要听官方的，不要听小道消息……之前封城官方一直辟这个谣、辟那个谣，后来都成为现实了，都变成预言了。这个官方消息不能听，从小就给我们洗脑，以前还有别的，六四、法轮功……都是官方消息。”

我就顺势给她讲了三退（退出共产党、共青团、少先队组织）保平安和法轮功真相，她很是认可，一再谢谢，我说：谢谢大法师父吧！
