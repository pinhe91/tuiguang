### 念“法轮大法好”得福报

【明慧网二零二二年九月二十九日】

#### 念“法轮大法好”得福报

〖中国大陆来稿〗一年的中秋我回老家，和哥哥在二侄女家吃饭。侄女的儿子吃鱼被鱼刺卡嗓子了，当时哥哥说：吃块馒头顶下去。但吃馒头没解决问题，而且鱼刺卡的更深了。小孩说难受，于是大侄女的丈夫开车拉二侄女和孩子上医院，我也跟上。我在路上跟孩子说：念“法轮大法好，真善忍好”，鱼刺就下去了。他不念。

到了医院是正中午时间，值班医生看看鱼刺太深，说得等下午上班打麻药做个小手术才能取。因为等的时间长，我们就先开车回家了。这次我带孩子念“法轮大法好，真善忍好”，他跟我念了一路，也就十分钟，小孩好了。
一次，县里的同修A激动的给我讲一件神奇事：

二零零一年左右，A被一位老板邀请一起去内蒙经商，因忙于工作，荒于学法，几乎要脱离修炼，他感觉这样不行，就要求回家。正好这位老板要回老家办事，于是他们开车回来。

从山西省进入河北省保定地区是崇山峻岭、峰峦叠嶂、山崖陡峭的太行山脉，一路是下坡的盘山公路，而且这时已是天开始黑下来了，因为离家近了，急于赶路，没有停歇，本应该拐弯没看清，就直着冲下了山崖，开车的老板吓得够呛，心想：完了！完了！同行的A由于很长一段疏于学法，大脑也是一片空白，但他忽然下意识的摸到自己带在身上的《转法轮》，大脑只是闪出一念：“我有《转法轮》！”就这么一念，正在往山崖下冲的车“啪”一下停在一个很小的台子上。

A说：“当时在路边的饭店外有几个人目睹了这令人惊悚、神奇的一幕。当我们稳住神儿，战战兢兢地从车里出来。他们大声对我们说：肯定有神仙保护你们，你们太幸运了，这经常出车祸的。他们帮我们把车推上来。当晚老板司机说啥不敢再走了，在饭店住了一晚。我把当时我的那一念告诉了老板。”

第二天早上，老板去出事的地方去看，回来感叹说：“车如果再朝下冲，就车毁人亡，没命，下边五十多米深的河谷，陡峭河谷是一块块凸起的怪石。真是谢谢李大师！”

#### 路遇一阿姨　听得一故事

〖中国黑龙江来稿〗我是一名黑龙江大法弟子。一天清晨我来到江边，迎面过来一位阿姨，看起来很善良，我笑着对她说：阿姨你好，送你一张真相护身符，记住“法轮大法好，真善忍好”，灾难来时命能保。

阿姨笑眯眯的没出声，伸手在自己的兜里掏出一个方便袋，从方便袋里又拿出一个方方正正的纸包，打开一看原来是一张看起来很旧的护身符。

阿姨抬起头来对我说：姑娘，这张护身符跟随我七年了，我今年七十多岁了，七年前我老伴去世了，孩子又在外国打工，我天天想念老伴，整天难过上火，慢慢身体出了毛病。到医院检查，说是乳腺癌晚期，怎么办呢，和孩子说吧，孩子在外国打工，不说吧，身体还一天不如一天，我真的绝望了。正在我走投无路时，遇见一位你们学法轮功的好心人，告诉我，诚心敬念“法轮大法好，真善忍好”，身心就会发生改变。我听了这位法轮功学员的话，就天天念，真是太神了，不但我身体的病好了，对老伴的情也放下了。这张护身符保我又活七年了。我太感谢法轮大法了，太感谢那位法轮功了。你看我天天出来遛弯，我现在每天诚心敬念“法轮大法好，真善忍好”一百遍呢。

阿姨笑着走了。我真为这位阿姨高兴。
