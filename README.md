### 警察说：“注意安全”

文: 辽宁大法弟子德慧 

【明慧网二零二三年三月十六日】我是二零零四年得法的女大法弟子，沐浴大法佛光一十八载。
由于动迁，在师父有序的安排下，我搬入了新家。这里的环境正适合，我就开个卖店，这样接触的人也就多了，正好讲真相。可是刚开始怕心很重，慢慢的人都熟悉了。有人夸我年轻。我就告诉他们，修大法可以祛病健身，是性命双修的功法，谁学谁都会年轻、健康。有人夸我对婆婆好。我就告诉他们：因为我修真、善、忍做好人，我师父叫我这么做的。

有一天，一对夫妻打赌，男的说我是婆婆的儿媳妇，女的说肯定是女儿，不信就赌一百元钱。第二天女的来问我，我笑着对她说：“你输了。”她不解的问我：“嫂子，你怎么能做到对婆婆这么好的？”我说：因为我是修真、善、忍的，大法师父告诉我们：“在各种环境中都得对别人好，与人为善，何况你的亲人。”[1]又给她讲了真相，帮她全家四口都做了三退。

二零一五年兴起全球起诉江泽民的潮流。我想不能只我自己起诉他，我要叫家里的亲朋好友和众邻里都起诉这个大魔头。于是我准备了支持全球起诉江泽民的征签表，在给来店里买东西的人讲真相后，请他们在征签表上按手印。他们都选择了美好的未来。

这些明白真相、相信“法轮大法好”的亲友、邻居们，也得到了福报。下面我把得福报的事举几例：

1、一个患腿血栓转为肺血栓的邻居小妹，卧床不起。在诚心敬念“法轮大法好、真善忍好”后重获新生，什么都能干了！现在她每天都看《转法轮》。

2、一个邻居姐妹诚念“法轮大法好，真善忍好”，她家种的香瓜，冰雹打不着，获得大丰收。

3、前年，姐夫患膀胱癌，诚心敬念“法轮大法好，真善忍好”。现在什么农活都能干了。

#### 讲真相就是我应该做的

师父说：“任何一个地区的大法弟子啊，基本上你们就是那个地区众生得救的希望了，而且是唯一的希望。那里的众生啊，要听到你们的福音，要听到你们在讲清真相中使他们认识到大法是什么，所以大法弟子的责任就很重大。”[2]

师父把这么重大的责任给予我们，作为弟子就应该认真做好，责无旁贷 。如何完成好自己的责任，我想首先要修好自己，用大法充实自己，多学法背法，遇事向内找，每天参加集体学法，在一言一行上下功夫。

通过大量学法、坚持背法，师父帮我去掉了很多执著心和这个党文化，使我在讲清真相时能够语重心长、和风细雨，无论世人如何表现，都无怨无恨，我抱着一颗纯真的心，善待每一个众生，顺着世人的执著讲，使他们明白真相而得救。

下面我把讲真相中的几个片段叙述给大家：

#### “大法师父真了不起”

一次，我遇到一位大哥坐在路边，好象等我给他讲真相似的。我走过去问候一声，就开始跟他讲真相。他说：“小妹呀，你年龄这么小，可别出事呀！现在共产党抓这事多厉害，你们为了啥呀？出了事怎么办呢？”我说：“大哥，你是不是认为我们傻啊！”他笑了。我说：“我不为名，不为利，不为搞政治，就是为在大灾大难前把善良人救下来，告诉你们诚念‘法轮大法好，真善忍好’就能留下来。但必须向老天退出你入过的党、团、队。因为你入党、团、队时已经把命献给这个邪党了！老天要灭它的时候，咱不得当陪葬品吗？现在有骗吃的、骗喝的，可没有骗让人平安的！”大哥听了我的一席话说：“是这样啊，那我退。可你们也太不容易了！可得注意安全呢！谢谢你！”我说：“不用谢我，谢我们师父吧！”他说：“大法师父真了不起”！

#### “孩子到家歇歇脚吧！”

一次我遇到一位大姨，我给她讲真相，做了三退后，她热情的非要我去她家不可。我婉言谢绝了。大姨说：“再路过这累了，就到家吃点东西，歇歇脚吧！”我当时感动的眼泪都要掉下来了，世人真的感受到我们大法弟子的慈悲了！她们也把我当亲人啦！

#### 警察说：“注意安全”

腊月的一天，我出去讲真相，遇到一便衣警察，他说：“你知道我是干什么的吗？”我一听可能遇到警察了，想到师父说：“一个不动能制万动！”[3]我说：“大哥，无论你是干什么的，首先咱得平安，你最知道我是冒着多大的风险，那你知道我为什么这么做吗？我就是想让象你这样的好人平平安安的躲过瘟疫，躲过灾难！”他说：“我是警察，你千万注意！年底了，我们大面积巡视，今天你遇到的是我，要是别人就麻烦了！”我说：“谢谢！善待大法一念，天赐幸福平安！大哥你是党员吧！”他说：“你说呢？”我说：“大哥，咱赶快退出来吧！带着这个印记谁都躲不过这次瘟疫，共产党都啥样了？咱可不能给它当陪葬！”他说：“行！谢谢你！”我说：“千万记住：法轮大法好！真善忍好！”他答应着，匆匆走了。

#### “修炼人不能见死不救啊！”

一天，我遇到一个什么都不信的人，他说：“共产党都骗人，信什么都没用。”我说：“大哥，我不是叫你和我一样，信仰法轮功那得看缘份，我是告诉你在大灾大难时怎样才能得救。”他说：“怎么得救？”我说：“诚念法轮大法好、真善忍好，同时把你入过的党、团、队向老天退出来就能得救。”他说：“不信。”我说：“大哥，你知道我们这么些年冒着生命危险告诉大家这九字吉言是为什么吗？就是灾难来了，大家好平安呢！我们是修炼人，不能见死不救啊！我能拿自己的安危开玩笑吗？真、善、忍是宇宙的正能量，正能量就能杀死病毒和细菌。”大哥点点头说：“我只入过少先队，退了吧！谢谢！”我说：“谢谢大法师父吧！是师父叫我救你的。”他说：“谢谢大法师父！”

这是我讲真相时的几个片段，因篇幅有限就不一一列举了。希望在今后的修炼路上和同修们共同精進做好三件事，和师父回家！如有不符合法的地方，请同修慈悲指正！

感恩师父！

谢谢所有帮助过我的同修！
