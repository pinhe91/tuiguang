### 被绑架迫害昏迷一年　长沙女护士成植物人

【明慧网二零二二年七月五日】（明慧网通讯员湖南报道）长沙市法轮功学员贺祥姑二零二一年五月十六日在住处被警察绑架，被非法关押在当地看守所。一个半月后，警察通知她家人说贺祥姑在医院昏迷不醒。如今近一年过去了，贺祥姑仍处于深度昏迷状态，不知她在看守所的一个半月里到底经历了什么样的摧残。

![image](https://user-images.githubusercontent.com/79625284/177292215-8004cf39-0d24-45ff-ab05-fc17c32cf2ae.png)

贺祥姑

![image](https://user-images.githubusercontent.com/79625284/177292278-7fb3f453-7b93-4184-81e7-1e25e3a19084.png)

贺祥姑仍处于深度昏迷状态

贺祥姑今年六十一岁，原湖南省妇幼保健院门诊手术室护士，一九九八年开始修炼法轮功。中共迫害法轮功后，她因坚持“真、善、忍”信仰，多年来一直遭到中共的残酷迫害，累计四次被非法劳教，三次被关入精神病院，长期不准上班，无理停发工资和保险，并被迫离婚。

二零二一年五月十六日，贺祥姑在住处再次被清水塘派出所警察绑架，被非法关押在长沙市第四看守所。警方不准她家人看望。她家人对办案警察说：如果出事了要他负责。该警察还对贺祥姑的家人发脾气。

二零二一年七月三日，贺祥姑的亲人突然接到电话，通知他们去长沙市第一医院接贺祥姑回家。亲人感觉不对劲，当亲人赶到医院时，贺祥姑已经是昏迷不醒了。经家人强烈要求立即抢救。七月四日，医院将贺祥姑送入重症监护室，靠着呼吸机维持着生命特征。几个月后，医生说，再过两个月还没醒来，就成了植物人了。

从二零二一年五月被绑架，到同年七月进重症监护室，对比其他重症监护室的病人，他们的家属能每天探视。而对贺祥姑的家人，警察只准每星期探视一次，每次限时五分钟。除直属家人外，其他亲朋好友探视要到清水塘派出所报备批准才能到医院探视。

家人每次探视都会在贺祥姑的耳边呼唤：“你快点好起来，只要你能吃饭，能说话，我就带你回家。”可是，无论怎样呼唤她，贺祥姑都没有任何反应。亲人哭干了泪，质问警察：我的亲人在家好好的，你们为什么要抓她去看守所？警察回复，不要为难他。

重症监护室主治医师黄康年前曾对贺祥姑的家人说，他请示了市公安局将贺祥姑转到普通病房，已批准由亲人陪护。可年后，黄医生又改口说上面没有批。面对主治医生前后不一的态度，家人万分无奈。

躺在重症监护室床上的贺祥姑，被绑架前身体健康，被劫持进看守所仅一个半月，就被迫害致深度昏迷，生命垂危，家人们都无法想象她在看守所里都经历了什么？
