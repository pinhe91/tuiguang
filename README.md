### 医生说活不过三月的表弟健康的活了十年了

文: 湖北大法弟子 

【明慧网二零二二年七月六日】二零一零年上半年，湖北省公安县沙场村一位五十岁的男子肺部长了瘤子，因家庭经济条件不好，直到病情恶化到不去医院不行了，才去医院就诊。
我与此人沾点亲，叫他表弟。他偶尔也看大法真相资料，我也曾经给他介绍过法轮大法。由于邪恶对大法弟子持续不断的迫害，使人不太理解，也不太在意大法怎么好了，他也一样。

就在他要动手术的前一天我去医院看他，就见病房里有几个医生，还有的病人也在围观，原来是正在抢救他。只听医生说他吐血止不住，要求家属签字，马上动手术，只有死马当活马医了。我也有点紧张，本来就是想在他手术前来给他打打气，让他進一步了解大法真相顺利过关的，于是就在他被推進手术室前一刻，我就对他说：“你认识我吗？”他点头，显然他头脑清醒，于是我就说：“什么事也别管，就念‘法轮大法好，真善忍好’九个字，你记住了吗？”他又点点头。

我陪着他的亲人在外等了五个小时。他的家人有的焦虑的说，不知人是不是能活着出来。手术做完了，医生出来对家属说，手术有难度，病灶位置不好，看来活不过三个月了。即使是能活下来，一年也得花两万元治疗费才能维持，即使如此，命也不长久。

他在医院又住了几天后就回家了。

过了几天我带着播放机、优盘、真相资料去家里看他。进门一抬头就看见一张很大的毛魔头像挂在墙上。交谈中我诚恳地告诉他们挂毛像的危害。他和家人也理解，他就告诉我，他的父母出身不好，一直被当地村官们欺压，父母只知道老实巴交的干活。生下他们姐弟八个，都没读几天书，就随着父母在家干农活。后来家庭情况稍微好点了，他自己的身体却不好了，还要供孩子读书，说他儿子是家族中唯一的大学生。

我们交谈了很多，最后我叫他摘下毛像，这对自己对家人都有好处，讲了相信大法好是会得福报的。我说以后我会送他们一张平安画。看我是真心为他好，他什么也没说就把毛魔像摘下来烧了。

过后不久我去给他送平安画，他跟我说：我家儿子毕业后想考公务员，连考四年没被录取，他劝儿子别考了，咱家没后台。儿子三十几岁了还没有女朋友，等到何时啊？我说：“明真相，会有福报的，只是时间问题。你不相信吗？我相信。”

二零一九年年底，我去乡下路过这个表弟家就去看看他们。表弟两口子满脸笑容告诉我，自从摘了毛像以后，家里变了，一切都变了。表弟说，他的身体恢复的很好，已经健健康康的活了十年了。他说：医生不是说我就能活三个月吗？还说我真能活过来一年也得花两万元药费，也管不了多久。可我啥药也没吃，活的好好的。家里环境好了，就连空气都变的新鲜了。儿子已考上公务员了，女朋友也有了，已领了结婚证，车、房都有了，我们做父母的只是给了一点钱，只是因疫情就没办酒席。

好事一桩桩一件件都来了，真替他们高兴！我说：“都是大法带来的福，你信吗？”他连声说：“信！信！”我说：“你把自己得福报的事讲给周围的人听，证实大法，功德无量。”
