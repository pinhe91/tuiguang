### 河北蠡县各乡镇派出所骚扰法轮功学员

【明慧网二零二二年十月六日】（明慧网通讯员河北报道）二零二二年九月初以来，河北省保定市蠡县蠡吾镇、百尺镇、留史镇、新兴镇、小陈乡、林堡乡、万安乡、郭丹乡、堡墟乡等派出所胁迫村干部到法轮功学员家骚扰，拍照，连去世的、不练的人的家都不放过。
派出所警察到法轮功学员家中骚扰时，非要给法轮功学员拍照，家人说该法轮功学员不在家，他们就给法轮功学员的家人照相，还给家门口照相。

蠡县城关派出所胁迫村干部把法轮功学员家的对联撕下来；蠡县城关派出所到法轮功学员家敲门，公然行骗、撒谎，说自己是社区的，要法轮功学员的健康码、行程码，却不要法轮功学员家人的。有法轮功学员给社区打电话问：是否派人查健康码？社区回答：根本没派人去查。
二零二二年九月二十八日、二十九日这两天，蠡县城关派出所警察开着警车，不穿警服，到所辖区的法轮功学员家转，给法轮功学员和家人造成很大的压力。有的家的老人有心脏病，被吓得吃不下饭，睡不着觉；有的家的孩子无奈地告诉妈妈：妈妈，你上了他们的黑名单了。给孩子造成心理压力。有的法轮功学员的家人害怕，把气出到法轮功学员身上，造成家庭不和。

有的老百姓不明白，看到警察骚扰，对法轮功学员说三道四的。也有同情法轮功学员的。有警察问一村民，法轮功学员住哪时，村民回答：为别的事告诉你，为这事不告诉你。有的村民听说警察骚扰法轮功学员，说：就该把他们哄出去。

法轮功学员面对各方面的压力，依然对上门骚扰的警察讲真相，告诉他们为了自己和家人的未来不要再这样干。
