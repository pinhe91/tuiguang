### 原上海闵行区公安分局局长郑文斌遭恶报

【明慧网二零二二年七月二十日】（明慧网通讯员上海报道）近日，上海市检察院第三分院以受贿罪对闵行区副区长兼区公安分局局长郑文斌批准逮捕。郑文斌在担任上海市普陀区公安分局局长、上海市闵行区公安分局局长期间，参与迫害法轮功，迫害大法弟子多人，郑文斌现已被逮捕，终遭恶报。
郑文斌，男，一九六七年出生，长期在上海公安系统工作，二零一六年十一月担任闵行区副区长兼区公安分局局长、党委书记，在此之前，郑文斌担任上海市普陀区公安分局局长。郑文斌于二零二一年十一月被免职，次月被查，二零二二年六月被双开，目前被逮捕。

据明慧网报导，显示郑文斌对法轮功学员被迫害负有责任。经查核，湖北仙桃法轮功学员谭秀芳，闵行区七宝镇大法弟子徐建兴，闵行区大法弟子王凤贞、李慧，梁霞，贺品琴，聂广丰，蔡玉芳，赵贤六，陈平，张懿，戎美英，邓成联等等有关信息42条。

以下是遭迫害的法轮功学员案例。

#### ▼郑文斌担任上海市闵行区公安分局局长期间

◎二零二一年九月十五日晚上10点多，在上海打工的湖北仙桃法轮功学员谭秀芳（女，36岁）在上海家中被绑架到闵行区看守所刑事拘留。

◎二零二一年八月二十一日明慧网报导，上海市闵行区七宝镇法轮功学员徐建兴，已失去联系两个多月。据悉，在六月份时，徐建兴已被秘密绑架，具体情况不明。

◎上海闵行区法轮功学员李慧、梁霞因二零二零年三月被闵行分局、古美派出所绑架后被取保候审，二零二一年三月十一日，两人再次被闵行分局非法行政拘留15日。

◎二零一九年九月八日，上海市闵行区71岁法轮功学员贺品琴女士在闵行区碧江广场打真相热点手机时，被闵行区公安分局碧江派出所警察绑架，被劫持到上海市奉贤区看守所关押迫害了三十天，十月八日被“取保候审”回家。二零二零年八月二十七日，贺品琴在家里又被绑架，据悉面临非法庭审。

◎二零二零年七月二十一日下午，上海闵行江川路街道红旗小区法轮功学员70岁的王凤贞，家门被敲开，碧江派出所红旗新村的片警李菁和江川路街道平安办某女士两人上门，要王写点什么，被拒绝，片警戴着执法仪，王让他关闭。

◎二零一九年十一月十三日早上，上海闵行区大法弟子聂广丰，被颛桥派出所绑架，警察把家里翻地乱七八糟的，2个电脑（笔记本1个，台式机1台）也被抄走了。

◎二零一九年九月三日，上海市法轮功学员蔡玉芳被上海市公安局闵行分局的国保和虹桥路派出所绑架。

◎二零一九年五月十四日，闵行国保在浦东三林非法绑架了徐家玉、郁归兰、董云芬、黄巧琴、严梅英等七人。

◎二零一九年四月十一日夜里11点左右，上海市法轮功学员赵贤六从甘肃老家回到上海的家，被守候在此的上海闵行国保、虹桥派出所等10多个警察绑架，家中的大法书等私人物品被搜走。

◎二零一八年十一月四日中午，上海徐汇区法轮功学员陈平在家中被闵行国保警察绑架。一同被绑架的还有普陀区的张懿、浦东的戎美英。

◎二零一八年三月二十三日，上海市闵行分局和虹桥派出所警察没有出示任何手续，将邓成联从家中绑架，关到闵行区看守所。六月十五日，邓成联被构陷到闵行区检察院。曾出资三十多万元为乡亲修路的好人邓成联，因坚持信仰，遭迫害致身体孱弱，不让会见律师。

◎上海普陀区大渡河路四十多岁女法轮功学员应钰于二零一七年四月十八日上午被普陀区国保绑架、抄家。十九日被非法刑事拘留。家中只剩七十多岁老母在家无人照顾。

◎二零一六年八月二十六日，上海黄浦区法轮功学员尤秀英在家中被普陀长寿路派出所绑架并野蛮抄家，九月二十九日被上海市普陀区检察院非法批捕。

#### ▼郑文斌担任上海市普陀区公安分局局长期间

◎二零一五年六月二日，上海法轮功学员翁萍在家中遭破门而入的普陀分局长征派出所、长宁区分局国保大队及新泾派出所警察绑架。之后警察将她构陷到上海市长宁区法院。当时，翁萍的身体出现严重病症，血压高达170-180，胆囊肿大，胆结石，随时有胆囊破裂的危险。翁萍的家属非常着急，呼吁当局立即释放翁萍。

◎二零一三年十月九日晚间，上海普陀区法轮功学员杨曼晔遭闯入家中的数十名国保警察绑架并非法抄家。一同被绑架的还有法轮功学员陈秀英、罗秀丽及另一老年苗兰英。杨曼晔的丈夫与儿子也被劫持，到次日早上才让回家。这是杨曼晔第五次被绑架、非法关押迫害，家中被抢走三台电脑，打印机，真相资料和钱。

◎二零一二年五月二十九日，上海普陀区公安先是派人跟踪68岁的刘文英（家住上海市普陀区灵芝公寓），下午国保邬菊伟带着7、8个人绑架了刘文英，说是因为刘文英寄真相信。
