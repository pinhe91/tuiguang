### 当年十一岁小弟子见证的九九年七二零

——大法修炼改变了一个叛逆少女

文: 澳洲青年法轮功学员

【明慧网二零二一年七月十六日】我在十岁的时候随母亲一起开始修炼法轮大法，从一个学习特别差、思想叛逆、没人愿意接近的孩子，变成了一个心里充满阳光、成绩拔尖的好学生，也真正体验到了家庭的和睦与温暖。但这一切都被一九九九年七月二十日开始的中共对大法的迫害摧毁了。十一岁的我，亲身参与并见证了迫害初期周围同修们反迫害的令人感动的过程。
在母亲被非法劳教后，我一度离开了大法。但内心对人生意义的疑问，让我又神奇般的走回了大法修炼。两年前有机缘来到海外，利用各种方式向中国人和海外各族裔民众讲述法轮功真相。在反迫害二十二周年来临之际，回想起当年的经历，仿佛就发生在昨天。

#### 年少时修炼大法后的神奇转变

我曾是一个孤独叛逆又自卑的孩子。因为在我两岁的时候，父母就因为家庭暴力离婚了。在一九九零年的中国，父母离婚是很罕见的。所以我就成了街头巷尾议论的焦点。走到哪个小巷子里就会听到一些上了年纪的人对我指指点点，说我是某某的孩子，我的父母离婚了等等。每次我都会匆匆跑开，而且造成很大的心理伤害。我变得很自卑，甚至几次梦中都梦到如此难堪的情景。

在中国单亲妈妈带着孩子不但没有福利待遇，还要承受各方面舆论的压力。我的爸爸酗酒，也没有支付抚养费，我们当时还要租房子住，日子过的很贫苦。

那时候妈妈和我身体都不好。妈妈有严重的冠心病和大动脉炎（俗称无脉症），浑身乏力。我经常感冒生病，小小年纪，肺炎、胃炎、肝炎就都得过，还有一种奇怪的病，就是嗓子肿痛，咽口水都特别疼，去医院也查不出结果。

五岁的时候妈妈给我找了一位继父，因为内心对继父的抗拒，我和妈妈的关系也变的紧张了。记得当时继父和母亲之间的争吵多数是在对于我的教育问题上。我变得倔强不听家长的管教，就算挨打我也不服软，常常用不吃饭逼迫妈妈妥协。学校老师看我学习总是考倒数第二名也不理我，小时候因为淘气，衣服总是脏兮兮的，同学也不愿意和我玩。对于未来，我不敢想象，因为大人们经常会说我，小时候不听父母管教，长大了就会進监狱等等。我就这样在没人关心理解、没人表扬鼓励的很负面的环境中成长着。

一九九八年春，当我看到妈妈开始学法轮功的时候，我就要跟着学。和多数人决定学法轮功不同的是，当时并没有人推荐我学，所以我是没抱有任何目地学习的法轮功。只是出于一个孩子天真的本性——直觉告诉我，这是一个好功法。因为从一个孩子的角度上看，法轮功修炼者面善，经常微笑，虽然他们也有各自的不足，但是他们绝对不会做损人利己的事。

就这样，我几乎每天早上五点钟起床跟着妈妈去广场和其他学员一起晨炼，晚饭后和大家一起读《转法轮》，然后再炼半小时第五套功法。看似做着简单而又平凡的事，但我的人生却因此发生了巨变。

不知不觉到了三年级时，我从“学渣”变成了好学生。在和同学的相处中，我不再象过去那样以自己为中心，通过学习《转法轮》我知道要珍惜缘份，我从心里接受了继父，我和妈妈身体变的特别健康，再也不用吃药了。偶尔继父也会跟着我们一起读《转法轮》。我感受到了家的温暖，全家人非常幸福和睦。不同社会阶层的法轮功修炼者都在努力按照真、善、忍去做好人，赢得了各界人士的敬佩与尊重。这段日子也是我人生中最快乐的时光。

#### 我所见证的九九年七二零

一九九九年“四二五”北京万人上访事件发生后，时不时就会有抹黑大法的报导刊登在报刊上。大家都意识到有必要把自己修炼大法后真实的转变向政府以及信访部门反应，让他们了解法轮功的真相。

一九九九年七月十九日，我和妈妈、姨妈一起去省会城市想跟领导们讲自己修炼法轮功的真实转变。到达时已近黄昏，领导们都已经下班了，于是我们决定第二天早上再来。在省领导办公楼前，越来越多的法轮功学员从各地赶来，大家的想法都是一样的，都是想和领导讲讲学法轮功后发生在自己身上的真实转变。

大概就是太阳快落山的时候我们走到一个广场，突然听到有人喊：看！天上有法轮！我们抬头，确实看到大法轮在头顶上空旋转，大家都拍手致敬，几分钟后法轮隐去，满天都是红色的祥云。作为修炼人，我们都很清楚这是慈悲的师父显神迹，对弟子的加持与鼓励。

因为身上没有多余的盘缠，很多外地法轮功学员都吃着最廉价的象饼一样的面包，我和妈妈与很多外地学員一起露宿街头。我盼着第二天快点到来，对和领导讲真相充满了期待。

天亮时，我们听到广场上有法轮大法炼功音乐，大家都开心的跑过去站好炼功。不到十分钟，几辆大巴车把我们包围，从车里出来大概四十多个持枪的防暴警察把我们围住，他们大声对我们咆哮：“抱头蹲下，不许动！”

看到警察的枪口正对着手无寸铁、祥和晨炼的我们，我完全不知道发生了什么事情，心中疑惑不解，为什么警察要枪毙我们？接着又来了几辆大巴车，我们被塞進车里，来到一所小学校。学校放暑假，就看到一车一车的法轮功学员被拉到这里。学员们先是在一个大礼堂，有个象公安局领导的人坐在讲台上，拿着纸宣讲，说的都是污蔑大法和师父的谎言。这时有人站起来高喊：“法轮大法好！”我小小的个子，坐在后排，还没等我找到是谁喊的，就看到前面的同修一个一个站起来，义正词严的说：“不许打人！”

接着又听到一个警察说：“你是老党员，要起带头作用，要跟法轮功决裂。”这个爷爷非常坚定的回答说：“那我退党。”

中午烈日当头，警察让我们排好队在操场上罚站。十一岁的我也不例外，我的汗顺脸而下，一个警察一会儿走到一个学员面前挖苦几句，又走到另一个学员面前讽刺一番。我记得他对一位瘦小的男学员说：你看你穷的饭都吃不起了，还炼法轮功？当我们结束体罚的时候，很多学员给那位男学员送去了吃的，我也送过去一个廉价的面包。那位男同修看我的眼神，那种正信者无畏一切，心中无怨无恨充满感激与善意的目光，我至今难忘。

晚上，很多教室里都坐着警察给我们单独做笔录，一个中年女警察问了我很多问题，我都如实回答，最后她又把我们之间的对话给我读了一遍，让我去签字按手印，签好后，女警察很为难的看着我，说：不行，你太小了，这个笔录不算了。我很纳闷，我自己明明说的实话为什么不算了？后来我才知道，这些签笔录的学员都被中共列入黑名单，作为重点迫害的对象。也许是中年女警察善良的一念，我没有被列在黑名单中。

连夜，当地的派出所就来车把各地学员拉回所在的城镇。那一次妈妈被拘留了七天。

#### 找到生命意义

在中共迫害法轮功修炼者的二十二年里，每个修炼人及家属都有心酸的故事。

我上初中时，母亲被关進女子劳教所迫害两年，家中的大法书籍都被警察抢走了。我生活在非常闭塞的农村，周围没有法轮功学员，也看不到任何法轮大法的真相资料。我的家人、老师不断给我洗脑，学校里宣讲着无神论、進化论，政治课宣传着斗争哲学。读初二时我有很多的疑问无法解开，于是我放弃修炼法轮功，决定做一个没有信仰的人。但是我渐渐意识到，当人没有信仰时，人世间便没有真理，万事没有衡量对与错的标准。制度法律更是如此，并不是公平的规范每一个人。

我很郁闷的苦苦思索，人为什么活着呢？我知道人应该珍惜生命，但是什么才是正确的人生目标呢？人活着为名吗？为利益吗？人世间的荣华富贵也就是几十年，生带不来，死带不去。那感情就更不用说了，到底人为何而生，难道这天地间就真的没有永恒不变的道理吗？我每天都在思索这些问题但不得其解。

渐渐的，我得了很严重的鼻窦炎，鼻塞，每晚张口呼吸睡觉，肺不好经常咳嗽，喉咙痛，耳朵痒，脑袋痛，颈椎痛，胃不能吃凉东西，痔疮，足廯，眼睛也经常发炎红肿。但病痛的折磨以及吃药、上药的痛苦我觉得都没什么，最让我痛苦的是我不知道人为什么活着。

突然有一天，一个念头闪过，我想看《转法轮》。这么多年过去了，我完全忘记了书中的内容。这个想法在当时那种情形下简直就是痴心妄想，所以我也没有和任何人提起过想看《转法轮》。但是神奇的是，很快大法书就送到了我的手上。时隔三、四年，当我从新读《转法轮》的时候，就象迷失的孩子终于找到了家，在读法的过程中，师父解开了我所有的疑问与心结：生命的来源，人活在世上的真正意义，在《转法轮》中我全部找到了答案。我的内心非常激动，我更加热爱生活，努力学习。我知道天地间衡量一切的标准是真、善、忍。在学《转法轮》的同时我也在看《九评共产党》。我明白了中共的本质，以及迫害法轮功的真相，就在我心中的疑问一个一个解开的时候，就在我努力按真、善、忍的标准改变自己的同时，也就是一两个星期的时间，我的身体又恢复到无病一身轻的状态。

我笃信法轮大法，坚信我们的师父李洪志先生，我的内心无比祥和平静，我会以最佳的心理状态处理工作、生活中突发的任何事情，也会高标准完成好工作，不用别人监督主动把事情做好。在自己不断按修炼人的标准提升的时候，也影响到我的家人和朋友，他们也坚信善的力量，克制自己的言行，不会为所欲为随波逐流。

长大后，我成为家乡一个小有名气的数学老师，无论是私立学校还是公立学校，老师们都对我特别认可，他们不仅把自己班里的学生推荐给我，还把自己的孩子也送来让我教。很多家长都说，孩子跟我学习数学后，不但成绩進步飞快，人也变得谦虚有礼貌了，真是跟啥人学啥人。能受到大家的尊敬和赞许，其实都是因为我是大法修炼人。

来到海外，除了通过给大陆民众打电话等途径帮助被中共谎言毒害的可贵的中国人，我也利用周末在唐人街讲真相。作为一个年轻人，我需要克服很大的心理障碍站在那里。因为年轻人总有一个好面子的心，当别人不接资料或者说一些不好的话的时候，难免会觉得下不来台。但没有什么比让被谎言蒙蔽的人们了解真相更重要的事情了。

#### 结语

风雨二十二载，我曾经认识的同修有被多次关押酷刑迫害的，有在监狱里做奴工折磨的，有被迫害致死的，我自己也经历过家破人亡的境遇。但是作为大法修炼者，我们的内心没有怨恨，因为迫害正信的，一定是邪恶的。法轮功学员之所以用最和平的方式讲真相反迫害，就是想让可贵的中国人了解真相，远离邪恶的政权，为自己选择一个美好的未来。法轮功修炼者没有政治诉求，当邪恶造谣抹黑大法修炼者与大法师父的时候，法轮功学员通过讲真相唤醒被谎言蒙蔽的世人。

愿迷中的世人快快找到真相，不要再上邪党的当。邪恶的迫害终会失败，愿真相大显时好人都平安。
