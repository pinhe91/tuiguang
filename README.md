### 同事都说我是个好人

文: 山东大法弟子喜缘

【明慧网二零二二年十月十九日】我是一九九四年喜得大法的。丈夫从小就爱好武术、气功，经别人介绍说法轮功很好，当时就给北京研究会去了一封信，说要学法轮功。过了几天，对方邮来《法轮功》一书，还有师父在济南讲法传功门票。到了六月份，师父传授班结束，丈夫带回来师父的讲法录音。我真是天天听，很多以前解不开的心结全都解开了，心里那高兴啊，逢人就说法轮功好。
学法轮大法以前，我身体很多病，如胆囊炎、牙痛、肚胀气，经常拉肚子；站的时间长了就受不了，仰脸向上看东西就头晕，还有月子病等等，整天无精打采，真是活的很苦、很累。学了法轮大法后，我一身的疾病全无，一身轻松，精力充沛，浑身有使不完的劲。是慈悲伟大的师父救了我，给了我美好未来，千言万语表达不了师父对我的佛恩浩荡。

#### 有惊无险

有一天，我骑电车上班，刚走到单位门口，突然里面冲出一辆电动四轮车，就听到一声闷响，两车就撞在了一起，我不知怎么就到了车的右边，也没受伤，也没害怕。我赶快问司机没事吧？她说我把她后车镜撞掉了一个。我问多少钱能修好？她说：那你给一百元钱吧。我就给了她一百元钱。这要是没修炼法轮大法我可真做不到。是大法的法理净化了我的身心，遇事为别人着想。

那人走了以后，我一看自己电车被撞变形了，车把儿都弯進去了。我一想，当时四轮车撞过来的时候，我应该在她的正前方，怎么我就躲开了正面撞击、到了车右边来了呢？要不是伟大的师父保护，后果真不敢想。谢谢师父救命之恩。

#### 放下利益之心

我的工作是计件工资制，就是多劳多得，少劳少得的管理方式。因为产品种类多，工作不好分配，所以每半年分工时，分配工就把产品分类，叫员工抓阄。这样，谁抓到好干的、量大的，谁就能多拿工资，当然谁也不愿意把自己的活让给别人。而抓到不好干的、量少的活的员工就经常没活干。这样同事之间就经常发生矛盾，搞的分配工也为难。

一次，我抓了数量有几十万的好货号，分配工找到我，要调我的货给别的同事。我说：行，你看着办吧。结果分配工把这个好的产品分的满车间都是。我也不去和分配工计较，因为我是按真善忍的标准要求自己的，放下名利之心，遇事先为别人着想。同事都知道我修炼法轮大法，都愿意和我一起干活，都说我是个好人。

#### 母亲在大法中受益

母亲今年八十一岁，她很相信神佛存在。因为她不认字，我就经常告诉她：念“法轮大法好，真善忍好”，保平安得福报。母亲就经常念这九字真言。我每次见到母亲就问她：念“法轮大法好，真善忍好”了吗？母亲说：我经常走着路念，在家里也念，晚上睡觉梦里还念呢。

母亲咳嗽，晚上睡不着觉，打了半月的吊针，输青霉素也不好使。母亲说她一合眼，就有人在她眼前说话，有认识的人，有不认识的人，使她无法入睡。母亲就念“法轮大法好，真善忍好”，瞬间那些东西都不见了，母亲就睡着觉了，病也好了。从此母亲更相信大法了，“法轮大法好，真善忍”九字真言不离口。
