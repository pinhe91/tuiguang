### 看看这些实例　谁还能说迫害好人没有报应？

文: 中国大陆大法弟子

【明慧网二零二二年十月五日】古人常说：“善恶有报”，现在的老年人大部份也相信这句话是千真万确的。可中共公检法司系统的警察，却一直都当作耳旁风。然而，这些年发生的许多恶报实例却是实实在在的，给不相信的人上了一堂课。
最近，明慧网上有一篇报道：《辽宁锦州市太和公安分局王超、吴亚凡等遭恶报》。

二零二二年九月中旬的一天，辽宁锦州市太和公安分局巡特警大队的警察季某，将自己的枪支让一个事业编的辅警临时保管（辅警是不允许配备枪支的）。这个辅警拿着枪就逃跑了。
据说此事惊动了公安部。锦州市公安局调动全员警察進行搜索，十天后才在市郊的一个高粱地里发现了该辅警的尸体，这个辅警开枪自杀了。

受这个事件的牵连，这个季姓警察已经被“双开”，并被刑事拘留；太和分局巡特警大队大队长和教导员二人被撤职，调离公安队伍；锦州市太和分局局长王超、政委吴亚凡二人均被免职；锦州市公安局局长、兼锦州市副市长张猛受到行政警告处分。

据报道，王超、吴亚凡自二零二一年九月份上任一年时间内，为争取分局系统办案第一，指挥该分局国保大队（包括下辖派出所）绑架法轮功学员三十三人，就连七、八十岁的老太太都被构陷判刑，给这些法轮功学员本人和他们的家庭带来了无尽的苦难。王超、吴亚凡等人积极参与迫害法轮功，报应之快，令人惊愕。

其实，类似这样的例子在明慧网已经报道过很多了。在此再列举几个锦州市公检法迫害好人遭恶报的例子，以警醒他们的同事。

例一：

据明慧网二零一六年七月十八日有篇报道：《升职八天　锦州警察刘文革遭恶报丧命》。

刘文革原在辽宁锦州市太和派出所任指导员。一九九九年七月中共迫害法轮功后，他认为升官的机会来了，就积极卖力迫害法轮功学员，追求所谓“政绩”。从一个指导员爬到了派出所所长，又到分局任职，再到市公安局“×教支队”任政委（中共才是邪教）。刘文革利欲熏心，一路往上爬，以迫害法轮功作为升官的垫脚石。因迫害好人，最终招来恶报：刚刚升职到市公安局“×教支队”任政委八天，二零一六年四月三十日，刘文革开车从市女儿河桥上坠落丧命。

例二：

二零一九年九月十九日明慧网刊登报道：《辽宁省锦州市公安局警察张海泉遭恶报死亡》。

张海泉生前积极追随江氏集团参与迫害法轮功学员。特别在王立军任锦州公安局长期间，张海泉伙同恶警王辉、陆昊、李嵋山、白宁等人疯狂迫害法轮功学员。他们先把法轮功学员利用特务手段抓捕，再操纵各分局、派出所進行劳教或判刑。有的法轮功学员被勒索巨款放回后，还被继续跟踪。

张海泉积极参与迫害，充当打手，罪行累累。一次在单位体检时医生发现他舌头异常，他和亲友立刻到北京看病。挂了第二天的号。结果当天晚上他就死在旅馆里。终年约五十八岁。

例三：

明慧网二零二一年十月三日报道：《从二十八天内锦州市两名官员相继自杀说起》。

文中的赵难，男，67岁左右，家住辽宁省锦州市古塔区万通公寓。中共迫害法轮功后，赵难在义县任职的三年半时间里，不遗余力地迫害法轮功，他经常在各种会议上布置对法轮功学员的迫害，叫嚣：“把法轮功（学员）都送监狱去。”

在赵难的邪恶指令和示范作用下，义县各级官员、警察纷纷参与迫害，这几年也是义县法轮功学员遭受迫害最严重的时期。二零二一年六月十一日，赵难从自家住所锦州万通公寓的顶楼二十楼跳楼自杀死亡。

例四：

明慧网二零二一年六月九日报道：《诬判多位法轮功学员　锦州市康赢健遭恶报死亡》。

康赢健，男，一九六五年出生，生前为锦州市太和区法院院长。在中共迫害法轮功的二十多年里，康赢健绝大部份时间在该法院任副、正院长。他罔顾事实和法律，积极执行中共对法轮功的迫害政策，与锦州市政法委及公安局、检察院的邪恶之徒沆瀣一气，制造了一桩桩司法冤案，诬判十多名法轮功学员（致一学员在狱中死亡）。作为法院的法定代表人、一把手，康赢健对这些枉法裁判负有不可推卸的责任。

二零二一年五月十四日早上，在辽宁省锦州市铁路南站站台附近的铁道线上，一个男子卧轨自杀身亡。从此人衣服兜里的工作证确认：死者为康赢健。康赢健在自杀前曾被辽宁巡视组约谈。

例五：

明慧网二零一五年十一月十七日报道：《惶恐中的悲惨结局——辽宁省锦州监狱副监狱长王洪博遭恶报自缢身亡》

一九九九年七月中共邪党之首江泽民迫害法轮功之后，王洪博成为江泽民的打手，而且一直不知改悔地干着。二零零九年以前，王洪博是凌源监狱的狱政处处长，也是当时凌源监狱迫害法轮功的主要负责人。二零零九年，王洪博被辽宁省司法厅厅长张家成提拔为锦州监狱的副监狱长，主管锦州监狱对法轮功的迫害。锦州监狱用各种残酷手段折磨、迫害法轮功学员，其中剥夺大法弟子的睡眠就是狱警最常用的残酷手段。锦州监狱为鼓励狱警与犯人参与迫害，给予物质奖励或减刑，他们便用老虎凳等刑具摧残多名坚定信仰的法轮功学员，对法轮功学员進行“严管”迫害，甚至不让家属探视学员。

据知情人士透露，二零一五年十月辽宁省司法厅厅长张家成被抓后，王洪博表现有些反常，经常有狱警拿文件材料到二楼办公室找他签字找不到人。

十一月六日下午，王洪博在自家的车库内自缢身亡，终年五十三岁。

这些年，从王立军、刘守林到郝铁君，从王洪博、张庆、刘文革、张海泉、张铁男到康赢健、赵难，这些人之所以死于非命，无一不是因迫害修心向善的法轮功学员而遭到的恶报。

古人很注重“修心行善”，有“养生先养德”的说法。人一生的福份，会因为个人的罪恶行为而改变。要想使自己的官位长久，“福禄寿”不在半路丧失殆尽，就得遵从古人的教诲，多行善事，积福德，自然就能达到福禄双全，延年益寿的目的。
