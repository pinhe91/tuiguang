### 参与迫害　河北三河市村官遭厄运十九例

【明慧网二零二二年二月十三日】（明慧网通讯员河北报道）河北省三河市是个县级市，地处京、津、唐的核心地带，与北京仅一河之隔。一九九五年，法轮佛法弘传到三河古城，以真、善、忍为核心，教导人们修心向善，无数人，包括绝症患者，修炼法轮功后获得身心健康。
一九九九年七月，江泽民出于小人妒嫉，挟持中共悍然发动了对法轮佛法的迫害。这场持续二十二年的血腥迫害中，三河至少23人被非法关押迫害致死；7人被迫害致残；18人（次）被非法判刑；至少260人次被非法拘禁在洗脑班迫害；更多的人被“敲门”“清零”等上门骚扰。

在国际上，迫害法轮功触犯的是反人类罪和群体灭绝罪。其行为一经曝光，就会上国际互联网法网恢恢恶人榜，每个上榜者都有恶人编号和详实的犯罪记录；几十个国家的法轮功学员，已经分批次将成千上万的行恶者名单递交给各国政府，要求依法对恶人及其家属禁止入境、甚至冻结资产等，他们将面临各种人间法律的惩处。

自古以来，迫害佛法修炼者罪大无边。中共迫害法轮功二十多年来，尽管中共当局极力掩盖，因积极参与迫害而遭恶报的事例仍然不断传出。

下面是三河市村一级官员遭恶报的19个案例，其中一例得闻真相、敬重大法，转为福报。

1、张希华，男，恶人编号E000055894，燕郊镇小胡庄村村长。利欲熏心，紧跟江氏流氓集团参与迫害。一九九九年十月份，吕春凤去北京上访，被当地警察拉回，劫持到三河看守所。有本村法轮功学员找到张，让他把吕春凤接回家。张却说：谁要是给我五千块钱，我就把她接回来。结果，吕春凤被非法劳教（后被迫害的精神失常）。家人去找村书记要给吕春凤送衣物，村书记同意让张开车去，张谎称车坏了，家人只好自己打车去唐山开平劳教所。当天夜里，张开着村委会的车，拉他的朋友到北京给小孩看病，汽车撞到北京八王坟附近的大望桥上，张当场死亡，他朋友一家三口都没有受伤。

2、冯士元，男，恶人编号E000150224，原段甲岭镇高家庄村书记。一九九九年开始积极配合监控、迫害本村法轮功学员，偷听、蹲坑，威胁不许发真相材料等；二零零零年指使村干部李小平带领警察，半夜翻墙跳院闯入大法弟子亲属家中，妄图绑架。二零零五年农历三月的一天上午十一点，冯士元骑摩托车去市医院，家人不让去，冯不听执意要去，在黄土庄沿口加油站附近被大货车撞的昏迷不醒、面目皆非，送三河市医院拒收，转北京两天后抢救无效死亡，才六十岁。冯的恶行殃及其妻二零二零年四月患脑血栓，被送养老院，二零二一年死亡；殃及其子二零二一年四月患脑出血住院治疗；冯士元不仅贪污腐败，还与儿媳妇多年不清不白。

3、于福，男，恶人编号E000150228，原黄土庄镇掘山头村书记。积极参与迫害法轮功学员，派人在门外偷听、监控本村大法学员，威胁不让发真相材料等，还经常半夜开车出去、妄图抓捕贴发真相资料的学员。二零一三年前后，于福几个人开车去唐山买海鲜，车门无故自行打开，其他人无事，只有于福一人掉下车，猝然摔死，才六十多岁。

4、陶德贵，男，恶人编号E000022196，原杨庄村治保主任。自一九九九年七二零以来，陶德贵紧跟江氏迫害法轮功，长期跟踪、盯梢、撕毁真相、多次谩骂大法师父、恶毒攻击大法。二零零三年二月，陶德贵死于突发脑溢血，才五十多岁。

5、周再生，男，恶人编号E000082927，原燕郊镇诸葛店村邪党书记，在高楼派出所上班时，就殴打过法轮功学员。周任村书记二年期间，配合当地六一零及公安人员，把四名法轮功学员绑架进拘留所、洗脑班、劳教所、监狱。法轮功学员多次给他讲真相，他不听不信，继续参与迫害。二零零六年十一月，周再生患肝癌死亡，才五十四岁。

6、吴永生，男，恶人编号E000082926，原李旗庄镇西杨庄村干部。曾撕毁大法真相，谩骂大法，为此遭报牙疼了一年多。吴永生不但不醒悟，还扬言：“我要是总书记，我把炼法轮功的全整死。”没几天，就得了尿毒症，肺部积水、下肢无知觉、嘴也不会动了；十几天后的二零零六年三月二十二日，吴永生在极其痛苦中死去。

7、朱宏（音），男，恶人编号E000082695，泃阳镇小曹庄村长。积极追随中共及江泽民流氓集团迫害法轮功，诋毁大法、破坏大法真相。二零一一年七月十五日，朱宏心脏病突发死亡，年仅四十六岁。

8、杨贺，男，恶人编号E000150239，燕郊镇诸葛店村主任，积极参与迫害本村法轮功学员。杨贺于二零一五年被抓，后被判重刑十六年。

9、杨春平，男，恶人编号E000014627，原燕郊镇诸葛店村支部书记，积极配合迫害抓捕法轮功学员，多次咒骂大法和大法师父，对善良的法轮功学员恶语相伤、迫害善良。二零零一年，因贪污被撤职，并令其退回贪污的赃款，他的儿子杨贺也锒铛入狱。

10、李永芹，女，恶人编号E000055887，原燕郊镇诸葛村代理书记，积极配合迫害抓捕法轮功学员，对学员罚款，多次骂大法、骂大法师父，还对善良的法轮功学员恶语相伤，不择手段进行迫害。二零零二年被撤职，身患多种疾病。

11、刘广生，男，恶人编号E000055889，原燕郊镇诸葛店村治保主任。任职期间，追随中共参与迫害，给大法造谣，谩骂大法和大法学员。二零零一年八月，刘广生指挥村民用钢刷把大法标语刷掉、摘掉横幅。法轮功学员善心告诉他们不要这样做，将来对自己不好，他们不听。二零零四年，刘广生在与他人争吵时突然倒地，一句话也没来得及说，几分钟内就暴毙，仅五十多岁。

12、李振福，男，恶人编号E000055888，原燕郊镇诸葛店村主任，积极配合迫害抓捕法轮功学员，盯梢、监控，给公安局通风报信，协助六一零去法轮功学员家中恐吓。二零零一年被撤职。

13、王强，男，恶人编号E000086539，六十九岁，原泃阳镇南关村书记。曾积极配合迫害法轮功学员，在镇党委会上发言恶毒攻击大法，还给法轮功学员家停电等。二零零三年九月被撤职，二零一八年患肺癌。

14、闫振明，男，恶人编号E000150247，燕郊镇东菜各庄村委会会计。经常带领不法人员到本村法轮功学员家骚扰、绑架，还主动代替法轮功学员签写不炼功保证。二零零七年前后，闫振明患肝癌死亡，才五十多岁。

15、贾广通，男，恶人编号E000082932，六十多岁，原燕郊镇诸葛店村村干部，积极迫害法轮功学员，诽谤大法师父和大法。贾广通患腰椎间盘突出，二零零七年，花了六万多元做手术，也未能治好，脑血管等处还有病。

16、王景义，男，恶人编号E000156563，原中门辛村书记，任职期间积极参与迫害本村学员，还派张淑青监视梁桂霞和王志茹。二零一六年十月，王景义病发死亡，才五十多岁。

17、邓朝玲，女，恶人编号E000073278，小胡庄村干部，积极参与迫害。邓朝玲曾欺骗法轮功学员说要看大法书，实际把书交给镇政府，没过几个月，她丈夫就得癌症死亡。

18、郭金华，男，燕郊开发区前赵村人，中共党员，当过民兵连长。二零一一年五月，大法弟子劝他三退（退出党团队组织）保平安，他说：“我费很大劲才入的党，我还想升官发财呐！”说了很多诽谤大法的话，犯下谤佛谤法大罪。刚过两个月，郭金华在焊一个大液化气罐时，发生爆炸：一条大腿被炸飞到二十多米以外；肚子被炸破，肠子流了一地；右手给炸没，血肉模糊，惨不忍睹，身体碎片有的被炸到一百五十米远，时年五十九岁。

19、孙某某，男，燕郊镇某村村主任，他母亲和小姨都是法轮功学员。一九九九年江氏集团搞迫害，孙某某母亲和小姨一起去北京为法轮功说公道话。孙某某知道后，就自己开车到天安门去找，将她们拉回家后，又怕镇政府知道，出于自保，将他母亲送镇政府非法关押三天，将他小姨送燕郊开发区洗脑班迫害多天。没过多长时间，孙某某出现肝病中期，并逐渐恶化。到二零零九年，人都快不行了，脸色煞白，头发也掉光了，整个人都脱相了；到北京大医院换血，一年至少花去十几万元。还找所谓神瞧（动物附体的巫医）看，能想到的办法都用了，也不见好转。法轮功学员多次给他讲真相后，孙某某终于明白真相，肝脏不再萎缩；经常诚心敬念“法轮大法好”，妻子也念，现在满面红光，头发又长了出来，生意兴隆，资产达千万元。

结语

法轮功，又称法轮大法，是一部佛家高德修炼大法，目前已经洪传世界一百多个国家和地区，让上亿的人道德回升、身体健康，福益于人类。

在江泽民发动的这场惨绝人寰的迫害中，那些曾经参与和正在参与迫害的人，从省、市到基层，明知法轮功学员都是善良的好人，为了职务、为了饭碗、为了自保，昧着良心犯罪。

奉劝那些被谎言欺骗还在参与迫害的人和不明真相的世人，别糊里糊涂追随中共一条路走到黑，为中共作陪葬。报应如影随形，绝不会缺席。人间的瘟疫就是针对道德败坏的人来的，希望你们倾听法轮功学员讲述的真相，停止迫害，择善而从，退出中共党、团、队组织，为自己和家人选择美好未来。把良知公义留给社会，把生命留给自己，把福德留给自己和子孙。
