### 明真相　家族人得福报

文: 中国大陆大法弟子 更新: 2022年10月31日

【明慧网二零二二年十月三十一日】我于一九九八年开始修炼法轮大法。我从小只念了三年书，上学时，还要捡柴、带弟妹、喂猪养鸡、做家务等，上学是三天打鱼两天晒网的，后来迫于生计辍学，常年辛苦劳作，很多字都忘了。修炼大法后，要读《转法轮》，有些字都不认识，但时间不长，师父把字的读音往我脑中打，很快《转法轮》能流利的通读了，《明慧周刊》也能看了，心中感恩大法的超常，为自己能成为师父的弟子而欣喜万分。
得法时间不长，中共迫害就开始了，我内心震惊：这么好的法，这么慈悲的师父，怎么能遭此诬蔑？我随同修们去省市上访，去北京为大法说句公道话，却遭绑架、关押、罚款，虽然这样，却更坚定了我修大法的决心。当时报纸、电台、电视铺天盖地的谎言使人们迷惑不解，甚至仇恨大法，对我们十分不理解。

我每天大量学法炼功，法能开启智慧，加强我的正念，我为自己在末世成为助师正法的大法徒感到荣耀，内心的喜悦使我眉眼含笑，别人见了我，都好奇问我：“你怎么这么高兴呢？家有喜事？”我自豪的说：“我修法轮大法了！”
我决定要向世人讲清真相，让他们从谎言中、从迷茫中清醒过来，从而得救得度。我首先从娘、婆二家亲人讲起，从“天安门自焚”骗局的疑点讲起。我说，头发很容易燃烧，王進东的脸烧得漆黑，可他的头发却没烧，完好无损，腿上的装汽油的雪碧瓶也没烧，也是完好无损；刘思影气管被切开后，不仅能清晰的讲话，还能唱歌，这有违常理。喝了半瓶汽油的刘葆荣老太太还能中气十足的讲话，肠胃却丝毫不受影响，这真是太离谱了！戒备森严的天安门广场怎么可能让人聚集了这么久，又自焚、又喝汽油，还拍下这么专业清晰的图片？没有专业的摄影器材，怎么可能完成呢？

我讲完后，家里人都恍然大悟，特别我娘家的大哥、二哥都是文化人，虽在体制内混，但他们心地善良，能明辨是非，对邪党的邪恶程度有了更深的理解。

我的亲人们明白了真相后，对自己的朋友、同事、左邻右舍在不同场合讲述了“天安门自焚”的真相，一传十，十传百，使更多善良的人明真相，得到救度。

我的亲人们明真相后，都不同程度的到福报。

#### ◎大哥拒绝“转化”我

我大哥是体制内的处级干部，他看到我修炼大法后，整天乐呵呵的，很佩服我们师父，说：“李大师了不起，我这个只念了三年书的妹妹，现在上知天文，下知地理，比我都有文化，不简单！我支持你修炼大法。”

我去北京上访，被非法关押在洗脑班，后来又遭迫害，被关押在市第一看守所，大哥没有责备我，“610”头目让他“转化”我，他说：“她原来脾气暴躁，现在整天笑口常开，相夫教子，家庭和睦，孝敬老娘，与人为善，妹妹有信仰，我这个大哥怎能横加干涉呢？开不了那个口哇。”

后来，“610”头目再也没有找过他。大哥大嫂夫妻和睦，相敬如宾，独子在省城成家立业，育有一子，生活和顺美满。

#### ◎二哥一家事业顺遂

二哥转业后，事业做得风生水起，独子也结婚成家，育有一男孩，家中房屋大小数套，他们都支持我修炼大法。

#### ◎老娘传九字真言

老娘是相信神佛存在的，常到庙里去。有一天，庙里有人心慌，面色苍白，我老娘对她说：“你赶快念‘法轮大法好，真善忍好’。我女儿炼法轮功，她让我常念‘法轮大法好，真善忍好’。我有天晚上心里十分难受，我就求李大师，说我相信法轮大法好，我女儿的经书和资料都在我家放着，我保存得好好的，我知道大法是冤枉的，现在我心里难受，请大师保护我，让我不要太难受了。就这样一念间，马上就好了，身体一点也不难受了。也请你们每天念九字真言，也会遇难呈祥，逢凶化吉的。”

当时那位老太太念了后，立刻恢复了正常。从此以后，那庙里老人每天念九字真言，还让自己家里人跟着诚心诚意念，众多人受益了。

#### ◎小弟弟挂了“法轮大法好”的横幅

我最小弟弟相信大法，一家人都办了三退。他家院墙上挂了“法轮大法好”的横幅，村里人去取，要拿走，他说：“你们走吧，我自己取，你们该干嘛干嘛去。”结果，他将横幅取下，挂到鸡场的围墙上。那年发鸡瘟，别人养的鸡都死了，弟弟家的鸡一只也没死，赚得盆满钵满，他到处说是挂了“法轮大法好”的横幅带来的福份。

#### ◎妯娌帮助发真相资料

我的妯娌，在人中头脑不太灵光，但她对我炼法轮功很支持，她常念九字真言，在邪恶猖獗时，我们经常被非法抄家，我将大法资料、设备等存放在她家，她热情接纳，还时常同我夜晚一起去发放资料，贴不干胶和标语，挂横幅等等。她家也得到了福报，娶了媳妇，生了一个胖小子，学习成绩始终名列前茅。我明白这是她相信大法，善待大法弟子得到的福报。

#### ◎侄儿放大法弟子平安回家

家族的一个侄儿在街道综治办上班。二零零八年邪党奥运期间，同修冤狱三年期满，“610”办企图绑架他到洗脑班继续迫害，当时侄儿是具体执行者。因为侄儿明白大法是好的，大法弟子是被冤枉的，所以在同修亲属们的强烈要求下，顶着巨大的压力，让同修登上家人的小车，平安归家，从而作出最明智的选择。

他后来身患主动脉血管夹层瘤，经抢救，平安痊愈。当时医生说，此类病患百分之九十八都不能存活，这真是神迹，一定是大菩萨在护佑！其实是侄儿保护大法弟子得到的福报。
