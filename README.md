### 诚念“法轮大法好”真受益

文: 中国大陆大法弟子

【明慧网二零二二年九月三日】去年我和妈妈去到超市，出来看见一个中年妇女在那儿讲，这个疫情把她害惨了。她说：“我背上三根肋骨都是骨质增生，又是类风湿，封城这两个月把我痛惨了，平时还可以去做按摩，贴膏药什么的，稍微好一点。在这疫情期间，按摩也没有，药也拿不到，简直痛死我了。”
听到后，我们就过去给她讲真相、劝三退。她说没有入党，只入过团、队。我说那你就对天发誓退出来，诚心念：“法轮大法好，真善忍好。”她说好，谢谢。

第三天去讲真相，我们见到她，她非常激动的说：“谢谢你们救了我，我回去一晚上就念好了。”第二天，她大女儿和女婿来，她叫他们也诚心念，大女儿不信，哪有那么灵？她给女儿说：“你看我昨天痛得腰都直不起来，今天这么精神，你是我女儿，我还骗你吗？这是真的呀，千真万确的呀。”
她说她大女儿两口子都有骨质增生，我就送了她U盘和一些真相资料叫她女儿看。她非常激动，再三说谢谢，我们给她讲：谢我们师父，师父是来救人的。

十几年前，我表妹的婆婆得了类风湿，瘫在床上大半年。我妈（大法弟子）回老家知道后，就去告诉她诚心念：“法轮大法好，真善忍好”，同时给她讲真相，让她退出邪党的一切组织。她就躺在床上真心念，有时自己还加几句说李大师救救我吧，我好可怜哟。不到一个星期，她站起来了，身体全正常了。

我回老家在半坡上碰到她，她激动不已给我讲这件事情，说你妈救了我的命。我给她讲：不是我妈救了你的命，我妈没有那本事，是师父救了你的命。她连连说：对对对，是李大师救了我。

我说你要告诉你的家人和你的亲朋好友哟，叫他们也受益得救哟。她说：我不敢嘛，我大儿子就是参加迫害法轮功的，大媳妇又是大队书记，这两个家伙是最狠毒的，从不管我的死活。我这大半年上吊三次，都是我小儿子把我救下了，虽然活着，但是活的非常痛苦，要不是你们告诉我这九字真言，我恐怕早就死了，谢谢你。

也是十几年前，我表舅妈得了直肠癌，在医院下不了床了，我妈知道后去给她讲真相，她明白了做了三退，叫她诚心念：“法轮大法好，真善忍好。”很快身体恢复正常了，身体好了。表舅妈就用现身说法去给人讲，她是诚念“法轮大法好，真善忍好”好的。到现在十几年了身体都好好的，每天非常开心。
