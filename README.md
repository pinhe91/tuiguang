### 善恶一念　天壤之别

文: 山东大法弟子　玉亮 

【明慧网二零二二年六月三日】我是一九九六年走入大法修炼的，在师父的慈悲保护下走到了今天，时时体悟到师父就在身边，无比的幸运、幸福，感恩之心，无以言表。经过二十多年风风雨雨的锤炼，我们越来越坚定、理智，感到大法修炼的无比神圣。大法的慈悲与威严时时展现出来。世人善恶的一念，决定着自己的未来！善良淳朴的民众，明白真相后，对大法弟子的那种纯真、诚挚的保护，对邪恶的抵制，那份善念永久留在我的心中。

#### 1、善待大法一念，天赐幸福平安

二零零零年春末，当地派出所的警察企图绑架我，引来了很多民众围观，有人直接大喊：绑匪来绑架人了。邻居王利（化名）的妻子直接上前扯警察的胳膊，我的妻子也正义相助，在她们的帮助下，我得以脱身，警察看此情形，围观的人越来越多，只好离开了。在当时邪恶铺天盖地的打压氛围下，她们能挺身而出、不畏强暴、正义相助、感天动地。她们无私的善举，也给他们带来了巨大的福报：现在她们家老俩口身体健康，子女生活美满，事事顺利，干什么什么挣钱，现在两部大挖掘机忙不停，真是人财两旺！

另一位是我村村民王明（化名）家住村边，二零零一年秋初，二十多名警察伙同镇政府和六一零人员，到处非法搜捕我。王明只有父子两人在家（儿子二十岁左右），我晚上好不容易到了他家，在他家呆了一天一夜。我叫王明告诉我家邻居老人，叫家人拿两件衣服来，因衣服实在没法穿了，我妻子送来衣服说，你快远走吧！他们那么多人白黑的监视你，到处找你、抓你。他们把村子的所有路口都堵住了。下半夜，在王利爷俩的帮助下，我从他家悄悄的出来，迅速進入了庄稼地，逃了出去，躲过了邪恶的绑架。二十多年过去了，现在王明儿孙满堂，家庭和睦，财运不断，个个身体健康，家门外摆着三、四辆大小车！这是善待大法弟子得到的福报！

#### 2、参与迫害大法弟子，结局可悲

相反，那些邪恶疯狂打压下顺从邪恶，助纣为虐，迫害大法弟子的；那些听信谎言，恶毒诽谤大法的，充当邪恶马前卒的，他们的结局是可悲的。

袁崇训不分青红皂白，多次配合邪恶、跟踪、监视大法弟子，不听劝阻，为了钱财、利益，良知泯灭，成了迫害大法弟子的帮凶。跟他讲真相一直不听，四轮拖拉机出了两次车祸，也不悟，那次我们三个遭绑架就是他跟踪告密造成的。最后我们又到他家跟他讲真相，他还不听，最后他得了胰腺癌死去！才三十多岁，家破人亡，老婆改嫁，墙倒屋塌房子都没了！

另一个是王贵（化名）民办小学教师，多次恶毒攻击大法、诽谤大法弟子。有一次大法弟子在村中贴了真相不干胶，他看到后破口大骂：抓住一个，枪毙一个……当时有个年龄比他大的人听不下去了，就说：人家贴是叫大家看的，你不看就拉倒，你说的不像人话。全镇民办教师没转正的就他自己，本来气管就不好，结果越来越严重，到后来腿也流脓、淌水，不久就死了，死时六十岁左右！

写出这些，希望广大大陆民众能分清善恶，明辨是非，做出正确的选择，不与邪恶为伍，赶快退出党团队，远离灾难。
