### 广州马民庆被枉判七年半入狱　妻子凄苦离世

【明慧网二零二二年九月十三日】（明慧网通讯员广东报道）广州市法轮功学员马民庆先生和母亲王雪祯女士，二零一九年七月九日被广州市公安局公共交通分局绑架，二零二一年分别被海珠区法院非法判刑七年六个月、四年，目前都被劫持到监狱迫害。他的妻子身患重病，失去照顾，二零二二年八月在无助中凄苦离世，留下女儿一人孤苦中生活。

![dfdzbphtp](https://user-images.githubusercontent.com/79625284/189860849-611c7c54-95e7-4ae4-9e7b-a41f623a6eca.png)

马民庆一九六九年出生，毕业于华南农业大学生物化学专业，现年五十三岁。因为坚持真、善、忍信仰，马民庆屡遭中共迫害。二零零一年，马明庆与妻子被迫流离失所、有家不能归。二零零六年七月，妻子被绑架、劳教迫害；马民庆同年十月被绑架，二零零七年七月被广州市越秀区法院非法判刑七年，非法关押到广东省四会监狱迫害，二零一一年底才结束冤狱回家。他回来后失去了工作，靠帮人送货维持一家人的生活。

二零一九年四月，广州地铁西塱站出现真相WIFI信号，地铁黄沙站派出所于二零一九年五月十三日非法立案。七月九日、十日广州市公安局公共交通分局侦查二大队对九名法轮功学员集中实施绑架，其中包括马民庆和母亲王雪祯。二零二一年十二月二十日，马民庆被广州市海珠区法院非法判刑七年六个月，被勒索罚金一万元。上诉后，二零二二年六月十三日，广州市中级法院非法维持冤判。二零二二年八月，马民庆被劫入广东省北江监狱。
马民庆的妻子郑洁儿曾修炼过法轮功，二零零六年七月，她被绑架，遭劳教迫害。从广州槎头女子劳教所回来后，郑洁儿放弃了修炼。二零一九年，马民庆和母亲被绑架时，郑洁儿已经患上了乳腺癌。丈夫和婆婆同时被绑架，郑洁儿失去照顾，于二零二二年八月前后，她在无助中凄苦离世。目前家中只有一个还在上学的女儿，孤苦的独自生活。

王雪祯一九四四年九月出生，现年七十八岁。二零二一年三月十七日，她被海珠区法院非法判刑四年，勒索罚金五千元。上诉后，二零二一年七月十五日，广州市中级法院非法维持冤判。后来，她被劫持到广东省女子监狱。

在海珠区看守所非法关押期间，自二零二零年六月份开始，王雪祯腰部剧烈疼痛，行动困难，严重时只能爬行。二零二零年十月份开始，王雪祯完全无法站立。律师会见时，王雪祯只能坐着轮椅，被人推着出来。在被迫害致残的情况下，王雪祯仍被劫入广东省女子监狱，导致病情加重。二零二二年二月二十三日，家属收到广东省女子监狱病情告知书，经监狱医院诊断，王雪祯患有：1、高血压（三级）；2、胸十二椎体及腰一、二椎体陈旧性压缩骨病；3、腰椎退行性骨关节病。目前她仍被非法关押在广东省女子监狱。

中共迫害法轮功二十多年来，无数法轮功学员被迫害致死、致残，也给家人带来重大伤害。马民庆一家人的遭遇，是广州市政法系统跟随中共迫害政策犯下的又一宗反人类罪行。

关于马民庆与他母亲遭受迫害的更多情况，请见明慧网报道《曾遭七年冤狱　广州马民庆又被非法判刑七年半》、《被迫害致无法站立　广州76岁王雪祯被判刑四年》等。
