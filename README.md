### 念法轮大法好　危急中保命

【明慧网二零二二年十月一日】

#### 邻居默念法轮大法好　转危为安

我是唐山市的一位大法弟子，我的邻居是一位六十岁左右的大姐，因急性胆囊结石、胆囊炎住院手术，切除胆囊，共住院十来天。在医院时，她就晕厥过两次，也未查出原因，在这种情况下，就让出院了。

回家后，她又晕厥了一次，并喘大气，活动后，更厉害，胸闷、气短。第二天早上，大姐去厕所，人就瘫在了厕所里，象软泥一样，被家人费了半天劲，才从厕所抬到了床上。大姐脸色灰白，毫无血色，全身冰凉，尤其手脚更厉害。用家里的血压表一测血压，根本就测不出来血压。
这时，我来探望，见此状，马上告诉大姐：“我知道你是共产党员，快退出来，保命啊！”大姐之前不同意退，她工作时，是单位的政工人员，被洗脑的厉害！此时，大姐说话都困难了，我又说：“你点头，就代表你同意了！”这位大姐赶紧点头，表示同意。

我抓紧说：“快快在心里默念‘法轮大法好，真善忍好’。你什么也不要想，只管默念，我们帮你联系120去医院。”

等120车来了，一测血压，血压有了！大姐喘息也平稳了。等到医院一查，考虑是肺栓塞，得赶紧转唐山市大医院。等到了唐山联合大学附属医院一查病情严重，又進了ICU。在ICU前三天，大姐失去意识三次，抢救了三次。然后，大姐转危为安，迅速好转。第五天，就从ICU转到了普通病房。

病房的主任说：“你活过来了，真是奇迹！我还以为你不行了呢！你可得好好谢谢你的邻居，要不是她你就完了。”

几天后，大姐出院，我问她都有什么感受？她说：当时，我就听你的，只管念法轮大法好，真善忍好，什么都没想，也没害怕，哪也不难受。现在就连做胆囊手术的部位也不难受了，做完手术后，吃什么都难受，现在吃什么都不难受了。谢谢大法师父救命啊！

现在，大姐天天都在默念法轮大法好，真善忍好！

#### 婴儿严重变形的头部变正常了

二零一六年七月的一天，我儿媳妇的弟媳生孩子，可是羊水破了，小孩生不出来。医院的两个大夫使劲挤压她的肚子，想帮助她把孩子生出来。结果，小孩生下来，头、脸严重变形，嘴巴张不开，吃不進东西。医生说有生命危险。

我儿媳妇哭着打电话给我，问我怎么办？因为我儿媳支持我修炼法轮大法，也认同大法，我就告诉她让所有在场的亲人都对着孩子念“法轮大法好，真善忍好”，孩子一定会好起来的。

儿媳听了我的话后，告诉家里在场的亲人，让他们天天对着孩子念。结果，孩子一天一个样，三天后，正常了！小脸正过来了，小脑袋圆圆的，不尖了，长的特别可爱。

医生看后，都觉的不可思议，三个月都不一定恢复正常，三天就正常了！在场的亲人们无不感到大法的神奇与超常。儿媳赶忙给我来电话说：“妈，一切都好了！孩子长的可漂亮了。感谢师父！”
