### 侄子保护大法书籍得福报

文: 陕西大法弟子 

【明慧网二零二二年五月三十一日】我是一个多灾多难的苦命人，有幸在一九九七年十一月喜得法轮大法，慈悲伟大的师父把满身业力的我从地狱里捞出来，净化了身体，创造了一个好的修炼环境。下面把身边几个曾经帮助过我的人明白真相得福报的事迹写出二例。
（一）

我的大弟弟俩口子人都老实，生了个侄子叫王选（化名），和他父母一样很老实，但傻呼呼的，说话还结巴，上学学不進去，也不好好上。一九九八年我哥来找我，叫我给王选侄在城里找个事，让他学着干干。我儿子高中的姓陈的好朋友，家里开了好几个商店，儿子就找他同学。他说，可以试一试。我把王选领去见了人家父母，勉强留下了试试，这样侄子就在那临时上班了。

一九九九年，江魔头迫害大法，抄家毁书，到处抓大法弟子。我所有的大法书，录音带、录像带等装了一大纸箱，打电话把王选叫来，我对他说：姑把这箱东西放在你那儿，你怕不怕？他说：我不怕。“不能告诉任何人，一定要保存好。”他说好，就把东西带走了。

从此王选改变了。师父看他保护大法书籍，给他开智开慧了。有时我去看他，老板比过去高兴了，对我说：你侄儿比过去聪明了，也能干，能吃苦，不胡来，我考验他几次，现在很放心。周围的商户也对我说：这娃娃实在，谁叫帮忙他都帮，有眼色、聪明，笑迷迷的，话也不多。我这才仔细看他：没有了以前傻样了，和谁说话都是笑呵呵的，说话也不结巴了。

二零零五年，旁边一老板看上了我侄子，把他的独生女介绍给王选当媳妇，女孩长的白净漂亮，王选把女孩领回村里，村上的人都炸锅了。我侄子当年结婚，有了一个可爱的女儿，也不给别人打工了，自己经营了个小食品仓库，给超市、商店、小卖部送货，生意红红火火的。国家鼓励二胎，媳妇又生了个儿子，小日子过的其乐融融。

（二）

二零零二年五、六月的一天晚上，我在农村的哥哥和弟弟半夜来我家，要接我回家去，我不知发生了什么事。原来是家委会的主任给他们打电话，让他们快来把我接走、说是明天邪恶要来绑架我，让我回家躲一躲（也不知道她从哪里知道我娘家（电话））。我跟他们回家了，第二天下午，果然来了两辆车、四、五个人，有派出所所长，单位领导，街道办事处的人，气势汹汹到我家，村里人不知道我犯了啥法，街道站了好多人，把我家人也吓的够呛。问我哥为啥把我接回来，我哥说父亲病重（当时87岁）在炕上躺着。我哥问他们到底咋回事？所长说我炼法轮功，怕我去北京，来看我是否真回家了。在家行，你们不要叫她到其它地方去，管好她。他们这一来，村里人都知道我炼法轮功了，家委会主任的一个善举，保护了大法弟子，她得福报了。

我在老家住的那段时间，央视天天放“天安门自焚”假案，村里好些人吓的不敢和我说话，我邻居一个小媳妇敢和我说话，问我法轮功的事。她说：电视放的是假的，法轮功不好为啥那么多学生、工人、大学教授都炼，都学。电视放天安门的事，我就把电视关了，不让丈夫和娃看。村里有人说法轮功“天安门自焚”，我直接就给怼回去了。她让我给她讲法轮功是啥，我就从我自身说起，原来满身是病，走路都困难，大夫说我五十岁就不能动了，瘫痪了，也可能就没命了。从此上山拜佛，学了好几种气功，花了不少钱也没啥效果。一九九七年十一月，炼了法轮功，学了《转法轮》全身病痛一扫而空，无病一身轻。法轮功只要你真学，祛病健身奇效。是真正的佛法，是千万年都难遇到的。作为一修炼人，首先要按照真、善、忍严格要求自己，做到打不还手，骂不还口，做一个比雷锋还要好的人。我还给她讲了我为还我师父清白去北京上访 。看到全国各地各行各业，不同阶层不同身份成千上万的工人、农民、学生、教授、军人等等去北京为师父还清白。为法轮大法正法的伟大壮举。

她很认真的听，对法轮功赞不绝口。我还给她讲了失与德、善念善行的法理，她很相信，也很爱听。从此以后，她给乡亲们讲，法轮功是真正的佛法，天安门自焚是假的。我给她伸出大拇指，好样的，你会得大福报的。她真得大福报了。我们村种菜，好卖的菜，她丈夫拉到远处去卖，剩下不好的，她骑三轮车四处村镇卖，一天又累又腿疼，现在她说，我每天念“法轮大法好，真善忍好”，骑上车子没感觉，不累，菜还卖的快，因为卖菜一个人，称菜算账，她不自己收钱，放个盒子让买菜的自己放進去，她不计较，有时回来一算，钱还卖多了，四处卖菜，碰见啥人都有，谁要说法轮功不好，她肯定给怼过去，说几句：不知道就别胡说，不好为啥那么多人炼？

在她家的变化影响下，我们那条街的人都明白真相了，都知道“法轮大法好”，我回去一次，带多少资料，都能发完。有的人来直接找我要，有的和我家比较近的人，把所带资料，光盘都拿走，他们在村里替我发，一会儿就发光了。这样的事很多。
