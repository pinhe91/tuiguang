#### 湖南省原耒阳市委书记廖炎秋遭恶报被逮捕

廖炎秋，原衡阳市政协主席，二零零二年九月至二零零六年十一月曾任中共湖南省耒阳市委书记，对任期内中共残酷迫害法轮功学员负有责任。

期间，耒阳市先后有法轮功学员曾昭秦、梁剑琴、廖高平、龙莲英、李英、谢会军、欧阳翠兰、艾菊英、伍文英、伍尤林、梁明英、刘冬生、张耀东、熊秋玲、陆秀花、梁年妹、梁士生等遭到迫害。

近日，湖南省检察院以涉嫌受贿罪对廖炎秋作出逮捕决定。

#### 湘潭市市委书记曹炯芳、原湘潭县县委书记傅国平遭恶报

二零二二年三月三日，湘潭市市委书记曹炯芳被“双规”，涉嫌贪腐和其它犯罪问题被调查。随后几天，湘潭县原县委书记傅国平在湘潭市任职时被“双规”调查。傅国平是二零二一年七月被调离湘潭县县委书记的，同一期间，湘潭市县有十多名大小官员被查办。

曹炯芳和傅国平被查办，彰显天理“善有善报，恶有恶报”。因为湘潭市大法弟子莫利琼于二零二一年一月二十五日在湘潭县杨嘉桥镇被绑架，且一直被非法关押在湘潭市马家河看守所未放，正是曹炯芳和傅国平作为市县一把手在职期间发生的，他们负有不可推卸的领导责任。

#### 河北沧州检察院原检察长杨浩遭恶报被审查

二零二一年十月十四日消息，原河北沧州市检察院检察长杨浩，因涉嫌严重违纪违法被“双开”。杨浩在任职期间，对当地法轮功学员受迫害负有责任。

杨浩从一九九二年开始，在河北唐山等地检察院工作，曾任唐山市开平区检察院检察员、副检察长；玉田县检察院检察长；二零零七年七月起，任唐山市检察院副检察长、反贪局局长；二零一三年三月起，任秦皇岛市检察院党组书记、副检察长、检察长；二零一六年十二月起，任沧州市检察院党组书记、副检察长、检察长、一级高级检察官。

杨浩在任唐山、沧州和秦皇岛检察院检察长期间，对唐山钢铁公司工程师陈立武、秦皇岛的徐秀娟、沧州的曹延香等法轮功学员被迫害负有责任。

#### 安徽省女子监狱原四监区教导员张辉遭恶报死亡

安徽省女子监狱是安徽省迫害女性法轮功学员的黑窝，采用长时间剥夺睡眠、不给吃饭喝水、野蛮灌食、铐、捆绑、电棍电击、毒打、打毒针、关禁闭等等酷刑，将多名法轮功学员致伤、致残、致疯、致死。详见《安徽省女子监狱对法轮功学员的迫害》。

该黑窝共有二十多个监区，没有专门迫害法轮功学员的监区。每个监区都参与迫害。十四监区为监狱医院。

张辉曾多次被评为“优秀党务工作者”、“个人三等功”、“二零一八年安徽司法行政年度榜样人物”提名奖、二零一九年司法部 “全国监狱工作先进个人”。

长期参与迫害的原四监区教导员张辉，已于二零二二年一月九日遭恶报，患癌症死亡，终年四十九岁。
