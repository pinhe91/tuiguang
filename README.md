### 德国十位政要致函谴责中共迫害法轮功

【明慧网二零二二年八月三日】（明慧德国记者站报道）今年是法轮功反迫害二十三周年，包括德国欧盟议员、国会议员和州议员在内的十位政要致函，谴责中共迫害法轮功。

![image](https://user-images.githubusercontent.com/79625284/182575536-62d736f6-8b10-4f3e-8f24-9e0631528f8f.png)

图1：法轮功反迫害二十三周年，包括德国欧盟议员、国会议员和州议员在内的十位政要致函，谴责中共迫害法轮功。

图上从左至右：欧盟议员马库斯·费伯（Markus Ferber）、国会议员弗兰克·施瓦博（Frank Schwabe）、国会议员乔纳斯·盖斯勒（Dr. Jonas Geissler）、德国议会议员萨宾娜·韦斯（Sabine Weiss）、国会议员约尔根·布朗（Jürgen Braun）；

图下从左至右：国会议员路易丝·阿姆茨伯格（Luise Amtsberg）、德国议会议员伯克瓦尔德·马蒂亚斯（Birkwald W. Matthias）、德国国会议员克里斯托弗·德·弗里斯（Christoph de Vries）、北威州议员本杰明·劳尔（Benjamin Rauer）、北威州议员马库斯·奥坦德伦克（Dr. Marcus Optendrenk）。

#### 欧盟议员费伯：欧洲人不能对迫害视而不见

欧盟议员、前欧盟议会基社盟欧洲集团主席马库斯·费伯（Markus Ferber）在信中提到，中共从二十三年前开始一直严厉打击这个修炼群体（法轮功），甚至有指控说被监禁的法轮功学员被系统性地盗取器官。在过去几年里，中（共）国显露了令人担忧的一面，我们欧洲人不能对此视而不见。

他并在信中说：中（共）国政府必须尊重法轮功学员的权利以及其他宗教和少数民族成员的权利，并立即停止目前的非人道待遇。在欧洲和国际层面，我们必须继续保持这些价值观并捍卫人权。欧盟及其成员国必须继续公开捍卫少数民族权利并谴责中国的活摘器官问题，同时呼吁中（共）国让步。我将继续在欧洲层面朝着这个方向努力。

#### 联邦政府宗教和信仰自由专员：会为维护法轮功权利而努力

国会议员、联邦政府宗教和信仰自由专员弗兰克·施瓦博（Frank Schwabe）在信中提到，这个日子提醒着人们想起（中共）针对法轮功学员犯下的罪行。我们必须全力以赴提醒人们关注这些人在中国的悲惨命运。

他说：骚扰、监禁、杀戮……二十多年来，中（共）国政府一直在对法轮功进行极其严厉的迫害。我对近年来数千名法轮功学员遭到非法拘留、酷刑和谋杀的报道深感关切。另外还有一项（对中共）令人难以想象的指控，那就是非法摘取法轮功学员器官。

他表示：我再次呼吁中（共）国政府遵守国际法，保护人权，包括宗教和信仰自由的权利，并公开在中国是如何获得供体器官并进行移植的。允许国际社会自由进入法庭听证会和监狱——这是（中共）应对这项非常严重的指控的唯一途径。

他最后说：请记得，你们有事可以继续找我，我会与你们一起为维护法轮功的权利而努力。

#### 德国国会议员谴责中共活摘器官

国会议员、选择党人权事务发言人约尔根·布朗（Jürgen Braun）在信中写道：二十三年前，中共开始有系统地迫害法轮功学员。一九九九年七月二十日，（时任）中共领导人江泽民提出“名誉上搞臭、经济上搞垮、肉体上消灭”，并发起了一场残酷的运动，致使数十万人被监禁、折磨和杀害。这种情况一直持续到今天。仅去年一年，就新发生一万起法轮功学员被抓捕和关押的事件。

他在信中谴责中共迫害法轮功。并说：多年来，一直有人指责（中共）国家领导层利用良心犯作为器官捐献者。受害者除藏族、基督教和维吾尔族人以外，主要是法轮功学员，因为他们生活方式健康，器官良好。

国会议员、基社盟（CSU）人权委员会成员乔纳斯·盖斯勒（Dr. Jonas Geissler）在信中谴责中共残暴与不人道的行为。他说：任意逮捕、强迫劳动、酷刑和摘取人体器官是必须受到惩罚的反人类罪行。

他表示：法轮功学员一再成为活摘器官的受难者，这是有史以来最令人发指的侵犯人权罪行之一。

他说：作为国会议员：我们不会停止抗议中共高层这种大规模违反国际法的行为。我要呼吁我们在德国和国际上的朋友们：我们也将在中国维护人权，将看到中共压迫和暴行的终结。

#### 德国议会议员韦斯衷心感谢法轮功学员不懈努力反迫害

德国议会议员萨宾娜·韦斯（Sabine Weiss）在信中写道：首先，我要对你们所有人表示衷心的感谢，感谢你们通过不懈的努力为人权树立了如此重要的榜样，感谢你们为所有不能让别人听到自己声音的人发声。

她说：对法轮功的残酷迫害已经在中国持续了二十三年。这些事实和数字不仅令人震惊，而且令人恐惧。

她还说：迄今为止，已确认死亡人数超过4800人，估计没有统计的死亡人数还有数万。维吾尔人和哈萨克人等其他少数民族也受到系统性的迫害和骚扰。超过一百万受迫害的人被关押在所谓的“再教育”营中，每天不得不忍受令人发指的侵犯人权的罪行。

她表示：我认为我们迫切需要在与中（共）国打交道时采取更明确的政策。那里的人权状况很糟糕而且还在继续恶化。少数民族在那里被系统地灭绝，侵犯人权的行为必须被指出来，并必须受到比以往更严厉的制裁。

她还提到：欧洲议会于二零二二年五月五日的决议谴责了中（共）国多年来发生的强制摘除器官的行为，这无疑是朝着正确方向迈出的一步。重要的是，由于在世界面前公然犯下的这些反人类罪行，中（共）国领导层被公开点名。

她表示：一、中共必须遵守联合国《世界人权宣言》，执行已签署的人权公约；二、中共必须尊重和保护少数民族的一切权利；三、中共必须最终停止​​对和平冥想运动法轮功的迫害和镇压。

她还说：我衷心感谢你们所有人，我请你们不要松懈对维护人权的责任感。这也是对我们的帮助和支持！

#### 国会议员阿姆茨伯格：

国会议员、外交部负责人权政策和人道主义援助的联邦政府专员路易丝·阿姆茨伯格（Luise Amtsberg）在信中写道：自一九九九年以来对法轮功的残酷迫害显示了中国宗教自由的紧张状态。有关数千名法轮功修炼者死亡的可怕报告表明，（中共）系统性、非自愿摘取器官的嫌疑远未消除。欧洲议会在五月份的一项决议中呼吁中方和独立观察员进行全面调查，在迫害周年之际，我附和欧洲议会的要求。

#### 德国议会议员马蒂亚斯：祝愿你们的努力取得成功

德国议会议员伯克瓦尔德·马蒂亚斯（Birkwald W. Matthias，Die Linke）在信中写道：我祝愿你们的努力取得成功，希望你们能在德国和欧洲的政界中得到关注，希望能找到合适的方法和手段来结束中（共）国对法轮功修炼者的迫害。

他说：我没有宗教信仰；但对我而言，与我自己没有信仰的自由同样重要的是，每个人都应该能没有恐惧的和平的生活，并实践自己的信仰。

#### 德国国会议员弗里斯：（中共的）罪行必须被制止

德国国会议员克里斯托弗·德·弗里斯 （Christoph de Vries）在信中写道：中国法轮功运动仍被迫害，其成员被任意逮捕和打压，这是个严峻的事实。最近泄露出来的有关维吾尔族人经历的信息表明，中共对所谓的异议人士采取了肆意的、残忍的和蔑视人性的行为。

他认为，这些调查结果令人震惊。您提到了欧洲议会的决议和摘取法轮功学员器官的证据。这些罪行必须被制止，并对其进行独立调查并处理。

他最后说：我相信，自由和人权最终会获胜。要做到这一点，我们不能胆怯地屈服于独裁者的压力，也不能放弃对基本自由和人权普世性的理念。

#### 北威州议员劳尔：我与法轮功修炼者站在一起

北威州议员本杰明·劳尔（Benjamin Rauer）在信中写道：在中国法轮功学员遭受迫害二十三周年的悲痛时刻，我祝愿所有法轮功学员能拥有巨大的勇气和力量。

他说：我与法轮功修炼者站在一起，并承诺在任职期间尽我所能提供支持。我牵挂着遭受迫害的人们，并向他们及其家人表示最深切的同情。

#### 北威州议员奥坦德伦克：我们不能视而不见

北威州议员马库斯·奥坦德伦克（Dr. Marcus Optendrenk）在信中表示赞赏和感谢法轮功学员反迫害的努力。

他在信中写道：我们非常关切地注意到，在世界许多地方，对人的尊严和人权的理解与我们并不一致。中（共）国对法轮功学员的迫害就是这种世界观分歧的例证。

在法轮功学员反迫害二十三周年之际，奥坦德伦克表示，我们要纪念成千上万因（中共）这种无视人权而受害的人。

他说：作为一个将人的尊严、身体完整和信仰自由视为共同生活的核心要素的自由社会，我们不能视而不见。我们必须引起人们对错误行为的关注，并抵制这种（错误）理念。
