### 吉林法轮功学员尹志波在家被警察推出窗外坠楼身亡

【明慧网二零二二年九月十八日】（明慧网通讯员吉林报道）吉林延吉市法轮功学员尹志波，二零二一年三月十六日在家被警察朴东杰及协警孙海涛推出窗外、坠楼身亡，终年44岁。

![image](https://user-images.githubusercontent.com/79625284/190894241-03f90fb0-1306-4fa7-92f7-141c1f8a05d6.png)

图：尹志波一家

尹志波，生于贫困的农民家庭。他天资聪颖，读书成绩比哥哥和姐姐好，是家人的骄傲。他喜爱机械，技校毕业后立志要用所学技术帮助乡亲们增产增收。乐于助人、勤劳忠厚的他每到春耕、秋收、农忙时节，他加班加点先帮乡亲们抢时、抢种、抢收后才干自己家的农活。因此错过了最佳农耕时机，每年他家的产量都少于同乡们。一九九六年他白手起家创办起德新农机租赁公司，拥有了拖拉机、收割机，播种机、铲车、玉米脱粒机、耙田机等农机，总资产超过50万元。勤劳善良的他在朝阳川镇是家喻户晓的助农增收致富带头人，为人豪爽大方。

尹志波于一九九九年三月份开始修炼法轮功，更加善良。他学法、背法，弘法、炼功都非常精进，并在一九九九年七月中共邪党迫害法轮功后，以各种方式向世人讲清真相，揭露邪恶，救度众生。
二零二一年三月十六日，尹志波被受谎言毒害的人诬告，中共警察在他家中翻出大法书籍和《九评共产党》等资料，他试图掩护资料及书籍时与警察朴东杰及协警孙海涛发生争抢。当时身穿睡衣的尹志波被朴东杰、孙海涛二人殴打推搡至窗边，在抗争中被二人推出窗外，坠楼身亡。

尹志波的妻子王秀华当时看到丈夫在家中被警察与协警袭击，被推出窗外坠楼身亡的丈夫尸体时，无法面对丈夫惨死的现实，伤心欲绝，几度昏死过去。他年幼的儿子尹东东趴在父亲的身上试图唤醒爸爸。

邻居们纷纷谴责推人坠楼的警察朴东杰、孙海涛二人，要求其偿命、要求政府给刽子手判刑。在有证人证言和众目睽睽之下的犯罪事实面前，延吉公安局却给出高空坠落的身亡证明，而不是故意伤害致死的真相。

家人们纷纷为其鸣冤鸣不平，却遭到村、乡政府和公安部门的威胁与恐吓。他的哥哥尹志丰多次为弟弟上访鸣冤，均被遣返回家，并收到村委会的威胁，若再为尹志波上访将没收土地，终止承包合同的通知。他的妹妹尹志萍多次到中国驻洛杉矶总领馆举牌抗议：要求中国大陆公安检察机关“彻查尹志波死因”。七月十五日他的外甥徐小刚因被怀疑是法轮功学员被无辜拘留15天。

延吉市朝阳川镇68岁的法轮功学员朴光勋老人，被迫害致残，很少出门，二零二一年二月六日遭延吉市进学派出所的几个警察威胁说：“不要上外边。”之后，二月末，他就失踪了。至今一年多，他音信全无。

据明慧网不完全统计：截至二零二零年十二月三十一日，延边地区法轮功学员被骚扰、绑架、抄家、经济迫害、非法关押至少1432人次：其中被强制洗脑至少801人次；被非法劳教学员至少146人次，劳教期1年至3年；被非法判刑的学员已知姓名的有89人次，刑期为6个月至13年；被迫害离世、致死的法轮功学员明慧网已经核实的有50人。

时至今日，尹志波被迫害去世已经一年多，尸骨未寒，真正的杀人凶手朴东杰还披着公安警察的外衣逍遥法外！但善恶必报，朴东杰、孙明海的罪责难逃！你们所做的坏事，历史都有记载，你们所干的坏事，都被记录在案。
