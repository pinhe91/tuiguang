### 北京赵春英被绑架一年半至今情况不明

【明慧网二零二二年六月二十八日】（明慧网通讯员北京报道）北京市朝阳区垡头地区法轮功学员赵春英，约于二零二零年八月十二月被当地警察绑架，非法关押在朝阳区看守所，至今已逾一年半，她仍然情况不明。
在二零二零年，赵春英就几次被骚扰、抄家、绑架。大约于二零二零年三至五月份期间，中共对法轮功学员实施“清零”迫害，她被绑架、非法抄家，并被强制体检，后因高血压症状被取保候审，回到家中。二零二零年十二月十日左右，派出所警察再次绑架赵春英，称所谓“收监”，将她非法关押在朝阳区看守所。

#### 赵春英修炼法轮功经历

赵春英于一九九八年开始修炼法轮功，一九九八年十月她在回家的路上看到很多人在炼法轮功，出于对传统文化的喜爱，她被深深地吸引了，上去询问后了解到法轮功是佛家上乘修炼大法，以宇宙最高特性真善忍为根本指导，要求修炼人真诚、善良，遇事忍让，并且配合五套简单易学的功法，修炼法轮功对稳定社会，提高人们的身体素质和道德水平有不可估量的正面作用，在“真善忍”的感召下，她走上了修炼法轮大法的道路。

修炼法轮功之后，赵春英的世界观发生了天翻地覆的变化，知道了做人要真诚、善良、忍让，看淡名利，返本归真，她的心胸开阔了，常年困扰她的萎缩性胃炎也在不知不觉好了，过去常犯的小毛病也不犯了，干活也不累了，得法后的赵春英，体会到久违的无病一身轻的幸福，丈夫看到她的变化也很高兴，家庭自此和和睦睦，其乐融融。

#### 赵春英遭迫害经历

但是没有想到这么好的功法，在一九九九年七月被以江泽民为首的中共政治流氓集团非法取缔，时任中共党和国家领导人的江泽民因为一己之私、嫉贤妒能，污蔑万古难遇的高德大法，并对成千上万的法轮功修炼者实行“肉体上消灭、经济上搞垮、名誉上搞臭”的灭绝政策，迫害气势如同文革再现，赵春英的丈夫被吓坏了，把家里的法轮大法书籍上缴了，为了让妻子放弃信仰，常常打她，并以离婚威胁，最终导致了赵春英的婚姻破裂，好端端的家庭就这样被江氏集团迫害拆散了。

离婚后的赵春英成了无家可归的人，年近五十岁的她失去了家庭和家人的呵护，无依无靠地搬回了单位的宿舍。

二零零八年，北京即将召开奥运会，举国欢庆，但是在这座城市光鲜的外表下却掩盖了很多百姓和家庭的苦楚和血泪，赵春英被不明真相的人举报，二零零八年六月三十日下午正在单位上班时 ，两名警察和一名保安闯进赵春英的单位，在没有搜查证的情况下，不分青红皂白的乱翻一气，把赵春英的电脑、光盘、书籍、资料等非法抄走，并在第二天把赵春英绑架到西城看守所，看守所的小牢房里，十多人关押在不到十平米左右的空间里，平时吃饭、睡觉、上厕所都在这里进行，因为空间太小，睡觉都是侧着睡，不能翻身，每天早上一睁眼，就被要求坐小板凳一直到晚上睡觉，看守所里的卫生纸、盥洗用品等生活品比市场上普遍高出两倍以上甚至更高，例如：市场上六角钱一卷的卫生纸在那里卖到二元钱，被褥是用过的被褥，但是价格要一百八十元一床。

一个月之后，赵春英被非法劳教，先是被非法拘禁在调遣处，那里俨然是人间地狱，一进入就被要求双手抱头蹲在地上，人格尊严尽失，尽管赵春英这些法轮功学员平时在社会上是遵纪守法的好公民，修炼法轮大法后更加严格要求自己做更好的，品格高尚的人，但是到了这里，不分青红皂白被要求认罪认错。在那里几天不让洗衣服，夏天的汗渍渗入衣服糊在身上很难受，走路不让抬头，上厕所限制时间，被虐待、折磨、失去尊严了一个月后，在某一天，赵春英等人被戴上手铐，又被一辆大巴车载着挟持到北京大兴区的北京市女子劳教所（自中国大陆取消劳教制度后，也相应取消），在那里赵春英被强制洗脑，期间被干警指使吸毒犯剥夺基本人权，非法限制睡觉和上厕所，长期保持一个姿势坐小板凳，不许动，稍微一动就拳打脚踢，不放弃信仰就不许见家属和采买（在劳教所的监管下采购日用品和食物，这些采购的商品价格一般都比市场价格贵），不许打亲情电话，不许给家里写信，此后的日子里还被要求参加各种奴工活动，如果不按她们的要求完成奴工劳动就会被说成是消极改造，受到包括吸毒犯等人的欺压。

二零一零年六月二十九日，赵春英终于走出了劳教所，但随后就被海淀区“610”办公室非法拘禁了半个多月。在拘禁期间，二十四小时轮流有人监视她，白天她被专门“转化”法轮功修炼者的所谓“帮教”洗脑，这些人不具备教育他人的资格，她们既不帮助赵春英，有病也不联系治疗，而是采用侵犯他人自由和尊严的手段折磨赵春英的身心，经过两年非法劳教，赵春英身体已经大不如前，被她们这么一折腾，旧病复发又添新病，白天肚子疼，拉肚子，晚上睡不着觉，整天咳嗽、气喘。“帮教”和“610”人员看她实在不行了，才还给她自由，但是每到每年的四月二十五，七月二十日等敏感日子，海淀派出所的片警就把赵春英叫去或打电话骚扰，干扰老人的正常生活，给她的身心带来很大的伤害。

为了躲避这种每年数次的骚扰，赵春英退休后独居在北京朝阳垡头地区，并于二零一五年顺应全球的诉江大潮，堂堂正正的起诉了迫害法轮功的元凶江泽民，其后在中共对法轮功学员的“清零”运动中被多次骚扰，二零二零年八月期间，赵春英再度被垡头警察骚扰，后被北京市公安警察绑架，此后一直没有下落，周边的人只知道赵春英有个女儿在海淀工作，但是因为害怕迫害政策受牵连不支持老人修炼，联系方式未知，打听不到她母亲的具体情况。

自从一九九九年中共和江泽民流氓集团抹黑法轮功，残酷迫害法轮功修炼者，至今已有二十多年，期间更有中共活摘法轮功学员器官卖钱的真相被揭露，这旷日持久的残酷迫害给象赵春英、雷中富等上亿法轮功修炼者和他们原本美满和幸福的家庭造成了难于愈合的创伤，给他们带来了巨大的经济和精神损失以及无尽的苦难。如今，赵春英在被强制体检、并被要求签字后被非法抓走，至今下落不明，这足以令每一个有良知的正义之士听闻后都深感揪心。

众所周知，法律是对犯罪行为的惩处，赵春英和其他法轮功学员没有任何违法犯罪行为，恰恰相反，他们按照真、善、忍的标准做一个好人，不计名利、踏实肯干、任劳任怨，他们以自己的实际行动善化着周围的环境，是社会稳定的中坚力量，相反，对他们的打压是对这个社会最善良民众的打压。
