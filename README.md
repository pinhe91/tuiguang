### “法轮大法好”喊声震天　危儿死而复活

【明慧网二零二二年七月十六日】

#### “法轮大法好”喊声震天　危儿死而复活

〖中国大陆来稿〗三义是位法轮功学员，他的家人都明白大法真相。

那是二零一三年大年三十，临近中午，正忙着做年饭的三义，突感身体不适，继而人事不省。当时家里只有大妹和八十八岁的老母，娘俩被眼前的状况吓得不轻。大妹赶紧拨打120急救车，同时通知其他的亲人们。八十八岁的老母亲很果断，她拽着儿子拼命的喊：“法轮大法好！法轮大法好！”大妹也同时在喊“法轮大法好”。

十分钟内，亲戚、朋友相继赶来，又掐人中、又摸脉搏。摸脉搏的朋友说：“没脉了。”再看瞳孔：“瞳孔也散了，人已走了。”老母亲就信大法能救儿子，一直不间断的高喊：“法轮大法好！法轮大法好！”

120救护车来了，救护人员把三义抬到担架上。老母亲急了，声音更大了：“法轮大法好！法轮大法好！”那真是震天动地呀。三义一下醒过来了，而且坐了起来。全屋的人惊呆了，大家好高兴，在场的十几人都感知到：“法轮大法好”神了。

三义回忆当时情形说：“我听到一声‘法轮大法好’，就在心里回应‘法轮大法好’，随即就感觉很大很厚的物质一下散开了，我立刻恢复了知觉，坐了起来。是师父救了我，我好感激！”

#### 断牙长齐了

〖中国大陆来稿〗我在小学一年级的时候，曾摔过一跤，刚换的门牙被摔掉了一个斜茬，门牙向外呲，从此牙齿不整齐，嘴上下闭不严。

一天，我丈夫告诉我说：“你的牙齿比年轻时还整齐。”我照镜子一看，果然是。我反问他：“我的牙是怎么齐的？”他认真的说：“是你师父给弄齐的。”

我丈夫不炼功，我让他跟我念“法轮大法好，真善忍好”，他就跟我念，结果他的胆结石也没了，心脏病也好了。

亲朋好友聚会时，我一讲真相，我丈夫就给我证明：没病、不用吃药、牙还长齐了。大家都称奇：“这年龄不掉牙就不错了，还把断掉的长出来了，大法真的很神奇！”

#### 修炼人的心胸

〖中国大陆来稿〗一九九五年末，我和丈夫开了一个公司。后来，丈夫的大嫂因车祸去世了，他侄儿就到了我们公司工作。我拿他当自家人，教他学习办理银行业务，教他写支票；一日三餐好吃好喝的，怕他思念去世的妈妈。

一天，他趁我家里没人，偷走了十几张支票，将所有家电洗劫一空，分别送到几家当铺给当了。我发现后没有报案，也没有生气。公司的员工很奇怪，问我：受这么大损失，你俩也不打仗，还乐呵呵的。我说：“炼功人讲真善忍，我善心待他，他会明白的。”对此，我丈夫、丈夫的大哥及我公公都很感动。
