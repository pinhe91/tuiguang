### 这九字真言太灵了

文: 大陆大法弟子口述　同修整理 

【明慧网二零二二年八月二十一日】一次我在外边遛弯儿，碰到同一小区的一位退休老师，跟她讲起了大法真相。我说，我是炼法轮功的，原来身体不好，有多种疾病，胃病、肝病、心脏病，还是个家属，没有固定收入，治病治不起。孩子让我练气功，没有效果，后来朋友推荐法轮功，我就试试看，按照书中要求，与人为善，遇事要忍，不争吵，做个好人，这样一点点去做。老头脾气不好，也不跟他生气了，也能忍了，身上这些病不知不觉都好了。
说到这，我又问这位老师入过党团队没？继续揭露邪恶：共产党讲无神论，与天斗、与地斗、与人斗，害死八千万中国人。咱老百姓一定要明白，大灾大难只有神佛才能保佑咱们，而只有退出这个不信神佛的党团队，才能得到神佛的保佑，从心里退了就行了。退休老师说，都入过了，那就退吧。随后她告诉我：孩子在派出所上班。我说，回去跟孩子讲善待大法弟子会得福报的。

过了一段时间，退休老师找到我说，说她有一个朋友，关系很不错，但不幸得了乳腺癌，吃药也不见效，情绪很低落，你能不能跟她说一说大法真相，哪天开车来接你。我说行。
过了一天，退休老师来接我，我马上带上真相小册子、真相光盘去见她的朋友。我跟她朋友说，诚心敬念“法轮大法好，真善忍好”，你可能会好的。她的朋友点点头，说想看看大法书。过了一个星期，我给她送去了《转法轮》这本书，还有师父的讲法磁带，教功光盘。我对她说，你觉的这本书好，就送给你。我又嘱咐她，这有师父教功光盘，你跟着炼，会好得快些。

不久，退休老师的这位朋友搬回了老家，而退休老师也搬离了我所在的这个小区，这样就没了她们的音信。又过了两年左右，一次在集市上偶然碰到了退休老师，她高兴的告诉我说，她的朋友全好了。

还有一次，是夏日的一天，我从市场回来，遇到了一位之前的同事。她愁眉苦脸地说，她丈夫精神不正常，每到春秋时节就犯病，老跑出去跟别人打架，最严重的一次把人家的耳朵给咬了下来了，赔了八万元钱，一年省吃俭用省下的钱都赔進去了。这眼看夏天要过去了，又该犯病了，我还得整天看着他，干不了活，阿姨，你说可咋办啊。我说，别犯愁，你就诚心敬念“法轮大法好，真善忍好”，会给你的丈夫带来福报的，这九字真言你就天天念，早上念，晚上念，有时间就念。她爽快的答应了。

到了第二年春天，我去问她的丈夫怎么样了，她高兴的告诉我说这九字真言真的是太灵了，真的没再犯病。我心里很是欣慰，看她也是个有缘人，就送给她一本《转法轮》书。

法轮大法在世间洪传，万古不遇，只要你抱着一颗虔诚的心对待他，就会给你带了善报、福寿，奇迹就会出现在你身上。
