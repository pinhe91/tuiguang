### 愿您登上那即将起航的金色法船

【明慧网二零二二年四月十九日】【投稿者注】从常人朋友那里，我看到了一封一位同修的家属写的劝善信。据了解，这位同修在单位工作出色，在领导和同事中，口碑很好。二零二一年，在邪党的所谓“清零行动”中，这位同修为抵制迫害，被迫流离失所。这封信就是这位同修的家属写给该同修单位领导的劝善信。
这封信行文贴近百姓的想法，据常人朋友说，信确实起到了一定的效果，领导的态度已经发生了明显的转变。现在原信基础上，稍加修改补充，可以作为基本真相资料的辅助，望能对同修讲真相有所帮助。

尊敬的某某某领导：

您好！我是某某的先生，也是一名信仰真、善、忍的法轮功学员。每次见面都很仓促，没有时间深聊，今天也是个机会，谈谈我的一些想法。

您也许会想，政府都不让炼了，你们也失去了那么多本该得到的，到底为了什么呢？值得吗？这个问题很现实，我们也都不止一次地认真思考过。其实，有些事情并不复杂，只要用人之常理判断一下，就能得出大致的结论。

中国有着悠久的历史文化，古老的东方智慧，各种经验教训、历史故事比比皆是，这一切造就了这个民族的理性和睿智。加上近代中国各种政治运动不断，人们曾有过追求和信仰，也有过失落。到了今天，历史久远的积淀中，很多优秀的传统文化和理念却被现代人视为守旧、愚昧和迷信，不知不觉中，唯利是图成了很多人的价值取向。在当今的中国，社会诚信缺失，尔虞我诈，假货横行。为了免受伤害，中国人在不断丰富自己的甄别能力中变得更加精明，想让人们轻易去相信什么真的是很难了。

上个世纪七十年代末期，中国大地出现了气功热，吸引了大批的健身爱好者，气功中出现的神奇现象和特异功能，也使刚刚从文革阴影中走出来的人们注意到“修炼”这个略带神秘的古老东方文化。到了改革开放的八十年代中期，社会上冒出一大批假气功师，为了赚钱，乱编一些东西骗人，搞得乌烟瘴气。到了九十年代初期，气功开始走下坡路，屡次上当受骗的人们开始怀疑气功的疗效，无论怎么宣传，气功师办班都很难了。

也正是在这样的背景下，倡导真、善、忍理念的法轮功出现在人们的视野，没有大张旗鼓的宣传，靠着神奇的祛病健身的疗效和修炼者实实在在的心灵升华，人传人，心传心，不到两年迅速传遍全国以至世界。

您也许注意到一个有趣的现象，一般的气功锻炼者群体都是以没有什么文化、体弱多病的老年人为主体，而修炼法轮功的这群男女老少中，既有青少年，也有中年和耄耋老者；既有文盲，也有大学生、艺术家和科学家；既有普通百姓，也有商人和达官显贵；既有军人，也有公检法公务员……他们遍布了社会各个阶层。当接触这群人后，您就不难发现，他们都有一颗修心向善的心，每个人都有修炼大法后身心受益、道德回升的感人故事，在各种环境中，实践着真、善、忍的信念。

他们为官清廉，淡泊名利；他们工作勤恳、不计得失，他们身体健康、豁达向上；他们敬老爱幼、子女品学兼优；先前社会上的混混和“瘾君子”学法后，浪子回头，从新做人；偷盗成风的村民学法后，变得民风淳朴；他们都是周围同事朋友、左邻右舍公认的好人。即便是在被当权者打压的二十多年中，在个人利益受到威胁时，还有那么多人仍然不离不弃。

人都是有思想的，更何况是当代的中国大陆人。您想，一个不好的东西，一个正常人不需要你多说，早就抛弃了。哪怕是个比较好的东西，涉及到自身利益时，都是要权衡取舍的。更何况这文革式的对法轮功的打压抹黑已经持续了二十多年，还有这么多有思想的善良百姓在坚守着真、善、忍的信念，修炼法轮功使人们身心受益的程度之大也可见一斑。就象当年人大委员长乔石经过大量调查后所说的那样：修炼法轮功对社会有百利而无一害。

我们修炼法轮功已有二十多年了，不光得到了健康的身体和道德的提升，也明白了人生的意义，大法的珍贵是无法用金钱利益来衡量的。这么好的东西，您想想，明白的人能不坚持吗？其实，大法对世上的每个人来说，都是极其珍贵的，只是人们在世俗滚滚的洪流中，追求着名利情，在当今这个古今中外预言中都提到的特殊历史时期，已无暇去思考这些身边发生的事情，变得麻木了。

那么，既然这么好的功法，国家为什么要取缔呢？其实，当时政治局常委中一意孤行，打着国家和党的名义要镇压法轮功的常委只有江泽民一人，其他人是反对的。而且这场迫害至今在中国现行法律框架内也找不到任何合法依据，只不过是凌驾于法律之上的人祸和一场注定的历史冤案。人以类聚，物以群分。看看当今那些在反腐中落马成为阶下囚的周永康、徐才厚、郭伯雄、薄熙来、王立军、孙力军、傅政华等人，哪个不是这场迫害发起者当年的得力干将和急先锋？自古正邪势不两立，对正信的恐慌和妒嫉、对权势的贪婪和小肚鸡肠，也许就是这场近乎荒唐的人间悲剧的起因。

在这场迫害中，一言堂的媒体编造出了很多污蔑之词和谎言，这也没有什么值得大惊吓怪的。小学语文课本中有一篇“狼与小羊”的故事： “狼来到小溪边，看见小羊在那儿喝水。狼很想吃小羊，就故意找碴儿，说：‘你把我喝的水弄脏了！你安的什么心？’小羊吃了一惊，温和地说：‘亲爱的狼先生，我怎么会把您喝的水弄脏呢？您在上游，我在下游，水是不会倒流的呀！’狼气冲冲地说：‘就算这样吧，你总是个坏家伙！我听说，去年你经常在背地里骂我，是不是？’可怜的小羊喊到：‘啊，这是不可能的，去年我还没出生呢！’狼不想再争辩了，大声喊道：‘你这个小坏蛋！骂我的不是你就是你爸爸，反正都一样！’说着，就往小羊身上扑去……”

清者自清，浊者自浊。粗制滥造的谎言真的不值一提，不值一辩。建议您有机会出国或翻墙看看外界真实的资讯就明白了。为什么中国要花那么多钱建金盾防火墙呢？因为谎言怕曝光，需要一个封闭的环境来维持。

事实胜于雄辩。三十年来，真、善、忍的信仰传遍了全世界，全世界各国各族裔的人们见证了大法弟子的和平理性、大善大忍，赢得了全世界正义力量的褒奖、支持和敬佩，在他们身上，人们看到了优秀的五千年华夏文明和传统文化再现，他们觉得这群有信仰的中国人才是真正的中国人，他们也想不通，为什么要去迫害这样一群善良的人，而且迫害延续了这么长时间！

有人说，法轮功反党、劝人退党搞政治。这也是很多中国人漠视这场迫害的主要原因之一。其实，“政治”一词本身并没有贬义，法律赋予了每个公民参政议政的权利。在正常社会，政治家还是个高阶层人士。只是在大陆一提到“参与政治”，人们就联想到几十年来的残酷政治斗争中的血雨腥风，成了“篡党夺权”的代名词，成了一根打人的棍子。“文革”造反派夺权时，你不参与政治，他会说你政治觉悟不高；现在一旦被扣上“参与政治”这顶帽子，就可以对你大打出手了。噤若寒蝉的中国人从此对“政治”避而远之。

自古以来，东方修炼文化都充满了神秘的色彩，得道者白日飞升，有术者洞察过去与未来，很多神奇的传说和历史故事代代相传，修炼人毕竟不是一般的人，有道行的修炼者知晓天机。自古以来人们都知道，“善待修炼人功德无量”、“宁搅千江水，不扰道人心”，迫害修行人是十八层地狱都装不下的大罪。

法轮功是佛家修炼，真、善、忍是宇宙的特性。您想想，这个历史上在各次运动中，造成八千万中国人非正常死亡的共产党，破坏传统文化，“文革”中毁寺砸庙，“六四”屠杀学生，贪污腐败，现在又迫害大法修炼人，甚至活摘器官牟取暴利，这铁证如山的罪恶，必遭上天的惩罚，将其党成员的未来也带入到一个非常悲惨的境地，因为他们曾经发毒誓要为其献身，只有抹去毒誓，才能躲过“天灭中共”的劫难，这就是大法弟子劝三退的真正原因。当您上网打开百度，搜索“藏字石”，就会看到那张彰显天意天然形成的六个大字“中国共产党亡”，天意不可为啊！
