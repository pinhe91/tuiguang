### 山东胶州市张秀兰被枉判八年、生命垂危

【明慧网二零二二年九月二十四日】（明慧网通讯员山东报道）二零二二年八月二十三日上午，青岛胶州市法轮功学员张秀兰在青岛市黄岛区被非法开庭，法院不准许她的任何家属旁听，并枉判张秀兰八年。张秀兰当庭表示不服，要上诉。
据知情人透露，目前张秀兰身体已极度虚弱，浑身瘦得皮包着骨头，生命时刻处于危险之中。即使这样，她在看守所里还被逼转化、被逼在诬判书上签字，她不答应转化、签字，狱警就打她骂她。

张秀兰家属和律师强烈要求有关部门重视老百姓的生命安全，让张秀兰保外就医。
张秀兰曾有一个幸福美满、令人艳羡的家庭。她丈夫刘福玺在市教体局、市纪委当秘书，后办了停薪留职手续，开始自己办公司做生意。一九九六年他们夫妻俩开始修炼法轮大法，刘福玺当秘书期间所患的严重神经性偏头痛以及胃病以及做生意后染上的烟瘾、酒瘾不翼而飞；张秀兰的妇科病、关节炎、让人焦心的阑尾炎等疾病都好了。不仅如此，刘福玺修炼法轮大法后，做生意不坑不骗，公司办的越来越红火，经济效益越来越好。被迫害之前，他的公司每年少说也有十几万的纯收入，他们一家人的日子过得富庶而惬意。

然而，一九九九年七月份江泽民发动了对法轮大法及其修炼者的残酷镇压后，张秀兰一家就遭到了残酷迫害，为躲避迫害，他们夫妇辗转外地，流离失所五年左右。期间，张秀兰母亲跟孩子在一租住的房子里生活，一天，门突然被打开了，几名便衣进入，张秀兰母亲愕然，问：“你们怎么进来的？”那些人竟说：“你没锁门。”老人说：“胡说，我刚送孩子上学回来，关上门的。”

二零一六年八月二十七日，张秀兰与大法学员一起张贴真相展板被绑架（后以“监视居住”的名义被放回家）。随后，二十八日凌晨一点左右，一帮警察闯进他们家，不顾刘福玺瘫痪在床不能自理，旁若无人的对他们家进行了查抄，他们肆无忌惮将家内翻了个遍，劫走大量私有物品。刘福玺又惊又吓，身体状况急转直下，过了不到二十天，他就永远的离开了人世，终年五十九岁。可怜他离世时张着嘴瞪着眼。张秀兰草草处理完刘福玺丧事后就离家出走，又过上了流离失所的日子。

一次次的被迫害，张秀兰无法象一般人那样安居乐业，一声警笛都要让她的心象被揪起来一样的痛，她昔日健壮的身躯日渐消瘦，挺直的腰板日趋佝偻，她的身体健康出了问题，最终，她不得不停止流离失所，在女儿家休养。可是还没等她身体有些许恢复，迫害再一次降临到她头上。

二零二一年十二月十六日，胶州中云派出所陈志杰等四个警察谎称是物业人员敲开张秀兰女儿的家门，一进门四个警察就不由分说把张秀兰绑架带走。可怜张秀兰的母亲在精神恐吓下于次日离世。

张秀兰在中云派出所被非法关押几天后，被送往青岛城阳，之后被非法关押在青岛即墨普东第三看守所非法关押、构陷至今。看守所罔顾人生命安全，逼迫她转化。

二零二二年八月二十三日上午，张秀兰被青岛市黄岛区法院非法开庭、枉判八年。她极度虚弱的身体更是雪上加霜，时刻处于危险之中。
