### 师父救了被摔脑震荡的孙女

文: 河北大法弟子 

【明慧网二零二二年一月三十一日】二零二零年八月份的一天上午十一点，我回到家，儿子在客厅辅导着孙子功课，我问儿子孙女呢？儿子说在屋里。我放下包，到屋里一看，七岁的孙女，满头大汗，头发都湿透了，在床上闭着眼躺着。
我把孙女抱起来，儿子过来没好气的说把她放下。我把孙女放下让躺在床上。这时我才知道，上午孙女去了同学家玩。同学把她背朝下摔了下来。孙女难受，同学才把她送回家。

孙女躺在床上闭着眼，我问孙女怎么了？孙女说道：“我想吐，就是吐不出来。”孙女反复的说着这句话，她说还想睡觉，我说你不能睡。

我在孙女耳边说：“快求师父救你，诚念‘法轮大法好，真善忍好’，师父救我。”懂事的孙女一遍一遍的念着“法轮大法好，真善忍好”，大约念了半个小时，我领着孙女给师父上了一炷香，孙女的精神越来越好。

中午了，我问孙女想吃什么？她说什么也不想吃。我给她做了一碗面汤，哄着让她吃完了，不一会，她就连续吐了几口水，可面条没吐出来，我哄着孙女到我屋子里的床上，继续念“法轮大法好”。

一直到下午三点，我问孙女，还想吐吗？还想睡吗？孙女说：不想了，就是有点头疼。我问孙女要出去玩吗？孙女说去。

我带着孙女，出去买了两包孙女平常爱喝的奶，去了她爷爷的店铺，玩了一个多小时。在这一个多小时里，她不停的念“法轮大法好，真善忍好”。

在回家的路上也是不停的念。快到家时，孙女说：“奶奶，我俩是世界上最幸福的人。”

晚饭我又给每人做了一碗面条汤。孙女玩到晚上十点多，跟着我安安稳稳的睡了一宿。

过后我和儿子儿媳说了整个过程。儿媳说：这是脑震荡。以前儿子儿媳不太理解大法，现在儿子儿媳都要了大法真相护身符，儿媳天天带在身上。

含着泪写完，我们无法用语言感谢师父的慈悲。
