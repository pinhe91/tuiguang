### 广西公安厅副厅长唐斌遭恶报被查

【明慧网二零二二年八月十九日】据二零二二年八月九日广西消息，中共广西公安厅中共党委副书记、副厅长唐斌被查。唐斌在广西政法系统任职期间积极跟随中共江泽民集团迫害法轮功学员。
公开资料显示，唐斌，一九六九年九月生，广西全州人，曾在广西政法系统任职三十一年，历任南宁市公安局副局长，钦州市公安局局长、副市长、市委政法委员会副书记（兼），南宁市副市长、公安局局长、市委政法委员会副书记（兼）等职。二零一八年四月起，唐斌任广西政法委副书记、维稳办主任，广西公安厅党委委员、副厅长、常务副厅长等职。

广西是少数民族人口众多的一个自治区。在中共发动的对法轮大法的迫害中，广西法轮功学员也遭到了中共政法部门残酷的迫害，据明慧网不完全统计，二零一八年至二零二零年期间，至少1人被迫害致死，16人被非法判刑，83人次被绑架，70人次被骚扰，26人次被劫持到洗脑班迫害，总计至少196人次被迫害。二零二一年间有10人被绑架，1人被非法判刑。

据调查，广西邪恶“610”在南宁市劳教所解体后，二零一三年将其迁至钦州，换了“广西钦州基地”的名称继续迫害法轮功学员，全自治区被绑架的法轮功学员都往那儿送。迫害手法：绑架、囚禁、诱惑、威胁（被判刑等）、暴力、人格侮辱（十分变态的无耻下流手段）、不给上厕所等等。整个洗脑迫害基地建筑构造形同一座监狱，层层铁门，有人看守，不许亲属探望。当时任“610”办公室主任的陆海琼和任公安局局长的唐斌，应负主要责任。
