#### “哪个国家都不反对法轮功，都说法轮功好”

〖中国大陆来稿〗这天，我碰到一位衣着时尚、气质优雅的崔姓女士。我给她讲真相时，她说：“我到过香港和其它一些海外国家，哪个国家都不反对法轮功，都能自由的炼功，都说法轮功好，都支持法轮功。可是在共产党统治下的中国就是异类，就不同，只能有一种声音。我说一句心里话，中国人真的好可怜，没有言论自由，没有信仰自由，没有人身自由，就象关在笼子里的动物一样，天天只能看新闻联播，接受中共的谎言洗脑宣传。”她还说：“我是不看新闻联播的。”

我听后，真为这些头脑清晰、能明辨是非的善良民众感到欣慰和由衷的喜悦！大陆百姓真的在越来越看清中共的邪恶本质！

#### 小伙说：“共产党是最坏的、最邪的！”

〖中国大陆来稿〗今年八月一日上午，我外出告诉民众法轮功真相时，碰到一位涂姓的小伙子。我给他讲法轮大法的美好时，他说：“法轮功是好的，修真、善、忍无可非议，共产党是最坏的、最邪的。”我问他上学时戴过红领巾没有？他说：“戴过，是被强迫的，不是我自愿加入的。”我说：“你也要把它退掉，抹掉兽的印记。”他连忙说：“退！退！我不要这个邪灵的东西！”

#### “这本书太好了！为什么不早点给我啊！”
〖中国大陆来稿〗一天，我送给单位同事一本《九评共产党》的书，同事看了这本书后，对我说：“这本书太好了！为什么不早点给我看啊，这本书太好了，要人人都给一本多好啊，一看不就都明白了吗！《九评共产党》一书是二零零四年十一月出版的，你怎么不早点给我看呢？”看到同事这样的反应，我真高兴，同事得救了，有了美好的未来。
