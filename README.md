### 乡亲大哥起死回生　同事明真相得福报

文: 河北大法弟子

【明慧网二零二二年一月十六日】我修炼法轮大法整整二十三个春秋，风风雨雨走到今天，体会很多。今天先不说自己，说说这些年来修炼中的真实故事，证实大法的美好与师父的慈悲与伟大！

#### 一、乡亲大哥起死回生讲真相

前些年，同村的一位乡亲大哥，打着氧气、输着液被从北京的医院用救护车送回了家。因与这位大哥那个家族的人不是很熟悉，我没到他家去看望他。有关他的情况基本是在村里的小卖店买菜时，听到大伙在议论而得知的。说他肿瘤扩散，没多少日子了，医院放弃不给他治了。

我听了心里很不好受，因我们是修善的，得有个慈悲心，要不就去看看他吧，跟他讲讲大法真相，转念又一想，他能不能接受啊？他两个儿子在村里都是有头有脸的人，比我也就小三、四岁。他本人也是个倔老头，能不能搭上话呀？再说我从来也没有去过他家，要是他们把我轰出来多没面子呀！又一想，要不先和他老伴讲讲吧，都是女的，比较好说话，求师父给安排吧！

过了十来天，我去大队交电费，正好碰到他老伴领着孙子在大队院里玩。我办完事后，就上前与她搭话：“嫂子，哄孩子玩儿呢？你家大哥怎么样了？”她说：“不怎么样，医院不给治了，说扩散了，一口东西吃不了，水也喝不了，就靠输点营养液，延长点时间罢了，早晚的事。”我说：“嫂子，我有个办法，你能不能接受？”她说：“有什么不能接受的，已经这样了，还能怎么的？你说吧。”

我问：“你听说过法轮功吗？”她说：“听说过，那电视上不是说不好吗？”我说：“嫂子，别管电视上怎么说，得听听身边炼法轮功的人自己怎么说，看看他们那些人怎么样。你说乡里乡亲的谁还不了解谁。你们这个年岁的人都是从文化大革命走过来的，多少人遭难不都看到过吗？（他们家文化大革命中就被迫害过，大哥曾被关在监狱好几年。）就拿我来说吧，以前身体不好，心眼小，动不动就生气，脸色都发暗。后来咱村有学炼法轮功的，开始我也不知道，整天不出门，身子不是这儿疼，就是那儿难受，还有子宫肌瘤，月经总不怎么干净。后来孩子的二婶来，让我去学学法轮大法，炼炼功吧，别整天憋在家里难受了！我一想，妯娌之间相处挺好的，她为我好，那就去吧。这一去就放不下了，原来法轮功这么好，那法轮功师父的书里讲的全是让人做好人的理，读了法轮功的书，炼了功，心里就顺了，身体也好了，干活一点也不累了。法轮功教人遇事要忍让，我再遇到不公平的事就不觉委屈了。”

“后来政府不让炼了，家里那口子管着不让我炼，书也不让我看，我身上的病就又犯了，后来就只能躺在炕上起不来了！身体难受不说，腿还抽筋，每次抽时间长了，丈夫就掐我的人中，才能过去。我心里想：年纪轻轻的就这样，啥时混到老啊！我妈就是老生病，我要是随我妈怎么办？这个法还学不学，功还炼不炼？丈夫一看我那么受罪，就说：‘你愿意炼功你就炼吧，我不管你了。’于是我就又开始炼法轮功了。一直炼到现在。”

我接着告诉她：这法轮功也叫法轮佛法。以前有好多得了绝症的人学炼法轮功都好了。现在咱们不能炼，先在心里念“法轮大法好，真善忍好”，这个坎也能过去。

我接着说：“大姐，你们老俩口都念，要是念好了，多好的事呀！既不花一分钱，也不痛苦，只要真心念，心诚则灵。”

她说：“行，回去就跟他说说。”我问：“你与大哥入过共青团和少先队吗？”她说：“什么都没入过。”

又过了十多天，有位同修给了我几本《绝处逢生》的书。路上我想给那个大哥家送一本去，让他看看有好处。真巧，半路正巧碰到他老伴抱着孙子买菜回来。我说：“嫂子，给你家大哥一本书，没事的时候看看，对他有帮助。”她说：“行，你老想着我家。”

后来再在路上遇到大哥的那两个儿子，他俩每次都会摇下车窗笑呵呵的跟我打招呼。

时间不长，在小卖店我就听到有人议论，说某某（这个大哥的名字），能出来遛弯了，面容和精神头都挺好。我想这是他明真相得福报了！

一次在大街上我看到他跟几个老头在唠嗑，说：“什么‘和谐社会’？是‘喝血社会’！你说这法轮功不挺好吗，硬说人家法轮功李洪志师父不好，还要抓人家！”

我一听这老头真的明大法真相得救度了！

#### 二、同事说：“大姐，真顺！”

二零一一年，我在箱包厂打工。有一位年岁比我小的同事，她有一个女儿叫小宇，才一岁半左右。小宇没有奶奶，奶奶在她爸爸十五岁的时候就去世了。自从有了小宇之后，每次小宇爸爸去给奶奶上坟回来，小宇晚上就不睡觉，就是哭。小宇会走路了之后老是挨磕，她妈也不知道怎么回事。

一次小宇妈妈就和我说起小宇的事，问我：“大姐，你们这个大法有办法吗？”我说有，给你一个护身符，你给孩子带在身上，你和孩子在一起时，就对着孩子念：“法轮大法好，真善忍好”，她就没事了。

隔了一天我去上班，听到小宇妈妈正在与别人说：“这一次她从一米高的阳台上头朝下摔了下来，既没有哭也没有伤。这个护身符真灵！”

二零一三年夏天，小宇妈妈又怀孕了。因为中国大陆计划生育规定，头胎是女孩的话，到孩子四岁才能生二胎，提前怀孕就得做流产。要不做流产，孩子生出来也不给上户口。

这事把小宇妈妈愁的呀，她跟我说：“大姐，怎么办？你能不能给我托个人帮帮我？我有个妹子在计生委上班，她帮我也没用。”我说：“孩子来了是好事，来了就有他的福份。什么也别想，该吃就吃，该喝就喝，你就在心里念：‘法轮大法好，真善忍好’。”

过年放假，她正月初六生了个男孩，也就是小宇有了个弟弟。同一个病房，一个素不相识的新生儿的父亲说他能帮忙上户口。后来小宇妈妈笑着对我说说：“大姐，真顺！”意思是说念“法轮大法好”真顺！

#### 三、“年画宝宝”出生了

还有个同事娇娇，结婚五年了，没有孩子。中医西医都看了，偏方也用了，钱也没少花，药也没少吃，都不管用。

二零一三年十二月中，老板让我和娇娇一起加班，我正好利用这个机会给她讲真相。她边听边笑。后来说到“三退”（退出中共的党、团、队组织），她说她入过队，我说：“退了吧？”她说：“行，退了。”

因聊天时间不多，也怕她面子上过不去，就没说到她没孩子的事。

有一次去她家串门，就我和她两个人在。我说：“告诉你一个好办法，你就真心念‘法轮大法好，真善忍好’，求大法师父，就会心想事成。”后来我给了她一张《明真相 得福报》的年画，告诉她：“贴在卧室，天天看，天天念‘法轮大法好’，一定能如愿。”

她一看画上那三个小孩儿，还是一个劲儿的笑。

正月十六日，大家正在上班，女老板来跟我们大伙说：“小宇妈妈生了个儿子，娇娇怀孕了！”大伙一听都说：“过年了，好事都来了！”

后来娇娇生了个儿子。那孩子单挑父母的优点长，还特别聪明。
