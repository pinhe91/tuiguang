### 河南省原劳教局副局长李军信遭恶报被查

【明慧网二零二一年六月二十五日】（明慧网通讯员河南报道）据河南二零一六年六月二十日消息：河南省漯河市副市长、市公安局局长、原河南省司法厅劳教局副局长李军信涉嫌严重违纪违法，投案自首，被审查。
一九九九年七月二十日，江泽民流氓集团开始迫害法轮功。次年，李军信担任河南省司法厅劳教局副局长。他所管辖的劳教所正式成为迫害法轮功的集中营。河南省迫害法轮功学员的劳教所共有五个：郑州、许昌、开封、洛阳、平顶山。

郑州十八里河女子劳教所，是全省集中迫害女性法轮功学员的邪恶场所；开封、洛阳、平顶山劳教所是迫害本地法轮功学员的邪恶场所；而许昌劳教所（即河南省第三劳教）被“六一零”（专门迫害法轮功的非法机构）称为“转化”基地，是集中迫害全省男性法轮功学员的邪恶场所。当时的劳教局对被非法关押人员可以不经法律授权，任意非法剥夺公民人身自由甚至生命。

李军信，男，一九六四年十一月生，河南信阳人。一九八五年开始工作，一九九一年入邪党，河南省司法学校司法专业毕业，省党校研究生。

一九八五年七月，历任河南省司法厅基层工作处干部、助理检察员、省司法厅律师管理处副处长；二零零零年八月，任河南省司法厅劳教局副局长；二零零六年九月，任河南省司法厅办公室调研员；二零零七年八月，任河南省许昌市司法局局长；二零零九年三月，任河南省许昌市公安局常务副局长、党委副书记；二零一四年三月，任河南省濮阳市政府党组成员、市公安局局长；二零一七年五月至今，任河南省漯河市副市长、市公安局局长。

李军信任河南省劳教局头目时的恶行：

#### 一、河南省劳教局的头目们亲自坐阵十八里河劳教所实施迫害

二零零三年，中共邪党政法委总头目罗干，借来河南视察“非典”之机，流窜到河南郑州十八里河劳教所，下密令加重迫害并要求该劳教所所谓的“转化”率必须达百分之九十八～百分之九十九，并有两个死亡指标。

河南省劳教局局长们亲自坐阵指挥，采用酷刑对坚定的法轮功学员实施迫害，并集中了省第三劳教（许昌劳教所）等几个单位最邪恶的人员到这里出谋划策并参与迫害。许昌劳教所为“转化”法轮功学员，采用了十三种酷刑。

### 在台湾传统市场送福音和避瘟疫良方

【明慧网二零二一年六月二十四日】（明慧记者郑语焉台湾双北地区报导）二零二一年五月母亲节过后，台湾本土疫情突然爆发，染疫人数从个位蹿升到百位，疫情指挥中心宣布进入三级警戒，并且相应采取各项策略，力阻疫情扩散。

策略之一的实联制：包括民众外出须戴口罩，到卖场购买民生用品或食材蔬果，需量体温，喷酒精干洗手，用手机扫描QR CODE，几秒钟内就完成登录卖场名称位置以及民众个资，或者是手写前述资料，方便检疫或疫调追踪之需。

这在超市或大卖场，可由其员工轮值负责执行人流控管，但在传统市场缺乏这样的人力。新北市一个传统市场的管委会理事长，对林女士说出他的烦恼：“现在疫情这么严峻，大家都很害怕，给工资也没有人要来做量额温，登录资料的工作。我需要你帮忙来做实联制控管人流的工作，你可以吗？”

#### 法轮功学员：我们都会尽力来帮忙！

林女士在市场设摊做生意已有三十几年，自一九九九年修炼法轮大法以来，经常向顾客讲述法轮功真相，市场人员都知道她是法轮功学员。林女士听理事长这么说，立即回答：“我可以！这是对社会和民众都好的公益，只要你需要怎么做，我不用工资也会全力配合。我和我法轮功的同修都会尽力来帮忙。”

其他法轮功学员也表示：“疫情中，人心惶恐不安，再则响应政府呼吁，大家都待在家里尽量不出门，但是为了三餐必须上市场采买，人流比其它地方多，在市场讲真相，送上‘法轮大法好 真善忍好’信息与真相资料，也可安定人心，帮助稳定社会情绪。”

#### 明真相　九字真言是避疫良方

林女士找到住在新店的碧云及几位法轮功学员，从六月四日起在传统市场执行防疫实联制的工作，此外，学员们还购买材料，制作带有“法轮大法好”、“真善忍好”信息的小莲花（喜获这种莲花的民众称为“防疫莲花”），并备妥法轮功真相资料，向民众传递大法的美好祝福与防疫良方。

法轮功学员碧云说：“此时民众面对疫情的生死关头，大家都想找一个可靠的救生圈。”学员们在量额温、执行实联制之际，告诉民众世界需要真善忍的重要，以及现今社会道德沦丧，人心变坏，易招天灾人祸，一旦人心归正向善，瘟疫（瘟神）自然会离去等等真相，并且送上“防疫莲花”与真相资料，教对方诚念“法轮大法好 真善忍好”九字真言，保平安，避瘟疫。
