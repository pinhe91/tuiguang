### 弟媳传真相得福报　家中好事连连

文: 中国山东大法弟子　重生 

【明慧网二零二二年八月二十三日】弟媳今年五十四岁了，为人淳朴厚道，明白法轮大法真相，敬重大法。我每次回老家，她都会要一些真相资料、大法真相护身符，真相台历，每次都说：“二嫂，你回家就多带一些回来，我帮你发。这可是好东西，谁看了谁说好，谁看了都有好处。俺一家都喜欢看小册子，都戴着护身符，所以干啥都顺。”
我每年都回家一两次，每次回家都带很多真相资料，当天发不完的，就放在弟媳家里，弟媳就发给她的一些亲戚朋友。有一次。我带一些真相大粘贴没贴完。弟媳说：“你放家里吧，我会贴出去的。”我嘱咐她要贴在平滑的电线杆上，将咱村东、西、南、北村头路边的电线杆上都贴上。让咱村的人都能看到“真善忍好，法轮大法好”。

等下一次我回家，刚走到村头就看到了弟媳贴的真相粘贴，大大的、醒目的“法轮大法好”、“真善忍好”！后来回家我就又带一些回去，告诉弟媳等时间长了，旧的自然破损了，就换上新的。
年复一年，我们村多处一直贴着那些“法轮大法好”、“真善忍好”粘贴。

我家是一个大家族，弟媳家有喜事，我一定会到场。每次都带很多真相小册子、护身符、光盘等救人的资料。我们家族上上下下一百多人全部都了解真相，家家都有真相小册子和护身符。该“三退”（退出中共的党、团、队组织）的也都退了。要说能让这一大家族中的人都明真相得到大法保护，弟媳的贡献很多。弟媳跟这大家族的人关系都很和谐、很溶洽，人人都夸弟媳是一个善良的人。这二十多年来，弟媳家的日子过得也很顺，亲戚街坊都羡慕弟媳一家人。

回头想想，在我学大法前，弟弟、弟媳家里的日子过得并不是很顺，也很累。她的身体上的小毛病不断。自从我学了大法，弟媳明白了真相，农闲时经常看我送给她的《转法轮》，特别是她开始发真相资料、传播真相后，她家的日子开始出现了奇迹：每年的收成都是最好的，一样的水果，她家每年都能卖一个好价钱。二十年来，家里盖上了新房；儿子和女儿都有了自己的小工厂，每年的收入都挺好；两个孩子还都在县城里买了楼房，儿子还娶了个远近闻名的最孝顺的媳妇。村里人都说我弟媳家太幸运了，娶的媳妇都是村里最好的、最孝顺的女孩子，弟媳是我们村里“最有福份的人”。

弟媳还给我讲了一件事：二零一七年正月的一天，弟媳骑电动车匆匆忙忙过马路，被一辆汽车撞上了，她当时已不省人事，被好心人送到了市区医院。当她醒来后，医生告诉她，她的脸颊骨断了，需要手术。她听后马上决定：不做手术治疗，马上回家。医生告诉家人：不做治疗出院，出了问题医院不负任何责任。弟媳说：“不用你们负任何责任，我马上出院。”回家后她坚持默念“真善忍好，法轮大法好”，没吃任何药，不长时间脸颊骨恢复原样，没留下任何隐患。她还对我说，是因为她身上戴着大法真相护身符，有大法师父保护，所以没有也不会出现大的危险。她还告诉我，她知道一个和她同样遇车祸的人死亡的事。她说：前年大伯哥遇车祸基本和自己一样，就是他不太相信大法，家里的护身符放在抽屉里没戴在身上，当然得不到大法师父的保护，被车撞了几个小时后就死了。“法轮大法好”护身符就会保护相信大法好的人。她说，我们全家会一直戴着的。相信大法和大法师父会保护我们一家人一生平安！

年前我回老家，弟媳让我帮忙，把她家得福报的事写出来，投到明慧网上，表表自己的心意——感谢大法！谢谢大法师父！
