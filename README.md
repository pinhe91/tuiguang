### 染疫的老父亲诚念九字真言奇迹生还

文: 大陆大法弟子夏花 

【明慧网二零二三年一月九日】十一天前，弟弟来电话说父亲不幸染疫。父亲已九十六岁，住在条件很差的农村，但生活自理，自己种菜做饭。父亲突然染疫后浑身疼痛难忍，肚子痛，大小便失禁，要求住院。中国现在的医院均人满为患，叫120救护车时，人家直接说了，没有空来接，医院也没有地方住。弟弟只好自己去找车，可父亲已经从床上拉不起来了，弟弟很绝望。
我在电话里告诉弟弟：给父亲念救命符（半月前我送给弟媳妇的）上的小故事，让父亲诚心默念九字真言“法轮大法好，真善忍好”。第二天父亲就能自己起床和吃饭了。于是弟弟叫了一辆车，送父亲去了县级、地州级所有医院，结果连急诊科都没有床位，转了大半天，好不容易在一家很小的私人医院住下了。父亲住在一个四人间的病房里。

他们可能以为住院了就等于上了保险，便不念九字真言了，每天打针吃药，前三天父亲没有发烧，但越来越吃不下饭，直到十二月三十日上午，弟弟又绝望的告诉我，父亲一天没吃任何东西了，昨晚又发烧，半昏迷状态，还折腾人，弟弟几个晚上都没有睡好觉。

弟弟还说这个小小私人医院每天都在死人，有时候一天就死十个人，父亲住的病房里前后就死了三个人，就这样还有病人不断的住進来，他问我：父亲是不是快死了？我问他：是不是没有念九字真言？他说是。

我说：现在已经是走投无路了，还是诚念九字真言吧，只有大法师父能救他。弟弟又重新念救命符上的小故事，并让父亲默念九字真言，神奇的事情再次发生了。

十二月三十一号早上，弟弟高兴地说：父亲没再发烧了，还自己起床坐在凳子上吃饭了。我说：继续诚念九字真言。下午，弟弟说：父亲居然起来锻炼身体了，今天精神更好了，昨天下午、晚上都在睡觉，吃饭也多了，坚持今天下午出院回家。

写到这里，我不知道怎么感恩师父，穷尽所有的感恩词语都无法表达其一二！作为能修炼大法的弟子，本身就是无比的幸运和幸福，弟子也尽一切能力让身边的人了解真相和让有缘人得法。可很多人就是那么迷，写出此文的目地，就是让这些还在迷中的人再次见证奇迹而惊醒吧！
