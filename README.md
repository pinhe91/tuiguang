### 谁相信“法轮大法好”谁受益

【明慧网二零二二年九月十二日】〖中国大陆来稿〗我认识一位八十三岁、明白真相的老太太。一天，我听说她有病住院后出院了，就去看她。
老太太说了她的故事：那天她突然很难受，呼吸困难，胸部疼的很厉害。儿子送她去医院，经检查说是心梗，得转院。去了一个大医院住了三天，不见好，又转到省城医院。经检查说是有三根血管堵塞，得做支架手术。先观察两天，星期一做支架手术。她在痛苦中熬到星期一，下午一点進的手术室。

老太太的孙女接着说：四十分钟后，手术医生出来找家属，当时孙女的心里就没了底了，可是医生却说，没给她奶奶做手术，因为三个堵塞的血管已经通了两个。医生问家属还需要做手术吗？孙女就打电话问主治医师，主治医生说先让她奶奶回病房观察。这时做手术的医生接过她孙女的电话问主治医师：你给这老太太用的什么特效药？结果真的改变了。
在场的医生和患者家属都感到神奇。有的说，这老太太一定是个大好人，做过大好事。有的医生看着老太太竖起了大拇指，表示祝福。

老太太对我说：進手术室后，又用仪器给她做全身检查，到处涂药水，觉的凉飕飕的，她心里想：可能回不去家了。当时也没想有人在身边，就喊了一声：“法轮大法好！”喊完后她还想：认命吧。

可是她没想到，就这发自真心的一念，改变了她的命运，换回了身体的健康。这位老太太在省城医院共住了七天就康复回家了。

#### 儿子、孙女化险为夷

〖中国大陆来稿〗我是大法弟子，今年六十六岁了，在大法修炼中，受益颇多，我们全家五口无不感谢大法师父的慈悲与救度之恩。仅举两例：

在今年四月末，我儿子突然鼻子流血不止，把鼻子堵上，血就从口里出，看了很吓人。两次去医院治疗，医生都说得需要烧烤，弄不好还有害处，而且必须得在鼻子出血时才能治。每次都在医院等了一大会儿，鼻子也没流血，就回家了。此时儿子鼻子流血已经四天了。

回到家，我对儿子说：妈什么也帮不了你，只有大法师父能帮你，你只要诚心念“法轮大法好，真善忍好”就管用。儿子说知道了。我还把播放器拿来，打开开关，让他听师父讲法。到吃饭时，我觉的听师父讲法对师父不敬，就把播放器放起来了。

六周岁的小孙女，吃鱼时不慎鱼刺卡在喉咙里，卡的嗷嗷直叫，眼泪直往下流。这时小孙女喊着找师父：师父呢？师父呢？我立即把播放器拿出来，小孙女双手抱着说：“师父救我！师父救我！我让鱼刺给卡了。”她爷爷说：快吃口大米饭。小孙女吃了米饭，鱼刺就顺下去了。这时小孙女睁大眼睛对我们说：“这可不是你们救了我，是师父救了我！”

确实是大法的威力，师父的威德，不然鱼刺怎能顺利的咽下去呢？我小孙女从会说话时，就念大法好，背师父的《洪吟》等诗词。就是因为小孙女相信师父能救她，她才哭着喊着找师父。

儿子也一直念“法轮大法好，真善忍好”，到第七天时，鼻子不流血了。就这样，师父把儿子、孙女的魔难化解了。

我发自内心的想对世人说：谁相信“法轮大法好、真善忍好”谁受益。特别是在多灾多难，疫情当下的今天，愿更多的人能够得到大法师父的护佑。
