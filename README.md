### 台湾灯会　法轮大法船花灯送祝福

【明慧网二零二二年三月二日】（明慧记者孙柏、苏容高雄采访报导）二零二二年台湾元宵节灯会在高雄隆重举办，台湾法轮大法学会共襄盛举，制作可以承载百人、六层楼高的巨型“法船”花灯，花灯在高雄冈山公园内从二月一日至二月二十八日展出一个月。
虽然受到疫情防疫限制，但仍不减民众高度兴致，阖家光临灯区。尤其在最后三天，适逢节假日，台湾疫情趋缓，数万民众纷至沓来，有不少政要、商界名人慕名前来，登船祈福，体验美好善愿的传统习俗。！

![image](https://user-images.githubusercontent.com/79625284/156327590-95cbd47d-99e5-4612-ac47-ef88e9936dfb.png)

图1：二月一日至二月二十八日，巨型“法船”花灯于高雄冈山公园展出一个月，天国乐团和腰鼓队前来表演，数万民众纷至沓来，登船祈福，体验美好善愿的传统习俗。

展出期间，每晚六时至九时各个整点，会开启逐层亮灯的灯效，象征光明希望。在二月二十八日连续假期期间，法轮大法天国乐团和腰鼓队前来演奏，更增添展场热闹丰富的艺术气氛。期间有不少政要、地方名人乡绅，高雄市长代表、冈山区区长，立法委员，前国大代表、冈山螺丝工业公司总裁、国际狮子会狮友们等前来，大家开开心心登船体验传统灯艺的文化内涵，灯船纳福，天赐平安。

这是“法船”第十三次参加台湾灯会，设计者法轮功学员庄逸中先生提到：“这是花费半年时间制作完成的法船花灯，最下层是以十二片大莲花瓣环绕衬托，船帆上九个法轮，法轮旋转时会正转九圈、反转九圈；船头有象征吉祥的大龙旗领航；船桅上三面‘真、善、忍’令旗发出的光芒万千，在夜色中金碧辉煌，璀璨亮丽。灯区现场还布置了‘神驹天车’、‘法轮功五套功法’花灯，全天候的免费义务教功，让民众进一步了解法轮功。”

图5：二月十五日元宵节晚上，台湾法轮大法学会理事长萧松山（左三）为法船正式点灯。

二月十五日元宵节晚上，台湾法轮大法学会理事长萧松山为法船正式点灯。他说：“这座由法轮功学员参与设计、建造、组装而成，唯盼法船花灯能在新年期间光照四方，彰显传统文化敬天重德的良善价值。”

萧松山还表示，“法船能带来舒缓疗愈的正能量，给地方民众带来福分。在疫情期间展出，希望大家都能平平安安的度过这个劫难。法船做得很精致，法轮能正转反转，意涵很深，期盼更多有缘民众一起加入学炼法轮功。”

#### 警察局少年志工队长：来到法船灯区像回到家一样

许多民众亲临现场，感受舒缓疗愈的能量。

屏东县警察局少年志工队分队长杨蕙珍、屏东县警察局志工队小队长林静慧，从屏东前来参观法船花灯。杨蕙珍表示：“来到法船灯区像回到家的感觉，好温暖、好温馨。本来在这疫情期间，我都不想出门了，我是带着一颗忐忑的心来参加的，可是在登上法船那一霎那，我内心好平稳，放眼看过去那么的光亮，看到法轮正转九圈、反转九圈，我整个心都放下了，似乎疫情已经离我很远了，非常感恩。”
