### 辽宁辽阳市七名法轮功学员遭绑架构陷、非法庭审

更新: 2022年07月03日

【明慧网二零二二年七月三日】（明慧网通讯员辽宁报道）辽宁省法轮功学员赵俊成、包凤芝、里培新、王存波等七人，二零二一年十二月八日被辽阳市警察绑架、非法关押在辽阳市看守所构陷，六月二十一日、三十日二次在灯塔法院遭非法开庭。里培新、王存波的两位维权律师从现行法律和信仰层面做了有理有据的无罪辩护。据悉公诉人建议法官判王存波三年、里培新判一年、崔荣一年半、赵俊兰三年半。
二零二一年十二月八日，辽阳市公安局国保大队、辽阳市白塔区公安分局国保支队、辽阳市白塔区文圣派出所警察跟踪赵俊成等法轮功学员。随后十二月八日，辽阳市白塔区文圣派出所警察非法闯入赵俊成和包凤芝（夫妻）家中，绑架他们与在他们家中做客的李英、崔荣、鞍山市法轮功学员里培新、王存波，和赵俊成的妹妹赵俊兰。

鞍山市法轮功学员里培新（男）由于身上带着身份证，被警察发现后，八日晚上八点左右有二十多个警察去他家非法抄家，抢劫走四十四本大法书、一台电脑、一个师父法像等私人财产。

![image](https://user-images.githubusercontent.com/79625284/177033370-980892f9-a3dc-4bbb-a2da-de2256e1e197.png)

酷刑演示：野蛮殴打

里培新于二零二一年十二月九日被辽阳市白塔区公安局非法刑事拘留，于二零二二年一月十四日被非法逮捕。

王存波（男）在被暴力殴打、刑讯逼供时，王存波说了一句话：你们打我，我也不恨你们。在他被绑架的半个月后，文圣派出所六七个警察去他家抄家，什么也没有抄到。

七名法轮功学员被绑架、构陷，二零二二年六月二十一日在灯塔法院被非法开庭，当时法官黄跃发，态度特别专横，不允许维权律师说话，恐吓维权律师要考虑后果。里培新说被抓时，被警察打了。法官黄跃发问里培新有没有证据。里培新说没有证据。因为当时被抓时，警察没出示任何证据、手续，进屋就是打人，由于里培新不配合采集指纹，被国保大队十几个警察暴力殴打，当时被打得肋骨疼痛、大腿粗肿。有个很瘦、很高、小眼睛的警察，里培新看了他一眼，就被他扇了两个耳光。开庭到12点左右，法官宣布休庭。

六月三十日继续开庭，法官态度变的温和了。里培新、王存波的两位维权律师从现行法律和信仰层面做了有理有据的无罪辩护。公诉人建议法官判王存波三年、里培新判一年、崔荣一年半、赵君兰三年半。王存波的律师当庭表示：免费替家属上诉。

这次开庭多一道手续，灯塔市公安局警察给每个参加开庭的家属拿手机拍身份证。

里培新生于一九六三年十月七日。一九九六年夏天他开始修炼法轮功，按真、善、忍做人，说真话、有善心、与他人有矛盾要找自己的过错，人变好了，身体也健康了。

王存波生于一九六七年五月二十八日，于一九九八年五月有幸得了大法，使身心得到了巨大的改变，他原来有肩周炎，右臂抬不过头顶，修炼大法后一个多月，右臂就能抬过头顶了；他还有久治不愈的痔疮、关节炎全部不治而愈了，做事先为别人着想。王存波就因为修炼法轮功在二零零四年十一月七日被鞍山市国保大队绑架构陷、非法判刑十年。这次是因为他去帮朋友修家电，被绑架、构陷。
