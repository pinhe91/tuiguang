### 遭恶报　河南焦作市郭国明突然丧命

【明慧网二零二二年八月二十二日】（明慧网通讯员河南报道）据河南消息，二零二二年八月十四日8时59分，原河南省焦作市人大常委会主任郭国明突然丧命，年74岁。
一九九九年一月至二零零四年二月，郭国明任河南省焦作市中共市委常委、政法委书记，610办公室副主任期间，不遗余力地追随江泽民、罗干迫害法轮功学员，焦作市至少有5名法轮功学员被迫害致死，10多人被非法判刑和劳教。

据明慧网二零零三年八月十三日曝光，河南省焦作市各个公共场所的人们都在议论：听公安的说：打死炼法轮功的，上报说是暴病死亡。郭国明的名字在明慧网“河南省焦作市参与迫害的恶人名单”中曝光。
郭国明任职焦作市政法委书记期间部份恶行：

#### 一、追随罗干疯狂迫害法轮功、强迫百姓观看大型污蔑法轮功的豫剧

中共政法委书记罗干二零零三年五月中旬流窜到河南省焦作市，罗干亲自部署下令，要求在7.22以前将坚修大法的法轮功学员抓起来。许多坚定的法轮功学员被强行绑架、抄家、毒打、洗脑。迫害法轮功的得力干将郑锋由政保科升为610组织的头目。焦作市开始办洗脑班，非法抓捕法轮功学员十多人。焦作市公安局、各派出所疯狂抄家、抓人，对法轮功学员进行骚扰，有的每天骚扰1次，有的2、3天骚扰1次，坚定的被送洗脑班。焦作市洗脑班恶徒围住法轮功学员刘富珍家两天两夜，跳窗入室将她绑架。

当时法轮功学员王桂花被折磨得走路要人扶着。刘富珍给帮教讲真相，被关禁闭，罚打扫厕所。有位法轮功学员第六天被折磨成精神病。两个月来它们利用各种手段造谣、诽谤、迫害法轮功学员。每天都要上操、跑步、体罚法轮功学员。610头目崔晓白每天都要出题骂大法，诽谤师尊。刘富珍被洗脑班罚款3000元，其他法轮功学员遭非法罚款2000元。时任焦作市政法委书记、610办公室副主任的郭国明是罪魁祸首。

二零零三年十一月五日，焦作市610歹徒下达焦防范办[2003]8号文件，标题为“关于组织观看大型现代反××豫剧《生日》的通知”，要求各市直单位按照文件所分配的名额购票并组织人员观看诬陷法轮功的豫剧《生日》，该剧十一月十六日下午2：30在焦作艺新剧院演出。据悉：该剧由河南省610歹徒、省文明办、省文化厅、省科协不法之徒决定在全省范围内巡演。

二零零三年十二月初以来，焦作市修武县为实现所谓的先进县三连冠的目标在全县范围内再掀对法轮功学员摸底排查迫害高潮，并唆使全县学生进行宣誓，组织县直有关单位大搞诬陷法轮功活动，毒害全县人民。

#### 二、五名法轮功学员被迫害致死

郭国明任职焦作市政法委书记期间，焦作市已有5位法轮功学员被迫害致死。白爱香二零零二年四月三十日被迫害致死，终年45岁。焦作市博爱县法轮功学员刘改，女，于二零零二年被迫害致死。孟州市缑村镇东葛村法轮功学员姚桂娇，于二零零二年六月被非法关押在孟州市看守所，受尽折磨，于二零零三年三月被孟州市看守所迫害致死。张春琴，女，50岁，二零零二年四月被迫害致死。柯玲被郑州十八里河女子劳教所迫害致死。

#### ◎河南博爱县法轮功学员刘改被迫害致死

刘改，女，55岁，焦作市博爱县柏山镇人，一九九八年得法，二零零零年进京上访后，多次被抓。二零零二年中秋节前后，刘改去娘家帮忙秋收。九月二十六日（农历八月二十）晚，柏山镇派出所刘大东等人追到张茹集（刘改娘家），骗刘改和儿媳上车后，把她们直接劫持到张茹集派出所。

![image](https://user-images.githubusercontent.com/79625284/185886212-fc2df304-c20c-489a-8a7e-2d919b6f0142.png)

刘改

到派出所后，刘改儿媳上厕所几分钟时间，出来一看，婆婆脸发青，不省人事。刘大东说：你婆婆刚才突然血压升高，昏迷了，要送她去医院抢救，你回家吧。但刘改儿媳回去不久，就传来消息说抢救无效，婆婆的尸体已在火葬场。

等家人匆匆赶到，不法人员们却不让见尸体。在家人的苦苦哀求下，勉强同意让他两个儿子进去。见到眼前的情况，儿子惊呆了，走时健康的母亲，却成了惨不忍睹、浑身是伤的尸体，衣服撕得破烂不堪，胳膊上全是刀伤。

儿子悲愤地质问为何这样，刘大东等人却支支吾吾，一会儿说是医院抢救时弄的，一会儿又说是火葬场没保管好。

事后柏山镇书记向刘改家人赔偿了一万五千元，便草草了事。

#### ◎法轮功学员白爱香被迫害致死

河南焦作市42岁的法轮功女学员白爱香，于二零零二年四月二十二日在市万方立交桥散发法轮功真相时不慎被恶警发现，被强制带到东方红派出所。白爱香为逃离恶警监控和迫害，从三楼跳下，摔成重伤。恶警对她继续折磨，直到她昏过去。等白爱香醒过来时，已在人民医院抢救室里，胸部剧痛，憋闷，后拍片检查发现肋骨被摔断7根，肺部被肋骨扎烂。恶警指令说不能做手术，只能让她恢复，自己慢慢长好。四月三十日，白爱香停止了呼吸。

#### 三、多名法轮功学员被劳教迫害、耿菊英遭警察强行堕胎

法轮功学员冯春荣（音），女，年近六十岁，焦作市人，因信仰“真善忍”，因修炼法轮功而两次被非法劳教。在一次非法抄家时，焦作市解放区公安局从她家拿走现金七千元，连白条也不打，冯春荣多次去要，得到的答复是，只要放弃法轮功钱才能退。冯春荣到焦作市政法委上访，并讲自己为什么炼法轮功，焦作市政法委书记郭国明当场给警察打电话，把她带走，非法劳教三年。二零零四年三月冯春荣被送往郑州十八里河，现被迫害得双腿无法行走也不放人。

焦作市博爱县柏山镇下期城村法轮功学员赵玲琴，曾被非法劳教两年，劫持到河南省十八里河女子劳教所。在劳教所内有个张管教，强迫赵玲琴吃精神病人吃的药。吃到第二天，赵玲琴便不会系纽扣、不会系裤子，生活不会自理。二零零一年十二月赵玲琴非法劳教期满回家。临出去时，张管教还威胁赵玲琴：不能说出去那药是她叫吃的。

法轮功学员张哲，焦作市工商银行职工，流离失所，二零零二年底被非法抓捕，被非法关押在郑州十八里河劳教所。

焦作市法轮功学员王俊英、颜爱英被非法劳教一年半后，再次被关押在郑州十八里河劳教所。法轮功学员张哲原流离失所，二零零二年底被非法抓捕，关押在十八里河劳教所。

法轮功学员耿菊英二零零三年二月初就被孟州县610再次绑架到河南省第一女子劳教所，关押在三大队，并被强迫隔离数天，后来又被强迫劳动，恶警指派两个犹大包夹。四月二十二日，三大队开始对法轮功学员迫害，穿“约束衣”。

明慧网二零零五年八月三日报道，二零零四年七月十五日，联合国言论自由特派专员与身心健康权利问题特派专员和妇女暴力和后果问题特派专员一起，联名发出一项紧急呼吁：

耿菊英，32岁，孟州市城伯乡罗庄村居民。二零零零年，她与8个月大的孩子一起被孟州市610办公室和警察分局人员抓捕并被关入看守所（特派专员知道这些人员的姓名）。据信，她被捕的原因是因为她是一名法轮功学员。由于看守所恶劣的环境造成的营养不良，耿菊英没有足够的奶水喂养婴儿。在她被释放后，二零零二年五月二十五日，警察（特派专员知道这些警察的姓名）在没有任何事先警告的情况下，翻墙而入，从家中将她带至警局。她当时正有身孕。有证据表明，为了将她送入劳教所，警察将她强行堕胎。他们给她注射堕胎药物，并观看全过程，在她处于痛苦中时挖苦嘲笑她。随后，她被送入河南省焦作市看守所。因为拒绝放弃法轮功，她被关入十八里河劳教所至今。有报告称，耿菊英的父亲耿学才，兄弟耿华明都曾因修炼法轮功而被多次拘捕。
