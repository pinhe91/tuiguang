### 善待大法天赐吉祥平安　迫害大法遭现世报应

文: 辽宁大法弟子　清莲 

【明慧网二零二二年八月二十五日】一九九九年大法被迫害初期，我遇到过三位善良正直的人，他们因无条件地帮助大法弟子，都得了大福报。相反，那些利用迫害大法弟子想升官发财的人却遭了恶报，生活都非常凄苦。
下面我就把这个经历写出来，给大家提个醒，让明白真相的世人都有个美好的未来。

第一位是我们单位公安处处长。一九九九年我进京上访，被人举报，被当地派出所非法抓捕，当时我给派出所多个警察讲了真相。我们单位的公安处处长，以这是我们单位的人，我们自己会妥善处理为由，当晚就把我从派出所接回单位，第二天就把我放回家。
与我同行的那位同修在看守所被非法关押了四十天后，要被送去劳教，在我处公安科停留了一会儿，我听说后抱着三个月大的孩子去了公安科，跟这位处长沟通后，这位处长把这位同修放了。

因为当时的心性有限，我警告派出所的片警，你记住大法是被一夜之间迫害的，他还会在一夜之间平反，到平反时我就找你。这个片警怕担责任，推到派出所所长那里，所长报告给了公安分局。过了几天，分局出动了二十来人来我们单位抓我。我当时正念很强，没有怕，静静的等着警察来，结果警察没有来。事后这位公安处处长跟我说，单位公款请那些警察吃了顿饭，事情就这样平息下来了。

又过了一段时间，派出所的片警又来到我们单位院里，告诉我交三千元押金，如果去北京上访，这个押金就不给了，我说我不会给他，丈夫为息事宁人，哀求我说：“给他们吧。”并把三千元塞到我手里，让我去公安科把钱交该片警。我去了公安科，我们处处长及片警都在屋里，片警诱惑我说：“交了吧，到最后这钱还都退还给你，其他那几位都交了，就差你了。”想到他们终究会退还我的，交就交了吧，说着就把钱交给了片警。这时，我们这位处长当着片警的面把我叫到他办公桌前说：“不要给他，他们就是为了钱。”我听了这位处长的话，觉的对，我走到片警跟前，没等片警反应过来，我把钱从他手里夺了过来，放到这位处长手里，让他代我保管。我们几个同修，除了我没交押金，其他几个人都被迫交了押金，到现在都没还给他们。

就是这位正直善良的公安处处长，在我三遇困境时，无条件的帮助了大法弟子，不久他被调到局里，升为了局级干部。最近我丈夫跟我说：“我们所在的城市有一个很高的奖励，这个奖励是很不容易的，全市的警察当中，只有十名额，其中就有他。他还有自己的设备放到单位挣钱，日子过的红红火火，真是名利双收。”

第二位，他是我丈夫的领导，二零零零年我因为发资料，被非法抓捕，我丈夫找到他，让他帮助我丈夫要人。他去公安分局，找到局长，局长答应无条件的放人，同时我自己也在看守所走脱。当时这位领导，竞选处长都没选上，自从帮了大法弟子后，被调到了别的局当了局长，这个局长的职位相当于副市长的级别，后来又出了国，在非洲工作。有一天甲方要求业主去开会，他和他的助理、秘书到达目的地时，他突然感觉心脏异常难受，就告诉他的秘书和助理去代他开会，他在车里休息。当秘书和助理赶到会场时，正赶上黑社会的人拿着冲锋枪，把在场的人全都扫射了。唯一的幸存者就是他，在车里没上楼。因为他善待了大法弟子，官越做越大，官运亨通。这就是善待大法弟子的福份。

还有一位，是以前在工作当中认识的。他儿子杀人，别人说他包庇儿子，以包庇罪在看守所被关押了四年，没有任何结果。他在看守所是给警察做饭的，他看我可怜，很同情我，每天给警察做饭时多做一碗面条，以提桶打水的形式，偷偷的把热乎乎的面条传递给我。大冬天，在那个环境中能吃上一碗热乎乎的面条，真是无比珍贵啊！这样他整整的给我送了一个月饭，突然有一天，出乎所有人的意料，让他收拾行李回家，他来跟我告别。我告诉他说这是你善待大法弟子得福报了。

时间又过了十年，一天我在老家的街里碰到了这位老乡。他告诉我说：“自从从看守所出来后，干啥啥顺，一年能赚十来万。”当时在二零一二年，那个时候一年能赚十来万，是高工资。我告诉他：这是因为你善良。在大法被迫害的时候你善待大法弟子，我师父给你的福报。他很高兴，他还千叮咛万嘱咐告诉我一定注意安全，他说看见我们被迫害太可怜了，充满了对大法弟子处境的担忧与同情，多么善良的人啊！

这是我在正法修炼中，遇到的三位善良的人。真是：“善待大法一念，天赐洪福平安啊！”

与以上善良人士不同的是，那些为了升官发财，对大法弟子跟踪、监视的人，也相应的遭到了恶报。在以上我提到的那位正义的处长调走以后，又上来一位处长，他想往上爬，对我及另一位同修派人跟踪，监视，并窃听我们电话，真是害人害己。他老是想绑架别人害人，结果他的妻子得了鼻癌，看病也花去大量医药费，他妻子长期饱受癌症的折磨，整个人形如枯木，痛苦不堪。前几天，我看见他跟我们单位人聊天，听到了他说长期腰腿疼痛，凄苦度日。

被他指使跟踪监视我们的人，也遭到了恶报，其中一人得了精神病，长期上不了班，他老婆没有工作，这个男的经常打他老婆，他老婆都不敢出门，只要出门，他就暴打她，日子过的非常紧张，他妻子更是苦不堪言。当时他们夫妻跟踪监视我们，同修多次给他讲真相，他们夫妻不听不信，结果遭了恶报。

还有一位因跟踪大法弟子的人，得了类风湿，手指脚趾都变形了，走路像鸭子似的步履蹒跚。还好他明白了真相，做了三退，日子过的还算稳定。

以上是我身边的人所经历的善恶有报的故事，写出来是想告诉那些现在还在对大法与大法弟子犯罪的人就此收手，不要为了蝇头小利断送了自己及家人的前程；抓紧时间，将功补过，为自己和家人选择一个好的未来。
