### 邻居常念九字真言　大法救了他

文: 山东大法弟子　雪莲 

【明慧网二零二二年七月十一日】我是一九九五年九月开始修炼法轮大法的。修炼前疾病缠身，哮喘病、子宫肌瘤、贫血等多种病，天天活的很痛苦。修炼大法后，所有的病都不治自愈，从此真是无病一身轻，我从内心体会到大法的超常，发自内心的感恩师父。
下面我把我遇到的明真相的邻居得福报的故事写出来。

（一）

一九九九年七月二十日以后，江泽民一伙疯狂的抓捕大法弟子，这时我的邻居大姐到我家说：“雪莲（化名），你别怕那些坏东西，他们要再抓你，我住一楼，我不给他们开电动门，他们進不来。你在二楼，你赶快把你师父的书包好放在我家，然后你从二楼窗上跳下来躲在我家，他们不敢抄我家。”我当时听了很受感动，发自内心的说了句，谢谢！

大姐的丈夫是一位高级工程师，他每天出门就念我贴在墙上的的真相标语“善待大法得福报！”并高声说：“法轮功教导我们‘法轮大法好，真善忍好’。”我丈夫见到此情景后，回家大发脾气，对我说：“又是你贴的，别再找事！”我说：“全市大街小巷贴的到处都是大法真相标语，难道都是我贴的？”他无奈地说：“反正咱楼道这些标语即使不是你贴的，人家也会以为是你干的。”这时我趁机告诉他邻居大姐要保护我的话，他惭愧的说：“还是好人多啊！”

转眼间春去秋来。一天晚上邻居大姐跳舞回家，突然发现丈夫昏倒在地，她马上打了120救护车，送医院抢救，事后抢救医生说再晚两分钟就没救了。大姐感激地说：“多亏了他常念‘法轮大法好’，是大法救了他，否则老命就没了！”

（二）

邻居陈平住在五楼，有一天下楼，见一女青年在揭写有“法轮大法好 真善忍好”的小标语，她马上制止说：“不要揭，你这样做会遭报应的！”该女子说：“我可不想遭报应，我再贴上行吗？”陈平说：“好啊，快贴上吧，那你就会得福报的！”

事后陈平告诉我说：“善待大法得福报是千真万确的事。”她的丈夫也姓陈，她丈夫家的兄弟好几个，下一辈就她家是一个男孩，所以都盼望她儿子能生个男孩。陈平一家人都特别相信大法，也经常念“法轮大法好”，结果如愿以偿，儿媳妇果真生了一个男孩，而且他儿子事业又特别顺利，一年赚的钱就买了一栋大房子。

她搬走后还经常回来要大法真相资料看。

（三）

有一天我到集市讲真相，遇儿时邻居家小妹在卖鱼，他是市水产公司的干部，单位破产后失业，她和丈夫在集市上卖鱼为生。一天，我去赶集，和她讲三退的事，小妹还没开口，她丈夫双手高举两条鱼高呼：“法轮大法好！真善忍好！大法师父好！”邻居小妹也连忙说：“大姐我们早就退了，天灭中共，早就该灭了。”

这时，在邻近摊位买菜的老人说：“我没入过党团队，我家成分不好不能入，你把资料都给我，我家的亲戚都相信大法好，都盼着天灭中共！这集市上特务很多，你要注意安全，我来帮你发，你早点回家吧！”

我为他们能明真相得救度感到高兴，发自内心感恩师父对众生的慈悲。
