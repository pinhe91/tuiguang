### 信阳司德利出狱后被610强制锁戴监控器

【明慧网二零二二年四月一日】（明慧网通讯员河南报道）中共对法轮功学员的迫害手段，可谓是无所不用其极。因恐惧法轮功学员传播真相，二十多年来，中共对法轮功学员实施各种迫害，以劳教、判刑的形式将他们关入狱中；以骚扰、跟踪的手段，令他们的生活形同在大监狱中。
最期，河南警察又发明了一种迫害手段，对结束冤狱的法轮功学员，在其身上强行锁戴人体监控器，如同一个行走的牢房，彻底剥夺法轮功学员作为一个自由人的权利。河南省信阳市法轮功学员司德利就遭到这样的侮辱与迫害。

司德利在二零二一年下半年结束三年多的冤狱后，经常遭中共人员的骚扰、监控和威胁。日前，信阳市政法委“610”人员，竟强制在他身上锁戴上监控仪器，并威胁他如果取下监控仪器，就要找他（麻烦）。

他们惧怕这样一位老人

司德利，男，一九五三年出生，曾是河南省信阳市浉河区文化馆美术骨干，副教授。司德利原患有严重的胃病、习惯性感冒、前列腺炎等多种疾病，自从一九九六年开始修炼法轮大法后，他身体奇迹般地恢复了健康，走路一身轻，多年来没吃过一片药，给国家和个人节省很多医药费。

司德利及家人炼了法轮功后，按真、善、忍的标准做一个好人，为社会做了很多好事。有次在报纸上报道本地一个患癌症的姑娘急需四千元钱治疗，他们冒着寒冷的天气，跑了很远的地方找到她，送去了一千元钱。

还有一次，司德利去医院看望病人，遇到医院有一个农民孩子被汽车轧断了腿，肇事司机逃走了，孩子无钱医治。司德利立即回家把家中现有的五百元钱拿来，捐给这个可怜的孩子。这个农民感动得直流眼泪，问他尊姓大名，司德利安慰了这个农民，不留姓名走了。此事曾被当地电视台热线报道：“好心人，你在哪里？”

两次劳教三次判刑，冤狱长达十七年

然而就是这样的好人，自一九九九年“七二零”后，司德利一直遭受着中共的迫害，他多次被绑架、抄家，两次被非法劳教，三次被非法判刑，遭冤狱长达十七年之久。

二零零零年七月及二零零一年三月，他两次被非法劳教，都被劫持到河南省第三劳教所迫害。

二零零一年三月，司德利被“610”和国保大队以莫须有的罪名非法劳教一年。

二零零二年出狱后，司德利坚持修炼，去唤醒被中共迷惑的其他法轮功学员，又被国保大队吴波等绑架，非法拘留一个月后，再被非法劳教两年。

![image](https://user-images.githubusercontent.com/79625284/161239364-5021ce90-cc5b-4520-8133-ef5ce3837fd5.png)

酷刑演示： 电棍电击

在许昌第二劳教所，狱警为了“转化”司德利，用细麻绳吊起来，嘴里堵上毛巾，用电棍电看不见的地方，造成司德利几天生活都无法自理，手和胳膊瘀血红肿。出来时，牙齿几乎全部脱落，头发花白。

在非法劳教期间，司德利的工资卡被“610”以吴天君为首的匪徒们强行抢走，几万元的工资被恶人据为己有。

二零零五年、二零一一年、二零一六年，司德利先后三次被绑架，后被中共法院非法判刑三年、五年半，五年半，均被劫持到郑州监狱迫害。

在第一次被非法关押在郑州监狱期间，司德利被迫害得牙齿脱落，吃饭都很困难，腰部受到严重损伤，走路非常艰难。

在第二次被非法关押在郑州监狱期间，司德利拒绝所谓的“转化”，被狱警关小号迫害，腰被迫害的直不起来了，只能卧床。

二零一六年九月六日，司德利再次被绑架，先被信阳市浉河区法院非法判刑三年六个月，又被驻马店市汝南法院非法判三年六个月，两个判决合并为五年六个月冤刑，并罚金一万元。

在第三次被非法关押在郑州监狱期间，司德利遭酷刑迫害，被折磨的体重只有七十多斤。
