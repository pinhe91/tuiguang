### 炼法轮功起死回生　山东季英梅遭迫害离世

【明慧网二零二二年十月二十七日】（明慧网通讯员山东报道）山东省东营市胜利油田季英梅，在中共发动迫害法轮功运动后放弃了修炼，2019年她被查出宫颈癌晚期，万分绝望中又重新修炼起法轮功，半年后身体完全恢复健康。但自从2021年7月15日季英梅被海滨分局警察绑架之日起，不到一年三个月的时间，季英梅就经历了中共邪党从身体上、经济上和精神上的高压迫害，于2022年10月9日清晨含冤离世，年仅45岁。

![image](https://user-images.githubusercontent.com/79625284/198247004-60cfa3f9-523c-4324-99d9-41f8a5f22a30.png)

季英梅生前照片

季英梅离世，她的追悼会还未开、亲人正处于悲恸中，各单位就接到通知，阻止本单位的法轮功学员去参加她的追悼会。

#### 重新修炼法轮功获新生

季英梅，胜利油田孤岛采油厂油气集输管理中心职工，45岁，有一个幸福美满的家庭。她本以为日子会一直幸福地过下去，天有不测风云，2019年6月季英梅被查出患上宫颈癌晚期，当时医生断定季英梅只有三个月的生命，本市医院不收治，后转到北京中科院肿瘤医院，北京中科院肿瘤医院也不收，后托亲戚关系，费尽周折才住上医院。虽然家人都没告诉季英梅，但是她自己很清楚病情的严重性，感觉生命快要走到了尽头。
万分绝望中，季英梅才想起了多年以前曾修炼过法轮功。记得那时刚炼功没多久，折磨季英梅多年的鼻炎、胃炎、颈椎病、失眠（每天最多只能睡三、四个小时）等多种顽疾就消失了，真正体会到无病一身轻的状态，十多年这些病症都没有出现过了，后来因为中共发动迫害法轮功运动后，她放弃了炼功。这次被查出癌症晚期，因为病痛的折磨和求生的欲望，季英梅决定再次修炼法轮功试试。

因为医生已经给季英梅制定了一套完整的治疗方案，她就一边住院、一边炼功，炼了没多久，她做正式治疗前的检查后，医生惊讶地发现肿瘤竟然已经迅速变小了。检查的结果在医生的预料之外，医生不得不重新制定新的治疗方案。

炼功令季英梅病情迅速好转，使她好好活下去的信心大增，化疗后可怕的副作用没有出现在她身上，食欲比以前还好，体重增加了20斤，气色越来越好，完全不像个癌症病人。同病房的人化疗后都非常痛苦，食水难进，季英梅却没有什么感觉，这本身就是一个奇迹。

半年后季英梅身体完全康复，又回到原来的工作岗位，同事们看到季英梅灿烂明媚的笑脸和充满活力的样子都很吃惊。季英梅的亲身实践可以证明，修炼法轮功对人身心健康确实能起到很大的作用，否则她当时就说自己可能早不在人世了。

#### 被威胁欺骗，成了构陷母亲的“证人证言”

2021年5月8日，孤岛小镇出现“庆祝5·13世界法轮大法日”的横幅，孤岛海滨分局国保大队警察抓捕了季英梅和母亲白兴文及妹妹在内的五名法轮功学员及家属。7月15日7点多，海滨分局警察李海滨、任国栋等及护卫队员和社区人员非法搜查了季英梅的家，非法扣押了她学习用的法轮功书籍和《明慧周刊》几十本以及价值近万元的平板电脑一台（后来返还），她被带去海滨分局办案区做笔录24小时以上。

2021年7月15日，季英梅与母亲白兴文和妹妹被绑架的当天，她们被带到海滨分局办案区，海滨分局警察对她们进行了信息采集非法审讯。非法审讯期间，在警察的疲劳审讯和诱骗之下，季英梅身体和精神状况越来越差，高度紧张，精神恍惚，小腿开始浮肿，稀里糊涂地按着警察的意思完成了审讯。

2022年3月22日前后，家人接到东营区检察院的电话，得知她母亲白兴文被构陷到了检察院。季英梅和妹妹的询问笔录竟然作为指控她们母亲的“证人证言”提交给了东营区检察院，详见《母亲被绑架构陷 姐妹俩还原事实真相》。

#### 提起控告被非法拘留迫害　走法律维权步步受阻

为了还母亲自由和自己清白，10月19日，季英梅和她的母亲白兴文依法向上级公安机关和检察院各个法律部门提交了法律文书：要求上级公安机关、检察院依法监督撤销她及母亲的治安案件和刑事案件；依法控告参与此次事件的任安远（国保大队长）、殷军（副大队长）、刘录英和朝阳派出所警察马玉强（教导员）、宋铭轩等十多个警察滥用职权、徇私枉法。（请详见《修炼法轮功获新生　胜利油田母女被构陷》）

控告发出后，2021年11月6日早上七点，朝阳派出所的警察马玉强、刘仁达等三人和小队队长一起进到值班室。刘仁达亮了一下证件，拿出一张传唤证说是因为季英梅违反了什么治安条例某某条，对她进行传唤。随后他们把季英梅带到了朝阳派出所。一个女警对季英梅进行了搜身检查，又强迫季英梅测量身高、体重；然后把季英梅带到审讯室，强迫季英梅坐在审讯椅上。上午9点左右，警察强迫季英梅在孤岛医院做了核酸检测后，11点左右又去了仙河镇医院强迫季英梅做了血常规、心电图、量血压。检查完又把季英梅带回审讯室。下午4点左右，警察韩传宝给季英梅念了对她进行拘留十天的行政处罚决定。下午6点左右，海滨分局警察将季英梅和另一名法轮功学员一起送到滨海公安局拘留所，非法拘留了十天。在拘留所里，拘留所警察不给她卫生纸等日用品，连牙刷都是别人用过的，特别爱洁净的季英梅这十天的拘留日子不知道是怎么度过的。

因为担心季英梅，她的家人为她聘请了一位律师。2021年11月9号上午，该律师前往拘留所要求会见季英梅，以确认《行政处罚决定书》的案号及相关事实依据，滨海拘留所告知：“行政处罚的被拘留人，一概无法会见。因为疫情期间，只能视频会见，但是看守所的相关电信通道尚未开通。”季英梅的亲人前往海滨分局朝阳派出所询要《行政处罚决定书》及通知书，被拒绝。

考虑到季英梅此前的身体状况，她的代理律师向海滨分局提交《行政拘留暂缓执行申请书》、《行政复议申请书》，并向海滨分局的上级机关山东省滨海公安局提交《行政复议申请书》。海滨分局回复说：行政拘留暂缓执行申请未予通过。同时告知：行政复议应向山东省政府行政复议办公室申请。

2021年11月16日季英梅从拘留所回到家后，代理律师所在的律师事务所接到司法局的电话通知，不许律师代理她的行政复议和诉讼，不准接跟法轮功有关的案子，逼迫律师写承诺书，保证不再接这样的案子。2021年12月初，司法局又找了他们所里两次，要求他们所里整顿，说省里来人督促的，再次强迫律师签承诺书，所里开会学习了好几天，重重压力之下，律师被迫退出季英梅提起行政复议的案子代理。

因对滨海公安局海滨分局的行政处罚不服，2022年1月20日，季英梅对海滨分局警察的违法行为提起行政诉讼。2022年1月25日，季英梅收到来自号码为12368行政诉讼案被退回的短信通知，一直没有任何书面通知。

#### 被单位扣发绩效工资、变相拘禁，依法申请劳动仲裁　立案又无故反悔销案

2021年11月16日是季英梅结束拘留所的日子，她所在单位领导接到派出所的电话，被告知季英梅即将出所。季英梅回家第一天，单位就通知她第二天到队上接受所谓“教育”，原因是季英梅不签所谓“承诺书”。

11月22日，季英梅到队部报到，单位领导再次强迫季英梅在承诺书上签字，被她拒绝，领导以季英梅违反单位内部条例为由强制季英梅离岗学习。大队综治办张青和其他三人拿着一张包括大队书记和各领导在内十六个人签字的《离岗员工停发绩效工资告知书》对季英梅宣布，从当天开始季英梅必须在队部离岗所谓的“学习”。领导告诉季英梅什么时候在承诺书上签了字，什么时候才能回到岗位上班。从2022年11月开始单位扣发了季英梅的绩效工资。

2022年1月6日，季英梅向东营市劳动争议仲裁委员会提起劳动仲裁申请，申请劳动争议仲裁委员会按照《劳动合同法》纠正单位对她的违法处罚行为。1月17日，东营市劳动争议仲裁委员会告知季英梅劳动仲裁申请已经立案，通知她领传票。接到通知的第二天，季英梅领回了《东营市劳动争议仲裁委员会受理通知书》和《东营市劳动争议仲裁委员会仲裁庭组成人员和开庭通知》。

2022年3月22日，季英梅突然接到劳动仲裁委员会打来的电话，工作人员口头通知季英梅她的劳动仲裁申请被销案了，说是因为不符合立案条件。季英梅问她哪里不符合立案条件，不符合立案条件你们之前怎么立上案的？工作人员支支吾吾答不上来。

3月28日，季英梅和家人来到东营市劳动争议仲裁委员会，要求仲裁委员会给出合理的理由。接待的几个工作人员拿不出法律依据，只是想方设法各种推诿。季英梅向他们说明：“我炼功并没有做任何危害社会和违反单位纪律的事情，也没有任何失职行为，单位不应该对我实行停职和停发工资。”她的律师也为她鸣不平：“季英梅已经在单位受到了不公正的待遇，本来寄予很大的期望能来这里解决问题，你们这样对待她，对于她来说如同雪上加霜。”

期间有不明身份的人一直在监视季英梅一行人的一举一动。保安和一个男子一直在大厅盯着，在停车场有个男子一直盯着季英梅的车，季英梅他们上车后，该男子也上了辆黑色的帕萨特（鲁ELV801），看见被人发现了，车上的三人都赶紧戴上了口罩。直到季英梅的车开出大门时，黑车才离开。（请详见《被扣绩效工资 山东省东营市季英梅申请劳动仲裁经过》）

#### 中共邪党是害死季英梅的罪魁祸首

老百姓常有一句话说：“事出反常，必有妖孽”。之前因为控告违法的警察，季英梅被非法拘留关押，她天真地以为事情就此结束了，不料她从拘留所刚回来，单位就开始了对她紧锣密鼓的迫害，逼迫她签“三书”。季英梅拒绝后，单位除了扣除她每个月的绩效工资外，并且强制她每天去单位会议室所谓的学习，变相地把她看管起来。2021年12月12日和13日，连续两天单位找来胜利油田“610”（中共迫害法轮功设立的非法组织）人员温曙明对季英梅谈话施压。

2022年4月20日和21日连续两天，胜利油田“610”人员温曙明、油田综治办“610”张姓（温曙明的领导）、采油厂综治办毕立国、王某某（自称姓王）再次到季英梅的单位骚扰，污蔑她到处控告和申请劳动仲裁。季英梅表示，自己走的是法律程序，不接受这种无理说教。“610”人员温曙明散布歪理邪说、威逼利诱什么现在有摘帽政策、什么人言可畏，劝她们母女不要控告了云云。采油厂综治办王某某临走时话语带威胁：“下次再见，就不是这种情形了。”

在单位里，为了达到转化季英梅的目的，单位工会主席牛雪伟曾经找到季英梅的丈夫，在背后挑拨季英梅的丈夫和她离婚。2022年8、9月份，当季英梅正被病魔折磨因此迷糊的情况下，牛雪伟还欺骗季英梅与她两人照像合影，目的是向“上头”交差，意即她已经在中共“20大”前做了对季英梅所谓的“思想工作”。

在延续中共江泽民犯罪集团的“名誉上搞臭、经济上搞垮、肉体上消灭”的高压迫害政策下，从2021年7月份到最后离世前，仅仅一年不到三个月的时间，季英梅经历了从被警察绑架、非法搜家、非法拘留的无理迫害，到被单位无理扣掉绩效工资、被变相拘禁以及被油田“610”及其他邪党人员骚扰而承受着精神上的沉重压力，她和她原本幸福安宁的家庭也因此蒙受着从经济上和精神上的巨大损失，这给她由于重新修炼法轮功刚刚恢复的身心健康造成了严重的损坏。从医院大夫口中得知，癌细胞已在季英梅全身扩散了，从肺部、骨髓一直侵蚀到大脑。

从2022年4月起，爱说爱笑的季英梅出现了持续干咳、嗓子沙哑，出现说话困难，到话语越来越少，以至郁郁寡言、心事重重。从2022年7月29日母亲白兴文被东营区法院非法庭审起，季英梅的健康状况开始急剧恶化，2022年9月20日，一直照顾她的母亲白兴文突然被海滨分局及朝阳派出所警察带走收押于河口看守所，之后几天季英梅就开始出现高烧昏迷不醒，于2022年10月9日清晨含冤辞世，离世前久久不肯闭上双眼。

一个鲜活美丽的生命就这样消失了，撇下了孤独的丈夫、年少的孩子及她所有深爱的人。季英梅生前，在单位里，由于她工作努力、人缘极好，得到了同事和朋友们的一致认可，当身边的同事看到单位领导对季英梅的不公正对待，都为她气愤不平：“小季人很好啊，为什么这么对她？”在家庭中，因为对妻子的深切相知，敦厚的丈夫不管季英梅的单位领导怎样挑拨，无论跟着妻子承受多大的压力，对她始终不离不弃；季英梅的亲人更是对她呵护备至，一直到她离世。当听到平时充满活力的季英梅突然病重的消息，她的同学好友都感到非常震惊，感念于她平时热情善良乐于助人结下的善缘，他们陆续赶来看望鼓励陪伴她。在季英梅最后的日子里，当他们了解了在这短短的一年多时间，她所走过的艰难的追寻人间公义之路，并为此所做出的努力和承受的一切，由此更加相信了季英梅的人品和她曾经给他们讲述的法轮大法被迫害的真相！
