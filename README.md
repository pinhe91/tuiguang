### 袭击纽约法轮功真相点嫌犯被捕
### 被控两项重罪

【明慧网二零二二年二月十七日】（明慧记者采菊综合报道）二零二二年二月十五日中午十一点五十五分左右，纽约市警109分局的警察在法拉盛缅街与凯辛娜大道的交界处，拘捕了近日多次暴力袭击法轮功多个真相点的疑犯郑步秋（Buqiu　Zheng，音译）。二月十六日在皇后区刑事法庭审理此案，郑步秋被指控两项E级重罪：四级“刑事损坏罪”（Criminal Mischief in the fourth degree）和“仇恨犯罪”（Hate Crime）。

![image](https://user-images.githubusercontent.com/79625284/154437088-f850fcc3-e24c-4cf8-9040-316dbdabed44.png)

图1：二月十五日中午时分，纽约市警109分局的警察在法拉盛缅街与凯辛娜大道的交界处，拘捕了近日来，多次暴力袭击法轮功多个真相点的疑犯。（法轮功学员提供）

![zphr](https://user-images.githubusercontent.com/79625284/154437486-496c0935-0a54-46b1-a5d1-c5a591c4f666.png)

纽约市警109分局的警察正在拘捕疑犯郑步秋（法轮功学员提供）

根据警方消息，这名华人男性嫌犯的名字叫郑步秋（Buqiu Zheng），今年三十二岁，家住法拉盛。在被捕之前警方已对他发出通缉令。

![image](https://user-images.githubusercontent.com/79625284/154437592-5208d381-10a6-42ff-9440-845b0b88fc3b.png)

图2：纽约市警方发出的通缉令，对从二零二二年二月十日以来，连续多天、多次袭击多个法轮功真相点的三十二岁嫌犯郑步秋进行通缉。（警方提供）

#### 疑犯被指控两项E级重罪

二月十六日，嫌犯郑步秋在皇后区刑事法庭出庭。地区助理检察官在庭上向法官劳里·彼得森（Laurie Peterson）陈述，郑步秋在法拉盛法轮功真相点将桌子掀翻、并说“你们的宗教信仰不好”，还毁坏了真相点的资料和播放器等物品。

郑步秋面临两项E级重罪，包括：四级“刑事损坏罪”（Criminal Mischief in the fourth degree）和“仇恨犯罪”（Hate Crime）。

郑步秋的下次出庭时间为三月三十日（下周三）。在等待下次出庭期间，郑步秋被有条件释放。法官要求他必须遵守的条件包括：不能再被逮捕、不得靠近法轮功真相点等。郑步秋在庭上用中文回答“知道”和“明白”自己的释放条件。

纽约法轮大法学会负责人易蓉感谢109分局迅速行动，成功逮住凶徒。易蓉表示：“感谢109分局警员们迅速行动，把这个凶徒缉拿归案，保护社区安全，保障信仰自由，为法拉盛民众和义工带来和平的环境。我们法轮功真相点的同修们，都会一如既往地坚持讲真相，和平理性地反迫害。”

#### 事件回放

纽约法拉盛中心地段共有五个法轮功修炼洪法真相点，分别位于图书馆、地铁口、黄金商场、邮局和置地大厦的前面，其中图书馆前的真相点已摆了二十年了，其余四个是二零零八年“法拉盛事件”后设立的，也有十四年的历史。一年三百六十五天酷暑寒冬风雨无阻，法轮功学员每天都在真相点上，告诉过往路人法轮大法的美好，揭露中共对法轮功信仰团体的迫害，劝中国人退出曾经入过的中共党、团、队。

上周四（二月十日）下午三点多，一名袒胸露臂身上多处刺青的华人男子（后来得知他叫郑步秋），先后攻击了位于法拉盛图书馆和黄金商场前的两个真相点，并将黄金商场前的真相点完全砸毁。凶徒先是来到图书馆前的真相点扯下展板，想继续动手时被义工制止；随后他就来到黄金商场前的真相点，用拳头袭击真相展板，踢翻摆放法轮功真相资料的桌子，之后还用脚踩展板，撕毁法轮功的宣传资料，并把已砸烂的展板拖走。

当时在黄金商场真相点的义工只有六十七岁的徐卫国女士一人，她说，她不认识这个人，与其无冤无仇，攻击场面令人震惊，这是一种什么仇恨？

![image](https://user-images.githubusercontent.com/79625284/154437735-90a69c78-219e-4e95-87d7-96228ec50889.png)

图3：郑步秋正在撕扯法轮功真相展板。（法轮功学员提供）

在第一次攻击后，媒体公布了凶徒作案的图片和视频，但此人并没有收敛。真相点义工Sun Hong说：凶徒于二月十一日下午又来到置地大厦前的真相点前，用脚踢放法轮功真相资料的桌子，被义工制止。随后，他又走到法拉盛图书馆前的真相点，欲破坏真相点。义工们马上报警，警察很快赶到，但此人逃脱了。报警的周小姐说：“警察说：‘只要这个人一出现，你们就报警。’”

十四日，这个凶徒又来到图书馆前的真相点，用脚踹放着真相资料的桌子。学员报警后，警方表示，对二月十日发生的暴力袭击法轮功真相点事件已立案调查，警方已经获得了事件的录像以及作案者的照片，并接到多次报警。警方说只要找到这个人，就会对他实施逮捕并依法判处，同时会加强对五个法轮功真相点的巡视，加强保护。

二月十五日的上午，这个凶徒又开始骚扰真相点，包括黄金商场点、图书馆点、地铁口点。据义工徐卫国介绍，该凶徒早上十点多就出现在黄金商场真相点上，用脚踹放着法轮功资料的桌子，之后逃离。

到了十点四十九分左右，此人又来到图书馆前的真相点，义工王Lily介绍说，凶徒踢翻了放法轮功资料的桌子，扯掉“法轮大法”、“真、善、忍”的小莲花并扔在地上，还想撕扯法轮功的真相展板但没能得逞，之后逃离。

学员们立即报警，市警109分局的警察们很快来到了该真相点现场了解情况，随后警察们开着警车一直在附近街区巡逻。

中午十一点五十五分左右，警察在缅街与凯辛娜大道的交界处锁住了嫌犯并成功逮捕了这名凶徒。

#### 袭击法轮功真相点背后　中共势力鬼影幢幢

暴力袭击事件的幕后黑手隐现，中共势力鬼影幢幢。二月十日下午在案发现场的真相点义工徐卫国说，她根本不认识这个作案者，与其无冤无仇，凶徒是有备而来的。更蹊跷的是，长期骚扰法轮功修炼洪法中心的中共走卒李华红，也出现在袭击现场。从二零零八年至今，李华红因街头攻击、骚扰他人等罪名被逮捕三次，二零一三年被裁定人身攻击罪成。

据知情人透露，来自中国天津的李华红，接受中共政法委、610组织的经费，执行中共迫害法轮功的任务，替中共在海外散播谣言和毒素，不时骚扰真相点。在这次暴力事件中，李华红早不出现晚不出现，在攻击事件发生时现身案发现场，并幸灾乐祸地对洪法的义工说：“拍我干什么，跟我有什么关系吗？哈哈哈，你们报警啊，你们为什么不报警啊？！”

徐卫国认为行凶者有人接应。她回忆：那个凶徒把摊位砸毁后，把真相展板拉走了，没有留在现场。而且她到附近垃圾桶查看，那里根本没有，估计是有同伙来接应他，把毁坏的真相板一起拉走了。

#### 法拉盛社区领袖：立即停止针对法轮功的仇恨犯罪

对法轮功真相点暴力袭击的事件发生后，二月十三日，纽约州第四十选区社区领袖（Democratic District Leader）玛莎（Martha Flores Vazquez）冒着大雪在法拉盛图书馆前举行集会，谴责暴力攻击信仰团体的犯罪行为，要求各级民意代表、警方和地区检察官重视这起仇恨犯罪、侵犯信仰自由的案件，将凶徒抓捕归案，绳之以法。

有过路民众表示，除了要依法惩处凶徒外，还应该将其递解出境。

玛莎（Martha Flores Vazquez）表示，“针对法轮功的仇恨犯罪必须立即停止！我们不允许任何仇恨犯罪，不允许任何针对亚裔的仇恨犯罪，也不允许任何针对法轮功信仰者的仇恨犯罪！这些暴力行为必须立即停止，这里是美国，不是中国！”

玛莎表示，“法轮功修炼者是一个和平的团体，他们在传承和弘扬中国文化，在二月五日的法拉盛大游行上，他们的队伍色彩亮丽、朝气蓬勃。我们需要法轮功修炼者，我们喜欢他们，我们欢迎他们，他们可以自由地信仰，可以自由地修炼法轮大法。我们必须保护他们的安全。”

玛莎进一步强调，“这个社区对仇恨犯罪是零容忍的。我将进一步向政府的官员讲述发生的针对法轮功信仰者的暴力事件，对法轮大法的攻击是绝对不能容忍的，这一切必须停止！让我们一起来保护他们（法轮功修炼者）。”

集会期间，109分局警官在现场维持治安。在集会发言的最后，玛莎表示感谢警察们保护社区民众的安全，并呼吁社区的人和她一起共同发声。她呼吁大家都要发声，现在就需要行动。

#### 民众支持

“中国民主人权联盟”总部第一副主席兼美国东部地区主席白节敏先生来到现场，谴责暴力，要求严惩凶手，保障法拉盛社区安全。白节敏说：“信仰自由是每个人的权利，法轮大法的传播深入民心，也鼓舞了我们这些不是炼法轮功的人。法轮功揭露真相，有的人非常害怕，根据视频和目击证人的证词，凶徒朝‘真、善、忍’三个字砸和踢，说明他对‘真、善、忍’非常地害怕，这是一个什么人，不是很清楚了吗？”

白节敏说：“凶徒是在挑战美国的法律和秩序，危害美国的国家安全，是对美国宪法的侵害，不仅是对法轮功学员的侵犯，也是对走在大街上的每一个民众的侵害。”

凶徒暴力袭击真相点事件激起民愤，在案发时，有围观的民众也拍下了视频，发到了社交媒体，有看到视频的民众路过真相点时，对法轮功学员表达声援。在缅街置地大厦法轮功真相点讲真相的义工谢女士说，“十一日早上我站点，一个年龄六十岁的女士走过来说：不要害怕，我们都支持你们（法轮功学员）。那些人（破坏者）都是败类。”

纽约人权律师叶宁针对事件表示：这是犯罪行为，表面上好像是破坏财物的耍流氓，但其实犯罪性质非常恶劣，“这和美国的教堂实施纵火的行为是一样的，是一种仇恨犯罪。其目标是践踏公民的自由表达。”

叶宁说，共产党出点小钱、雇用了一批人，只要共产党扔出一点肉骨头渣子，他们就会往上冲，这种是共产党花钱雇来的暴徒，目的性很强，冲着真善忍，在光天化日之下进行破坏，这是仇恨犯罪。“如果是仇恨犯罪，可以提起联邦控罪，可以判五年以上的有期徒刑。”

叶宁还认为，暴力事件发生在法拉盛中国新年游行之后，这个时间点值得思考，“这次中国新年庆祝的时候，法轮功展示了最大的阵容，触动了中共海外情治机构的神经，做出这种所谓的’业绩’来，继续骗取中共经费。”

香港移民林威廉（William Lin）先生路过集会现场时，向法轮功学员挥手表达声援。他接受媒体采访时表示：应该将这个破坏美国信仰自由的凶徒驱逐出境，“他们（法轮功学员）在做好事，抓住他后，应该把他驱逐出境。就好像那些人拿了点钱在街上插（红）旗，贪钱，没有廉耻。如果这种人放过他，治安就不会好，是不是？他犯了法，无论大小都要抓他，最好将他驱逐出境。”
