### 由江澤民和習近平之比想得到的

文: 唯真 

【明慧網二零二二年十二月六日】江澤民死了，有人因為近兩三年在疫情清零中的切身感受，忽然覺得原來習近平最可惡、江澤民畢竟讓自己生活好轉了。這其實是一種牆內最容易有的幻覺，也是中共常年洗腦造成的。
到底是誰讓中國老百姓生活好了的呢？是老百姓自己。

同時，中國老百姓吃苦耐勞，在任何自由社會、任何年代都應該能過上像樣的日子，為甚麼在中國就不行？是因為共產黨在所有的層面、所有的時期壓榨中國人、剝奪中國人的基本人權。

中國人被迫養肥了共產黨，共產黨還給中國人洗腦、讓中國人感恩共產黨。

哪個黨魁都代表著中共的利益，不代表老百姓的利益。

江澤民當政時期，中共吃肉你喝湯，並不是中共為了讓你喝湯你才喝上，而是中國老百姓自己自帶的福德。就像一棵飽滿的樹種，遇到合適的土壤、陽光、水分，它自己就會發芽、生根、成長。中共不是土壤、不是陽光、也不是水分，而是碩鼠──你沒發芽之前可能把你生吞活吃了，你結出了果實它會來搶奪果實，還說那些果實本來就屬於它的。

習近平時期，疫情清零讓中國經濟倒退讓你受苦，那種清零本身正是中共的看家本事，習近平在手段上加了高科技，骨子裏都是中共那套思想。而且習近平上台，也是江澤民的意願和支持，二人交易不斷。共產黨內部就是個最大的黑社會，你說誰在裏面是好的？不沾人民的鮮血？放棄幻想吧。

認清中共的本質，才不會對任何中共黨魁抱有幻想，更不會在天滅中共時成為中共的陪葬。
