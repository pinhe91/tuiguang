### 等死的女医生获新生的奇迹

文: 黑龙江大法弟子

【明慧网二零二二年十月二十二日】张女士是黑龙江省某医院的大夫，身为医生的她却对自己的病无能为力。她一身的病：风湿性关节炎，全身的大关节整天有膏药贴着，满身的膏药味；慢性萎缩性胃炎，慢性结肠炎，只能吃点煮烂的面条，馒头、水果都不能吃，油大一点就恶心，人瘦的象麻秆，感觉风大一点就能吹倒；神经衰弱，经常是晚上不能入睡，慢性鼻炎，经常被憋的头痛难忍。
病痛折磨得她性格也变得急躁，经常发脾气。张女士去了许多医院，看了许多医生。在北京协和医院一住就住了一个半月，最后医生告诉张女士：“什么药在你身上不过三天就不管用了，如此的抗药，我们也没办法，你回家慢慢休养吧！”

人都有求生的欲望，张女士回来后又找中医，中医也说身体状况太差，没法再用药了，活几天算几天，回家等着吧。张女士走投无路又找偏方，人家说：“我治不了，别找我了。”

身体越来越差，张女士已经瘦得不到六十斤，真是只剩下皮包骨了，由于长期腹痛，腿佝偻着能好受点，双腿已经无法伸直，不能走路了。张女士二十多天水米没進，喝口水都吐，就等着死了。所有的人包括张女士自己都认为不行了，等着往外抬了。

就在这绝望之时，张女士有幸看到了《转法轮》这本书，张女士打开《转法轮》看到李洪志老师的法像，立刻就感到头发根根都立起来了，有冰凉的东西顺着头发在往外走，头顶象被凉水浇了，冰凉的顺着脊柱往下走，持续了很长一段时间，脑子一下就清醒了。

张女士如饥似渴的坚持看完了一讲《转法轮》，就睡着了。醒来后，她神奇般的能喝水，吃饭了。大法实在是太神奇了。

就这样张女士开始天天在家看法轮大法书，二十天后张女士能盘腿打坐了，又过了几天，腿能伸直了，能下地炼动功了。

法轮大法把张女士从死亡线上拉了回来，给了张女士第二次生命，挽救了她濒临破碎的家庭。当年年底张女士就回到了工作岗位，能正常上班了。

直到现在张女士一直身体健康。前几年张女士家里的钱都用在看病上了，孩子上学、生活都要靠借钱，借债总也还不上，心情不好，两口子干仗闹离婚。学大法后，张女士身心健康，不但还上了外债，家里还购置了小轿车，一家三口日子过得开心幸福。

当地人们都在流传着医院的张医生病得要死了，人瘦的只剩几十斤，就是学法轮功学好的，都感到了法轮大法的神奇。
