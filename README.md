#### 八十七岁的老人扔掉了拐杖
——农村老妪保护真相资料 得福报

文：山东青岛农村老人的女儿口
有这样一位八十七岁的农村老妇人，年轻时，长年的劳累使她身体落下一身病，特别近几年，她腿疼的几乎行走不了，只能依靠拐杖挪动。

老人未曾上过学，大字不识一个，但是她对法轮功特别相信。邻村的法轮功学员告诉她“法轮大法好”的真相，并让她诚念：“法轮大法好，真善忍好”九字真言，能遇难呈祥，逢凶化吉，老人家对此深信不疑。一有时间就念诵九字真言。

村里每周有个集市，法轮功学员免费发放的真相资料，有时会被不明真相的人丢弃，老人家看到后非常心疼，每次她都会一本本、一张张的捡回家，用布擦干净，存放起来。

有一次，老人到邻居家串门，看现邻居正在将几本真相小册子放進灶台烧火，她急了，大声喊：“不能烧！不能烧！这可是宝贝！”说完，就抢过去，抱着真相资料一挪一挪的回了家。她如获至宝，把真相资料擦拭干净。

虽然老人不识字，但是每天都会把真相资料捧出来，用手翻看一遍，一边翻一边对着真相资料说：“你也不认识我，我也不认识你。”女儿看到她那虔诚的样子，又好笑又感动，便对母亲说：“娘，你不要说（真相资料）不认识你了，你的诚心，神佛看的到。”

老人家保护大法真相资料的善举得了福报，困扰她多年的腿痛症状消失了！老人家扔掉了伴随她多年的拐杖，身体愈发健壮，浑身有使不完的劲。

老人家用大锄头把自家菜园刨好，种上了各种各样的菜，供给子女和邻居们吃，气色比自己的儿女都好。前两天，老人的女儿回老家，和母亲一起去山坡挖野菜，挖了整整一下午。回到家，女儿累的腰酸背痛，躺在炕上都不想起来，而老人竟然什么事都没有，坐在院子里择菜呢。

还有一次，老人到城里去，坐电梯时，由于慌乱没站稳，向后倒了下。危急时刻，外孙瞬间就抓住了她，没有发生危险。老人全家都知道这是大法师父保护了她。

老人家的女儿说：“我虽未正式走入大法修炼，但我们全家都对大法和师父深信不疑，我要让更多的人知道大法在我家显现的神迹，让更多善良的人相信法轮大法好，从而也得福报。”

#### 诚念“真、善、忍好　法轮大法好”　邻居胃癌消

文：山东大法弟子

我的邻居先明（化名），去年很长一段时间感到胃痛，吃不下饭，自己也没当会事，再说他家里经济也不宽余。后来他胃痛的实在受不了，就在家人的逼迫下，去医院做了检查。医生告诉家人说是十二指肠癌晩期，手术都不能做了，回家他想吃啥买点好吃的，保养保养吧。没办法，先明就从医院回家了。

我听说后，去看他。他躺在床上有气无力，说话都睁不开眼睛，一百七十多斤的人现在瘦的皮包着骨头。我就凑近耳边上，告诉他大法的真相。他原来对法轮功也有所了解，我对他说你这病只有法轮功师父李洪志大师能救你，你要诚心诚意的念“真善忍好”、“法轮大法好”，这是宇宙的特性、最高的佛法，你这点病算什么？打起精神来。我就教他一句一句的念：“真善忍好”、“法轮大法好”，念了十几分钟，他就睡着了。

就这样，我天天去看他，進门就说“真善忍好”、“法轮大法好”，他也在炕上说“真善忍好”、“法轮大法好”，我告诉他这是九字真言，你要牢记在心，有空就默念他。他说，婶子，我一定天天念。

有一天，我進门后，他就抢着跟我说，念“真善忍好”、“法轮大法好”真管用，真是太神奇了，你看我精神也好了，也能吃饭了，胃也不痛了，身体舒服多了。又过些日子，我再去看他，他脸色红润，脸上身上都长肉了，从那以后，一天天好起来了。

几个月后，我再去看他，他说；“婶子，你看我身体已经全好了，饭也吃的多了，也有力气了，现在都去上班了，我真得好好谢谢你呀。”我说：“你可别谢我，你要谢谢慈悲伟大的法轮功师父李洪志大师吧！”他说：“我们全家人都非常感恩李大师给了我这第二次生命，成全了我们这一家人，我一定要坚信师父，坚信大法，‘真善忍好’、‘法轮大法好’，我会念一辈子的。”
