### 曝光山东招远市洗脑班头目季晓东的恶行

【明慧网二零二二年十月十八日】（明慧网通讯员山东报道）季晓东自二零零八年春天从大秦家镇计生办调到招远洗脑班任头目，因迫害法轮功学员，二零一一年任招远卫生防疫站副主任，混到了副科级。下面曝光季晓东在招远洗脑班的部份犯罪事实。
季晓东对被非法关押的法轮功学员经常骂骂咧咧，用脏话侮辱学员的人格，指使洗脑班打手曲涛、赵秀江、徐建圣等酷刑折磨法轮功学员，手段与她的前任宋书琴一样：关小黑屋、熬鹰、电刑、吊铐、毒打、野蛮灌食等。季晓东长挂在口边的话：我告诉你，我这是第一关，我这一关过不去你别想出去。

二零零八年春天，法轮功学员崔桂芬被绑架到洗脑班。因崔桂芬拒绝所谓的转化，不放弃对真善忍的信仰，被关押在小黑屋里，吃喝拉撒全在里面，不准与家人见面，长时间断绝与外界任何联系，被这样非法关押了近一年，连过年都是在小黑屋过的，季晓东等恶人用这种寂寞残忍的方式逼迫她屈服。
二零零九年十月，法轮功学员李玉凤被招远国保绑架到洗脑班，先被恶人宋少昌扇耳光，后被恶警徐建圣指挥五个同伙把她绑在铁椅子上拽着头发扇耳光。徐建圣一拳打得她牙齿松动，几乎掉了下来。季晓东指使曲涛强行给她戴上手铐关进小黑屋，七天七夜不解手铐。李玉凤绝食抗议十九天，季晓东又指使曲涛和一宋姓恶人对她野蛮灌食，季玉风被非法关押了三个多月，身心受到很大的伤害。

六十多岁的张淑香老人被绑架到洗脑班，她绝食抗议非法关押十八天，季晓东指使打手曲涛、刘学东用钢棍把她打得全身是伤，多次昏死，精神恍惚，被非法关押三个月后，被季晓东和招远610送山东王村劳教所迫。

二零一零年五月八日，季晓东与招远国保王玉成、610宋少昌等策划绑架了考福全和妻子宋桂华，之后又有多名法轮功学员被绑架到洗脑班酷刑迫害，其中七人被非法判刑四到八年。

明慧网曾曝光过季晓东的恶行，请见《山东省招远市洗脑班头目季晓东犯罪事实》一文。

季晓东的个人及家庭信息：

季晓东，女，一九七零年九月生，山东省招远市玲珑镇沟上村人。
现工作单位：山东省招远市卫生防疫站。职务：副主任。
家庭住址：招远市城东区59号（在烟台已买房）
季晓东丈夫：刘明国　工作单位：招远市玲珑镇机电设备公司机电技术科 。
季晓东女儿：刘佳璇　大学毕业已工作。
