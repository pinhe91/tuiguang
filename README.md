### 诚念“法轮大法好”　武汉舅妈死里逃生

文: 湖北大法弟子 

【明慧网二零二二年三月二十七日】二零二零年一月，中共病毒在武汉爆发。作为武汉的大法弟子，我亲眼见证了这场灾祸的惨烈，而我的两位亲属在染上病毒之后，幸运的得到了大法的救度！我讲讲这两位亲属的真实故事。

#### 舅妈死里逃生

当时快要过年了，武汉的市民都在忙着置办年货和走亲访友，大街小巷、商场超市都是熙熙攘攘的人群，人们沉浸在节日的欢乐中，却不知一场灭顶之灾正在悄悄降临！

我的舅舅去闹市区购物和探访亲友，回家之后感觉身体不适就到医院去检查，确诊感染了中共病毒。但那时医院里早已挤满了感染病毒的人，院方只得让我舅舅回家自行隔离。无奈之下，舅舅只能住在家里的阳台上。但很快舅妈就被传染上且发展成了重症。

几年前，我曾经给舅舅、舅妈讲过大法真相，舅舅非常接受，还看过《转法轮》，虽然没有走入修炼，但一直对大法有正面认识，所以舅舅染疫后一直都是轻症。而舅妈在当时不接受真相，还很排斥。她染疫后很快发展成重症，我知道一定与她对大法的态度有关。

舅妈的状况很不好，舅舅就带着她到处去看病。可那时公交车和地铁都已停运，老俩口就乘坐出租车，一个医院一个医院的跑，结果是所有的医院都人满为患，根本连挂个号都没希望。老俩口只得回到家中。進门舅妈就卧床不起了，高烧、呼吸困难、不能进食……舅舅急的像热锅上的蚂蚁，不停的拨打市长热线、110、120等电话求助，依然没有结果，直到在床上躺了八天之后，才有一家郊区的小医院接收了舅妈。此时的舅妈身体已经极度虚弱。

拍片显示她的肺部大部分变白，因为过度紧张和害怕，她又患上了抑郁症，几天几夜不能睡觉，真是雪上加霜！医院几次下了病危通知单，并且一再让舅舅做好心理准备，说舅妈可能挺不过去。

当我知道这些情况的时候，舅妈已经住院半个多月了，舅舅已住进方舱医院。我在电话中安慰舅舅不要担心，接着准备在电话中再次给舅妈讲真相，希望她能转变，但心里有点担心：会不会有安全隐患呢？但我想这是救命的大事，是最正的，不会有问题的，并求师父加持。

电话接通后，传来了舅妈微弱的声音，我让舅妈少说话，保存体力，听我讲。接着我进一步详细的讲述了大法的真相，和自己亲身受益的体会，并且用科学上同频共振的原理，讲解了为什么诚念“九字真言”——“法轮大法好 真善忍好”会有祛病健身的奇效，并且让她马上做三退。最后，我真诚的对舅妈说：“人的生命和健康是最重要的，您就按照我说的，试一试吧！”这一次我怀着纯净的心态和真心希望她得救的心，效果非常好，舅妈接受了真相，并且答应我一定诚念“九字真言”。

此后，我每隔一个星期就给舅妈打一次电话，了解她的情况，第一次电话中，舅妈告诉我，本来医院要把她转到方舱，突然又改变决定，转到了武汉最好的大医院。这里的环境和医院条件都好很多。我告诉舅妈：“你转运了，这是大法师父在管你。”舅妈身体一次比一次好，先是退了烧，接着可以进食了，后来又拔掉了输氧管，还能围着病床走几圈了。

一个月后舅妈康复出院！

回家后，舅妈在电话里告诉我：原来我给她讲真相，她不但不接受，还很反感，而这一次她真心接受了，没想到就过了这一大难！我对舅妈说：“这不怪你，是我没有修好自己，没让你在我身上看到大法的美好。但你能在紧要关头选择相信大法，说明你和大法是有缘的。”我一再嘱咐舅妈，平时有时间还是要多念“法轮大法好”这句真言，舅妈高兴的接受了！

#### 婆婆的故事

我的婆婆今年八十九岁，和我们生活在一起四年了。从她来到我家，我就给婆婆讲述了大法的真相，并且把大法的书籍以及真相资料拿给她看，让她多了解大法。因婆婆平时看到大法弟子的为人处事都是在按照真、善、忍的标准来要求自己，所以她十分认同大法，而且平时有空时就念一念“法轮大法好”，身体一直非常健康！

二零一九年的夏天，可能是由于空调调的温度偏低，她受凉了。一天早上她起床后喊我，我过去一看，婆婆的一只眼睛既睁不开也不能完全闭上，嘴巴歪向一边，喝水吞咽困难，并且顺着嘴角往外流，说话几乎发不出声音，只有沙沙的气流声。

我赶紧带婆婆去了武汉同济医院，挂了专家号。这位专家是一位七十多岁的老教授，医学经验丰富，她一看婆婆的状况，立即诊断为颜面神经炎，俗称面瘫。因为婆婆年事已高，不能完全恢复，只开了一些营养神经的药，就打发我们回家了！

家里的亲戚听说后来探望，出主意说让婆婆看中医、做按摩、扎针灸等等。

亲戚走后，我真诚的对婆婆说：“妈，医院专家的话你也听到了，开的那些药你想吃就吃，但是你也看了法轮大法的真相资料，里面有很多身患绝症的人通过诚念‘法轮大法好，真善忍好’身体康复的故事，您也诚心的念那九个字吧！”婆婆非常坚定的点了点头。

之后，我看到婆婆每天除了吃饭睡觉，只要有空就在不停的念“九字真言”，嘴里发不出声，就在心里默默的念。

不到一个星期，婆婆完全恢复了：脸上一点面瘫的痕迹也没有，说话声音正常了。亲戚们得知了消息，夸我照顾的好。我说：“我什么也没做，是婆婆自己相信并诚念‘法轮大法好，真善忍好’得福报了！”

婆婆告诉我，从那以后，她每天都要念五百遍“法轮大法好”，她怕记不住数字，就拿一个算盘，每念十遍就拨一个算盘珠，念到五百遍为止。婆婆十分的虔诚！现在她的身体很好，有什么小毛病出现从不当回事，因坚持念“法轮大法好”，很快就好了！

这是发生在我身边的真实故事。看到亲人们了解大法真相后的变化，我感受到师父的洪大慈悲和大法的无边威力。如果没有邪党的迫害，将有多少生命得到大法的救度啊！同时，也知道大法弟子的责任的巨大。我们一定要修好自己，才能在师父的加持下，救更多的世人！
