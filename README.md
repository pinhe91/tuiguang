### 一大家人长期轮番遭迫害　内蒙古赵福霞含冤离世

【明慧网二零二二年五月二十一日】（明慧网通讯员内蒙古报道）内蒙古赤峰市巴林左旗法轮功学员赵福霞，与三个姐妹及父母一大家人长期轮番被抓被关押等迫害，二零一七年正月十五日饱经苦难的母亲含冤离世，前两年大姐赵桂荣在一次被骚扰过后含冤离世。二零二二年五月十日，赵福霞含冤离世，终年59岁。
赵福霞的父母赵玉、张桂芝是农村人，生性善良，两位老人与四个女儿赵桂荣、赵福霞、赵春霞、赵春华，及两个女婿，都修炼法轮大法，身心受益，家庭和睦幸福。

自一九九九年七月中共发动对法轮功的疯狂迫害后，这个按真、善、忍标准修心向善的大家庭承受了巨大苦难，祖孙三代人在被绑架、抢掠、关押、骚扰、恐吓等迫害中度日。老老幼幼，多人多次被绑架关押、遭毒打；大女儿赵桂荣多次被绑架、抄家，被非法劳教、判刑七年；二女儿赵福霞被绑架、关押、毒打、群殴；三女儿赵春霞遭非法判刑等迫害，三女婿贾彬两次被非法判刑五年；小女儿赵春华多次被绑架关押、小女婿吴井刚被非法判刑六年。

二零一二年六月四日，赵福霞去姐姐赵桂荣家，也被巴林左旗公安局警察非法抄家、抢走了家中的人民币六百八十元，后又上房将卫星收视大锅用脚踹碎，还将房东家的大锅给拆了下来。赵福霞被绑架、毒打（被打晕）、非法关押一天、被用刑具折磨一天。

当天早晨五点多，巴林左旗“610”的王立新、林西县公安局、左旗公安局、国保大队教导员李宏柱、大队长田立成、高树军、齐柏林等十多名警察、便衣闯到妹妹赵春霞家中，见大门上锁，就像土匪一样跳墙入院，在没有任何手续的情况下非法抄家，翻箱倒柜，将家翻得一片狼藉，抢走电脑、打印机等个人物品，抢走价值两万多元的财物，还抢走一万四千多元现金。两个年幼的孩子遭到恐吓。

赵春霞被非法关押在巴林左旗看守所，丈夫贾彬被迫流离失所。两个未成年的孩子二零一二年八月十四日身披“我要妈妈，我要吃饭”的条幅喊冤，一度被中共警察关押。

二零一三年一月，妹妹赵春霞被非法关押在左旗看守所七个多月，赵福霞带着赵春霞的两个孩子去巴林左旗检察院、法院等处要人。巴林左旗法院接待她的人说让她星期五来，到了星期五，也就是二零一三年一月十一日上午九点多，赵福霞领着赵春霞的两个孩子来到法院门口，把门人不让进，赵福霞和他们理论，冲出五个男人，都是法院人员，拖着赵福霞拳打脚踢。

![image](https://user-images.githubusercontent.com/79625284/169644751-9be11a0c-e46b-4ceb-995d-d39350dea177.png)

酷刑演示：拳打脚踢

光天化日之下，法院门口，五个男人打一个女人，吓得两个孩子哆哆嗦嗦的，不敢哭不敢叫。赵福霞大声抗议：法轮功没犯法！因而引来了不少围观的人，法院的人一看人多了，才赶紧跑回去。

二零一四年二月十日下午三点钟左右，赵福霞准备坐车去呼市一监看望被非法判刑的姐姐赵桂荣，在林东镇火车站买好票刚到候车室安检完等车，候车室一专门检查身份证的男警突然闯到她跟前问：你是赵福霞吗？并索要赵福霞的身份证和火车票，立即没收；随即又抢了她的包，抓住她不放手，又来了几个警察，其中有两个女的，把她带到安检桌旁边。另一男警把包里的东西全部倒在桌子上，抢走包里的一千五百元钱和二十二张一元真相币。

赵福霞被巴林左旗铁路派出所警察和巴林左旗公安局国保大队李宏柱、高树军等人劫持，手机里的一切联系号码被删除，无法和家人联系。当天半夜，巴林左旗公安局国保大队警察和特警十多人，逼迫赵福霞带着他们去非法抄家。警察一个一个地从赵福霞家后墙跳到院子里，砸开房门，翻箱倒柜，连水缸都翻了，搞得一片狼藉。之后，警察再次非法将赵福霞绑架。

二月十二日早晨，李宏柱、高树军和一个拿相机的人又到赵福霞家，赵福霞家的小孩给他们开了门。李宏柱等人再次非法抄家，到处拍照。

赵福霞被绑架、拘留家人都不知道，她的两个妹妹多次去左旗610、公安局国保大队要人610的高艳国才告诉她们。赵福霞被非法关押在巴林右旗看守所，看见两个妹妹领着两个小外孙女来到看守所，赵福霞就商量希望见孩子，警察不让，赵福霞说让孩子在窗户上看一下也行，因孩子太小没离开过她，怕孩子想坏了。警察告诉她是你们巴林左旗不让见。从那时起赵福霞不吃不喝，躺下不起床了。女警白丽（音）踢她三脚。二月二十六日赵福霞被妹妹接回了家。

二零一六年六月十三日，赵春霞的大女儿贾雪冬因给学生讲真相被左旗国保黄建、时玉国等和白音诺尔镇派出所警察绑架。六月二十七日，赵春霞与大姐赵桂荣、二姐赵福霞、小妹赵春华、赵春霞的父母赵玉、张桂枝等众亲友二十多人去公安局找黄建等要人。十点三十分，黄建准备开车回家，赵春霞和赵春华看见了黄建，要找黄建谈话，宋海龙冲过来指着赵春华骂骂咧咧，好几个警察冲过来，先后将赵春霞、赵桂荣、赵福霞、赵春华及她们的父母赵玉、张桂枝连拖带打拖进公安局大楼审讯室。赵福霞的女儿女婿在外地打工，她要照顾两个上小学的外孙女。中午十一点多，赵福霞的手机开始叫了，她要接孩子，警察不让她走，后来老师打来电话让赵福霞去接孩子，警察也不让赵福霞走。下午三点， 警察把赵春霞、赵桂荣、赵福霞、赵春华塞进特警车，劫持到拘留所。姐妹四人无端被非法关押十天。

二零一七年四月二十一日上午，三妹赵春霞和出狱不久的三妹夫贾彬，去给老人收拾屋子搞卫生，擦玻璃时，无端被巴林左旗公安局国保大队绑架、非法抄家，后被非法判刑。

二零一八年九月十四日，赵福霞坐了十多个小时的火车赶了几千里的路程，去看望身陷冤狱的妹妹赵春霞，内蒙古女子监狱以赵春霞没“转化”为由不让她们姐妹相见，剥夺了她们的接见权。

此后，赵福霞不仅要照顾两个上小学的外孙女，还要照顾八十多岁的老父亲，加上自己和一家人被迫害的痛苦，再加上两位亲人的含冤离世的痛苦，这几年赵福霞日渐消瘦，于二零二二年五月十日含冤离世。

关于赵福霞一家人遭受迫害情况，请见明慧网文章《内蒙古巴林左旗祖孙三代十多人遭受的迫害 ——百个遭中共残害的家庭（59）》。
