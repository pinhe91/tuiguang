### 保定市雄县杨智雄在医院离世　仍戴着手铐脚镣

【明慧网二零二二年八月十三日】（明慧网通讯员河北报道）保定市雄县五十八岁的法轮功学员杨智雄于二零二二年八月十日晚上八点四十六分在河北唐山协和医院重病监护室离世。他离世时还戴着脚镣手铐，瘦得皮包骨，眼睛睁得大大的。又一个善良人就这样被迫害致死，死不瞑目。

![image](https://user-images.githubusercontent.com/79625284/184475847-26da9d3c-10fd-4e17-8380-3aff0c04430e.png)

杨智雄

杨智雄曾进修于河北经贸大学，原是河北《法制日报》社记者，后在石家庄卓达房地产公司主管销售，后在北京某外企公司任办公室主任。因坚持修炼法轮大法，二零零三年六月他被绑架、非法判刑四年，妻子祁荭瑾被非法劳教两年。杨智雄出狱后失去工作，靠打工维持生活，二零零八年他被骗至保定市所谓“法制教育中心”洗脑班非法拘禁一个多月。

二零一二年十月上旬，杨智雄到北京中德环保公司应聘做销售主管，上班不到一个月，二零一二年十一月三日在北京工作单位的宿舍被雄县国保警察郭军学等人绑架、构陷，十二月二十五日雄县法院对杨智雄非法开庭。整个开庭过程证明了六一零、公、检、法是没有任何法律依据的恶意构陷，恰恰证明了杨智雄在哪都是做好人。然而雄县法院相关人员枉判杨智雄三年，二零一三年一月十六日他被劫持到唐山冀东监狱迫害。

二零一九年九月二十六日，杨智雄驾车在容城高速检查站被容城县公安局非法拘留。在获知他是法轮功学员身份后，上报雄安新区公安局，雄安新区公安局伙同容城公安局、雄县公安局非法抄了他的家。在容城拘留所数日后，十月十日，杨智雄被转入雄县看守所继续关押。国保警察给他罗列了一系列罪名。

二零二零年十二月一日，杨智雄被非法开庭，被非法判刑六年半。杨智雄上诉后，二零二零年十二月九号，被非法维持冤判。大约在二零二一年四、五月，他被劫持到冀东分局第五监狱第五监区。

在一年多的时间里，监狱不让杨智雄给家里打一次电话，更没有视频会见过。家属去过，监狱以疫情为由，不让见。

二零二二年五月及六月，监狱给家属打过两个电话，一次是给杨智雄要生活费500元，一次是给杨志雄治疗腰椎间盘突出3000元。

七月份，家属接到监狱的电话，说杨智雄病重，住进唐山协和医院重症监护室，让家属去看。家属来到唐山协和医院，见到杨智雄身体非常消瘦，身上戴着手铐、脚镣，插着各种管子。医生介绍说：“杨智雄被诊断为骨髓瘤，身体状况不太好，不敢对其化疗，只能给其输血及蛋白，让其恢复一些后，再进行化疗。”

看护的警察悄悄对家属说，赶快接回家吧，想见谁就见见谁，想吃点什么就吃点什么，这说明杨智雄的病情非常严重。

但是，中共邪党多年对法轮功的迫害，对杨智雄的迫害，给他的家庭带来了巨大的灾难，他妻子祁荭瑾现在音讯全无，不知在哪里打工，多年没有回家；岳母被迫害致死；儿子因父母多次遭到绑架及关押，加上邪党的造谣宣传，对大法产生误解，不想马上签字接父亲回家。

短短的一年多的时间，从一个健康、开朗的壮汉，被迫害得虚弱、多病到倒下直至死去，不知杨智雄在唐山冀东监狱遭受了怎样的折磨和虐待。

关于杨智雄遭受的更多迫害情况，请见明慧网文章《曾遭二次冤狱　河北雄县杨智雄又被非法判七年》、《从绑架到庭审杨智雄，看河北雄县公检法违法犯罪》、《原河北《法制日报》社记者“被失踪” 面临冤狱》等。
