### 黑龍江肇東610接爆電話　法輪功學員蘇廣明無罪獲釋

【明慧網二零二二年八月四日】（明慧網通訊員黑龍江報導）法輪功學員蘇廣明於七月二十八日從黑龍江省肇東市看守所被無罪釋放回家。而綁架他的肇東市公安局「610」頭目王立軍，在此之前大叫：「這幾天我的電話都快給接爆了。」
蘇廣明的老家是黑龍江省肇東市昌五鎮，因遭迫害，在外地顛沛流離二十年。二零二二年六月二十七日晚，蘇廣明在大慶市出租的樓裏被突然而至的警察綁架。原來，肇東市公安局「610」頭目王立軍，在二零二二年六月份的核酸報告中發現了蘇廣明的信息。

於是六月二十七日白天，王立軍冒充防疫人員給蘇廣明家人打電話，索要現在的家庭住址，然後帶「610」警察與肇東市昌五鎮派出所所長肖建國等警察，於當晚驅車到大慶市蘇廣明的居所，拉閘斷電，在蘇廣明開門查看停電原因時，他們破門而入，綁架了蘇廣明。

王立軍把蘇廣明非法關押到肇東市看守所。通常警察將法輪功學員關入看守所，就是打算要進行司法迫害。

七月初，王立軍多次找蘇廣明的妻子，問蘇廣明家的樓是誰的名字，蘇妻回答是自己的名字。王立軍當時說：「如果沒有節外生枝的事情發生，半個月就放人。」七月十日，王立軍又叫蘇廣明的妻子去公安局，並偽善地說：煉法輪功的人都是好人。蘇妻反問：那你們為甚麼還抓好人？你不說半個月就放人嗎？此時王立軍大吼：「蘇廣明是網上通緝的在逃犯，這幾天我的電話都快給接爆了。」聲音很大，也很心虛。

王立軍的幫兇、警察王國傑也在一旁拍著桌子大吼大罵。蘇廣明的妻子沒有怕他們，說：「善惡有報。迫害煉法輪功的人會遭報應，並會殃及子孫的。」王立軍連忙陰險說：「你是不是也是煉法輪功的？你怎麼啥都知道？」蘇妻說：「法輪功真相資料誰家都有，你家沒有呀？」

蘇廣明的家人後來請了維權律師，律師到看守所見到蘇廣明。現得知，蘇廣明已於七月二十八日被無罪釋放。

據悉，王立軍從來不敢見維權律師，每次維權律師來他都逃避。
