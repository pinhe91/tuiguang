### 肺癌晚期的弟弟又活过来了

文: 东北大法弟子 

【明慧网二零二二年六月二十三日】我是一九九九年“七·二零”之前得法的大法弟子。修炼中每一步都离不开师尊的慈悲保护。下面把我弟弟起死回生的事例写出来，见证法轮大法的超常与神奇。
二零二一年十月的一天，弟媳给我打来电话说：“姐，现在我们在医院，你弟弟有病住院了。”我问她现在是什么情况？弟媳带着哭腔说：“先前孩子们也不跟我说实话，我从宁夏回来后孩子们才告诉我，他爸是肺癌晚期，而且气管长个瘤子不能做手术了，只能保守治疗。饭、水果什么都吃不了，只能喝一点点稀的不见几个小米粒的稀粥。已经二十多天不能躺了，只能坐着。医生说没救了，孩子已经把寿衣都准备好了。”说着弟媳在电话中大哭起来。

我听了心里一点儿也没着急，也没害怕，告诉她没事儿。因为我修炼法轮大法二十五年了，知道大法师父无所不能。我安慰弟媳：“别急，肯定有救，前提只要你们诚心相信。”弟媳知道我是修炼法轮大法的，她说：“只要能救了他怎么都行。”

第二天，我带了真相MP3赶到医院，让弟弟先听大法真相，并对他们讲：“要诚心敬念九字真言——法轮大法好，真善忍好。”弟媳和侄儿们都很认可。同时我给弟弟讲了当人的目的和人得病的根本原因，并提醒他：“你现在的情况，现代医学根本就束手无策了，只有大法能救你，但前提你要做个真正的修炼人，因为大法不是给常人治病的，是来度人的。”

弟弟很认可。当天他的身体就发生了变化，由原来的喝口水都痛的直喊“妈呀”的状态，转变为能喝一水杯稀粥，还能吃点水果也不怎么疼了。

几天后，我把师父在大连的讲法录音MP3带给他，弟弟用耳机天天听。我又用手机把五套功法的动作拍下来，给他看。弟弟在身体允许的情况下就跟着炼，有时炼的多一些，有时炼的少一些，但天天都在坚持炼。

通过不断的听师父的讲法，弟弟从内心深处明白了人得病的根本原因。弟弟这回真的明白了，彻底改变了原来的观念，坚定的要做个真正的修炼人，无论剩下的时间还有多少，都要真心修炼。奇迹发生了：二十多天来一直都坐着躺不下的弟弟，这回能躺下了，还能睡个好觉了，而且基本上什么都能吃了。

如今，半年的时间过去了，曾经被医院诊断不治的弟弟告别了死神，健康快乐的走在修炼法轮大法的金光大道上。弟弟逢人就讲法轮大法好。弟媳也因此走入了法轮大法修炼的行列，全家人沐浴在法光中。

弟弟全家人对大法对师父的感激之情无以言表。弟弟说，是大法师父给了他第二次生命。弟媳说，是大法师父赐给他们一个完整的家。

弟子叩谢师尊慈悲救度之恩！
