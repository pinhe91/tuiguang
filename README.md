# 诚念真言神迹显　尊重大法洪福伴

【明慧网二零二一年五月十九日】（明慧记者玉紫菱综合报道）一声声恭敬的祝贺，一句句诚挚的感恩，来自中国大陆和世界各地的肺腑心声通过明慧网敬献给慈悲伟大的师父——法轮功创始人李洪志先生。其中不仅包括修炼多年的法轮大法弟子，也有刚刚走入修炼的新学员，还有很多明白真相、受益于大法的世人。
今年的5.13，很多尚未走入修炼的大陆各界民众表达对李洪志大师的虔诚感恩。他们见证了身边大法弟子的善良坚忍，体会到大法的慈悲美好，体验到九字真言的神奇威力，也亲身经历了尊重大法、善待大法弟子所得到的福报和保护。

很多人讲述在退出中共邪党组织、诚心敬念“法轮大法好，真善忍好”九字真言后，转危为安、遇难呈祥的神奇经历：有的绝症病人恢复，医生都称“奇迹”；有的遭遇车祸有惊无险；有的危重病人昏迷不醒，身边的家人坚持念九字真言，而化险为夷。还有的讲述大法带给全家人身心健康、平安幸福、生意红火的福祉。

一个个震撼人心的神迹，令亲身经历者及身边亲友深信大法的超常威力和神圣美好，深深感恩大法师父的救命之恩和慈悲看护。很多人由对大法的善念，开始用实际行动支持大法弟子，帮助传播大法真相，制止中共邪恶对修炼家人的迫害。

重庆市南岸区一位大法弟子写道：“我的家人及亲友都明白大法真相，深知法轮大法好，支持我修炼，全力站出来制止邪恶对我的迫害，他们也因此而福报连连！”

山东烟台招远乡村一老伯写道：“我是个直肠癌患者，我能活到今天，就是因为有大法师父保护，要不六年前我就不在人世了。就因为我知道大法好，常年念‘法轮大法好，真善忍好’得到李大师的保护，不但癌症好了，现在变的无病一身轻，还越活越年轻！我常念叨：‘李大师啊，您辛苦了！我的命是您给的，我怎么样感谢您啊！’”

这位老伯还常对家人说：“谁也不许说大法一个不字，他是咱的救命恩人哪！”他的孩子都相信大法好，也都退出了中共邪党组织。

# 【庆祝513】如梦方醒

文: 中国大陆大法弟子　

【明慧网二零二一年五月十八日】我是刚修炼法轮大法一年多的大法新学员。在修炼法轮大法之前，我是一个自私自利，天天只想着怎么挣钱，如何吃好喝好的人。我认为自己经商小有成就，所以什么都不信。那时的我，是个完完全全的无神论者，而且脾气特别暴躁。
然而一年多来的经历让我有了非常大的变化，现在的我，和那时判若两人。

我家是搞个体经营的，我妻子是修炼多年的法轮大法弟子。这么多年来，家里什么事都很顺利。到了二零一八年年底，我的身体经常感觉不舒服。逐渐的，越来越严重。最严重的时候，胃里面全都是气，胀的象打满了气的皮球，一点食物都吃不下。那种胀痛的感觉真是让我难以承受。

到二零一九年年初的时候，我的病情加重了，从胃到小肚子整个腹部胀的非常厉害，根本不敢吃饭，只能少喝一点粥充饥。到医院经检查，结果显示消化系统没有好的地方，全都有炎症。从此以后，我开始了四处的询医、治病。

我是西医也看了，中医也看了，省城里最好的医院都看了。吃药无数，打针无数，用尽了一切办法，也没有治好我的病。体重从一百八十多斤掉到了一百五十多斤。医生摸了我的脉，告诉我说，我体质弱的不如一个妇女。那时，我真是感觉活着没什么意思。

我妻子经常和我说，如果我修炼大法就好了。可是，当时我根本不相信，还经常和她辩论。就这样，我每天度日如年、心情压抑的忙碌着生意。

到了二零一九年年底的时候，我的病情更严重了，阑尾部位也时常疼的厉害，家里的生意都不能打理了。妻子又和我说让我修炼法轮大法。当时我和她开玩笑的说：“让我修炼也行，给我托个梦，我就信。”妻子没说什么，我也没太在意。

没想到第三天的晚上，我真的就做了一个梦：早上，我去路边的摊位吃早餐。天还是黑的，东方的天空刚有一点点发白。我坐在了靠路边的座位上，刚坐下，就听见靠东侧邻桌的人喊：“看看，那是啥？”别人都起身到棚子外面去看。我也好奇的一侧身，就出了早餐棚。

我看到东方的天空上，有一小块粉红色的亮点，逐渐的，越来越大，最后一下子就散开了，变成了方方整整的四个大字——“法轮大法”，这四个形容不出来的粉色的大字，太亮了。随后，有位男子的声音说：“到现在为止，人类社会已结束。”

其次，不要被中共妖魔化法轮功的妖言迷惑，不参与迫害法轮功，而是善待法轮功学员，诚心敬念“法轮大法好、真善忍好”，这已经是积德善行了，如果心有余力还可以广传真相救人，那是积无量功德的巨大善举，你想一想生命的未来该有多大的福报？真的是不可限量，是古人想遇都遇不到的万古机缘，今天的人类遇到了。但是不知道珍惜，错过了这个道德回升的机缘，失去的将是永远都不可弥补的生命损失，用痛悔都形容不了那个时刻绝望的心情。

古人讲“仁义值千金”，很珍惜这个道德标准，按照这个仁义标准做事，福报就不可思议，而今天人遇到的宇宙特性真、善、忍，是多少钱都买不来的，珍贵的已经无法用语言来形容了，但又不是高不可攀，只要人愿意按照真、善、忍去做好人，人所得到的福报是古人无法高攀的；同理，人拒绝真相，继续被中共谎言带动着迫害法轮功，诋毁真、善、忍，生命所遭受的恶报也是前所未有的。

今天这个特殊的选择时刻，谎言与真相并存，恶报与福报也同时并存，人的选择就尤为关键。一念之差，就是地狱与天堂之别。孰轻孰重，相信人心中自有一杆秤，能看的分明，衡量的明白。
