### 修大法获健康　长沙王娟讲真相遭迫害风雨飘零

【明慧网二零二二年十月十二日】（明慧网通讯员湖南报道）二零二二年九月二日，长沙市马王堆派出所的黄姓警察打电话给王娟的先生，说要到他家来看看，随后来了两个人（包括黄姓警察在内），他们要给王娟照相，要求和王娟坐在一起，王娟没听他们的，就直接走到他们对面，向他们反应自己无缘无故被抓被打的情况，他们说：我们是不会打人的，如果你说的情况属实，打人者是会受处罚的。他们也知道自己在说谎。
王娟也知道他们在说谎，就直接问他们：我做一个好人，有什么错，为什么反反复复抓我打我？他们问王娟有证据吗，王娟说证据都在你们那儿！他们说：你之前是因为法轮功被抓，你现在还练吗？王娟的先生被吓怕了，就顺着说是没练了，王娟对先生说别乱讲，并说中国十四种邪教里没有法轮功，做“真善忍”没有错！炼法轮功身体好了，不炼功一身是病。他们说有病就去医院，有医保，做真善忍可以，就是不要炼法轮功。他们明知道法轮功就是“真善忍”，岂不是自相矛盾？

后来不知什么时候，他们偷偷拍了王娟的照片，直到王娟看到两个警察互使眼色说拍了，然后起身找她先生东扯下西扯下，就说有事要走了才知道的。这是他们迫害大法弟子惯用的伎俩！
这次已经记不清是第几次来骚扰了，此前，王娟被多次骚扰过，还多次被绑架，惨遭毒打，甚至判刑。

#### 遭残忍毒打

最初是在二零一九年十月二号，王娟先生驾车带着王娟、王娟婆婆和女儿去重庆游玩，下午三点多钟的样子，刚到重庆边上，就被仙女山派出所的警察绑架，他们审问王娟，王娟不配合，结果电脑坏了，又将王娟送到酉阳县，到达时天已经黑了，王娟就被关进酉阳县公安局的地下室里，警察连夜审问，王娟说自己从来不做坏事，为什么要抓好人？所以不配合他们的无理要求。

当时是三个警察（其中一个叫明承东，一个叫张涛），有两个抓住王娟，张涛就使劲掰王娟的手腕，掰到了极限还不放手，痛的王娟瘫倒在地上，他们还大声叫骂：你装你装！他们几个轮番审，还把王娟拖到体检室里反掰手腕，审问时威胁王娟不能闭眼，否则就泼凉水。反反复复折腾了一个晚上，直至王娟手腕处鼓起一个大坨，整个手臂又青又肿，手筋连着食指、拇指都严重损伤！王娟痛得死去活来，他们还扬言说：我们就是要活摘你的器官！恶警一边折磨着一个弱女子，一边享受着折磨别人带来的刺激，在王娟痛苦的喊叫中哈哈大笑！毫无半点人性，变态至极！

当晚他们还把王娟先生关在王娟隔壁的小屋单独审问，还鼓动王娟的婆婆劝其儿子与王娟离婚，非常无耻！第二天，逼着王娟的先生带着一帮警察到湖南长沙王娟的家里，非法抄家，抢劫走了王娟所有的大法书等私人物品，未开任何清单。

#### 被枉判一年

随后王娟被非法关押在黔江看守所，被酉阳县检察院、法院非法判刑一年。在看守所里，王娟被罚洗厕所，搞卫生，不许坐等，受尽凌辱，那个被伤害的手，足足十个月才慢慢恢复。在狱中，善良的王娟无怨无恨，尽管先生每月都打了钱给她用，她自己却省吃俭用，她想只要能维持生命就行，把省下的钱全部用来帮助别人，最后狱警和犯人都被王娟的善良和无私所感动！

出狱回来几个月后，酉阳县公安局来了两人到王娟的家里，联系长沙市芙蓉区马王堆派出所，芙蓉区公安局及其他相关单位，继续加害王娟，长沙市马王堆派出所经常打电话来骚扰，并联合街道、社区监视和威胁王娟，王娟不配合他们。

#### 再遭毒打

二零二二年二月二十八日早上六点多钟，王娟打开房门准备出去，被三个躲在门外的便衣强行拦截，他们说是芙蓉区派出所的，进门就抄家抢劫，搜走了大法书和三张师父的法像，王娟和先生双双被绑架到文艺路派出所，先审问了王娟先生，后在审问王娟时，要王娟不炼了，王娟不配合，以丁俊为首的几个警察就破口大骂，骂的不堪入耳，非法审到凌晨两点，强行将王娟拖到长沙市八医院去做体检。期间，他们换了几班人马轮流折磨王娟。

三月一日早上，他们中又来了一个一米八高、头大耳肥的警察，他对王娟说：你昨天没配合，今天必须配合，要求王娟放弃大法，王娟没有理会，他很嚣张，马上将王娟的双手一上一下扳到后背铐起来，又提着手铐突然将双手扳到前面，王娟的双手顿时感觉断了一般疼痛难忍，接着又提起手铐将王娟整个人吊起又放下，反反复复的折腾，王娟在无法承受的痛苦中喊叫，他们不但没有收敛，继续逼着王娟配合体检，另一（协）警看了跑上来猛地对王娟就是一耳光又一耳光的一顿痛打，打了一阵后，把衣服一脱，跳起脚来又使劲打。

王娟质问他：我在家就是做个好人，我手无寸铁，为什么要把我强行关进地下室，为什么要这么野蛮打人？他大言不惭地说“我叫某某（后来知道他故意报的假名），我辞职了，今天是最后一天，我就是变态，就是要打人！”其他人一直逼着王娟说不练了，否则就威胁王娟要坐一辈子的牢。

中共暴徒们逼一阵打一阵，在肉体和精神的双重折磨下，王娟违心地答应他们说不练了，刚刚说完，他们马上变本加厉，又逼着王娟踩师父的法像，王娟猛然惊醒，马上说：我后悔了，我不该说不练了，你们所说的我永远也不会做！“真善忍”太好了，你们这是魔鬼！并给他们劝善，告诉他们大法的美好。而他们不但不听，更加使劲的打王娟的耳光，一只手用力打在王娟的脸上，打的王娟的头撞到墙上，另一只手又猛地打回来，打的王娟晕头转向，打累了，将王娟关进小屋，稍稍停顿，又拉出来打，反反复复从早上一直打到下午，整个人被打的惨不可言！

之后逼着王娟检查身体，体检的整个过程中都在暴打王娟，无比残忍！下午四点，来提审的警察丁俊等人，拿来一张送拘留所的单子，逼着王娟签字，因为上面有“莫须有”的罪名，王娟拒签，丁俊就鼓起眼睛说，不签就判三至五年，之后非法关押到长沙市拘留所，后来又转到望城。

十五天后，王娟回到家中，被当地的户籍监管，后来就出现了本文开头的一幕。

#### 在大法中受益无穷、遭迫害风雨飘零

王娟从小身体本来很好，后来因为职业原因，生活没有规律，无法按时饮食，得了胃病，婚后体质更加虚弱，首次怀孕几个月就胎死腹中，因处理不当，导致子宫粘连，得了妇科病，非常痛苦，各种进补都无济于事，并严重贫血，体重只有七十斤，多站一会就累的不行，头昏无力，还有鼻炎，一年只有几天不感冒，先生工资全用来买药吃了，以后一直没怀上孩子，种种原因导致家里矛盾重重。年纪轻轻的王娟，感觉人生灰暗一片！

后来机缘巧合，在二零零四年，王娟有幸走进了法轮大法的修炼中，刚得大法时，知道“真善忍”，叫人做好人非常好，却不知道还会对身体好，王娟就按照法轮大法“真善忍”的要求做一个善良的好人，没想到身体却奇迹般的好了！从此胃再没痛过，也不感冒了，也没胃炎了，太神奇了！如果不是亲身经历，只听别人说是难以置信的！而今，自己就是实实在在的见证人，用现在的科学是解释不了的！此后王娟每天都感到轻松而美妙，与之前病恹恹的样子判若两人，不久就怀孕了，几个月后就生下了活泼可爱的女儿，王娟一边带孩子，一边努力工作，勤俭持家，二零一四年一家人在省城长沙买了房子和车子，三口之家，幸福祥和。
