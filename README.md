### 诚念九字真言是走出瘟疫的灵丹

——在医院急诊呼吸科讲真相的体悟

 文: 河北法轮功学员 

【明慧网二零二三年一月二十五日】近期，我们这个城市也出现大面积疫情，家家户户都被传染，出现发烧，咳嗽、嗓子疼、严重的停止呼吸失去生命。现在去医院都得去排队，不排号，就住不上医院。
在这疫情中，我妻子也阳了。妻子是多年脑萎缩，不能自理。二零二三年快过年了，妻子出现发烧，她不会说话，体温在38～39度，嗓子有痰，身体虚弱。晚上十点时，我把她送往医院。找了三个大医院，都没有床位。后来就在市一医院急诊科排号。在急诊楼道等候，到夜四点，还没有排到号。后来，那个大夫在楼道往外走，我就拉住他说；能先给看看吗？大夫看了看说；是很严重、得下管子，就是在嗓子下一个粗管子。

到凌晨五点，呼吸科才给了妻子一个床位。呼吸科大夫让我们先去做CT。我和儿子马上把她送去做CT。做完CT，马上把片给了大夫，大夫一看片子说；是很严重的。大夫给我讲，像她这年龄、又是常年卧床的人很难度过这个关。大夫和护士马上给妻子上了呼吸机和监护仪。当时仪器的显示心跳为135、血氧70-80、血压也很低。

当时我就想，我是修炼人，有师父管，我的心很坦然，我就在妻子跟前一直诚心念九字真言“法轮大法好，真善忍好”。妻子的血氧往上升了，血氧87-90，到第二天，血氧92-96，第三天，血氧96-100，心跳70-80。到第四天，就给妻子撤去了呼吸机，血压也正常了，人也有精神了，就是嗓子还有点痰，大夫不敢给去掉嗓子上那个粗管子。

我和大夫说，得去掉嗓子上的管子呀。大夫说，有痰，怕去掉后，有痰堵。大夫怕担责任，写了一份保证书，让我签了字，才敢把嗓子上的管子去掉。去掉后，我观测了一天。妻子都正常，我们就出院了。

我感谢慈悲的师父给了妻子新的生命。

#### 在病房里讲真相　好坏自选啊

1、女儿们相信真相并三退 82岁老太渡过难关

和妻子同一个病房的患者、同一病房一号床的是一个82岁的老太太，也阳了，也是同一天住的医院。她发烧、呼吸困难，做CT，片子显示肺上也是白白的，到病房，也上了呼吸机、监护仪。老太太血氧86-88，也很严重，老太太喘的很厉害。

护理老太太的是她两个女儿。那时我就想一定要给她讲真相，能明白真相、做三退，就能解除白肺病对他母亲生命的威胁，就能救了她妈，我不能错过这个机会。中午吃饭时，我对她两个女儿说：“你们念‘法轮大法好，真善忍好’吧，这样能使老人病情好转。”我接着给他们讲：“这九个字是宇宙的法，有无限的法力，能调动宇宙正的能量净化身体，净化空间场，消灭病毒，你们能认真的念诵他，妈妈的病就能向好的方向转变。”她俩答应了。

我又问她俩：你们入党、团、队吗？大女儿叫小云、入过队；二女儿叫爱云，入过团、队。说着，小云的女儿也来看她姥姥了，我又问她，你都入过什么呀？她说入过少先队，她们都同意做了三退。小云的女儿还说：“我知道法轮大法好，我有一亲戚就是炼这个的。”由于孩子们的心诚，老太太在第三天血氧上升到96，也不那么喘了，不再用呼吸机了。大夫说，再观测两天看看。老太太在医院又住了三天，血氧、心跳、血压都正常，老太太顺利出院了。

2、好坏自选啊

老太太出院后不到一个小时，又来了一个阳性患者，是个68岁的妇女，也是喘的很厉害。大夫看她片子也是那样，马上上了呼吸机和监护仪，血氧70-80。大夫又叫人给她拍背，血氧还是升不上去。

在患者身边的是她的两个儿子。我看到患者那样受罪，我就想自己无论是在什么环境和什么情况，都应该是我讲真相的机会，都不应该错过有缘人。于是，我就拉住老太太大儿子的手到一边给他讲，让他诚心念“法轮大法，真善忍好”这九字真言，就能帮你妈解除痛苦。她大儿子看了看我，没有吱声，看来他是不相信的。

大夫叫她大儿子拉来六个大氧气瓶，一夜里六瓶氧气就都用完了，血氧还是没有上来。早晨，又拉来六个瓶的氧气。到了早八点，大夫看血氧不但不升，还在往下降，70、60、50，到上午十点，血氧降到40，心脏停止了跳动。就这样，这位68岁的阳性患者不幸失去了生命。

同一个病房的三号床患者，是男性，五十多岁，也是阳性。他晚一天住的院，也是高烧、气喘、痰多，血氧在80-90，状态也不好。我和患者——他的妻子讲了四、五次，让她诚心念“法轮大法好，真善忍好”，神佛能保平安。他们家人说，我们不相信那个。没办法，我只好离去。

原来的一号床的老太太和我的家人，因诚念“法轮大法好，真善忍好”很快就都出院了。可三号床患者还是那个状态，没有任何好转。

3、一清洁工：“这时候还送福，真好！”

我给一清洁工人讲真相，我问她是否入过党、团、队？她说，我什么都没入过。我说没入更好，你诚心念“法轮大法好，真善忍好”就能保平安，她答应了。我问，给你个真相护身符吧？她说，“给我了，你咋办呀？”我说，我还有一个。她拿着护身符，高兴的说：“这时候还送福，真好！”她重复说了两遍。

#### 诚念九字真言　走出瘟疫的灵丹

现在医院用的生命监护仪能测出一个人的物质身体好坏强弱来，诚心念“法轮大法好，真善忍好”是正能量，就能加强呼吸机、监护仪的正能量。同样在病房里，几个患者在监护仪测的数值和患者物质身体的变化来看，诚心念“法轮大法好、真善忍好”的患者，做过三退、相信“法轮大法好 真善忍好”的患者，他的物质身体变化就很快、很大，康复就很快。不相信九字真言的人，他的物质身体变化就不大。监测仪没有正能量的加强，数值变化不太大，有的甚至失去生命。

我身边看到的这些人事，我感到真的是在对号。师父教导说：“人只要退出了中共邪党，他就不被中共邪党再管了。那人一旦不被中共邪党管了的时候哪，他紧接着又发生一个变化。什么变化呢？他的身体中就会被清除中共邪党的那些毒素、因素，他就会从新去认识法轮功和被迫害这些问题。这是相应带来的，所以他就会走入未来。但是哪，他如果说“我反对中共邪党、我也不同意法轮功”，没有第三选择”[1]。

这是我在医院急诊呼吸科讲真相救人点滴体悟，只要能讲好真相，世人就能得到救护。
