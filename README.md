### 念大法好　婆婆度过两次病危

文: 辽宁大法弟子 

【明慧网二零二二年六月十四日】我婆婆今年八十四岁，心衰多年了，去年春眼见她手脚浮肿，而且心脏疼的喘上不来气，一天比一天严重，去当地打点滴也不见好转，整夜不能睡觉。
小姑子带着婆婆去了市里大医院，经检查是肺积水和心脏积水，治疗一段时间，医生说对婆婆没有什么救治的方法了，告诉我们，老人能吃啥就给买啥吃吧，意思就是时日不多了。

婆婆回家后，喘气很费劲，躺不下，只能坐着睡觉，吃不下东西。我和丈夫都修炼大法，我们知道很多人诚念“法轮大法好，真善忍好”转危为安的例子。丈夫对婆婆说：妈，你相信大法吗？现在只有大法师父能救你。婆婆说：我相信。平时婆婆很支持我们学大法，丈夫就拿了两幅扑克牌，拿一张让婆婆念一句“法轮大法好，真善忍好”。她一天拿着扑克牌念几遍，家人也跟着念，几天功夫婆婆能吃饭了，也能躺着睡觉了，心脏也不疼了，医院都说治不了了，婆婆念“法轮大法好，真善忍好”就好起来了，全家人亲眼见证了念九字真言的奇效，都非常认同法轮大法。

今年五月初，小姑子带着婆婆去大姑姐家串门，小姑子跟婆婆开玩笑说：妈，我不要你了，你就在姐家住着吧。婆婆说：那可不行，你不要我我就得死了。婆婆和小姑子生活在一起，就她做的饭菜合婆婆的口味，小姑子真有事回家了，准备过几天去接婆婆，婆婆晚上犯病了，心脏疼的喘不上来气，大姑姐吓的赶紧说：妈，你快念“法轮大法好”，婆婆家的人全相信大法，婆婆念着念着缓了过来。小姑子闻信连夜赶去把婆婆接回了家，打了几天吊瓶不见好转又去了市医院住院，吃不進去饭，喝水都吐，针扎進去药顺着针眼往外趟，天天心脏疼得又喊又叫，两分钟也躺不下，医生说该用的药都用了，年纪大了，所以器官都衰竭，没有办法了，你们有个心理准备，把她走时穿的衣服拿来吧！

全家人听了都哭，小姑子说：妈，只有法轮大法能救你，你快念“法轮大法好”，婆婆用微弱的声音说：我念不了了，我不行了，遭不起罪了。我和丈夫也赶来了，丈夫说，妈，你就心里不停的念：法轮大法好，一定能好！我攥着婆婆的手说：妈，你看这一帮儿女，平时好时你就忘记念了，现在有信心念好吗？婆婆哭着点了点头。

第二天回到了我们当地医院，亲朋好友去了好多人，在众人面前就听见婆婆喊：“法轮大法好！”声音很大，谁進屋看她时也不说别的话，对着人就喊：法轮大法好！把人都逗笑了，婆婆象换个人似的特别精神，小姑子高兴的说：我妈也得法了！医生说老太太没事了，回家吧。婆婆念法轮大法好又活了过来。
