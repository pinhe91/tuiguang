### 瘫痪十多年　诚信大法获康复

文: 中国大陆大法弟子 

【明慧网二零二二年十二月十五日】我修炼法轮大法二十多年了，明白了师尊为了正法，为了众生，承受了我们难以想象的苦难，却给了众生最光明、最美好的一切。下面我分享讲真相中的一个感人故事。
在我新居住地的附近，有一位四十七、八岁的妇女带着她二十多岁的儿子收废品。我每次看到她，她都是心情不好，满脸愁容，说话很冲。我观察了好一阵子，没有见到她丈夫。

有一天，我问了邻居老太太：“怎么没看见收废品那个女人的丈夫呢？”老太太说：“你不知道，她丈夫瘫痪十多年了。”听后我心想：怎么能让这一家人明白大法真相，得到救度呢？我平时就把废品存好，存多点就去她那卖。有时我就不要钱，送给她，为了和她多接触，给讲真相打下基础。有一次，我家以前的几个铁床不要了，我和丈夫用车拉到她那去卖。她没给我市场价，扣了我不少钱。我问了一句：“我这么多铁架子，就给这么点钱？”她说：“就这些。”我没有和她计较。我丈夫在后边等着，问：“给多少钱？”我说：“就这些。”他说：“是少的太多了。”我说：“算了，他们母子也挺可怜的，家里还有病人。”我丈夫不修炼，听我这么说，也没有再说什么，我们就回家了。

可是过后几天，我心里总是往上翻这事，我想，这不是利益心吗？我把心放下了，我怎么能被利益心所带动，我还要给她讲真相，救度她呢。

我平时看见她都和她打声招呼。有一天我路过那儿，看她闲着，就走过去对她说：“妹妹，我看你一天也挺不容易的。”她说：“可不是吗！”看她心情挺好，我接着说：“你可别介意，我听说你丈夫瘫痪了十多年了。”她说：“是，你怎么知道的？”我说：“听别人说的。你相信我吗？”她说：“我相信，你平时挺好，每次见我都说话，看着你就觉的挺善良的。”

我说：“那我就告诉你一个好消息。你知道法轮功吗？”她说：“法轮功不是国家不让炼吗？”我说：“不是国家不让炼，是江泽民妒嫉修炼法轮功的人太多，才迫害法轮功的，制造‘天安门自焚’假案。法轮功是最高的佛法，是救人的。是以真、善、忍为准则做好人，做更高尚的人。有多少危重病人、绝症病人、瘫痪多少年的人，通过炼法轮功得救的。”她说：“是吗？”我说：“是呀！从今天开始，让你丈夫和你一起诚心敬念‘法轮大法好，真善忍好’，他的身体就会出现奇迹。”她说：“我记不住。”我说：“你拿笔，我给你写下来。”她说：“太好了，我听了这几个字就觉的心里亮堂。”我说：“那你每天都在心里念几遍。”

过了几天，我又去她那，她老远就喊我过去，见我就说：“你太好了。你告诉我那几个字我天天都念。”这一次，我给她带去了几个大法真相护身符。她看了上面写的内容，说：“这上面写的更好，更详细。”看她很接受，我就给她劝三退（退出中共的党、团、队组织）。她说入过少先队，同意退了。我说：“这个护身符一定要给你丈夫一个，让他也诚心念。”我又问她丈夫入过什么没有？她说他都那样了，能行吗？我说：“一定行。我觉的他也是个有缘人，瘫痪那么多年还活着，就等着大法救度的。”她说：“好。”

当我又去她那时，她简直就象变了一个人，满脸笑容。她拉住我的手，高兴的告诉我她丈夫好了，能下地走了。说她丈夫什么也没有加入过。她丈夫说：“这法轮功太神了！我瘫痪了这么多年，什么药、方法都没治好。这没吃药就好了，这比吃药还灵呀！”

她的一双儿女也明白真相，退出了中共的党、团、队组织。我又送给她一本新年真相台历，告诉他们好好看看真相。她要给我钱，我说这都是大法弟子自己拿钱做的，免费赠送的。大法弟子救人是不为名、不为利的。这都是大法师父教我们做的，都是大法师父在救人。

今年八月份我再去见她时，她问我：“怎么这么长时间没见你呢？”我说疫情隔离，在老家回不来。她说：“我得给你发二百元红包。”我问：“为什么？”她说：“感谢你给我讲真相。”我说：“那可不行。不是我救你，是大法师父救你，我们共同感谢大法师父的救度！而且大法救人是坚决不收钱、物的。这是大法师父在法中要求的。”

她又说：“这大法太好了，你怎么就想救我呢？”我说：“我听了你的事以后，觉的你这个人心还是很善良，没有因家庭负担重而放弃，而是坚强的承担家里的一切，维持着这个家，这也是我们中华民族的传统美德。我就想把这个大法告诉你，让好人都能被大法救度。”我告诉她：“你既然相信大法，那么做买卖一定要把心摆正，不欺骗别人，不伤害别人，一切顺其自然。不能缺斤少两，不妒嫉别人。”

后来她对面又来了一家收废品的，我也给他们讲了真相。由于两家收购废品，有时她收的就少了。我问她：“今天收的少了点？”她说：“你不是说顺其自然吗。我心里一点也不急，这要是过去不得把我急成啥样了。”

她儿子也说：“收太多也很累的。钱生带不来，死带不去的。知道了大法真相，我们心里可高兴了。”我说：“是呀，我们真得从心里感谢大法师父的救度。”她说：“认识你太晚了。早点认识也许不能遭这么大的罪。”我说：“不晚，那时也许缘份不到。”

她还告诉我：“我有时累了，心里就念几遍九字真言，就觉的浑身有劲了。有一天，风刮的可大了，外面铁皮刮的叮当响。这大风刮的废品都收不了，因纸壳、泡沫皮都轻，被大风刮的到处跑。我就坐椅子上，心里念‘法轮大法好，真善忍好’，一会儿风就停了。这是怎么回事？”

我说：“你真相信大法了，当然大法师父就管你了。”她说：“是。”他们这一家四口人都真心相信大法，感谢师父的慈悲救度，我也为这一家人能明真相而高兴。感谢师尊的慈悲苦度。
