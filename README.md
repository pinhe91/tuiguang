### 老妇人哭诉：将来白骨遍地

世人讲述 曌宇整理

【正见网2022年11月09日】

一次我在街上讲真相，一位中年妇女说，我相信法轮大法好，我给你讲个真实的故事。

她说：我娘家是河北省保定市涞水县新庄头村，有老两口，老头叫张旺（老两口都已去世了），他老伴原来是个看相的，有点特异功能。就是谁有病了，看不好，或孩子吓着了，就请她烧香看个究竟。但是后来她就不看香了。

2008年前后那些年，这个老妇人经常在街上哭，边哭边说：人呐，心坏了，坏事干多了，神来救你们来了，你们不听神的话呀，不知死的鬼呀，别看你们拼命的干，将来呀人有大难呐，老天爷要收割这一批人了。我看见将来呀，人死了没人埋，到处是死尸，遍地是白骨，庄稼没人收，有地没人种。有房没人住，有衣没人穿，有饭没人吃，有钱没人花。一个村子一个村子的都空了。一座一座的楼房都空了，怎么办哪？怎么办哪？

老天爷呀，你老人家救救他们吧！

人们都觉她得了疯病了，没人理她，也不信她的话。

好几年呐，她总是在家、或在街上哭，重复着这些话：人呐，心坏了，坏事干多了，神来救你们来了，你们不听神的话呀，不知死的鬼呀，别看你们拼命的干，将来呀人有大难呐，老天爷要收割这一批人了。我看见将来呀，人死了没人埋，死尸遍地，白骨遍地，庄稼没人收，有地没人种。有房没人住，有衣没人穿，有饭没人吃，有钱没人花。一个村子一个村子的都空了。一座一座的楼房都空了，怎么办哪？怎么办哪？

她就这样在街上哭了好几年，前几年她在极度的遗憾、忧伤和无奈中离开了人世。

直到2020年武汉引发的全球大瘟疫出现的大量死人，人们才明白，老妇人的话是真的。但是人们哪里知道，灾难也仅仅是开始。

庄头村边就是涞水县武装部打靶场，涞水县党、政机关、政法委、“610”法西斯办公室、公、检、法、司等机关，在这个打靶场多次办洗脑班，使用几十种酷刑迫害大法弟子。在那些日日夜夜，恶人对拒不转化的大法弟子用电线凝成的鞭子、皮带、沾水的毛巾，木板、警棍、铁棍、板凳腿、鞋底抽打、捆上、拷上，吊起来打，用电棍电击，大法弟子的惨叫声撕心裂肺，久久回荡在空中！一墙之隔的辛庄头老百姓听的真真切切，吓得浑身发抖。可是谁也不敢吱声。

也许那位老妇人所说的“人呐，心坏了，坏事干多了，神来救你们来了，你们不听神的话呀，不知死的鬼呀”，其中就指的是这部分人和事。
