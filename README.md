### 浅谈信神

文: 中国大陆法轮功学员　小慧 

【明慧网二零二二年七月十五日】修炼法轮功之前，谁一提神，我就笑话谁，说人家无知、迷信。其实，就说“迷信”二字吧，你要不着迷，就不信；你能着迷，你就信，就能够做好。其实，这是个很普通的词。现在好了，一说“迷信”二字，把人吓的够呛，了不得了——谁谁讲迷信，你说这人傻不傻？
就说我现在修炼法轮功吧，很多人不理解，说我傻——上面（中共）不让炼，就不炼了，为什么吃这苦？我就告诉他们说：我当时看人家学，我就想学，谁知这一学，就放不下了。为什么呢？因为我一学法轮功，感觉身心巨变，心胸宽广，慈悲、祥和，整天感觉好像生活在幸福的海洋里，庆幸我真有福，能得这么一个好法，能有这么一个好师父；一个人偷偷地哭，哭我遭受人生的苦，这回到头了。

为什么这么说呢？

因为修炼法轮功前，为抢生二胎，让上面（中共）罚了款。结扎时，又留下后遗症，浑身没有不痛的地方，头痛得睡不好觉，吃不好饭，胳膊痛的干不了活，腿痛得下不了炕，坐下起不来。

特别脚背上长了一个花生米大小的筋疙瘩，走路时，只要让小石头压上这根筋，不管在哪里，不管有没有人，双膝盖一齐就跪下了，同时痛得“啊”的一声。特别是秋天打玉米粒季节，真吓坏了我，满场院玉米粒，说不定什么时候，玉米粒就压上这个筋疙瘩了。你们说，我活得苦不苦？

还不止这些，那时候，孩子小，我又有病，公、婆、姐、妹、弟、媳都不靠近我，甚至连话都不跟我说。还有夫妻吵架。你说，我活得累不累？我真的有死的念头，可是，我下不了狠心死，因为孩子太小。每当看到两个孩子放学，在门角一边一个，靠墙快睡了，甚至睡着了，心里真的象刀绞一样痛，可又没办法。

我说这些呢，就是想告诉你们，我修炼了法轮功后，浑身病好了，家庭矛盾没有了，我不记恨他们，善待他们，现在我们都和好了。你们说神不神？！

古书上记载的，像孙思邈、华佗、李时珍、扁鹊等等都是大医学家，都有特异功能。华佗看见曹操脑中有瘤子，想给曹操取瘤，曹操不信，最后曹操得这个病死了。还有太极、河图、洛书、周易、八卦等等。你们说有没有神？反正让我说，千真万确地有神在。

还有的说，神看不见摸不到，谁信？你看不见的东西太多了，收音机的电波你看不见，现在的网络信号也看不见，你能说没有吗？你吸的空气看不见，你能说没有吗？还有白天出太阳，晚上出月亮，一年四季，春、夏、秋、冬都很有规律，你说有神没神？

别的不说，就说炼法轮功的人，中共就这么抓、打、压，可法轮功学员就是不放弃修炼，法轮功不但没被打倒，反而传播到了一百多个国家和地区。你们说神不神！真的有神！

我说这么多，只想让朋友们把心放大一点，大脑清醒一点，用心看看法轮功真相资料，好明白世态的变化。

现在人类道德的败坏，不道德的人会被未来淘汰。法轮大法师父来救度众生，传下天法真、善、忍。用心记住：“法轮大法好，真善忍好。”生命能走入未来，相反，真的没有未来。你们说说与你们有没有关系？

大法师父把大法弟子救了，师父让大法弟子救世人。我们真的感恩大法师父的救度之恩！其实我在大法修炼中的神奇事还很多，在这里，我想让你们自己思考、自己选择。
