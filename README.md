### 长期被迫害　甘肃淳朴妇女曹强强含冤离世

【明慧网二零二二年二月八日】（明慧网通讯员甘肃报道）法轮功学员曹强强，一九五六年出生，家住甘肃省庆阳市西峰市肖金镇，是一位善良的农村妇女。一九九八年，在她四十二岁时，开始修炼法轮大法。自一九九九年七月二十日中共邪党开始打压迫害法轮大法之后，曹强强两次冲破阻力进京，为大法讨公道，先后多次被戒毒所迫害，曾经被迫害生命垂危。在长达二十二年的迫害中，曹强强身心遭受了严重的伤害，于二零二二年一月一日晚十点，含冤离世，享年六十六岁。

![image](https://user-images.githubusercontent.com/79625284/152954529-6fb9f7b6-a6a3-481b-abb9-355febfb6d38.png)

曹强强遗照

#### 两次进京上访　被关押、凌辱

二零零零年四月九日（农历三月初五），曹强强和五名法轮功学员去北京上访。当时由于曹强强的丈夫听信了中共媒体的谎言宣传，在恐惧和压力下，赶到西峰市肖金车站，拦截住曹强强，以离婚相威逼，不许她去北京，最后强行把曹强强拉到本地乡政府。

乡政府执法干事高清亮强令曹强强的丈夫将她带回家严加看管，不许离家外出，强行拘禁在家。两天后，曹强强想办法从家中走脱，坚持踏上了通往北京之路。

在北京上访后，曹强强被北京警察押送到西峰驻京办事处，交给当地处置。五天后，曹强强被遣返回老家，交给了当地肖金派出所。该所副所长范××强迫曹强强坐了一夜冷板凳，第二天，把她劫持到西峰市看守所（戒毒所）非法关押三十天。（注：当时西峰还是县级市，二零零二年六月，西峰撤市成立地级庆阳市，原县级西峰市改为庆阳市西峰区）

在曹强强被非法关押期间，肖金镇政府人员把曹强强等多名法轮功学员强行拉出去，在本地学校、机关等万人批判法轮功大会上示众，然后游街，场景如“文革”再现。游街示众后，曹强强还被强制罚款一千元，关满了三十天，才被放回家。

回家当天，乡政府人员马上上门，逼迫她写“不炼功的保证”，曹强强坚决不写，被当众铐在大门上五个多小时，又被重新押回西峰看守所（戒毒所），加关了十五天。

被非法关押期间，曹强强多次受到吸毒人员的辱骂和欺侮。最后，肖金镇派出所警察强迫曹强强的丈夫出面担保其不炼，才将曹强强放回家。

派出所警察还从曹强强家非法抄走所有大法书籍，并强迫她每天两次到派出所签字报到，还派一不明真相的当地人三天两头到她家监视、盯梢。

二零零零年六月，曹强强被肖金镇派出所绑架到肖金镇幼儿园办的“法培中心”（专门“转化”法轮功学员的洗脑班），强制洗脑迫害七天。

二零零零年十一月，曹强强和其他法轮功学员，再次到北京上访证实大法。期间，被北京警察关押在地下室，辱骂威胁和殴打，铐了三天，搜去了仅有的六百元生活费和其它物品之后，才被非法押送回西峰。西峰警察郑翔恶毒咒骂曹强强，最后，将曹强强在西峰看守所关押了一个月，才放她回家。

#### 两次被非法劳教未成　遭戒毒所关押迫害

二零零一年一月，曹强强被警察张正华、贺××绑架到西峰戒毒所关押。在戒毒所，曹强强和另一名法轮功学员李玉被关在一个大房子里，庆阳的元月是最寒冷的冬天，常常都是零下十几度，警察故意关掉暖气，房间内严寒彻骨，冻得她们晚上睡不着。当时，李玉的腿脚被冻痛得厉害，整夜都不能入睡。在戒毒所被非法关押了七天后，曹强强被非法判了一年劳教。

二零零一年一月十八日，西峰市公安局三个警察把李玉和曹强强转押到兰州市平安台劳教所七大队一中队。曹强强正念否定邪恶的劳教迫害，结果因体检不合格，劳教所拒收。

西峰市警察又将曹强强非法关押到西峰戒毒所七个月，最后，曹强强开始绝食反迫害。四天后，戒毒所怕出人命担责任，就逼迫她的家人交了一千元，放她回家。

二零零二年六月，肖金镇派出所又将曹强强强行绑架到西峰戒毒所，非法关押了十六天后，以伙食费的名义向她家人索要了3160元才放人，平均每天伙食费接近200元钱，而当时西峰戒毒所伙食很差。

二零零三年二月，曹强强发法轮功真相资料时被人举报。警察将她绑架到庆阳市镇原县派出所铐了一夜，不给吃，不给喝，第二天下午两点左右，把她转送关押到镇原县看守所，非法拘留了十五天。

释放回家十二天后，急于立功表现的政府人员又将她欺骗到肖金镇派出所，铐上手铐，劫持到西峰看守所，非法关押了四十多天之后，非法判曹强强二年三个月劳教，这是对曹强强的第二次迫害劳教。

将曹强强送劳教所时，警察不给家里人通知，连件衣服都没拿上。路上，她被折磨得连吐带晕。曹强强思想中再次正念否定劳教迫害，结果劳教所体检又不合格，拒绝收留，警察只好将曹强强又送回西峰看守所。曹强强开始绝食反迫害，十六天后，生命奄奄一息，警察怕出人命，才放她回家。

#### 被非法抄家、监控等迫害

二零零四年四月，曹强强正在家里看大法经文，肖金镇派出所所长张少华带两个恶警气势汹汹地闯进曹强强家，大吵大闹，还到邻居家调查曹强强是否出去发放真相资料，最后没有收获，就抢走了曹强强看的大法经文。自此之后，警察隔三差五地来骚扰、恐吓她，并对她家实施监控，给她和家人造成很大的精神压力与伤害。

二零一一年六月二十六日，庆阳市西峰区国保大队的朱长锁，肖金镇派出所副所长席凯、王治，乡政府干事罗强和司机等人，突然闯入曹强强家，他们没有出示任何证件及手续，便强行乱翻进行抄家，他们把所有房间都翻遍后，还要到租住在她家的饭店服务员房间去翻，因房子门锁着，钥匙在去上班的服务员身上，他们就要强行撬门，家人不让动。僵持之后，他们站在凳子上从窗户向里探查，没发现什么后，便带着从她家抢的两幅法轮功真相挂历走了。临走时，他们还威胁曹强强不要出去联系其他法轮功学员，否则就会抓她。

二零一三年七月十五日下午两点左右，庆阳市西峰区肖金镇派出所三男一女，开了一辆大警车，闯入曹强强家，再次骚扰，他们强行闯入曹强强房间乱翻一通，没有发现他们要找的东西就走了。

二零一五年七月七日，因曹强强发了“诉江状”，高检签收，西峰区肖金派出所八、九个警察到曹强强家骚扰抄家，发现大法书籍、真相资料、光盘就全部抢走。

二零一五年十月，肖金村不法人员又到曹强强家骚扰，并诱惑让她参加西峰办的洗脑班，被她拒绝。

#### 长期迫害导致曹强强半瘫痪

由于遭受长期的持续不断的迫害和骚扰，曹强强的身体受到很大的伤害，时常出现头痛、心困、很累，身体没有劲的症状，有时连饭都做不了，走路也很困难，炼功都很难保证。发展到后来，眼睛也看不清东西了。

到二零一八年五月二日，曹强强突然晕倒，失去知觉，送去医院抢救，在当地医院近一个月时间，一直都处于昏迷状态，又转送西安的医院治疗后，才慢慢恢复了知觉，思想恢复了清醒，但身体处于半瘫痪状态。

回家后，曹强强这种生活不能自理半瘫痪状态一直持续了一年多时间。

二零二零年腊月，曹强强突然又不能吃饭了，家人又把她送到西峰住院抢救，腊月二十八日才回家。回家后，曹强强一直卧床不起，生活不能自理，家人给她请了保姆照顾她的生活。

#### 不堪长期精神、身体的折磨　含冤离世

长期的监禁折磨、骚扰抄家、威胁恐吓、监控监视等各种迫害，给曹强强及其家人造成巨大的伤害，她的身心遭受严重摧残。家庭遭受很大的经济损失，后来家人不堪忍受迫害，开始反对曹强强炼功，丈夫一度以离婚相威胁逼迫她放弃信仰，经常对曹强强辱骂。

后期在曹强强失去生活自理能力时，丈夫、女儿都禁止她学法炼功，不让其他同修去看望，使曹强强处于和外界基本隔离的状态，无法通过炼功、学法来恢复身体健康，只能无助地躺在床上，耗到了生命的最后一刻，撒手人寰。

曹强强走了，这位善良的大法学员，在时光刚刚迈入二零二二年的第一天，就离我们而去。一位淳朴的中国西部农村妇女，仅仅因为坚持对真、善、忍的信仰，二十二年来，遭受了这么多邪恶的迫害，最后含冤离世。

这种惨剧绝非个案，明慧纪念馆数据显示：自一九九九年至今，被迫害致死的法轮功学员能够统计到的、有名有姓的已经有4622例，这都是一个个鲜活的生命啊，他们也都跟曹强强一样，是善良的中国普通民众。

一个什么样的政权、什么样的国度，才会不让人信仰真、善、忍，才会不择手段打压迫害一个只为做好人的善良修炼群体？只有基因中充斥着假、恶、斗的中共邪党才会视真、善、忍为天敌。希望人们唤醒良知，制止这场已经持续了二十三年的迫害，使曹强强这样的惨剧不要再发生。
