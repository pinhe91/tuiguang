### 地震幸存者成为大法真相的活传媒

文: 河北大法弟子　净缘 

【明慧网二零二二年六月十日】一九四九年前，我家在长春农村。爷爷有个粮食加工厂和几亩地，中共篡权后，被定为“地主”兼“资本家”，不多的家产全部被掠夺。为了生活，全家被迫迁往河北农村居住。
因家庭困难，上不起学，生活习惯、说话口音与当地不同，小伙伴们叫我“小侉子”，我心里老生闷气，总也不爱说话，家里大人叫我“二傻子”。我得了一种病“气脖”，结婚后，才知医学上称甲亢，严重到影响心脏，经常休克。去医院做了手术，也不成功，做了左边，右边还没做，就昏死过去，立即抢救，总算活了过来。

一九七六年，唐山大地震，由于邪党不顾百姓死活，尽管有的专业人员给上面打了报告，上面却置之不理，结果一场超级大地震造成数十万民众死亡，伤者无数。我丈夫在地震中去世，而我的腰部、左边骨盆、胳膊、脖子、等多处骨折，右眼失明。

#### 用自身的变化证实大法的美好

唐山大地震几乎将我变成一个半残疾人。一九九七年，我为祛病走入大法修炼。得法后，各种疾病都消失了，如今七十八岁的我，骑自行车象有人推我一样轻巧，大家都说我像六十岁左右的人。现在我每天凌晨三点就自然醒来，参加全国统一时间炼功，上午出去讲真相救人，中午回家做真相资料，下午和晚上背法到九点，发完正念，休息。每天有十七、八个小时心都在三件事上，心里美滋滋的什么也不想，自己把自己当作“专修”弟子，所以只要环境一静或睡醒觉，脑子自然就背法或发正念，每天忙忙碌碌过得充实愉快，做三件事已经是我每天生活的主要内容。

外出讲真相，基本上见人就能讲，用我自身身体的变化讲真相很容易，一说我的岁数，对方就羡慕的不得了，接着顺势讲我炼法轮功如何受益，二十四年来身体无病，没吃过药没打过针等等；再让对方看脖子上的疤痕，告诉他是因自身的遭遇，总生气得的病，气能伤人、能杀人，只要按真、善、忍修，永远不会生气等，自然就讲出邪党怎么害了我、大法如何救了我等真相。大家都认同，二十多年来，走到哪儿讲到哪儿，从没被恶意举报过，也没遇到骂我打我的。当然有不信的、不听我讲的。

每天出去穿着整齐，大方合体，按照师父讲的：“你平时总是保持一颗慈悲的心，一个祥和的心态，遇到问题就会做好，因为它有缓冲余地。”[1]以这种心态去讲真相一点怕心也没有了，总是面带微笑轻声细语，象见到熟人一样亲切，自然大方的从日常生活中切入话题，常常能感觉对方与我的场是融合在一起的，这时效果都会比较好。

#### 三退后的世人

给一位在一大超市遇到的女士做了“三退”后，隔段时间又碰到她。她说：“大姨，我可找到你了！”我说：“咱俩真有缘，有事吗？”她说：“我越想你说的话越有道理，你给我儿子也退了吧，他是飞行员，入过党，叫某某（真名）。”

我和同修从超市出来，我走在前面，同修在后面给一男士讲真相，男士说：“我早就退了，就是前面那位大姐（指我）给我退的，讲得可好了，我还认得她呢。”

今年一月底，超市里有位年纪大的男士在挑苹果，我走过去，边挑边讲真相。过一会，一个年轻女子也来挑苹果，她挑完后，把苹果给了我，说了一句：“这几个挺好。”转身就走了。我想这是有缘人，给男士讲完后，就去追她。追上后，她说：“大姨！您不认识我了？您给我三退时起的名字叫‘新宇’，要不我咋给您挑苹果呢？我知道您在给那个老头讲三退的事呢，没心思挑苹果。他退了没？”“退了！”我俩都笑了。

我每次讲法轮功是佛家大法，是修真、善、忍的，做到这三个字，你就不会生气，没有怨气，不会不高兴，气伤人、气杀人，病都是从气上得的。讲到这儿，几乎百分之百的人都认同，这时对方会主动与我互动，有要电话的、要地址的等等。

每次给对方讲完真相后，我都会告诉他们：“记住‘法轮大法好，真善忍好’遇到任何困难和大灾大难，只要你念，就会遇难呈祥！”对方都会说：“谢谢大姨！”

注：
[1] 李洪志师父著作：《转法轮》
