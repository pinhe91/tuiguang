### 九十岁老人得福报

文: 大陆大法弟子 

【明慧网二零二二年五月二十八日】昨天，我去看望以前的老邻居阿叔阿婶，阿叔今年九十四岁了，阿婶今年九十岁，老俩口耳不聋、眼不花，身体硬朗，儿孙满堂，福寿双全。
阿婶看见我就高兴的说：“你又看我来啦！你告诉的九个字我天天念。”说着就双手合十又虔诚的念诵：“法轮大法好，真善忍好。”

老人家告诉我说：“前几年我摔了一跤把胳膊摔坏了，手抬不起来，梳头都够不着，我天天念九字真言，没几天胳膊就好了，就能够着梳头了，现在啥事没有了。”

阿婶又说：“不光我自己念，我还告诉咱们邻居阿杰也念，他上山拉烧柴就对着山林大声念，别人也去捡烧柴就问他：你喊啥呢？他说我唱歌哪！阿杰回来看见我就说，大娘啊！我念九字真言，拉柴火就一点也不累，太好了。”

阿婶还说：“你大舅（就是阿婶的亲弟弟）从老家来我也告诉他念，他也信，他怕记不住都写在纸上了，我说你回去告诉全家念。多传多念是做好事啊！我还告诉别人：你不信也别拔犟眼子（方言：别抬杠），千万不能让人家不信！这对你可不好，你不信还不让别人信吗？那可不好！”

我感动的对阿婶说：“阿婶您太了不起了！大法给了您福报，您的未来一定太美好啦！”

阿婶还给我讲了一个故事，她说：“这是真事儿，都有二十年了吧，我们那时在四方台住的时候，有一个村子着火了，村里有个人背着他妈妈就往外跑，屋里还有两个小孩，他妈妈就捶他儿子说你把我放下，我都这么大岁数了，死就死呗，快救孩子啊！他也不吱声就把他妈妈背到安全地带，回过头就往村里走，一看全村都烧圆盆了，進不去了，等大火烧完了一看，就他家的两间房没有烧着，两个孩子也好好的。有人问那两个孩子，你们在屋里干啥了？孩子说就看见两只羊在门口，一边一个往外拱火，那火就没進来。”阿婶接着说：“那不就是神吗？孝心惊动天和地呀！神看他有孝心就帮把他家保护起来了！我们事后路过村里，看到别人家都用簸箕簸那烧过的米呀黑黑的，黑也没办法，那也得吃啊不然没吃的了，唯独那一家啥都没损失。”阿婶讲完故事总结的说：“人啊！还得要心眼好啊！不能做坏事啊！”

我把这些写出来是想告诉所有还想参与迫害大法弟子的人员，别再做傻事了，做好事有福报，做坏事有恶报，这是天理呀！现在瘟疫肆虐，来势汹汹，人人自危。两年前从武汉传出的消息，有人走着走着就倒下了；医院死者尸体有时来不及拉走，医生和病人还在那儿走动上班；医院床位不够，病人只能呆在家里；许多一线医生护士被传染；前去武汉一线的二十八岁湖南男医生过劳致死。一件件，一幕幕，一个一个又一个的病患，一个又一个的逝者，触目惊心，令人心酸落泪，悲叹万分。而今上海因为疫情封控，造成的次生灾害不亚于瘟疫死亡人数。有的被饿死，有的跳楼自杀，不当封控引发的乱象丛生。惹得民怨沸腾，使民众逐渐觉醒，认清了中共的邪恶本质，纷纷出来反抗，网民有的唱国歌：起来不愿做奴隶的人们……中共极为恐慌，在网路上极力封杀。它为啥恐慌？它怕民众明白真相后觉醒了不受它的摆布。

中共邪恶政权迫害死那么多修佛修道之人，令天地为之愤怒，老天能不灭它吗？跟随它迫害好人的人能有好下场吗？遭恶报的例子数不胜数。所以奉劝世人远离邪恶，别给恶党做替罪羊当陪葬品，给自己选择一个美好的未来！
