### 告诉租户避疫良方　哈尔滨孙太荣被非法批捕

【明慧网二零二二年三月二十六日】（明慧网通讯员黑龙江报道）哈尔滨香坊区法轮功学员孙太荣，一月二十日被南岗区公安局国保大队吕炳涛等人绑架。二月十六日家属接到办案人吕炳涛的电话让去国保签字，说是孙太荣已被批捕。孙太荣被非法关押到哈尔滨鸭子圈看守所已两月。因疫情看守所不许律师会见。
孙太荣，现年五十二岁，是哈市宾县宁远镇爱国村王油坊屯人。她多年来一直在哈市打零工为生。

![image](https://user-images.githubusercontent.com/79625284/160233904-765cf7b0-038c-46ea-b0c3-cdc3f4a510e9.png)

孙太荣

孙太荣将自家位于南岗区的楼房租给了原籍肇东的王柄皓和李唐的两个年轻人，他俩是同学，在租房中复习考公务员。

孙太荣看到租房的这两个与自己孩子相仿年纪、又挺上进的年轻人就非常喜爱，处处替他们着想。柄皓说考公务员要安装宽带，她就把她丈夫手机的套餐给了这孩子，为他省了840元钱；孩子租房没到期就提出不租了，孙太荣也考虑孩子自己在外不容易，哪里都需要钱，就毫不犹豫地退了按他们希望的房租钱（按中介公司出具的合同那是一分不退的）。

孙太荣看到国内疫情此起彼落的不消停，在去给孩子送退房租款时，就告诉了两个孩子诚心敬念“法轮大法好，真善忍好”九字吉言能提高免疫力，有效躲避瘟疫的良方，以及为什么要“三退”，就是真心想让他们多一个保命、保健康的方法！因为孙太荣亲身实践过了，法轮功提高人体免疫力，祛病健身是真实不虚的，这是科学，不是迷信。

天下做父母的不都是一个心思，希望远在他乡的儿女平安健康吗。这两个孩子当时也没有提出疑议。可是，遗憾的是两个年轻人被中共洗脑灌输的是非不清了，背后把孙太荣炼法轮功的事报告给了南岗区政法委，继而区公安国保跟踪绑架了孙太荣。

即使这样，二零二二年一月二十至二十二日孙太荣在被非法连续羁押了五十三小时的逆境中，在警察软硬兼施逼迫她写“三书”的高压威逼下，她还是告诉被警察利用来劝其放弃信仰的亲人们，不要仇恨绑架她的警察，不要去找那两个可怜孩子的麻烦。

让我们看看孙太荣平日是个怎样的人吧。

#### 一、修大法重获健康

孙太荣二十多岁就浑身是病，月子里受了风，得了风湿病；起“攻心番”（一种地方病）差点死了；胃下面肚子里长了一个拳头大的包，用手一摸梆梆硬，都说不是啥好玩意儿；脑袋也不知道是啥毛病，人说晕倒就晕倒，经常毫无预兆的就昏过去。一次，她端着一盆刚出锅的菜汤，一转身功夫就栽歪过去了，多亏在身边的丈夫手疾眼快接住了盆，才避免再烫伤她。她还有胆囊炎、肝郁气滞等等，还有好多说不上名的病……

家人领着她县里、市里的大医院跑遍了，西药、中药、偏方吃了不知道有多少了，打吊瓶是家常事，针灸、理疗、拔罐、糊药都听大夫的。大窗户台上摆着大大小小、各式各样的瓶子、罐子、盒子，堆的满满的，都是药。成年地熬中药，就是十多年后炉灶上烧开水，房梁还出中药味呢。别人告诉喝蒜辫子水能治受风病，她就赶紧煮了水喝；听说哪里有好大夫，她就赶紧去抓药，一年到头挣的钱都治了病，买了药了，可她的病还不见好转。一米六的个子，体重只有就七、八十斤。

她婆婆总是嫌弃她说：“外边争块板，家里丢扇门”，这日子没法过！年纪轻轻的她不但浑身难受，受着罪，心里更是憋屈苦闷，谁想有病啊？谁不想拿钱买糖不买药啊？！她怨老天对她不公，更怨恨婆家人不理解她，偏心不帮她！那时的她真是到了家不保，命不保的地步了。

然而，天无绝人之路。一九九五年法轮功传到了他们屯，为了好病，孙太荣也跟着去炼了。让她和家人没想到的是，只几天工夫啊，她肚子里拳头大的硬包就没了！一段时间后，她头也不晕了，人也不倒了，脸上也红扑扑的，吃鸡蛋胆囊也不疼了，体重迅速增加到了一百一十多斤。

家里、地里的活都能干了！她家人和全屯的人都见证了她起死回生的变化。这脱胎换骨的变化，只因她炼了不要人一分钱的法轮功！她常发自内心地说：“我这命是大法给的”！

孙太荣炼法轮功还不止一身病都好了，大法还把她整个人都变了，提起老杨家大儿媳妇（孙太荣），四邻八舍的人没有不竖大拇指的。

#### 二、好媳妇人见人夸

早年因婆家重男轻女，孙太荣生了女儿后，婆婆不高兴，也不给看孩子。女儿五岁时跟着一帮大点的孩子到河边玩，一不小心掉河里了，孩子们吓的连喊带叫地都跑了，多亏被河道边种地的人看到救起，她女儿捡了一条小命。可小叔子家的孩子公婆却天天手传手的抱着、扛着；公公家的地，她们花钱买都不给种，小叔子家却白种；婆婆把她结婚时给的彩礼钱“借”回去了，多少年不还，却总拿钱贴补别的孩子。婆家偏心的大事、小事、憋屈事多了……连婆家亲戚、邻居都为她打抱不平。她对婆家积怨压在心里没地方说，大夫也说她一大半的病是从气上来的。

可修炼了大法，她明白了做好人的道理，看淡了钱财，看淡了人与人之间的得失恩怨，也体谅了公婆拉扯儿孙的不易。大法从心底解开了多年她与婆家的怨，她用真心对待公婆和兄弟妯娌们。

他们俩口整年在外给人刮大白打零工，成年一身涂料，一身灰，包的再严实，鼻孔、耳朵、嘴，甚至嗓子里都是粉，连眼毛、眉毛都挂满白灰。起早爬半夜、手挪嘴攒的，挣的那叫个辛苦钱。可每次回屯她都是大包小裹地给公公、婆婆买吃的、穿的、用的。屯子里的人说起孙太荣都挑大拇指，说人家老杨家大媳妇对公婆那是“衣裳是衣裳，钱是钱”，没的说！

#### 三、村里公认一顶一的大好人

农村全家人的吃喝年用全指望那点地。农民都知道，岗地和川地打粮可差多了。一年她家买别人家的一块好川地种，可播种时买的这块地却被另一家先播上种子了，换给她家一块岗地。她丈夫一看明摆着欺负人吗？张罗着去大队查地账，还要把那家播完种子的地垄豁了，他重播。屯里人都知道是她家占理。也有打抱不平的、也有“架火”的，眼看着一场大仗难免，村干部也为难了。

孙太荣拦着丈夫拽着劝：乡里乡亲的，打坏了对谁都不好。老天不会欺负好人的，岗地就岗地吧，能差多少？别计较了。最后在她的劝说下就种了那块岗地。大伙都敬佩地说，好人呢！人家炼法轮功的就是大量，为人处世就是不一样！

一年孙太荣婆家的叔伯弟弟盖房子，占了她家两根拢的地，还不明说。农村人对土地那是寸厘儿不让，更别说两根垄的地了。因为盖房前后延出一点打破脑袋，都不是新鲜事！孙太荣的丈夫觉得这兄弟处事也太欺负人了，这事要被熊住了，他在屯里一辈子都抬不起头了。他可不能吃这哑巴亏，于是蹦高要找人家干仗。

孙太荣一直劝说丈夫：大法给了咱一个完好的家，给了咱好身体，咱干啥活都不犯愁了，这几年功夫咱家盖了大砖房，还添了大儿子，咱一家四口没病没灾、和和气气的还有啥不满足的？别为这事伤了家族的和气，真把谁打坏了就更犯不上了，都是亲戚里道的；大法让咱做好人，那不能光说在嘴上的，得实实在在的做出个样儿——“让他三尺有何妨”？！

她丈夫当时就睁大了眼睛。他不得不佩服，这大法能把他媳妇变成这么大量的人。大伙也都说不服（法轮功处事）还真不行！

有一年，就是占地的这个弟弟家的媳妇得了子宫癌，到城里看病。孙太荣当自己亲弟弟一样，跑前跑后地帮忙，送饭送粥地照顾弟媳；弟弟家钱不足，她还给凑了一万块钱先垫上手术费，感动的弟弟、亲戚直掉泪。屯里人听说了，都竖着大拇指说她是一顶一的好人啊！

二零一零年夏天，孙太荣在哈尔滨会展中心干活时，从梯子上掉下来，直接摔晕了过去。包工老板和其他人赶紧把她抬车上，往省医院拉。她醒后说：我没事，不用去医院了，检查得花多少钱啊。包工老板不放心，怕算后账，还是要去医院检查。孙太荣说，你放心啦，我是炼法轮功的。一不会讹你，二有师父保护，我不会有事（后来依着老板去检查了，真是哪也没坏）。包工老板感激地说：我算真见识炼法轮功的了！换别人没事也得讹俩钱花，这要真出了事，我这趟活就都白包了！

十好几年，孙太荣和丈夫给人刮大白、刷房子，不论雇主在不在现场，她们都凭良心做，不偷工减料，不糊弄。她的活都是上家给介绍下家，活源不断。

孙太荣俩口起早贪黑、凭着勤劳的双手，硬是在城里置办起了一个安稳的家，还给儿子也买了房子。她总是感叹说，要不是学了大法，能不能活到这时候都不好说了，更别说买房了。这家的一切都是大法给的。

她无限感激法轮大法带给自己及全家的福德，更想让她身边接触到的善良人们都得到大法的福佑。尤其是面对全国此起彼伏疫情频现的当下。

就是这样一个勤劳本份、知恩感念的好人，这样一个村里屯外人见人夸的好人，却在大年前无辜被抓了！

真是让老百姓想不明白啊！法律理应是保护好人的呀！被警察抓的应该是对他人人身和生命财产造成危害的坏人啊！可是，现在谁是被孙太荣伤害的人呢？！

国保警察、政法官员，你们都是执法之人，你们比老百姓更知道国家的根本大法《宪法》35、36条明确规定了公民有信仰的自由。《宪法》保护公民拥有信仰书籍、资料不受侵害。你们就照实说，像孙太荣这样讲良心的农村人，能妨碍着国家哪条法律实施不了呢？！

请你们依法行事，设身处地给为老百姓摆摆理，凭着良心为百姓说句公道话，帮助孙太荣早日回家吧。家里老人、孩子过日子都指望她呢。
