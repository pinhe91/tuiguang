### 犯人诚念“法轮大法好”身体神奇康复了

文: 内蒙古大法弟子 

【明慧网二零二二年七月二十三日】二零一四年我被非法关押在邪恶的监狱里，一天，有个犯人被收监回来，她是某某部门的一个科长，因为经济案被判刑七年。为了保护她个人的安全，我给她起个化名叫小蕾。小蕾因为身体不好，病太严重，被保外就医，监狱的狱医都以为她死了，没想到她现在还活着，现在又收监回来了。
因为我拒绝做奴工，长期被关在监舍里，有包夹看管。每天出工以后，他们都把她送到我呆的监舍里边来。我当时住在三层楼、她住的监舍在一层楼。每天出工以后，狱警都把一楼二楼的老弱病残集中到三层楼来，分到每个监舍。小蕾每次从一楼走到三楼，都得歇好几歇。我看见她脸色苍白，嘴唇发紫，脸上流着汗，气喘吁吁的進门以后，急忙把窗户打开，脸面向窗外一口一口的喘气。

看她样子极其痛苦，我走到她的身边问她怎么了。她就跟我说她的身体情况，她说她有好几种病：气管炎、支气管扩张、心脏病、高血压、长期吃降压药。她说她从小就有大肠干燥的毛病，十天半个月才解一次大便，她说她在外边时经常洗肠。小蕾说她为了治好病，走遍了全国各地有名的大医院。经专家确诊：她的支气管扩张都已经变形了，根本就无法医治，专家说是死不了的癌症。小蕾说她家中有好几种仪器，经常输氧，打针、吃药、输液、长期吃激素来控制。

在夜间，小蕾根本就无法躺下睡觉，躺下就憋得上不来气，只能坐着，把被子、褥子、叠成方块摞在一起，再把枕头放上，垫得高高的只坐着睡一会，有时不大一会就被憋醒，只能到厕所里面去咳嗽。

有一天，小蕾昏了过去，被抢救过来，她流着泪对我说：“大姐这次我熬不出去了，我也活不出去了，我还有十个月的期限。”我说我有一个办法能救你，不知道你能不能相信。她说大姐你有什么办法，你快说吧！

我开始给她讲法轮功的真相，大法的美好，揭露邪党的杀人历史，我说把你的党、团、队退了，诚心敬念“法轮大法好，真善忍好”就能逢凶化吉、遇难呈祥，就能出现奇迹。她问：能管用吗？我说你还有十个月的期限，是真是假你自己真心念一念，不就知道了吗？

她很高兴的同意把党、团、队全部都退了。她说大姐我相信你，那我就真心的念一念吧！自从那以后，她的身体状况一天比一天好，她很高兴，我说你不但诚心念，还得按真善忍去做。我对小蕾说，我看见监狱很多犯人在吃饭的时候，把馒头皮全部剥下来倒掉，我看着太可惜了，我跟她们说那样浪费粮食对你不好，她们也不听，我说你可不能那样做呀？她说我以后知道怎么做了。

又过了两三个月，有一天她激动的握着我的双手对我说：“大姐我太谢谢你了，我的病好了！”我说你别谢我、你谢谢大法师父吧！是师父救了你。她急忙双手合十，说谢谢师父！谢谢师父！

她说：“我太幸运了，这次监狱我没有白進来呀！全国有名的大医院，都没有治好我的病，诚心敬念‘法轮大法好，真善忍好’就能把我的病治好，我觉真的是不可思议，太神奇了！我的丈夫和儿子来接见我，很惊讶的说发现我的身体状态越来越好，我笑着对他们说：‘这是秘密，现在不能说，回家再告诉你们。’”

她对我说：大姐你看看我以前什么样，现在什么样。以前吃激素把脸都吃变形了，嘴唇发紫，现在我的脸已经恢复正常了，嘴唇也变红了，从小就有大肠干燥毛病也消失了，身体一切恢复正常，什么活都能干了。我回家的第一件事，找你们炼法轮功的人，我想要看一看你们的书。
