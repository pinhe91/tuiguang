### 传播大法真相　河北衡水赵长余又被枉判六年

【明慧网二零二二年九月十日】（明慧网通讯员河北报道）河北省衡水市法轮功学员赵长余先生给人真相时，被警察绑架；二零二二年四月中旬被衡水市冀州区（原冀州市）法院非法判刑三年半（监外执行）。六月中旬，衡水市冀州区法院人员跑到赵长余家中非法开庭，将三年半非法刑期又改判为六年。
赵长余（赵长愈）先生，64岁，衡水市枣强县枣强镇康马村人，一九九八年开始修炼法轮大法，按真、善、忍标准做好人。修炼几个月后，困扰他多年的关节炎、胃病、神经衰弱等病症不药而愈，一家幸福和睦。好多人从他身上见证了大法的美好，纷纷走入修炼大法的行列。在中共邪党对法轮功的疯狂迫害中，赵长余多次被中共邪党迫害。

#### 被枉判五年，在冀东监狱遭迫害命危

一九九九年七月二十日，中共邪党开始疯狂迫害法轮功，赵长余四次被当地关押迫害，累计一百八十多天，被勒索现金数千元。二零零二年四月，赵长余被中共邪党法院非法判刑五年，关押在冀东监狱迫害。
冀东监狱位于河北省唐山市丰南区的海滩盐场，方圆约百里内分布九个支队，总队设在岸边的尖坨子。一、二、五支队离总队较近，一二十里；四支队最远，位于离尖坨子七十里的南堡，离渤海只有十来里。道路和各支队的监舍都是斜向的，就是晴天也很难辨别方向。监狱采用关禁闭、毒打、熬鹰（连续几天几夜不让睡觉）、戴手铐、蹚脚镣、老虎凳、死人床等酷刑折磨法轮功学员。

二零零三年八月二十三日，冀东监狱将所有被非法关押的法轮功学员（除了绝食和住院的）都转移到离陆地最远的第四支队集中“转化”迫害。赵长余在第二支队拒绝“转化”，腿被警察折磨得走路困难，被转到四支队迫害一个多月后，人已脱相、骨瘦如柴、上厕所都得要两人架着慢慢挪，生命垂危。

#### 传播真相，再被枉判六年

赵长余在给人讲真相时，遭不明真相的人构陷，被警察绑架到恩察派出所非法关押几天后，被家属接回。不长时间枣强县公安局枣强镇派出所联合恩察派出所警察闯到家中骚扰、恐吓。赵长余身心受到严重创伤，双腿不会走路去石家庄医院治疗。期间，警察还追到医院继续骚扰、恐吓。

二零二二年四月中旬，赵长余被衡水市冀州区法院非法视频开庭，被非法判刑三年半，因身体有病，监外执行。

六月中旬，冀州区法院人员又跑到赵长余家中非法开庭，将三年半非法刑期又改判为六年。法院人员还说：检查身体没事就抓走。之后，以“去医院做核酸”为名将赵长余绑架。
