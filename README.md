### 医生奇怪：囊肿怎么就不见了？

文: 海外大法弟子

【明慧网二零二二年九月二十七日】父亲年轻的时候，正好赶上“文化大革命”，受无神论的教育，父亲一直以来不相信有神的存在。他经常说的话就是：人死如灯灭，哪有什么鬼啊神啊的。谁一说神啊，佛啊，他就笑这个人愚昧。
头几年，我跟他说大法的神奇和修炼的事儿，他根本就不相信。妈妈因为念诵“法轮大法好，真善忍好”九字真言，扔掉了药罐子，而且越老越健康了，他也无动于衷。

当年劝他三退的时候，他也痛快地答应了，但是就是不相信大法的事儿。虽然去年因为一些事，跟他说了九字真言的神奇，他勉强答应了念诵九字真言，但是还是念诵几天，就不念了。
二零二二年九月的一天，父亲因为前列腺疾病，排不出尿来，在当地小医院打点滴，数日不见好转。不得不插排尿管，才能排出尿来，而且排出来的都是血水血块。他非常痛苦。一向心疼钱的他，自己喊着要去大医院了。

当我知道这件事的时候，我就打电话告诉他，赶紧去大医院，该用什么药就用什么药，不要心疼钱。同时叮嘱他一定要诚心敬念九字真言“法轮大法好，真善忍好”。他很不情愿的“嗯”了一声，我知道他很反感，只是出于对我的爱，没有反驳我。

我就给他讲了个癌症病人因为诚心念诵九字真言“法轮大法好，真善忍好”，上手术台前检查身体，癌细胞竟然消失了的福报故事。他听了后，似乎有所触动，声音比之前轻快了许多。

在他入院期间，我每天给他打国际长途，鼓励他念诵九字真言。他都答应好好念。

父亲在医院做了两天的检查后，医生说，前列腺有个囊肿，需要马上手术。于是，准备第二天手术了，手术的相关物品也付了钱，买好了。

就在他准备手术的前一天晚上，弟弟告诉我，核磁共振结果显示，父亲的癌细胞值很高，需要再做个加强核磁共振，确认一下到底是不是癌症。

当时，我听了之后，脑袋“嗡”一下，眼泪就下来了。但是，随即我稳住了自己，否定这一切，我告诉自己，父亲不会有事的，师父都在管。父亲已经在念诵九字真言了，就看他自己的诚心了。他有他的命，不是我能决定的。于是，我发正念否定这一切。

第二天中午，我再次打电话，弟弟说：一切正常，而且手术也不需要立即做了，再过两天，父亲就可以回家了，自行消炎，然后再决定要不要手术。听到这个消息，我流泪了，我知道是师父救了父亲。

加强核磁共振做完后两天，父亲就出院了。

回到家，我也叮咛他一定要念诵九字真言，他说：“你放心吧，我每天都念的。”但是，我还是不放心，担心他和上次一样，中途就放弃了，于是就叮嘱妈妈提醒父亲。妈妈说，这次你不用担心他了，这次他真的信了。

原来，就在父亲做加强核磁共振的时候，医生非常疑惑地说：奇怪了，你这里明明有个囊肿，今天怎么就不见了呢？你这消炎消得也太好了，昨天囊肿还挺大，今天就没有，真是奇怪了！

父亲回家后，越想这事儿越神奇。因为当父亲转到大医院的时候，医生说：“你来的太晚了，你再稍稍晚一点来，命都保不住了。”

住院的几天里，和父亲同一个病房里几个患者都是同样的病，比他病情轻的人都做手术了。唯有他最重，却是最早一个出院的，而且没有做手术。

经历了这件事，父亲彻底相信了：大法是救人的。父亲从此彻底改变了无神论的观念，每天念诵九字真言数百遍，身体越来越好。
