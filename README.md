### 丈夫明真相后　五癌消失　健康生活

文: 中国大陆大法弟子　晶莲 

【明慧网二零二一年九月二十二日】我是一九九六年底得法修炼的，二十多年来，沐浴在佛恩浩荡之中，身心得到了净化，道德提升，身体的多种疾病都不翼而飞，身体健康，家庭和睦，儿孙满堂。是我修炼了法轮功，得到了福报，师父给了我一个幸福美满的家庭。家人们看到我身心的变化，都知道大法好，修炼法轮功的人都是好人，明白真相后，我们这个大家族有几十人做了三退，都为自己选择了一个美好的未来。同时有的开始学法炼功。下面我讲几个我一人炼功亲人们身心受益的真实的故事。

#### 我丈夫原身患五种癌症　现健康的活着

我丈夫今年六十七岁，教育系统退休。在二零一四年五月二十八日，在体检时，确诊为肺癌，当时我们全家人都不敢相信这是真的。当年六月二日，丈夫到北京三零一医院就诊，经过各种检查，确诊为小细胞肺癌。医生告诉我说，这种类型的癌症发展非常快，生命很短，一年到一年半，让我有个思想准备。听到这个消息，我当时感到好像天塌下来了，背着丈夫偷偷流眼泪。在北京三零一医院做了一个疗程的化疗，就回家了。

回家后，丈夫休息了二十一天（一个化疗后，要休息三周），紧接着开始往返省医院化疗，期间医生建议，由于小细胞肺癌可以直接转移到头部，医生说在头部做个加强维护治疗，再做一个月的头部放疗，过后看这是多余，当时只能听医生的。在一年的治疗中，该去的医院都走到了，治疗的方法能用的，都用到了，比如，放疗、化疗、冷冻、介入疗法、生物疗法、服用中西药、偏方、营养品等，都没能把癌细胞控制住，一年内，癌细胞爆发式的转移，转移左肾、颈淋巴、头部、还有腮腺癌（是二零一九年得的），成为一个身患五种癌症的人。

二零一五年上半年，我丈夫的身体消瘦、虚弱、浑身没力气，脸的气色灰白色，出门不敢见人，戴帽子把脸遮住。体重由原来的一百八十多斤，减了三十多斤。丈夫跟我说：“我可能要不行了，连洗脸的力气都没有了，”六个化疗疗程没能坚持完，身体就支持不住了，心脏病也犯了，就放弃了化疗。

我丈夫由于受邪党文化无神论毒害很深，根本不接受任何信仰和所有的气功。当我开始修炼法轮功时，他极力反对。当我遭受迫害时，给他打击很大，有对师父和大法不敬的言行，有时还打骂我，用尽了他所能用的办法来阻挡我修炼大法。还有一次，他把我娘家十几口人请到饭店吃饭，让我的娘家人来劝我放弃法轮功。我多次给他讲真相都不听，他瞪大眼睛，凶巴巴大喊大叫，根本就不听。

丈夫的病情就像医生说的，发展的非常快，越来越严重，眼看着一天不如一天，他自己也失去了生存的信心，脾气暴躁，自己认为生命就要到尽头了，钱也没用了，吃喝玩乐，享受一天是一天，破罐子破摔。

有一天，我看他情绪好一些，我就告诉他，你的病只有大法师父能救你，他用乞求的眼神看着我，静静的听我给他讲大法被迫害的真相，然后我告诉他三退保平安，把你入过的党团队声明退出来，神佛保佑你平安，他听明白了，问我说：“那得怎么退呀？”我就告诉他，神佛慈悲于人，大法师父来世间是救度众生的，只要你同意表个态，我就帮你退了，不用你真名，用小名、化名、笔名都可以，神佛看人心，他听到这里，把右手举过头，嘴里喊着我同意！我同意退、退！

当时我很激动，眼含热泪，我又问他一遍，你是真心的同意退吗？他又把右手举过头，说一遍我同意退，我说，那我就用你的小名给你退出党团队组织吧。他说行。这时我激动的心情无法控制，也无法表达，激动的眼泪夺眶而出，我喊他的名字，某某某，你有救了！你有救了！

从此以后，七年多来，我丈夫随身带的钱包里有真相护身符，走哪带哪，病情一天比一天好，现在的体重已经二百零四斤了，红光满面，白里透红，谁看见他，都说你命真大，你家哪辈子积大德了，我妯娌都说，是大嫂修炼法轮功，大哥才有今天。

丈夫暂时还没能走入修炼，但是他在默默的支持我，在邪党搞所谓的“清零”行动时，社区给丈夫打电话，让他到社区去免费检查身体，丈夫问我，你说我去不去？我就告诉他不能去，咱们不图那个便宜，也许是为我炼法轮功，要强制签字的事。丈夫反问我一句，那你为啥不签呢？我说，你知道签的是什么字吗？丈夫说：“不知道。”我说就是不让我修炼法轮功的“三书”，你说我能签吗？你也看到了，我修炼法轮功二十多年了，我无论在社会上或在家庭中，为人处世都按照真、善、忍的标准做好人，凡是了解我的人，都说我是个大好人，难得的好媳妇。而且我身体健康，二十多年不吃药，不打针，没让你操过心，给咱家省了多少钱？！还有你得了大病，还健健康康的活着，这不是奇迹吗！ 这不是法轮大法给我们带来的福份吗？我怎么能背叛师父，背叛大法呢？后来社区再来骚扰电话，丈夫就不接了或关机。签字一事就不了了之了。

丈夫默默支持我、保护我，不跟邪党站队，得到了福报，是师父给了丈夫第二次生命，我们全家感恩师父的大恩大德！感谢师父！感谢大法！

#### 妹妹学法炼功　抑郁症好了

十年前，妹妹身体一直不好，不爱吃饭，有时整夜不睡觉，浑身哪都不舒服，到几个医院看病，也没看出来是啥病。医生说：“可能是更年期的反应。”就开始按照更年期吃药治疗。可是病情越来越严重，都不想活了，总想跳楼死了，一了百了。

妹妹的朋友看到妹妹的病情很严重，医院还看不出是什么病，就介绍给妹妹看“大仙”，妹妹看好病心切，听说哪里“大仙”，花多少钱都不在乎，看了几年的“大仙”，不但没好，还招来了各种动物附体，严重时住了几次院，还住進精神病院。后来经医院大夫确诊是抑郁症。

我多次跟妹妹讲修炼法轮大法的美好，祛病健身有奇效，妹妹看到我身心的变化，也知道大法好，由于看到炼法轮功的人遭到迫害，所以她很害怕，不敢学炼法轮功。我就告诉妹妹只有学炼法轮功，才能使你身体得到健康，你将来会亲自来找我学法炼功的。

三年前一天，妹妹哭丧着脸来到我家，当我看到她时，把我吓一跳，整个人变的精神恍惚，脸的气色青灰色，说话没有底气。妹妹见到我后，眼睛含着眼泪，大姐，我的病哪都看到了，可还越来越重，我生活的勇气都没有了，我今天是来跟你学法炼功的。我听到这，悬着的心一下子就轻松了。我说，好啊！你这才是选择了正路。

从这天起，妹妹开始学法炼功，因为妹妹身体带的东西很乱，干扰很大，我去她家，她很不高兴，对我的态度很不友好，好像害怕的感觉，还恶狠狠的说：“你不要再上我家来了。”我当时感到一惊，妹妹是怎么了，从来对我没有这个态度。后来我明白了，是她身体带的附体起的作用。

经妹妹同意，我把她家里都彻底清理一遍，该送走的送走，该烧的烧，该扔的扔，妹妹看到此景，问我说：“你处理这些东西，你不怕招惹麻烦吗？”我告诉妹妹，我什么本事都没有，表面是我在做，其实都是大法师父做的，大法师父无所不能。

后来，妹妹就能静心学法炼功了，还把所有的药都扔掉了。不到一个月，妹妹能吃能睡，脸的气色变的白里透红，白白胖胖的，一米六零的个子，体重现在都达到一百一十多斤了，抑郁症神奇般的好了！

身体的病都好了，妹妹每天都生活在幸福之中，是大法的神奇改变了妹妹命运。全家人都见证了师父的伟大！

在二零二一年八月十二日，妹妹骑电瓶车，追赶前面的朋友，车速快一些，根本就没看见前边停一辆四轮车，不知不觉的“咣当”一声撞上了，妹妹连人带车撞出去十几米远，可是身体安然无恙，哪都没伤着，只是电瓶车的后视镜摔坏了。她从地上爬起来后，骑着电瓶车就走了。妹妹感慨的说，是师父救了我的命，谢谢师父的救命之恩！

#### 大姑姐：你再拉、尿床上，我也不打你、骂你了

妹妹的大姑姐，是个下岗（失业）职工，已退休，今年六十二岁，家里还有个脑血栓后遗症卧床多年的丈夫。大姑姐多年来身体一直不好，腰腿疼、后背疼、胃疼、不爱吃饭、失眠症，吃药打针都不见好，1.72米的个子，瘦得皮包骨，体重不到一百斤，还得照顾卧床的丈夫，对这个收入不高的家庭来说，真是雪上加霜。

妹妹看到她大姑姐常年有病不见好，想用自己学炼法轮功身心受益的亲身经历，和妹夫商量让姐姐也学炼法轮功吧，妹夫当然知道大法好，祛病健身有奇效。

二零二一年三月末的一天，我妹夫来我家说：“我姐姐身体很不好，你也教我姐姐学法炼功吧。”我说你姐姐同意吗？我俩都跟我姐姐说了，她同意学炼法轮功。我说那好啊！那咱们现在就去。

我们俩来到他姐姐家，说明来意，她姐姐很高兴，愿意学法炼法轮功。我首先给他们（他姐夫脑血栓后遗症，不能说话，但能听明白）讲了法轮功被迫害的真相，讲了为什么要“三退，”两人都听明白了，退出了邪党的团队组织。然后，我又给他们看MP4里的短视频，《骨癌患者祈求的奇迹出现了》、《瘟疫中的千金良方》，给她增加了学炼法轮功的信心，也看到了大法的超常与神奇，然后我跟他姐姐看MP4里的师尊教功录像，边看边学，学的很用心。

第二天，我给她买一个播放机，里面装上师尊的教功录像、炼功音乐、师父在广州讲法。她姐姐用了三天的时间，就听完了师父的广州讲法。我去她家，她乐乐呵呵告诉我说：“师父的讲法，我听明白了，原来法轮功是高德大法呀！师父说，修炼人打不还手，骂不还口，得忍，打人失德，把德给了对方，那我得给他（指丈夫）多少德啊！”

原来她丈夫得脑血栓后遗症后，生活不能自理，大小便经常弄到床上、地上、衣服、裤子、还有时甩到墙上，她边给丈夫收拾粪便，边生气，骂他、打他。她听了师父讲法后，明白了做人的道理。她来到丈夫面前拍拍丈夫肩膀说：“我今后再也不打你、骂你了，我错了。丈夫听明白了媳妇说的话，抿着嘴开心的笑了。

妹妹的大姑姐在不知不觉中身体的病都好了，她说这个功法太神奇了！
