### 不可思议的生命奇迹

文: 中国辽宁大法弟子　夏莲 

【明慧网二零二二年一月一日】我今年六十四岁。一九九七年四月走進大法修炼。拜读《转法轮》后，按“真、善、忍”归正自己，师父就为我净化了身体。炼功不长时间，原来的胆囊炎、胃溃疡、神经衰弱、慢性阑尾炎、关节炎、心律不齐、腰椎间盘突出、糖尿病、乳腺炎等等疾病就都好了，无病一身轻，真快乐！
而且，修炼大法后，我的心性提高很快，在家中，是孝敬公婆的好儿媳，也是关心丈夫的贤妻良母，整个家庭和谐美满；在单位，善待同事，帮人排忧解难，当时身为办公室主任，从未用职权之便办过私事，工作中身体力行，尽职尽责，得到朋友、同事的称赞。

一九九九年七月二十日，中共开始迫害法轮功后，我因为坚持信仰，分别在二零零五年和二零一六年两次遭非法判刑七年和四年半，在被非法关押期间惨遭酷刑，身体损伤严重。然而，就这样一个面临残疾和死亡的生命，师父给我重塑了身体，出现了不可思议的生命奇迹。

#### 生命衰竭　监狱、看守所不敢留

二零零五年十月二十五日，我被绑架到黑山看守所。在那里，警察给我上背铐，我不从，他就把我右胳膊小臂掰伤。我绝食反迫害，狱医带几个警察和犯人给我野蛮灌食，硬把手指粗的胶皮管子从鼻孔往胃里插，我的鼻子和嗓子被插得鲜血直流，出不来气，又恶心，吐又吐不出来。我的鼻子和嘴里流着血、双手铐在背后、胃里插着管子，从上午十点直到下午三点，这样被残酷迫害长达五个小时。

二零零六年一月十六日，在我几次被迫害致昏迷的情况下，黑山法院人员在看守所对我非法判刑七年。二零零六年二月九日，黑山看守所三个警察把我抬上警车，送入大北监狱。

在要将我关進大北监狱体检时，发现我出现严重肾衰、心衰、酸中毒、极度营养不良四种重病，随时都有生命危险，监狱拒收。

为了把我留在监狱，黑山看守所警察拿出五千元钱给了大北监狱。按规定：地方看守所给监狱送一个人，监狱要给地方看守所八百元，而此时黑山看守所为了把我关進监狱，却不惜重金贿赂。但是，监狱看我的身体状况，怕我死在监狱担责任，就是不肯收。结果，黑山看守所警察又将我拉回黑山看守所。

回到看守所后，警察就让人给我强行输液、灌食。这期间，一个副所长或许好意，跟我说：“把《转法轮》给你拿来，你看书吧！”我说：“这里不是我学法的地方，我要回家学法。”

二零零六年二月十三日早上，在给我灌食、输液时，我吐了两口紫黑色的血。看守所警察看我要死了，就赶紧把输液管拔了，叫家人把我接回了家。

此时，我的身体浑身浮肿，腿肿的走不了路，有尿憋不住，只能爬着去卫生间。同时，又出现了心衰，已说不出话来了。我心里喊：“师父救我！”我醒来时，浑身衣服都被汗水湿透了。师父又一次把我救活了。在这之前，几次出现心衰我都是喊师父救我才活下来的。

一个监狱拒收、医院判了死刑的人，坚持继续学法炼功。仅一个月，身体就完全恢复正常。

#### 警察掰断的胳膊自动回位

二零零七年九月十一日，我被铁岭警察绑架，在铁岭看守所被关押四十多天。二零零七年十月二十三日，警察将我劫持到马三家女子劳教所。

二零零八年十月十日，在马三家劳教所，因几名法轮功学员不在“考核表”上签字，被上吊刑，因我抗议对法轮功学员的这种迫害，狱警给我也上了吊刑（或叫抻刑）。

这种刑是利用上下两层床，从床头把人的上身推進去，下身在床的外面。在床底下横绑着一根角铁。先把人的两条腿绑在一起，再把腿和角铁绑在一起。两只胳膊用手铐铐在手腕上，在床的两侧吊起来，手铐固定在上床的梁上。此时，腰直不起来，腿蹲不下，整个身体的重量都在两只胳膊上。

只几分钟，我就大汗淋漓，全身湿透，手肿成黑紫色，痛不欲生。时间一长，手铐铐進肉里，血顺着手往下滴。我被整整抻吊了两天两夜。当把我放下来时，我的右半身失去知觉，吃不進东西。听警察说，这种刑，如果能挺过两天两宿的人，不被抻死，也得抻残废了。

二十多天后，我的右半身还是没有知觉。我要求检查身体。在监狱医院，我问医生：“我的身体能不能好了？”医生支吾着：“这得看你怎么锻炼了，也许一个月、二个月，也许一年、两年，也许十年、八年。”我说：“你的意思是不是神经坏死了？”他说：“咱这没仪器，不能确诊。”

在我的右半身没有知觉的情况下，二零零八年十二月三十日，因为我不在“考核表”上签字，三个警察强行掰着我失去知觉的右手在“考核表”上按手印，把我右胳膊从肩膀根处掰断，当时胳膊错位，变成往上举了。

监狱医院治不了，转到沈阳医大。医生说，只能做手术，才能接上。我没同意。我说，只要让我炼功，就能好。他们就请了沈阳正骨医院的主任给我保守治疗。因为胳膊已折断三天了，在打牵引时，我痛的受不了，就想：我是大法弟子，怎么能这样呢？就用另一只手把牵引拆了下来。

马三家劳教所警察把我关在四楼的一间仓库里，由一个湖北籍的人照顾我的起居。我倒在床上背大法，背《论语》、背《洪吟》、《洪吟二》等所有会背的法。背着背着，觉的胳膊就象有人拽一样，一下一下的往下拽，但是不疼。

二十天后，我被带去正骨医院复查，胳膊竟然完全复位了！但右半身还是没有知觉。一直到二零零九年三月十日，我被放回家时，生活都不能自理。

回到家后，我继续学法、炼功。仅二十多天，身体就完全恢复了，而且接好的胳膊没有一点硬结。骨折的人都知道，断骨是长不上的，是外面的肉把断裂部位包起来了，所以骨折的部位接好后，都有很明显的硬结。可我的胳膊断裂的部位接好后，是正常的，没有一点硬结。

这样一个面临严重残疾的生命，师父给我重塑了一个健康的身体。

#### 医生说：你的胆就象做过手术一样

二零一六年六月，我再次被绑架，被非法关押在锦州市看守所。夏季体检时，做B超的医生问我：“你的胆做过手术吗？”我说：“没做过，我的胆怎么了？”他说：“你的胆就象做过手术一样，胆壁象被刀割过的一样，既薄又光滑，而且又均匀。”

我告诉他，炼法轮功之前，我得过胆囊炎，炼功以后就好了，但我没去医院检查过。医生听后，连连摇头，自言自语的说：“不可思议，真不可思议……”

这让我想起了二零零五年十月二十五日，那天我被绑架、非法关押在黑山看守所。因为我绝食，警察把我弄到县医院做胃镜。

回来时，两个警察在车上小声说：“尽撒谎，她的胆壁那么薄，还说得过胆囊炎！”因为当时我身体非常虚弱，听到这话，心想，我修炼法轮大法前，确实得过胆囊炎。那时，医生还告诉我，胆壁相当厚了，早晚得做手术，我没撒谎啊！

当时，我就没问车上的两个警察为什么这么说我，而此时，医生说到我的胆壁的状况，我才明白，原来是我修炼法轮大法后，师父早就把我的胆净化成一个健康的、象做过手术一样的胆了。

#### 老太太的各项指标比年轻人还好

二零一六年六月我被绑架后，因为坚持信仰法轮大法，被非法判刑四年半，关押到二零二零年十月二十四日才结束冤狱，那时我已经六十三岁。

在结束冤狱之前，监狱给我检查身体，在场的医生们说：“这老太太的心脏、血压等各项指标比年轻人还好。”我告诉她们，我曾经得过肾衰、心衰、酸中毒，我现在还活着就是个奇迹！

于是，我给她们讲了我于二零零五年十月二十五日被绑架后非法判刑的经历，也就是上文讲到的——生命衰竭，监狱、看守所不敢留。然而，我回家学法、炼功一个月，就完全恢复健康，我告诉她们，也就是现在你们看到的我这样的身体。

她们说：“这真是个奇迹！”

我修炼了法轮大法，找到了生命的归宿。在中共残酷的迫害中，尽管我的身体遭到中共严重的残害，但是，我生命的本质已经被师父改变了。师父重塑了我的身体，也重塑了我的心灵，我不知道用什么语言能表达对师父的感激之恩，叩谢师尊！

师父，做您的弟子真好！
