### 曾陷冤狱11年半　上海68岁罗伟又遭枉判

【明慧网二零二二年八月九日】（明慧网通讯员报道）上海市徐汇区法轮功学员罗伟老先生，二零二一年三月一日在家中被该市普陀区国安警察绑架，非法关押在普陀区看守所。二零二二年八月四日下午两点左右，普陀区法院非法庭审罗伟，非法判他一年半。
罗伟六十八岁，大专学历，原上海石油公司职员，修炼法轮大法后，按真、善、忍标准做好人，身心受益。在中共邪党对法轮功的迫害中，罗伟曾一次被非法劳教、两次被非法判刑，累计陷冤狱十一年半。

以下是罗伟被中共邪党迫害的部份事实：

#### 一、进京和平请愿，被非法劳教一年半

一九九九年七月二十二日，中共邪党开始迫害法轮功。二零零零年二月十五日中午，罗伟、池波夫妇和十六岁的女儿罗丹，和几位法轮功学员到天安门广场，在旗杆西侧展开了法轮大法好的横幅。顷刻，警车呼啸而至，冲下来十多个恶警，抢夺横幅，并对法轮功学员拳打脚踢，罗丹后脑勺挨了一脚……

![image](https://user-images.githubusercontent.com/79625284/183607675-71b806fe-5d55-4363-8992-de4b1053d3f3.png)

中共酷刑示意图：殴打

罗伟一家被劫回到上海后，被非法关押到徐汇看守所迫害一个月。后罗伟被非法劳教一年半。

之后，女儿罗丹因在学校作文中写出了法轮大法是受诬陷的，此事震动当地教育系统。

#### 二、全家被绑架，夫妻被非法判刑

二零零二年一月底，罗伟的妻子池波发放真相资料被绑架，随后罗伟、女儿罗丹也相继被徐家汇警察绑架，罗伟被非法判刑三年，池波也被非法判刑（刑期待查）。十七岁的罗丹被劫持到“上海市法制教育学校”洗脑迫害。

罗伟在被非法劳教的同时被所在单位非法开除了公职。

#### 三、被非法判刑七年，在提篮桥监狱遭迫害

二零零九年三月十日，罗伟被上海市金山区公安分局警察绑架、抄家，非法关押在金山看守所，后被非法判刑七年，被转到上海市提篮桥监狱继续迫害。

提篮桥监狱是迫害法轮功学员的黑窝，采用的迫害手段主要有：

二十四小时“包夹”：三人一间牢房，恶警指使二个犯人对法轮功学员一天二十四小时“包夹”。“不转化”就呆在3.3平米的牢房内，白天坐窄面的小矮凳，挨打、受饿，辱骂更是随时发生。

多根警棍同时电击：多个恶警用警棍同时电击法轮功学员身体的各个部位，如头皮、嘴唇、手心、脚心、背部、胸部无所顾忌，残忍至极。

![image](https://user-images.githubusercontent.com/79625284/183607843-7dcf2a35-b8d4-4725-bf3c-1417a8cfe8a5.png)

酷刑演示：电棍电击

剥夺睡眠：恶警授意犯人轮流看着法轮功学员，不让其睡觉，稍有不顺招来的就是拳打脚踢。

野蛮灌食：针对绝食反迫害的法轮功学员，灌入高温烫粥或非常咸的食物，甚至灌入不明药物，学员常常被灌的透不过气来。

绑死人床：人成大字形，手脚分别被绑在床上，严重的嘴被胶布封上。

![image](https://user-images.githubusercontent.com/79625284/183607974-96581f9e-5c7f-4d29-9b95-bdff7fe1bc6f.png)

酷刑演示：死人床（呈“大”字型绑在抻床上）

奴役：监狱为追求利润，进行大量劳务加工，超时劳动更是家常便饭，恶警强迫法轮功学员干活，强加奴役指标。拒绝奴役则以“抗拒管教”为由施以电警棍电击的私刑。
