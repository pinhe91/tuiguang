### 保护大法弟子　患结肠癌的哥哥得福报

文: 四川大法弟子　青莲 

【明慧网二零二二年五月六日】今年我哥哥六十八岁。二零二一年二月二十二日早上，他突然给我打电话说：他几天都没吃东西，不吃不排，肚子发胀，叫我陪他去看病。我到那一看，赶快叫他敬念“法轮大法好，真善忍好”，他就开始在心里念。
我们到了一家三甲医院，因他以前有疝气没有重视，我几次叫他去医院做手术，他都没去。这次发病严重，三甲医院都无法救治，医生叫我们赶快到四川大学华西医院看急诊。

到了华西医院急诊科，医生开出各种检查单，这些检查单全部检查完后，已是晚上十一点过。急诊医生通知疝气教授会诊，急忙收入住院。住院后，立即动手术。把腹部打开，医生发现他的腹腔肠子已溃烂，液体开始有一点外流，小肠已发黑。教授说：可能是结肠癌，要做活检，肛门要改道（也就是在肚子上造口，粪便从造口排出），要家属签字。手术是两个，本来是看疝气的，结果肠子也动了手术。是他的小女婿陪伴他。

手术很顺利，哥哥在手术室待了十一小时，才到病房。后来教授查房时，对哥哥说：“你如果第二天到医院，就没命了，好在来的及时。”

因受疫情影响，医院只留一名家属陪伴，中途不得探视。一个星期，哥哥顺利出院。出院病历上写的是结肠癌，鉴定是中、晚期。医院又建议他到康复医院，他到康复医院，家属就可以探视病人。

第二天，我到医院去看他，给他带了真相护身符，我问他要不要？他说要，他让我给他放在衣服的上衣兜里。我让他念“法轮大法好，真善忍好”，他说好。

又过了两天，去看他，我给他带去师父在广州讲法录音，问他听不听师父的讲法？他说要听。因哥哥很支持我修炼大法，知道大法好，早就三退保平安了（退出邪恶的党、团、队一切组织）。以前，他经常佩戴真相护身符。我告诉哥哥，这是师父在保护你、在管你，所以，你才那么顺利。他很相信，伤口恢复的很好。后来，医生要他做化疗。我说：你回家，我教你炼功，修炼法轮大法。

他的女儿要给他化疗，我看他也想化疗，也就没阻止。结果，在医院，医生给他开了八个疗程的化疗。每次化疗都很轻松，身体没有痛苦。病房的病人都觉得很奇怪，连护士每次都要问他，大爷，有没有身体不适的地方？他都说没有。不象其他病人身体反应那么痛苦。我知道这是师父在为他承受一切痛苦。写到这里，我的心很感恩师父。

一切都那么顺利，身体也恢复得很健康。身体体检各项指标都很正常，转眼改做肛门修复。可华西的教授还要观察、又要预约，原因是：初期到医院是治疗疝气，做疝气手术，腹部打开后，才发现结肠癌的，鉴定是中、晚期。

自从哥哥开始听师父的广州讲法录音后，我又帮他把同修的修炼故事录音、同修们写的忆师恩录音、明慧广播、《九评共产党》等都叫他听。有些不能理解的、听不懂的，按照师父讲的法，帮他理解。他每天早晨四、五点钟，就开始听师父的讲法录音，晚上睡觉之前，念九字真言——“法轮大法好，真善忍好”。

预约是二零二一年十二月二十二日到二零二二年一月二十二日之间，是四周的时间。期间几次去问医院，咋还没通知，医院都叫等待。到二零二二年三月十二日，才通知哥哥到医院住院，做肛门修复手术。期间，又是身体各项检查后，各项指标都很正常，才做的肛门修复手术。手术后恢复的很好。

#### 哥哥的福报

一九九九年七月二十日后，中共江泽民邪恶集团发动迫害法轮功，利用一切邪恶手段打压大法弟子、抄家、收缴法轮功的所有书籍。为了保护好大法书，有的大法弟子都要把大法书收藏好，我也是其中的一个。为了保护好大法书，我把大法书包装好，放在哥哥那里，小心翼翼的收藏好。过了很长的时间，我才取回家。

派出所的警察一有所谓的敏感日，就骚扰大法弟子。哥哥利用工作之便，保护大法弟子。这些都是后来哥哥告诉我的。我说，你这是积了大德，行大善，天赐洪福，保平安。

后来哥哥真是得到师父的加持，福报满满的。本来哥哥是没有社保、医保的，单位要补办一批没有社保、医保的员工，要符合条件的补办。哥哥刚好在边缘上，补办了社保，但医保被另一个当官的给卡住了，当时也不知道，也就过了这么几年。

二零一七年三月份，单位重新上任的新领导发现，只有哥哥的医保没有办理，这才开始清查，并且马上补办医保卡。我知道，这一切都是师父给哥哥的福份，他的工资也越来越多。

二零二一年一月，因我讲真相救人，被警察绑架到派出所，关了一晚上，第二天被非法抄家。五、六个警察把我带回家，被哥哥看见了，他就跟在后面，因警察一進屋就把门关上，到处翻找。哥哥跟上楼，就敲门，警察问他，你是干什么的（警察的意思是不是炼法轮功的）？你是她什么人？哥哥理直气壮的说：我是她哥哥，你们这是干什么？我妹妹干什么了？你们这样对她。警察说：她到处宣传法轮功。

因在家没有搜查到他们要的东西就下楼，警察边下楼、边给哥哥说了不知什么。我从派出所回来，第一时间去看哥哥，看他躺在床上生气。哥哥看到我回来，就特别的高兴，问我吃饭没有，马上给我弄午饭吃。

二零二一年二月二十二日，也就是哥哥给我打电话的早上，我炼第五套静功时，师父把我的天目一下打开了，从我的左上方看到有两人架着一人放在我的左边，我告诉（指意念）他们：我看是不是我们大法弟子，刚说完，“唰”一下这两人撒腿就跑。马上天目又关了。我知道这两个人是不好的生命，但我没想那么多。

后来，我告诉哥哥，是师父救了你。你要感谢师父。哥哥就把他晚上做的梦给我诉说了一遍。哥哥说，梦到是两个人架住他往水里按，他不配合，挣扎着，问他们干啥子？这时来了一人，跟他们打了起来，从地面一直打到房顶上，我就跑了，也醒了。

我告诉哥哥，师父早就在管你了，那些把你往水里按的是水鬼，它们要害死你。在你危难时，是师父来救你了。

今天写出这些，是感恩师父救了哥哥的命，告诉世人我们师父的伟大，大法的伟大。
