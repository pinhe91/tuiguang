文: 河北大法学员 

【明慧网二零二三年二月十日】河北高阳劳教所的邪恶程度不亚于辽宁马三家劳教所。曾经有恶警把蛇放进女学员的裤子里去，女学员当场精神失常。多少年过去后，一提高阳劳教所，这位学员就脸色大变。
