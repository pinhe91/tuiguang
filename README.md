### 相信法轮大法　肺癌变炎症

文: 大陆大法弟子 

【明慧网二零二二年十二月二十一日】二零一九年八月，二姐感冒咳嗽，去了医院。可是两个多月过去了，感冒一点没好，她就去了医院检查，拍了片子。她让大姐去给她取拍的片子。大姐取出来一看，是恶性肿瘤，当时就如五雷轰顶。她立即给我打电话让我马上去医院。得知结果后，我回到店里，双手合十，求慈悲的师父救救二姐。
大姐不敢告诉二姐实情，就让二姐夫带二姐去了医院。胸内科大夫建议二姐住院治疗。二姐预感到病情的严重性，一直默默无语。晚上，我们都去了大姐家。我看出二姐非常绝望，精神几乎崩溃。我首先告诫自己：不要被亲情带动。然后走到二姐身边坐下，告诉她：不要害怕，不要担心，没有过不去的坎。

我以前给二姐、二姐夫讲过真相。二姐夫是警察，他不愿意听我讲的真相，当时还把我从他家赶了出来，并说：“你以后再也不要来我家。”二姐也说：“你以后不要再给我提大法了，你一说大法我全身颤抖，每个汗毛孔都要竖起来了！你要知道你一次次被绑架，一次次被酷刑折磨，家人受到多大刺激，多大的伤害。每当想起这些我都不寒而栗，我们再也经受不了了，请你不要再讲了。”

出于顾虑，这次我没有马上对二姐讲真相，只是每天和大姐陪她去医院做检查。常人的医院当然治不好二姐的病，我想不能这样下去，人命关天！我怎么能怕自己受到伤害而置二姐的生命于不顾呢？看到二姐痛苦绝望的表情，我怎么能忍心沉默下去呢？这不是强大的自私吗？现在我只有放下自我，放下怕心，不被他们的人心带动，救人要紧。于是我对二姐说：“姐姐，你也看到我从监狱刚回来时身体状况那么差，回来不长时间身体就恢复如初，难道你还没看出来吗？你不要再执迷不悟了，现在只有一条路可走，那就是修炼法轮大法，唯有大法能救你！”

二姐说：“我知道我已无路可走了，我也知道这个病的最终结果是什么。现在我只有按你说的求大法师父救我了！”听她这么说，我马上打开带来的师父的讲法录音。自那天起，我天天陪二姐听师父讲法。很快，二姐就有明显变化，心情好了，走路也变的轻松多了。我又给她下载了《细语人生》、《绝处逢生》视频，里面有得肺癌和其它不治之症的人修炼大法后康复的故事。这些神奇的经历使二姐看到了大法的神奇与超常，看到了生命的希望，并说要跟我一起学法炼功。

二姐决定不住院也不做手术，要求出院。这事被大姐知道了，对我破口大骂：“你炼什么法轮功，你得到了什么？你炼的家破人散，一无所有，你把家人都害得够苦了！还想害你二姐？！”大姐边说边放声大哭。接着又指责我：“你去我家，在我拜的挂图上（是附体功）写的什么全灭，你是想害我，还想灭我们全家！我对你怎么样？你一次次被绑架，一次次被残酷折磨，我为你痛苦，为你揪心，为你东奔西跑，为你少遭迫害去求人。而你却如此这样对待我，伤害我！”我一个劲的向大姐解释、道歉，说：“你误会了，你看错了，我在上面写的是正法口诀，完全是为了你好，你是我的亲人，我怎么可能害你呢？”

二姐也哭着说：“姐呀，我都快没命了，生命已经快走到尽头了，你就别吵、别闹了。不管怎么样妹妹都是为你好、为我好呀！她怎么可能害我们呢？”这时大姐静下来了，说：“你如果病能炼好，那我就天天陪你一起学法，陪你一起炼功。”

于是我就打开真相视频和她俩一起看，又看了有关《九评共产党》和其他真相视频。我陪她俩整整看了一个晚上。看完大姐恍然大悟，说：“原来大法师父是来救人的呀！电视上演的全是假的、骗人的！”大姐含着眼泪对我说：“多年以来我在伤害着你，冤枉着你，对你冷眼相看，我不知道你怎么走过来的，我好内疚呀！”

真是一波未平，一波又起。二姐夫来了，看到二姐办了出院手续，对我大发雷霆：“这么大的事你不和我商量，胆大包天，自作主张。你把自己家害惨了不说，又来害我们家！”二姐平静的对他说：“是我让妺妹回家教我炼法轮功的，你别怪她。”二姐夫一听就对二姐炸开了：“你敢！”二姐说：“我决心已定！谁也阻挡不了我。请你不要发火，看看真相视频，你就什么都明白了。”看到二姐这么坚定，二姐夫态度突然变的平静了，说：“那我就看看吧！”看完一个真相视频后，二姐夫说：“难道大法真的就有那么神奇？如果你真的身体炼好了，我不但也学，还要去推广。”

自那我就每天陪二姐学法炼功。十几天后，二姐去医院做了检查，结果是：那地方只是炎症而已。恶性肿瘤变成炎症！当时二姐、二姐夫都难以置信：“这是真的吗？”确实是真的！二姐夫放心了，说要去上班。走之前嘱咐二姐：“好好炼功吧，什么都不要想。”

二姐夫没去上班，直奔我爸妈家了，告诉我爸妈：“法轮功太神奇了！法轮功太神奇了！检查结果出来了，肺癌变成炎症！”我妈激动的泪流满面，连声说：“谢谢大法师父！谢谢大法师父！”

短短的十多天，大法就挽救了二姐的生命！在事实面前，大姐对大法佩服的五体投地，并说她也要修炼法轮功。

自那天开始，我们姐妹每天一起学法、炼功、看真相视频。师父为大姐清理了附体，大姐的气色越来越好，出现了奇迹。六十岁的人居然来例假了！她还开了天目，看到了很多美妙景象；原来严重的风湿病、高血压、心脏病都消失了，也变的无病一身轻了。

大法无边的法力再次震撼了我们整个家族及亲朋好友。

叩谢师父！
