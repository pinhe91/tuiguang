### 刘乐国任辽宁省营口市公安局副局长期间的恶行

【明慧网二零二二年六月一日】（明慧网通讯员辽宁报道）据二零二二年五月七日辽宁消息，辽宁省公安厅原副厅长、大连市公安局原局长刘乐国，被宣布涉嫌严重违纪违法被查。
刘乐国一九五七年二月生，此人在辽宁省公安系统长达二十多年。一九九九年十月刘乐国任辽宁省营口市公安局副局长；二零零三年十月任辽宁省公安厅治安总队总队长，二零零八年九月任辽宁省公安厅党委委员、副厅长；二零一三年六月任大连市副市长，大连市公安局党委书记、局长、督察长，大连市委政法委副书记；二零一七年十一月任辽宁省公安厅党委委员；二零一九年退休。

刘乐国在营口市是一位基建工人，他利用十余年时间就爬到了公安系统，他任营口市公安局副局长期间正值江泽民裹挟中共迫害修炼真、善、忍的好人。从一九九九年七月二十日至离开营口市，刘乐国为了升官发财，不遗余力地执行江泽民打压法轮功的政策，迫害法轮功修炼者。在营口地区形成一股恶势力，充当了江泽民“名誉上搞臭，肉体上消灭，经济上截断”、“打死白打死，打死了算自杀，不查身源直接火化”的凶手恶棍，丧失人性的大面积迫害法轮功学员。

据明慧网报道不完全统计，获知刘乐国任职营口市公安局副局长四年多时间内，营口市公检法非法关押法轮功学员106人，非法判刑27人，非法劳教33人，流离失所13人，迫害伤残3人，迫害致死11人，罚款58万之多。

#### 刘乐国任职营口市公安局期间部份恶行：

#### 1、迫害致死法轮功学员马天庸、鲁桂芳夫妻

![image](https://user-images.githubusercontent.com/79625284/171369207-1fbb0d0b-aded-4acc-a954-f53db2afc00a.png)

马天庸遗照

原辽宁省营口市国家安全局纪检书记、政治处主任，一级警督，马天庸，因不放弃修炼法轮功，多次遭到恶党疯狂迫害，开除公职、停发工资。

二零零二年一月份，马天庸同妻子鲁桂芳，儿子马旭辉一家三口去大连，被营口安全局特务一路跟踪，并联络大连市国安特务将一家三口在大连绑架，非法关押。

马天庸被非法劳教三年，关押在大连姚家看守所；他妻子鲁桂芳被非法两年劳教，送到臭名昭著的沈阳马三家教养院非法关押；他儿子马旭辉被劳教一年，被非法关押在大连教养院

遭受迫害。一个美满幸福的家庭被中共营口市公安、国家安全局（中共）迫害得支离破碎。马天庸一个大家公认的憨厚的好人，被营口市国家安全局局长、局党委书记谢洪军等不择手段的迫害。

在没有任何法律依据的情况下，原营口市国家安全局局长党委书记谢洪军、原营口市国家安全局机关党委书记曹洪霖（曹遭恶报死于癌症，谢得了治不了的怪病）等人为捞取政治资本，将退休多年并已丧失劳动能力的马天庸开除公职，终止了一切退休待遇，并长期监控，盯梢。在这种精神及肉体长期摧残下，马天庸于二零零七年三月二十九日含冤离世。

马天庸妻子鲁桂芳，原营口卷烟厂总会计师，修炼法轮功前，曾患有十几种严重疾病。心脏病、贫血、甲状腺瘤、风湿病、长期浮肿、妇科病、颈椎、腰椎骨质增生压迫神经曾造成昏迷，脱肛、内外痔等等。长年服大剂量的中西药，几次住院，每次药费近万元，多次在工作岗位突然病情发作，昏迷被送到医院抢救。

![image](https://user-images.githubusercontent.com/79625284/171369329-ab76630b-407f-4ca3-9343-b608004be0f2.png)

鲁桂芳遗照

一九九五年九月，鲁桂芳开始修炼法轮大法后，各种疾病不翼而飞，六年来没吃一粒药。二零零二年被非法关押到马三家教养院，迫害两年。残酷的迫害中，鲁桂芳身体出现生命垂危，后被保外就医，因长期的折磨，最终于二零零二年十一月二十六日离开人世，年仅五十岁。

熟悉鲁会计的人都知道，她为人正直，工作尽职尽责，不贪图私利，不为金钱所动，营口卷烟厂是当地经济实力较优厚的单位。别说那里的会计师，就是卷烟厂的工人当年都令人羡慕。但她用真、善、忍的标准严格要求自己，造福社会、关爱他（她）人，实现着一位法轮功学员的人生价值。

#### 2、迫害致死苏晶岩后扔入水洼

大石桥市法轮功学员苏晶岩，女，三十八岁。她身体健康，心地善良、勤劳手巧，曾是做裁剪衣服的个体户。一九九九年大法被迫害后，她为证实大法的美好，被迫害关押、拘留在戒毒所、看守所、劳教所。

![image](https://user-images.githubusercontent.com/79625284/171369517-92789da4-7414-4df2-a74f-a2f12883a6b8.png)

苏晶岩遗照

二零零一年八月初，苏晶岩被非法劳教三年，但中途逃脱魔掌。
二零零一年八月二十一日，苏晶岩与法轮功学员马素艳、赵桂琴一起到砬山地区发大法真相资料，她们是晚上九点钟左右动的身，而她们三人却一去未归。八月二十三日传来信息，她们三人在讲真相时被绑架，马素艳、赵桂琴被警察非法抓捕，分别被关押在拘留所、看守所，而苏晶岩却死在大石桥市夏屯的一个水洼里。

一大石桥市金桥分局里的内部人士（也是苏晶岩冤案的参与者）直言：苏晶岩在二零零一年八月二十一日晚，并没有走脱，而是被金桥分局的警察非法抓进了金桥分局，并问她砬山地区的传单是不是她撒的，而苏晶岩不配合他们的审讯，于是警察就对她大打出手。苏晶岩被迫害致死后抛尸于水洼里。

#### 3、迫害致残王爱云教师

一九九九年七月二十日，江泽民发起了邪恶的镇压，用王爱云老师的话说：“从一九九九年迫害开始后，我家终日不得安宁，天天都在被骚扰中。二十四小时被监控，警察经常来抄家。”熊岳公安分局的警察成了王爱云老师家的“常客”。他们时常派人开车到她家骚扰，翻书，搞得人心惶惶。

王爱云老师晚上睡不好觉，一听警车响，心就怦怦直跳。他老伴身体不好，片警来抄家一次，他就突发心脏病到医院治疗一次，家里人都跟着担惊受怕。

一九九九年末，营口市公安局伙同熊岳公安分局到王爱云老师家非法抄家，并把王爱云老师带至盖州市拘留所，非法拘留十五天。

二零零三年八月份，鲅鱼圈四个警察到王爱云老师家跳墙而入，将穿着衬衣衬裤的王爱云老师强行拖走，将其绑架至鲅鱼圈公安局，后送至看守所关押。

在看守所期间，王爱云老师一直吃不下饭，曾昏死过两次，在第二次醒后，感觉浑身疼痛难忍，狱警知道说：“上边有令，死了给你拉出去，不许喊。”

十天后，鲅鱼圈公安局强行将身体虚弱的王爱云老师送进沈阳马三家教养院迫害一年，她遭受药物迫害，生命垂危，王爱云老师在教养院煎熬二个月，便出现大面积脑出血现象，半身不遂，生活不能自理，至今瘫痪在床。

![image](https://user-images.githubusercontent.com/79625284/171369672-3b9f29f9-26d7-429a-ae33-8480e477968a.png)

王爱云教师被迫害致残
