### 曾遭冤狱15年几度命危　辽宁刘梅又被冤判2年

【明慧网二零二三年一月六日】（明慧网通讯员辽宁报道）辽宁省东港市今年五十六岁的法轮功学员刘梅，因坚持修炼“真善忍”法轮大法，曾经遭非法劳教两年，非法判刑十三年，她被迫害出心脏病、肺结核、小脑萎缩，几度命危。二零二一年十一月，她再次被绑架、构陷，二零二二年十二月十三日在丹东振安区法院被非法开庭，被非法判刑两年、勒索三千元。

#### 遭绑架、非法拘留

二零二一年十一月七日上午，刘梅因为去市场的路上偶遇同修张丽，就一起去市场，在往回返的途中被警察堵住，下来几个人就搜查刘梅的空包和张丽的包，然后把她们劫持到向阳派出所，又强行搜身，刘梅身上除了一点钱和钥匙什么都没有。刘梅她们要求回家，派出所警察不允许，要她们的姓名和住址，她们没有配合，因为知道他们要抄家。一直到傍晚，警察周兴宇非法提审刘梅，让刘梅坐铁椅。刘梅不坐，说自己不是犯人，有什么事你就问吧。刘梅告诉他，她包里除了一个记事本，什么都没有。他就火了，并说要拿记事本摔她身上，刘梅没有吱声。

这时教导员于本过来对周说，你下来，我来问，你在旁边坐着。于边说边过来轻松地对刘梅说，这都不是个事儿，赶快弄弄，两小时就放你们回家，我说了就算，我来给你们分。这时他就把一本台历等东西拿走，边走边说，这些是她的。回来就对刘梅说这五张小卡片（护身符）是你的，就这么点事，快弄弄得了，我们也得下班，快点签字。于本说就这么个程序，就走个程序，签个字马上就放你们回家，你们这儿根本就不是个事儿。刘梅不放心又问他一句：你说话算数，两小时之内保证放我们回家？于本还应声说，对，我说了就算，两小时保证放你们回家，我们还有事，快点弄得了，这不算什么事，快签吧，签了得了。

就这样在教导员于本的诱骗、逼迫下，刘梅签了字。期间，派出所停电，他们把刘梅带到可能是新兴派出所提审的，都是在这种诱骗、逼迫下让刘梅签字的。晚上刘梅要求回家，于本又说，现在是疫情，得给你们做核酸看看你们有没有感染，因为你们在派出所这么长时间，如果你们感染了，派出所的人和你们都得隔离，如果没有感染，才能放你们回家。就这样一直等到第二天出结果才放，结果刘梅却被非法行政拘留十五天。

#### 被欺骗构陷

十五天后，刘梅因医院检查身体不合格，看守所不收，于本特意赶到看守所，花四个多小时的时间，强行把她送进看守所的。

十一月三十日，东港市公安局国保和向阳派出所将构陷刘梅的所谓“案子”移交到丹东市振安区检察院。十二月七日，刘梅被丹东市振安区检察院非法批捕。

因身体原因，刘梅二零二一年十二月十七日从看守所回家，于本告诉她是“监视居住”。

二零二二年，于本多次找刘梅，骗她在取保单上签字，欺骗她说撤案。直到十月十一日于和警察周兴宇通知刘梅去检察院帮她“撤案”，因此刘梅还带着撤案申请，当见到检察官刘梅才知道，九月十六日于本又编造假材料报送了检察院，检察院下了起诉书，派出所根本就没有通知刘梅，他们利用刘梅去市场遇到的同修，伪造假材料说同修指证刘梅等等等等，实在是荒唐至极！

二零二二年十月十一日是检察院通知他带刘梅到检察院，是把构陷刘梅的案子往法院移交。当天所谓案子被法院受理，因身体不合格被看守所拒收。

二零二二年十二月十三日在丹东振安区法院非法开庭，律师做了有力的无罪辩护，并且刘梅和律师都把国务院50号令等文件提交法官，律师还把二零二二年“两高”新出台的解释提交法官，并且指出解释中没有明确法轮功是×教。

与刘梅偶遇的同修也出庭证明材料和卡片（护身符）都是自己的，刘梅没有，她也没有给刘梅，提审中她从没有指证刘梅什么，整个过程有监控录音录像为证。公诉人声称提审当天停电，没有监控录音录像。

十二月二十九日，刘梅被非法判两年，罚金三千元。刘梅上诉到丹东市中级法院。

#### 做好人遭牢狱残忍迫害

刘梅从小出生在农村，父母都是当地有名的教师，父亲是高中语文教师，擅长文艺和体育，而且会很多乐器，所以他们兄妹几个也会不同的乐器，每天晚上几乎都是在歌声和乐器声中度过，周围的亲友、邻居都羡慕不已。后来刘梅父亲工作调动，他们全家都进城了。随着父亲工作的调动和职位的提升，父亲的思想、观念也随着发生了很大的变化，在八十年代提出与母亲离婚，这对刘梅的母亲和他们兄妹来说，犹如晴天霹雳。看着母亲痛苦流泪，他们兄妹就更加悲伤、怨恨父亲。

一九九五年，刘梅万幸修炼法轮大法了，做人做事都尽量按“真善忍”的标准要求自己，遇事向内找，找自己的不足，慢慢地从内心原谅了父亲。无论在单位、在社会、在家庭，刘梅都按真、善、忍的标准要求自己做一个好人、更好的好人。在单位上班的时候，每年都是单位的先进个人。

一九九九年七月二十日中共疯狂迫害法轮功以后，刘梅因为到北京上访被非法抓捕，一次在看守所里，被几个警察摁趴在地上用三角带和竹皮子抽打，不知多少时间，他们就把刘梅打得昏死过去，刘梅只觉得自己轻飘飘的在空中飘荡着，盲无目的的往前飘着，后来听到远处有人在喊她的名字，她就随着喊声往回飘。过了一会儿，看到一帮人围着自己的身体，有在扎她的人中的，有哭的，有在哭喊她的名字的。看着看着，她感到全身疼痛难忍，却不知道自己在哪儿。又过了一会儿，她想起来了自己是在看守所里，这时听到有人说她眼睛动了、她活了，随之刘梅就醒过来了。那一次她似乎真的体会到灵魂的存在。当时她的臀、腰、大腿肿得像一个黑黑的、厚厚的大锅底扣在身上，动一点都疼痛难忍。

二零零二年刘梅与丈夫都被绑架、非法判刑十三年。刘梅被劫持到辽宁省女子监狱，到二零零七年年底，她的身体已经被折磨得非常不好了：高血压，心脏病，犯病时，身体一动也动不了，动一下小手指、眨一下眼睛的力气都没有；肺结核天天吐血、内分泌紊乱，肚子肿得比胸还高，便脓、便血，小脑萎缩，上厕所需要人扶；眼球萎缩、塌陷、视力模糊，脸一半胀红，一半萎缩，萎缩这面的那只耳朵薄得像一层纸，双脚脚背烂得袜子都粘在脓血上，两条腿肿成四楞型，无论黑夜白天，脑袋像镇流器一样铮铮作响，脑袋里面就像一个驯兽场，总感到里面有无数个野兽在里面乱跑乱叫；脖子左侧动脉处烂得每天都在冒脓、淌血，脖子异常难受，有时就像被谁掐住脖子一样喘不过气来，乳房里有肿块等，身体瘦弱到不足百斤，生活不能自理，走路需要人扶。刘梅当时曾多次找科长队长谈话讲真相，后来她多次给狱主要领导写信，狱里一位领导到监狱医院多次找她谈话，几次谈话狱领导对刘梅的人品、修养、口才和文化等一些方面很赞赏。从那以后，她的环境宽松了一些。在回家前的一段时间里，她的身体和心情在大法的法理中和在诚心敬念“法轮大法好，真善忍好”的口诀中得到了很大的调整和改变。

二零一五年刘梅刑满回家时，真是房无一间、地无一垄，借住在亲属的房子里。刚回来不久，她的身体还没有完全恢复好，婆婆因病住院，刘梅和丈夫就主动护理老人，二十四小时陪护，将近一个月出院。回家后，刘梅和丈夫每天都靠打工生活，丈夫长期出差。刘梅回家半年左右，老人因为年龄大了，生活不能自理，腿不好，心脏、肾等等都不好。老人生活在农村时也没有什么积蓄，脾气不好，强势，丈夫有两个弟弟和一个妹妹，都说不方便照顾老人，在这种情况下刘梅与丈夫就无条件把老人接回家，把大而有阳光的房间倒给老人住，由刘梅来照顾老人（其实他们结婚时，老人只给了一千元钱，因为刘梅手有工伤，老人不同意）。刘梅每天都尽量做老人爱吃的饭菜，尽管当时生活比较困难，但是只要是老人爱吃的，就尽量给老人买、给老人做。还经常给老人擦身、常给老人洗澡，也常给老人买些她爱吃的零食。有一年，老人卧床一个月左右，刘梅天天给端屎端尿 ，翻身、擦身。邻居、亲戚、朋友都赞叹不已，都说刘梅是难得的好人！
