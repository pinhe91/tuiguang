### 河北昌黎县警察花16万元安12个摄像头监控周向阳

【明慧网二零二二年七月十九日】（明慧网通讯员河北报道）河北省秦皇岛市法轮功学员周向阳遭受七年冤狱迫害，于二零二二年三月一日走出天津滨海监狱，但他并没有恢复人身自由，仍遭严密监控。中共人员花十六万元安设十多个监控器日夜监控他。
周向阳，四十九岁，当年从北方交通大学毕业，分配到天津铁道第三勘探设计院工经处，因工作出色，单位送他到天津大学，获得投资经济学位。一九九八年他考取了全国首批造价工程师职业资格，成为当时全国仅有的六十位造价工程师之一。在法轮大法真、善、忍法理的指导下，他工作兢兢业业，从来不要客户私下给的好处，是世风日下的社会中难得一见的卓然独立的好青年。

因为修炼法轮功，周向阳曾多次被中共警察绑架，几次被非法劳教，两次遭非法判刑，先后被非法关押在天津铁路看守所、天津青泊洼劳教所、天津双口劳教所、天津蓟县渔山劳教所、天津河西看守所。

![image](https://user-images.githubusercontent.com/79625284/179714226-8e9e1143-2c74-4021-9fed-84f50f4c8944.png)

酷刑演示：电棍电击

二零零三年五月三十一日，周向阳被非法判刑九年，在天津滨海监狱期间遭受无数酷刑：被彻夜电击至遍体鳞伤、连续三十天熬夜、多次关小号、野蛮灌食等等。二零零八年六月底，周向阳为抵制迫害，在港北监狱绝食一年多，体重只剩八十多斤，身体虚弱无法行走，大小便不能自理，直至二零零九年七月二十八日保外就医。二零一一年三月五日，周向阳在唐山的租住房内再遭绑架，劫持回监狱。

他妻子李珊珊因坚持为丈夫申冤，两次被非法劳教共计三年多。二零一三年劳教制度解体了，李珊珊是从石家庄女子劳教所走出来的最后一人。

一家人团聚的日子没有多久。二零一五年三月二日早晨七点，天津警察再一次破门而入，把周向阳和妻子李珊珊抓走了。李珊珊被非法判刑六年。周向阳被非法判刑七年，又被劫持到天津滨海监狱。

据知情人透露，在狱中，周向阳遭到狱警、犯人残酷的折磨、性迫害等，他以生命维护法，绝食抗争长达五年之久，被迫害致骨瘦如柴。他坚持反迫害，坚持打坐，坐的轮椅长出了美丽的优昙婆罗花，打坐中看到天女散花等。周向阳多次在人多的场合高呼“法轮大法好”，极大的震慑了邪恶，惊慌地将他按倒，往他嘴里塞布、塞馒头，往他脸上喷辣椒水，令他几近窒息；他曾被迫害致器官衰竭，被送监狱医院（新生医院）抢救。

周向阳约于二零二二年三月结束七年冤刑，但他仍不能和妻子团圆，被逼回到秦皇岛昌黎县马驼店镇马坨村老父母家中居住。他并没有真正恢复人身自由。在河北省、秦皇岛市两级政法委、“610”、公安局的指使下，昌黎县公安局国保警察耗资十六万，在村里、周向阳父母家里及他哥嫂家里至少安装了十二个摄像头，连他嫂子父母家附近也有。只要周向阳走出家门，就会被跟踪、盯梢。周向阳等于生活在另类监狱中。

周向阳的家人也日夜生活在恐惧中，怕自己的亲人还会被中共抓走、关押、折磨。

奉劝追随中共迫害法轮功的人，多行不义必自毙，人不治天治，好自为之吧。如恶名昭彰的天津市滨海监狱，里面的狱警折磨起法轮功学员手段残忍，灭绝人性，近期陆续传出消息，有多名青壮年狱警或上吊、或因急症不治而非正常死亡。里面的狱警因此终日精神惶恐、惴惴不安。
