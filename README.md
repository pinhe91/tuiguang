### 明白真相的众人声援大法弟子

文: 河北大法弟子金莲 

【明慧网二零二二年四月二十日】今年三月四日下午外出去讲真相，一路都很顺，看到前面有一位老者，我迎上去说：大哥你好，遛弯儿呢？送你个护身符，记住法轮大法好，瘟疫来时能保命。
他拿着护身符仔细辨认上面的字，看清后大声喊叫：“法轮功，你敢发这个，走，上派出所。”我不慌不忙的调转车头回去给他讲真相。他厉声说：“我是某派出所退休的。”讲真相他也不听，就自顾自的胡言乱语：“炼法轮功，孩子不能上大学、不能当兵、消除你的低保、扣除你的养老金……”他疯狂的大吵大叫，我一看没法讲真相，就骑车要走，他一把拽住我的车后座说：“走，去派出所。”我一看走不了了，就站在那向内找。

这一会儿工夫，过来了十几个人，其中一个老大哥和他吵了起来说：你干嘛欺负法轮功，人家真善忍，打不还手，骂不还口，我们社区的一个老太太是炼法轮功的，谁家有事儿她都帮忙。

这时又过来一个骑三轮车的老大姐对他说：“你这个老头子，干嘛欺负女人，让人家走。”这人就又说：“我是派出所的，她是法轮功（学员）。”大姐就说：“派出所的没好人，法轮功比你们强百倍。”他听后，就又跟大姐大喊大叫，坐在三轮车后座上的人说：“走吧，别找事儿。”她们走后，又过来了两个年轻人，看了一会儿说：“你抓杀人犯去呀！抓黑社会去呀！干嘛欺负法轮功？”过路的大哥还跟他理论说：“把钥匙给人家。”

我一边心里求师父，救救他，不许邪恶利用众生对大法弟子犯罪，一边跟他说：“大哥，我和你无怨无仇的，你这是干嘛？你这样对你不好。”说着，我从他手里把钥匙抢了过来，他就拿出手机开始找电话号码，要给派出所打电话。我马上想：不能让他打，那样犯罪的人会更多。我就一把抓过他的手机，他又抢回去。过路的大哥一边拉架，一边指责他说：“你这个老东西，那么多坏人不抓，专抓好人，你的良心招狗吃了……”

旁边的所有人都指责他、骂他，都愤怒的瞪着他，他看看没有一个向着他说话的，就气急败坏的说：你走吧！那个仗义的大哥向我摆着手，让我赶快走，我谢过大哥，骑上车离开了。

#### 突破怕心再讲真相

往回走的时候，我的心情糟透了，心里跟师父说：“师父，我哪错了？哪不在法上了？请您点化一下弟子吧，我一定要利用好这次机会找到执着，去掉它，提高上来。”这时头脑中出现一个字“怕”，我谢谢师父。想想真是：现在怕心出来了，每次出干扰后就不敢讲了，看谁都不可靠，错过了很多能讲真相的机会。我就跟师父说：师父，我错了，现在突破怕心，继续讲。

迎面是一个小区，有护栏，里面一个老者正在停放小三轮车，我隔着栏杆和他搭话：“大哥，遛弯儿回来啦？”他说是，我说送你一本书，这书可好啦，是告诉人怎样躲避瘟疫、病毒的。他说：“好。”高兴的接过小册子翻着看，我说再送你一个护身符保平安，你是党员吗？你知道退党保平安吗？他说：“不知道。”我说：帮你退了行吗？他说：“好。”分手时我告诉他心里记住：“真善忍好，法轮大法好”，关难来时命能保。他说知道了，谢谢你。我告诉他：你要谢李大师，谢法轮功。他合十说：谢谢李洪志，谢法轮功。然后，他说：我早就讨厌这个党了，没好人，说直理的人，它就排挤你；拍马屁的小人，直线上升。我赞许的说：大哥，你真是个好人，祝你平安健康，再见。

这时心里激动万分，充满了对师父的感恩，师父看着呢，众生真的觉醒了。刚才的正邪较量中，我虽然没有直接给那些人讲真相，但是这十几个人也在这次事件中，选择了站在大法一边，做了最正的选择，摆放了他们的位置。这也是同修们日复一日的讲真相，和同修们按照真善忍做好人，把众生被邪党谎言欺骗的假相破除了，使世人都知道了邪党的坏，都知道了法轮功的好。

师父说：“明白了真相的人有的可能会动念要修炼，有的人会很同情，有的会用行动来支持。这些讲真相中所带来的反应，也是人传人、心传心的扩充着。邪恶因素在失去环境。大法与正的因素也在覆盖与充实着一切，同时针对这场迫害，它会使社会、使方方面面都发生变化，叫恶人没有市场，叫邪恶的生命无机可乘，它必然会起到这样的作用。大家在讲清真相时看似平常，作用巨大。”[1]

写到这，我也非常感激全力以赴讲真相，平时在日常生活中做的好的同修。按真善忍标准做人的大法弟子的表现，就能洗去邪党抹黑法轮功的那些惑众谣言，刚才在一片混乱中，就有一个人说：过去的土匪在深山，现在的土匪在公安，派出所里没一个好东西，让人家走。还有一个好心大哥，一直数落那个人，说他这样做是干缺德事。我把钥匙从那人手里抢回来之后，好心大哥马上站在中间指责那人不对，又对我挥手叫我走。

#### 回家后继续向内找

回到家，在师父法像前合十请师父点化，我肯定是哪错了，有什么执着招来了迫害与干扰，拿起《转法轮》一翻开是“提高心性”。真是应该提高了，这一阵子，怨气太大了，看新唐人电视台每天报三退人数才四、五万人，过年期间更少，才两万多人，什么时候才能救下足够的人数？二月份记录了一下我讲真相总的三退人数，二十八天退了一千零三十个，过年我都不停的。我知道在讲真相的这几年又生出了一个大执着，就是怨恨心，对同修不讲真相找各种借口、散漫、求安逸、享受、办护照出国游玩儿、买好房子买好车，把心思用在过常人生活上，一想起这个我就一肚子气。

同修知道我的想法后告诫我：你的求名心够重，爱显示，证实自我。我还找到了喜欢被别人崇拜的心，当别人赞扬自己时，心里美滋滋的，这个心必须得去，太危险。求名心、显示心这都是对大法不敬，对师父不敬的行为，此心一起就已经在大法之外了，危险至极啊！还有急躁心，一急就跑回人这儿了。一次给一个老太太讲真相，让她记住法轮大法好，三退保平安，说了两遍，她没听明白，就说：丫头，你说慢点儿，我有点儿耳背，听不清你在讲什么，但我看到你那颗善良的心，是为我好。

我又慢慢的给她讲了一遍基本真相，她听明白后，高兴的大声说：谢谢你。我说：要谢谢李洪志大师，是我师父教叫我们传真相救人的。我又给她真相小册子、护身符和《共产主义终极目的》书，跟她说看一看这书就明白了，我父亲就在“文化大革命”中差点儿被害死。她说：“我老头子（丈夫）就是在文革中被打死的，好，我一定要看看。”说着就把资料装进书包里合十向天高喊：“谢李大师，我终于和这个坏党划清界限了。”

还有争斗心，不是在利益中争，而是在讲真相时，人家不听，或者不接受真相，对大法不敬，我马上恼怒和人家争起来，有时能控制自己的心态，有时就不行了。这都是党文化的恨、斗，造成这次魔难里面也有这个因素。还有怕心，感觉这个怕在我修炼中如影随形，许多时候意识不到它的存在。

清除这些人心执着，静心学好法，在法中归正自己，让自己更加纯净的去讲真相，救更多的世人。感恩慈悲伟大的师父！谢谢同修的无私帮助！

个人体会，层次有限，不足之处敬请慈悲指正。
