### 安徽女监狱警詹咏梅用多种手段迫害法轮功学员

文：安徽法轮功学员 

【明慧网二零二二年九月十七日】詹咏梅，约50多岁，在安徽女子监狱是专门负责强制转化法轮功学员的狱警。被送到监狱的法轮功学员，都要到监狱转化班进行所谓的“学习”，由詹咏梅带着三～五个被迫转化了的人，去包夹这个刚来参加“学习”的法轮功学员。
詹咏梅采用的手段很多，开始是假惺惺地关心法轮功学员，试探学员的态度，如果三、四天后学员仍然坚持修炼，就开始罚站，不准上厕所，并在吃饭的时候，安排打饭的人只给打一点点饭菜。

法轮功学员回到监区后，詹咏梅对监区的狱警说，对这个学员严加看管，其他刑事犯都可以任意打骂该学员。法轮功学员在监区同样是被几个刑事犯包夹，也不让洗澡。再到转化班“学习”后，詹咏梅威胁说再不转化不允许家人会见。
有律师来会见被迫害的法轮功学员，詹咏梅曾欺骗律师说，某某说了不见律师，事后律师打电话给被迫害学员的家属，家属会见过学员，家属说学员没有说不见律师。詹咏梅有时又对律师说，法轮功学员不转化不给会见。

詹咏梅有时整晚不让法轮功学员睡觉，不给吃饭，拿一个空碗放在学员面前，说“你师父无所不能，就让你师父给你饭吃”。大法师父告诉学员可以用正念使电流与药物转到施暴者身上去。狱警詹咏梅就拿电棍电法轮功学员身，说看看电是不是倒流，如果不倒流，就叫其他被迫转化的人骂师父，从而欺骗学员转化。

詹咏梅让人把师父的像粘在凳子底下，让法轮功学员坐在凳子上，由于粘在凳子反面，学员不知道。过一会，詹咏梅就说：“你尊敬你师父是假的，都把你师父的像放在屁股底下坐着，这是尊敬师父吗？”还让被迫转化的人轮番坐在贴有师父像的凳子上，让被迫转化的人不停的喊污蔑法轮功的言辞。

詹咏采用各种下流手段，逼迫被冤狱迫害的法轮功学员转化，做了十几年的“转化”迫害法轮功学员的恶事，曾在安徽省监狱第二届所谓“教育转化”（实则洗脑迫害）技能大赛决赛获得“二等奖”。
