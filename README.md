### 诚念九字真言　先天性心脏病女孩康复

文: 四川大法弟子

【明慧网二零二三年二月二十四日】法轮大法是性命双修的佛家高德大法，于国于民有百利而无一害。大法在世间弘传三十年间，有幸了解大法真相，诚念“法轮大法好、真善忍好”九字真言从而遇难呈祥逢凶化吉、得福报的事例比比皆是。我在修炼大法二十多年中，亲眼见证家人及亲朋诚念九字真言得福报的事例很多。现仅举几例：

#### 例一：先天性心脏病女孩康复

我妹夫家侄女两岁多就检查出患有先天性心脏病，医生说要等长大了才能做手术医治。亲戚都说这孩子蓝蓝的大眼睛，皮肤白如凝脂，很漂亮，可惜得心脏病还要做手术。几岁时父母离异，父亲多年不知去向，母亲工厂倒闭。她就跟随大姨生活、读书。大姨家住的工厂宿舍，生活环境恶劣，每天煤气咔喉。

到十六、七岁读中专时，订好在中心医院由北京来的专家做手术。各项术前检查后决定第三天做手术。这期间她大姨每天以泪洗面，说自己工资低，既要供她读书，又要从牙缝中挤钱出来凑足手术费。还怕万一手术有闪失，人财两空。我们就给她和侄女说，天天念“真善忍好、法轮大法好”九字真言，就可以保手术顺利平安。她和侄女听后，就很诚心每天念。侄女的表姐，还有我们了解这个情况的家人都帮她念这九字真言。

手术前复查，意外的惊喜出现了，专家说心脏缝隙变小了，不需要做手术了，今后不影响结婚和生育。全家人听此大好消息，皆大欢喜，都非常感动、感激。她大姨失声大哭说：孩子，你命大，是大法救了你，我们全家都感谢大法师父。她们全家都做了三退。

几年后这个女孩结了婚，后生了一对双胞胎，长得十分可爱。

#### 例二：两口子说：每天有空就念大法好，身体好、生意好

我地农贸市场有一对给别人加工生肉的夫妇。女的热情勤劳朴实，男的个头偏低，腿瘸，皮肤粗糙，满脸油腻，还时常长满疹子之类的，态度生硬，给人印象很不好。经我们给这对夫妇讲了真相，他们都退出了团队。从此每天有空就念九字真言。

不久这夫妇就发生了很大的变化。女的健康快乐，男的皮肤变的白而细腻，脸上一点疹子都没了，经常都是面带微笑，和气待人，两口子和睦了，生意越做越好。我们去洗肉或打肉糜女主人高兴的大声说：你们看我们那个人长好了、长漂亮了。这人真是从内到外都发生了相当大的变化，一点都不丑了，真的变美好了。

两口子说：我们每天有空就念大法好，身体好、生意好，再不信造谣谎言了。

#### 例三：小王说：“太好了，太神奇了！”

小妹家装新房，给她家做墙面的工人小王，再做一天他的工作就完成了。小妹请小王和电工吃晚饭。席间，小王闷闷不乐，心情不好。小妹问小王怎么了，他说：“我哪有心情吃饭啰！我女儿生病住院，前额长个疔疮，头痛、发烧，我有个朋友的孩子也是得这个病，没医好，死了。所以我担心得很！”

小妹告诉他别急，你马上去告诉她念“法轮大法好、真善忍好”九字真言，你也和她一起念。好多病人甚至危重病人念九字真言都神奇般的好了。这小王说有这等好事，她马上去医院叫女儿念。饭都顾不得吃，就急着走了。

第二天小王没来，第三天来了，高高兴兴的做工作。小妹问：小王，你女儿好没？小王说：哎呀，我还忘了说了，好神奇哟，太神奇了！那天我赶到医院去，女儿正痛苦难忍。我告诉她马上念“法轮大法好，真善忍好”，而且我和她一起都小声念，有时默念。很快她就感到轻松了，到晚上十一点多，又不烧又不痛了。当晚美美的睡了一觉，直到天亮。早晨起来精神好得很。第二天就出院，今天都上学去了。

我们听后都很高兴。小王说：“太好了，太神奇了！感谢小姨！”小妹说你谢谢大法师父吧。小王说：谢谢大法师父！谢谢大法师父！

还有我们亲人中，有得了肺结核的，我给她讲了真相后，她也经常念九字真言，还看了一遍《转法轮》，两个月就痊愈了。类似的例子还有很多，在此就不一一列举。世界需要真善忍，希望大家快来了解大法真相，诚念九字真言，顺利度过大瘟疫、大劫难，迎接生命美好的未来！
