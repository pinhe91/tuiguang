### 【720评论】公检法司人心渐醒 中共迫害法轮功难以为继

文: 大陆法轮功学员

【明慧网二零二二年七月二十一日】大家都知道，中共一直都在大陆搞愚民政策，特别是二十多年来残酷迫害法轮功，把公检法司人员当傻子用，当流氓用，当黑社会用，当恐怖分子用，当杀人犯用。有的人可能不同意我的观点，说那公检法司人员都很精明，怎么会被利用呢？这就是中共愚民政策“成功”的一面，它用暴力、谎言、欺骗、洗脑就做到了这一点，迫害法轮功就是一个例子。它让这些看似精明的应该为人民服务的执法人员，都当了中共自己统治的保镖、打手，指使这些人对善良的百姓任意剥夺人权、自由，做了普通流氓、黑社会、杀人犯等都做不到的邪恶之事，这就是中共邪教愚民的关键，让人党、国不分，以为维护党就是维护国家，错了，中共是来毁人的。
每当哪个公检法司人员对法轮功学员做了酷刑残害等恶事，法轮功学员都认为这是中共让他们这么干的，如果背后没有中共的指使，他们都不会干那么邪恶的事。但是天理明鉴，谁行了恶，谁要遭恶报。而当哪一个公检法司人员做了同情、保护法轮功学员的好事，法轮功学员都认为那是他的个人行为，不是共产党让他那样做的，那是人的天性、良知返出来了，在中共依然保持打压态势的情况下，公检法司人员能认清它的邪恶，不配合、不参与迫害好人，那么就显出这些人非常的难能可贵，他们都会得好报。

公检法司人员觉醒的例子很多，因篇幅有限下面就举几个例子，我们来看看这些载入史册的善行义举。为了保护这些觉醒的公检法司人员，我们只说事件，不提单位姓名。

#### 一、警察明真相后感人的改变

某地一名警察以前不明白真相，迫害过本辖区的法轮功学员。法轮功学员不但不恨他，而还善意地给他讲法轮功的真相，他每次来法轮功学员家执行所谓上级给的“任务”时，都能受到法轮功学员的热情招待。他从这些法轮功学员的身上，体会到法轮功善的力量，看清了中共造谣诬蔑的骗局。

一次，他的上级指名让他去抓捕他辖区的法轮功学员，他当时就拒绝说：“他们都是好人，抓他干啥？我不抓。”他的上级回答说：“你不去，我带其他人去。”他一听，人一定得抓，这我挡不住，可我在他们来之前给法轮功学员通个信，还是能做得到的。结果，那些人来抓人时，都扑了个空。使他辖区的法轮功学员免遭损失。

#### 二、检察官的觉醒

一位法轮功学员自述：妻子的哥哥是一名检察官，给他讲真相最难，一讲他就又吼又叫的，嗓门特别大。他说：“共产党给我钱，我不能退党，我得同中共保持一致。”一次，他来我家，谈话间他说：我们检察院和法院，在同一个办公楼办公，一家一半楼。这几年，同在一个楼办公的法官、检察官得癌症的一个接着一个地死。而且人死得很奇怪，也很有规律……

我给妻子的哥哥，讲了大法真相，还把检察院、法院相继都遭恶报的事写成文章在明慧网曝了光。后来，妻子的哥哥见到我说：“你很会造句。”我知道他们检察院的人都知道了。妻子的哥哥说，现在没人愿意接法轮功的案子。

一天，妻子的哥哥叫我们帮助干点活，他说：法轮功早晚有一天一定会有一个说法的，你们等着听好消息吧。

#### 三、法官的善举

二零一五年五月二十五日，河南淮阳县法院对法轮功学员王宪淮二审开庭，十多分钟就结束了。一审的审判长马骏说：是送你回家还是叫你家里来人接你？王宪淮说：我自己打车回家。最后家里来人把他接了回去。要知道，王宪淮一审已经被判了三年，可一上诉竟然就让回家了。对于一般的刑事案件，也确实有上诉后改判，或被无罪释放的，而对法轮功学员的上诉案，“610”从来坚持的就是维持原判。可如今王宪淮竟然在上诉后什么也没说就让他回家了，这样的变化不可谓不大。

#### 四、司法部门官员明真相做出善的选择

学员自述：一天，我告诉大妹，你们监狱的监狱长、副监狱长还有许多恶警都被明慧网曝光了，你的名字也上明慧网了。当时，她的面部没有表情，很痛苦。我告诉她，迫害法轮功学员是要遭报应的。我给她讲了念“法轮大法好”、“真善忍好”得福报的小故事，我叫已经患病的妹妹念念试试，她说知道了。

劝他们退党保平安，大妹夫当时就说：“哥，你给我退了党、团、队吧。”母亲、大妹、小妹也退出了党团队组织，并同意发表严正声明，今后相信法轮大法好，维护法轮大法，并严正声明以前所说、所写污蔑大法的言辞全部作废。

发表严正声明后，大妹的身体奇迹般地康复了。大妹夫本来要晋升副监狱长的，大妹说他放弃了，在监狱当副监狱长难免不参与迫害法轮功学员，这样每个月会少收入五、六百元，但是心里踏实。两个妹妹说，以后不参与迫害法轮功学员，他们现在的岁数和爸妈的寿龄差不多。

#### 五、政法委书记带来“三退”名单

学员自述：有一次，丈夫与他的朋友们一起去爬山，我也去了。一个县的县委副书记兼政法委书记问我：“你怎么爬得这么快呀？”我说：“我是炼法轮功的嘛！你可能不知道我过去什么样——一个瘫痪在床将近半年的废人！是法轮功师父救了我。师父不但给了我一个健康的身体，还使我明白了做人的道理和人生的意义。”

他说：“哎呀，原来法轮功这么好啊！”于是，我就进一步给他讲了真相，还给了他一本《九评共产党》和一些真相资料，让他看完后再给其他人看。那天，不但他退出了中共的党团队，连他的司机也“三退”了。

有一天，这位县委副书记给了我一张折起来的纸。我打开一看，原来是一张十多个人的“三退”名单。显然是《九评共产党》及法轮功真相资料让他彻底醒悟，他就把自己的亲友劝退了。

#### 六、610人员的醒悟

一位八十多的法轮功学员，把一份资料递给一名五十多岁的男子，慈悲地告诉他希望他了解真相，希望他平安。

可那位男子被中共对法轮功的诬陷宣传迷惑，就对大法学员说：“你别走，站那别动，我得给‘六一零’打个电话。”他把电话打了过去，告诉“六一零”说：“这里有法轮功学员发资料。”因为那人的电话开着免提键，老人就听到那边回了他一句：“法轮功学员发资料，碍着你X疼？！”这人听后一下愣在那里了，然后把电话关了放回兜里。自觉没趣，推着车子走了。

#### 七、结语

大陆各地公检法司人员很多人都在觉醒中，都不愿做这种犯罪事了，邪恶的迫害也就难以为继了。我们盼望着从公检法司内部发生质的变化，当觉醒者回归正义，保护好人时，不仅削弱了中共的邪恶力量，也保护了他们自己。人在做，天在看，邪恶的大厦轰然倒塌的那一天已经不远了，希望好人都登上渡船，走向美好的未来。
