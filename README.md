# 【庆祝513】仁者之勇——写给父亲的家书

文: 海外大法弟子

【明慧网二零二一年五月二十三日】
爸爸您好！

好多年没有给您写信了。虽然我们父女几十年中生活在一起的时间不多，彼此很难说完全了解，但是，我知道您是一个作风正派、坦荡光明的人。这一点，其实也被我继承了下来。您未曾改变的遵循着一种传统的生活方式，尽管经济宽裕，却简朴、自律。在眼下这个道德下滑、乱象尽显的社会里不随波逐流，也是难能可贵。

爸爸，如果您觉的您一生不为利益与人勾心斗角是值得自豪的话，那么，您也可以为您的后人——我以及您的外孙铭铭感到高兴，因为我们同样继承了这一点，对物质利益少有贪欲，不会见利忘义。爸爸，谢谢您和母亲把我影响成为一个品行端正、是非分明、活的心底踏实的人。

爸爸，今天想通过书信和您说说心里话，因为我们缺少沟通，您可能对我人生中的一些选择不理解，不认同。非常抱歉，是我做的不好，没有向您陈述自己的经历，也没有真诚倾听您的意见。不是我不尊重您，而是我在您和母亲面前说话，心理压力比较大。虽然我今年五十有二，但在父母面前，总觉的自己象个时常犯错的小学生。我脸皮薄，心理承受能力差，使我常常逃避和你们的交流。

我年少时，体弱多病。让你们，特别是母亲，对我操的心比妹妹多很多。尤其遗尿的毛病，母亲很痛苦，我也很痛苦。记得多少次半夜里，母亲发现我又把床上的被子、褥子全弄湿了，她一面快速的收拾，一面大声的发泄着情绪。寂静的午夜，我真的觉的自己是家里的拖累，不知道活着有什么意义，时常冒出结束生命的念头，却没有勇气。

因为我身体不好，在同学中我的毛病也被拿来取外号、嘲笑。因此，我觉的自己低人一等，自卑、报复心强。我不思進取，没有奋斗目标。浑浑噩噩中，不知道怎样考上的大学。浑浑噩噩中，不知道怎样结的婚。讲台站了十五年，在第一个任职的学校，用“混口饭吃”形容很恰当。学校不正规，大家都是混日子的心态，我也心安理得的混着。

生孩子时，我大病一场。夏天要穿厚衣服，身上忽冷忽热；夜里常常失眠，白天无精打采。混日子都没有精力混了，只好搞个停薪留职，在家养病。这个医院進，那个医院出，我没有起色。中药喝的恶心，还是没有好转。病急乱投医，还跑去见一个搞巫术的，别人说灵验，我却没有感觉。

有一次，我从小何那里提着中药回你们那里时，看到口腔医院门口盖楼的脚手架，我站住了，希望它倒下来，让我长痛不如短痛。那时，我就是这样卑微的仅仅留着一口气。

一个朋友看我活的痛苦，借给我一本《转法轮》，说看看也许会对我有帮助。当时，我不相信看一本书能治好我的病。但是看了书以后，我的身体确实在短短的几个月后有了很大的变化。我没有進医院，也不再去小何那里拿中药。

半年后，我重返工作岗位。此前，一堂课四十五分钟，我只能站着讲十分钟左右，就必须坐椅子。课堂纪律不好，我就只能任由学生吵闹，因为我提不起气来制止他们。

修炼法轮大法后，上课四十五分钟，我不仅可以站着讲到下课铃响，还可以根据情况调节课堂气氛，既让孩子们活跃又不至于无秩序。要知道，这对六、七十人的班级来说，既需要脑力也需要体力，我却比从前任何时候都教的轻松。

到如今，我修炼法轮大法二十二年，从未吃药、進医院，无病一身轻，这是不争的事实。“滴水之恩，当涌泉相报”。爸爸，您说我能不感恩吗？更重要的是，《转法轮》给了我人生的答案。

我一直认为命运对我不公，多病痛；脑子不开窍，受歧视。大三我的眼睛出问题后，我心灵更加苦闷，有报复社会的扭曲心态，开始偷东西。在大学商场假装买东西，趁店员不注意，就把柜台上的商品装進自己的包里。甚至对我没用的东西也拿，然后扔掉，好象从中能得到一点心理平衡。

工作后，买菜时顺手牵羊拿人家的鸡蛋。这种行为，一直持续到一九九三年。在华生园，当我连买带偷拿着人家的蛋糕从商店出来时，猛然感到头顶上有双眼睛看着我，我不自觉的抬头。深邃的天空中，那双眼睛定定的罩住我，使我惶恐不安，不知道怎么回的家。这是我人生最后一次偷东西。华生园门口那双眼睛令我生畏，永生难忘！

虽然我杜绝了这种肮脏的行为，然而，我的心态还是不健康，怨天尤人。现在的女人生孩子，没有几个象我这样生出大病来的。这一点，我更是又怨又恨。看了《转法轮》之后，我明白了做人的道理，明白了自己为什么病痛多，被别人看不起。从此，我告诉自己，不管什么境遇，再大的不公，我都永远用好人的标准要求自己。

我庆幸自己能恰逢法轮大法洪传，如果不是走入法轮大法修炼，我无法想象生活在这样一个做好人都难的社会里，我怎样才能不随波逐流。在各种不公面前，怎样才能保持自己的良知不被泯灭。

孔子讲：四十不惑，五十知天命。我庆幸自己在不惑之年踏上了返本归真之路，找到了人生的真正意义。

爸爸，您也知道我以前脾气暴躁，点火就着。初中时，同学对我不公，我竟然站在课桌上拿扫把打他们；在您和母亲面前也是，被冤枉了，会歇斯底里的发泄；后来和学生相处，也战火不断。有一次，我和一个初二男生发生矛盾，他甚至抢过我的教鞭要打我。还有一次，我骂一个女生，她告到校长那里，要教育处通报批评我。唉！现在回忆起来，我都为自己脸红。

然而，法轮大法改变了我。我后来任职的学校，是当地重点中学，老师们为了职称奖金，人际关系挺复杂。然而，我却不时听到同事当面对我的赞扬。他们说我是现今社会上少有的“只讲付出，不求回报”的人。开玩笑说，我是“濒临灭绝的物种”。

年级主任是个不苟言笑的老太太，常常对我说：“全校就你心态好，我要向你学。”我的大学同学是副校长。有一次，他在我们年级办公室问一个老教师：“我这师妹怎么样啊？”老太太笑呵呵的说：“个个员工都象她，你们当官的就轻松了。”教导主任对别人说：“你看夏老师，干工作从不讨价还价，学生给她打的分最高。”爸爸，如果不是修炼大法，我不会得到这些好评的。

教师办公室的清洁卫生，都是找学生来做的，我也从没觉的不好。修炼后，我不再接受老师翘着腿，让学生拖地的现象了。我每天到了办公室，就在默默无声中把拖地、擦桌子做了。有一天，我正在拖地，一个老教师端着茶杯笑呵呵的说：“看来法轮功是厉害啊，把我们这懒丫头教勤快了。”当我离开学校的时候，年级组聚餐，有个男同事对我说：“夏老师，我很佩服你。你人很正，我这个人平时喜欢吊儿郎当开玩笑，在你面前，我却不敢放肆。”

我的学生常说我和别的老师不一样。他们私下告诉我：“年级只有你和某老师（一个受私塾教育的返聘的老太太）没有绰号，因为我们是尊敬你们的！”同事的孩子是我的学生，同事说：“只有你的话，我女儿会放在心上。家长把孩子交给你，就省心了。”我的学生送给我的笔记本扉页上写着：“我为自己感到庆幸，在人生迷茫的时候，有一个高贵的灵魂来到了我的身旁，为我指引方向。老师，我会永远记得您！”

一个男孩子，当我告诉他“天安门自焚”是中共的造假新闻，法轮功不是电视上说的那样，孩子大睁着眼睛，很吃惊。过后，他神情庄重的对我说：“老师，只要是您说的，我都信。”爸爸，这些话，是我教书生涯得到的最大回报。而这些回报，是在一九九九年我开始修炼法轮大法之后。

我的同事夸赞我，我的学生尊敬我。而我，感恩我的师父！感恩法轮大法！没有真、善、忍信仰的力量，在大染缸中，我也只能随波逐流，难以洁身自好。因为逆流而行是需要勇气的。

修炼前，我也对利益看的很重。记得学校开教研会回来可以报出差费，同事们都是尽可能多报几天，车费也尽可能多写，我也一样。修炼后，我不再这样做了，因为我明白，这样做，会损德。

二零零零年初，单位有分房指标，据说是最后一次福利分房了，就一个名额，正好排到我。后来，一个同事利用他和领导关系好的优势，得到了房子，同事都替我不平，劝我去找校长，说以后没机会了，不能太软弱。最后，我没有去。校长知道自己做的不公，看见我，都有点不自在。

谁知半年后，又来了一个新房指标，说这回才是最后一次。校长二话没说，直接叫我去填分房表。同事们都很惊叹！因为这个房子的面积、地段，是我校福利房中最好的。

这件事后，同事对我的态度似乎多了一些尊重。我没有争强，却得到了更好的结果。我想，这也是给我的同事一个无言的思考吧。如果没有修炼法轮大法，我会和许多人一样，为利益而争而斗，活的又苦又累，绝不会有这样的洒脱。

爸爸，每个人的人生追求不一样，价值观也不同，可谓人各有志。今天，我为追寻正义、良知而努力，奔走，我很自豪。我的丈夫，他大学加入了中共党组织，曾经对共产党相信到了感恩戴德的程度。我的信仰，让他经历了几年的内心挣扎（包括想离婚），我们家庭为此有过争吵甚至冷战。他心爱的儿子，就因为说“進化论是错误的”，被他一脚踢在屁股上，然后甩下我们，怒气而走。那一年，铭铭九岁。

二零零八年，我被绑架到派出所的时候，我丈夫被警察打电话找去。从家里走到派出所，十几分钟的路程，他竟然膝盖在流血。后来我问他在哪里摔倒，他却不知道。看着他在警察面前努力保持镇定和恭顺，我既心痛又难过。身为外资企业的副总，他平日里给人的印象都是沉着冷静、遇事不惊。而在中共的淫威下，他目光中透出的惊恐与胆怯，让我宁可接受他与我离婚，由我独自面对这一切。

几经挣扎后，他没有选择离开我。我曾经问他：“为什么没有提出离婚？”他说：“不会找到象你这样的好人了。”如今，他精神觉醒，已经给自己退了党。他默默的在经济与精神两方面给予我支持，因为他知道我在做着一件正确的事情。他也慢慢相信，他会因为支持我而拥有福份。

我们虽然经济不是很富足，因为种种原因，三口之家如今还未能团聚，但我们彼此信任，尽管有矛盾，也互谅互敬。铭铭在我们的影响下，身心健康，孝敬、仁厚。见到他的长辈，大多都会对他表达喜爱和认同。然而，这并不是我这个母亲的功劳，而是法轮大法的佛光普照。若没有修炼法轮大法，我都不会做好人，谈何教育子女？

爸爸，请相信，无论现在还是将来，我都不会给你们丢脸。这种代代传承的心胸坦荡的作风，也许就是一种家风吧？只会让您在人生的回忆中引以自豪。

目前，我和我的同修做的事情，有很多人不理解，骂我们蠢货，吃饱饭没事干，甚至包括亲人也会恶语相向。但我知道，不是同胞手足不善良，而是因为几十年走过来，大家已经习惯了在中共的强权下低头、做顺民，这样才能让自己有一点暂时的安全感。

从我的外公、奶奶开始，亲身经历了亲人们的遭遇后，我能理解你们只是想获得安宁生活的愿望。母亲也是希望我离开中国，让她不要有压力。我理解，真的理解，你们经历的太多了。

爸爸，您可能还不知道，我曾经因为和两个路边的中学生谈论法轮功而被抓進拘留所。家人不敢告诉您，是怕您上火。这就是为什么他们希望我离开国内的原因。可是，爸爸您知道吗？拘留所的经历，不是我人生的耻辱。我在那里得到的是默默无声的同情，掷地有声的赞叹！

一个警察在周围没人时，悄悄的问我：“你会恨我吗？”我说：“不会。”他叹了口气：“回去还炼吗？”我说：“还炼。”他低下头，轻声说：“以后要小心啊！”临离开时，他望着我，目光停留了好几秒，用告别亲友一样的语气说：“你自己多注意啊，我们走了啊。”那一刻，我忘了自己的处境，为他的善念而身心愉悦。

拘留所里一个偷钢材被关進来的老婆婆，大声说：“这里面的人，都是干了坏事進来的。老师你是冤枉的，你和我们不一样。”一个吸毒的年轻女孩，详详细细的问了我很多问题，最后她说：“阿姨，您要是我妈妈就好了，我就不会有今天了。”还说出去一定找大法的书看。

爸爸，您别因为我曾经被中共关押觉的丢脸。相反，您女儿不向强权低头，不卑不亢，坚定中没有对立，慈善而不失尊严，警察也暗自佩服。我相信，未来，我的家人将以我为荣。

仁者之勇，不是没有恐惧，而是在高压下，能够挺直脊梁；在暴力下，坚持和平理性；在克服恐惧中，依然勇往直前。这种同化了真、善、忍而自然流露出的胸怀、德行，也一定能感化世人。我们正在实践这一点。

有人说：中国历史上从来只有顺民和暴民。而法轮功学员既没有在强权下低眉顺目，也没有在不堪暴力时以恶治恶，这是中国历史上的一个特例。因为我们心中有伟大的法轮大法！

亲爱的父亲，您也是一个正直的人，请您相信：乌云不会遮住天，春天总会来临。“你可以是一束光，去照亮黑暗；你也可以不是一束光，但是你需要做的只是选择，选择睁开眼睛，迎接光明的到来。”这句话说的真好！爸爸，您也睁开眼睛吧。

爸爸，我眷念我的家乡，梦中都会怀念那静静流淌的嘉陵江和那些朴实的乡邻。我也感恩养育我的故土和双亲。我为自己是一个中国人深感自豪，因为我的血脉里，同样注入了中华文化的理念：真诚、善良，知恩图报。不久的将来，当我们那个历尽沧桑而拥有美德的民族再度辉煌的时候；当真相大显、人归正道的时候，您会因为有我而自豪！——您的女儿，曾经是亿万个为这一天的到来而付出努力的好人中的一员！

您甚至可以感到荣耀：因为我们做了一件许多有良知的中国人想做、却没有做到的事情。没有暴力、恶言，只有平和、善良，却改写了历史。如果没有信念的坚守，没有上天的看护，这是不可能的。我们不想做英雄豪杰，但真、善、忍信仰的力量让我们慈悲和无畏，心中光明坦荡。

未来新世纪的曙光，一定属于好人，属于善良！

爸爸，请用您一生拥有的正直和从不虚伪，在心底默默的支持正义、支持我吧！上天将赐福于您！愿您和母亲多保重！

女儿
二零二一年三月

# 迫害法轮功　四川绵阳地区恶报12例

【明慧网二零二一年五月二十二日】（明慧网通讯员四川综合报道）自从中共江泽民集团发动对法轮功的全面迫害后，四川绵阳地区政法部门的人员为了捞取政绩，在中共胁迫下，在名利的驱使下，参与迫害法轮功学员，有的人可能得到了一点黑钱，仕途得到了升迁。可人做了什么，上天都有记载，都得在什么时候偿还。

### 1、绵阳梓潼县公安局国保大队胡世然遭恶报死亡

绵阳梓潼县公安局国保大队胡世然，二零一六年腊月二十九日遭恶报去世。胡世然于二零零五年五月非法抓捕梓潼县法轮功学员夏全富及妻子谢某，并非法关押一个月，后谢某离世。

### 2、三台县公安局副局长谢贤武遭车祸死亡

绵阳地区三台县公安局副局长谢贤武，从迫害法轮功开始，紧跟江氏之流对法轮功学员进行登记、监控、抄家，又对多名法轮功学员非法判刑、劳教等迫害，只要是法轮功学员的案子交到他那里他还会加重处理，他的这些恶行给法轮功学员和他们的家人带来了无法弥补的伤害。二零零五年十一月谢贤武驾车去绵阳途中翻车死亡，死相极惨。

# 达尔文拱门崩塌　最新研究挑战进化论

【明慧网二零二一年五月二十二日】（明慧记者玉紫菱报道）二零二一年五月十七日，南美洲国家厄瓜多尔环境部通过社交媒体脸书（Facebook）公布，加拉帕戈斯群岛（Galapagos Archipelago）地标“达尔文拱门”崩塌。照片显示，原本弯曲的拱状岩石已经倒塌落入海中，仅剩下两根柱子。

![image](https://user-images.githubusercontent.com/79625284/119224380-b8e87b00-bb30-11eb-9661-bf816a2cbac5.png)

图：二零二一年五月十七日，厄瓜多尔环境部公布：“达尔文拱门”崩塌。

同日，旅游公司Aggressor Adventures也在脸书发文表示，他们的旅客当天经历了一生难遇的惊人事件，在当地时间上午十一时二十分，达尔文拱门在他们眼前崩塌。众人在事发后合照。

达尔文拱门距离达尔文岛不到一公里，而这个岛屿是达尔文假想进化论的起源地。进化论不断被科学家们指出漏洞百出。如今拱门崩塌，被认为“冥冥中似有天意”。

最新一期《科学》杂志五月七日发表文章指出，自达尔文进化论以来的关于人类起源的大多数解释都与化石记录不一致。

这篇文章的主要作者、美国自然历史博物馆人类学部门高级科学家塞尔吉奥·阿尔梅西亚（Sergio Almécija）在声明中表示：“当你阅读人类起源的叙述时，它简直就是一片混乱，没有任何共识。”

另一位作者、美国自然历史博物馆人类学部门助理馆长阿什莉·哈蒙德（Ashley Hammond）说：“当我们把所有的证据考虑在内，也就是现存的和化石的猿类和类人猿，我们很明显发现，以现存的少数猿类物种为理论依据的人类进化学说，实在是一孔之见。”

当初达尔文提出进化论时，只是作为一种假说，并没有提供确凿证据。事实上，能支持进化论的考古证据从未被发现过，人类从未发现过任何两个物种中间过渡状态的生物化石，人类文明史中也从未出现过所谓进化的中间状态的人或动物。相反，科学家们发现的大量考古证据足以推翻进化论。

近一个世纪以来，在世界各地发现的许多史前生物化石和具有高超智慧的史前人类文明遗迹，其范围之大、分布之广、年代之久远，完全打破了进化论的时间框架，是进化论根本无法解释的。

例如，一九八四年七月，在云南省澄江县发现了大量寒武纪（距今约5.3亿年）多门类生物化石群后，更多科学家加入否定进化论的行列。达尔文在书中就胆胆突突地提到过，复杂动物在寒武纪突然出现是对进化论的巨大挑战。让达尔文心虚的寒武纪动物突然大量出现的现象，在学术界被论定为“寒武纪生命大爆发”，很多科学家认为这种现象直接推翻了进化论。

《考古学禁区》（Forbidden Archeology）一书，由考古学家迈克尔·克莱默和理查德·汤姆森（Michael A.Cremo & Richard Thompson）所著。书中列举了五百个证据，几万年，几百万年，甚至于几亿年前人类文明的遗迹，许多都有着高度发达的科技和艺术水平。个个事例都足以否定进化论。

美国加州大学伯克利分校法学教授菲利普·约翰逊（Phillip Johnson）曾任美国大法官华伦（Warren）的助手，他的专长是分析明辨律师在辩论时所用的措辞和逻辑。当他读到进化论的文献时，马上发现里面充斥大量逻辑问题。他以法官的身份，多次质问：“确凿的证据何在？”

约翰逊在自己的著作《审判达尔文》（Darwin On Trial）中列举大量事实，从逻辑的角度说明进化论是错误的。书中指出：“化石向我们展示的都是突然出现的有机体，没有逐步进化的任何迹象……这些有机体一旦出现，基本上就不再变了，哪怕过了几百万年，不管气候环境如何变化。如果达尔文的理论成立，这些条件本应该引起物种的巨大变化。”

英国著名古文明研究专家葛瑞姆·汉卡克（Graham Hancock）所著《上帝的指纹》一书，讲述了考古学家经过大量实地考察的科学发现。作者踏遍中南美和埃及的古老土地，以一种全新而大胆的视角回溯人类文明史。汉卡克称世界各古老文明的传说及史前文化呼唤着他去解开人类从何而来之谜，以及古老文明为何被毁灭。

二十世纪末，多位世界著名考古学家编撰出版了《世界伟大考古纪实报告》系列丛书。该丛书通过《水晶头骨之谜》、《上帝的指纹》、《破译圣经》、《废墟的真相》、《众神的宫殿》等分册图文并茂的展现了世界考古工作的重大发现。书中揭示，在多少亿万年前，都存在着连现代人都无法望其项背的超级文明。

当越来越多的无可辩驳的证据公开后，进化论被批判为荒唐至极。新西兰遗传学家迈克尔·丹顿（Michael Denton）在其所著的《出现危机的理论：进化论》一书中直截了当的指出：“达尔文的进化论是二十世纪最大的谎言。”

最神奇的是，那年老家发了一场百年不遇的大洪水，山体滑坡，院墙被冲倒，当时正赶在晚上，大水涌進屋里，水位跟炕沿一般高。院子里水齐腰深，堂哥和堂嫂没害怕，顺利撤出，房子也没冲倒。大水过后，周围好几户人家房子被洪水损害的不能住了，都搬走了，堂哥的房子清理完了啥事没有。他家日子比以前过的更好了。

### 表姐的胃穿孔神奇的好了

表姐是药篓子，常年吃药，浑身没劲。表姐夫种地、放蚕，收入的钱几乎都给表姐治病了。有一年，我特意去她家讲法轮功真相和劝三退，两口子非常相信我讲的，都做了三退。每次去时，表姐都扯着我的手说：“你人厚道，有福。你看我，这一身病，啥时是个头？”我说：“你就相信法轮大法好，没有过不去的坎。”

虽然我的手头不宽绰，她的身体不好，我每次去看望她都会给她买点苹果和香蕉什么的，一点心意吧。

有一年，表姐忽然昏迷不醒，住進了县医院。一检查，是胃穿孔，赶紧输血。医生会诊后对姐夫和孩子说，穿孔有鸡蛋大小，腹腔大面积积血，心脏还不好，怕下不了手术台。建议家人马上把表姐送到省陆军总院。孩子赶紧张罗钱，准备送表姐去省医院。

去医院的当天晚上，表姐夫一夜未睡。别看表姐是病秧子，也是家里的顶梁柱，没了她，老实巴交的表姐夫等于塌了半个天。从表姐一出家门，表姐夫就开始念：“法轮大法好！真善忍好！求李老师保佑！”到了省陆军总院后，专家给表姐会诊，结论是：“不敢做手术，开点药，回家养吧。”儿子傻眼了，回家“养”着？那不是等死吗？可医院拒收，只能把人拉回家。

表姐病情严重，但一路上意识清醒，想起我告诉她的话，就在心里一直默念：“法轮大法好，真善忍好”。表姐夫也从未停止的念。后来听表姐说没想到的是，那天一進家门她就觉的病情有些缓解。

我给了表姐一个装有师父讲法录音的播放器，让表姐有空就听师父讲法。每次播放，他们夫妻俩都正襟危坐，恭恭敬敬的听。渐渐的，表姐脸上有了血色，能喝点粥了，接着能下地了，一天比一天好。也就两个多月吧，她的身体恢复正常，只是有点虚。村里人到她家看望时，表姐就跟他们说：“是大法师父救了我，要不就交代了。”

村里人都知道表姐底细，也惊奇她的变化，不少人都相信法轮大法好。

我去表姐家时，只要有真相小册子，我都给她几份，我说：“你看完后别扔了，放到村里的小商店里或谁家的窗台上，让别人看看也受益。”表姐夫痛快的答应了。

有一次，表姐夫说去小店逛逛，遇到一个熟人。这个村民对他说，早晨在窗台上捡到一本法轮功小册子，要扔还没扔呢。表姐夫说：“看看有好处，有病的就不犯病了。”村里人知道表姐是在大法中受益的，对小册子也格外留心起来，看完后还会送别人看。有时我跟表姐说：“大法这么好，真相资料谁看谁受益。老百姓就是叫恶党吓唬的不敢看。有多少人被共产党害了。”

十多年过去了，当初被医院诊断不治只能等死的表姐，如今七十多岁了，身体一直很好。表姐夫种地、养猪、打点零工，每年收入不再花冤枉钱给表姐看病了，都用在家庭生活上。老俩口和和顺顺，也给孩子减轻不少负担。

表姐和姐夫从心里感恩大法师父的洪恩。时间长了我不去她家，他们就给我打个电话，说惦记我了。

### 九十八岁的老姑历险

老姑今年九十八岁了。老人没上过学，一辈子吃苦，不会说不会道。我父亲在世时常跟我说：“你老姑受气，孩子不省心，你要经常去看看她。”老姑每次见到我，都眼泪巴巴的，说：“我命苦呀，兄弟姐妹几个就剩下我了，就我命不济。”我说：“老姑，您的命最好，我爸和大姑、二姑都走了，您能健康活着，这不是福嘛，咋说命不好呢？！”老姑说：“我受一辈子累，嘴也笨，尽受气。”我说：“吃苦也是福，您这辈的人，就剩下您自己了，这是让您等着听到法轮大法的福音呀，这福份多大呀，您应该高兴呀！”每次去，我都跟她这样说。

老姑很珍惜我给她的大法真相护身符，经常念“法轮大法好”，她最愿意听我讲的相信大法好得福报的故事，听得津津有味，还问我：“光念这句话就管用？”我说：“这可不是平常话，是神佛让人念的，威力大着呢，您就真心念吧，肯定应验。”

有一次，我進城办事，路过老姑家去看她，她隔着窗户看见我去了，赶紧下地迎我，说：“就盼你来呢！你看看我，现在多精神？”我打量她，的确比以前精神多了。我问：“您每天都念那句话吗？”老姑说：“念，每天都念！‘法轮大法好，真善忍好’，对吧？”我说：“对！对！”她又说：“别看老姑年老，身体硬朗着呢，就是耳朵背点儿，眼不花，背也不驼，能吃能喝，心里亮堂多了。”

老姑住在儿子家，条件并不好，特别是儿媳动不动就给她甩脸子，老姑没少生闷气。她能活九十多岁，用她的话说：“是老天照顾。”可我清楚，生命都是为了大法来的，人来到世上，不管多有钱，转眼也就几十年，啥能留下？只有接受大法真相的人，那才是最聪明的人，才有大福报。

有一次，我去老姑家，她给我说了她的一次历险：有天上午，儿子和儿媳都下地干活去了，就她一人在家，她想到街上走走。她家大门柱的木头烂了，一推大门，“轰”的一下，大门倒了，当即把她压在大门底下，那门是铁的，有上百斤重。

老姑说：“开始我喊‘救命！’没人应，邻居都干活去了，门压在身上推不开，蹬不动，我忽然想起来‘法轮大法好’，就开始念，念着念着，就有邻居过来了，赶忙把我扶起来，又给儿子打电话。儿子吓得够呛，把我拉到县医院，全身查了一遍，哪都没伤，就是脚上破了点儿皮，买了两袋跌打损伤药就回家了。你说说，我这么单薄，又是小脚，要不信大法，还不压扁了？”

老姑说着说着，眼泪就下来了。我常听说，上了岁数的人，走路怕摔倒，不是骨折，就是卧床不起，大限将至，这例子太多了。老姑被百斤重的大铁门压在身上，啥事也没有，真正见证了大法的神奇！

### 表弟：“法轮大法好！”

小表弟年轻时偷鸡摸狗，打架斗殴啥都干。有一年，遭了车祸，落下残疾，媳妇也走了。我给他讲大法真相，讲善恶有报，他相信，说自己落得这步田地，纯粹是自己折腾的。他接受三退，常念“法轮大法好”，日子越来越顺。

我没事时给他打电话聊聊天，知道他生活非比从前，他说现在再也不干那下道事儿了。这几年用小三轮拉水果卖，什么下来卖什么，一年不少挣，生活过得满滋润的。年前，又组织了新家庭，夫唱妇随，又开起了小餐馆。

前些日子，我问他生意怎样？他说：“挺好！有时遇到不顺心的事，我一念‘法轮大法好’，心情就好了。”

这些年，我在讲大法真相过程中，还看到一个现象：有的人你跟他讲大法真相能接受，有人一听说直摇头。我曾想过：不听也不信大法的，都是被恶党造谣毒害的，这是主要原因，更重要一点可能是我讲真相时没考虑对方的感受，自己知道恶党在害人，但心急却没说清，让对方误会。

曾有这样一个教训：一个熟人的孩子升学，他的母亲找到我，说：“你帮孩子办个共青团的关系。”我说：“现在都在三退保平安，你办那干啥？快退掉吧！”说时有点急躁，口气也不善。孩子的母亲瞪大眼睛看着我，满脸不解。在她看来，“这是关系到孩子前途的大事，你怎还这么说？”当时，如果我能站在她的角度上想，不急躁，把道理说透，她会同意的。可我没那样做，直往前拱，我越这样说，她越反感。我一着急，大声说：“你这不等于给自己孩子兜里揣个炸弹吗？”这句话让她更不能接受了，后来她跟不少人说我不希望她家孩子好，我是在反党。这事传到我妻子耳朵里，妻子给我好顿修理。当我再给那个孩子的母亲讲真相时，她嘴上哼哈着，心里很抵触。我为此后悔了很久，本来我是为她孩子好，却被误解了，使她对大法产生抵触。

后来，我听说她家遭水灾，搬進城里了，她又得了糖尿病，挺重的，眼睛也不好使，日子紧巴，生活挺困难，菜都舍不得买。于是，我拔了一些自家园子里的菜，去城里看她，又借机给她讲大法真相。这一次她很感动，一再谢我。我说：“你就诚心念‘法轮大法好，真善忍好！’一定会管用的。”

我又把孩子转团员关系那件事向她道歉，给她讲大法基本真相，和大法在世界洪传盛况等，她很高兴，痛快的退了团、队。

在我老家的农村，方圆几十里就我一人修大法。尽管我一直在讲真相和劝三退，总的说来数量少，我能做的，也就是给自己的亲戚、族人、朋友、同事等这些人讲大法好和劝退。只要师父正法没结束，我会继续不停的做下去。愿更多的乡亲们能明真相得福报，拥有美好的未来。

谢谢师父！
