### 从锦州学员家属被诬判 看中共对人性的摧残

文:大陆大法弟子 

【明慧网二零二二年四月八日】近日，辽宁锦州市法轮功学员许清芳未修炼的女儿和侄子，因去派出所询问情况和保护法轮功书籍、物品，与她一同被诬判。震惊之余，更加看清了中共的邪恶与残暴，及其党徒的张狂。

#### 一、事件回放

二零二一年十二月十二日，辽宁锦州市法轮功学员许清芳在早市发明慧台历时，被锦州市公安局太和分局国保大队的便衣警察绑架到锦州市大薛派出所。之后，许清芳未修炼的女儿孟涛听说后，赶到派出所询问情况；许清芳未修炼的侄子许驰听到消息后，也来到派出所与警察交涉，未果。接下来许驰便赶到姑姑许清芳的住处，收拾姑姑的法轮功书籍、物品，随即，他被跟踪而来的警察扣押。许清芳的女儿孟涛也被扣押。十二月二十日，孟涛和许池被以“销毁证据罪”非法批捕，而许清芳也被批捕。二零二二年三月，三人被同时非法判刑：许清芳被诬判五年；孟涛和许池均被诬判六个月。至此，一家三人一同落难。

#### 二、当事人的善良本性

熟悉许清芳女儿孟涛的人们都知道，孟涛在真善忍理念的熏陶下，为人善良，善解人意，在家里，她是好妻子、好母亲；在单位，她是领导和同事都放心的好员工；在住宅小区，她是一个人人称赞的好姑娘。提起孟涛，那是人人赞许。而许清芳的侄子许驰，也是一个为人正直的好青年，他虽说未修炼法轮功，但他知道姑姑修炼法轮大法没有错，法轮功是冤枉的，天安门自焚是假的，姑姑修炼法轮功，身心受益了，他愿意姑姑修炼法轮功，在道义理念的引领下，去姑姑家收拾法轮功书籍、物品，他没有想到警察会跟踪他，最后将他绑架。

信仰自由是天赋人权，法轮功学员被诬判都是冤案，特别在这个迫害案例中锦州太和警察还株连了法轮功学员不修炼的家人，也让他们与亲人一起蒙羞受难，实在是丧失了道德底线，这是中共残暴本性的再次展现。

亲情乃天经地义。当亲人被诬陷时，其他家庭成员或亲属挺身而出，或用力所能及的方式保护亲人，那是大义之举，更应该予以肯定和称赞。孟涛和许驰的所为就是这样的义举。这不但是善良人性的彰显，更是法轮功学员的家属在道德勇气的感召下，对亲人奋不顾身的庇护。

#### 三、中共对基本人性的摧残

在许清芳这个案例中，许清芳并没有犯罪。事实上，到现在为止，在中国大陆找不到任何一个可以迫害法轮功的“法律依据”。中国的任何一部法律都没有把法轮功说成是什么“邪教”。我们马上在网上搜索“中国政府认定的邪教组织”，找到公通字（2000）39号文件全文，会发现，其中明确认定的14种邪教中根本没有法轮功。

其实对法轮功学员抄家，收缴法轮功书籍和相关物品也是违法的。《国务院公报》2011年第28期刊登了2011年3月1日的《国务院新闻出版总署第50号文件》（请搜索：http://www.gov.cn/gongbao/content/2011/content_1960695.htm），  其中99项和100项表明中国新闻出版总署废除了对法轮功书籍的出版禁令，50号文件说明，法轮功书籍已被解禁，属于合法出版物。就象所有正常的出版物一样，法轮功的一切书籍、宣传品可以正常的出版、发行、使用、传播。

那么既然许清芳没有犯罪，她女儿和侄子也知道她没有犯罪，那么，孟涛去派出所打听母亲被抓情况、许驰去姑姑家把法轮功书籍收拾好，是违法行为吗？怎么能以“销毁证据罪”被判刑呢？这些书籍和资料都是教人做好人的，怎么能说是犯罪证据呢？里面的哪句话是教人干坏事的呢？这到底是谁在犯罪？众所周知，判断一个出版物是否合法，权威机构是国家新闻出版署，公安人员执法也得按照法律法规来执行呀。

许清芳与女儿、侄子三人同一天被抓，接下来又都被非法判刑，犹如晴天霹雳，这三个家庭瞬间被甩入低谷之中。这是何等的悲剧，何等的苦难？中共的以民为敌，在其对中国人基本人性的肆意摧残中，残忍狠毒地发挥着。

孟涛的同学同事、许驰的乡里乡亲都在为他们鸣不平。

从个体层面上说，许清芳及她的女儿、侄子被迫害的案例，起因是锦州太和公安分局新上任的局长吴亚凡和政委王超为了争取年度“办案数量第一”，指使手下人滥抓无辜，使这个分局的国保大队头目刘长杰和李蕾肆无忌惮地绑架法轮功学员造成的。他们把法轮功学员家属也非法抓捕并予以构陷判刑，凑数，要政绩，为了自己的官职名利不顾别人死活。

但吴亚凡、刘长杰、李蕾等人之所以这样残忍地欺压百姓没有底线，是因为他们背后有中共邪党给他们撑腰壮胆。李蕾甚至还用这个案子恐吓其他法轮功学员家属。

《九评共产党》一书中指出：“而共产党在破坏宗教的同时，破坏传统文化。它把传统、道德、人伦说成是封建、迷信、反动，以革命的名义而铲除。通过不间断的各种政治运动，中国共产党把人变成狼，甚至比虎狼更凶残。虎毒不食子，但在中共统治下，父母，子女，夫妻之间互相揭发，断绝亲属关系的比比皆是。传统文化中讲‘孝’，共产党可以把不赡养父母的人抓到监狱里去，但却是因为儿女不赡养父母，父母就成了政府的‘负担’，而共产党需要的时候，儿女还要和父母划清界限。”

许清芳的女儿和侄子，在真善忍理念的感召下，实现了人性的传统回归。这是中共害怕的，也是它容忍不了的。共产党容不得人心向善，它要的是全中国人民都与它保持一致，与它划定的所谓“敌人”划清界限，哪怕这所谓的“敌人”是至爱亲朋。

#### 结语

上天不会无视中共的邪恶。《九评共产党》引发的“三退”运动，迄今已经齐聚了全球3.9亿退党勇士的正义力量。没有中共的中国很快就会来临。

还望世人抓紧拜读奇书《九评共产党》，退出中共党团队，回归人性，回归传统。也希望锦州地区还在参与迫害法轮功的各级官员，从锦州地区官员遭恶报的实例中警醒。你们只有立刻停止迫害，保护法轮功学员，才能减轻你们的罪责，才能为你和你的家族留一条出路，为自己和家人赎回未来。
