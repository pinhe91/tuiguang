### 累遭十五年冤狱迫害　云南昆明王汇真女士含冤离世

【明慧网二零二一年十一月一日】（明慧网通讯员云南报道）昆明法轮功学员王汇真女士，因信仰真善忍，曾经多次被邪党绑架、抄家、关押，劳教、判刑三次，遭冤狱达十五年。王汇真第三次被关押在云南省第二女子监狱期间，受尽各种折磨，出现严重腹水；二零一七年出狱回家后又不断遭到西山区国保警察和西山区东陆派出所警察骚扰、绑架，情况越来越严重，于二零二一年五月含冤离世。
王汇真一九五七年出生，新华社云南分社下属新华广告公司职员。王汇真女士自一九九九年开始修炼法轮功，身心受益，所患的各种疾病消失，工作兢兢业业，在单位是大家称赞的好职工。

#### 一、被单位诬告非法劳教两年

可是一九九九年“七二零”中共江泽民集团开始迫害法轮功后，王汇真只因坚持对真、善、忍的信仰，向民众讲述法轮功被迫害的真相，在单位复印真相信，被办公室主任恶告。

二零零一年五月三十日下午，昆明市公安局政保科警察，伙同单位不法人员，对王汇真非法审讯并抄家，随后将其劫持到昆明市第一看守所关押，非法关押三十七天后，被昆明市劳教委非法劳教二年。

二零零二年八月十八日，王汇真从劳教所回家后，单位不安排工作，还停了她的养老保险，从当年十月开始，每月只给她七百元的生活费。

#### 二、二次被非法判刑四年

二零零三年九月二日，王汇真在昆明市西华园公园向民众发真相资料，被公园管理人员构陷，被官渡区国保大队冯军等人绑架、抄家。当天晚上，直接把她送到昆明市第二看守所。

王汇真被非法关押八个多月，被昆明市中级法院非法判刑四年。二零零四年五月八日，王汇真被送到云南省第二女子监狱四监区关押，二零零六年十二月一日，王汇真回家。回来后，王汇真再次向单位写了要求补办养老保险和要求安排工作的申请，依然没有回复。

二零零八年四月二十二日，王汇真在小区发真相资料，被门卫诬告，被西山区国保大队和永昌派出所警察绑架、抄家，抄家时，没有搜查证，抢走的物品也没有清单。

随后，王汇真被昆明中级法院再次非法判刑四年。王汇真被关押在云南省第二女子监狱九监区期间，被强迫坐小凳子折磨三个月，导致腿肿，臀部坐烂，血压升高。

#### 三、再次被非法判刑五年，被迫害致“严重腹水”

二零一二年五月四、五日两天，中共云南国保警察在全省多个州、市统一对数十名法轮功学员大抓捕。

二零一二年五月五日早上八点，昆明市西山区国保大队警察温永祥突然闯入王汇真家中，同时控制了正在敲王汇真家门的法轮功学员蔡文慧。

随后，赶到的有西山区国保大队队长邱学彦，还有警察王中芳、张为民、贺勇，还有不知名的四个辅警，其中一个是西华派出所的，一共九人，都是男的。这些人不亮身份和搜查证，就动手非法抄家。从家中抢走激光打印机四台、彩喷机二台、电脑、刻录机、碎纸机、热胶机、过塑机、切纸机、定装机各一台，手机一部，各种真相资料及耗材，满满一车，价值二万八千元左右。

二零一二年九月二十九日，昆明市检察院对王汇真和蔡文慧非法起诉，代理检察员许龄心。

之后二零一二年十二月二十八日，王汇真被非法判刑五年，蔡文慧三年，审判长：杨捷，代理审判员：李世超、李兴虎，书记员：金坤。

二零一三年五月二十日，王汇真和蔡文慧被劫持到云南省第二女子监狱，关押在九监区（集训队）的专管组（专门迫害法轮功的组），迫害王汇真的主管警察是彭舒，派犯人张玉香和杨云建做王汇真的包夹，寸步不离地监视王汇真，连夜间起夜，都必须叫一个包夹犯人跟着。

王汇真在被非法关押在九监区期间，每天被强迫从早上七点到晚上十一点三十分坐小板凳。每天只能在规定时间上厕所。严管期，不给洗澡，每星期只提供一桶冷水，一个热水瓶的热水，在监室里脱光衣服擦身上，再用擦身上的水来洗衣服、洗鞋。

同时严管期，还限制一切行动，购物是最低标准，不能买食品，还要写申请（其他犯人不用写），必须写：“自己因犯利用××组织破坏法律实施罪入狱”否则不让购买生活用品。由于残酷迫害致使王汇真血压升高，出血头晕，臀部红肿，双下肢浮肿，随后还出现了严重腹水。

二零一七年，王汇真被释放回家时，依然有严重腹水。

#### 四、不断的绑架、骚扰、恐吓

二零一七年五月，王汇真出狱回家后，西山区国保大队、东陆派出所、西坝北社区等不法人员不顾王汇真腹水日渐加重（如怀胎十个月），还经常上门恐吓、骚扰。

二零一九年十月一日前，昆明市公安机关对全市法轮功学员大规模骚扰、绑架。九月二十五日下午三点多，西坝北社区警察夏黔山突然闯进王汇真家骚扰，当时王汇真家中，有六位法轮功学员正在学法（一起读李洪志师父著作）。夏黔山看到这个情况后，就立即通知西山区国保大队、东陆派出所、西坝北社区。

不一会儿，一大帮人赶到王汇真家，有西坝北社区主任卢州、白惠泉，东陆派出所所长艾雄及两女三男警察，非法抄家，抄走李洪志师父法像一张、李洪志师父著作近百本、播放机一台和其它一些私人物品。

警察抄家后，将王汇真在内的六位法轮功学员，年龄最大的八十二岁，最年轻的也有六十二岁，带到东陆派出所非法审讯、做笔录。之后，将王汇真与另一位法轮功学员张良（男，六十七岁，见明慧网《昆明张良自述遭十年冤狱迫害经历》）带到圣约翰医院体检，做B超。王汇真当时已经有严重腹水，走路都要人搀扶，一直被折腾到凌晨。

随后，警察将七十岁以上的四位法轮功学员送回家，将王汇真和张良两人送到安宁市太平拘留所。王汇真和张良（高血压）的身体都不适合关押，拘留所拒绝收押。最后，折腾到第二天，警察才将两人释放回家。

由于东陆派出所警察和西坝北社区不法人员不断骚扰，致使王汇真腹水日渐加重。二零二一年五月，王汇真在家中含冤离世。

王汇真女士前期被迫害的经历，详见明慧网二零一一年十二月十六日《新华社云南分社职员王汇真遭受的迫害》、《遭15年冤狱　昆明王汇真出狱后仍被抄家骚扰》。
