#### 老汉说“九字真言”救了自己

文：中国大陆大法弟子

二零二一年十月的一天上午，我出去倒垃圾碰到一个人拄着棍子在走路，有时还要跺跺脚，面色很难看。

我一看这是我以前帮他做过“三退”的人，就问他：“你怎么了？”他说：“我得脑血栓了，现在又得了肾结石。”我说：“你没去医院吗？”他说：“去了，医生说石头太小打不了，叫我回家呆几天再去治。可是我好几天都不能睡觉，疼得不行。”我说：“你没有念‘法轮大法好，真善忍好’吗？”他说：“念了，不管用。”听他的口气好像他并不是很相信，我说：“你今天只念一遍，明天也只念一遍，那能好用吗？”他说：“那怎么办？”我让他在那等着，我就回家拿了个护身符送给他，我说：“照这上面的字念，尽量多念，心诚则灵。”

过了二十天左右，我看到他站在那边笑眯眯的看着我，我就走到他跟前问他：“你好了？”他说：“好了，这‘九字真言’救了我！”他接着说：“那天你告诉我后，我没有回家，拿着护身符就在村子里转了一大圈，转了有三个多小时，嘴里不停的念‘法轮大法好，真善忍好’。快到家的时候就觉的不怎么疼了。到家后真的就不疼了。大姐，谢谢你！”我说：“你别谢我，你得谢谢法轮大法师父！师父看你这次是真心实意念相信了，就给你治好了。”

我又接着说：“你家不总有二、三桌打扑克的人吗，你回去把念九字真言救了你的事告诉大家，你这个拄棍也会扔掉的。”

一个星期后，我看到他真的把拄棍扔掉了。

#### 念“法轮大法好”眼睛好了

文：中国大陆大法弟子

今年四月一天，陈三（小名）见到我，掩不住内心的喜悦的对我说：“你叫我念‘法轮大法好’，我的眼睛能看见了！”接着说：“前晚我洗了一个澡，洗完走出来，突然觉的我的眼睛能看见东西了。再仔细一看，不是幻觉，真的看见东西了。我好高兴啊！我能大大的跨步走路了，我好久没有这样走路了。今天赶集，我特意来告诉你。”

我也为他高兴。

陈三是一位喜欢听大法真相的老人，还常跟我要真相资料。半个多月前我碰到他，和他打招呼，他说他的眼睛看不见东西了，心里很苦闷。我告诉他，诚心敬念“法轮大法好，真善忍好”，一定会有奇迹出现。

陈三按照我告诉他的做了，每天坚持念，出声念，奇迹就在他身上发生了。

我在为陈三高兴的同时，就把发生在他身上的奇迹告诉身边亲朋好友及听过我讲真相的人们，让众生都能相信师尊的无量慈悲与大法的法力是真实不虚的。
