### 掉到大水沟底　念九字真言绝处逢生

文: 中国大陆大法弟子家属 更新: 2022年09月01日

【明慧网二零二二年九月一日】我是湖南岳阳农村的一名普通农民，靠给人打水井和给百年后的人扎灵屋子等为生。
我母亲七十多岁，修炼法轮大法二十多年了。我家里有个学法小组，不管迫害怎样严酷，母亲和很多法轮功学员都一如既往地在一起学法。母亲经常叮嘱我：你经常开车出外挣钱，一定要多诚心敬念“法轮大法好，真善忍好”九字真言，能遇难呈祥，常念得福报。我也很支持并相信大法，但我总是信而不坚，偶尔念念。

二零二二年三月二十九日，我开车到邻村去打水井，车上装满材料和工具，当车开到一拐弯处倒车时，因刹车失灵，连人带车掉到约四米深的排渠大水沟底。在我感到万分无助的时候，我想起了母亲的叮嘱，从心底诚心地大喊“法轮大法好，真善忍好”，求李大师保护我。
当时巨大的响声惊动了周围的人，众人都纷纷跑过来，只听他们都说：完了、完了。大家七手八脚地将我从沟里抬上来，发现我身体没受一点伤、人也很清醒，都很惊喜。然后又帮我联系吊车，将车从沟底吊上来，将材料和工具捞上来。

我检查了一下车也没坏，众人都说：你家真是烧高香了，有神佛保佑。但我心里知道是法轮大法师父救了我。我谢过他们，然后就开车回家了。回家后我赶快到李大师法像前烧香跪拜谢恩！

我虽然没走入大法修炼，但我也会在心里记住“真、善、忍”三个字，做一个好人。愿世人都能诚心敬念九字真言，都能平安。
