### 借口二十大　多省政法委操控不法人员迫害法轮功

【明慧网二零二二年十月二日】（明慧网通讯员大陆报道）进入二零二二年九月，中国大陆多个省份的政法委人员操控派出所警察、某些单位的职能人员、村干部等，以各种方式闯入法轮功学员家非法抄家、抢劫，有的打电话骚扰，威胁法轮功学员不要外出等，并非法强行照相、录像等，其借口是中共邪党要开“十二大”。正像一位被骚扰的法轮功学员的妻子说：“‘二十大’跟我们有啥关系啊？！”

#### 河北省唐山市法轮功学员王克泉遭到警察骚扰

九月三十日下午四点左右，钢城分局四个人来到法轮功学员王克泉家，当时王克泉没有在家，他们要求进家里。他们进到家里之后，先自报身份，并给家人看了警察证。警察跟他妻子说：告诉他，现在要开“二十大”了，要他不要外出。
王克泉的妻子说：他妈妈在卧床，他在照顾他妈妈。“二十大”跟我们有啥关系啊？！钢厂不是给他开除了吗？怎么还要管他？！不给他安排工作，不给开支，这个时候找来了。

年龄大的警察说，工作的事跟我说没用，你去找公司。我只是来完成上面交给我的事。

#### 重庆市沙坪坝双碑地区十几位法轮功学员遭骚扰、抄家和绑架迫害

法轮功学员曾志君，女，八十二岁，九月二十日下午三点被骚扰。晚九点半被五、六个警察在曾志君家非法抄家二十分钟，警察非法抄走几十本大法书，连墙上的大法真相的画，也被抢走。后曾志君被绑架到派出所，半夜十一点半，被放回家。参与的警察有个叫熊其，社区书记是女的，姓夏。

九月二十日下午三点，法轮功学员李继兰不在家，丈夫在家，五个警察闯入家中，非法抄走九本大法书。当晚八点，警察将李继兰绑架到井口派出所。九月二十一日凌晨一点半，李继兰被放回家。

另外，还有许多法轮功学员遭骚扰、抄家。

这次对沙坪坝双碑法轮功学员大面积的骚扰、抄家等犯法行为，其借口说是邪党要开“二十大”。

◎二零二二年九月二十一日，警察和工作单位领导到法轮功学员汪文英家中骚扰，并强迫给汪文英照了相。

◎赤壁市法轮功学员老杨被警察非法抄家。据悉，多名警察突然闯入老杨家中，将老杨家中的大法书籍及电脑打印机等私人物品非法抄走。

◎赤壁市法轮功学员张占英被警察骚扰，并在笔录上签字。

◎赤壁市法轮功学员余春学，二零一六年三月十一日，在东莞市粘贴真相资料，被绑架，被东莞市第一法院非法判刑七年三个月。余春学现在在赤壁，被迫害得住在医院，全身不能动，无法自理，目前由家人护理。

#### 黑龙江省齐齐哈尔市富裕县派出所警察骚扰当地法轮功学员

近期，富裕县各派出所警察给多名法轮功学员打骚扰电话，说要开“二十大”了，这些日子别走，过这些天就好了，并让拍照，发个定位等等。参与骚扰的有郭姓警察（15147604567）、孙姓警察（15174609234）。

#### 山东省潍坊市寿光市大面积骚扰法轮功学员

二零二二年中国新年前后，山东省潍坊市寿光市的各派出所对所辖区内的部份法轮功学员骚扰，说是见见面，照个像，应付工作。也有的打电话问还炼不炼。

进入九月份以来，各派出所、街道、社区对法轮功学员进行了大面积骚扰。先是派出所警察打电话询问，有的到家中问还炼不炼。有的派出所警察打电话要求见面，被学员拒绝后，警察找到社区或街道人员强行到家中照相，再被学员拒绝后，只好在门口照相，应付回去交差。

#### 吉林省临江市多名法轮功学员及家人被警察骚扰

近期，吉林省临江市多名法轮功学员及家人被警察骚扰，就是警察对自己辖区的“重点迫害的”法轮功学员拍照，摸清个人信息。警察骚扰的手机电话：17804399562。

#### 吉林省吉林市昌邑区多名法轮功学员被绑架和抄家

九月二十七日，吉林省吉林市昌邑区多名法轮功学员被绑架和抄家。已知在铁路住宅十一号楼张姓女法轮功学员和来家中做客叫“小丽”的女法轮功学员被绑架，并被非法抄家，警察抄走许多大法书，电脑。她们被非法“取保候审”后，晚上才回家。

在麻棉小区居住的七十七岁男法轮功学员赵旭东和来看望的女儿的法轮功学员赵国坤被绑架并抄家，警察抄走许多大法书，电脑和打印机。赵国坤被非法“取保候审”后，晚上回家。

九月二十九日，赵国坤被绑架到舒兰市看守所。有目击者证实，这次绑架和抄家是昌邑区一次有预谋的迫害行动，还有多名法轮功学员被绑架，具体情况待收集和反馈。

#### 河北邢台市多位法轮功学员被骚扰

近一段时间以来，邢台市多个派出所警察，有给法轮功学员打电话的、有上门骚扰的，有让填表签字的，有给拍照的，骚扰多个法轮功学员。

#### 四川省眉山市东坡区多位法轮功学员或家属被骚扰

九月下旬，由眉山市东坡区防邪办、政法委、街道办、社区或乡镇派出所、村委多部门人员上门要法轮功学员签字（“清零”），威胁说不签字就办“学习班”（即洗脑班）；或给法轮功学员家属打电话，让其录制法轮功学员在日常生活中的录像发给他们。有的家属悄悄地给录了像。具体情况待查。

#### 近期成都市政法委人员骚扰迫害多名天府新区法轮功学员

四川省成都市政法委近期在天府新区骚扰迫害多名法轮功学员，据说，华阳地区很多法轮功学员被干扰。市，区政法委，街道，社区人员一起找到法轮功学员恐吓，欺骗，要求签字，不签就威胁送洗脑班。

九月二十七日，市、区政法委三人、新兴镇孔雀村佘世林等人，骚扰法轮功学员吕升云一家。

法轮功也叫法轮大法，是上乘的佛家修炼大法，于一九九二年由李洪志师父传出，他以宇宙最高特性真、善、忍为原则指导人修炼，辅以简单优美的五套功法，可以使修炼人在极短的时间内达到身心净化，道德回升。

中共开“二十大”与法轮功学员没有任何关系。相反，修炼法轮大法福益家庭、社会，是应该受到表彰的。法轮大法在中国是合法的，法轮功学员的个人信仰自由、人身自由和他们的财产是受宪法与法律保护的。

二十三年来，中共对法轮功学员的持续迫害，违反良知、道德和现行法律，是江氏集团利用国力资源迫害中国最善良的人——法轮功学员，到今天，骑虎难下，越加害怕善良人，以其阴暗的心理，利用中共的“二十大”制造借口，继续迫害法轮功学员。

善恶有报，在未来法制昌明之时，所有参与迫害法轮功学员的人都面临未来正义法庭审判和终身追责。
