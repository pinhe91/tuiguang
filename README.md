### 唐山的社会乱象是谁造成的

文: 中国大陆法轮功学员 

【明慧网二零二二年八月十四日】河北省唐山市的治安状况堪忧，从烧烤店打人事件火遍全国网络，余波尚未平息，又传出越野车辗压女友视频，接连撞破人性道德底线，造成非常恶劣的社会影响。据传，今年唐山境内的几所大学招生遇冷，情况非往年可比。
有家长调侃：“去唐山上学可能得自带护盾，不然下了火车没到学校就出事了。”有的家长说：“决不让孩子去唐山的任何一所大学！生命安全比事业前途更重要！”家长的心情可以理解，含辛茹苦养大的孩子，当然希望他有一个平安美好的未来，这倒让我想起另外一件事。

我认识一个年轻小伙，有很好的工作，曾经在唐山上过学，他跟我讲了这么一件事：在唐山上学期间，有一天走在街上，碰到一个老大爷，亲切地跟他叫“大外甥”！他心里一惊，想：这人生地不熟的，怎么有人认识我？紧接着，这位老大爷就跟他法轮功真相、中共为什么迫害法轮功，以及退中共的一切组织，不与恶为伍，能保幸福平安等等。

我问他：“这老大爷说的，你相信吗？你爸妈可都是警察呀！”他说：“我还感觉挺好的，虽然有些东西一时不能完全接受，但是以后我可以慢慢去了解。老人很善良，很真诚，陪我走了一路，他说的那些对我没坏处，就是在心里面退出中共就行了，记住真、善、忍好，我同意真、善、忍，也不喜欢共产党。”

我就想，走在唐山的大街小巷，主动跟你搭讪的，不是熟人，多半就是推销的，他管你要手机号，再加个微信，目的是想赚你的钱。而有这么一群人，他们慈眉善目，和蔼可亲，他们或曾身患疾病，修炼法轮功后恢复了健康，所以他们碰到有缘人，就会真诚地告诉你记住“法轮大法好”、“真善忍好”，既不会加你微信，又不跟你要钱，还可能会免费送你一个精美的小挂坠，或者一本劝善的期刊或者书。我想，如果我是家长，我会非常乐意自己的孩子也能偶遇这么一个美好的瞬间。

然而，这样的善良人，却常常遭到警察的抓捕。从唐山打人事件事后，我关注到近期的法轮功明慧网上，有几篇关于唐山本地的报道。其中《唐山2019～2021年：三年五次绑架逾150人》提到，最近三年里，唐山地区发生了五次公安系统秘密策划的大规模绑架案。经过少则半月、多则半年的对法轮功学员跟踪、手机窃听、小区监控、录音、录像等，然后拟定名单，再出动大批警察统一行动，每次绑架少则几人，多则数十人，有些法轮功学员的家人也一同被绑架。

例如二零一九年七月六日凌晨三点，仅遵化一地就出动300多警察，按内定法轮功学员名单30多人，统一实施的绑架。这三年的五次绑架中，至少150名法轮功学员被非法抓捕。41人被非法判刑，刑期最短的6个月，最长的9年，被非法判刑的法轮功学员的平均年龄66岁，大多属于老年人，还有两位被迫害致死。

最具讽刺意义的，烧烤店打人事件发生后，公安部门开展了所谓的“百日行动”作为回应，唐山警察开始大面积骚扰修心向善的法轮功学员，敲门、录像、询问炼不炼功、逼迫签字等，多个派出所和社区人员都参与其中。还有至少十位法轮功学员被非法抄家，被抢走法轮功书籍等，有六位被非法抓捕，她们有的是钢厂的退休职员、高级工程师，还有曾在高校任职的教师，年龄大多在五、六十岁左右。（详情见《七月河北唐山地区警察绑架、骚扰多位法轮功学员》）

这些法轮功学员为什么被抓，大多是像前文提到的老大爷一样，在街上告诉人“真善忍好”，然后就被警察抓捕判刑了。从另一个角度讲，当这个社会不允许传扬真、善、忍的时候，必定滋养虚假、暴力和恶性争斗，以牙还牙，以恶制恶，长此以往，社会上积累了相当大的怨力。

作为河北省的一个经济强市，唐山毗邻京津，政治方面不甘“落后”，多年来紧随江泽民余毒迫害法轮功，使这片曾经多慷慨悲歌之士的燕赵大地，如今成为打击真、善、忍的重灾区，各种社会乱象的根源似乎就是这些执法犯法的公安警察。我这么说，您怎么看？试想一想：做好人遭迫害、讲真话遭迫害的社会，可不可怕？你愿意你的孩子生活在那样的社会吗？

对法轮功学员的打压意味着善恶颠倒、正邪不分，会导致社会道德沦丧、人心荒漠化。把一个“打不还手、骂不还口”的人逮捕关押，会让全社会的民众都迷茫司法机关为什么迫害好人；会進一步摧毁社会的道德价值观。中共江泽民集团对法轮功的迫害使中国的法治建设出现了大倒退，依法治国不仅成为口号，甚至成了玩笑，中国之所以出现今天如此之乱象，完全是由于这场对法轮功的迫害所引发。
