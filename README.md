### 原宁波市市长卢子跃遭报被判无期徒刑

【明慧网二零二二年八月三十日】原中共浙江省宁波市委副书记、市长卢子跃涉嫌严重违纪，受贿共计折合人民币1.47亿余元，于二零一六年六月二十一日被立案审查，二零一七年五月三十一日被判处无期徒刑，并处没收个人全部财产。
卢子跃简历：二零零一年，任义乌市委副书记、代市长，二零零五年四月，任台州市委常委、秘书长，二零零六年七月任台州市委常委、临海市委书记；二零零八年五月，任丽水市委副书记，市政府副市长；二零零九年一月出任市长；二零一一年三月升丽水市委书记。二零一三年三月，五十一岁的卢子跃升任浙江省政府副省长。同年，他调任宁波市委副书记、市长。

![image](https://user-images.githubusercontent.com/79625284/187393456-12abf14f-9043-4d6c-a21c-004265ef7175.png)

原中共浙江省宁波市委副书记、市长卢子跃被判无期徒刑

卢子跃同其他落马官员一样，在任临海市委书记、丽水市委书记及宁波市委副书记、市长期间曾在当地积极推行中共对法轮功的迫害政策，参与迫害法轮功，是中共迫害法轮功的重要帮凶。法轮功学员被非法判刑、在监狱中被酷刑折磨、被迫害致伤、致残、亲人所受精神折磨，卢子跃是负有领导和主使的责任，难辞其咎，罪责难逃。

#### 一、在台州临海市任市委书记期间（二零零六年七月至二零零八年四月），发生的迫害法轮功学员主要案例：

二零零七年五月二十一日，台州临海市吴素英、吴素美、吴素月、吴飞行、冯和平、王端善、王健伟、王丽丽、良维新等多名法轮功学员被邪恶绑架，吴素英、王莉莉被分别非法判处八年和四年徒刑；吴飞行被非法判处四年徒刑；王端善、王建伟被分别非法判处二年和三年。吴素美、吴素月被非法劳教一年六个月。审判长：朱胜。

1、二零一四年四月十八日，宁波余姚市王春燕、戚长星、姚红文、余志英、黄桂娣、陈珍和、邵红妮等法轮功学员在发真相资料时被恶警跟踪后绑架，都被非法抄家。抄走王春燕电脑、打印机等做真相的全部东西。据内部传出，已经盯梢跟踪很长时间了，蹲坑监视、跟踪、安装监控、电话窃听等手段进行迫害，这次绑架事件是有预谋的。

非法关押期间，七名法轮功学员被剥夺通信权，受到警察的哄骗恐吓以及身心折磨，恶警动用各种手段逼迫法轮功学员放弃信仰。其中陈珍和老伴在此期间悲愤离世，另有家属在余姚看守所给家人存钱近一万元，而黄桂娣称根本没有收到。

开庭之前，余姚市法院里外特警重重，便衣重重，黑车重重。门口警卫对参加出庭人员严格检查，家属被搜身，正义律师被要求翻查公文包，并有专人录像。但看出恶人心虚，有女法官从家属身边经过拿书挡脸。开庭时，庭内有大批便衣混充保安数名。

开庭时，法官多次打断法轮功学员发言，对其象死刑犯一样戴手铐脚镣，后经律师争取当庭解铐。下午，年龄最大的法轮功学员陈珍和（女）出现昏倒休克病态，法院仍旧强行让已无法说话的陈珍和躺在推床上出庭。

二零一五年十一月四日，宁波中级法院裁定，维持二审结果——姚红文、余志英、王春艳被非法秘判五年，黄桂娣四年半。过程中，无判决书及任何法律程序，只有所谓“裁定”。

2、二零一四年五月十四日，近七十岁宁波市法轮功学员戴翠英因在路上发破网软件，被宁波市西门派出所警察绑架。被冤判六年，上诉后，维持原判。在冤狱期间惨遭酷刑折磨，出狱时骨瘦如柴。

3、二零一四年五月十九日，舟山法轮功学员金华被宁波镇海公安分局国保大队长柴亚玲和庄市派出所警察徐征绑架抄家，抢走大法书籍电脑移动硬盘播放器手机等私人物品。于二零一四年十二月，被宁波镇海法院冤判三年六个月，后上诉，维持冤判。在监狱受过酷刑折磨，被单位开除，三十多年工龄被清零，养老金被剥夺。

4、二零一四年六月二十六日，宁波市法轮功学员秦莲玉和女儿洪洁静被绑架抄家，宁波市江北区“610”及洪塘派出所警察参与了此次绑架。在此事发生前10天左右，在小区内突然新加装了直对她们家楼门前位置的摄像头。母女被非法关押一个月后回家。

5、二零一四年七月十九日左右至八月十六日，宁波市610伙同国保、街道等不法人员，在宁海雄风山庄办洗脑班，先后在宁波市绑架了四人，在余姚市绑架了六人，非法关押迫害近一个月，期间不法人员用各种卑劣手段逼迫法轮功学员放弃修炼，甚至从省里找了个什么“转化高手”，此恶徒狂叫要把不“转化”的人立即扔监狱迫害。

6、二零一四年十一月二十五日14点，宁波七十多岁法轮功学员沈金凤遭镇海区公安分局九龙湖派出所警察绑架。当天晚上，一帮警察押着沈金凤，到家里非法抄家，抢走大法师父法像、播放器等私人物品。晚上，沈金凤被非法关押在派出所，整夜不让睡觉。第二天又非法关进宁波北仑看守所，非法关押九个月后，被宁波镇海区法院冤判三年。在冤狱期间遭酷刑虐待，派了三个包夹，逼迫看造谣污蔑法轮功的材料，逼着抄写他们写好的材料。

7、二零一五年四月十五日，宁波市镇海区公安局国保柴亚玲、严凯等，把法轮功学员杨庆华非法关押到康宁医院（精神病医院）。据说杨庆华被镇海公安国保反反复复绑架二十次左右，多次被送到精神病院迫害。

8、二零一五年五月四日，宁波市公安局、海曙区公安分局、西门派出所等一群人，绑架法轮功学员何小波夫妻。何小波已于二零一五年六月十日被非法关押三十多天后取保回家。二零一六年初又再次绑架何小波。同年六月二日，被余姚市法院非法判刑三年。上诉后维持冤判。其妻罗雪梅，在看守所被非法关押一年两个月后，被余姚市法院非法判缓刑三年。

9、二零一五年九月，宁波市象山县公安派出所一起出动，绑架了大法学员陈菊芳、在兰家庵修行的爱花和一名石浦大法学员（具体名字待核实），还有奉化市法轮功学员程秀霞及宁波市法轮功学员朱素芬、陈秀娥等约10位法轮功学员也被当地警察绑架。十几位同修被送到鄞州区章水镇“西山阁宾馆”洗脑班迫害。因为这些学员邮寄了举报江泽民的控告信。二零一五年十一月二十四日，宁波市海曙区法轮功学员朱培婷因起诉江魔头被海曙区公安分局西门派出所绑架，非法拘留15天；镇海区法轮功学员陈月霞也因起诉江魔头被镇海区公安分局国保林坚民、王志山等7人绑架，被非法拘留十天。

10、莫桂花，女，六十多岁。二零零九年大约六月，莫桂花在宁波市江北区被绑架，被非法关押在宁波市看守所遭受迫害，后被非法判刑三年。二零一六年初，莫桂花在其子女家中再次被绑架，被宁波市鄞州区法院非法判刑四年。
