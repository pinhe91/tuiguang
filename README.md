### 中国民众：你们是未来人类的种子

【明慧网二零二三年五月四日】（明慧记者夏昀台湾采访报道）二十四年前的“四·二五”，有万名中国的法轮功学员和平上访，后来，中共前党魁一意孤行开始迫害法轮功，从那时起，世界各国的法轮功学员开始了向各界人士讲述法轮功的真相。年轻的缇娜（化名）就是其中的一位。
缇娜六年前跟随先生从大陆回台定居。三年前，她开始每天在晚上七点半到十点往大陆打电话讲真相，向中国同胞讲述法轮功的真相，为误解法轮功的人剖析中共那一套造假宣传的谬论，阐明中共暴政危害中国人的严重性，让人在善恶面前做出正确选择。

有很多人因此明白了真相，有情绪激动的小粉红、有体制内的利益既得者，他们从谩骂、不认同，到真心道谢与敬佩。还有早已明白真相、关心支持法轮功的民众，高呼“法轮大法好，李大师好”。更有的提醒她，“你们一定要保护好自己，你们是未来人类的种子”……这些都让她充满感动，激励着她坚持打电话讲真相。

缇娜讲述几个印象深刻的例子。

世人的转变与支持

一位三十岁左右的年轻男子，全家都是党员，他自称崇拜毛泽东。缇娜告诉他共产党干的坏事和法轮功的真相，他都非常排斥，也听不进去。缇娜一边安抚他的情绪，一边梳理他党国不分的误区。男子则不断细数：“我陪你聊了三十分钟啦。”“我陪你聊了四十分钟啦。”“你挺有耐心啊，碰到我这样的人，你还这么有耐心。”直到电话最后他说，“在你身上我也学习到很多东西。”

缇娜不放弃，又找了时间打第二通电话，对方正在忙。第三通电话打通后，男子象朋友一样和她聊了起来。她感觉到对方没有了之前的误解，能开始自己思考问题，也同意三退（退出党、团、队等中共相关组织）了。最后她告诉男子，“您可别再崇拜毛××了。”他说：“我不崇拜毛××了，现在我崇拜（法轮功修炼者）。”

另一位原本同样听信中共宣传的小粉红，通话过程情绪激动，完全无法听完完整的一句话。缇娜不断提醒自己保持平和，抽丝剥茧的解开他每一个心结。这通电话长达八十分钟。终于，他敞开心扉象朋友一样的交谈。缇娜向他推荐了法轮功的主要著作《转法轮》，相信对解除他心中迷思有帮助。他欣然接受了，并再三道谢。最后他同意三退（退出中共相关组织）。

一位中年的基督徒曾因信仰被中共迫害，在监狱里遇到法轮功学员。他告诉缇娜原本对法轮功有误解，是因为在监狱接触了那些学员而了解了真相。他说：“法轮功学员太善良了。”另一位中壮年男士也已经知道了真相，他认为法轮功学员“是世界上最善良的人”。他一再叮咛缇娜，“（中共）这个组织还很邪恶，还在迫害你们，一定要保护好自己。”最后他说：“你们是世界上最清新的空气，你们是人类未来的种子。”

缇娜回顾这三年多所经历的，面对形形色色的中国同胞，过程中有苦有甜。她说：“我们就想埋下一颗善良的种子，也许无法立即看到开花结果，但是我相信势必会生根发芽。”
