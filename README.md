### 反迫害23周年　乌克兰民众声援法轮功

文: 乌克兰法轮功学员 

【明慧网二零二二年七月二十九日】二零二二年七月二十三日，乌克兰法轮功学员在首都基辅（Kyiv）举行反迫害活动。历年法轮功学员是在中国驻基辅使馆前举行和平请愿。由于战争原因中使馆不工作，法轮功学员在市中心大道举行了反迫害23周年活动。

![image](https://user-images.githubusercontent.com/79625284/181719032-4a616890-3407-487c-8641-f93ad263301b.png)

图1～2：乌克兰法轮功学员在首都基辅（Kyiv）市中心大道举行反迫害23周年活动

![image](https://user-images.githubusercontent.com/79625284/181719145-c14e42a9-1dbc-4406-b831-3db43c0d753d.png)

图3：悼念被迫害致死的中国大陆法轮功学员

![image](https://user-images.githubusercontent.com/79625284/181719239-ba9f55a1-b179-4c5e-b77a-24b7abfba7e0.png)

图4：基辅市军人签名声援法轮功

![image](https://user-images.githubusercontent.com/79625284/181719578-000d4d39-e787-46fd-affc-d9ddc6437579.png)

![image](https://user-images.githubusercontent.com/79625284/181719625-bc0bb500-1a88-4876-8f53-7c5ac51c5baf.png)

图5～6：民众驻足签名声援法轮功

乌克兰目前处于战争状态，首都夜晚实行宵禁，人们不能在晚上走动，法轮功学员无法在夜晚的街面上点燃蜡烛，缅怀那些被中共迫害致死的中国法轮功学员，于是今年的反迫害活动便改在白天举行。

尽管乌克兰战争还在持续进行，不过基辅街头还是会有很多人。乌克兰东部和南部战区的很多国民迁移到了首都。他们中的很多人是第一次看到法轮功学员的活动，这使他们有机会了解大法。

活动中，法轮功学员向世人展示功法动作，派发关于法轮大法真相的报纸和传单，展示被中共迫害致死的法轮功学员的遗像。同时，学员们还向世人征集反迫害的签名。

当人们询问，为什么大法学员的神态看起来如此严肃庄重，手里还拿着照片？经过学员讲解，民众得知这是纪念那些在中国遭受迫害而死的法轮功学员。有人说他们已经听说了（中共的）这些罪行。还有一些人，因为乌克兰战争，最初对发生在中国的这场迫害表现出漠不关心。经过法轮功学员讲述后，他们改变了原先的态度，在请愿书上签名。

一位名叫安娜的老妇走到请愿台前签下了她的名字。她向学员解释说，她对法轮功学员遭受迫害的事很了解。她说：“我知道对良心犯进行身体检查和测试，尤其是那些没有亲属的犯人。检查他们的器官状况。如有必要，会杀死囚犯并出售器官。这太可怕了。”这名女士临走时祝愿大法学员的活动取得成功。

![image](https://user-images.githubusercontent.com/79625284/181719752-aba8cba4-60b5-4706-a3c4-871c460523e9.png)

图7：安娜女士签名支持法轮功

一位女士停下来，说法轮功学员的动作中散发出非常善良的能量。

哈尔科夫（Kharkiv）市是乌克兰的第二大城市，也是战争期间遭受重创的城市之一。哈尔科夫学员说，一九九九年七月二十日中共发动了对法轮功的疯狂迫害。从这天起，法轮功学员也开始了反迫害的历程。每年在这一天，哈尔科夫市大法学员会在市中心举行反迫害活动。由于目前哈尔科夫市仍遭到持续轰炸，许多居民已经撤离，市内中央街道和公园空无一人。在七月二十日之前，该市大法学员们就在火车站附近的广场上举行活动，揭露中共的迫害。

![image](https://user-images.githubusercontent.com/79625284/181719889-855ac641-7341-4340-bee0-ae69bdf29444.png)

图8：在哈尔科夫（Kharkiv）的广场上举行活动揭露中共迫害

在惨烈的战争中，哈尔科夫居民遭受了惨痛的身心创伤。大法学员向饱受惊吓的人们赠送写有“真、善、忍”的吊坠小莲花，告诉人们要改变世界，先从自我做起，坚信真、善、忍理念，能给人们的生活带来改善的希望。人们看着小莲花，脸上露出喜悦的笑容，感谢大法学员赠予的礼物，并表示一定会挂在家里，以便提醒自己应该努力变得更好。

民众与法轮功学员交谈，他们很感谢大法学员在这么艰难的情况下，仍能提醒人们保持善良，做反对迫害这些最重要的事。
