### 惊心动魄的一幕彻底改变了我

文: 中国大陆大法弟子的儿子

【明慧网二零二二年七月四日】我在一家私企开大货车，经常出远门，虽然工资挣的多一些，但是这活很苦、很累，也很危险。
我妈是修炼法轮功的，我见证了大法给她带来的变化：满身的病没花一分钱，炼功后都好了，人变的乐观、开朗，我家又恢复了往日的温馨和欢乐。妈妈经常外出给民众讲法轮功真相，我也早就做了“三退”（退出中共的党、团、队组织），并且也向最高检察院和法院控告过江泽民迫害法轮功。可是我对大法弟子所说的法轮功的那些神奇事半信半疑，甚至不相信，还不止一次的扔给老妈一句话：“人不吃饭不行，没钱花不行！”

虽然如此，可大法的真相护身符我却总是随身带着。

二零二零年六月的一天，我驾驶着满载石料的长箱车行至河北地界，这段路左面临山，右面是深沟，路况不好。车开了不远，因为是重载，还是上坡，只听车头与车厢的连接部位，“咔！”的一声，这时我的大脑的第一反应就是：“不好，前桥断了！”随即车厢猛的一个侧翻，便“哐！”的一声闷响便往沟下滚。我象那待宰的鸡，吓的紧闭双眼等死……

过了一会儿，没动静了，睁眼一看，我还在车里坐着，我没死啊？！怎么回事？就战战兢兢的从车里出来，眼前的一幕让我倒吸了一口凉气：巨大的载重车厢倒栽着，车头并没有被滚下的车厢拖下沟，而是稳稳的卧在沟沿上，否则此时我这条小命就……

这翻车事件超乎寻常，太离奇，太不可思议！当时我只觉的奇怪，想不明白怎么会这样？

惊魂未定的我回到家，与家人说了这无法解释的事，全家的人只是为我感到很后怕……这时只听妈妈大声喊：“儿子，还不快去给大法师父磕头，是师父救了你的命呀！”

我的心一震，啥都不相信的我如梦初醒，眼泪大滴大滴的往下落，清洗我的这颗散发着铜臭的灵魂，啥都不以为然的我，在生命的深处升起了从未有过的对大法的敬畏和对大法师父的感恩！

我真心的叩拜师父，感谢师父的救命之恩！
