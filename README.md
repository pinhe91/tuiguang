### 上网学法轮功 　九十余国两万多人受益

【明慧网二零二二年一月二十日】（明慧记者英梓报道）自二零二零年四月以来，据不完全统计，来自全球超过九十个国家的两万两千多人参加了法轮功（又称法轮大法）免费网上教功班。目前，因为需求不断增长，网上教功班已经扩展到二十种语言，而且还在不断增加。疫情中，网上教功班为民众提供了祛病健身、提升道德的功法教学。

#### 法轮功网上教功班令九十多国民众身心受益

网上教功的起源是一些偏远地方或因各种原因不便于前往的国家或地区，难以举办面对面的教功，法轮功学员希望以网上教功的方式，帮助那里的人们获得机会学习法轮功。二零二零年初，中共病毒导致的疫情席卷全球，户外和面对面的活动都受到限制。动荡中，人们渴望找到有益身心健康的良方。网上法轮功免费教功班 https://LearnFalunGong.com/ 为民众带来了福音。

加拿大法轮功学员、网上教功协调人雷时忠博士向明慧记者介绍，在各地法轮大法学会的支持下，俄罗斯、印度法轮功学员在二零二零年春开始了网上教功班，其它国家也开始积极筹备。二零二零年十月三十一日，加拿大法轮功学员举办了首次英语网上教功，有三十六人参加。之后网上教功班不断增多，几乎遍布全球。

“到目前为止，仅限于参加英语、法语、西班牙语、汉语、俄语、越南语、德语、印尼语以及印度语（包括印地语、马拉地语、孟加拉语和卡纳达语）网上学功的就有来自九十多个国家和地区的两万两千多不同族裔、职业、年龄的人，遍及五大洲，甚至北极。”

他补充说，之前在媒体报道出的数字只是北美（英、法、中文）和印度班的参与人数，当时其它教功班的参加人数还没来得及统计。

#### 二十种语言传福音

![swxflg](https://user-images.githubusercontent.com/79625284/150312422-3dbb1c1e-bb90-4e82-b9d1-6be656b875b9.png)

雷博士说，目前提供的语种，除了上述十种语言外，还有日语、瑞典语、荷兰语、土耳其语、波斯语、韩语、匈牙利语、意大利语，共二十个语种；很快还会推出波兰语、芬兰语、克罗地亚语、乌克兰语和犹太语等语言。每天不同的时段都有不同的语种开班。

中文教功班协调人黄（Wendy Huang）教授介绍，中文教学网上学功班自二零二一年七月开班，参加学功的大约四百二十人，来自十九个国家和地区。目前，中文学功班已经由最初的一个班，增加到三个班，两个班适合亚太时间，另一个班适合北美时间。

据法语班负责人阮平善（Prashant Raina）博士介绍，法语课程于二零二一年四月开始。到目前为止，来自十个国家的六百人参加了每周两次的法语课程。

据印度教功班负责人苏伦德拉·饶（Surendra Rao）介绍，自二零二零年四月开班以来，有五千六百多人参加，其中超过 80% 的参与者是过去九个月加入的，目前人数还在增长。到二零二一年底，印度已经开设五种语言的课程——英语、印地语、马拉地语、孟加拉语（主要适用于印度西孟加拉邦和孟加拉国）和卡纳达语。网络辅导员还为青少年举办了专门的学习班。

越南学功班协调人约翰（John）介绍，越南法轮功学员从二零二一年七月开始举办网上学功班，半年内就有一千五百人参加，他们都是口耳相传进来的。

德语班负责人王先生介绍，德语班从二零二零年十二月份开始，到现在已经有约六百人参加。

雷博士说，很多人参加了网上学习班后，迫不及待地找到当地炼功点，参加面对面集体炼功。
