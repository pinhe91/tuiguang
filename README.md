### 明真相世人对法轮功学员诉心声

文: 爱尔兰大法弟子守诺 

【明慧网二零二二年十月十一日】我是爱尔兰学员，说一说最近上街讲真相的经历。
现在已经四亿多华人退出中共组织，随着这个巨大的三退浪潮，越来越多中国人也勇于表达自己真实的心声。

一位来自沈阳的梁先生说，自己在大陆的亲人刚刚去世，心情很沉痛，出来上街转一转。他说亲人去世仅一天就被埋葬了，由于疫情不允许停放，他在海外也回不去，不能见亲人最后一面，共产党真是太不讲人权了！他又气愤又难过。他说自己曾经遇到法轮功学员，已做过三退，法轮功学员留给他的印象是健康的，善良的。他说：中共不倒台，老百姓就跟着遭殃！

一位来自北京的王先生带着女儿来到我们的真相点，拿了小册子。他说自己是亲眼目睹八九年六四的那一代人，见证了中共的腐败、虚假与残忍，他清醒地认识到这个政权不可能走向自由和民主！他很失望！他现在希望自己的女儿能勤奋学习，留在海外。他表示要回去好好看一看有关法轮功的资料。

一位来自福建的余女士讲述她的故事，她的哥哥是一名教师，被打成右派。文革后的平反潮中，既没有给她的哥哥平反，也没有给任何的经济补偿，哥哥生活很困苦。她当时正怀着第二个孩子，因为中共的一胎政策不允许要二胎，她一直东躲西藏，好不容易生下老二，她的丈夫又被叫去交罚款。后来几经周折来到爱尔兰。移民官问她为什么来，她说想多要几个小孩。移民官笑着说，在这里可以随便生啊。我们跟她谈到四亿多人已三退。这位余女士说：共产党越早倒台越好，大快人心！

一位台湾女士在路边车里坐着，我走过去递给她一本小册子。她说：我在这里看你讲了十几年了，每次看到了小册子上有关活摘法轮功学员器官的内容，我都很痛心。

这时从另外的车里走下来一位男士，他说他来自四川，他在国内的生意场拼搏许多年，积累了可观的财富。他说中国的公检法和税务部门都是流氓操作，如果不能符合他们的流氓逻辑，就不能赚到钱，活得真是太累了。两年前和妻子商量全家人移民爱尔兰，结果两家的亲戚都不理解，他们国内生活优渥，夫妻俩又不会讲英语，为什么自讨苦吃？他叹了口气说：我只想呼吸自由的空气。他说现在体制内的人都知道中共完了，都想着如何把钱移到海外，或者是自己一跑了之，而不明真相的底层老百姓却还跟着中共摇旗呐喊。

台湾女士说：现在很奇怪，有些海外人认为国内好，国内的人认为国外好。男士说：现在的中国人已经不知道自己的文化源泉，找不到自己的根了……

每一个背井离乡的国人都有自己的故事，而其中有太多的缘由是源于这个邪恶的中共政权！希望有更多的人能站出来讲一讲自己的经历，认清中共的本质。
