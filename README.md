### “共产党是大骗子，我看穿了它！”

文: 江西大法弟子 

【明慧网二零二二年四月三十日】今年，我和同修到人流多的地方，给百姓传真相，遇到不少明白真相的人士，今摘选两位体制内人的明智之言。

#### 1.“共产党是大骗子，我看穿了它！”

一月中旬，我和一位大法弟子到一个集贸市场讲真相时，碰到一位六十多岁的男士。当时我送给他一个真相护身符，他问是什么护身符？我告诉他是法轮功的“法轮大法好，真善忍好”九字真言护身符。他高兴地接过了护身符，并退出了曾加入过的中共团、队组织。

他告诉我，他知道法轮大法是好的。他知道有一个大法弟子捡到一个金镯子后，及时归还了失主。其实这个大法弟子家里并不富裕，可她却能够拾金不昧，这在当今的社会里是难能可贵的。当时，大法弟子上班的超市老板知道后，还奖励给她200元钱。

可这么好的人不久却被中共警察绑架，并被非法判刑两年。刑满被释放，大法弟子回家后，超市老板又聘请她回超市上班。但不久，该大法弟子又一次被绑架并非法判刑。

这位男士神情严肃的说：“这个共产党真是太邪恶了！这么好的人都不放过（迫害）！它靠谎言起家，靠谎言执政，中国十四亿人就象被关在铁笼子里，任它宰割。”

“说实在话，我和我妻子虽然都是体制内的人，我就是要说公道话：共产党是大骗子，不得人心，我看穿了它！我也希望中国人快快清醒过来，不要再上中共魔鬼的当。”

#### 2.“中共才是真正的邪教，天天用谎言给中国人灌毒”

二月中旬，我和另一位大法弟子到一个粮油批发市场去讲真相，碰到一位男士，大约五十多岁。我给他讲大法真相，送了一个护身符给他，他象孩子得到宝物一样高兴，连声说：“谢谢你！谢谢你！”他还告诉我他在行政单位上班。

我说：“我猜你是个党员。”他说：“我已经退了，党费我也没交了。”我说：“是大法弟子给你退的吗？”他说：“我自己退的。”我说：“这不算，要向神退、神认可才行。今天你碰到了我，就是缘份，我帮你退。”他说谢谢我。

我们边走边聊，他一连串说了很多：“翻墙就能看到很多的东西，中共才是真正的邪教，天天用谎言给中国人灌毒……污蔑法轮功……事实跟中共讲的全是反的，实际上法轮功是救人的，法轮功的师父是来救全人类的。迫害法轮功二十多年，大法弟子吃了很多苦，我打心里佩服法轮功。”

临分手时，他还紧接着说：“中国人真可怜，被共产党欺骗的好坏不分。如果所有的中国人都能突破网络封锁，中共的谎言就会不攻自破，这样我们中国人才能真正自己主宰自己！”

九十一岁老妈的神奇故事

文: 中国大陆大法弟子

【明慧网二零二二年四月二十九日】我母亲今年九十一岁，前两天我给大弟弟打电话，问母亲怎么样，他说：“你不用挂念，妈身体好的很，全村人都羡慕。”
十年前，母亲患腰椎间盘突出，腰、腿都很疼，睡觉时翻身也困难，在老家吃药治疗没效果，我就把她接到了省城来。我带她到医院检查，医生说：“年龄太大了，没法做手术，只能保守治疗。”当时开了几帖膏药，贴了几天也没有效果。

我对母亲说：“我教你念：法轮大法好，真善忍好。诚心念，师父就会管你。”她说：“好。”就跟着我一起念九字真言。之后，母亲每天都念“法轮大法好，真善忍好”，腰腿渐渐的不那么疼了，睡觉也能翻身了。

在我家住了二十多天后，母亲要回老家，我就把她送了回去。她到家就把以前供的乱七八糟的东西烧掉了，每天就是诚心的念九字真言，腰腿都好了。

四弟媳妇一看这么神奇，也跟着母亲念了起来，她女儿、儿子也都退出了邪党组织，女儿考上了研究生，儿子考上了大学。

母亲八十四岁那年冬天，老家下大雪，她在大弟弟的院子里滑倒摔了一跤，大弟弟吓坏了，喊：“妈，我拉你起来，摔坏没？”母亲说：“没事，没事，别拉我，我自己站起来。”她自己真的站起来了，嘴里念着：“法轮大法好，真善忍好。”回自己屋去了。大弟弟也乐了。

二零一九年，母亲左眼长白内障，在县城做手术，术前检查身体各项指标都正常。医生说：“身体怎么那么好呢？那么大年龄。”母亲说：“我有信仰。”手术很成功，也没有住院。

母亲现在身体特棒，满头黑发，皮肤白里透红，没有老年斑，眼睛能纫小针，不但能照顾自己，还种了两个院的菜，整天乐呵呵的。家里儿媳、孙男孙女都很孝顺，几十口人都很和睦。

法轮大法是正法，法轮大法是救人的高德大法。大疫当前，“法轮大法好，真善忍好”九字真言是走出劫难的良方、妙方。希望有缘人明白真相，转变观念，不要再被中共邪党的谎言蒙蔽。
