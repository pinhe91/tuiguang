### 诚信“法轮大法好”的人走过危难

文: 陕西大法弟子  

【明慧网二零二一年十二月十九日】我地大法弟子在讲真相救人的过程中，见证了大法所创造的一桩桩神迹，现讲几件与大家分享。

#### 突来车祸无大碍　世人明真相得救

二零二一年十一月九日十点多，七十多岁的A同修骑着三轮电动车，带着近七十岁的B同修，给邻区同修送完真相资料，在返回的路途中，明明看着后面无人无车，才打着左转向灯，慢慢拐弯。刹那间，只听“嘭”的一声，不知身后从哪儿窜出一辆黑色小轿车，直奔电动车而来。

A同修瞬间被甩出车外，电动车左右猛烈摇晃了几下，才停了下来。后座上的B同修急忙跳下车，去看A同修，心里求师父救弟子，嘴里不停念叨：“法轮大法好！真善忍好！没事！不会有事！”

她推推趴在地上的A同修，问：“你怎样？没事吧？”这时被撞懵的A同修才回过神来，慢慢从地上爬起来，活动活动，说：“嗯！没事儿！”

此时，再回头看后面的司机和他的同伴，傻傻的站在那里，脸色煞白，一副惊慌失措的样子。司机手指着路边的医院，声音颤抖着说：“赶快送医院，送医院！”

两个同修摆摆手说：“不用不用，我们不会有事的，我们有师父保护，有神保护，你们不用害怕，我们是修炼人，是炼法轮功的，不会讹你的。”

司机又说，他打电话叫保险公司来做鉴定。两个同修很认真的告诉他们：“我们真的不会有事，请相信我们，我们修大法的人，有师父保护，你们今天是遇到我们炼法轮功的了。如果撞到常人，你们今天就惨了。”

同修很耐心、细致的给司机两人讲了大法真相，送了真相资料和护身符，他们还退出了团、队组织。司机两人拿着真相资料，看着精致的护身符，激动的不停念叨：“谢谢阿姨，谢谢大法！原来法轮功这么好啊？ 我们今天真的是遇到好人了。”

一场车祸，在师父的保护下，两位老年同修安然无恙。第二天，A同修腿上、臀部发现有好大一片的青紫色淤斑，但是疼痛感却是明显可以承受的了。弟子发自内心的感谢伟大的师尊，是师父为弟子还了命债，承受了痛苦，救了世人，弟子还长了功。

#### 肠癌患者枯木逢春

我女儿在市一家医院工作，下班回来，告诉我说，早上上班，在医院楼道口，遇到C阿姨（同修）和D阿姨（同修）姐妹俩，说她们的弟弟在我科要做手术。女儿对她们说：“说是有个大手术，原来是你弟呀？现在疫情严重，也進不去，你们等着，我去看看手术安排情况，再告诉你们。”

我问女儿：“什么病？”女儿说是结直肠肿瘤，做肠镜病检，结果不太好，需要做个手术，把这段肠子切除（我女儿怕我担心，没有把实情说的太明确）。C和D的弟弟的手术被安排在了第二天十点三十分。我叮嘱女儿，瞅机会，告诉叔叔记住心里默念九字真言，女儿答应了。

C夫妇同修和妹妹D同修都是修炼了二十多年的老大法弟子了，面对亲人的不幸，更多的是担心和无奈，只能是提醒和劝慰。在医生给她俩的弟弟做术前的准备时，女儿悄悄安慰他说：“叔叔，你不要紧张，心里多念那九个字，一切都会顺利的。” C、D同修的弟弟望着我女儿，一脸茫然，女儿笑着对他说：“我妈和你姐、你妹她们都是修大法的。”病人会心的点头答应了。

手术在无影灯下紧张的進行着，手术室外，C同修和她的丈夫与妹妹D及家人在焦急的等待着，全家人心里都在一遍接一遍的默默念诵着“法轮大法好，真善忍好”。

这时，C同修的丈夫突然看见手术室门口一个大大的金色法轮在旋转着，不停旋转着，顿时，两行热泪夺眶而出，双手合十，感恩师尊之心无以言表。

手术進行的很顺利，病人恢复的也很快，第二天，就能下床活动了，第三天，能吃点流食了。五天后，病检报告出来了，被诊断为：右半结肠加横结肠粘膜下纤维脂肪瘤。医生拿着病检报告，既疑惑又兴奋地说：“这个病人简直太幸运、太幸运了，竟然是个良性的。”

几天后，女儿告诉我说；“明天，我就要给叔叔拆线了，他明天就出院。”我说：“师父慈悲，又替弟子的家人受苦受罪了，他知不知道他的生命是延续来的，他该走入修炼啦！”

第二天拆线时，女儿说：“叔叔，你是不是该入（修炼）门啦？”C同修的弟弟高兴的回答：“是呀，家人不停的督促我，该学法，该修炼了，看来真是该入门啦！”

通过此事，C同修的弟弟真的相信了“法轮大法好，真善忍好”，现在天天念。

#### 学法炼功　淋巴癌痊愈

E同修今年五十多岁，也是修炼了二十多年的老大法弟子了，高挑的身材，娇好的脸庞，时常给人一种充满活力的印象。她父母生活在几十里外的农村，这么多年来，她每次回家看望老人，都是一半乘车，一半徒步行走。身背着满满一挎包真相资料，走着，发着，讲着，救度着。虽然两次被不明真相的世人举报、关押，但E同修正念正行，很快就回家了。

她丈夫在镇政府机关担任要职，在中共迫害的环境里，为了求得官位升迁，反对妻子修炼。E同修一遍一遍的学师父的讲法，向内找，悟法理。在师父的加持下，E同修心怀善念，讲真相，发正念，终于清除了邪恶因素，丈夫的态度也缓和了，为自己开创了一个宽松的修炼环境。

二零二零年八月的一天，E同修的小姑子哭着来到她家，说她被诊断为淋巴癌恶性肿瘤晚期，医院要她尽快做手术。消息传来，顿时全家乱作一团，她丈夫一走已十年，她和孩子相依为命，甚是凄苦。

手术完后，接着就是一月一次的化疗。E同修主动帮她，经济上，生活上，关心她，想让小姑子尽快从痛苦中解脱出来。可吃药、化疗竟无济于事，病情还是在不断的加重，包块扩散了全身。医生说，也就是二、三个月的时间了。三次化疗后，小姑子的头发、眉毛、连睫毛都掉光了，面貌脱相。

由于癌细胞扩散全身，医生还告诉说：“回家筹备二十万，准备给你做肝细胞移植。”对一个经济无可靠来源的弱女人来说，筹备二十万，简直是天方夜谭。小姑子听后，无法面对后面要发生的一切，只有死的心了。她悄悄的给自己在准备后事。

面对眼前的一切，E同修说：“炼法轮功吧！只有炼法轮功，才会有希望。咱一起炼！”E同修劝慰她，引导她，并送小姑子大法书。她几乎天天骑着自行车，往返七、八里地，穿梭在两家之间。这时，失踪十年的小姑的丈夫也回家了。看着妻子凄惨的模样，痛苦的表情，他羞愧难当。听说炼法轮功能救命，他也陪着妻子学法炼功，念九字真言，尽量为她分担痛苦，希望她能早日康复。

E同修天天和小姑子学法、炼功、发正念、听交流文章。佛度有缘人，小姑子悟性好，也能吃苦，明白了有病是由自己的业力而造，怨不了天，怪不着地，吃苦消业，病情好转一天一个样，人有精神了，面目恢复正常了，黑黑的头发也长出来了。半年时间，小姑子去市医院检查，一切正常，全身的肿瘤包块消失的无影无踪。市医院不相信，认为此前是误诊，又让她去省医院做加强CT，后又把医院做的肿瘤标本航运北京肿瘤医院检查，结果：一切正常。

一个被医生诊断不治只有等死的人，竟然炼法轮功炼好了，太神奇！事实摆在面前，就连以前反对修炼的家人及亲戚朋友哑口无言了，不得不赞叹大法的超常，很多人明白了真相，都“三退”保平安了。

E同修的丈夫经过此事也明白了，象变了一个人一样，明白了邪党的假恶斗欺骗宣传，退出了邪党，也开始学大法炼功了。E同修感慨的说：“今天有缘遇到宇宙大法，真是天大的缘份。是我们伟大的师尊救了我们，也救了我们一个又一个即将破碎的家。叩谢师尊！感谢大法！”

#### 弟弟和弟妹得福报

一九九九年，邪党的欺骗宣传，尤其是“天安门自焚”的造假，更让世人信以为真，铺天盖地的迫害，大法弟子家个个被搜查，邪党强行收缴大法书。一天，我到弟弟家串门，说起此事，弟妹果断的说：“把你们的书拿我家来，看谁能搜到我家？！”看似一句玩笑话，却彰显了弟妹的正义感。

不久，弟妹做妇科检查，折磨她多年的子宫肌瘤却不翼而飞。她不解的问我为什么？我说是她有保护大法书的一念，得到了福报，并告诉她一定要记住“法轮大法好，真善忍好”会出现很多神奇的事。

夏日的一天傍晚，弟弟俩口骑车回家。弟妹提醒弟弟说：你眼神不好，你心里念着九个字。她见我弟不吱声，就说：你不念，我念。

他们骑车刚走到大马路中间，一辆摩托奔驰而过，弟妹一下被摔倒在地。一看有人摔倒，傍晚乘凉看热闹的人“呼啦”围了一大帮：撞人了，撞人了，看看谁让摩托撞了，截住摩托。一看，都是熟人，有人说：送医院检查；有人说：赔多少钱，说啥的都有。

我弟妹从地上站起来，说：“人家没撞我，是我自己摔倒的，让人家走，让人家走。”看热闹的人笑话她说：“你真傻，本想给你弄几个钱花花，你却执意让小伙走啦。”接着，就是一片嘲笑声。

回家后，弟妹脱下鞋袜，脚踝骨处有一个核桃大的紫血包。我弟说：“让你去检查，你硬说没事，这下你受着吧。”弟妹说：“人家真的没撞我，是我的裤脚挂着脚蹬摔倒的，不能怪人家。”一边说一边念着“法轮大法好，真善忍好”。

第二天醒来，弟妹脚踝骨上的紫血包一扫而光。弟妹一会儿一遍的述说着：这九个字太灵了，太神奇了。

《洪吟五》发表后，我买来笔记本，让弟弟帮我抄写一本。抄完后，B同修一看说：“字写的这么好，也帮我抄一本。”我明白同修的用意，想让我弟多学点法，也明真相，于是，他很快帮同修抄了一本。

去年九月一天晚上，十点多，弟弟突感心慌，大汗淋漓，可坐起来，就好了。原以为是血糖过低，可一会就好了，他也没当回事。第二天九点多，他去镇医院检查后，医生一看结果：急性心肌梗死。医生立即告知家属，病情十分危急，需要手术，立即转院。

弟妹立即打电话告知我女儿，联系她医院的心内科的主治大夫。我叮咛他们说：快念九字真言。他们说：“念着呢，一直念着呢。”

中午十一点多，弟弟被送進手术室。手术進行的很顺利，术后，医生告诉我女儿：“你舅太危险了，心肌梗死，做了两个支架。他怎能从昨天支撑到现在？这么长时间，危险！太危险啦！”我们都知道是师父救了他，是大法救了他，是他念九字真言，才躲过了这一劫。

现在，我弟弟在家开始抄写《转法轮》了。他说，已经抄了一半啦。我为他们明白真相由衷的高兴。虽说他们还没有正式走入修炼，但是他们都知道“法轮大法好，真善忍好”。
