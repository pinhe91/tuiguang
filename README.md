### 三个弟弟见证大法神迹

文: 辽宁大法弟子 

【明慧网二零二二年六月二十七日】一九九九年“七·二零”之前，我们家族中除我之外还有多人学炼法轮大法。这些年中，我的家人遇到了许多神奇事，写出几件与同修分享，见证大法的殊胜与美好。

#### 师父救了大弟的家人

我的大弟是其中之一，而且比较精進。他的孩子也支持他修大法。

几年前的一个深秋，大弟的女儿、女婿等七人坐他家的三轮车去十几里以外的大山上采蘑菇。当车开到一个下坡时，突然颠了几下后，接着就停住不动了。于是大家都下车去查看，发现有个车轮子没在轴上，而是立在了车底下，把车卡住了。这可把他们都吓坏了！因为道的左边是深沟，这车要是翻了后果不堪设想……

大伙忙着把车身抬起来，把掉下来的轮子拉出来，安在轴上。可找不到卡轮子的穿钉了，有人就往来的路上去找，结果在一个坡路顶上找到了。令大家惊讶的是：穿钉距车停下来的地方相距四十多米远！那么远，又是下坡，轮子外边没有穿钉，车开出去四十多米轮子才掉下来呢？轮子掉了，车咋没翻呢？轮子怎么会是立在车下……一串串的不解之谜困扰着大家，解不开。

穿钉找到了，就把掉下来的轮子安上吧。当打开工具箱一看，一个“法轮大法好”的护身符出现在眼前！不言而明，人人从内心感恩大法，敬佩大法，感谢大法师父救命之恩！

安好车轮后他们继续往山上开去。

#### 敬师敬法 二弟得福报

我二弟不修炼，但他敬师信法，喜欢看大法真相资料，他家还一直供着大法师父的照片，每天不落的给师父敬香。说起来惭愧，他比我这个修炼人做的都好！

他平时也收废品，有时其中偶有大法书和大法真相资料，他都会完整的处理干净，谁要就送给谁。

一天他开着三轮车正在穿过一条小横道，突然一辆满载货物的大货车从上坡冲了下来，直接拦腰撞上了他的小三轮车。三轮车头撞坏了，车把歪到另一边去了，而他人啥事没有，连皮都没破！大货上的司机一看人没事，开车就跑了。二弟当时也没叫对方赔偿。只是自己租了一辆车，把他的三轮车拉到修车厂，花了五、六百元修好了。

他敬师信法，得了大法的福报。

#### 九死一生 三弟惊醒了

我三弟妹修大法。三弟却非常反对。三弟性情暴躁，爱好喝酒。我多次给他讲真相，他就是不听，他甚至曾经撕坏过大法书，把他妻子给师父敬香的香炉摔了。类似的事情还有。

一天他使用砂轮时，砂轮直接冲着他的脸崩了过去，眼球立即掉出一个，头骨塌进去一大块，大脑严重受损，满身到处是血，人事不知。家人忙叫车把他送到大医院。他家经济很困难，在家族人倾尽能力帮助下，经过八个多小时的抢救，总算保住了他的命。

他清醒后我们都对他说：接受教训，不要再反对大法，更不能做破坏大法的事。他知道自己是遭到报应了，此后不再干扰三弟妹修炼了，有时还提醒她到点了，该发正念了。

三弟也收废品。一次，一个老太太把占地动迁村里给的八万多元钱放在破棉袄的袖子里，时间久了忘记了，就把破棉袄当废品卖给了三弟。当三弟装车时，觉得这破棉袄很重，一看，哎呀有这么多钱！他马上就去找卖破棉袄的老太太，把所有的钱还给了老人。

老人非常感谢他，说：“你要不送回来，到用钱时找不到这钱，我就得急死啊！”

为此三弟得了福报：不久自己买了个大点的房子；儿子的生意越来越好，买了好房和车；女儿工作收入不错，也有了自己不错的房子。

善恶必报是天理！

我亲身遇到的大法的神奇事还有很多，一时说不完，今天先写几例吧！愿世人都能相信法轮大法好，敬师信法，早日得到大法的救度！

* * * * * * *
法轮大法神奇殊胜，法力遍布洪微。本文所讲的事例，仅仅是大法威力在世俗层面的一个小小体现。大法能给人类带来世间的福祉，而大法的神奇和殊胜更是为了让人返本归真。
