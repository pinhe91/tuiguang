### 九字真言带来的福报

更新: 2021年12月13日

【明慧网二零二一年十二月十三日】（明慧记者郑语焉采访报道）自“中共病毒”（武汉肺炎，Covid-19）爆发后，传播全球，导致近2.9亿人染疫，至今有超过530万人死亡。为了救人，法轮功学员尽所能告诉人们九字真言“法轮大法好 真善美好”，诚心念诵能帮助躲过灾难。现居斯洛伐克的苏珊是其中的法轮功学员之一。
苏珊是香港人，两年前随先生移民斯洛伐克，在中共病毒蔓延全球的时候，她告诉自己认识的亲朋好友九字真言，她说：“不仅九字真言让人在疫情中化险为夷，自从我修炼大法之后，身心受益，家人、朋友和周遭的人也都受益。”在接受明慧记者的采访中，她讲述了这些故事。

![image](https://user-images.githubusercontent.com/79625284/145788485-d6a50c04-6e60-4c07-b75b-340c97032386.png)

图1：苏珊和两个儿子（本人提供）

#### 常念“法轮大法好，真善忍好”，瘟疫不近身

二零二零年秋冬之际，苏珊一家从香港搬回斯洛伐克定居不久，听闻小姑的同窗好友因为接触“中共病毒”（武汉肺炎，Covid-19）确诊者，需在家隔离，很害怕自己染疫。苏珊用电话录音录好“法轮大法好，真善忍好”九字真言，请小姑转发给好友，说：“这是我给她的祝福（My blessing），并叮嘱每天诚心常念十到三十遍，会有意想不到的效果”，小姑好友接受了并表示感谢，之后她检测呈阴性反应（没有染疫）。

今年十月，这位好友的大儿子回家后，身体不适，检测结果感染了武汉肺炎，因此一家四口在家隔离，家里三人都确诊，包括好友同床共眠的先生也染疫了，只有接受“法轮大法好，真善忍好”九字真言的好友没事，他们觉得不可思议。苏珊对小姑说：“所有我的亲朋好友，诚心念九字真言的，都没有人感染中共病毒。”小姑非常惊讶，因为她有些朋友确诊，而且病得很厉害。苏珊委婉建议小姑也把“法轮大法好，真善忍好”的祝福送给朋友。

#### “一定是你炼了法轮功，所以我才没事”

苏珊的先生是斯洛伐克人，他虽然知道法轮大法好，但由于顾虑心，私下劝阻苏珊在聚会或饭局上讲“中共活摘器官”等真相。这两年发生了一些事情，先生的态度转变了，现在不但主动为苏珊翻译，也常帮忙讲真相，整个人更加精神清朗。

最近，苏珊的先生因为工作关系，需要打疫苗，眼看许多友人和工作伙伴打了第二剂之后，身体不舒服的反应大，因此他特地安排两天休假“备战”，结果打疫苗之后，身体没有任何不良反应，精神也没受到影响，先生对苏珊说：“一定是你炼了法轮功，所以我才没事。”

苏珊两个儿子的学校，许多班级因为有人感染“中共病毒”，全班都需在家隔离观察，不得到校上课。而两个儿子的全班同学都安然无恙，照常上学，许多人觉得神奇，苏珊对儿子说：“你们有在炼功，你周围的人也会跟着受益。”他们也会把握机会给同学讲真相。

苏珊说：“不仅九字真言让人在疫情中化险为夷，自从我修炼大法之后，身心受益，家人、朋友和周遭的人也都受益。”

例如，苏珊的朋友期盼第二个孩子，可是连续两次滑胎流产，第三次怀孕时战战兢兢，唯恐再三失望。苏珊发简讯给朋友，建议她常念“法轮大法好，真善忍好”，朋友接受祝福并表感谢，结果胎儿安好顺产，孩子健康成长，活泼又可爱。

那么，苏珊是如何走进大法修炼？修炼之后又有那些变化呢？

#### 两岁儿念九字真言，恼人湿疹不药而愈

苏珊婚后头胎生了个儿子，她享受初为人母的喜悦才过几个月，没想到儿子罹患湿疹，皮肤红肿、痒、粗糙，甚至渗出血水，全身找不到一处完好的皮肤，看了许多西医、中医，病情反反复复，无法痊愈。病在儿身，痛在娘心，儿子哭、她也哭。她经常望着沉睡中的儿子，眼泪噗簌簌地掉个不停，感到做人真是苦﹗

苏珊把她的痛苦向同事倾吐，同事安慰她要宽心，放下这个痛苦的纠结，建议她学炼法轮功。她听取同事的建议，于二零零七年夏开始学炼法轮功，每天念“法轮大法好，真善忍好”九字真言给儿子听，当时一岁半的儿子听着听着，情况日渐安稳，苏珊也感到心灵有所安顿。

儿子两岁生日那天，刚学会说话的儿子自己开口说：“法轮大法好，真善忍好”，从此，皮肤开始变得细嫩光滑，回复到婴幼儿那样漂亮的肌肤。曾经笼罩的愁云散尽，小家庭又见欢笑幸福绵绵。

![image](https://user-images.githubusercontent.com/79625284/145788628-bccdf99e-72bc-4049-bee0-697721c5b632.png)

#### 修炼法轮功后，腰椎旧伤后遗症和红肿肉芽不见了

修炼法轮功后，苏珊不仅身体获得健康，心理素质和心性也有意想不到的提升。她说：“修炼之后，我的心感到非常平静祥和，是以前从来没有过的。”

而且，苏珊遵照真、善、忍的指导原则修炼，事事为他人着想，先他后我，遇事向内查找自己的不足，提高心性，没想到自己身心和亲情都有莫大获益。

苏珊开刀生产大儿子之后，手术缝合处长出红肿肉芽，又痛又痒，非常难受，用药一年多也无法消肿。另一个让她长久以来备受折磨的是，小时候溜直排轮（即溜旱冰）多次跌跤，导致腰骨受伤，留下了酸痛的后遗症，天气不好腰椎就酸痛，她无法久坐，更不能坐在地上或冰冷的地方。在外学法炼功，必是仔细铺好坐垫，在家则是在床上打坐。

修炼法轮功三个月左右的某天，苏珊应同修建议硬着头皮直接坐在地上炼功，“咦，怎么没有酸痛？﹗”苏珊猛然发现腰椎旧伤竟然不药而愈。不知何时起，生产时的开刀处不再疼痒，红肿肉芽消失了，只剩一条细线隐痕。

#### 坏习惯也戒掉了，感觉从未有过的神清气爽

苏珊还谈到，修炼法轮功之前，她抽烟、喝酒、打麻将、赌博样样来；修炼之后，明白这些是不好的习惯，很自然地说戒就戒掉了，感觉从未有过的神清气爽，心宁神静。

#### 为他人设想，海阔天空又自在

香港生活步调快又紧张，每个人都有不同程度的压力。公司老板的脾气大，个性和行事风格急躁严厉，说话语气重时，大家都提心吊胆非常紧张，员工表现不够理想时开口就骂，几乎无人幸免。

有时莫名其妙挨骂，苏珊心情起伏难平，需要深呼吸来调解情绪。得法之后，苏珊换位思考，为老板设想，理解老板承担的压力太大，他不是对员工发脾气，是在对他自己发脾气。之后老板发脾气，苏珊心平气和地先查找自己，是否自己有需要改进的地方，心湖不起波澜，泰然对待。苏珊说：“我内心非常平静，是修炼之前从来没有的。而且自从我跟老板讲真相之后，感觉老板对我更加尊重了，也爽快解囊赞助讲真相的项目。”

夫妻的观念、习性和意见相左在所难免，苏珊知道先生不喜欢被过多管束，常强忍着，她说：“但是我忍得很辛苦，强忍着不发脾气，但是内心沸腾，好象被火烧的一样，很难控制，积累到超过临界线时就爆发开来，回骂他一顿。”

修炼法轮功之后，苏珊遵循真、善、忍原则的指导，也明白了许多事情是有因缘关系的，她明白了可能是自己以前有过对先生不好，欠债要还，那么就是现在要还给他；所以，后来先生发脾气时，她只笑笑，不再耿耿于怀，而且心情轻松安泰。先生看到苏珊的改变，他的态度也变了，尊重苏珊带两个儿子一起学法炼功，还买花送给苏珊供奉在师父的法像前。

#### 念诵九字真言得福报

其实，从苏珊获得九字真言，从而接连得到福报的例子还有，她说，她在香港时的一位同事的腹部生了一个纤维瘤，每次痛时，都折磨得她很难忍受；没想到，念诵九字真言就不痛了。

念诵九字真言得福报者还包括苏珊的父亲。苏珊从小跟父亲的关系特别紧张，父亲对待她特别不公平的严厉，造成日积月累的伤害，种下仇恨父亲的种子。修炼之后，苏珊明白了夙怨恶业轮回的法理，清除内心仇怨之恨，主动关心父亲，与父亲修好，带他上馆子，为他夹菜递茶，亲子关系日渐和睦。

父亲是个老烟枪，习惯大吃大喝，患有肺炎，肺部长了一个肿瘤，非常接近血管，医生说不能动手术，因为百分之八十会死在手术台上，采用电疗跟化疗时，发现他的白血球低，医生说不能做任何疗程，叫父亲回家休养，这无异宣布无法施行医疗救治，只能在家自生自灭。父亲情绪低落到谷底。

刚得法不久的苏珊，看了明慧网同修交流的文章，开始每天读《转法轮》给父亲听，读法之后让父亲向师父法像合十礼敬，教父亲持续常念“法轮大法好，真善忍好”，父亲的坏个性改变了，觉得身体和精神越来越好，不顾家人担心癌细胞已扩散全身的劝阻，坚持到医院做CT检查，结果出炉，医生说看不到肿瘤，只有一丝丝的痕迹。恢复健康后的父亲故态复萌，又过起任性的生活，尽管如此，寿命还是多延长了五年，才因肺积水而辞世。
