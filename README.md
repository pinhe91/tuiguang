### 新华社新疆分社副社长陈晓虎诬蔑法轮功遭恶报身亡

【明慧网二零二一年六月二十二日】（明慧网通讯员大陆报道）新华社新疆分社副社长、分党组成员、中共恶党党员、高级记者陈晓虎，因在网络上大量发表污蔑法轮功的文章，迫害法轮大法，颠倒黑白，制造仇恨，遭了恶报，于二零二一年六月七日在北京抢救无效死亡，卒48岁。
据公开资料显示，陈晓虎一九七三年五月出生，曾任新华社宁夏分社常务副总编辑、新华社宁夏分社副社长，后调任新华社新疆分社副社长。

新华社作为中共重要的喉舌之一，在迫害法轮功中，从头至尾都起着带头犯罪的作用。自迫害法轮功始，新华社及新华网就大规模编造了对法轮功进行诋毁和诽谤的文章。仅二零零零年一月至二零零三年十月，诋毁文章就达522篇之多。

二零零零年九

月六日，宁夏维吾尔自治区灵武市法轮功学员陆红枫副校长在致人大呼吁书中签字，呼吁停止迫害法轮功，遭陆红枫的丈夫秦玉焕与灵武市精神病院药物致死后，秦玉焕这个泯灭了人性和良知的中共支部书记虐杀妻子后，不仅毫无悔意，反而变本加厉的在报纸、电视等媒体采访中诽谤法轮功。当地政府利用媒体渲染陆红枫的死亡，但却对陆红枫的真正死因秘而不宣，散布谎言，欺瞒世人。

### 诚念“法轮大法好”　三哥的小脑萎缩好了

 文: 中国大陆大法弟子 

【明慧网二零二一年六月二十一日】法轮大法神奇殊胜，法力遍布洪微。本文所讲的事例，仅仅是大法威力在世俗层面的一个小小体现。大法能给人类带来世间的福祉，而大法的神奇和殊胜更是为了让人返本归真。
* * * * * * *
已八十岁高龄的三哥去年患了小脑萎缩。今年三月份我和丈夫去看他，看上去他的气色很不好，以前就不爱吱声的他现在更不怎么说话了，行动迟缓，当时是他的女儿在照顾他。

侄女说上厕所得搀着他，要不然就可能摔倒，有时大小便弄在床上和裤子里，记忆力也不好，有时还说些不着边的胡话，侄女很是无奈。

《明慧周刊》第998期里有一篇同修的交流文章，里面有一则诚念“九字真言”得福报的事例给我的印象很深——一位七旬老太太九十度弯腰，通过诵念“法轮大法好”、“真善忍好”，几天腰板挺起来了，和原来判若两人。我找出那期周刊，带上播报器，四月一日那天又去了三哥家。

我把这个小故事念给他听，并嘱咐他一定要常念“法轮大法好”、“真善忍好”，他点头答应，我一遍一遍的教他念，他也学着念，我又让侄女放《九评共产党》给他听。

事隔几天，我又去看了三哥。進门看见他坐在沙发上，脸色不那么难看了，这日我拿来了装有《绝处逢生》和《祝你平安》的TF卡让他听，更有针对性，并叮嘱他继续念九字真言。

四月二十几号，我去三哥家。刚一敲门，没想到是三哥来给我开的门。我一看，他气色比上次还好，在家里活动不需要侄女看护着了。他家住一楼，临走时，他还和侄女一起把我送出了单元楼大门。

五月十几号我骑着电动车走進他家楼群时，就听见侄女对他爸说：“我老姑……”我一看，三哥正在和一帮老头儿在一起坐着，听他们唠嗑呢。

见我来了，他们爷俩就和我一起往家走。三哥家在西面，中间隔两栋楼和一条道。我推着车跟三哥和侄女一起往他家走，看到他走路已经完全正常。之前他出门可都是靠侄女用轮椅推着的。

我对三哥说：“你身体现在恢复的这么好，你得谢谢李洪志大师，是大法师父救了你的命！”三哥跟着说：“谢谢李大师救了我！”

知道三哥病情的人都说：他小脑萎缩病情严重到那种程度，靠医院的医生治疗根本不可能恢复的这么好！只是因为大法至今仍然遭中共严重迫害，侄女出于怕和顾虑，不敢向人们说出实情，但家人心里都明白：这是大法师父的慈悲和大法的威力在三哥身上的展现！
