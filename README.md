### 默念九字真言解决了困扰女儿近十年的问题

文: 山东大法弟子 

【明慧网二零二二年六月十一日】二零一二年初秋的一天下午，我在银行遇到老同事S，办完业务后就和她聊天。因工作原因，我在原单位辞职后很少有时间再见到她，S非常善良，工作中从不计较个人得失、乐于助人，也非常敬业、加班加点从不喊冤叫屈、总是默默的尽自己所能，所以我非常信赖她。
我把自己这几年发生过的好多事或不知道如何解决的事情一股脑讲给她听。当把多年来一直困扰我的，女儿每天半夜总要惊醒害怕的事情说出来时（女儿五岁时因为她爷爷把我家门窗玻璃全砸了、受到惊吓后就每天半夜十二点左右哭醒）。S当即从脖子上取下她的护身符递到我手里，我在好奇的看这个护身符时，她告诉我要把护身符给孩子随身带着或戴到脖子上、让她每天默念“法轮大法好 真善忍好”这九个字，说睡觉的问题可能就会解决。

我听到“法轮功”，同时看到护身符上莲花图案的两边写有“法轮大法好 真善忍好”时，心里咯噔一下：难道她现在炼法轮功了？！她还正常吗？她没和我说过她学法轮功了啊……带着这大大的疑惑，我又抬头看了看她，心想：她挺正常啊……

在此之前，我从没听到过法轮功真相，但听说过有“自焚”，也没亲眼从电视上看过。大学毕业后也听高中同学提起过我们一位老师因炼法轮功，后来去了省城，再后来人就没了，那时也没多问只是心想，这老师怎么这样了呢？直到我得法修炼以后，進明慧网查询，才知道真相——我的那位老师是被非法关進监狱受到迫害。

女儿当时初二住校， 那个周末给她接回家后，我就拿出护身符告诉她是S阿姨给的，让她戴到脖子上每天没事时就默念“法轮大法好，真善忍好”，晚上睡觉就不会再半夜醒了，女儿没任何疑问就接受了。

之后，女儿不管在家里还是在学校，晚上都睡的好好的，并且告诉我哪怕在学校有时晚上起夜独自出去上厕所都不害怕了，说她每天都默念。这件事让我感到太不可思议了，怎么会这么神奇？！

我在修炼法轮功之前是个邪党灌输下的“无神论”，什么都不懂，也不知道该怎么帮她解决这个问题。直到遇到S给了法轮功护身符并每天默念九字真言，才真正解决了困扰女儿近十年的睡觉问题。所以后来又在路上遇到S时，我高兴的告诉她女儿睡觉好了，她又送了我小光盘，说用它翻墙上网可以了解好多国内看不到的消息。

通过翻墙软件，我真的看到了太多的信息，看到了世界各地法轮大法的洪传盛况，看到了各国政府、百姓及不同阶层的社会群体、包括世界有名的大学等等对法轮功的认可与欢迎，都给予法轮功极高的评价与赞誉，看到了法轮功在世界洪传以来已获得过数千项荣誉证书及褒奖等等诸多的社会赞誉。法轮功根本就不是共产党对民众宣传的那样，明白了大陆所有的宣传都是对大法和师父的诬蔑、诽谤，看到了“天安门自焚”伪案的全过程分析，那个造假的种种迹象人只要不糊涂都能看出破绽的，可是大陆百姓却被政府利用的所有宣传工具狂轰滥炸式的给骗住了，甚至对法轮功产生了敌对、偏见、排斥和莫名的恐惧……其实，被蒙在鼓里、不明真相的百姓才是最可怜可悲的人啊。法轮功教人向善做好人，真正修炼的人还能祛病健身。社会上好人多了，整体社会道德提升了，我们还用再担心父母在外面摔倒了无人敢扶、孩子在外面被骗被欺负吗？人人都按真善忍去做，还会再有地沟油、毒奶粉、毒疫苗等等坑害人的事件发生吗？

明白真相后的我毅然走入了大法修炼，修炼后的我身心收益，我的亲朋好友也都跟着受益良多。

我的两只眼睛一个375度，一个325度，虽然度数不是特别大，但近视眼的人都知道，只要平时离不开眼镜的人，一旦摘下就会感觉啥都看不清很闷人，对眼镜有种依赖感，我以前就是这样。可是修炼一段时间后我发现，戴着眼镜时总感觉眼睛特别不舒服，所以在家我就不戴只有出门时才戴上，有次我着急出门，下楼后想起来忘戴眼镜了，可是看看周围的环境，再抬头看看蓝天白云，一切感觉那么清晰，我怀疑自己是不是戴了眼镜，就习惯性的用手触了触鼻梁处，就是没戴，又对着旁边的车玻璃照了照，确认真的没戴，可是我居然看的那么清楚，我高兴极了，从此以后就再也不戴眼镜了。

二零一九年夏天，女儿想换副眼镜，陪她去配镜时，请验光师帮我测了一下，一只眼睛175度，一只125度，两只眼睛居然同时减少了200度，做手术可能都不会这么准确吧！这时我才悟到，是师父帮我调好了眼睛啊！因为我有个非常迷你的小MP3，里面装有电子版的大法经文，每天晚上都会在关灯后学习师父的讲法，那个字非常非常小，我都离眼睛很远的距离看，看的特别清楚，也没有眼睛疲劳的感觉。应该是师父看到我这颗真想修炼的心就帮我把视力提高了吧。

还有好多明真相得福报、三退保平安的事例，在此就不一一列举了，当下疫情形势严峻，如果不慎被隔离被封闭或被确诊感染病毒，如果自身还有其它疾病或突发意外，不要等到无药可医、无人可靠时才想起向天求助，希望更多的还不明白真相的有缘人都能冷静下来，抓紧时间了解真相吧。
