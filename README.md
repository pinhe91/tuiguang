### 我家老人敬念“法轮大法好”得福报

文:大陆大法弟子 

【明慧网二零二二年十二月二十九日】说说我家老人的事。老人遇难，诚念“法轮大法好，真善忍好”，闯过了晚年中的一关又一关。这里只举几件事例。

#### “他怎么能这么快就好了？”

前几年有一段时间，老人咳嗽了很长时间，吃了不少药也不见好转，后来还出现了低烧，身体日渐消瘦。一天，我们挂了医院内科主任医生的号。主任给老人做了检查，结果出来后，医生异常紧张的说老人肺的一侧中下部有一块很大的阴影，像是个实体，并说病人很危险要马上住院。我们说那就先回家做些住院的准备，主任说不行，并把我叫到一旁说，病人八、九成患的是肿瘤。这么大的肿块，如果不小心碰破裂或者肿块侵蚀血管引起血管破裂，那是很危险的。

入院后，老人一边治疗一边又做了一系列的检查，医生认为肿瘤的可能性很大，但是确诊还需要做病理切片检查。考虑到病人当时的情况，医生说等治疗一个阶段，略微好些再做。

这一切来得太突然，太可怕了。冷静下来后我想到了师父。我求师父帮助和加持我和家中老人，清除所有不好的东西，让老人能顺利走过这场魔难。同时我也劝老人：“您现在什么都不要想了，每天就是诚心敬念‘法轮大法好，真善忍好’吧。”老人可能也意识到自己病的不轻，于是每天躺在病床上不干别的，就是敬念“法轮大法好，真善忍好”。

几天后，医生给老人做复查。结果说肿块开始液化了。我心里高兴，知道病情是在向好的方向发展，我和老人说了复查结果，并让老人和我一起诚心感谢师父的救度和帮助。

大约又过了一周，老人被推去做病理切片检查。因手术進行了很长时间，我在外面就很担心。医生终于出来了，说：病人做术前检查过程比较困难比较麻烦，管子插進后病人就不断的吐，吐出很多黄色粘稠的痰，只能做一做，停一停。医生说现在已经做完了，叫我進去一起扶病人起来。

把老人扶起来后，我下意识的拍了拍他的背部，结果他又吐出了一大堆浓痰。那个痰不像一般的痰，让人很恶心。医生和护士都感到诧异，说没见过这种情况。出来后老人说呼吸顺畅了许多，感觉也轻松多了，还高兴的跟我说：“在手术室里我一直都在念‘法轮大法好，真善忍好’。”

病理检查结果出来了：只是炎症。前后两周老人就康复出院。这让医生们觉的很意外，但也很替老人高兴，同病区的病人更是惊讶：他怎么能这么快就好了？

出院后，我想到了中医一般都将肿瘤列为“痰痈”一类的，我还想到了中医和民间也有一个说法就是“邪有出路”。我想是师父将计就计把肿瘤打散后，再利用医院做活检穿刺的过程，巧妙的将“痰痈”引出，既简单又快捷，对病人损伤又小。

退一步说，即使是炎症，这么严重，两周就痊愈了，也是个奇迹啊！大法的威力真是妙不可言！老人的一场巨大的魔难就这样消失了。我们全家人都恭恭敬敬的给师父上香，感谢师父的救命之恩！

#### “哪里有什么错位突出啊？”

有一段时间老人出现比较严重的腰背疼痛，直不起腰来。到医院检查，医生说是脊柱退行性病变，还有颈椎、胸椎、腰椎各有一节错位突出，医生一边打开电脑给我们看，一边指着病变部位说，从上到下这三个部位都有一节脊椎有问题，就像一栋高楼承重的几个点都歪了，高楼怎么能负重呢？怎么能立起来？病人现在的情况只适合做保守治疗，但是保守治疗也没有更多更好的办法，能控制病情不发展就不错了。多吃些钙片和舒经活络的药吧。老人吃了一段时间的钙片和从医院拿来的药，没什么改善的感觉，就不吃了。

一天，老人不慎摔了一大跤躺在地上，怎么办？我们立即想到了请师父帮帮忙，并让他念“法轮大法好，真善忍好”。老人相信大法，躺在地上一直在念。我们查看他外表没有受伤，等他缓过来后，我们费了好大劲将他扶了起来。有的家人为慎重起见坚持要送老人到医院检查。

医生给他拍X光片又做CT，说没有什么大事，脊柱情况良好，我们说不对呀，他前一段时间检查时发现颈、胸、腰椎都有一节错位突出的呀。医生把电脑屏幕移给我们看，说：哪里有什么错位突出啊？骨质的亮度也不错。

我们都愕然！我立即想到是师父将老人的脊柱调整好了，谢谢师父！如果换成一般的老人，摔了这么重的一跤，真不知道会摔成什么样呢！同时病变的脊椎需要做多长时间的治疗才能好转、复位，年纪这么大的老人更不容易治愈。

#### 敬念法轮大法好　三天出院

几个月前的一天晚上，老人拿起手机看，看了很长时间，突然晕了，趴在桌子上。我们扶起他的头，看到他的脸色苍白，嘴角歪斜、流口水，失去知觉。用手拍拍他的脸，一点反应都没有，一摸他的手，冰凉！开始还有脉搏，但很微弱，很快就摸不到了，接着就没有了呼吸，脸色变的惨白。家人已经打了“120”。我抱着老人大声喊：“请师父救救他！请师父救救他！法轮大法好！真善忍好！”我不停的喊。

过了一会我用手拍拍他的背，看看他有没有反应，能否醒过来。突然他吐出一口痰，再拍，他又吐出一口痰，接着有了呼吸，脸色好转，也能摸到脉搏了，我流着眼泪连声说：“感谢师父的救命之恩！谢谢师父的救命之恩！”一会儿“120”急救车来了，医护人员把他抬上了救护车。半路上老人醒了，大家都松了口气。

在急诊室，我在老人耳边对他说：“你心里要念‘法轮大法好，真善忍好’，记住啊！”他点头答应。老人在急诊室做了各种检查。CT结果出来后，医生把我叫到办公室坐在他身旁，他一边看着电脑上的大脑图像，一边详细询问老人的情况，他用手指着图像说，病人的大脑图像有点奇怪，大脑上下的血管还好，但是中间两条主要的血管有一段看不到，好像是堵塞了，要么就是没拍好而看不到，一般是不会拍不到的，主血管旁边有血管侧枝，顶端的血管分布也好，小血管都可以看到。他从各个角度翻来覆去看了许久还是拿不定主意，最后，他打通了主任的电话，夜深了，还希望主任能帮忙看看。我心里很平静，心想师父已经帮老人的血管打通了，并在心里不停的谢师父！

主任看后对主治医生说：“先在急诊室观察治疗，我想大概是不需要做手术了。”第二天，神经科医生来会诊，说病人的基本情况还好，可以住院也可以不住院。老人自己说不住院了。他知道师父又一次救了他。躺在病床上在静心敬念“法轮大法好”。第三天，老人出院了。

老人这次发生的意外真是命悬一线啊。事后我们笑着说，他在阎王门口走了一趟。

老人的经历讲起来平淡，但是在那些过程中步步都惊心动魄。

慈悲的师父无时无刻不在看护着大法弟子和相信大法好的弟子的家人，真是“一人炼功全家受益”[1]。在危难时，诚心敬念“法轮大法好，真善忍好”，就会遇难呈祥得福报，真实不虚！当今的世界混乱不堪，天灾人祸连绵不断，谁能秉承善念，认识到“真、善、忍”是人类得救的希望，法轮大法的光辉一定会洒落在谁的身上。法轮大法救护着每一个善良的生命。
