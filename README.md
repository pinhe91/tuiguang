### 【征文选登】“人民英雄纪念碑”帮你看清真假抗日

文: 章世明 

【明慧网二零二二年三月九日】几十年前，“抗日剧”只有几部来回来去播放：“地雷战”、“地道战”、“铁道游击战”。随着影视编剧的市场越来越发达，五花八门的“抗日剧”层出不穷。中共的抗日英雄真有那么多吗？为什么在象征中共政治中心的天安门广场上，他们竖起的“人民英雄纪念碑”却只纪念三年内战的“英雄”呢？

#### “抗日剧”的迷魂阵

在“地雷战”时代，从来没有过一部影片是反映“正面战场”（国民政府抗日事迹）；近年来，随着互联网的发展，以及中共统战需要，放开了对于国军在“正面战场”抗战的内容管制，人们才知道：原来有那么多国军将士阵亡了！

目前，在国内只要打开电视机，黄金时段播出的十有八九是“抗日剧”。编剧水平高的，在描写国军正面战场的同时，总要加进去一个主角或关键人物，这个人是共产党，虽然国军抗日也付出了，但是这位关键人物起到了重要作用。这样的手法叫“掺沙子”，表面上是说国军抗日，实际上是说其中共产党抗日起到了重要作用。最迷惑人的，其实是这一类影视剧。

编剧水平低劣的，比比皆是，一部部虚构的荒唐剧情频频出现，如：“我爷爷九岁的时候就被日本人残忍地杀害了”，“八百里开外，一枪干掉鬼子机枪手”，“手榴弹炸飞机”等等。

这么多的“共军抗日事迹”？为什么在天安门广场的人民英雄碑上，却见不到一个共军抗日英雄的名字呢？

#### 谁是“抗日战争的中流砥柱”？

按照中共一九八七年通过的《档案法》第十九条规定，一般档案超过三十年即可解密，普通公民皆可查阅，然而中共至今仍不敢开放涉及抗日、内战的任何档案资料。

在中共的所有教科书与宣传中，都一直自称领导中国人民抗击日本侵略者，是“抗日战争的中流砥柱”，而蒋介石领导的国民政府则是“真投降、假抗日”。

我们来通过几个简单的事实分析一下，到底谁是“抗日战争的中流砥柱”：

1、军力数据：

在抗日全面爆发的一九三七年，国民政府拥有陆军二百七十万，而这时逃到延安的中共只有两万红军。在一九三七年八月十三日开始的淞沪会战中，国民党七十万军队与五十万日军浴血奋战，粉碎了日军三个月横扫中华的计划。远在陕北的 两万红军如何能主导一百二十万人的会战？

2、伤亡数据：

自一九三七年七月至一九四五年八月，中华民国政府军发动大型会战二十二次，重要战斗一千一百一十七次，国军伤亡三百二十二万多人，共有二百零六名国军将领在抗战中捐躯（据中华民国国防部一九四六年统计）。

二零一五年七月十四日，中共中央党史研究室对外公布，抗日战争中，中国军民伤亡三千五百万以上，其中军队伤亡三百八十余万，对于国民政府颁布的数据，中共没有否认。但是没有列示中共军队伤亡的具体数据，中共提供的是一个笼统的数据。

3、战役数据：

一九八三年，读卖新闻社所出版的《战争：中国侵略》所给出的数据是自一九三七年七月七日至战争结束为止，总共有七十多万日军死于中国战场。根据日本统计，日本军在整个二战死亡约一百八十五万人。

日军在华毙命的一百二十九名将领之中，死于与中共作战的只有三个，其余都是死于国民政府军队手中。

“平型关大捷”、“百团大战”是在中共教科书中，能拿得出手的两次战役。中共以前一直宣传“平型关大捷歼敌一万多”，近年悄悄改为一千多人。事实上，日本军史记载，“平型关战斗”日军仅伤亡二百多人。

蒋介石侍卫长、国军前参谋总长、国防部长郝柏村这样评价中共的抗战贡献：“‘平型关’、‘百团大战’是他们认为最了不起的贡献，如此而已，从整个抗战来说，不到百分之一！”

#### 假的可能猖獗 而真的总会留下痕迹

事实上，中共是不是“真抗日”，在中共自己出版物中都有明确的记载：

1、《毛泽东选集》中的记录

一九六一年一月二十四日，日本社会党代表团访华、国会议员黑田寿男、浅沼稻次郎等人被毛泽东接见。毛说：“日本的南乡三郎（日中输出入组理事长）见我时，一见面就说：日本侵略了中国，对不住你们。我对他说：我们不这样看，是日本军阀占领了大半个中国，因此教育了中国人民，不然我们……到现在也还在山上，不能到北京来看京戏……所以日本军阀丶垄断资本干了件好事，如果要感谢的话，我宁愿感谢日本军阀。”（《毛泽东文集第八卷》）

一九六四年七月十日，毛泽东对于日本社会党人士佐佐木的侵华道歉，毛说：“我们为什么要感谢日本皇军呢？就是日本皇军来了，我们和日本皇军打，才又和蒋介石合作。二万五千军队，打了八年，我们又发展到一百二十万军队，有一亿人口的根据地。你们说要不要感谢啊？”（一九六九年《毛泽东思想万岁》原文复刻，（716页版本）p.532—545）

2、《庐山会议实录》，作者李锐，曾任毛泽东秘书。

“林彪：平型关吃了亏，头脑发热。

毛泽东：一些同志认为日本占地越少越好，后来才统一认识：让日本多占地，才爱国。否则变成爱蒋介石的国了。”

“百团大战是无组织无纪律”，“反对毛主席的军事指挥”。八路军副司令员彭德怀后来在文革被清算时，进行“百团大战”是他“反对毛主席”的“罪状”之一。

3、《延安日记》

作者彼得•弗拉季米洛夫作为共产国际驻中共的联络员兼塔斯社随军记者，于一九四二年五月——一九四五年十一月，由斯大林从苏联派驻延安。

“毛泽东的原则是：进行这次战争为的是保存他自己的实力，而不是消灭敌人。要达到这个目的，就得对敌人放松抵抗和让出更多的地方。”

“在第120师师部里，讨论的中心不是战斗任务、作战和其他军事问题，而是怎么做买卖和赚钱。这一切都是奉命行事的。例如，已严令八路军和新四军各部队不得对日本人采取任何有利的作战活动。一句话，就是不准打仗，遭到攻击就往后撤，有可能就休战。”

到八年抗战结束时，国民政府的军力伤亡三百余万人，而中共军队从逃到延安的不足三万人，已经扩充到了一百二十万人，并最终通过三年内战窃取了政权，这也是为什么天安门广场的“人民英雄纪念碑”上刻写了这样的话：“三年以来，在人民解放战争和人民革命中牺牲的人民英雄们永垂不朽！”三年内战是实实在在的事实。三年内战，才是中共要纪念的“英雄”。

事实毕竟是事实，不会因谎言宣传的嚣张而改变。七十余年来，中共颠倒黑白，隐瞒历史真相，的确导致许多年轻人的思想充满了仇恨。了解真实的历史，事关我们的现在和将来。让大家一起来还原真相，让今天的年轻人走出谎言，找到历史真相！
