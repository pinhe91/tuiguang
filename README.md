### 诚念九字真言　大法师父帮我清除癌细胞

文: 中国大陆大法弟子家属 

【明慧网二零二二年六月五日】我今年七十多岁，老伴是一名大法弟子，她对大法非常坚定，经常出去讲真相救人，我知道大法好，所以对她所做的救人的事情非常支持。她经常告诉我念“法轮大法好，真善忍好”，我有时也念，但我还达不到百分之百相信。
我们家安装了一个大锅可以收看新唐人电视台节目，其中的电视新闻是我最喜欢的节目，几乎每天必看，通过收看新闻我对中共的腐败深有感触，并且了解到了最新的中国大陆真实情况。有时我坐大巴车去子女家时，在车上，我就会大声的给全车人讲电视台播放的新闻内容，一直讲到下车，车里的人都静静的听，没人反驳。

正因为我正面支持大法，所以大法师父帮我化解了一次大难，让我起死回生，事情经过是这样的：

两个多月前的一天，老伴刚好从她妹妹家回来。当天晚上半夜两点半左右，我突然感觉胃很不舒服，伴随恶心想吐，就赶忙爬起来，准备下炕去外面吐，结果刚走到炕沿边上，我一下就忍不住了，趴到炕沿上就吐了一口，全吐在了地上。

老伴听到我的声音，赶忙起身把灯打开，当时看到我吐的全部都是血，她第一念就是大声求师父：师父，求您救救某某某（我的名字）。然后她跟我说：你也赶紧求师父救你。我心里默默向大法师父求救，同时打手势示意她靠边站。

我忍不住吐了第二口血。老伴赶紧下地，边打电话给远在外地的子女，让他们打120救护车过来拉我，边拿来扫把和小桶，把地上的血收拾了一下，结果整整装了小半桶。收拾完，我让她给村里的一个老乡电话，让他过来陪我一起去医院。过了一会，我想去厕所，她扶我到厕所之后，我感觉自己拉的都是血，拉完后我站起来准备往屋里走，结果腿一软一下就瘫倒在地上，缓了一会儿后，我让老伴扶我挪到屋里，我坐到炕沿上，她出去把院子的大门打开。

等了一会，救护车和老乡正好也都过来了，就这样我被送到了市人民医院。一路上，我一直不断的念“法轮大法好，真善忍好”。

老伴没有跟去，她后来对我说，早上七点多钟，家里特别听话的狗突然死掉了，这条狗是我最喜欢的，它非常懂事，我很疼它，给它打针都是最贵的，平时它看家，有一点动静都会叫，告诉我外面有人。但我得病那天，救护车过来它一声没吭，应该是心里知道车是来救我的。家里的一个亲戚说：这条狗替我去了。我自己也想，事情发生的这么巧合，一定不是偶然的。

我在市医院呆了四天，子女就帮我转院到了锦州医院。这期间，我一直都默念九字真言。老伴给儿子和闺女都打电话，让他们全家一起帮我念“法轮大法好，真善忍好”，他们都很诚心的念，连我小孙子也很虔诚的帮我念。到了锦州医院，医生检查后说要做手术，需要切除三分之二的胃。

做手术那天，先打麻药，然后开始手术，進行的非常顺利。当把胃切下来之后，医生惊奇的发现，切下来的胃有三层，在第二层和三层之间有癌细胞，他还拿给我的孩子看，并对他说：你父亲非常幸运，一个手术解决了两个病症，这种概率非常小。

手术之后，我心里仍然是不断的默念九字真言。等麻醉药过劲后，我伤口只疼过一次，孩子让医生给我打了一次止痛针，而同病房的其他人，都会疼四五次。我知道这都是大法师父帮我减轻了病痛。

我出院回到家后身体恢复的很快，现在我能吃能干，已经基本恢复到得病前的状态了。在这里我要再次感谢大法师父，谢谢大法师父救了我的命！
