### 不分善恶追随迫害　各地民众遭恶报案例

【明慧网二零二二年七月九日】（明慧网通讯员大陆报道）人权恶棍江泽民和中共邪党相互利用，发动了对善良的法轮功学员的迫害，采用肉体上、精神上、经济上全方位迫害的政策，动用国家全部宣传机器，欲将谎言伪装成真相，欺骗民众，诱使他们参与迫害修心向善的法轮功学员。
然而，苍天有眼，善恶有报的天理不会放过每一个恶人和恶事。下面是参与迫害法轮功学员的各地民众遭恶报的事实。

#### 辽宁黑山县太和镇恶人张文典遭恶报突然死亡

辽宁黑山县太和镇包屯村恶人张文典，现年五十八岁，原包屯村民兵连长。

一九九九年“七•二零”以来，张文典一直追随江氏集团参与迫害本村法轮功学员，后来村干部选举时落选，一直心里愤愤不平，四处告状。

二零一零年初，由于张文典的告状引来辽宁乡村台记者刘海鹏，张文典拿出法轮功学员善意送给他的二零一零年年历，刘海鹏恶意诬陷法轮功学员的报导发表后，辽宁省公安厅对黑山县公安局施压。在二零一零年二月二十三日，由市、县公安局的头头亲自坐镇，当晚对黑山县法轮功学员大清查，绑架了十二名法轮功学员。后多数被非法劳教。

二零二二年以来，张文典接连遭恶报。二零二二年四月十二日左右，张文典请同村的一个身体硬朗的人帮他家干活。吃晚饭时，那个人突然死亡。张文典给了他家人两万元作为补偿。

二零二二年五月二十一日晚上，在什么症状都没有的情况下，张文典竟突然死亡。家里还有一个三十多岁的儿子没订婚。

#### 河北省廊坊市堂二里镇九街村民张志军遭恶报死亡

河北省廊坊市堂二里镇九街村民张志军，在堂二里镇镇政府信访办工作，执行恶党命令，带领镇政府和居民委员会七、八个人到法轮功学员王清敏家，非法抄走大法书四本和真相资料以及光盘等，同时还诽谤师父、诽谤大法，不让法轮功学员讲真相，并且积极响应恶党指令，要求所有民众对法轮功人员进行举报。

张志军还参与迫害了法轮功学员荣桂侠和王清敏两人。

张志军坏事做尽，于二零二二年五月份，得鼻癌死亡，终年四十二岁。张志军的死亡印证了凡是迫害法轮功学员的恶人、恶警终将会遭到报应，善恶有报是天理。

#### 四川省德阳市孝感镇金马村村民李长长遭恶报

四川省德阳市孝感镇金马村一组村民，李长长，男，二十三岁，一九九八年出生，当过两年兵，退伍后，经由其原时任村主任的幺爷，进了派出所当协警。上班后，没多久，二零一九年九月二日，李长长诬告自己住处对面的一对老年夫妻法轮功学员，使他们夫妻同遭绑架、被非法抄家、拘留。男法轮功学员，因年龄太大，在家执行，让法轮功学员和他们的子女受到了严重干扰与迫害。

二零一九年十一月二十一日，本组一老年女法轮功学员在外向民众讲大法真相，被不明真相的人构陷，遭绑架至派出所。由于法轮功学员不配合，警察拉法轮功学员按手印按不上。

李长长一点不顾及本村组、并且是一个家族的人的情面，他说，按不上手印，就弄点水，然后强行拉上该法轮功学员的手往上按。该法轮功学员善意地对他说：你不要这样做，这样做对你不好，我们都是做好人的。该法轮功学员给他说，他不听，还把该法轮功学员强行推上车，拉去做各种检查，准备迫害该法轮功学员。

过后没多久，李长长在知悉派出所警察去抓吸毒犯时，私下给吸毒者通风报信，后李长长被派出所开除。

二零二一年的夏天，李长长与他人出去游玩，他在坐滑板车时，被重重地摔了一跤，脚被摔断，后因感染，一只脚被锯掉，花了几十万元钱，还在网上呼吁大家捐钱，差点送命。虽然他才二十三岁，年纪轻轻的，但不能工作，还常常神经兮兮的，很是可怜。

真心的希望世人都明真相，善待大法和大法弟子得福报。不要去听所谓上级的迫害指令，受蒙蔽上当受骗当枪使，恶报祸及自己连累家人。

#### 广西隆安县雁江镇中学校长曾祖德遭恶报

原隆安县雁江镇中学校长曾祖德，曾经带领学生指证一名法轮功学员的讲真相活动，致使这名法轮功学员遭非法判刑。曾祖德是迫害该法轮功学员的主要元凶之一，二零二二年六月得知，曾祖德遭恶报，现在因心脏病而做心脏搭桥手术，生命处于危险之中。

#### 贵州省贵阳市修文县一小区宋姓门卫遭恶报

在修文县龙场镇一个住宅小区的门卫因为敌视法轮大法而遭到了恶报，从发病到死亡不到一个月时间。以此证明了善恶有报的天理。愿世人不要跟随共产党作恶，珍惜自己的生命。

在贵州省贵阳市修文县的一个住宅小区，有一个看大门的门卫。人称宋老师，是某中学校的一位退休教师，因为身体好，退了休，在这个小区当门卫。

十几年前，本小区就有一家母女两人都是修炼法轮功的，这个门卫被人指使经常跟踪、监视她们。经过她们的讲真相，门卫嘴上说不反对了，其实心里根本不相信。

二零一四年，这家女儿被绑架、冤判入狱了，这个门卫更是积极、卖力地跟踪她母亲，并监视她家庭出入的人员。

二零一六年这家母亲过八十岁生日，四川老家的亲戚开车来，给老人过生日。门卫看到有人、有车来到她家，就报告派出所，把她家亲戚的车牌号抄下，报告到四川当地派出所。

她亲戚回四川后，当地派出所就上门问他们为什么到法轮功家去。以后每到过年、过节时，四川当地派出所都要上门骚扰她家亲戚，问有没有再到法轮功家去？搞得她家亲戚从此以后都不敢和她家有接触了，甚至电话都不敢打了。

二零一六年八月份，宋老师做了这个损德的恶事，谁知到了二零一七年一月份，他就得了癌症，很快二月份，人就死亡了，从发病到死亡不到一个月时间。

我们听了这个事，为他难过又痛心。为什么给他讲真相他不但不听，还要助纣为虐呢？这个报应来得好快啊。真是人容天都不容啊。
