### 相信大法好　老伴、儿子、孙女都得福报

文: 中国大陆大法弟子 

【明慧网二零二二年八月一日】我今年七十二岁，修炼之前，体弱多病，生孩子差点走了，走路还得扶着墙。熟人见了老伴儿就问：“你怎么找了这么老的太太？”一九九七年修炼法轮大法之后，我神采奕奕，无病一身轻，走路生风。同事见了我就问：“你怎么找了这么老的物件，看你多显年轻呀！”我自豪的说：“我炼法轮功了，身体没有病，所以显得年轻了。”
在二十多年修炼中，我都坚持天天学法、炼功。伺候儿媳妇坐月子，半夜回家，我也要学法、炼功，从不懈怠。全家人虽然只有我一人炼功，可都知道大法好，多年来人人都受益显神奇。

#### 一、老伴相信大法好，糖尿病不治自愈

前几年，老伴得了严重的糖尿病，脚趾头溃烂，肿的像面包，根本没法穿鞋。一个星期六，老伴去医院就诊，医生看了摇摇头说：“没有好办法，再厉害就得截肢。”老伴害怕了，心事重重去找按摩师推拿，结果越来越厉害，回来连路都不能走了。

我安慰老伴说：“老伴儿，你看你那么胖，哪像糖尿病人？你放心，不会截肢的。你没有这个罪，我也没有这个难，师父没有给我安排这样的路，一切都是师父说了算。我炼功，你也受益。你看医院没有好办法，推拿更厉害了。咱还是相信大法，求师父帮助你吧。”

老伴信服的说：“好吧！”就在心里默念九字真言：“法轮大法好！真善忍好!”

我想：二十多年来，老伴儿一直支持我修炼。我在修炼中，有时出现消病业的情况，或者不守心性和他人出现摩擦时，老伴儿曾经对我修炼产生怀疑，对大法有看法。我就对老伴说：“不是大法不好，是我没做好，就像学生有考100分，有及格的，也有不及格的。是我做的不对，没有符合大法的要求。”

我告诉他：人得病或遇到什么麻烦事都是业力造成的。你就常念“法轮大法好，真善忍好”，师父会帮着消业，实际上师父为我们承担了很多，咱只是承受一点点。老伴儿听后笑了，消去了一切疑虑。

星期一早晨，老伴儿的脚奇迹般的消肿，脚趾溃烂消失，两脚完全恢复正常了，他穿上皮鞋上班去了。后来医院的大夫知道了，惊叹道：“不可思议，不可能的事，太神奇了。”

#### 二、儿子明真相受益，躲过灾难

儿子和儿媳明白大法真相，都做了三退，他们也支持我修炼。每次儿媳看到我发正念，她就赶快抱着孙女悄悄躲开，害怕影响我发正念。

几年前，儿子和朋友合伙做生意，朋友两次骗儿子二百多万。面临这一场灾难，儿子愁的整天睡不着觉，儿媳妇也是愁眉不展，觉的在亲朋好友面前很没有面子，抬不起头来。儿子和儿媳妇急的找到我们老俩口想办法，让我们帮着还债。当时，就是倾家荡产也还不起。我对他们说：“别着急，不失就不会得，凡事都有因缘关系，也许生生世世在轮回中，咱欠了人家的，欠债就得还。放心吧，都能解决的。”

此事我没有动心，放下儿女情。我就是信师信法，一切都有师尊说了算，我一如既往的还是天天出门讲真相、救人。

不久，全国各地房价猛涨，儿子原来用103万买了一套房子，结果卖出了260万，欠债的问题一下子就解决了，儿子马上还了所有的欠款。

#### 三、孙女学大法得福报

孙女从小聪明伶俐，跟我长大。孙女有时身体不舒服，儿媳要送孙女去医院看病。孙女就说：“我去奶奶家，奶奶家有师父，有能量，去了就好了。”果真跟着我住上一、二天就好了。

孙女从小就和我老伴不对脾气，祖孙俩一见面，没说几句就得吵架，她从不和我老伴玩耍。我想只有大法能解开祖孙俩世俗的恩怨，我就教孙女背师尊的诗词《洪吟》，给她讲古代尊老爱幼传统文化故事。

有一天晚上，孙女跟着我睡觉，半夜三点醒来，坐起来对我说：“奶奶，您去叫爷爷和我一起下棋好吗？”老伴一听乐的都合不上嘴了。从此祖孙俩一见面就有说有笑，非常开心。

一天，孙女放学回家，一進门就说：“奶奶，今天辅导员说邪党怎么好，说大法不好。我告诉同学别相信那些谎言。”孙女问同学：“台湾和青岛哪里的水清啊？”同学都说：“当然是台湾的水清。”孙女说：“对吧，报纸上的宣传都是相反的，能相信邪党的话吗？”孙女告诉我，当时，八个同学议论纷纷，有五个同学都表示反对邪党，说邪党太坏了！

听了孙女的话，我既感到高兴又有些担心，高兴更多的孩子开始觉醒，看清邪党的丑恶面目，也担心那些深受邪党毒害的孩子。为此，我知道得加快救人的步伐，救救那些被邪党蒙骗的世人。

这些年，孙女在大法中受益得福报了，从小学到上中学成绩优秀。去年，外语考试全级部前二十名，成了免费生。

在此我代表一家人叩谢师尊慈悲救度！
