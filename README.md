### 齐齐哈尔法轮功学员夏婷婷被绑架迫害经过

【明慧网二零二二年五月九日】（明慧网通讯员黑龙江报道）齐齐哈尔市龙沙区、铁锋区警察二零二二年四月份绑架、骚扰多位法轮功学员。据不完全统计，至少有16名法轮功学员被绑架、骚扰；另有学员被监控中。
法轮功学员夏婷婷四月十一日被龙沙区国保大队长姜浩等警察绑架，警察抄家抢走了许多私人物品。下面是夏婷婷诉述被绑架迫害经过：

二零二二年四月十一日十八时三十分，有人敲门，问是谁？没吱声，我就把门打开了，闯进来四名便衣（国保）和一名着装警察（五龙派出所警察），为首的是龙沙分局国保警察姜浩，他进来就问我认识他吗，我说你是姜浩。姜浩问我最近忙什么呢？我说我忙着要服刑期间被无理扣发的养老金呢。姜浩说让你去分局询问点情况。姜浩手里拿着一张纸，没让我看是什么。我说我先换衣服。

没等我换衣服，姜浩和另一名便衣拿着录像的东西闯进我住的小屋，边翻边录像，把床、衣柜、褥子底下、梳妆台的所有抽屉都翻个遍，还把洗衣机盖打开看了看。最后非法抄走了一台笔记本电脑、一台打印机、十几本大法书、一本手抄大法书、一个学习笔记本、三部手机、一个优盘、两个读卡器、一个小音箱、四十多份真相资料、八十元真相币、一包A4打印纸、半包B5打印纸、六包塑料袋、十多个美纹纸胶袋等。

他们把我带到龙沙分局刑警队，问我上网揭露迫害的文章是我发表的吗？我拒绝回答，姜浩说拒绝回答就是“是”，还问一些问题，我都拒绝回答，只回答一些自然信息。

我在龙沙分局坐了一宿，十二日上午姜浩和一名国保头目（好像姓曹）问我电脑密码，我说我不能说，说了是害你们。十二日中午，把我和另六人拉到五龙派出所，一名正阳派出所姓董的警察负责看管。董姓警察说前些年每个派出所迫害法轮功他都参与了，还污蔑大法，说其他人都没啥事能回家，只有我不能回家。

十二日晚上，五龙派出所警察徐喜山和黄姓警察又做讯问笔录，多数问题我都拒绝回答。他们按照国保给的笔录打字，打完让我看，有很多都是瞎编的，例如有个问题是问我都往哪发资料了，回答是记不清都发哪里了，这个问题是编造的，还有说杨淑芹揭露迫害的文章是我给上网发表的，那篇文章是二零二零年七月二十一日发表的，那个时间我还在女子监狱被迫害呢，还有很多伪造口供的地方，我给他们一一指出，过后我看到还是那些，根本没有改。

十二日晚上，五龙派出所所长王文军来劝我吃饭，因为我从被绑架开始一直绝食绝水，我告诉他我不在派出所吃饭，我的身体没问题，他看我精神状态很好，也就放心了，他说今天晚上或明天上午就让我回家。

十三日上午，分局来了两名女警察给我照相，晚上派出所的两名男警察给我照相后又采集指纹，我不配合采集指纹，他们说吃完饭强制采集。晚上九点多，徐喜山把我丈夫找来，要让我丈夫领我到附近饭店吃点饭，我拒绝。晚上十点多，在非法关押我五十一个小时后，放我回家。让我丈夫签的字。我始终未签字。

齐齐哈尔市法轮功学员夏婷婷，二零一七年六月二十八日早上七点钟，被姜浩等五、六名警察闯入家中绑架，被构陷、枉判三年六个月，二零一八年十一月七日被送到女子监狱继续非法关押迫害，二零二零年十二月二十八日才结束冤狱回家。
