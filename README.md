### 三退是真福

文: 大陆大法弟子

【明慧网二零二二年五月二十六日】在这些年讲真相劝三退过程中，发生了许多世人三退后得福报的感人故事，现在与大家共同分享几例。

#### 一、我家这几年可顺了

二零一三年我去山东，目地是走亲访友救人，我顺便到了我堂兄的孙子家。孙子大强和孙媳妇小凤两人在二零零六年我来山东时给他们做过三退。到了他们家有点认不出来了，已经大变样了。小凤见到我后特高兴，進屋落座后我问：这几年你家变化怎么这么大？小凤说：二爷爷你不知道啊，我家这几年可顺了，变化可大了！接着，小凤就讲了起来：

二零零六年秋，大强去工厂打工了 ，厂长叫大强干着两份工作，开两份儿工资，每月七千元，今天大强上班去了不在家。二零零八年秋我也到工厂打工了每月一千八百元。这几年，我家盖了新房，建了车库，还建了两米高的院墙，重建了大门口，所以一切都是新的了，二零一零年还生了个小男孩，现在去幼儿园了。还有两个逢凶化吉，遇难呈祥的事呢！

一个是二零零七年秋一天大强休息，开着拖拉机和他四叔去拉玉米秸子，没注意把拖拉机开到了公路边儿的沟里了，他先掉進沟里的，拖拉机滑下去后车轮压住了大强的双腿，他四叔急忙去救他，但怎么也抬不动车轮，他四叔又开拖拉机回村找人，来了一帮人才把大强从车轮下拖出来，大强站起来试了试哪儿也没坏，大伙儿帮忙把拖拉机从沟里弄上来，大强又去把玉米秸拉回家了。

第二件事是零八年春，一天大强骑摩托车去上夜班，光顾骑车往前跑，不知道公路前面还停着一辆汽车，一下子头就撞到了汽车上，幸好戴着头盔，当时就撞昏了什么也不知道了，醒来后发现自己在公路边儿和摩托车一块儿躺着，只知道撞车时的一点记忆其它什么也不知道，大概是司机发现后，把他拖到路边儿了，司机开车跑了。他起来后试了试没啥事儿，就又骑摩托车上班去了。这就是大强所经历的不幸中万幸的两件事。

二爷爷，现在村里人都羡慕我们，他们说，哎呦，人家怎么那么好、那么年轻就过得那么好，那么走运，过得那么富，出那么大的事都啥事儿没有，老天爷，光看护他们了。

我对小凤说：你知道你家为啥这么顺吗？就是因为你们做了三退的原因，你们顺应天意神佛就保护你们，其实真正保护你们的是法轮功师父，福报都是师父给的，这些年做了三退后得福报的例子很多，你以后把你家的事，向亲戚朋友讲讲，让更多人明白法轮功真相，要更多人的得救受益，你家的福报会更多。小凤说：行。

我要走了，小凤儿希望我晚上能来她家吃饭，他们夫妻俩早就盼望着，我能来他们家在给他们多讲讲法轮功的故事。

#### 二、宝马车的来历

玉洁是个农村妹，二零零二年十八岁進城打工，所干的活都是站柜台当卖货员，玉洁姑娘人虽小，但很有志气，就想将来当老板，为了积累经验、增长知识，她换过好多岗位，卖过各种商品。后来自己真的开起了超市，为了找个靠山、有个帮手，二零零四年与我妻侄结婚了（我妻侄是進城农民开出租的），自那时起，我们经常见面同时给他们做了三退。以后玉洁把他父母、弟弟、妹妹也接到城里，一同打拼，体验城市生活。

玉洁很认同大法，经常到我家拿大法资料看。二零一四年夏，玉洁在我家看到了印有“法轮大法好，真善忍好”救人用的塑料小条幅（25厘米乘13厘米）。我告诉她这是救人用的九字真言。这个挂在家里是最吉祥的，干什么都顺。玉洁说：那么好，我要几个，给我家的人各房间都挂上。于是就选了五个放兜里了。

这几年，玉洁和弟弟、妹妹合资办了个较大的超市，玉洁是老板，从超市开业效益就很好。玉洁的婆婆说：这几年更发了。

去年，二零二一年玉洁弟弟花七十万元买了一个家属楼，玉洁花四十万元买了辆宝马车，二零二一年女儿又考上了大学，真是双喜临门。认识玉洁的人都说十八岁的小姑娘進城打工，步步顺步步旺，财也发了，宝马车也开上了，老板也当上了真有福。其实这都是大法给的！

#### 三、享过五福的老太太

二零零二年秋，我们夫妻为了讲真相和证实大法，去了离我家几百里路的一个小山村，这个村很小只有几十户人家。这个村人大部份都是我岳母家族的人，以前这个村的人根本不知道法轮功真相，真实的法轮功他们一无所知。他们所知道的就是中共电视台对法轮功的造谣，抹黑宣传，我妻弟还和孩子们说：离你大姑远点儿，她是炼法轮功的，别叫她给杀了。另外这里的人都喜欢抓鱼、抓鸟、抓林蛙、抓野猪、抓狍子等，这些成了他们的第二职业，挣钱的门路。

为了接近他们，讲真相，我们就主动的帮他们干活。一天，有十多人在村里路边修水沟，我也拿了一件儿工具和他们一起干，同时给他们讲真相，休息的时候，村里一个队长说：听说你炼法轮功？我说：是。他说：“法轮功天安门自焚”、“又杀人、又上吊”，国家都不让炼。我说：都是共产党给法轮功造谣，抹黑，那些人不是真炼法轮功的。法轮功是禁止杀生的，杀生是有很大罪的，小鸟、小鱼、小动物都不准杀，能杀人吗？自杀也是杀生，有很大的罪，杀生问题在法轮功书籍里都写得明明白白的。“自焚”是骗人的，二零零一年八月十四日，联合国教育发展组织在联合国会议上，就“天安门自焚”事件，强烈谴责中共当局的“国家恐怖主义行径”，声明指出，录像分析表明，整个事件是“政府一手导演的”，在场的中共代表面对确凿的证据，没有辩词，该声明当时被联合国备案。当时江泽民集团导演自焚假案的目地是栽赃法轮功，挑动和激化民众对法轮功的仇恨。

我问：队长，你知道法轮功是怎么回事吗？队长摇了摇头说：不清楚。我说：是修炼，是按照宇宙特性真、善、忍修炼。真就是说真话、办真事、不说谎、不骗人；善就是重德行善，做好人、做好事；忍就是遇到矛盾要忍让、宽容，做到打不还手，骂不还口。还要做到不吸烟、不喝酒；利益面前不争不斗，有了矛盾向内找，看自己，不贪不占，就是按照这样的要求去做好人，做更好的人，另外还有五套炼功动作。法轮功还有个更大的好处，就是祛病健身有奇效，无论你身上有几种病，多难治的病包括癌症，只要你诚心修炼法轮功，就能痊愈。你会无病一身轻，我见过很多这样的人。法轮功被迫害之前，炼法轮功人数飞速增长，从一九九二年五月传出开始至一九九九年十月，炼功人数达到一亿人。队长说：怪不得共产党不让炼，这么好、人数这么多，它能不害怕吗？！

通过我们到各家各户的去讲，这个村的人基本都明白了法轮功是好的，电视是在给法轮功造谣，“天安门自焚”的不是炼法轮功的，是栽赃法轮功。

我岳母对法轮功从不理解到支持。一天她三女儿来岳母家动员老太太信基督教，老太太说我不信基督教，我信法轮功。还有一天，在我四小姨子家炕上放着两本书，一本是《转法轮》，一本是普通杂志，我岳母手指着《转法轮》对我四连襟说：你看这个、你看这个。又用手指着那本杂志说：你别看那个、你别看那个。这老太太是对大法表了态了。

这老太太一辈子生了九个孩子，五男四女，每个儿女家又都育有两个孩子，每家四口人共三十六人，有三十五人“三退”（一个信基督教的没退）。

孩子们都成家后，我岳母为了图清静自由，不与儿女们一起住，自己单过。老太太烟酒茶都好，自己随便。养鸡、种菜她认为锻炼身体很好，饮食方面鸡鸭鱼肉，样样不缺，养老费花不完，村里人说老太太真有福。

二零一三年十一月一天早上老太太吃饭以后就吐了，小女儿来了，煮了几个鸡蛋，吃了两个，这身体正常没啥事儿，两个儿子也来看看，没啥事儿，都在那儿坐着说话。九点多的时候，小女儿说：我回家包点饺子。十点小女儿把饺子端来了，一看老太太不在屋里，两个哥哥说刚出去一小会儿，说出去活动活动，小女在院子里找，看到老太太在柴草边坐着，小女说：娘，你怎么在这坐着？话音刚落老太太头就低下了，小女儿过去摸已经没气了。老太太就这样走了，老太太时年八十八岁，是无病善终。

民间说，长寿、富贵、安宁、好德、善终为“五福”。老太太就是享了“五福”的人。
