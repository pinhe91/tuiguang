### 诚念“法轮大法好”显奇迹

文: 中国大陆大法弟子 更新: 2022年10月08日

【明慧网二零二二年十月八日】

#### 姥姥的故事

同学的姥姥只有他母亲这一个独生女。可是问题来了，随着姥姥年龄大了，到八十来岁的时候，母亲也五十多岁了，要照顾老公（同学的父亲是当官的，不做家务）、要给儿子媳妇做饭、还要带孙女，就没精力照顾老母亲了。所以，姥姥一个人自己住，自己买菜做饭，自己照顾自己。

姥姥有个毛病就是视力很不好，而且年龄大了，经常爱碰着、摔着，而且一摔跤，就容易骨折。所以，姥姥身上好多处都骨折过。一骨折，就挤在同学家，由同学的母亲照顾，骨折好后，又回到自己住处，一个人生活。年龄大的姥姥一个人过得其实蛮艰难的。
后来有人送给同学一本法轮功的小册子，里面讲了诚念“法轮大法好”显神迹的故事。同学念给姥姥听，姥姥听了，也要念“法轮大法好”。

同学就给姥姥买了念佛的串珠，方便姥姥记数。从此以后，姥姥每天都念“法轮大法好”了。姥姥心很诚，每天都念一千遍以上。如果哪天有事要忙，那最少也要念够五百遍。

姥姥视力不好，还是会经常碰着、摔着，但是神奇的是，从此以后，姥姥再也没有骨折过！最重的一次是头上撞起一个小包。就这样，姥姥靠着诚念“法轮大法好”，又顽强地活了十多年，直到九十多岁高寿，才安然去世。

这就是姥姥的神奇故事。

#### 我的故事

二零零六年，因为工作太累，无法休息，肺部出了状况，住院治疗。医生开了检查单，要做纤支镜检查，检查室和住院部不是同一栋大楼。检查时，医生将纤支镜从我的鼻腔插入、進入气管、支气管、再進入肺部检查。

在進入气管（咽喉）处喷射了麻药。可能麻药喷射过多，检查结束时，我从检查室去住院部的路上，麻药劲一下上来了，整个脖子都麻痹，无法呼吸了。这种感觉很恐怖，就好象有人勒住脖子，要憋死一样。当我走進住院部电梯间的时候，感觉人快要“挂了”，望着空荡荡的电梯间，我要独自面对死亡了。

记得当时我很平静，这时，我突然想起以前同学教我念的“法轮大法好”，我想，好吧，在死之前，我就多念几遍“法轮大法好”！神奇的是，我还没有念几遍，也就是电梯刚到三楼的时候，突然感觉气管“啪”被冲开个小口子，我能艰难地吸点气了。当时求生欲一下上来了，我想赶紧去找医生给我上氧气。

当电梯上到五楼，我赶到医生办公门口的时候，这时气管“啪”又冲开个小口了，我可以艰难的呼吸了。我想，那就不用上氧气了，我可以挺过来的。就这样，我坐在病房床上，靠着这两个小口子慢慢呼吸，闯了过来！过了很长时间，麻痹状态才慢慢消失。

现在想想都觉的神奇，我只不过念了几遍“法轮大法好”，就从死亡边缘拉回来！感恩师尊！

后来，有机会我就把“姥姥的故事”和“我的故事”讲给别人听。这两个故事算得上是我的珍藏。

这两天突然有个念头出来要把这两个故事写到网上去与大家分享，让更多的人也能像姥姥和我一样受益。
