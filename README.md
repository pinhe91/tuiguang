### 明真相保护大法弟子的警察得福报

文: 大陆大法弟子

【明慧网二零二二年九月十五日】我老家在中原某省，后来在南方打工。二零零一年四月，我因到北京为法轮功讨公道，被中共610（中共为迫害法轮功专门成立的非法机构）绑架到某劳教所非法关押。该劳教所中队长是刚刚调来的，因为那段时间生病，一切事宜就由刚调来的杨副队长（化名，人称杨队）负责，杨队长得五大三粗，说话言语粗暴。包夹告诉我：他是专门调来迫害法轮功学员的，并嘱咐我小心提防。
第二天杨队找我到中队办公室谈话，问我为何高兴，唱歌唱得挺起劲，自己不休息还为百多号人洗衣，又为病号做菜。我讲他们出工辛苦，很多人半夜才下班，有的连手脚都没精力洗就睡了，有的个别人虱子成堆。我是修真、善、忍的大法弟子，看不下去，所以就每天帮他们义务洗一点。我又告诉他，前队长曾宣布三条：一不准炼功，二不准讲真相，三不准自由行走活动。我告诉他我做不到，他威胁说做不到就关禁闭，再往后就加教一年。我对他讲死都没怕过还怕这些？我本来也不怎么会唱歌，唱歌之前就请师父帮我。杨队说：难怪他们说你唱的好听，原来请你师父帮你了。我说没有师父帮我怎么能唱出这么好的歌？这一唱就打开了讲真相的局面。杨队问我讲些什么？我说主要讲“天安门自焚”是伪案。在没有迫害法轮功前电视报纸都是正面宣传法轮功，是江泽民出于小人妒嫉搞迫害。

杨队又问我：“四·二五”去中南海何为？我讲“四·二五”是象林冲误入白虎堂，大家去是为了讲明真相而表达诉求。九八年乔石委员长率全国人大对法轮功做过调查，认可大法治病有效率达百分之九十七，于国家安定，于人民身体健康，及社会治安的好转有百利而无一害。江罗一伙为挑起全国人民仇恨法轮功编造“自焚”伪案，小思影气管割开还唱歌，一个王進东成了三个王進东的相貌，成了三种耳朵（长耳、小耳、大圆耳），王進东盘腿姿式是军人式的姿式，不是法轮功的姿式。我演示了大法弟子盘腿的姿式给他看。装汽油的塑料瓶不燃烧，结印姿式也是错的，我也演示结印给他看了。喊的口号不是大法师父讲的，这分明是假的。我师父在讲法中讲了自杀是有罪的。这本书我看过一遍，你们可以去找这本书看。结果中队六个警察都找书看了，还有大队所有官员们也都看了，他们都知道了自焚是栽赃法轮功的。
最后杨队说相信我讲的。我要他善待大法弟子，自己会得福报的。他愁眉苦脸地对我说他妻子怀孕是一支花（女娃），又只准生一个，想生个儿子传宗接代。我说你这么善良一定会有福报的，只要你真心相信我师父，你一定会生儿子的，这是要在行为上做到才行的。他讲他会办到的，看到我言行一致，不象别人讲的那样，愿意相信我讲的。

那年冬天，一个吸毒犯解教回家时，因从他身上搜出为我们几个大法弟子带的家信，正职要挟不放人走了，激起几十人在办公室殴打一个同修，我出面制止后，另一同修告诉我只有马上炼功才能解决问题。我们就开始炼功，正职叫包夹把我们劝拉开后，说他正在给杨队打电话，杨队马上赶回中队商量解决问题。所领导扬言要关禁闭呀加期呀，这样那样的叫嚷不停。我们都发正念，否定他的一切迫害。

杨队来后宣布：一是马上放解教人回家；二是不同意关我们禁闭；三是炼功问题，讲前任不早就知道他们炼功吗？不是我们来了他们才炼的，默认他们炼的自己升迁走了，别人当聪明人，我们当傻子？！四是不让我们看诬蔑法轮功的“自焚”电视和出邪恶黑板报（用粉笔写）。又智慧的说：让他们看电视，他们就讲真相，结果我们七个警察都知道了真相，这种事别人怎么看我们？他们（指大法弟子）言行一致，连上面都默认他们，今天的事就当大家都不知道，仍象过去那样睁只眼闭只眼，只当没发生，搞生产为重要，如果一搞他们，可能全队人员都会罢工罢餐了。那时你我就要下台了。陈警察（化名）马上发言支持杨队，这件事就不了了之了。

一天早上，两个年轻同修起床晚了，早上集合时还在炼功。那天是正职上班，他很邪恶，就把两名同修捆吊在晒衣的铁栏杆上。我看见后很严厉的指着他，要他马上放人，他说一放又炼功咋办？我说那我不管，现在是吃早餐的时候，雷公不打吃饭人，出了事我就找你，你就得下台，你老婆就会和你离婚！现在回想起来当时我缺乏慈悲心，就是义正词严的震慑他。他就赶紧给杨队打电话，这时陈警察传达杨队的意见，叫包夹们放人。

杨队来后，这两个年轻同修就一起给他讲真相，我当时还以为杨队要大骂我一顿，没想到他却和善的说：你不要管他们炼不炼，喊别人炼就成了集体炼功了。你叫我好意思在桌面上讲吗？我体谅到他的难处，赶紧向他道了歉，承认自己没有和善地对待正职，请他转告正职，他也应承了。后来正职再也没有搞迫害了。

杨队因为善待大法弟子，后来他妻子分娩时果真生了个儿子，他非常高兴地告诉我：他亲友们都称谢法轮功师父伟大，说今后还要为我们多做善事。我叫他以后为人行事按照真、善、忍去做，他答应了。说他有环境也会修炼大法的。

再后来杨队升迁走了，陈警也得福报接任了杨队的职位。杨队走之前特意向我表示谢意，我叫他谢师父，不要谢我，告诉他无论在哪都要善待大法及大法弟子，自己才会有好的未来，他答应了。

以上是两个警察得福报的故事。当时的劳教所规定：警察要同我们大法弟子谈话必须要有两人以上，说是防止我们转化他们。那时警察们的心理状态是人人都想踩着别人往上爬，而又人人都想背着别人先了解真相。他们要了解真相是多么难。

大疫当前，希望广大公检法司警察不要听信邪党的谎言宣传，要理智辨别是非真假。早日退出中共党团队组织，并真心念“法轮大法好，真善忍好”九字真言，你会拥有美好未来，各种灾难都会远离你。
