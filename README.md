### 丈夫为大法说公道话得福报

文: 中国大陆大法弟子  

【明慧网二零二二年六月十二日】我是一九九七年开始修炼大法的，当时我还不到四十岁，这是人一生中最好的时光。可我却是一个体弱多病，弱不禁风的人，干什么都力不从心，整天昏昏沉沉，四肢无力，就连给学生讲课的力气都没有。
我经常请病假，身体不舒服，心里也烦，看见谁都不高兴，经常发脾气。丈夫带我去医院，中医、西医都看了，买了很多药，根本不管用，家人为我的身体担心、发愁。

那年秋天，姐姐来到我家，让我炼法轮功，说法轮功祛病健身有奇效，还教人做好人。我当时听了真高兴，就请了大法书。从那以后，我每天学法炼功。没过多久，不知不觉中，我一身的病全都好了，真是无病一身轻。

丈夫也从每天给我拿药、倒水、做饭和各种家务中解脱出来。我时刻按真、善、忍的标准要求自己，我每天笑呵呵的对待每一个人。丈夫和儿子都说我变了，不发火了，也会笑了。

#### 一、为大法说公道话得福报

二零一零年刚入冬，丈夫在学校烧锅炉。一天上班时，他在四楼看见烧开锅了，就赶紧跑到锅炉房去关阀门，结果阀门坏了。顷刻间，从四楼压下来的开水哗哗的往外喷，满屋子都是热气，什么也看不见了。外边的人吓的腿都打颤了，不知所措。

这时，只见丈夫从锅炉房里出来了，浑身上下全湿透了，可他却啥事没有，外边的人这才长出了一口气，惊叹：“太神奇了！不可思议！”原来是我丈夫为大法说公道话，得到了福报。

一九九九年七月二十日以后，江泽民集团对法轮大法和大法弟子疯狂打压的时候，局长找我谈话，丈夫也跟我去了。他们让我放弃修炼法轮功，没有达到目地，就对我丈夫说：“你劝劝她（劝我不要炼法轮功了）。”丈夫说：“我劝她干什么呀？她原来一身的病，哪儿也治不好。现在她一片药不吃了，还身体健康。”局长也无话可说了。真是善待大法一念，天赐幸福平安，因此我们才有幸福美满的家庭。

#### 二、婆婆相信大法身体康复

二零二一年的一个早晨，婆婆在炕上摔了一跤，当时婆婆已经八十六岁了。她觉的腿不能动了，我们就带她去医院拍个片子。片子上显示右腿股骨头摔坏了，大夫说得做手术，不然的话就永远不能下地走路了。于是我们就带着婆婆到医院做了手术，换了股骨头。

在做手术前，我告诉婆婆诚心敬念“法轮大法好，真善忍好”。手术非常顺利，不到一个小时，就做完了，而且术后没有一点疼痛感。在住院和回家的日子里，都是我和丈夫白天黑夜，不离左右的悉心照顾着，没有让弟弟、妹妹们照顾一天。婆婆非常受感动，逢人就说儿媳对她怎么怎么好，给她倒屎、倒尿不嫌脏，还给她洗头、洗脚、擦身子。

一天我出去买菜，碰到一位八十岁的老太太。她对我说：“我们那块的人都说你对婆婆特别好，照顾的很周到。现在的人要都象你这样就好了，也不用担心老了没人照顾了。”我说，其实我也没做啥，照顾老人是应该的，离大法师父讲的真、善、忍还差的远着呢。

我说，大法师父讲：“我们在常人社会中修炼，孝敬父母、管教孩子都是应该的，在各种环境中都得对别人好，与人为善，何况你的亲人。对谁也一样，对父母、对儿女都好，处处考虑别人，这个心就不是自私的了，都是慈善之心，是慈悲。”[1]

老太太说：“要都按照大法师父说的去做，那多好啊！你看我们这些老人们，子女回家哪是看望老人，是到老人这儿吃、拿来了。总嫌拿的少，没个够。有的老人病了，住院了，儿媳妇都不登门，更甭说照顾了。”

是啊，现在世风日下，道德越来越来败坏，我要是不修法轮大法，也会随波逐流，也得和弟弟妹妹们斤斤计较，哪有我们现在的和睦家庭啊！感恩师父慈悲救度！

注：
[1] 李洪志师父著作：《转法轮》
