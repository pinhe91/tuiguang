### 支持大法真的有福报

文: 大陆大法弟子 

【明慧网二零二二年十月十七日】我于二零零四年修炼大法，自己在大法中身心受益，知道了人来世的目地是返本归真，懂得了“真、善、忍是衡量好坏人的唯一标准”[1]，从此人生有了目标，生活有了希望。不修炼的家人也在大法的保护下遵循真、善、忍，并且得到福报。下面分享给大家他们的故事。

#### 1、我丈夫的故事

我的丈夫和我是大学时期的校友，现年五十多岁了，人很善良。大学毕业后，我们被分配到国营企业，但企业效益不好，没法生活，下岗（失业）后，一九九九年打工来到现在的城市定居。

二零零三年夏天的时候，有大法弟子给我一张真相光盘，我当时还受中共谎言的毒害，不了解真相，我们两个一起看了盘，丈夫说共产党什么坏事都干的出来，这些内容肯定是真的，还跟我讲了他见过原国企单位很多工程师们炼法轮功，他们人都很好。
丈夫在一家私企工作，工作累，工资低，孩子刚上小学，房子还是贷款，这一年多一直在想怎么能换个工资高的单位，但一直没机会。就在看过真相光盘后几个月，就有人跟他联系，去了外地一线城市的外资企业，工资和各种待遇提升了很多，我们的生活好了很多。

二零一七年，丈夫从外企单位离职回到居住地工作，在家里他经常看新唐人节目，经常夸赞新唐人节目办的好，说邪党的节目没什么看头，只有一个新唐人就足够了。他还经常看大法弟子办的自媒体，说非常感谢这些节目让他明白了很多。

二零二零年我因发资料被抄家和流离失所的那段时间里，当时儿子考公务员面临政审，丈夫虽然压力也很大，但是没指责过我修大法，还尽力帮助我躲避骚扰。美国大选的时候，他经常说我也要向川普总统那样做一个有正义感的好人。他的善念善行也得到了回报，现年五十多岁的他经常被人叫做小伙子，这么多年几乎没吃过药。在疫情泛滥封锁严重企业蒙受很大损失时，他同事的年终奖都折半的情况下，丈夫的年终奖却是以前的三倍。我有时候问他：你是不是得到福报，你感受到神对你的照应了吗？他说感受到了。有几次还特别认真的对我说：幸亏没有進到体制内工作，否则像他工作这么认真的人，在邪党的欺骗和胁迫下，不知道会做多少恶，造多少业呢！

丈夫虽然还没有走入大法修炼，但他做事情也用真、善、忍去衡量。今年过年的时候，厨房下水总管道堵了，楼上用水就漏到我家来了。谁也不愿去花钱找人疏通，因为住户有的不住在这里，没法摊钱。楼下的人自己将厨房改道了，我说要不咱家也改吧，要不都漏我们家了。他没这样做，过了年后自己花钱把下水道疏通好了。只收了楼上的那一家的钱，其他家找不到人，就我们自己拿了。还有一次坐公交车，有两个老人坐车，不会扫码付款，手里还没有零钱，车上没有人能帮他们换钱，丈夫就帮他们两人付了款。这样的事情还有不少。

上天从来不亏待善良人，儿子的公务员录取没有受影响，而且很快处了对象，女孩是他的同学，而且很懂事，已经领证了。我有时候问他：你没感到你的事情都很顺利吗？没有因为我修大法而受影响吧？他说没有。支持大法真的有福报。

#### 2、儿子的故事

儿子九岁那年我开始修炼大法，发现我看《转法轮》书，他有点害怕，就把他的课本《思想品德》拿给我看，里面是污蔑法轮功的自焚谎言。这么小的孩子就被谎言欺骗，中共真是太邪恶了！我把真相告诉了孩子，让他不要害怕，法轮功是教人做好人的。后来孩子也拿我的大法书看，也知道按照真、善、忍去做。

有一次，孩子姥爷送他去上学，走到校园里的时候，一个女同学在后面推了他一把，他差点摔倒，他没做任何反应，继续往前走；这个女孩在后面又跟着他，踢了他一脚，他还是没反应，继续背着书包往教学楼里走。这一切被姥爷看到了，姥爷都生气了，孩子没招惹她，她为什么欺负人，让我告诉孩子别太老实被人欺负。其实是孩子在遵照真、善、忍的原则处理问题呢。

学校里老师同学都喜欢我儿子，他是老师的好帮手，家里的电话成了他的专线，同学们有问题都打电话问他。儿子从小就身体不好，几乎是每个月都要上医院，每次开工资都犯愁，又得去医院了。上学后虽然不是每月都去医院，但也经常发烧，还有严重的鼻炎和扁桃体炎。渐渐的孩子身体越来越好，有几次高烧，没去医院，只是听大法师父的讲法录音就好了。后来又声明退了队，孩子的鼻炎和扁桃体炎都好了，这么多年也没去过医院。而且学习也好，最后被保送到排名前几位的大学。

现在许多孩子上了大学后，离开了父母的监护，在社会这个大染缸中很难把握自己，手机电脑成瘾，讲吃讲穿，男女之间也很不检点，不认真钻研学业，令父母担心。儿子从不乱花钱，也不沉迷于网络手机，衣服也不追求时髦，有的中学时穿的衣服现在还在穿呢。

儿子研究生毕业之后正赶上疫情，考取公务员。正在他面临政审的时候，我被抄家绑架，孩子没有指责我。在师父的保护下孩子顺利入职，连派出所所长都奇怪的说：政审是怎么过去的呢？他哪里知道，因为我们尊敬大法，支持大法，大法师父在保护我们家呢！

儿子刚刚上班几个月，派出所因为找不到我，政法委就去给儿子施压，儿子压力很大，但并没有埋怨我。我当时也有压力，担心影响儿子的工作和生活。然而孩子的善行得到了回报，很快处了对象，女孩是他的同学，贤惠懂事。虽然因为疫情还没有举行婚礼仪式，但两个人买了自己的房子，登记结婚，尽管疫情封锁，但我们家处处受到大法的保护，生活工作都很顺利。当然，因为这个社会被共产党破坏了传统文化，我们文革后出生的人都是在共产党的歪理邪说洗脑下长大的，不懂得传统文化中待人接物的道理和礼节，更不懂如何教育自己的孩子了。幸遇法轮大法，让我们明白和体验到了善恶有报的天理，把真、善、忍的种子播种在孩子心里，在这个物欲横流、道德急速下滑的世界里能坚守道德，从而得到大法的加持和保护。

这里只是讲了我家里的亲人明真相得福报的几个故事，是不是印证了善恶有报的天理呢？实际上还有很多，亲朋好友明白大法真相得福报的故事更多。诋毁大法得恶报的事情也不少，这里就不说它了。

朋友们，从这两年的瘟疫流行中，人类的各种措施都不能阻止病毒，您是否感受到上天的力量？现在虽然病毒全世界大流行，但没有造成巨大的死亡，是不是神在盼着人类的醒悟在给人机会？人类是不是已经到了危险的边缘？留给人的时间还有多少？法轮功修炼者二十三年顶着迫害的压力传递真相，告诉我们诚念“法轮大法好，真善忍好，灾难来时命能保”，心存善良的人你们可否听明白？

衷心希望朋友们能从我们家的小故事中体悟大法的美好和超常，了解真相，走出人类的大劫难，做一个配神救度的生命。
