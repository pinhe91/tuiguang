## 廖光慧在成都女子监狱遭迫害　昏迷至今已三个月

【明慧网二零二一年六月十一日】（明慧网通讯员四川报道）廖光慧的丈夫李双全和儿子三月十四日到华西医院，见到廖光慧的头全部被药棉纱包裹着，气管被切开，插着输氧管，身上还插着其它什么管子。李双全和儿子大声呼喊着廖光慧、“妈妈……”廖光慧却没有任何反应。
四川省绵阳市68岁的法轮功学员廖光慧，二零二一年一月二十日被劫持到成都龙泉驿女子监狱。三月十日，监狱打电话告知其家人，声称：“廖光慧上厕所时摔倒，头部伤得很严重，在华西医院动手术，需十几万元钱，赶快拿钱来。”任何反应，眼皮也未能动一下。廖光慧丈夫李双全拿出家中仅有的5850元钱给狱警交给华西医院。

廖光慧被辗转到成都华西医院开颅手术，在重症监护室一直处于昏迷状态。三月三十日，监狱警察无视生命，欺骗家属说：“将廖光慧转到监狱医院去，家人签字，以后家属去探望廖光慧更容易。”

廖光慧被欺骗转到监狱医院后，家属一次也没能亲眼见到她人。

五月七日，监狱姓吴的警察给廖光慧儿子打电话让他们去看望廖光慧。五月十日，廖光慧丈夫和儿子去监狱后，与监狱警察交涉了很久，监狱医院只是让他们在视频里看了看依旧躺在病床上的廖光慧的头，视频没有让家人看到整个身体。

监狱警察要廖光慧儿子交钱，廖光慧的儿子和儿媳靠打工抚养两个孩子。儿子除了返回家的路费，身上仅有的20元钱被监狱警察勒索去。

廖光慧，一九五三年二月二十六日出生，这次被绑架迫害前，身体很强壮，走路健步如飞，每天很忙，烧锅煮饭、买菜洗衣、打扫卫生、照顾孙儿孙女，全家六口人的生活一切包干。她丈夫李双全因不放弃修炼真、善、忍，二零一二年被中共非法判刑五年。

二零一九年七月二十日下午一点，十几个身穿便衣的彪形大汉（后了解是涪城区公安局国保大队、花园派出所警察），突然闯入绵阳市涪城区新华厂家属区廖光慧的家中，未出示任何搜捕证，和身份证明，警号等，就强行控制家人自由行动，像土匪一样抄家抢劫近三个小时之久，强行抢走私人物品：笔记本电脑两台，手机、打印机、刻录机、现金几百元，大法书籍等，并绑架了廖光慧。

廖光慧被非法关押到绵阳市看守所一年多。二零二零年九月一日遭绵阳市涪城区法院非法庭审，维权律师为廖光慧作了有理有据的无罪辩护。九月二十五日，廖光慧被绵阳市涪城区法院非法判刑三年、勒索罚金三千元；枉判的借口是用刻录机刻录让人了解真相的光盘来赠发给人。刻录的内容都是事实，如一九九八年七月十日的《中国经济时报》以题为《我站起来了》的报导：河北邯郸的谢秀芬，丈夫在北京部队。她一九七九年做结扎手术致瘫痪，经301医院查鉴后，残联给发了“下肢瘫痪残疾证”。一躺就是16年，一九九六年七月一日，丈夫用轮椅把她推到法轮功炼功点，她终于站起来了！这些事实让人明白真相、身心受益，传播它何罪之有？

二零二一年一月二十日，廖光慧被劫持到成都龙泉驿女子监狱继续关押迫害。收监时全面体检，身体无病。然而，目前却被迫害成这样！

截至六月十日，廖光慧已经昏迷整整三个月。请国内外正义、善良人士，关注廖光慧目前的处境。

《刑法》第二百四十八条的规定，虐待被监管人罪是指监狱、拘留所、看守所等监管机构的监管人员，对被监管人进行殴打或者体罚虐待，情节严重的行为。犯本罪的，处三年以下有期徒刑或者拘役；情节特别严重的，处三年以上十年以下有期徒刑。致人伤残、死亡的，依照本法第二百三十四条、第二百三十二条的规定定罪从重处罚。

《刑法》第三百九十七条的规定，滥用职权罪是指国家机关工作人员故意逾越职权，不按或违反法律决定、处理其无权决定、处理的事项，或者违反规定处理公务，致使侵吞公共财产、国家和人民遭受重大财产损失等行为。

《监狱法》第十四条规定：监狱警察不得有下列行为：（一）索要、收受、侵占罪犯及其亲属的财物；（二）私放罪犯或者玩忽职守造成罪犯脱逃；（三）刑讯逼供或者体罚、虐待罪犯；（四）侮辱罪犯的人格；（五）殴打或者纵容他人殴打罪犯；（六）为谋取私利，利用罪犯提供劳务；（七）违反规定，私自为罪犯传递信件或者物品；（八）非法将监管罪犯的职权交予他人行使；（九）其他违法行为。监狱的人民警察有前款所列行为，构成犯罪的，依法追究刑事责任；尚未构成犯罪的，应当予以行政处分。

## 法轮大法救了我的多位武汉家人

文: 中国大陆大法弟子 

【明慧网二零二一年六月十日】我把中共病毒泛滥武汉期间，我的武汉家人的亲身经历写出来，感谢师父的救命之恩！当前疫情仍然在中国传播，威胁着不明真相的中国民众的性命。望我们都能按照师父的要求，做好三件事，救度更多的众生。

### 老伴的弟弟一家感谢大法师父的救命之恩

二零二零年大年初二早上八点左右，我正在上班，突然接到侄儿（老伴弟弟的孩子）的电话，问我们武汉家的钥匙放哪儿了？我说：“我们家的钥匙你大伯都带回来了，你要我家的钥匙干什么呀？你们不是都在武汉吗？”侄儿说他们腊月二十九就回武汉了。我说，你们都没事吧？侄儿说：“我……我……我们都没事。大妈大伯你们都好吧？”我说：“我们都很好，就是很担心你们。你们都没事，我们就放心了。”侄儿说：“大妈，祝你们新年好！”就没再说什么了。

中午回到家，老伴就急着跟我说他的弟弟给他打电话的事。我问，是不是早上八点左右？老伴说应该是这个时间。接着老伴就把他和他弟弟的对话给我重复了一遍：

弟弟说：“哥哥！你快点把你们回来时说的那九个东西拿给我们！”老伴说：“你说的那九个东西我不知道是什么，也不知在哪儿呀，等你嫂子回来我问问她吧。”弟弟说：“就……就……就那九个能救命的东西呀！”老伴说：“哦，你说的是‘九字真言’？”弟弟说：“对！对！快点拿过来，我们等着。”老伴说：“怎么拿过去，这么远的路，怎么拿？能拿也拿不去了，我们这边腊月二十九就封城了。”弟弟一听，说：“哎呀，那怎么办啊？！”老伴问：“你们怎么了？是不是有事了？得（肺炎）了？”弟弟说：“我们没事，我们都很好，你放心，都很好……”老伴想了一下，说：“可以用微信发过去，救人要紧。”弟弟说：“你还会用微信？”老伴说：“会，马上发给你！”

老伴给我重复完他和弟弟的对话，我也把早上侄儿打电话的事说了。老伴说：“你觉的他们那边是不是有问题（感染肺炎）了？”我说：“我觉的可能是他们当中哪一个有问题了，要不怎么那么着急？”老伴说：“不怕，我去年年初就梦见师父了，师父的身体大的看不到师父的整个头。当时我看到师父是朝着武汉方向望着，我也就朝着师父看的方向看去，看见武汉天空中黑浪滚滚，好像要把整个武汉吞没了似的，很吓人！我心里想武汉要出大劫难了，师父知道。醒来后我不是跟你说过这个梦吗？”

过了十天左右，弟弟又打电话给老伴，说：“哥哥，他们（上面领导）明天要强行把我们全家都弄去做检查，我们都好好的，我们不想去。”老伴说：“他们为什么要弄你们去呀？”弟弟说：“本来不想告诉你，怕你担心。某某（弟媳）初二早上病得快不行了，状况跟肺炎的状况是一样的。所以儿子就急着打电话给你们，要那个‘九字真言’，同时想用车把他妈送到你们家去隔离。她在家隔离了几天后被人举报，弄到医院检查。检查确诊为阳性。”老伴说：“怕什么，要把你们都弄医院去检查，去就去吧，听我的。”弟弟说：“其实我们都不怕了。我们全家日夜都念着哪！请求你们的师父保护，我们向你们的师父忏悔：我们全家以前不知道真相，对你们的师父、对你们说了一些不好的话。现在我们都相信你们的师父，我们会没事的！”

又过了二十多天，弟弟和侄儿很高兴的给我们打电话说：“我们全家六口人都回家了！感谢你们的师父！感谢你们去年回老家来告诉我们真相，不然，我们全家就完了！”

### 只信“科学”的小妹体验到法轮大法的威力

二零二零年的大年初一，早上我给我的小妹和两个外甥女打电话，祝福她们全家新年好！我接着说：“你们那里（武汉）现在疫情最严重，全家要诚念‘法轮大法好，真善忍好’就能保平安，不然的话后果不可想象。”小妹一听，哈哈一笑，说：“都这个时候了，你们还相信那个？我可不相信，现在医学这么发达，你们那个东西管用吗？请你不要再给我讲这些，讲点别的吧。”我说：“那，就再见了！”我就把电话关了。

过了几天，一位同修来我家，自然谈起疫情，我顺便讲了我小妹的事。同修劝我再给小妹打个电话。我不高兴了，说：“我根本不想再给她打电话，她认为她自己是教师，觉的自己了不起，讥笑我们搞迷信，瞧不起我们！”虽然嘴上这么说着，手上的电话已经拨通小妹的电话。电话响一会儿才听到小妹说：“谢……谢谢！”电话就挂了。这时，我有点生气，就对同修说：“我说不打，你要我打，真是的！”

不记得过了多少天后，我梦见了小妹。醒来就想给她打个电话，于是就打了过去。小妹接到电话马上就说：“感谢你们的法轮大法师父！”我说：“小妹，你怎么啦？”小妹说：“你不知道呀，我以为你知道我今天全好了才给我打电话的呢。”我说：“我真的不知道你怎么了？”她说：“其实你上次打电话给我时，我正病的快要死了，那状况就和肺炎的症状一模一样，真的快要死了，连拿手机的力气都没有了，想再多说几句都没力气了。所以你那天给我打电话，我只能说‘谢谢’，根本没力气多说什么。”我问：“你是怎么好的呢？”她说：“哪还有什么办法？我就和我的两个女儿按你说的日夜念‘法轮大法好，真善忍好’，就这样，我一天比一天好，这不我全好了！真得谢谢你们的法轮大法师父！谢谢，谢谢！也谢谢你的关心……二姐，你知不知道哥哥也跟我出现一样的状况，都快要死了！他们也是全家就都念‘法轮大法好！真善忍好！’不停的念。现在他也全好了！”我问：“你是怎么知道哥哥的情况的？”她说：“昨天哥哥打电话跟我说的。我们谈了很长时间。”我说：“小妹，哥哥的情况我很清楚。他们全家念的是‘法轮大法好，真善忍好’。”小妹说：“哦，‘法轮大法好，真善忍好’，记住了！谢谢！谢谢！谢谢！”

写到这儿，泪水已湿了眼眶……无法表达对师父的感恩！师父无时无刻不在看护着我和我的家人，也救了丈夫的弟弟一家和我的小妹和哥哥。

写出我的家人的亲身经历，替家人感谢师父的救命大恩！也愿看到我的家人的真实故事的您，能早日识正邪，了解更多大法真相，沐浴佛恩，得到佛法救度！在这多灾多难的时代，愿您平安！

谢师恩！
