### 老太太从小落下的头痛病一下子全好了

【明慧网二零二二年十二月十八日】

#### 老太太从小落下的头痛病一下子全好了

〖中国大陆来稿〗有位老太太看到有大法弟子在附近讲真相，就冲着大法弟子喊：“法轮大法太神奇了，太灵了！”老太太就跟大法弟子讲了她的亲身经历。

老太太说，她从小家里就穷，四岁就送给人家当童养媳，吃了不少的苦。有一天老太太遇到了一个大法弟子，大法弟子送给老太太一个护身符，告诉她诚念：“法轮大法好，真善忍好”九字真言，遇到危险能化险为夷，还能祛病，身体好。老太太记下了。

有一次老太太在自家院子里做事，不知从什么地方飞来了一群蜜蜂，急速的朝老太太飞去，老太太来不及躲，蜜蜂就朝老太太头上蜇，情急之下，老太太就大喊：“法轮大法好，真善忍好！”也不知怎的，蜜蜂又全飞走了。随后，老太太感觉从小落下的头痛病一下全好了。

老太太逢人就讲自己的亲身经历，那些人听了都说法轮功太神奇了。老太太对大法弟子讲，她要让更多的人知道：诚念法轮大法好，真善忍好，就能得福报，保平安。

#### “真是奇迹！”

〖中国黑龙江来稿〗这是发生在我家的事。二零一八年，二十五岁的外孙因工作单位离家远，就买一辆汽车自己开车上下班。二零一九年的一天，他下班很晚。开车回家的路上，车的方向盘突然向一侧偏离，力量还很大，车就直接冲着八米多深的沟里冲过去，打着滚翻到沟底，他就什么都不知道了。

不知过了多久，他猛然清醒过来，一看时间已是夜间十二点左右。他闻到有汽油味，担心车会爆炸，就用力把车门踹开爬了出去。他又突然想起手机还在车里，就转到车的另一侧，伸手从破碎的玻璃车窗把副驾驶座位上的手机拿出来。车没有爆炸，整辆车只有他坐的驾驶位没有塌，其它部位全变形了，全报废了。车里的东西都没了，钱包也没了，就连我给他的保平安的护身符也不见了。

外孙明白是师父保护了他，他才没受到一点伤害，还留下手机让他与家人联系，如果手机也没了那可就真麻烦了。深更半夜，前不着村后不着店，想想都后怕。

我女儿和外孙来我家，一進门，女儿就对外孙说：“快给师父磕头，感谢师父的救命之恩！是大法师父救了你的命！”外孙马上给师父法像磕头，感谢师父救命之恩！我悟到外孙是用这种方式在还他生生世世的业债，师父帮他化解了，保住了他的命。

我女儿的同学、同事、朋友知道此事后，都很惊讶，有的说：“发生这么大的事，太可怕了，也真是奇迹！你家一定是信什么，才受到神佛的保护。”女儿说：“我妈是修法轮大法的，我们都相信法轮大法好。是法轮大法师父保护了我儿子。”

我们全家在这里再次感谢师父，感谢师父的救命之恩！弟子让师父操心了，弟子给师父叩头，感谢师父的救度！弟子一定精進再精進，修好自己，做好三件事，不辜负师尊的慈悲苦度。
