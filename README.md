### 越来越多的世人知道“法轮大法好”

文: 大陆大法弟子 

【明慧网二零二一年十月二十二日】我是一九九六年开始修炼法轮大法的。在我的家人、族人、朋友、同学、同事、邻居以及学生中，越来越多的人相信“法轮大法好”并因此出现许许多多神奇事。

#### 一、九十六岁远房哥哥的故事

我的一个远房哥哥今年九十六岁。他因为了抗日而参加过国民党军队，一九五七年被打成了“历史反革命”，在当地被监管劳动。他非常听话，从不“乱说乱动”，还坚决不信神佛。他母亲去世时，他弟弟请了锣鼓队，他硬是没让吹打，认为那是迷信，更不符合政府的政策。

二零零六年我回乡后得知，就是这个远房哥哥的老伴因与人打架，被打成脑震荡，久治不愈。我告诉嫂子：“你只要相信大法师父，悄悄的天天念‘法轮大法好，真善忍好’，你一定会好的。”我给了她一个大法真相护身符让她戴着，告诉她：“千万不能让哥知道啊！”

没想到两个月后，远房哥哥给我打电话说：“你给你嫂子那个东西比吃药还好用，是怎么回事啊？你也给我一个揣着嘛，我身上经常痒得不行……”我以包裹的方式寄给了他一个“法轮大法好”护身符和一些真相资料。从那时，护身符他成天戴着，经常跟嫂子一起念“法轮大法好，真善忍好”。

二零零九年，我回去看哥哥，他告诉我说：“我本来是不敢信鬼神的。你嫂子的脑震荡吃药两年也没好，可是戴上护身符之后，不吃药脑袋不疼了，我戴上后身上也不发痒了。大法的师父是什么人？人间真的有神吗？国家为什么要镇压？妹夫是个官，他怎么敢让你炼法轮功呢？”问了很多很多他不解的问题。

我给他讲了江泽民其人、到底有没有神、什么是神、大法师父是为什么来世的，等等，并让他思索：“共产党对你、对老百姓怎么样？人是从哪儿来的？九大行星为什么能按照规律围着太阳转？”他开始怀疑“无神论”了，还敢给他的妻弟讲了他们夫妻俩受益大法的事，把我给他的《转法轮》也让他的妻弟看。

二零一五年九月，我回去给哥哥祝九十大寿。正好他因痛风疼的日夜不能眠，腿、脚肿的没鞋可穿，在家都坐着轮椅活动。当晚，我给他听大法师父的讲法录音。没想到他听到凌晨三点腿脚就恢复正常！

第二天，他自己走到酒店，参加晚辈们为他办的九十大寿寿宴。他高兴的给亲人、族人说：“没办法不信啦！我这是活生生的事实啊！”

年底，他在自己家门前摔了一跤，把腿摔肿了，不能上街散步了。这时，他怀疑的问我修炼的妹妹：“不是说保护我吗？怎么还让我摔跟头呢？我不信了！”

我打电话问他：“哥，你骨头摔坏没有？你的邻居才七十三岁，是不是跟你在同一个地方摔的？他还没有你摔的重，可是已经住院半年了，是不是还在由护工喂饭？”他说：“哦，我错了！是神保护了我！”

我告诉哥哥：“延寿，不能无条件、无限期的延哎！你摔一跤，就是消大灾，让你吃点苦，遭点罪。但不让你摔坏，是在继续延你的寿。你说，大法师父保护你没有？”他说：“哦，我明白了，以后我坚信大法师父，不再怀疑了。”

二零一九年暑期，我家乡高温四十一度，哥哥坚持去社区内散步，回家后，就高烧、失去了知觉。家人、亲人都觉的这次他必死无疑了。嫂子跟我妹妹说：“你让姐姐（指我）求求大法师父，让他再多活几年吧。”我没照做。

嫂子自己守在哥哥耳边，整天不断的念“法轮大法好，真善忍好”。一周后，我打电话问哥哥的情况，说他能吃流食了；两周后，他能自己下地去厕所了。

二零二一年正月初一，我们和哥嫂视频，哥哥正在吃午饭。他女婿问他：“他是谁（指我老伴儿）？”哥哥答：“妹夫。”我问他：“哥，你今年高寿多少？”他答：“九十六。”

#### 二、患直肠、肝脏、肺癌的同窗延寿了

我的一个初中同窗，是位英语教师，在当地小有名气。二零零九年春天，我们同学聚会时没见到他，我问其他同学他为什么没来？同学说：“他患直肠癌做了第二次手术，现在已经转移到肝脏和肺部了。医生告诉家人，最多只能活三个月了。”

我去看望这位同学。我告诉他：“我不到五十五岁就病退了。我原来有心脏病、慢性胃炎、低血压、经常头晕、长期失眠、关节炎，这些病跟了我几十年，五十出头又查出血液粘稠，做脑血流图、脑电图等等，没有一项指标是正常的。我在家都不能独立行走，身边总得有人，就提前退休了。一九九六年，经学生家长介绍，我炼了法轮功。只学炼了一个多月，百病全无。”

他当即就让我给他买MP3，让我给他录制大法师父的讲法录音。从此，他身上不痛了，睡眠好、吃饭香。他让他妻子给我绣花袜底表示感谢。

可惜的是他悟性差，身体好了就天天下午出去打麻将，而忘记了自己的病是怎么好的，没做他该做的。三年后的二零一二年十一月，他再次住院了。二十多天后走了。

走之前，他告诉他妻子：“我那同学（指我）再来时，你要感谢一下！我每月五千多元工资，我多活了三年多，给我们家添了多少钱啊！”还嘱咐他妻子说：“你也跟她炼功吧。”现在他妻子也天天念“法轮大法好，真善忍好”。

#### 三、“我的家人、亲人，这回都心服口服了。”

二零一二年夏，我们附近发生了七级地震。我的新邻居正走在楼梯上，一下从二楼滚到一楼，摔成腰椎压缩性骨折。住院一个月了还不能翻身，靠老伴喂饭。

我去看望她，告诉她：“你若相信大法师父，你就回家吧，我给你读《转法轮》，保证你好的快。”她说：“我信！一九九九年七二零前我炼过半个月，镇压后因家人反对，我放弃了。这事我谁也没告诉过。”

她出院了。我就开始去给她读《转法轮》。因怕她老伴儿反对，都是趁她老伴儿出去打麻将走后，才能去给她读。刚刚读了两个下午，读到一百五十页的时候，她就可以自己扶着墙上洗手间了；用了五个半天读完一遍《转法轮》后，她告诉我：“我在家能自由行走，还能做饭了。我老伴可高兴了。明天娘家人要来看我，你不用来给我读了。”

从此，她堂堂正正的走回修炼了，老伴和其他家人再也不反对了。

二零一六年秋，邻居的亲家公被确诊为肺癌晚期，邻居要我帮他。我给邻居的亲家公讲了真相，做了三退（退出中共的党、团、队组织），给了他《转法轮》和MP3，他就出院了。整天听大法师父的讲法录音或看《转法轮》。

直到二零一九年秋，邻居的这位亲家公住院二十多天后就走了。邻居告诉我：“我亲家公的家人都知道是大法师父让他多活了三年，还给他们家省了钱。”

今年年前，邻居的大哥出车祸，被汽车把脑浆都撞出来了，家人已经准备了后事。邻居守在她的大哥耳边坚持念“法轮大法好，真善忍好”。一星期后，她大哥醒来了；邻居继续给他哥念，半个月时就出院了，现在一切恢复正常。

邻居说：“万分感谢师父！我的家人、亲人，这回都心服口服了！”

#### 四、快递员的母亲恢复了

一位快递员因经常帮我们邮寄真相资料，所以彼此很熟悉了。连他的母亲也成了熟人。二零二零年八月，他的母亲在家门口把右大腿的骨头摔断了，断成三截。手术前，他母亲用电话发短信对我说：“某某，请你给我祷告一下。我后天做手术，我害怕……”第二天，我让老伴给她送去大法师父的讲法录音，告诉她和她的家人都诚心诚意的念“法轮大法好，真善忍好”会有意想不到的效果。

手术后五天，她居然自己就能下床了。她儿女见状，就架着她在屋里走了一段路。过了几天医生给她拆线后，她立即就出院了。第二天，她到我家门市来走给我看。

我让她继续听大法师父的讲法录音。一个月后，她就恢复到和摔断腿之前一样了。

#### 五、患癌症的同学说：“我现在挺好。”

我的一个师范同窗，去年八月电话给我说：“老同学，我一年前得了喉癌，现已转移到肝脏和肺脏了，时日不多了……”我以前给他讲过真相，他也做了三退，他很相信大法师父，想炼功，但因家人反对，未炼成。

我给他寄去资料，告诉他：诚心念“法轮大法好，真善忍好”吧，会好的。

一个多月后，他在电话中告诉我：“我到颐养院来了。我老伴脑梗，我俩一起住在这里。我好多了，感谢了！”之后，他两次打电话告诉我：“我现在挺好，感谢……”我让他把自己好转的情况告诉亲人，让亲人也受益。

我修炼法轮大法二十五年来，家人、亲人全都受益匪浅：老伴因全力支持我修炼，帮我做讲真相的事，他已二十年没看过医生了；孙子及孙媳从美国一所名校毕业，又在美国最好公司就业四年了；小外孙今年九月也要去美国一所名校读研，还与家孙在同一个州，相距很近。

叩谢慈悲伟大的师父！
