### 武汉市法轮功学员陈俊、田化珍等五人被非法批捕

【明慧网二零二二年七月二十八日】（明慧网通讯员湖北报道）据悉，湖北省武汉市江夏区法轮功学员陈俊、田化珍、张娇娥、李春连、邱咏枝已于七月二十二日被江夏区检察院非法批捕。田化珍，张娇娥，邱咏枝现被非法关押在武汉市第一看守所；陈俊、李春连被非法关押在武汉市汉阳区安吉精神病院，情况堪忧。据说李春连正在被迫绝食。
陈俊和李春连的家属于七月二十二日接江夏区纸坊派出所黎姓警察通知为他们二人办了“取保候审”手续，但派出所负责人只办手续、却不放人。家属要求放人，但警察说上面不让放。

二零二二年六月十六日中午十一时至十二时左右，武汉市江夏区纸坊派出所一伙警察开了一辆依维柯白色警车，来到法轮功学员陈俊家，绑架了陈俊、田化珍、张娇娥、李春连、邱咏枝五位法轮功学员，并非法抄了陈俊的家。

家属收到江夏区纸坊派出所寄来的刑事拘留通知书，通知书上说陈俊被非法关押在武汉市江夏区看守所（地址：武汉市江夏区纸坊街东林村）；田化珍、张娇娥、李春连、邱婆婆被非法关押在武汉市第一看守所（地址：武汉市东西湖区三秀路）。

田化珍，女，七十岁，一九五二年十一月十六日出生，修炼前她有严重的类风湿，两腿常常不能走路，心里烦躁，脾气也不好；一九九七年十月炼功后，类风湿再也没犯过，她按真、善、忍标准要求自己，脾气也变好了，每天都生活得很充实。

邱咏枝，女，六十九岁，患有好多病走进法轮功修炼的，例如：胃病、风湿、尿道炎等。一九九七年元月一日开始修炼后，她什么病都好了，人活得轻松愉快，性格开朗，善良，宽容，真诚。

一九九九年八月中旬的一天，夜里十点钟江夏区五里界派出所一帮警察连喊带叫的敲田化珍的家门。田化珍和老伴开了门，他们一下就闯了进来，说你儿子炼法轮功，我们要搜查。家中被翻得乱七八糟，儿子被绑架迫害。二零零三年儿子因挂大法条幅，被非法判劳教十五个月，全家陷入痛苦的煎熬之中，老伴因思念儿子，不久就痛苦离世了。二零一三年八月十五号，江夏纸坊的一个干警闯到田化珍家中，问她炼不炼法轮功。迫害发生后，每年逢年过节，江夏区居委会的人都要去田化珍家里骚扰。
