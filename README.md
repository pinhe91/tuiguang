### 广东省所谓的“反邪教协会”的罪恶簿

【明慧网二零二二年二月二日】二十多年来，广东省政法委操控公检法司对法轮功学员进行残酷迫害的同时，也对广东民众进行持续的精神洗脑和迫害，并把迫害延伸到社会的各个阶层和角落。广东省所谓的“反邪教协会”（以下称为广东省中共邪恶协会，中共是真正的邪教，历次运动害死八千多万中华儿女）直接或间接操控了所有对法轮功的诽谤和对民众的洗脑迫害。
广东省中共邪恶协会设立于二零零一年三月十四日，其上级单位是广东省社会科学界联合会（简称广东省社科联），邪恶协会和社科联的办公地址都在广州市天河区天河北路618号，邪恶协会盘踞在709、711室。邪恶协会的历任会长和秘书长均为社科联的主要领导人员。邪恶协会从广东省政法委反×教工作处领取迫害指令后，通过邪恶的枢纽将指令传递至各个基层领域，由基层人员具体实施迫害。省政法委反×教工作处的上司为广东省历届政法委书记。

邪恶协会第一届会长为时任广东省社科联主席张磊，时任广东省社科联党组委员张硕城为秘书长。法轮功教人向善，中共是真正的邪教。中共对法轮功的迫害，是对公民信仰权利的践踏。中共邪教却贼喊捉贼，邪恶协会一设立便喊出“崇尚科学，反对邪教”的口令，并在广州市农讲所进行了十五天的诬陷图片展。这个口令一直沿用到今天，成为中共邪教欺骗民众、挑起民众仇视法轮佛法,和邪教一起走向共毁的招魂幡。二零一一年十二月，邪恶协会换届产生第二届所谓理事会，张硕城成为会长，社科联秘书长黄庆明为专职副会长，社科联专职副主席周华为法人代表，兼副会长和秘书长，省政法委反×教工作处巡视员颜建国为副会长。邪恶协会的第三届所谓理事会产生于二零一八年十二月十九日，所谓大会由广东省社科联巡视员李夏铭主持，广东省政法委专职委员黄文平参与并发表煽动演说。省社科联专职副主席李旭明成为第三届理事会会长，省社科联的陈文汉为法人代表兼秘书长，省社科院骆梅芬为监事长，省直有关单位、高校和省五大宗教团体负责人等八人为副会长，其中包括广东省监狱管理局教育改造处副处长韩保锋、南方医科大学通识部主任严金海等不法人员。

二十多年来，对法轮功的诬陷、诽谤，对广东民众的洗脑、毒害，全部由广东省中共邪恶协会直接或间接操控，它的罪恶罄竹难书，具体反映在以下几个方面。

#### 策划设立基层邪恶协会　将迫害延伸到各个基层

在广东省政法委邪教工作处的部署下，该邪恶协会象中共病毒一样迅速复制、扩张，目前在全省30多个市、县建立了反×教协会，市、县再在各区、街道、镇、村设立分会，将这个散播谎言、迫害信仰的犯罪组织延伸到广东省的各个基层，通过基层实施对民众的洗脑迫害。目前广东省各个市、县、区的公园、广场基本都被改造成了洗脑班，设有固定的诬陷造型、标记，常年挂有诬陷展板，不时开展各种活动，这些横幅、展板等的内容由邪恶协会统一编制。所有街道、居委组织的活动，几乎都掺进了洗脑诬陷元素。邪恶协会的秘书长陈文汉还经常窜到各个村镇，担任驻村第一书记，以“脱贫功坚”的名义，组织基层单位对当地居民进行大规模毒害。

#### 实施诬陷诽谤常态化　毒害青少年学生

自二零零六年开始，邪恶协会在广东省政法委、社科联的鼓动下，与各市的教育局相勾结，在学校实行反×教诽谤洗脑常态化机制，在学校开设《中小学生反×教警示教育课程》，安排专门教师、采用专门教案，并规定课时，该害人课程首先在广州市越秀区中小学生综合实践教育基地进行试点，然后推广到全省。二零零七年，邪恶协会又编写了中小学法制副校长反×教教学大纲，由广东省“610”办下发至全省，使校园洗脑迫害更常态化。邪恶协会还每年挑选一至两个地级市的大专院校进行墙报、演讲比赛等诬陷害人活动。青少年学生涉世未深、天真烂漫，对老师、学校高度信任、毫无防备，加上这些诽谤宣传是在正常上课时间进行，大部份学生都难逃魔掌。

#### 组织编写出版洗脑毒本《36名×教亲历者实录》　毒害世人

该毒本经时任副省长林少春同意并作序，由省政法委牵头，邪恶协会协调省监狱管理局等不法人员杜撰而成，所谓主编为时任广东省政法委专职副书记陈少波，执行主编为时任广东省政法委专职副书记、“610”办主任叶敏辉，执行副主编为邪恶协会秘书长陈文汉。该毒本于二零一七年三月由南方日报社出版，被列为“广东省‘七五’普法读物”，发行量超过12万册，邪恶协会副会长严金海积极到各个学校推广、贱送。

#### 策划“广东省反×教宣传周”诬陷诽谤活动

自二零一六年开始，广东省中共邪恶协会每年在全省范围内策划反×教诽谤宣传周活动。活动中，省、市、区、县、镇、村各级反×教协会和有关单位联动，线下线上同时进行，多个媒体大规模跟踪报道，是一种集中式高密度全省范围内的诬陷洗脑运动。这个时候，学校摆展板、发单张、看视频、搞签名墙，组织反×教宣誓、书画比赛、主题班会、开学第一课等等洗脑活动。大部份学校会用整个下午集中进行，在课室采用PPT宣讲，强制全校学生同步观看，观看后再看展板、游戏、签名等活动；社区广场也是各种各样的活动，邪恶协会打着“平安创建”、“崇尚科学”等骗人的幌子，各街道、镇、村、居委，利用一切方式，对法轮功进行污蔑性宣传，对民众进行洗脑欺骗，诽谤毒害中要求“镇不漏村、村不漏户”。

#### 纠集设立“广东省×教现象研究基地”

二零一六年十二月，广东省中共邪恶协会纠集南方医科大学，在南方医科大学设立了“广东省×教现象研究基地”，豢养一批学术痞子为中共对法轮功的迫害、对民众的洗脑诬陷提供所谓理论依据。基地主任为严金海，核心人员包括：南方医科大学卫生管理学院党支部书记常路，通识教育部高一飞，马克思主义学院罗海滢、谢传仓、宋剑，中山大学教育学院社科系德育研究室主任古南永，广州中医药大学经济与管理学院院长邱鸿钟，广东省民族宗教研究院副所长陈晓毅，中山大学叶春生，广东工业大学邵俊武等。二零一九年十一月二十一日，邪恶协会联合×教现象研究基地，举办了所谓“广东省X邪教理论研讨会”。严金海、古南永、邱鸿钟目前为广东省反×教讲坛所谓专家。二零二一年十二月，广州市番禺区政法委不法人员以“营造美好社会”为幌子，对法轮功进行诬陷诽谤，严金海进行了直播，据说线上参与人员近20万。
