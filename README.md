### 表妹“复阳”　念九字真言四天痊愈

文: 辽宁大法弟子扬帆 

【明慧网二零二三年一月十五日】二零二二年年底，大陆疫情爆发，相继传来朋友同事的亲人去世的消息。姐夫是某大学教授，单位同事四天走了八人；丈夫同修的单位有三个同事的亲戚去世；我们小区每天早上都有放炮的（按当地风俗，家人去世一早出殡要放炮）。我们感到时间的紧迫，作为法轮大法弟子要和邪恶抢人，抓紧叫世人了解真相、知道避难的良方——念“法轮大法好，真善忍好”和“三退保平安”。
利用元旦放假，我和丈夫到他的亲戚家拜访，再次把大法真相告诉他们。听表嫂说表妹复阳，症状很严重。从表嫂家出来，我们带着礼物直奔表妹家。敲开门，表妹戴着口罩，不敢让我们進屋，怕传染我们。在我们的坚持下，她才让我们進屋，让我们坐在房间一边，她坐另一边。得知她和女儿、丈夫都阳了，她没好利索又出去，结果复阳，现在很难受。

我们告诉她念“法轮大法好，真善忍好”九字真言。解释为什么念“九字真言”可以好：中医认为人得病是“邪气入侵”，念“法轮大法好，真善忍好”增加正能量，人就会好。而且“真善忍”这三个字没有任何问题，真诚、善良、忍耐没有错，法轮功修炼人按照真善忍的要求做，法轮功没有错。

我们还讲了为什么中共迫害法轮功：法轮功一九九二年传出，到一九九九年有一亿人在炼，人数远远超过党员的人数，上至高官下至平民百姓，当时七个常委的家属都有人在炼。看到千百万人都发自内心地尊敬法轮功师父，按照法轮功师父的教导做人、修炼，江泽民出于妒嫉发动对法轮功的迫害。但是到目前为止，中国没有一条法律说法轮功是邪教，公安部和国务院认定的十四种邪教里面没有法轮功，法轮功在中国是合法的。那么“邪教”之说怎么来的呢？是江泽民接受法国记者采访说的，之后《人民日报》跟风报道，个人和媒体说的不能成为法律依据。为了给迫害法轮功找理由，中共自编自导了天安门自焚事件。如果自焚能圆满，找个山沟自焚得了，为什么跑到天安门去“自焚”呢？电视里自焚的人坐的姿势和手势完全不对，两腿之间还有个完好的雪碧塑料瓶……

问他们知道为什么突然“解封”吗？是乌鲁木齐的火灾，因为封控将消防通道堵死，发生火灾消防车進不去，烧死了好多人。这把老百姓的积怨点燃，北京、上海等各地学生拿白纸上街抗议，有的喊出：“共产党，下台！”当局再看疫情也按不住了（重庆、北京、广州等地都非常严重），就突然放开，让老百姓毫无准备，疯狂抢药囤药，哪个政府会这样不顾老百姓死活？中共从篡政以来，发动各种政治运动，迫害死很多无辜的百姓。中共干尽坏事，上天要惩罚它，会有天灾、人祸、瘟疫发生。疫情面前人人平等，有钱有权你不照样死吗？现在挣多少钱不重要，全家人平平安安渡过疫情最重要。中共宣传美国怎么不好，可高官家属都往美国跑，每人手里攥着好几本护照。

共产党是不会为老百姓考虑的。咱老百姓得想办法自保，念“九字真言”是最好的办法，只要诚心的念，没人的时候出声念，有人的时候心里念，不用花一分钱，而且安全。我们还举了念“九字真言”恢复健康的例子；并讲了藏字石上的“中国共产党亡”是天意的显现，是上天要灭共产党。解释为什么要三退保平安，他们说以前的熟人给他们退了。临走时我们给表妹留下了真相U盘和护身符，教会他们用手机看里面的小视频。

我们是周一晚上去表妹家的，周五表妹发来信息说：“我好了。祝你们全家真善忍好，法轮大法好。”丈夫叮嘱她：别忘告诉亲朋好友和你妈妈这个办法有效。表妹答应了。当时看到信息的时候，我被感动的眼泪流下来了，感恩师父的救度。

回来这么多天了，我俩至今啥事也没有。法轮大法就是这么神奇！
