### 世人见证神迹后传播大法福音

文: 大陆大法弟子慕奇恩 

【明慧网二零二二年十一月七日】一九九九年“七·二零”邪党江氏集团捏造谎言诋毁法轮大法创始人及迫害大法弟子以后，大法弟子无畏强暴，冒着生命危险，经历各种魔难、艰辛万苦，不断的向世人讲清大法真相，告诉世人：法轮大法好，真善忍好。现在世人明白真相神迹显的实例比比皆是，因篇幅有限，现仅举几例，证实大法师父伟大！法轮大法超常！

#### 一、世人见证神迹后传播大法福音

小静是一个温柔贤惠的年轻妈妈。她看到父母修炼法轮大法以后，身体健康，相互关心，家庭气氛温馨。虽然她没有修炼，但她深知法轮大法的美好！即使邪恶疯狂迫害大法弟子，她也从来不去阻挡父母修炼。

小静的腿不能向里撇，只能向外撇。小静生活很节俭，为了省钱供孩子念书，她也没有治疗。时间长了，她行动越来越不方便，而且还很痛。外地打工的丈夫得知后，催促她去医院检查，经医生诊断是筋腱坏死。小静没去医治，而是天天在心里诚心默念：法轮大法好，真善忍好。也不知过了多长时间，她发现腿不疼了，基本恢复正常了！小静亲身见证了法轮大法神奇，更加敬重大法师父！相信大法好！
年前，小静无意中触摸到乳房里有两个瘤子，她就抓紧时间默念九字真言，不知不觉瘤子小了。快过年了，丈夫回来了，陪着小静上市医院检查，医生说：“瘤子不大，没事。”小静不放心，要去矿务局医院再检查一遍，如果两家医院检查结果都一样，才确定没事了。丈夫又陪着去矿务局医院检查，医生看完片子说：“瘤子已经没有了。”小静放心的笑了！

小静得知孩子的辅导老师得了子宫癌，不能教学生了。她就去看望这位老师，临走时给老师二百元钱，老师没要，说：“我不缺钱，我就想吃农村养的小鸡下的笨鸡蛋。现在市场卖的笨鸡蛋都是假的。”善良的小静回农村娘家，跟母亲说了此事。母亲说：“冬天小鸡不爱下蛋，咱家就攒了二十来个，你都拿去吧，再送给老师一个大法护身符。”小静说：“也不知道老师是否相信？她丈夫可挺反感的。”小静妈说：“你把护身符和鸡蛋放在一起，方便你就跟她讲自己的亲身经历，让她诚心默念九字真言，她的病就会好转的。”小静就按着妈妈的办法做了。

后来，辅导老师主动给小静打电话：“你给我那个真是个好宝贝！我很想看看那本书，就想了解一下，怎么那么多人都在学呢？”于是小静又特意回娘家取了一本《转法轮》送给老师，老师高兴的爱不释手。小静一再叮嘱：“看书前一定要先洗手，看完后把书包好，放在高处保存好。你要学呢，就送给你；你要不想看了，一定要把书还给我。”老师很认真的照办了，而且还三退了。（退出党、团、队邪恶组织）后来老师的病不治而愈全好了。

老师又开班辅导学生了，每堂课要结束时，老师都很智慧的讲法轮大法真相，潜移默化的劝学生三退保平安。

#### 二、小涵默念九字真言得福报

小涵从小就体弱多病，上市医院检查是血液病，婚后不宜生育。家里花了很多钱四处为她医治，也治不好，能维持现状、不再恶化就已经不错了。后来小涵的父亲修炼了法轮大法，“一人炼功全家受益”[1]。从此小涵的病再也没有恶化。

小涵大学毕业后，在政府做临时工。她经常默念：“法轮大法好，真善忍好。”不知不觉病就好了！她更加相信大法的神奇！后来小涵结婚生了一个大胖小子，孩子很健康。小涵和丈夫都认同法轮大法是正法，经常默念九字真言，如今双双都考上公务员。

#### 三、小晴认同大法看淡名利

小晴从小就争强好胜，追逐名利，不肯吃亏。自从小晴的奶奶修炼法轮大法以后，就经常给小晴讲大法的美好，讲修炼故事。在大法法理的教化下，小晴不知不觉的在改变。大法也给小晴开智开慧，她的学习成绩直线上升，后来考上了一所理想的大学。

小晴毕业后又顺利的進入一家企业工作。小晴工作认真负责，受到大家一致好评。年终被评为“工资一等奖”。没想到“工资二等奖”得主给领导送礼，领导偷梁换柱，把“二等奖”换成“一等奖”，小晴由“一等奖”变为“二等奖”。小晴没有找领导理论，而是一笑了之过去了。

一次，奶奶给小晴邮了一包新鲜豌豆角和一包东北煎饼，这些都是小晴在家时爱吃的。当小晴打开邮包，傻眼了：菜都烂了，煎饼也碎了。姑姑得知后，让她找快递公司赔偿。小晴淡淡的说：“算了，坏就坏了吧，不找了。”

#### 四、老汉明真相后小喇叭不离身

有一位农村老汉六十多岁，经医院确诊：有心脏病、轻微脑血栓。他也没钱看病，就只能躺在炕上，痛苦的呻吟。大法弟子向他讲大法真相，并让他诚心敬念：“法轮大法好，真善忍好。”老汉很认同大法好！并爽快的同意三退（退出党、团、队邪恶组织）

老汉明真相后，诚心默念九字真言，一天天的精神起来了，也能下地干活了。后来大法弟子又送给他一个真相小喇叭，让他听。这一听，老汉如获珍宝、爱不释手，小喇叭不离身，吃饭时、走路时、干活时、睡觉前，每时每刻都很认真的听，还不住的赞叹。

老汉听了好长时间了，里面的内容他都能讲述出来。后来，他找到大法弟子，请求再给换换其它大法真相内容。于是大法弟子又重新更换真相内容，老汉满意的笑了。现在老汉红光满面，精神十足，他也说不清自己的病什么时候全好了，他只知道是大法师父救了自己。

世人明真相见证大法的神奇，无不感激大法师父救度之恩！赞叹法轮大法的美好！同时又是个活传媒，把大法的福音传递给更多有缘人！
