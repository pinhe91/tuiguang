### 不同的选择　不同的结局

文: 中国大陆大法弟子 

【明慧网二零二二年四月二十四日】我在某市一家企业担任要职期间，目睹了两任中共市委书记的不同沉浮。对大法的不同态度，注定了他们不同的命运与结局。
A在二零零八年任市委书记，当时该市迫害法轮功也很严重，但没听到过A书记对法轮功有什么言论。二零一零年我由于讲真相不注意安全，被人举报。市公安局国保大队及分局派人跟踪我一个多月后，于二零一零年十二月十四日晚，闯到我的住处绑架了我。我被关在铁笼子里，戴着脚镣、手铐，坐铁椅子。

第二天早晨公安局通知了我工作的企业。老板联系上了市委A书记。上班没多久，总经理带人来公安局先见了我，公安局国保、“610”头目等陪着。一进来他们就向总经理告了我一状，说我不配合、不签字、不认错。总经理了解我，也知道法轮功真相，没说什么，只对我说：“老板指示我到公安局参加局长例会，专门研究你的事，我马上就来了，企业各部门领导也都来了，企业暂时还离不开你，上下都惦记你。”我明白总经理的画外音：你别担心。

午餐时间刚过，总经理、市局国保、“610”头目等人一大帮人进来了，边给我打开脚镣、手铐，边说着话。那个“610”头目直转圈儿，说：“就这么放了？怎么这么快就放了？昨晚抓的今天就放了？！”

回来后，总经理对我说：“A书记表态了，要不他们准备把你送劳教的。”我说：“A书记及所有人的善言善行都会有福报的。”

A书记任市委书记期间，各种不利于他的传闻日甚一日。中纪委、省纪委明察暗访，多次来调查他，我也代表企业被多次询问过。但基本没影响他的升迁，现在他已经官至副部级多年。

B书记接替了A书记的位置。B书记长期在本地为官，“爪牙”遍地，该市对法轮功学员的迫害他负有不可逃脱的责任。

二零一九年一月十八日辖区派出所一位熟人给我打电话，很客气地说：“我们领导让我给您打个电话，问问您是不是人大代表或者政协委员？我们问了几个人，都不知道，让直接问问您。”说具体啥事他也不清楚，就让他了解一下。

我没回答他，立即给老板打了个电话。经老板了解，是公安局知道我炼法轮功，要找我的麻烦。市公安局C副局长（兼分局长）开会时亲自下的令，说是“政治任务”。

C局长是B书记一手提拔起来的心腹之人，唯B书记马首是瞻，是个骄横跋扈的角色。此时B书记已经调走，但仕途看涨；C局长一心追随B书记保持着所谓“政治站位”，想在迫害法轮功问题上做点文章，捞点政绩。

老板很紧张，找了几个人，都说从C局长的态度看，找谁都没用，如果找B书记，别的事行，法轮功的事恐怕不行。让我处理好手头的事，躲躲，让公安局具体办案人给拖着点，实在不行，他找省公安厅领导说句话。这样，我把工作处理了一下，离开了该市。

没过几天，B书记的各种腐败及要落马的传闻沸沸扬扬，C局长要出事的传言也是遍布大街小巷；市纪检委常务副书记（也是老板的朋友）用各种方式向C局长直接或间接释放要双规他的信息，C局长彻底蔫了，找我麻烦的“政治任务”不了了之。

我又回到某市，一切照旧！

二零一九年十二月新闻报道了B书记被双规的消息，后又被判无期徒刑。表面上看，是他由于贪腐等问题落马，实质上是由于积极迫害法轮功遭了恶报。C局长每天惊魂落魄，后被调到一个闲置部门。

B书记落马后，不明真相的市民为他鸣冤，说他太倒霉了，A书记比他腐败多了，不但没事还升官了！世人哪里知道，A书记在大是大非面前，有自己的立场，也算积了大福份，消减了他的一些业力。B书记是非不辨，盲目追随江泽民政治流氓集团迫害法轮功遭恶报，下场纯属必然！也是老天给他忏悔的机会，如果再不醒悟，那由于迫害法轮功所犯下的滔天大罪，恐怕下地狱都没资格，将在无尽的形神全灭的痛苦中偿还，非常可怕！

善恶有报，不是不报，时候未到，时候一到，一切都报！看看那些积极迫害法轮功捞取政治资本削尖脑袋往上爬的人，哪个有好下场了？！退出中共邪党保平安，这是神佛给你的自我救赎机会，但转瞬即逝！抓住吧！
