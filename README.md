### 法官求撤名与博士退党

文: 大陆大法弟子 

【明慧网二零二二年六月七日】共产党的灭亡是天定的，现在越来越多的人开始觉醒和反思。当善恶有报一步步向人推进，曾经参与迫害和拒绝真相的中共拥护者开始担心和觉醒。下面举两个真实的例子
刑庭庭长的请求

一天，以前认识的一个刑庭庭长，突然要和我说一件事。他说得知自己在国外被列入了被追查的名单，他恳请我帮助联系去掉他的名字。

该庭长称，当初他给法轮功学员判刑，是“610”和政法委强制性的，他不判别人也得判，所以没有办法。现在知道被列入了被追查名单，十分担心影响到自己及子孙后代，并说再也不参与迫害法轮功了。

博士终于退党

一天，再次给一个认识的博士讲真相。她是一个比较优秀的人才，也是一个十分单纯、善良、倔强的人，之前给她讲真相她固执的不听，但是这次见她却满脸的委屈与无奈。

我再次给她讲真相，从中共篡政以来的各种运动，到文化大革命、六四、迫害法轮功、迫害异议人士一点点地讲，她静静地听，和之前的高傲、反对形成鲜明的对比。

随后她讲了自己的经历，讲到伤心处不断地流泪，她说自己一心工作，不断地研发，都是自己投资，为了创业，为了实验，卖掉房子，她踏踏实实地钻研，因为不善言辞与社交，家人都是普通农民，没有背景，最终的成果却成了政府和别人的，而自己还背负着一身债务。她的泪水透露出极度的无奈。

说到退出党员、团员、少先队员，这一次她不再拒绝，而是发自内心的同意。她说，当初是真心加入中共，感觉加入了能更好的利用所学专业造福百姓，不辜负所学，但是加入后才知道，中共完全是说一套做一套，宣传中共如何好，如何为百姓好等等都是骗人的。她后悔加入，尤其是遭遇种种不公后，她更加后悔，并说如果不是亲身经历她不会相信中共是如此邪恶的，完全是反人性的。

可贵的中国人快快觉醒吧，我们是神的子民，不是马列子孙，时间不等人，不要充当邪党的陪葬，快快找法轮功学员退出中共党员、团员、少先队员。
