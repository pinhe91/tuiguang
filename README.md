### 坚守善念　重庆张利群女士讲真相又遭绑架折磨

【明慧网二零二二年六月二十日】（明慧网通讯员重庆报道）重庆市永川区法轮功学员张利群讲真相遭人恶告，二零二二年六月十日被萱花派出所警察绑架，在看守所被非法关押一周，于六月十七日回家。期间，张利群遭到朱冠熙、“胖娃”等警察的残忍折磨。
十几年来，张利群女士因遭中共迫害，失去家庭，无户口、无身份证，无法找工作，真是一无所有。在极为艰难的日子里，她仍然坚守着善念，坚持给受中共毒害的民众讲真相，告诉他们大难来时保命的方法。

二零二二年六月十日下午，张利群在永川区体育馆附近向三名学生介绍如何度过大劫难，不料被这三个学生报告给警察，被胜利路萱花派出所俩警察绑架。

当晚，张利群遭强制体检，因她不配合，遭到警察的虐待，如体检时，警察将她双手反铐在背后，一个叫朱冠熙（音）的警察用手在张利群后背使劲向上拉扯手铐，令她剧痛不已。第二天上午，张利群被劫持去验核酸，刚一出萱花派出所，一个叫胖娃的警察一手在张利群后背使劲拉扯手铐，一手用拳头猛击张利群的后背。在萱花派出所，张利群双手双脚始终被铐在铁椅子上，头上方是空调对着吹冷风，连续二十七小时，张利群被冻的不停地颤抖。

第二天晚六点时，两个永川区公安分局的警察到派出所，将张利群连夜劫持到合川看守所。因张利群一直未说出任何个人信息，警察就用虚假信息办理入看守所手续，看守所人员表示，电脑不能正常录入（虚假个人信息），无法收监，只能将人退回。永川公安分局警察坚持要看守所想办法录入或修改录入程序。一个多小时后，双方警察终于用造假信息把张利群塞进看守所。

几天后，警察通过天网工程查到张利群的住处，随即非法抄家，据悉搜走了一本《转法轮》书籍。

六月十七日，警察将张利群从合川看守所拉到永川公安局，搞了一个问询笔录。之前已搞过两次，每次都是警察自编自答自己签字。当天下午，张利群又被重庆沙坪坝公安局、重庆大学社区、重庆大学公安局一行四人劫持到重庆大学公安局做笔录，并给同去的张利群的姐姐也做了一个笔录，之后才将张利群释放。

张利群被迫害致一无所有

张利群原是重庆大学金沙电气研究所职工，是一名会计师。她在二零零零年修炼法轮功后，身体状况得到极大的改善。

二零零二年初，张利群去南坪小泉参加法轮功学员修炼交流会，途中被警察绑架，先后被劫持到沙坪坝看守所、沙坪坝强制戒毒所、歌乐山洗脑班非法关押、迫害。二零零二年四月，张利群从戒备森严的洗脑班智慧走脱。

沙坪坝区公安分局警察对此恼羞成怒，到处追找张利群，包括骚扰她的同事、同学及外地老家的亲戚，并在网上通缉她，还冻结了她的户口，企图封死她的生存空间。沙坪坝公安局还给张利群当时所在工作单位施压，使得单位在压力下违心地开除了张利群，并收回了公司分给她的住房。

中共人员还给张利群的丈夫李运良施压，逼他在单位写检查。李运良承受不住压力，无奈下到法院单方面解除了婚约。

张利群被迫流离在外。十几年来，她无家、无户口、无身份证，无法找工作，真是一无所有，还要躲避中共的迫害。
