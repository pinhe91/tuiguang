### 河南荥阳市七旬老人聶俊花被迫害成植物人

【明慧网二零二二年十二月十三日】（明慧网通讯员河南报道）河南省郑州市荥阳市七旬法轮功学员聂俊花，去年六月被绑架，后被非法判刑五年，期间遭到看守所、法院、监狱的迫害，她现已被迫害成植物人。
二零二一年六月二十七日，聂俊花老人在外出办事时，被荥阳市国保警察伙同京城路派出所警察绑架。聂俊花被非法审讯两天后，被关押在郑州市第三看守所。由于她不配合所谓的“转化”，被迫害得身体非常虚弱，于二零二一年九月二十六日晕倒在监室里。两天后，看守所才通知家人：聂俊花查出多种疾病，高血压、糖尿病、脑梗塞、子宫肌瘤等。家人非常震惊，不相信，因为聂俊花在家时身体健康、红光满面，走路生风。

二零二一年十二月，郑州市中原区法院不通知家属，秘密开庭，又一次对聂俊花老人非法判刑五年，罚金两万元。

二零二二年一月份，警察不通知家属，将生活已不能自理的聂俊花拉到郑州市第二人民医院做手术，直到二零二二年二月十四日才通知家属。家属赶到医院时，看到聂俊花已被迫害得面目全非，体重从原来一百四十多斤降到只有七、八十斤，生命处于垂危状态。而中共郑州市中原区法院的邪恶之徒还把这样状态的聂俊花拉到河南新乡女子监狱继续迫害。

仅几个月的时间，聂俊花就被新乡女子监狱的狱警迫害成了植物人。狱方于二零二二年十一月份用保外就医的邪恶手段通知她的家人把她弄到郑州市中医医院医治。治疗二十多天没有好转，现又转到另一个医院就治。据悉，聂俊花现在是全身不会动，她的一切全靠她丈夫照料。

聂俊花老人曾于二零零八年被非法判刑两年，于二零零九年十二月十日结束冤狱。
