### 山东青岛曲辉芳十三年前被迫害离世的事实

【明慧网二零二二年九月二日】（明慧网通讯员山东报道）法轮功学员曲辉芳女士，一九七零年出生，高中毕业，原山东省青岛平度市香店（现凤台）街道办事处西郝家疃村人。一九九九年七月江泽民发动对法轮功的残酷迫害以来，她遭受了种种非人的迫害，身体健康每况愈下，于二零零九年九月二十四日凌晨凄惨离世，年仅39岁。

![image](https://user-images.githubusercontent.com/79625284/188109187-2554cd5d-f618-4b84-b70a-814def50fb7a.png)

曲辉芳

曲辉芳从小身体虚弱，瘦得皮包骨，干不了重活。一九九七年，看到她的大姑姐修炼法轮功后无病一身轻，变得善良宽容，曲辉芳也开始修炼法轮功。

修炼法轮功后，曲辉芳经常与当地的法轮功学员一起交流修炼心得，她的心胸开阔了，不怕吃亏，每当遇到矛盾时，她都能按照法轮功的教导找自己的不足，与人相处和睦。修炼后，曲辉芳的身体也变得强壮了，自己一个人能搬动一麻袋粮食，这简直是奇迹！她与丈夫整天出去赶集收粮食，日子过得蒸蒸日上。
大约在二零零二年，曲辉芳的女儿上小学一、二年级时，家中老人骑着自行车送孩子去上学，快到学校的一个红绿灯路口，突然被后面急驶而来的一辆摩托车撞倒在地，孩子的头撞在电线杆上，骑摩托车的小两口（麻兰镇千里村人）吓坏了。这一幕恰好被同村的人看见，急忙给曲辉芳打电话。曲辉芳赶到后，看着老人和孩子，心疼不已，但是她还是宽慰肇事的小两口：“不用担心，我们是修炼法轮功的，不会讹你们。你们走吧。”没有要他们任何赔偿。小两口万分感激地说：“可遇到好人了！”后来，小两口又去看望孩子，提出领孩子去青岛玩，曲辉芳安慰他们不要心存愧疚，他们也不是故意的，不要花钱。老人和孩子在师父的护佑下，仅是皮外轻伤。

曲辉芳曾经感叹：一九九七年修炼法轮功以来，到一九九九年七月江泽民发动对法轮功的残酷迫害之前，这短暂的两年时间是她一生中最快乐的时光。

一九九九年七月二十二日凌晨，曲辉芳去平度市市政府反映法轮功教人修心向善的真实情况，被有关人员拉到平度党校，下午又被拉到香店党委。香店党委的有关人员不断地给她施加压力，逼迫她写“不炼法轮功的保证书”，二十三日凌晨才让她回家。

之后，中共有关人员对曲辉芳的骚扰一直不断。

大约二零零零年农历十月份，一个寒冷的日子，曲辉芳一大家人正在她家烤着火炉闲聊，香店派出所四、五个警察突然非法闯入，以阻止她去北京为法轮功上访为由，把曲辉芳强行拖着抬到警车上，拉到香店派出所非法关押好几天，并敲诈勒索她三千元钱。

大约二零零零年农历十一月份，曲辉芳依法到北京为法轮功上访，在潍坊被中共人员截回，被非法关押在香店党委。在那里，中共人员逼迫曲辉芳只穿单薄的秋衣秋裤，赤着脚站在雪地里，双手手心贴在冰冷的水泥墙上，在阴暗的楼后罚站，曲辉芳被冻了整整一宿。第二天，中共人员把她关在一间小屋里，一直不让她睡觉，一睡觉就打她。

在香店党委被折磨两三天后，曲辉芳又被绑架到平度拘留所；非法拘留半个月后又被拉回香店党委。因为曲辉芳炼功，当时的香店党委副书记王明乐气急败坏地将她绑架回平度拘留所，再次非法拘留她半个月。面对时时对她的强制洗脑，逼迫她放弃对法轮功的信仰，曲辉芳承受着巨大的精神压力。

快过年了，拘留所里被非法关押的法轮功学员陆陆续续地都回家了，阴森恐怖的牢狱里只剩曲辉芳一个人，她睡在冰凉的炕上，盖着散发着臭味的破被褥，她牵挂着刚刚五岁的女儿……

半个月后，曲辉芳被拉回到香店党委。因为不放弃修炼法轮功，曲辉芳再次被香店派出所非法关押十天。这次被迫害，曲辉芳共被非法关押四十多天，被敲诈五千元钱。

从香店派出所被放回家后，曲辉芳的身体出现了严重问题：面色发黄、食欲不佳，最明显的是口腔有氨臭味。后来医生告诉家人说：惊恐伤肾，再加上曲辉芳的手脚被长时间冰冻，犹如雪上加霜，这一切导致她肾脏受到严重伤害。

二零零一年四月份，小妹曲辉莉被绑架并失去教师工作。曲辉芳再次承受了沉重的打击。

二零零一年六月份，警察又去曲辉芳的村里骚扰，要去看曲辉芳。曲辉芳吓得不敢回家。

一到所谓的敏感日，中共参与迫害人员就去骚扰曲辉芳，一年去两三次。

一次次的被绑架、非法关押，一次次的骚扰、一次次的惊吓，亲人的被迫害离世……这一切迫害令曲辉芳每天生活在巨大的恐惧中，她宁愿上街溜达，也不敢一个人在家，根本没法静心学法炼功，她的身体每况愈下，于二零零九年九月二十四日凌晨凄惨离世，年仅39岁，留下年仅14岁的女儿和花甲之年的父母。
