### 丈夫正气震慑邪恶

文: 山东大法弟子　爱莲 

【明慧网二零二一年九月十四日】二零一五年五月，中共最高法院宣布所谓“有案必立，有诉必理”，法轮功学员纷纷起诉发起这场迫害的恶首江泽民。多地大法弟子由于起诉江泽民，遭到610、派出所人员的骚扰，甚至绑架。
我也起诉了江泽民，发往最高检察院、最高法院各一封信。有一天，来了一大帮子人闯進我家，有市610、派出所警察等一伙人，其中一人问我：“你起诉江泽民了？ ”我说：“是呀，我还写了两封呢，高检高法各一封。”为首的人说：“你为什么起诉江泽民，那可是国家元首？”我说：“我炼法轮功，修真善忍做好人，江泽民迫害法轮功，我不应该起诉它吗？再说这也是我的权利。”

就在这时，我丈夫从外面回来了，一看这个阵势，就说：“有什么事找我。我老婆做好人不犯法。”那人说：“你老婆起诉江泽民。”我丈夫说：“起诉江泽民怎么了，我也要起诉江泽民。”那人说：“你老婆炼法轮功起诉江泽民，你又不炼法轮功。”我丈夫说：“这个江大蛤蟆上台，就没干点什么好事。你说，这个江大蛤蟆干了什么好事，我现在打电话找十个人来，有一个说这个江大蛤蟆好，就算我输了。它不光迫害法轮功，还是个卖国贼，出卖国土……”

听着丈夫这些话，来的两个年轻人在那笑。610主任一看我丈夫这样，坐不住了，站起来就往外走。一帮人就都跟着往外走。

我丈夫说：“别走，我还没说完呢！”这帮人来的时候气焰嚣张，走的时候却是灰溜溜。
