### 江西省萍乡市副市长叶华林遭报被查

【明慧网二零二二年四月二十六日】（明慧网通讯员江西报道）据二零二二年四月二日江西消息，江西省萍乡市政府原中共党组成员、副市长叶华林因严重违纪违法，主动投案，已被开除公职，并立案调查。
叶华林，男，汉族，一九七一年六月出生，江西安远人，于二零一六年十月至二零二一年九月，担任江西省萍乡市政府党组成员、副市长。叶华林作为萍乡市的主要领导，对当地法轮功学员被非法骚扰、监视、绑架、关押洗脑班、拘留、判刑等迫害，负有不可推卸的领导责任。

以下是叶华林任职江西省萍乡市政府党组成员、副市长期间，发生在萍乡市对法轮功学员迫害的部份事实：

#### 一、萍乡市法轮功学员刘旺遭监视、骚扰

刘旺，男，萍乡市八一街罗家塘社区的法轮功学员。二零一八年三月四日下午，萍乡市公安局国保支队，李姓和杨姓两个警察到刘旺家骚扰。

从二零一八年三月六日晚上开始，警察对刘旺实施监视居住，为所谓防止刘旺在两会期间去北京上访，警察在刘旺外出期间进行跟踪和监视。参与监视的人员有八一街派出所警察、协警和安源区公安分局处理突发事件大队的部分警察。

二零二零年十二月十五日下午四点多，由萍乡市安源区政法委聂姓书记带队，共六人，到刘旺家骚扰。整个二零二零年，安源区公检法人员上门骚扰共达六次之多。

#### 二、萍乡市十四位法轮功学员被非法拘留

二零一九年四月二十三日下午，萍乡市十四位法轮功学员在萍乡市富利昌大厦八楼阅读法轮的书籍时，遭十多位警察非法抓捕。

警察在没有出示任何搜查手续的情况下，把整个场地翻抄了个遍，四处一片狼藉。然后对法轮功学员进行非法搜身，从法轮功学员王金娥的包里搜出了大厦八楼的钥匙和大量真相币，就认为她是所谓的“领导”并强制抄她家，从她家中抄去一大袋法轮大法书籍和资料。

傍晚五点左右，警察把所有法轮功学员绑架到萍乡市安源区的所谓“扫黑基地”，通知各派出所分别将人接回并进行非法讯问后，把法轮功学员都劫持到萍乡市拘留所非法关押迫害。

#### 三、萍乡市安源区杨秀英等老年法轮功学员遭绑架

二零一九年五月二十日下午四点左右，萍乡市安源区国保大队大队长江萍，带领下属的后埠派出所七、八个警察，早早蹲守在法轮功学员杨秀英家门外。

屋内的法轮功学员开门出去时，江萍等人就气势汹汹地冲进去，一边强制她们坐着不许动，一边开始在屋内翻箱倒柜搜查，不仅把法轮功书籍和法轮功师父的法像全部抢走，还把八十多岁高龄的杨秀英和六十多岁的刘金明、赖芳萍劫持到后埠派出所进行非法讯问、逼迫签字，直到晚上十点钟才将她们释放回家。

#### 四、萍乡市的杨庭先老人被非法判刑

杨庭先，男，六十二岁，萍乡市法轮功学员。二零一九年十二月十六日，杨庭先携带法轮大法经书和真相挂历，预备乘坐火车前往宜春市分宜县时，在火车站的进站口遭五陂下派出所警察拦截、绑架，后被劫持到萍乡市看守所。看守所因杨庭先血压高达230mm汞柱而拒收，杨庭先就又被带到萍乡市三医院体检，医院要求住院，警察说没人看守，就又将杨庭先押回看守所强行关押。

在看守所，杨庭先天天被强迫吃降压药，一个多月后，不仅血压没有降下来，脚板底还起泡流水。二零二零年二月二日傍晚，杨庭先被“取保候审”回家。

二零二零年六月二十二日，杨庭先被萍乡市芦溪县检察院构陷到芦溪县法院。芦溪县法院从六月二十三日开始对杨庭先实行“监视居住”。二零二零年七月十日，杨庭先被芦溪县法院非法判刑一年六个月、勒索罚金两千元。

#### 五、萍乡市肖秋文等三位法轮功学员被非法判刑 回家后遭骚扰

二零一九年十二月十七日上午，萍乡市法轮功学员肖秋文、肖放梅和胡梅香，到萍乡市开发区玉湖小区免费向民众赠送真相台历，遭人恶意构陷。当天下午，肖秋文等三人在另一位法轮功学员邬春梅家阅读《转法轮》书的时候，被突然闯入的一大伙气势汹汹的警察绑架。同时被绑架的还有邬春梅、姚和珍、姜启兰和易美兰四位法轮功学员。邬春梅、姚和珍、易美兰被释放回家；姜启兰被送萍乡市拘留所，非法关押十天；肖秋文、肖放梅和胡梅香三人则被绑架到开发区公安分局非法讯问，晚上十一点半左右，三人被绑架到萍乡市看守所非法关押迫害。

二零二零年七月一日，萍乡市芦溪县法院对三人非法视频开庭，三人分别被冤判：肖秋文九个月刑期、肖放梅八个月刑期、胡梅香七个月刑期，三人均分别被勒索罚金一千元。

三人被释放回家后，又遭各个部门骚扰。肖秋文和刘旺母子，前后被八一街派出所、八一街街道办和罗家塘社区共计骚扰五次；胡梅香被五陂镇派出所的警察上门骚扰；肖放梅的儿子多次被八一街派出所的警察打电话骚扰。

#### 六、萍乡市多位法轮功学员遭政法委等骚扰迫害

从二零二零年十一月中旬开始，萍乡市政法委指使公检法对法轮功学员进行有组织、有计划、有详细周密部署的“清零”迫害，已知被骚扰的法轮功学员有：荣爱梅、黎心尧、肖秋文、刘旺、邬春梅、赖欢萍、周爱清、刘淑兰、施冬芬、胡梅香等人。“清零”行动的目的就是利用一切手段迫使、诱骗法轮功学员放弃对真、善、忍的信仰。

在这次“清零”行动中，对法轮功学员采取威逼利诱、恐吓威胁、死缠烂打等卑鄙手段进行迫害。刘旺被威胁如果坚持信仰，和其他法轮功学员“串联”或者翻墙上网，就要遭到严厉打击；邬春梅和她的丈夫被诱骗到安源山庄（所谓“扫黑基地”），被二、三十个公检法人员恐吓威胁；刘淑兰则被安源区凤凰街道办商城社区的人员长期多次骚扰，被逼签字放弃信仰。

#### 七、萍乡市安源区洗脑班对法轮功学员强制洗脑

从二零二一年三月十五日开始，萍乡市安源区政法委，在位于萍乡市芦溪县高坑镇泉江小学旧址开办针对法轮功学员进行“清零”迫害的洗脑班，这个洗脑班叫做所谓的“关爱之家”。

洗脑班总投资两百多万元，在洗脑班的黑板报上写满了污蔑大法和大法师父的邪恶文字。每期洗脑班为期两周，几乎全市所有的法轮功学员都被要求参加，洗脑班上或者用恐吓威胁、或者用欺骗伪善的手段，强行逼迫法轮功学员放弃对法轮大法的信仰。

已知被强制参加洗脑班的法轮功学员有：黎心尧、荣爱梅、金玉莲、王金娥、李久香、赖欢萍、陈艳、李久香、施冬芬、杨秀英、胡梅香等人。
