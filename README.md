### 美国制裁迫害法轮功的中共官员

文: 中国大陆大法弟子 

【明慧网二零二二年十二月十一日】（明慧记者王英编译报道）美国国务院在十二月九日国际人权日前夕宣布，对65名个人和实体实施制裁，其中包括任意关押法轮功学员的前中国重庆地区监狱副局长唐勇。
美国国务院确认前中国重庆地区监狱副局长唐勇，任意关押法轮功学员，严重侵犯人权，极其严重地违反宗教信仰自由而被列入制裁名单。

美国国务卿布林肯九日发表声明，他说，在国际反腐败日和人权日前夕，美国政府对超过65名个人和实体实施金融制裁和签证限制，以促进对腐败行为和侵犯人权行为的问责。人权属于世界各地的每一个人。

![image](https://user-images.githubusercontent.com/79625284/206895636-de79909d-d218-4548-936d-2dafdad42628.png)

图1：美国国务院首席副发言人韦丹特•帕特尔举办新闻发布会

国务院首席副发言人韦丹特•帕特尔（Vedant Patel）在当天的新闻发布会上说，在国际反腐败日和国际人权日前夕，美国对十七个国家的几十名个人和实体实施制裁，这些人犯下了贪污罪和侵犯人权的罪行。美国将利用一切适当的工具和权力，包括限制签证，让他们承担责任。

同时被制裁的还包括前中共西藏党委书记吴英杰，西藏公安厅长张洪波，以及大连海洋渔业公司和平潭海洋实业公司的李振宇、卓新荣。

前中共西藏党委书记吴英杰在二零一六年至二零二一年担任西藏党委书记期间，法轮功学员受到严重迫害。吴英杰至少在两起迫害案例中，被列为参与迫害的部份主要责任人。

这不是美国政府第一次制裁迫害法轮功的中共官员。二零二一年五月十二日，美国国务院宣布，制裁中共迫害法轮功的官员、四川省成都市前“610”办公室主任余辉。

余辉担任成都“610”办公室主任期间，法轮功学员遭到严重迫害。二零零一年十一月，藏族法轮功学员扎桑在四川省成都家中被成都“610”浆洗街派出所非法抓捕，非法关押在成都郫县犀浦看守所，并被非法判刑三年。由于在狱中关押期间饮食、住宿条件恶劣，出现旧病复发状态，后保外就医，二零零二年十月在拉萨家中含冤离世，时年56岁。

二零二零年十二月十日，美国国务院宣布，对17名迫害人权的外国官员进行制裁，其中包括迫害法轮功学员的中共执法部门官员黄元雄及其配偶。

中国厦门公安局梧村派出所主管黄元雄，参与了严重侵犯人权的行为。黄元雄极其严重地侵犯了法轮功学员的宗教自由。他参与了对法轮功学员拘留和审讯。制裁也适用于黄元雄的配偶。
