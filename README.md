### 云南大理州公安系统八名局长遭报被查

【明慧网二零二一年十月二十八日】（明慧网通讯员云南报道）大理白族自治州地处云南省中部偏西，东邻楚雄州，南靠普洱市、临沧市，辖一个县级市：大理市，八个县：祥云县、宾川县、弥渡县、永平县、云龙县、洱源县、剑川县、鹤庆县，三个自治县：漾濞彝族自治县、南涧彝族自治县、巍山彝族回族自治县。
自一九九九年中共迫害法轮功以来，大理州公安系统对当地法轮功学员实施抄家、绑架、恐吓、罚款、拘留、劳教包括构陷至检察院、法院使法轮功学员被枉判送入监狱遭受更进一步迫害。

善恶有报是天理，近几年来，大理州公安系统不断“地震”，前后几十名公安系统人员遭报落马，其中八名是公安局长，这正应了一句古话：“以害人始，以害己终。”

八名落马被查的公安局长是：

1、吴学着，大理州公安局局长。据云南二零二一年十月消息，吴学着（落马前任迪庆州公安局局长）被免职并调离当地十二天后被查。
2、严亚铭，大理市公安局副局长、刑侦大队原大队长，二零一九年四月被查。
3、杨建军，大理市公安局副局长，二零一九年四月被查。
4、杨仁伟，大理市公安局副局长，二零一九年三月底被查。
5、李彪，大理市公安局局长。二零一九年五月，时任昭通市副市长、公安局局长的李彪被查，同年九月被提起公诉。
6、潘峰，大理市公安局副局长，二零一九年四月被查。
7、刘鸿俊，大理市公安局原副局长，二零一九年四月被查。
8、梁正军，大理州公安局副局长，大理市公安局局长，二零一七年十一月落马。二零一九年三月二十九日，云南省曲靖市中级法院对外公布梁正军受贿一审刑事判决结果，梁正军被判处有期徒刑十一年，并处罚金一百万元。

（一）吴学着，男，汉族，一九六五年十二月生，云南巍山人。一九八七年七月至二零零二年十月在大理州巍山县公安局工作；二零零二年十月至二零零六年三月任大理州漾濞县公安局党委书记、局长；二零一三年五月至二零一七年十一月任大理州公安局副局长；二零一七年十一月至二零一九年六月任大理州公安局党委副书记、常务副局长；二零一九年六月至二零二一年九月任迪庆州副州长、州公安局局长；二零二一年十月被查。

吴学着任职期间，法轮功学员遭迫害的案例（部份）：

#### 1、宾川县石建伟、肖竹夫妇遭冤判，石建伟在省一监被迫害至“肝癌”

石建伟与妻子肖竹都曾是宾川县一中的教师，石建伟是英语教师，肖竹是体育教师。在中共对法轮功学员的残酷迫害中，夫妇二人被强迫调离原单位工作。俩人被教育局调到距离宾川县城四十多公里的半山区的力角镇海良完小，并派专人监视夫妻俩人的一举一动，随时上报，同时还把他们夫妇两个人由原先的中教二级降到最低一级的工资待遇。二零零六年，肖竹被调到力角镇中心学校教体育，石建伟辞职离开了学校。

二零一五年九月七日下午，宾川县国保大队警察闯到石建伟家，进屋后不由分说将石建伟和他的女儿石佳按倒在地殴打。之后将石建伟、肖竹夫妇和女儿石佳绑架到宾川县公安局，抢走私人物品包括家门钥匙、车钥匙。石建伟、肖竹夫妇分别被非法关押在宾川县看守所和大理州看守所。

石建伟和肖竹夫妇被绑架后，在宾川县国保大队长杨瑜和副队长刘靖宇、教导员夏建国以及宾川县公安局的刘姓副政委的指使下，国保大队到宾川县十几户法轮功学员的家中进行非法搜查，抢劫法轮大法书籍、光盘若干，并且对多名法轮功学员和他们的亲属非法“传唤”，威胁、刑讯逼供，要求他们提供所谓的“证据、证言”，以此作为迫害石建伟、肖竹夫妻二人的“证据”。

二零一六年六月二十三日，石建伟、肖竹夫妇在祥云县法院被非法开庭，石建伟讲述了自己在公安局的地下室被国保大队长杨瑜指使的十几名警察三次殴打，反扭他的手臂，用膝盖猛顶腰部，用脚猛踢腹部，并且用脚猛踩头部。肖竹也揭露了国保警察用其女儿的人身安全威胁她，诱供、刑讯逼供。夫妻二人都坚持信仰法轮功，坚持修炼无罪。

二零一六年八月律师收到了大理州祥云法院对石建伟、肖竹夫妻二人的非法判决书，石建伟被非法判刑六年半，肖竹五年。夫妻二人不服判决，上诉至大理州中级法院，但是大理州中院依然维持了原判，之后石建伟被送到云南省第一监狱，肖竹被送到云南省第二女子监狱。

石建伟的家人于二零二一年八月二十五日接到监狱打来的电话，称石建伟疑似“肝癌”，已经被送往监区中心医院。目前石建伟体重下降十多公斤，情况已经很严重。

#### 2、朱会林、王顺仙夫妇遭绑架，王顺仙被冤判

朱会林与王顺仙夫妇均是大理州宾川县法轮功学员。二零一八年一月二十三日新年前夕，大理市公安局下辖的宾川县公安局、“610”（专门迫害法轮功非法组织）对法轮功学员挨家挨户骚扰，到朱会林、王顺仙夫妇开的百货店时，看到店里挂着法轮功的物品，就强行进入非法抄店，并绑架了夫妻二人，朱会林被非法关押在宾川看守所，王顺仙被非法关押在下关市看守所。二零一八年十月，王顺仙被当地法院非法判刑三年半，非法罚款八千元。

#### 3、巍山县苏慧芝等四位法轮功学员参与“诉江”被绑架、判刑

二零一五年十一月八日十日，巍山县“610”、国保大队警察绑架了巍山县参与诉江的法轮功学员苏慧芝、程玉梅、左释光、胡开兰，其中左释光，胡开兰当天被送派出所审讯后回家，苏慧芝、程玉梅被劫持到大理州看守所非法关押，之后被非法判刑，苏慧芝被非法判刑四年，程玉梅刑期不详。之后俩人被送到云南省第二女子监狱继续迫害。巍山县苏慧芝67岁，原巍山县建筑公司的工人，后自营一家书店，一九九九年中共迫害法轮功后，苏慧芝多次遭当地不法人员骚扰、恐吓（当时吴学着正在巍山县公安局工作）最终书店被迫关闭。之后苏慧芝又被非法劳教一次、冤判两次，近九年的时间在邪恶的黑窝中遭受迫害。

#### （二）大理市公安局副局长严亚铭、杨建军，二零一九年四月落马被查。其任职期间法轮功学员遭迫害的案例（部份）：

#### 、大理大学毕业生李慧敏被非法判刑三年半

李慧敏，女，一九九五年四月十八日出生，大理大学生物工程系大四毕业班学生，福建人。二零一六年一月九日同学聚会，赠送29个U盘（内含有九评，活摘器官等真相资料及自由门软件等）给同学，却被不明真相的同学告知班主任，班主任报告学校保卫处，由保卫处报大理公安局。一月十二日，李慧敏被绑架并被抄走手提电脑及几张神韵光盘与真相传单等。后被关押在下头镇文献村看守所，在看守所内李慧敏向同室人员讲真相，被看守所戴上四十斤重的脚镣刑罚一周，并且被罚站。四月七日案卷移送大理市检察院审查起诉，大理市法院八月一日对李慧敏非法开庭，之后做出非法判决，对李慧敏非法判刑三年半并罚金一万元。家属不服结果，上诉到大理州中级法院，十二月大理州中级法院非法维持了邪恶的原判，李慧敏被送到云南省第二女子监狱继续迫害。

（三）二零一九年三月底，大理市公安局副局长杨仁伟落马，接受纪律审查和监察调查。杨仁伟对法轮功学员朱会林、王顺仙、石建伟、肖竹夫妻被迫害负有不可推卸的责任。

（四）时任昭通市副市长、公安局局长的李彪，二零一九年五月落马被查，被双开，同年九月被提起公诉。

李彪，男，一九九八年零三月任大理州弥渡县公安局局长；二零零三年七月任大理州公安局国保支队政委；二零零五年零七月后大理州公安局副局长、局长、大理市副市长；二零一五年七月任昭通市中共党组成员、市公安局党委书记；二零一五年八月任昭通市副市长、市公安局局长、党委书记。

其任职期间迫害案例（部份）：

#### 1、刘国花被冤判三年半，二零二一年又遭绑架

刘国花，女，53岁，宾川县太和农场柳家湾十五队农民，因向民众讲述法轮功真相，二零一二年七月二十四日被宾川县桥殿镇派出所警察绑架、关押，后被包括派出所在内的县610、县国保大队十多人非法抄家。宾川县法院二零一三年一月二十四日对刘国花非法庭审，之后对刘国花非法判刑三年半，并把刘国花偷偷劫持入云南省第二女子监狱。

二零二一年五月十四日，刘国花再次在自己家中遭到绑架，来了二十多个人，其中有特警，国保、610人员等，这些人当场给刘国花下了逮捕证，并在没有出示搜查证的情况下，把家里翻了个底朝天，抢走大量刘国花的私人物品，刘国花被绑架后一直被关押在大理州看守所至今。

#### 2、王玉林被非法判刑三年

王玉林，男，宾川县金牛镇金殿后坝田村人。二零一二年三月二十日遭到了宾川县公安局国保大队警察非法抄家、绑架，并构陷到宾川县检察院，二零一二年十月十二日早上宾川县法院非法对王玉林开庭审理，宾川县法院（2012）宾刑初字第143号对王玉林非法判刑三年，但判决迟迟未给王玉林。过了上诉期限两个多月法院才给家属打来电话叫王玉林上诉。王玉林于二零一二年十一月二十六日向大理市中级法院提出上诉。家属三周后向大理市法院询问，一个法官说只能维持原判。

二零一三年二月，王玉林的家属接到宾川县看守所的通知，说要将王玉林送往监狱，叫他们来看守所和王玉林见一面。下午两点钟，家属在看守所的接待室见到被非法关押近一年的王玉林，发现他的门牙已经掉了两颗，终日见不到阳光，身体非常虚弱。二月十七日王玉林被送到大理州监狱。

#### 3、黄玉芳被非法判刑五年

二零零五年五月，大理市电信公司职工黄玉芳发放真相资料时被不明真相者构陷，遭警察绑架，后被非法判刑五年，送到云南省第二女子监狱遭受迫害。

在江泽民发动的这场惨绝人寰的迫害中，那些曾经参与和正在参与迫害的人，从省、市到基层，明知法轮功学员都是善良的好人，为了职务、为了饭碗、为了自保，昧着良心犯罪，都将面临正义的审判，善恶必报。现在没有遭报，是老天爷想给其中还有可能改过的人留下希望与机会，其实他们才是这场迫害的受害者、牺牲品。而且人间的报应只是为了警醒世人，地狱的报应那才是偿还恶业的过程，还会殃及子子孙孙。古今中外，迫害正信的强权暴政从来没有一个成功过的，所有残害良善的元凶、爪牙没有一个善终的。
