### 大妮说：这“九字真言”真好

【明慧网二零二二年十月十五日】

#### 大妮说：这“九字真言”真好

我赶小集做生意，讲真相的环境比较宽松。有一次，我给大妮和她得脑溢血的丈夫讲了真相，并给她丈夫做了“三退”，没几天，她的丈夫竟然自己能走路了。她的左邻右舍看到她丈夫的变化后，都先后找她要大法真相资料。

大妮说：自我看了大法真相以后，我心里常念：“法轮大法好，真善忍好”，我的庄稼也长得好了。别人说我什么，我也不往心里去了，什么事也不往心里去了。这“九字真言” 真好，没事我就看法轮功的真相资料。
我为这些明真相的人高兴。

#### 大法给予五岁孩子的福报

〖中国江苏来稿〗我是一名专职带宝宝的阿姨。二零二零年十二月，我去一家照顾一个五岁的男孩。这个孩子不但体弱多病，且容易过敏、脾气特别暴躁。

刚去的时候，孩子没上学，孩子的妈妈叫我每天早晚给他吃多种抗过敏的药。当时我心里就想：我是大法弟子，我要把大法的福音告诉他，他能够得到福音，以后就不会生病了。

于是我就先对着他默默的念“法轮大法好，真善忍好”。念了一段时间我发觉孩子脾气变的好多了，吃东西也不那么容易过敏了。我心里很感激师父。过了一些日子，我在陪孩子写字的时候，就叫他跟我一起念“法轮大法好，真善忍好”。他很听话，总是开心的跟着我大声的念，那声音大的能震撼整个三楼。

一天，他突然笑眯眯的对我说：“阿姨，我知道你是修善的。”还说难怪我为什么对他不凶。他说以前的阿姨都对他很凶，会骂他，就我从来不对他凶。他还说他知道我是修真善忍的。

我当时就抱起了他，开心的对着他说了一句：“你真是个聪明宝宝！”感谢师父，都是师父赐予了他智慧。

现在他能背九十二首唐诗，《三字经》、《木兰词》随口而出。很多汉字他都认识了，简单的英语句子和算数的加减他都会说、会算了。

他妈妈都说自从我去了她家，孩子的脾气好多了，身体也比以前好了许多，每个星期都能正常上课了。我去前的那年，一个学期加起来也没上过半个月的课。

现在他的那些治疗过敏的药都不吃了。

佛恩浩荡！感谢师父的慈悲苦度！


#### 有福之人
〖中国大陆来稿〗因为我经常到集市上讲真相，很多人都认识我，有的象亲人一样打招呼，有的急着向我要真相资料，有的问：有没有厚本的（指《九评共产党》、《共产主义的终极目的》等书），有的要护身符，有的要真相卡等，很少遇到不要资料的。

今天我又和往常一样到集市上去讲真相，在一条东西走向的胡同口，看到一个妇女正在她家外面用铁锨铲门外的土。我知道这是一个有缘人，便径直走了过去，从包里拿出一本真相小册子递给她说：“给你一本真相资料，你看看，谁看谁得福报。”她听我一说，高兴地接过去。我又跟她讲大法的真相，她都非常接受，我又劝她三退，她说她只入过队，很痛快的退出了邪党的少先队。我又送了她真相护身符，她高兴的握住我的手连连说：“谢谢你，你快到我家坐坐歇歇吧！”我说：“你还是谢谢我师父吧，不用了，我还得去赶集呢。”她说：“谢谢你师父。”

在谈话中她告诉我，她的孩子们都在外面，个个都有出息，对她也很好。我打量了一下她，在农村是穿的很讲究很好的那种人，不象是个种地的人，我老远看到她的打扮还以为她六十来岁，一问她说：“我都快八十岁了，我的孙子都上大学了。”她又说：“我很相信法轮功说的，我的孩子们也很相信。”我说：“怪不得你家孩子都这么有出息呢。”

看她自豪的样子，我也替她高兴。真是善良人有善报，明真相得厚福呀。
