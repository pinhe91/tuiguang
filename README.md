###  念九字真言　大姐的癌细胞没有了

文: 中国辽宁大法弟子　一山 

【明慧网二零二二年九月二十三日】我把我姐姐的真实经历写出来，证实大法的超常、神奇，真实不虚！
我大姐今年六十三岁，家住河北省。二零一八年的一天，因肚子不舒服，她去医院做检查。检查结果是：肠子上有癌细胞。她又到北京友谊医院做進一步检查，确诊她得了结肠癌。医生要她做手术。

这突如其来的一切让全家人都懵了！大家都知道癌症的结果是什么。我和妻子（同修）因学大法，身份证里有相关信息，去不了北京，我就给大姐打电话，告诉她：“念‘九字真言’就一定能康复！”同时告诉大姐的儿子帮着他妈一起念。我也告诉全家人：要想让大姐病好，就求大法师父，念“法轮大法好，真善忍好”，帮助大姐早日恢复健康。
以前我给全家人都讲过真相，家人也都认可。我儿子不修大法，但从小就认同大法好。他特意去看他大姑，并给他大姑讲了大法真相，把他大姑感动的都落泪了。

也许到了性命攸关的时候了，此时的大姐非常听话，从此每天不停的念“法轮大法好”、“真善忍好”。我三姐和弟弟去医院看她，回来跟我说大姐除了吃饭睡觉外，一有时间就念“法轮大法好”。大姐的小姑子去看大姐，问她：“你嘴里叨咕啥呢？”我大姐说：“念佛呢！”她小姑子问：“管用吗？”大姐说：“管用。”

在准备做手术的前一天晚上，我大姐突然感到眩晕，脑袋迷糊，但过了一会就好了。因是在手术前，以前她从来没有这种现象，医生就赶紧给她做了头部检查。结果显示：癌细胞已经扩散到头部了，脑中有三个大瘤子！医生说不能做手术了，即使把肠子上的癌细胞切除了，脑中的瘤子也做不了，只能保守治疗，做做化疗。其实就是根本治不了了。但作为权威医院的医生又不能这样说，说了家人难以接受。

家人得知大姐的癌细胞已转移到脑子里，开始对医院抱有的一些希望已一丝没有了！于是就把大姐带回家了，只是定期到医院做化疗。

做化疗的时候我大姐并不怎么难受，也没象别人那样脱发和恶心，更没耽误吃也没耽误睡，只是皮肤有些发黑。

在这期间我坚持给大姐打电话，叮嘱她一定要相信大法师父、相信大法，要坚持念“法轮大法好，真善忍好”这九字真言。

到大姐做第三次化疗时，检查身体显示：肠子上的癌细胞没有了，头部的大瘤子已全部钙化。医生们都觉的太不可思议！因为他们心里都明白化疗根本不起啥大作用。况且大姐刚做了三次化疗，癌细胞就全没了？这根本不可能！

可事实就是如此。医生只好说：“好吧，接着做化疗吧。”

我外甥没听医生的，做完第四次化疗后就再也不带他妈去做化疗了。

现在我大姐的身体非常健康，做过检查，什么问题都没有了。她非常开心，无比感谢大法师父的救命之恩！

感谢慈悲伟大的师父！师父不但救了我姐姐的命，还为我们节省了很多钱。因为做这种手术最少也得十万，甚至更多，现在我姐姐没遭那个大罪就神奇般的好了！

这是发生在我的亲人身上的真实故事。写出此文，是要告诉大家：法轮大法好！法轮大法是正法！法轮大法是超常的，大法师父是来救度众生的。谁真正相信大法好，祛病健身就有奇效！

人啊，快醒来吧，千万不要听信邪党的谎言！

感恩师尊的慈悲救度！弟子只有修炼精進再精進，兑现誓约，不负大法弟子的使命，救度更多众生，以报师恩！

弟子叩谢师尊！
