### 诚念“法轮大法好”的奇迹

文: 湖北大法弟子 

【明慧网二零二二年七月八日】我今年七十三岁，一九九五年底开始修炼法轮大法。写出修炼路上几个片段，以感恩师父，证实大法，救度更多的世人。

#### 诚念法轮大法好，旧疾不治而愈

在我居住的小区，有一个姓毛的老头，以前我给他讲真相他不听。一次，他生病了，住院治疗也不见明显好转。我去他家里看他，发现他家的客厅里挂着毛魔头的象，我就告诉他，毛魔头在历史上做了哪些坏事。这么坏的人，你还把它的象挂在客厅里，等于是尊敬它，供奉它，它就会害你。我叫他烧掉。开始他不愿意，说：这是我们本家，它会保护我发财。我继续给他讲毛魔头、共产党的真实历史，启发他的善念。慢慢的，他知道毛魔头确实不好，就同意撕掉，当时就把它扯下来，烧掉。

我继续给他讲法轮功真相，告诉他真心诚意念“法轮大法好，真善忍好”九字真言会得福报。他记住了，并天天念这九字真言。后来他的病不治而愈。他遇到我时说：“你说的是真的，我听你的话，病真的好了，还没花一分钱。”

在他的影响下，他的一家人，包括儿子儿媳、孙子共五人，都非常相信诚念“法轮大法好”会得福报。

#### 真心念“法轮大法好”，一切都变顺了

我有一个邻居，是个九十七岁的老奶奶。她有一个儿媳，两人经常闹矛盾，让这个老奶奶非常不顺心。

一次，我遇到老奶奶，她就把自己的苦恼告诉我，我就给她讲真相。她明白真相，愿意每天念“法轮大法好，真善忍好”，心里平静许多，身体越来越硬朗。我也给她儿媳讲了真相，做了三退，也念“法轮大法好，真善忍好”九字真言，她儿媳变了，也善待这位老奶奶了。

有一天这位老奶奶告诉我，她的儿子儿媳只用了五万元买到了一套房子，很高兴。他们全家人心里都非常感谢师父恩赐自己的福份。

真的是真心念“法轮大法好”，一切都变顺了。

#### 诚念“法轮大法好”，起死回生

一个七十五岁的老婆婆，长期卧床不起，只有一口气，为了满足老人的心愿，家人把她送到老家乡下，以待寿终正寝。

我了解了她的情况，就决定去救她。她的女儿坚决反对，不让我去。我就多方打听，才打听到她的住处，就驱车赶到那里。到了那里，我就给她讲真相，她听懂了真相，诚念“法轮大法好，真善忍好”，看到她在变，要吃饭，要喝水，精神也变好了。我问她愿意听师父的讲法录音吗，她非常愿意。我就把师父的讲法录音给她听，她很爱听。她身体好了，就要求家人送自己回到城里，家人看到她康复了，就把她接回了城里的家。

得知濒死的妈妈因念九字真言神奇康复的经历，她女儿一改过去对我的态度，由敌视变为尊敬，由反对变为欢迎。老婆婆全家人见证了“法轮大法好”，对师父非常感激。

#### 儿子静心学法后，腰椎间盘突出症好了

儿子在一个工厂上班，患了腰椎间盘突出症，腰痛的很厉害，不能自由行走，就休假在家中治疗。

我就给儿子讲真相，并引导他学师父的讲法。儿子很愿意听师父的讲法录音，很愿意看师父的大法书，学炼师父的五套动作，他的腰椎间盘突出症很快消失了。

儿子非常相信法轮大法，非常支持我修炼大法。不仅如此，在社区人员、派出所警察骚扰我的时候，儿子极力制止他们行恶。他还用自己工作的便利条件，义务为大法弟子安装维护网络，起到很好的正面作用。他的善得到了很大的福报。他结婚，生子，家庭和睦、幸福、平安。

#### 遇车祸无恙，肇事者得救

一天早上，我出门才发现下大雨，因没带雨伞，就罩上羽绒服帽子在头上遮雨。过马路时，被一辆突然疾驰而来的农用车从后面撞上，我正面扑倒在地。司机急刹车，因害怕不敢下车，就在车里坐着看我。

我慢慢从地上爬起来，一看双手连皮都没破，穿的白色羽绒服也很干净。我主动安慰司机说：“别怕，我是炼法轮功的，有师父看护，不会有事的。”司机是个女人，问我：“你到哪里去？”我说去西河桥。她说：“正好跟我同路，你上车，我带你去。”在车上我们交谈，她问我：“你多大年纪？”我告诉她我七十多岁。她惊讶的说：“啊，你七十多岁？看不出来，你看起来很年轻。”我就跟她讲法轮功真相，劝她三退。她同意退团退队。我又告诉她记住“法轮大法好，真善忍好”九字真言。她说记住了。

回家后，我一点事都没有，继续做我该做的事。
