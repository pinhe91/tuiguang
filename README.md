### 天赐良方　绝处逢生

文: 玉洁 

【明慧网二零二二年七月十二日】瘟疫不仅发生在当今，中国古代也发生过各种瘟疫，也有治不好的绝症，但有些人却能绝处逢生，道理在哪里呢？我们来看看东汉时期神医邳彤救母的故事。

#### 行善之家有余庆

邳彤不仅是汉光武帝刘秀手下的一名开国功臣，还被封为“药王”。他治病救人简直到了废寝忘食的地步，而且难能可贵的是，他治病不分贫富贵贱，全都一视同仁。

当他听说南方瘟疫流行，便离开家乡前往灾区为百姓治病。谁知邳彤离家后，母亲得了重病，百医无效。哥哥邳祝只好送母亲去南方见邳彤，但邳彤亦无良药，只好让哥哥再带母亲回家。一天，哥哥邳祝搀着母亲走在半路上，母亲口渴难忍，可前不着村，后不着店，哪里去讨口水喝？

邳祝将母亲安置在路边，四处寻找。后来走到一片槐树林里，看见一个死人头骨，里面还存有些雨水，还有两条细细的小蛇在水中嬉戏。邳祝轰走小蛇，把水端给母亲说：“母亲，您就闭上眼睛把这点雨水喝下去吧。”母亲实在渴极了，也顾不上干净不干净，闭上眼一饮而尽，觉得心里好受多了。

这天来到一个村寨，母亲又觉饥饿难挨，小村里没有饭铺，邳祝只好上门讨饭。可巧有一家人家刚生了一对双胞胎。这家的婆婆是个瞎子，公公是个拐子，这样的人家竟然一胎添了两个胖小子，自然十分高兴。一听有人讨饭，赶紧把产妇吃剩下的一碗薏米饭和一个鸡蛋给了邳祝。没想到那鸡蛋还是一个双黄蛋。母亲吃了薏米饭和双黄蛋，身上觉得有力气多了，终于回到了家乡。

不久，邳彤托人捎来一封家信，信上写道：“如母亲能喝到二龙戏珠天然水，吃到一胎双子双黄蛋，牛马小姑碾薏米，拐公瞎婆做成的饭，自然病除。这四种药非人力可为，若母亲命大，自然天缘巧合，药到病除。不知母亲一路如何，十分悬念。”邳祝看了信，回想母亲在路上的遭遇，才知道喝的吃的正是弟弟开的方子上的药，赶紧写信告诉了弟弟。

神医邳彤能开出药方，但如果不是上苍的垂怜，怎么可能凑齐这副“非人力可为”的药呢？正是邳彤医德高尚才得到“天赐良药”，才使得母亲绝症痊愈。

从二零二零年至今, 新冠病毒时不时的在中国大陆卷土重来。在这危难时刻，有些人因诚念“法轮大法好，真善忍好”就能好转，“三退”者密接不染疫。很多实例证明，“九字真言”、“三退”就是避疫祛病、逢凶化吉的最简单但却最有效的天赐良方。

#### 密接被传染　没用任何治疗方法神奇康复

二零二二年五月上旬，黑龙江省哈尔滨市疫情非常严重的时候，我一邻居的妹妹小达和确诊人员密接，在哈尔滨市隔离后出现症状，没用任何治疗方法神奇康复。

小达家在哈尔滨市开超市，在今年春天疫情最严重时，有两个顾客去她家买东西，后来确认这两人是染疫人员，而小达成了密接者。小达吓得直哭，当时其他密接人员都是大巴车接走的，小达是被120救护车接走的，小达更害怕了。

到隔离点时，小达就出现了症状，发烧、头痛、咳嗽、浑身疼，但是小达没告诉别人，她想起我以前跟她讲过法轮大法的真相，想起我告诉她念九字真言并“三退”就能躲过危难，她就开始诚心念“法轮大法好，真善忍好”。结果核酸检查是阴性。小达更有信心了，就一直坚持念。她还打电话告诉她三个姐姐都帮助她念九字真言，她姐姐每天都帮着念三十遍，小达还让她姐姐找到我，让我求求大法师父救救她。之后小达每次核酸检测都是阴性，三四天后一切症状都没有了，不发烧也不头痛了，隔离十四天回家。没用任何治疗方法，神奇康复。

现在小达生意正常，小达和她的姐姐们都知道她是靠念“法轮大法好，真善忍好”九字真言帮她度过了劫难，都知道大法太神奇了。

#### 兄弟染疫全家隔离，念九字真言得救

肖爹八十岁，湖北某乡人，他有五个儿女，都在外工作，已是儿孙满堂。

二零二零年过年时，因中共隐瞒疫情，很多人被蒙在鼓里，肖爹的儿女都回家过年了，一共回来二十多人，其中大儿子从武汉回来。不久疫情越来越凶猛。我女儿跟肖爹女儿肖缘认识，肖缘告诉我女儿一个不好的消息，说前些天他大哥染上武汉肺炎，转到县医院去了，她弟弟也染上了，直接给送去了县医院。一家二人染疫，全家人天天在一起吃饭、密切接触，都感到恐慌。现在家里有二十人被送去镇上的一个宾馆隔离。

我女儿听后立即给我打电话，让我赶快想办法求法轮大法师父救他们全家。我告诉女儿稳住心，快快叫他们一家人都念：“法轮大法好，真善忍好。”这九字真言只要真心诚念，威力无比，就能得到大法师父的保护。

女儿将我的话告诉了肖缘，并让她转告住院的哥哥和弟弟叫他们全家都念，念出声和默念都行。几天后，女儿打电话得知，肖缘的哥哥和弟弟康复出院，只有弟媳妇还在隔离，其他人都回家了。肖缘在电话里不停地说：“谢谢你妈妈告诉我们救命的大法，救了我们全家的命。”

《黄帝内经》中说：“正气存内，邪不可干”。法轮大法是高德佛家大法，诚心诵念九字真言或“三退”， 会与宇宙中的正能量产生共振，从而增强体内正气，抵御病毒。

#### 瘟疫不侵的大家族

二零二二年三月份，新冠病毒肆虐我地。官宣阳性病例以每天四位数的速度上升，百姓惊呼超过当年的武汉。

在当初武汉疫情爆发前，我已经给整个家族几十口人都做了“三退”，在这次疫情爆发中，几十口人的家族分散居住在这个城市的各个小区，有的居住在高风险区，他们很多都是志愿者，他们有的是市委、市政府的干部，有的在公安部门工作，有的在检察院部门工作，有的在宣传部门工作。他们都在疫情的一线做防疫工作，直接接触阳性患者，可是，我们家族的几十口人都没有一个染疫的。

疫情爆发中，这个城市小到几人的店铺、大到几千人、上万人的工厂都因疫情而关门停产。我们家族中，有一个是保供企业（疫情时供应食品、物品的企业）的老板，手下有几百人的员工，他手下的几个中层干部都通过我做了“三退”，有些员工也通过他们的家人（修炼大法的家人）做了“三退”。整个疫情爆发期间，企业周边的大大小小的店铺都因出现阳性病例而关门、歇店，员工全部被拉走、隔离，有的因为一个店铺出现阳性病例，影响到附近小区居民都被拉走、隔离。可是，直到我地区官宣全部清零时，这个几百人的保供企业竟然没有一个染疫的，疫情期间都正常生产。

疫情结束后，家族人在一起聚会，他们都感叹：疫情如此严重，家族人却没有一个“小阳人”，“三退”真的保平安。

新冠（中共）病毒是针对那些相信中共并与之亲近的人而来，因而“三退”将是最佳的避疫方法。

#### 瘟疫来时 传真言派发《九评》

香港学员杨女士的家临近中港边境，中共病毒来临之际悟到救人急，决定放下顾虑，在家门外挂起“瘟疫面前保命良方 诚念：法轮大法好 真善忍好”的大横幅，并获得先生支持。自此，她的村外神奇地长出一片芒草原，仙境般的美景吸引了大量香港人前来观赏。她家门口的救命良方，也变成了旅游景点。有村民说：“‘法轮大法好，真善忍好’保护我们村在严重的疫情下没有一个确诊。”

一次,杨女士派发《九评》、《共产主义的终极目的》时，一位中年妇人气汹汹地地当场报警，一会儿便来了十多个警察。她想：不就是身边有几百本《九评》书吗？这书讲错话了吗？没有。于是她不再害怕，热情地与警察打招呼，祝福他们在疫情中记住“法轮大法好，真善忍好”，一定可以逢凶化吉，保平安。警察赶忙说谢谢。

二十多年来，法轮功学员不顾风吹日晒、严寒酷暑，冒着被抓、被非法关押、判刑的风险，只为了让您明白大法真相，请您珍惜这份善缘，这是您能渡过末劫大难的希望。
