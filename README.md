### 你们是红朝最大的受害者

文: 台湾夏祷 

【明慧网二零二二年八月二日】（称呼省略）
这些年我们写给你的信很多了，这或许是我们写给你的最后一封信。你也看出来了，一切已到了最后的临界点——红船将沉。

郑州、深圳银行挤兑，明眼人知道，这是经济崩溃的导火索，是最终将会要了共产党命的事。你记得，当年苏联垮台也是由于经济崩溃。多年来，中共的领导们一直在为这最后的跳船做准备，你准备好了吗？


在红色中国有太多身不由己的事，而公检法司或许是最身不由己的一群人，也是被害最深的群体。

善良的警察知道：自己是红朝最大的受害者。唯有切断和这贪婪的外来政权罪恶的牵系，唯有远离这手上沾血的政权，才能得到平安。我们给你写信，无非是希望你在这最后的时刻保护自己，得到平安。

最后了，我们一定得让你知道：即将来到的危险是无比真实的。你不快行动，保护自己，面临的痛苦和悔恨将是难以承受的。

#### 善与恶的鸿沟

在这红潮落、红船沉的最后时刻，我们得快快醒来，把一切看清。

“我党”？是苏联移植入中国的，不是祖先留给我们的。我们的祖先缔造了夏商周、唐宋明悠悠五千文明，留给我们敦厚的传统、忠孝节义，告诉我们如何做一个人。你看唐诗宋词，看那些山水画，先哲的智慧，再看红朝的样板戏、今天红彤彤的党文化，就明白两者之间巨大的断层。这不是文化的断层，而是善与恶两种力量之间的鸿沟。

在这鸿沟后是一个根本的差异：那就是对于神灵的态度。你知道，我们的祖先是敬天信神的。邪党建政后做的第一件事就是思想改造，把无神论、进化论灌入古老民族的脑中。直到今天，许多人以为作恶不会受罚，无法无天，无所畏惧。江泽民密令对法轮功“打死白死，打死算自杀”，叫你们公检法司做刽子手，把你们带入了万劫深渊。

七十多年来，邪党把“恶”笼罩在这块国土上，毁坏中华民族。

可我们深知，追随邪党犯罪的不是真正的你。是被谎言欺骗，被国家机器制造的“假”与“恶”污染，忘了善恶有报的你。忘了自己那颗善良的心的你。我们知道真正的你是谁，我们知道你被假恶斗侵蚀的外表下那颗柔软的心。你的善良没有变，一直在那里。

我们还知道，你的生命来源高贵。今天的你深陷危险，可你真正的生命是高贵的，而且是何等的宝贵！我们看到警察为法轮功修炼人的遭遇而流泪，那真诚的心象金子一样。我们怎么能让你们被这外来的“恶”的政权绑架，落入深渊？

#### 灵魂确实存在

二零一九年，一千位有造就的科学家对进化论提出质疑；同时，这些年来，达尔文进化论的一些考古证据被证实是学者所伪造的。这是科学界最大的丑闻。进化论早已破产。

新世纪之交，现代人视为迷信的灵魂跃上新书架，成为热门课题。科学家以临床科学证实了灵魂确实存在，濒死经验NDE更开启了人们对生死的全新认识。许多不同民族的人死而复生，说出了这不可思议的灵魂之旅。透过这神奇而普遍的经验，人们不得不承认灵魂真实存在，不随死亡而消失。

唐山大地震的幸存者也被做过濒死研究，他们的经历和国外的研究发现十分相近。科学家开始和神学家联手，探索人类生命的真谛。

#### 怎么能不告诉你？

为什么说公检法司是比法轮功修炼人更深的受害者？如果灵魂不灭，你想，这意味着什么？如果进化论已破产，又意味着什么？局势诡谲，人世间的惩罚说到就到，看那些落马的高官就知道。然而你要担心的，是永远跟随你，如影随形的老天的惩罚。这才是我们今天想跟你说的，你真正要畏惧的东西。

善恶有报的古训已在今天的中国大量应验。它已在遭报的两万公检法司人员身上应验。你的同行遭雷击的事你聼说了吧？天打雷劈，这可是古代的天罚，却发生在今天的中国。河北省赞皇县纪检委常委滑海英儿子死后灵魂附体，气急败坏地要父亲停手迫害法轮功的故事你聼过吧？这可是上级下来调查过的真人真事。

还有一件事。到这节骨眼了，为了你的未来，这最重要的事情，我们怎么能不告诉你？

早在希腊，柏拉图留下一个著名的伊尔寓言：士兵伊尔死而复生，判官命他回到人间，告诉世人行恶者在阴司受苦，以警世人。而今天，二零二一年，明慧网刊登一篇《我到地狱走一遭的亲身经历》，一位退伍老兵述说自己濒死时在阴曹看见无数公检法司人员在地狱遭受各种酷刑的惨烈景象，一切历历在目，叫人出一身冷汗。

看了这篇文章，心情沉重，不得不提笔给你写信。怎么忍心让你承受那样的极刑？那是比法轮功修炼人所承受的恐怖千百倍的刑罚，那是来自上天，没有止境的永刑。我们深深为追随邪党犯下罪行的公检法司人员忧虑。

#### 上哪去见马克思？

最后想跟你说个事。在中共话语，“去见马克思”成了“仙逝”的代语。那么马克思在哪儿跟咱们大有关系。

在前南斯拉夫共和国黑山境内一座教堂壁画上，画着马克思、恩格斯在地狱受刑的苦状。这不是画家的想象。《共产主义宣言》中明白道出了共产党摧毁人类文明，摧毁家庭，伦理道德的野心。十八岁时，马克思从一名基督徒转而信奉撒旦，诅咒人类。他自己说过：“我这曾经信仰上帝的灵魂，现已注定要下地狱”。（温布兰特，《马克思与撒旦》）

入了党团队的人，曾宣誓把生命献给党，就是与魔鬼签下了誓约，从此被邪灵握在掌心，额头上打上印记，那么死后是要去地狱见马克思的。太多故事告诉我们，誓言是要兑现的。在这最后的时刻，你要快快解脱这与魔鬼的誓约，退出党团队，不受马克思捆绑。这样，你的生命，你永生的灵魂才能自由。

银行挤兑、外资大批撤离，失业率飙高，经济危机的导火索已引燃，天下没有不倒的江山，你的领导上司都在偷偷退党、跳船自保。

天边的惊雷已炸响，在这最后的时刻，衷心祝愿你寻回真正的自己，永保生命的平安！
