### 高喊法轮大法好　瞬间化险为夷

【明慧网二零二二年六月九日】

#### 高喊法轮大法好，瞬间化险为夷

〖中国大陆来稿〗几年前的一天，我儿子、儿媳开车去市里办事，回来的路上，轿车不知何因，一下子出了状况，当时既没有提速，也没有刹车，轿车突然间尖叫着在马路上横向着划圈。这时后面正好过来一辆轿车，眼看就撞到一起的瞬间，我儿子突然高喊：“法轮大法好！”车就停下来了，后面的轿车可能也采取了紧急应对，两辆车只是轻微擦伤。

我儿子下车后，看了看双方的车子都没什么大碍，只是轻轻撞坏一层皮，我儿子和对方说：“好险啊，真是不幸中之万幸，好在都没什么大毛病，咱们谁也别给谁找麻烦了，我们各自修修自己的车好了。”对方同意了我儿子的提议，没说什么也就走了。

儿子儿媳回家后，感慨的说起此事，由衷的感激师父的救命之恩。后来儿子花了三百多元钱就把车修好了。

#### 明白真相的人见证大法神奇

〖中国大陆来稿〗去年的一个冬天，我和同修到集市上讲真相，到一个中年男子车旁停住，这人是个卖菜的，车旁围了四、五个人在挑菜，我们稍等了一会，就开始跟他搭话，问他白菜多少钱一斤，他很客气地回答。紧接着我说：你们农民挣个钱很不容易，起早摸黑，像你这个岁数的人都是家里的顶梁柱，当前瘟疫不断，愿你天天平安，远离瘟疫。诚念：法轮大法好，真善忍好，就会得到神佛的护佑。

中年人马上高兴地说：我都念了六年了，没出过一次事故。我每天一拿车钥匙就念几遍，我的菜比谁卖的都快。我看到他高兴的样子，小声说着：谢谢师父！谢谢师父！最后我们离开时送了他一本真相资料，他双手接过去，赶快放到车里。

过了两个集后，我和同修又来集市。那个中年男子看到我们迫不及待的讲他在上一个集，回家的路上，汽车突然灭火了，离家还远，赶紧打一个维修电话，回答说：等半个小时，这时想起求大法师父，说大法师父救救我，大法师父救救我吧。不过两分钟，一试车好了，赶紧给维修部回了电话，不用来了。看到他美滋滋的样子，我赶快告诉他：你谢谢师父。他说是谢谢师父，连说了好几遍。为了不影响他生意，我们就离开了。

#### 我家的锅一周烧不坏

〖中国大陆来稿〗那是二零一二年的事了。我在儿子家看孙子，都是周六上午回自己家。有一次星期天下午去儿子家，走的有点急，炉子忘关了，等周六回来时，听见炉子那边咔嚓咔嚓的响。赶紧跑進厨房一看，哎呀，炉子怎么没关火？一看锅一点都没烧坏，完好无损，神奇吧，我家的锅一周烧不坏。谢谢慈悲伟大的师尊，没有您的保护，后果不堪设想。
