【明慧网二零二二年八月二十八日】

#### 世人明真相后传播真相

〖中国大陆来稿〗前两年我在讲真相时遇到一个六十多岁的熟人，她明真相后，接过资料和护身符并做了“三退”（退出中共的党、团、队组织），她说：“我很相信修炼的事，前些年我炼过张三丰的那种功。”我告诉她：“诚心敬念‘法轮大法好，真善忍好’就能逢凶化吉。”她说：“我有高血压和糖尿病，我回去好好念念。”
过一段时间，她看见我时就惊喜的说：“自上次我回去后，每天都念几次‘法轮大法好，真善忍好’，我的血糖正常了！”她让我多给她几个护身符，她要拿去发给她的亲朋好友。过后又找我要了两次护身符。

又过了一段时间，她就对我说：“我很想炼法轮功，就是我的事太多了，还得在单位煮十多个人的饭。我拿一百元钱做真相资料吧。”我看她是真心，就接过了她给的钱转给了资料点。过后见面，我们都要聊几句。

我家出门不远有座新修的桥，桥的两边种了花，有条凳，经常有人在桥上耍，我在桥上也救了一些有缘人。我除了上午出门讲真相外，每天吃晚饭后也要出去找有缘人。
一天看见两个中学生小伙子，我说：“小伙子，你们好，我是修炼人，我希望你们平安，现在给你们每人一个护身符保平安，瘟疫来了远离你们，各种大灾难也会远离你们。”听我这么说他俩都退出了各自加入的团、队组织。我说你们把我讲的真相告诉你们的亲朋好友，给他们看看这个护身符，他们如果相信，他们也能得救，你们也功德无量。他俩说好。

第二天我有事，去桥上讲真相比平时晚了半小时，又碰到那两个中学生。他俩说他们也是刚到。我一看旁边还来了一个小伙子，他说他是来做三退的。

又过了几天他们见着我就说：我们有个同学在外地，他也要三退，并用视频和那个同学连接上，让我和他对话。对方说他要做三退。

我正和他们几个讲着话，一个骑自行车的小伙子路过，他们马上喊他过来。我给了这个小伙子一个护身符，一讲真相他就相信并做了“三退”。

前两年，我曾给一个清洁工大姐讲真相，我一说，她就相信，并退出了加入过的邪党组织。之后我俩经常见面，一见面她都会和我打招呼。一天她看见我就说：“大姐，你给我写几个人名字，帮我家里的人都退了。”我说，那你得让你家里人明白为什么要退，要自己答应才有效。她说：“那好办。”

过了几天她给了我一张六个人的“三退”名单，还有一个是退出共青团，说是她孙女自己写的。

#### “新闻联播全是骗人的！”

〖中国山东来稿〗随着师父正法進程的快速推進，大量邪恶被销毁，明白真相的世人越来越多，这是师父慈悲救度的结果！在最后的时间里让我们更加精進，抓紧时间讲真相救下更多的人。

在最近这段时间里，我明显感到大部分世人越来越清醒了，能做出自己的判断了。

仅举几例：

一天，碰到一位七十岁左右的男士，我给他讲了大法的洪传，邪党对我们的迫害，让他记住“法轮大法好，真善忍好”，退出邪党组织保平安。他说：“你给我退了吧，我是党员。我早就不看新闻联播了，全是骗人的，就连天气预报都能造假。”

我为他的清醒而高兴。邪党越来越不得民心，世人也看清了它的真面目，知道它的本质就是邪的。

前几天我遇到一位六十岁左右的男士。我上前搭话，告诉他瘟疫仍在横行，记住“法轮大法好，真善忍好”会有神佛保平安。看他很接受，就又跟他讲了法轮功真相，讲了邪党这么多年对我们修炼人的迫害，告诉他“天灭中共，三退保命”。

他听后也说：“我早就不信这个邪党了，早就不看电视新闻了，说的全是骗咱老百姓的。我只看看天气预报。”他说曾加入过共青团和少先队。我用化名给他退了出来，他说谢谢我。我说：“不要谢我，谢大法师父吧，是大法师父让我们讲真相救人的。”他说：“那就谢谢你们的师父。”

这类事例很多。当然，国内没得救的众生还在等着我们去救，师父留给我们这段时间就是让我们去救人的，特别是农村人，好多都没听到过真相，他们纯朴、善良，这一生吃苦、受累，每天面向黄土背朝天苦干，他们正在等待大法的救度。我们没有理由懈怠。希望没有出来面对面讲真相的同修能放下怕心，走出来讲真相救人。
