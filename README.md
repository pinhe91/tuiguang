### 退休工人主动退党

文: 四川大法弟子 

【明慧网二零二二年十月二十三日】我修炼法轮大法二十多年了，一直按师父的要求做着三件事，有这么几件记忆深刻的事，激励着我坚定的走在讲真相救人的路上。

#### 退休工人主动退党

二零一六年的一天，我上街买菜，看见有俩夫妻的摊位上有几种菜。我买好菜，付钱时，大爷小声的问我：“老孃，你认识炼法轮功的吗？”我心里一惊，脱口而出：“我就是炼法轮功的，你有什么事吗？”

大爷看了他妻子一眼，靠近我说：“我是某煤矿退休的，听说炼法轮功的可以帮忙退党，是吗？”我说：“大爷，你真是个好人，我正想给你讲真相，没想到你主动问上门来了。”
我问了他的简单情况和姓名，并告诉他怎样退党，为什么要退党？大爷高兴地说：“太好了，共产党确实太邪，你就帮我退了，还需要做什么吗？”

我拿一张护身符，告诉他，全家人都念九字真言，还要给亲朋好友讲真相三退的事，要他们心里装着九字真言，才能保平安健康。他再三谢谢我，我说要谢的是法轮功师父。

#### 主动要真相护身符的小伙子

一次买梨时，我问小伙子：“你上学入过团队吗？入团队时，要举手宣誓，要把一生献给共产党的，现在共产党这么腐败、邪恶，要退出才能保平安，用真名、小名、代名都可以。”他说：“高中没毕业，因家庭困难，没有读了，团、队都入了，就用真名退。”“好的，我给你办了。你是个好小伙子，你会有美好的未来。”

看我要走了，他着急说：“你有装在身上的那个小块的东西吗？”我明白了：“是真相护身符吗？”他点头。“我带来的已经发完了，你若要，你在这儿等着，我去拿。”

半个钟头后，我给他，他笑嘻嘻说：“谢谢老孃，对不起让你跑路了。”我说：“小伙子，我多给了几张，你可以给同学、亲朋好友，告诉他们记住法轮大法好，真善忍好，平安幸福。”“好的，记住了。谢谢你。”

#### “我们全家都三退了”

我有个朋友小赵，家在农村，高中毕业后，就有了小家庭，当上了生产队长，入了（邪）党。她丈夫是团员，在自家门前开了个小卖部，卖杂货，俩口做起生意。

一次，他们请我到她家去玩。我给她们讲真相，劝三退，他俩都没有退。

几年后，小赵发病了，检查是脑瘤，叫我去她家，全家人哭成一团。我安慰后，说：你去动手术吧，同时再劝他们要三退。我给了真相资料和护身符，叫他们好好地看真相资料，念九字真言，这才是唯一得救的希望。只有相信法轮大法好，才能保平安。小赵立即同意三退了，她丈夫也退了团队。

这些年，小赵身体恢复较好。经过那次发病，神奇的好了之后，她的同学、亲朋好友都知道了法轮大法好。

小赵有两个儿子，大儿子是大专毕业，在一家企业上班。小儿大学毕业，是公务员。我给小赵的儿子们讲真相时，两个儿子都说，知道了，请我帮他们三退，是他们两兄弟的心愿。小赵高兴地说：“我们全家都三退了。”
