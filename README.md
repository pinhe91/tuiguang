### 山东招远市国保大队警察王海波恶行录

【明慧网二零二二年十月十四日】（明慧网通讯员山东报道）山东省招远市国保大队警察王海波积极参与迫害法轮功，是招远国保大队臭名昭著的一名打手。
王海波，男，一九七一年十月出生，住址：山东省招远市建筑新村11号楼；其妻闫春红的单位是招远市国大，其子王超的单位是招远市龙大。

以下是王海波对法轮功学员实施迫害的部份罪行。
◎二零零七年十一月一日，招远国保大队王玉成、王海波和杨某等六个警察闯入法轮功学员王均光家，先抄家抢劫，随后把王均光夫妻绑架到招远洗脑班分别关押在小黑屋里一个星期。

十一月八日，王海波、李建光等四人把王均光逼到二楼一行刑室，李建光非法审讯他，李先用硬帽盖左右开弓打王均光的脸，王海波用紧攥的拳头从王均光的右下颚向上猛捣，连续击打，王均光被打得口吐鲜血，王海波仍不停拳击；午饭铃响了，王海波继续拳击。王均光右腮内肉被牙齿咬烂，鲜血不停地流出，几个牙都被打活动了，疼得他吃不了饭，睡不着觉，连续七天不能进食。

几天后，王海波等几个恶徒又把王均光逼到二楼审讯室，恐吓他不配合就打死扔到矿井里，因他们当时有事，又把王均光押走。王海波把王均光背铐在床头上，蹲不下，坐不起，一直铐了一上午。

◎二零零九年黄历二月二十八日，金岭镇北冯家村法轮功学员于红被绑架到招远洗脑班，王海波一把抓住她的头发把她摔倒在地，王又和另一个警察对于红用脚猛踢，嘴里骂着不堪入耳的脏话，直到他们踢打累了。王海波又抓起苍蝇拍朝于红的头乱抽乱打，还专门抽于红的嘴，于红嘴被抽肿，右胳膊被打得抖动的厉害，王海波就狠踩她的右胳膊，碾踩她的右手，还骂于红装的。于红头发被揪下一堆落在地上，心身伤害很大。

![image](https://user-images.githubusercontent.com/79625284/195809816-91991d5a-b51f-47ab-8781-d38ecc49a1d4.png)

酷刑演示：毒打

◎二零零九年七月二十八日，齐山镇下林庄村法轮功学员王美菊在去姐姐家的路上被绑架到招远洗脑班。王海波、李汉超酷刑毒打她，王美菊被打得昏死了过去，后半夜才醒了过来，发现自己嘴里被黑血块堵住，满身苍蝇蚊子，她浑身疼痛，头部麻木，恶心呕吐，睁不开眼，说不出话，全身瘫软，大小便不能自理。之后又被王海波多次毒打，被打掉四颗牙齿，一个月嘴还张不开，被野蛮灌食，8月27日被构陷劳教一年半。

◎二零零九年九月，法轮功学员杨松美被绑架到招远洗脑班。一进屋就被恶徒宋少昌踢倒，当时腰被踢坏，几天不能动，之后被逼到二楼。恶警王海波把她双手铐在铁椅子上，双脚铐在铁椅子腿上，王海波恶毒的说：今天专门修理你。王海波和几个恶徒先用条子狠狠的抽打，后换成粗木棍抽，最后又用螺纹钢棍打，还专门打头、砍脖子。杨松美被打得浑身青紫，不能动弹。

![image](https://user-images.githubusercontent.com/79625284/195809967-42ff51d4-cc9e-41b0-9ae4-7aeba61124fb.png)

酷刑演示：毒打

◎二零一五年七月，齐山镇下林庄村七十六岁的男法轮功学员吴克林被绑架到齐山镇派出所，王海波一脚把吴克林踢得没气了，人倒地昏死，王海波不以为然，毫不收敛，还要对被一起被绑架的王美菊行凶，因吴克林生死不明，在场的人怕出人命，挡住了王海波，使他对王美菊行凶未成。吴克林三十五天回家后，发现被王海波脚踢的大腿根处还有一块巴掌大小的青紫淤血。

◎二零一八年四月下旬，王和波和另一国保警察闯入毕郭镇沙沟村迟瑞美家中，恐吓迟瑞美。威逼她说出被逼已流离失所多年的丈夫王华洋的下落。几年来，国保大队王海波等人多次去迟瑞美家骚扰，妄图抓捕王华洋。

◎二零一八年九月二十五日，法轮功学员战春华在公交车站点讲真相救人，被着便装的恶警王海波等人绑架到国保大队，王海波先狠狠的扇了战春华几个耳光子，又用脚使劲的踢她的腿，之后把她一脚踹倒。战春华腿被踢的痛了一个多月。后战春华被非法关押在烟台看守所和招远洗脑班七十二天。
