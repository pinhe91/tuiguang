### 辽宁村民举报法轮功学员遭恶报死亡并殃及家人

【明慧网二零二二年五月二十日】（明慧网通讯员辽宁报道）辽宁省葫芦岛市建昌县大屯乡头道沟村韩锡国，二零一九年举报法轮功学员，几个月之后遭恶报死亡，并殃及家人，其妻也于二零二二年患病离世。
二零一九年三月二十八日下午，建昌县大屯乡头道沟村村民韩锡国恶意举报了去本村发真相资料的七名法轮功学员，造成沈文伶被非法判刑一年六个月，送进辽宁省女子监狱；另有四名法轮功学员王玉珊，曹玉娥、胡继霞、齐向前被非法判处拘役六个月缓刑一年，并处罚金一万元，其中齐向前被非法开除公职，胡继霞现在每月少发300元工资。王玉珊，曹玉娥都被降一级工资。陈奎修被非法关押二十天；另有一名法轮功学员高淑芬被非法抄家，被迫流离失所至今，并被非法设为“网逃”。

举报法轮功学员几个月之后，韩锡国遭报患病死亡，其妻也于三年后得病死亡。

法轮大法是佛法修炼，法轮功学员向世人讲真相是在救被中共邪党谎言毒害的人。特别是在当前瘟疫盛行的时候，阻止法轮功学员救人、迫害法轮功学员，就是与佛法为敌、断送自己未来的行为，并且会殃及家人。这样的例子已经很多了，希望所有的父老乡亲都能明真相，善待大法得福报。
