# 【庆祝513】最幸福的儿媳妇]

文: 大陆大法弟子　昭日 

【明慧网二零二一年五月二十五日】婆媳相处，是千古难题。不争不斗，已属难得。而我却是非常的幸运，因为我婆婆修炼法轮大法，所以我成了世界上最幸福的儿媳。

### 一、修大法　结圣缘

一九九九年三月，我开始修炼法轮大法。七月二十日，中共开始迫害法轮功。电视台整天播放污蔑法轮大法的新闻，不需要很高的智商，就知道全是撒谎和造谣。经过严肃的思考，我决定要为法轮大法说句公道话。

虽然我没有经历过文革，但我知道，一旦站出来维护法轮大法将意味着什么。面对铺天盖地的镇压声势，我也怕。那时，单位要人人过关，逼着表态，写不炼功保证都不够，还要写揭批，没完没了。但我决心已下，挥笔写下“法轮大法，千古奇冤”八个大字，就走出去维护大法了。

一晃，十多年过去了。我为法轮功鸣冤，被累计非法关押了五年，失去了一切，婚姻也解体了。由于没有经历过真正的心性修炼，在被迫害中，我走了极端，身心受到了很大的伤害。我自己也纳闷：为什么我就修不出慈悲、祥和的心态呢？

一天，我见到一位同修，他母亲也是大法弟子。这位阿姨总是乐呵呵的，自从看见她，我第一次体会到了什么是祥和，我浑身暖融融的。此前，因为遭受迫害，我和同修们总是被冲散，真正接触的老同修并不多。我身边缺乏一个安定实修的环境，身体也很差，脸色总是蜡黄的。

阿姨对我的境况深为关心。她的儿子是单身，与我的性格也互补，经过慎重考虑，大家同意组成家庭，开创一个新的修炼环境。就这样，阿姨成了我的婆婆。一时间，亲友当中都传遍了：因为修炼法轮大法，这两家结了好姻缘。

### 二、大法纯善　治愈创伤

我是父母眼中的乖乖女，高材生，有着稳定的工作，光明的前途。只因为我说了一句真话，就被停职、停工资、软禁、被绑架、非法劳教、非法判刑、被迫离婚……母亲为此哭干了眼泪，父亲在高压中性格变的怪僻。一开始，他们被中共的谎言蒙蔽，对大法很不理解，很激烈的反对我修炼。可是，我坚定的选择，又让他们明白了，暴力不能使人放弃信仰。十多年，我的父母承受着巨大的痛苦。

我从黑窝回来后，性格渐渐变的敏感、易怒，有时甚至象个火药桶，一点就着。我给爸爸劝三退（退出中共的党、团、队组织）时，他不但不退，还维护邪党。我生气，时常和他争吵。我自己也知道不对，就是控制不住。妈妈一听我和爸爸吵架，也跟着神经质的喊天喊地。一家人整日愁眉苦脸。

和未来的亲家见面后，我父母感受到了温暖，看到有人能真正关心自己的女儿，他们的内心是接受的、也是感动的。尽管如此，两家人刚搬到一起时，还是挺难的。

婆婆准备在自己的房间里把师尊的法像供起来。我妈妈一开始反对，后来竟象疯了一样大哭大闹。婆婆惊呆了：十几年来，自己和儿子堂堂正正的修大法，婆婆的老伴健在时，也是百分之百的支持，单位也尽可能的保护大法弟子，婆婆什么时候见过这种架势啊？她感到了眼下这个环境的艰难，但她什么也没说，开始无微不至的关心和照顾这个家。

结婚后，我和丈夫出去工作。婆婆就象老姐姐一样，带着我父母从新开始生活。她非常勤快、手脚麻利、厨艺超级棒，煎炸炒炖，色香味全；面点小食，样样精通。逢年过节，她滚青团、包春卷、做生煎、制调料、卤肉蛋……美味不输专业厨师。我爸妈从未见过这等能人，惊叹不已。

三位老人乐呵呵的一块过日子，原来冷冷清清的家变的热闹起来，邻居也接二连三的沾到口福。婆婆整日笑眯眯的，好象有一股高兴的泉水不停的从她的心里往外冒。只要看到她的脸，心情就会开朗起来。婆婆无论在哪儿住，不超过三个月，邻里关系就处的非常溶洽。

婆婆也挺爱聊天，她柔声细语、家里家外、轶闻趣事，绘声绘色、无所不谈。讲着讲着，就融汇成修炼人的见识和经历。婆婆把我爸妈讲明白了，我父母知道了法轮大法好，大法弟子善良；知道了中共为了镇压法轮功，编造了弥天大谎；明白了那些跟着中共跑的人有多可悲；也知道了世界上还有很多帮助大法弟子的好人等等。

渐渐的，我父母的态度有了一百八十度的大转弯。我母亲很后悔自己过去激烈的反对法轮功。在很多场合，她都真诚的忏悔，责怪自己听信了中共的谎言。她上网做了三退、写了“郑重声明”。面对前来骚扰的中共邪党人员，她义正词严，用亲身经历指控江泽民的迫害政策给老百姓带来的灭顶之灾。她甚至勇敢的和我们一起实名控告元凶江泽民。

我父亲也转变了观念，他能够正确认识到江泽民对法轮功的迫害是多么的邪恶。法轮大法是如何为人带来了身心健康。他亲手为婆婆做了漂亮的镜框，把师父的法像端端正正的挂了起来。他还和我妈去小区交涉，要求他们不要来骚扰大法弟子的家庭。

就是这么一个善良的婆婆，把原先笼罩着全家的阴霾一扫而光。我有时在旁边看着轻松欢快的爸爸妈妈，看着老大姐一样的婆婆，心中感慨万千：真修大法原来是这样啊！那水晶般的纯善、那金灿灿的欢乐光芒，好神奇啊！我太幸运了，得遇这样的缘份。

### 三、伟大的法造就的无私修炼人

修炼法轮大法前，婆婆完全是另外一个人。生活的沧桑和磨难，养成了她争强好胜的个性。她幼年幸福，可自从共产党来了，家产被掠夺一空。作为长女，为了帮衬家里，她早早辍学去干活。卖杂货、做佣人、進工厂，干过很多行当。她手脚麻利，勤快聪慧，大家都喜欢她。但别人也很难占到她的便宜。

虽然她精明强干，可四十刚出头，健康就恶化了，不得不提前内退。儿子又不幸罹患罕见的肝病，四处求医，不见好转。拖了几年，变的奄奄一息。一家人心力交瘁，在绝望中挣扎。

一九九六年，儿子修炼了法轮大法。仅仅三个月，病就痊愈了。这天大的好事震撼了所有的亲友、同事。法轮功究竟神奇到什么地步呢？我丈夫回忆道：“当时，《转法轮》还没出版，就是跟着炼动作。由于常年吃药，满嘴都是苦味儿，没一点食欲。炼功才一天，我就想吃饭了。渐渐的，身上有了力气，人一天天好了起来。我体内的毒素不断的被清理出来，手上戴的银戒指都变黑了。三个月后，妈妈让我把面粉从楼下扛上来。以前我上个楼，都三步一喘、五步一挪。结果那一天，我跑上跑下，把三袋五十斤的面粉一口气扛回了家。妈妈抱着我说：‘好了！好了！我儿子好了！’”

儿子痊愈后，婆婆也开始修炼法轮大法。很快，所有的毛病不翼而飞。有一次，婆婆感到浑身难受，想吐。“哇”的一大口鲜血喷到了地上。她赶快到卫生间里，吐了很多。感觉还想吐，干脆拿了一个大脸盆来接着。结果，就这么一口又一口的，鲜血接了大半个脸盆，里面还有象鸡蛋那么大的发黑的血坨坨。

婆婆并不觉的害怕，她浑身发热，头脑清醒。她明白，是师父在给自己清理身体呢。吐完以后，肚子又开始疼，挣扎着去了卫生间，开始拉肚子，拉了许多煤黑色的东西。就这么折腾了整整一天，到晚上了，人瘫在了床上，面如金纸。邻居都说：“她是要死了。”

在全家人的理解和支持下，婆婆没有上医院，就在家听大法师父讲法的录音，听大法音乐《普度》、《济世》。婆婆坚持着，能炼功多久就炼功多久。就这样，慢慢从喝白开水，到开始喝粥，再开始吃饭。半年之后，婆婆完全康复了。从那以后，婆婆好象变了一个人，原来瘦削的脸、瘦削的身材，变成了圆圆的脸、胖胖的身材；原来的愁眉苦脸也变成了笑容满面、白里透红。

摆脱了生命的沉疴，婆婆心里敞亮、松快。精明好强不见了，取而代之的是爽朗欢乐、热情体贴。她发自内心的感恩：“师父把我整个人都换成新的了。”全家人重获新生，沐浴在真、善、忍的佛光之中，真是用语言无法形容的幸福。

一九九九年中共迫害法轮功以后，婆婆和同修们义无反顾的走上了维护大法、证实大法、救度众生之路。她数次去北京、去天安门广场打横幅，高喊：“法轮大法好！还师父清白！”并平安归来；为了保护大法书、保护同修，她想尽了办法，承受了巨大的压力。她和同修们一道，把法轮功真相传遍了本城和外地：贴标语、发真相传单、送真相资料、挂横幅……她走到哪里，讲真相、劝三退就做到哪里，即使面对老伴被迫害离世、自己被非法判刑、儿子流离失所、天各一方，她从未懈怠过一天，十多年如一日的做好大法弟子应该做的事。

有一次，婆婆去发法轮功真相资料。進入居民楼楼道后，她从上往下发。发到三楼的时候，只听四楼的一个男人大喊：“什么人？！给我站住！”紧接着，就听到“咚、咚、咚”沉重的脚步声从楼上跑下来。婆婆急忙下楼，边跑边心里求师父：“求师父把这个人定住！”回头的瞬间，就见那个男子象傻了一样，愣在原地。婆婆就这样走脱了。

还有一次，发完真相资料天已经很晚，为了抄近路回家，婆婆走了一条田间小径。脚下正走的急，突然一滑，掉進了一个大水坑里。当时情况很危险，天黑的伸手不见五指，水一下就没了顶。婆婆不会游泳，人又胖，眼看着就没命了。她立刻心中大叫一声：“师父救我！！”说时迟，那时快，一股力量一下子把她托了起来，婆婆的头就露出了水面。很快，她的脚就踩到了地——到岸了！婆婆一点点的爬了上来。她激动万分，赶快叩谢师父的救命之恩！

修炼法轮大法后，婆婆从一个普通的家庭妇女成长为一位大善大忍的大法弟子。她也救助受难的同修。曾有一位女同修，被中共迫害的瘫痪在床。家人不理解，不管她了。婆婆在不耽误学法、炼功、救人的前提下，承担起照顾同修的责任。每天一早，婆婆就去同修家，为她做饭、煲汤；擦身、洗衣；鼓励她，陪她学法……照料瘫痪病人是很劳累的，婆婆一照顾就是好几年，一直善心的陪护着同修。虽然这位同修最后还是被迫害的离世了，但她的家人终于明白了，大法弟子都是真正的好人。

### 四、永远的妈妈

我太幸运了，婆婆待我如亲生女儿，百般呵护。刚结婚那会儿，我消业非常厉害，好几次躺床上起不来。婆婆一直充满关怀的照看我，为我掖被子、熬粥……在修炼的路上，她更是帮助我走上了正道。

知识份子学大法有一个很大的障碍，就是被自己复杂的思想困住。就拿“向内找”来说，我一直以为向内找就是反思自己，遇事能剖析自己，勇于认错，勇于改進，所以觉得自己修的还行。直到遇见婆婆，才明白什么是真正的向内找。

婆婆没有我的想法多，她的心态很纯净，认为做到是修。她每天三点半起床，炼完功就学法，非常规律，一天神清气爽。遇到矛盾和关难，她不会象我那样苦思冥想、眉头打结，而是很沉静的面对、平和中就把问题解决了。我的丈夫也是这样。

我虽然想法很多，但最基本的几件事做的怎样呢？早晨炼功起不来，双盘也盘不好，学法没有规律，一天总是过的很紧张、心态也不稳。长期陷于相似的关难中，总也过不好，把控起情绪来虎头蛇尾。结婚以后，才知道自己根本不在修炼状态之中。

师尊说：“一个人要想修炼，可不是那么一件容易的事情。我讲了这是一件非常严肃的事情，而且它是超出常人的，比常人中任何事情都要难一些。”[2]

对我来说，修炼真的是比其它所有事情都难、都奥妙。常人的书我没少读，理论知道一大堆。但修炼了大法，我竟然会天天捧着《转法轮》，却没有真正得法。别看所思、所想都是三件事，谁知这思维的本身就是问题，它让我陷在事情当中打转转，而没有找到每一动念背后隐藏的执著，也就很难体会到大法的无边内涵。

以婆婆为镜，我总算明白自己复杂的思想，就是我修炼上最大的障碍。要解决这个问题，我先从最基本的开始——有规律的晨炼、学法、发正念。做到才是修。晨炼对我来说，一直好难。好不容易哪一天四点半起床了，第二天准补觉。我找各种理由，工作忙啊，等等。婆婆没有对我说教，她就是尽一切能力为我减轻家庭负担，让我腾出更多的时间早点忙完工作。

可是人的思想太狡猾，做不好时，为自己开脱不说，还对婆婆产生很大的依赖心。觉的和这么精進的老同修在一起，我又对大法这么坚定，自己也能算一个合格的大法弟子吧。于是即便做不好，我也渐渐的心安理得起来。婆婆看到我是把她作为榜样来依赖，从修炼的角度她明白，这是要让她放下对子女的情。看到我们生活的各个方面基本稳定之后，她就开始隔段时间去外地走走亲朋，一住两、三个月。通过这种方式，我的依赖之心减轻了不少。

两年后，婆婆决定搬回老家，不再和我们住在一起。纵然万般不舍，我们还是尊重她的心愿。依依不舍的告别后，婆婆走了，我们看着她的身影消失在阳光中。

婆婆走后，我竟然能三点半起床了：闹钟一响，爬起来，竟然很轻松。炼完功，一天也没发困，第二天又准点起来，不觉的累。我简直不敢相信，怎么就能起来了呢？开始坚持晨炼以后，我精力充沛多了，打坐逐渐能够入静了，学法也入心了。

师尊说：“静而不思　玄妙可见”[3]。

我再回过头来看自己的思维方式，就能看到过去的起心、动念，全是有为，全是执著。真正的智慧根本就不是自己绞尽脑汁“想”出来的，而是来自于大法中的，是师尊点化给我的，是妙不可言的。

如今，我和丈夫一起，保证每天三点半起床晨炼。一个小时抱轮的炼功音乐发表以后，我们也延长抱轮一小时；清晨学法，保证每天至少学一讲《转法轮》，已经这样坚持三年了。有了好的修炼基础，证实法修炼的路也越走越宽。

婆婆回到老家已经三年了。回首过去，我越发能体会到，作为一位母亲，婆婆为我们付出了最伟大和无私的母爱；作为一名大法修炼者，她向我们展现了在法轮大法中修炼出的纯净与慈悲。我们共同沐浴在师尊的佛恩浩荡之中，一定不辜负师尊的慈悲救度。

[2] 李洪志师父著作：《转法轮》
[3] 李洪志师父诗词：《洪吟》〈道中〉

# 【庆祝513】八旬老翁：大法在迷世中度人

文: 四川大法弟子　

【明慧网二零二一年五月二十四日】我出生在山区的一个县城里，今年八十三岁。清朝末年，我家祖上修了几条街巷，供后人住用；祖上也给县城修了桥、修了路；也建了小学，供给后人免费读书。我家附近有几座庙，我总喜欢到那几个庙里去，因为庙里有千里眼、顺风耳等神仙的像。

### 法轮大法是超常的科学

我是某大学的退休职工。去年夏天，我到学校退休处领取物品。退休处的一位领导认识我，她曾在中共迫害法轮大法的初期到过我家，叫我不要修炼法轮大法，我给她讲过法轮大法好的真相。

我去学校退休处时，她对我说：“某某老师，你已经八十多岁了，可你的头发是黑的，面部白里透红，你要活到一百岁！”我到市场买东西，学校同事碰到我，都说：“你都八十多岁了，还来市场买东西？”

修炼法轮大法之前，我病魔缠身。戴老花眼镜，且有散光；患有十二指肠溃疡，还有慢性萎缩性胃炎。医生说，慢性萎缩性胃炎有变癌症的可能，我的思想包袱就很重。我天天靠吃药缓解这些病症，我吃了几十年的药。

邻居院长说，他愿意为我做外科手术。可是我想：父母给我的身体，我只有爱护。不到非做手术不可时，我就不做。我脚底还长了鸡眼，走路脚痛，经常去街上摆摊人那里，让他给我挖鸡眼，要不就贴鸡眼膏药；到了冬天，我的头上就要戴帽子。

一九九四年的一天，我在书店里看到了《法轮功》这本书。书里讲到了真、善、忍是宇宙的特性，我感到很震惊。于是，我就按照书中法轮大法师父所讲的法理，用真、善、忍要求自己。

没想到，我的胃病彻底断根了。眼睛老花眼、散光不存在了，不用戴眼镜了，书上的小字也能看清楚了。

之后，师父帮我清理了身体。我拉肚子，拉的是脓中带血的东西。从此以后，我吃冰糕等冷食品，胃肠完全正常。我无病一身轻，精力旺盛，也不用戴眼镜了，视力就象年轻时一样。

在炼静功双盘时，我看见脚上的鸡眼象一根根针一样，从脚内往外冒出。从此，鸡眼没有了，脚不痛了，走路轻松。冬天也不用戴帽子了，也不用靠涂万金油来提神了。我的脸色白里透红，没有病的感觉真好。

我妻子是医生，看到我修炼法轮大法无病一身轻，她也按照真、善、忍的标准要求自己，不收红包了。

我的大儿子曾染上吸毒毒瘾，花了几十万元，也没戒掉他的毒瘾。那时，我已经得到了大法宝书《转法轮》。我对儿子说：“命是你自己的。你想要你的命，你就抄《转法轮》。”他就抄《转法轮》，边抄，边全身发热。他的毒瘾就这样戒掉了。

他的儿子即我的大孙子，都长的很大了，还不会说话。带他到重庆医院去看，可是医不好。后来我想，法轮大法的威力大。吃完晚饭，我领大孙子去坐公交车，让他听大法音乐《普度》，他喜欢听。

后来我念“法轮大法好，真善忍好”，大孙子也愿意听。最后，他就自己念。再后来，我念《论语》，他跟着学，跟着背。大孙子说话正常了。法轮大法是高德大法，是超常的科学。

我修炼法轮大法，家人受益了，我也希望大家都能受益。我就在单位播放大法师父的讲法录像、教功录像给大家看，我义务教功，来学大法的人很多。放寒假、暑假，我就去外地，包括我自己的家乡去洪法，让更多的人受益。

### “你们是我们的希望”

一九九九年七月二十日，中共开始疯狂迫害法轮功。二零零零年，我和重庆的大法弟子到北京为法轮大法说句公道话，到中南海上访。我被中南海的警察电击了四十分钟，满屋都是被烧焦的焦肉味。

我心中默念师父的诗词：“生无所求　死不惜留　荡尽妄念　佛不难修”[1]，我就走了出去。一看，自己全身正常，真是神奇！

后来，我单位派人去北京把我们劫回重庆。来抓我们的这些人，坐在一起就是打扑克牌，吵吵闹闹，声音很大，吃零食把地上搞的很脏，被火车的乘务员警告。我们大法弟子所在的车厢整齐、清洁，特别安静，受到列车员的称赞。下车时，有个年轻的工作人员在我面前竖起大拇指，说：“你们是我们的希望。”

从北京回到重庆后，我被非法关押在重庆沙坪坝白鹤林看守所。看守所的一男一女两个年轻警察问我：“你为什么要到北京？”我说：“法轮功师父教我们修炼真、善、忍，我们在按真、善、忍做好人，做更好的人，最后做个超常的人。我原来有十二指肠溃疡、慢性萎缩性胃炎，在医院治了二十多年治不好。修炼大法后，全好了。这多超常啊！”男警察说：“那你的师父是大神，你是小神喽！”

因为我不放弃修炼法轮大法，在中共对大法的迫害中，我被非法关進重庆西山坪劳教所遭受迫害。

### 青年学生背诵大法

在重庆西山坪劳教所，有个大足县的青年吸毒者，是个大学生。他的床铺与我相连，他知道我是大法弟子。有一天，他对我说：“你能不能把法轮功背一段给我听？”我说：“行。”我就背《论语》给他听。

我背完后，他说想记下来。于是，我背一句，他记一句，我全部背完，他全部都记下来了。都记下来之后，他对我说：“大法把我变成了另外一个人。”

一天，单位退休处的党委书记带了一个人来看我。狱警就把我带到门口，跟书记见面。学校书记严肃的对警察说：“你怎么把我们学校的好人关在了这里？”警察无言以对。我退休前，学校评职称，大家争夺激烈。唯有我不争。

时任学校党委书记问我：“你要共产党，还是要法轮功？”我果断的说：“我选择法轮功。”

### 法轮大法的威力真大

师父让我们大法弟子讲真相，救人，这是大法弟子的责任和使命。开始时，我有怕心，我就经常背师父的法：“你有怕　它就抓　念一正　恶就垮　修炼人　装着法　发正念　烂鬼炸　神在世　证实法”[2]。

我先从亲戚朋友入手，积累了一定的经验后，我就利用买东西和外出时讲真相。现在，路上碰到的人、车上遇到的人，我都能给他们讲真相了。我平时注意让自己的心态要稳，要多学法，正念强。讲真相时要心生慈悲，心平气和的讲，效果比较好。

我有个远房亲戚，家住广东省陆河县。一天，这个亲戚来我家，见面后，他有气无力的说：“我已经七十多岁了，身体不好。我家里人已经给我买好了墓地，买了一口棺材，还做好了寿衣。我将不久于人世。”

我先请他听大法音乐《普度》。我问他：“好听吧？”他说：“好听。”我又给他讲了大法真相，他听的很认真，也顺利的做了三退（退出中共的党、团、队组织）。我告诉他要诚心念：“法轮大法好，真善忍好”，他答应了。他再次来我家玩时，他的身体已经变的很好了。

我的一个亲戚是客家人，家在广东省的农村。一天，他见到我兴奋的说：“你给我的这个（大法真相护身符）可真灵！”我问：“怎么个灵法？”他说：“我骑摩托车，在路上不小心连人带车摔到了悬崖下。摩托车摔坏了，我却没有事。”

那是一个多月前，我见到这位亲戚时，给他讲了大法真相。他说：“我什么都没有参加，我不相信这些。”我说：“你天天骑摩托车在外面跑，有危险。我送一个真相护身符给你，保个平安吧！”他还说他不相信这些。我说：“我们是亲戚，我不会害你的。你把护身符放在钱包里，随身带着，不妨碍你干任何事。”他收下了。没想到，法轮大法救了他一条命。从此，他全家四口人都相信了法轮大法好。

我妻子是医院放疗科的医生。她有个同事是技术员，我叫她郑三妹。有一年，我碰到郑三妹，她面呈黑色。我问她：“怎么了？”她说自己得了肺癌。我告诉她真心退出中共的党、团、队组织，诚心念“法轮大法好，真善忍好”，病就会好。生死关头，她马上就接受了。

过了几个月，我再见到她，她的脸色正常了。又过了两年，她的脸色白里透红。法轮大法的威力真大！

我有个亲家是空军的师级干部。有一年，我得知他進了重症监护室，我就去看他。我说：“你若想病好，我说一句话，你同意的话，就眨眼睛表示同意。”我说：“我帮你退出中共的组织吧？”他眨了一下眼睛；我说：“你愿意念‘法轮大法好，真善忍好’吗？”他又眨了一下眼睛。

第二天，我又去医院看他，他已经出院了。现在，他健康的活着。

人出生在世上，是个迷的空间，看不到真相，所以会做错事，有生死病死的存在。大法正在迷世中度人。

# 【庆祝513】仁者之勇——写给父亲的家书

文: 海外大法弟子

【明慧网二零二一年五月二十三日】

爸爸您好！

好多年没有给您写信了。虽然我们父女几十年中生活在一起的时间不多，彼此很难说完全了解，但是，我知道您是一个作风正派、坦荡光明的人。这一点，其实也被我继承了下来。您未曾改变的遵循着一种传统的生活方式，尽管经济宽裕，却简朴、自律。在眼下这个道德下滑、乱象尽显的社会里不随波逐流，也是难能可贵。

爸爸，如果您觉的您一生不为利益与人勾心斗角是值得自豪的话，那么，您也可以为您的后人——我以及您的外孙铭铭感到高兴，因为我们同样继承了这一点，对物质利益少有贪欲，不会见利忘义。爸爸，谢谢您和母亲把我影响成为一个品行端正、是非分明、活的心底踏实的人。

爸爸，今天想通过书信和您说说心里话，因为我们缺少沟通，您可能对我人生中的一些选择不理解，不认同。非常抱歉，是我做的不好，没有向您陈述自己的经历，也没有真诚倾听您的意见。不是我不尊重您，而是我在您和母亲面前说话，心理压力比较大。虽然我今年五十有二，但在父母面前，总觉的自己象个时常犯错的小学生。我脸皮薄，心理承受能力差，使我常常逃避和你们的交流。

我年少时，体弱多病。让你们，特别是母亲，对我操的心比妹妹多很多。尤其遗尿的毛病，母亲很痛苦，我也很痛苦。记得多少次半夜里，母亲发现我又把床上的被子、褥子全弄湿了，她一面快速的收拾，一面大声的发泄着情绪。寂静的午夜，我真的觉的自己是家里的拖累，不知道活着有什么意义，时常冒出结束生命的念头，却没有勇气。

因为我身体不好，在同学中我的毛病也被拿来取外号、嘲笑。因此，我觉的自己低人一等，自卑、报复心强。我不思進取，没有奋斗目标。浑浑噩噩中，不知道怎样考上的大学。浑浑噩噩中，不知道怎样结的婚。讲台站了十五年，在第一个任职的学校，用“混口饭吃”形容很恰当。学校不正规，大家都是混日子的心态，我也心安理得的混着。

生孩子时，我大病一场。夏天要穿厚衣服，身上忽冷忽热；夜里常常失眠，白天无精打采。混日子都没有精力混了，只好搞个停薪留职，在家养病。这个医院進，那个医院出，我没有起色。中药喝的恶心，还是没有好转。病急乱投医，还跑去见一个搞巫术的，别人说灵验，我却没有感觉。

有一次，我从小何那里提着中药回你们那里时，看到口腔医院门口盖楼的脚手架，我站住了，希望它倒下来，让我长痛不如短痛。那时，我就是这样卑微的仅仅留着一口气。

一个朋友看我活的痛苦，借给我一本《转法轮》，说看看也许会对我有帮助。当时，我不相信看一本书能治好我的病。但是看了书以后，我的身体确实在短短的几个月后有了很大的变化。我没有進医院，也不再去小何那里拿中药。

半年后，我重返工作岗位。此前，一堂课四十五分钟，我只能站着讲十分钟左右，就必须坐椅子。课堂纪律不好，我就只能任由学生吵闹，因为我提不起气来制止他们。

修炼法轮大法后，上课四十五分钟，我不仅可以站着讲到下课铃响，还可以根据情况调节课堂气氛，既让孩子们活跃又不至于无秩序。要知道，这对六、七十人的班级来说，既需要脑力也需要体力，我却比从前任何时候都教的轻松。

到如今，我修炼法轮大法二十二年，从未吃药、進医院，无病一身轻，这是不争的事实。“滴水之恩，当涌泉相报”。爸爸，您说我能不感恩吗？更重要的是，《转法轮》给了我人生的答案。

我一直认为命运对我不公，多病痛；脑子不开窍，受歧视。大三我的眼睛出问题后，我心灵更加苦闷，有报复社会的扭曲心态，开始偷东西。在大学商场假装买东西，趁店员不注意，就把柜台上的商品装進自己的包里。甚至对我没用的东西也拿，然后扔掉，好象从中能得到一点心理平衡。

工作后，买菜时顺手牵羊拿人家的鸡蛋。这种行为，一直持续到一九九三年。在华生园，当我连买带偷拿着人家的蛋糕从商店出来时，猛然感到头顶上有双眼睛看着我，我不自觉的抬头。深邃的天空中，那双眼睛定定的罩住我，使我惶恐不安，不知道怎么回的家。这是我人生最后一次偷东西。华生园门口那双眼睛令我生畏，永生难忘！

虽然我杜绝了这种肮脏的行为，然而，我的心态还是不健康，怨天尤人。现在的女人生孩子，没有几个象我这样生出大病来的。这一点，我更是又怨又恨。看了《转法轮》之后，我明白了做人的道理，明白了自己为什么病痛多，被别人看不起。从此，我告诉自己，不管什么境遇，再大的不公，我都永远用好人的标准要求自己。

我庆幸自己能恰逢法轮大法洪传，如果不是走入法轮大法修炼，我无法想象生活在这样一个做好人都难的社会里，我怎样才能不随波逐流。在各种不公面前，怎样才能保持自己的良知不被泯灭。

孔子讲：四十不惑，五十知天命。我庆幸自己在不惑之年踏上了返本归真之路，找到了人生的真正意义。

爸爸，您也知道我以前脾气暴躁，点火就着。初中时，同学对我不公，我竟然站在课桌上拿扫把打他们；在您和母亲面前也是，被冤枉了，会歇斯底里的发泄；后来和学生相处，也战火不断。有一次，我和一个初二男生发生矛盾，他甚至抢过我的教鞭要打我。还有一次，我骂一个女生，她告到校长那里，要教育处通报批评我。唉！现在回忆起来，我都为自己脸红。

然而，法轮大法改变了我。我后来任职的学校，是当地重点中学，老师们为了职称奖金，人际关系挺复杂。然而，我却不时听到同事当面对我的赞扬。他们说我是现今社会上少有的“只讲付出，不求回报”的人。开玩笑说，我是“濒临灭绝的物种”。

年级主任是个不苟言笑的老太太，常常对我说：“全校就你心态好，我要向你学。”我的大学同学是副校长。有一次，他在我们年级办公室问一个老教师：“我这师妹怎么样啊？”老太太笑呵呵的说：“个个员工都象她，你们当官的就轻松了。”教导主任对别人说：“你看夏老师，干工作从不讨价还价，学生给她打的分最高。”爸爸，如果不是修炼大法，我不会得到这些好评的。

教师办公室的清洁卫生，都是找学生来做的，我也从没觉的不好。修炼后，我不再接受老师翘着腿，让学生拖地的现象了。我每天到了办公室，就在默默无声中把拖地、擦桌子做了。有一天，我正在拖地，一个老教师端着茶杯笑呵呵的说：“看来法轮功是厉害啊，把我们这懒丫头教勤快了。”当我离开学校的时候，年级组聚餐，有个男同事对我说：“夏老师，我很佩服你。你人很正，我这个人平时喜欢吊儿郎当开玩笑，在你面前，我却不敢放肆。”

我的学生常说我和别的老师不一样。他们私下告诉我：“年级只有你和某老师（一个受私塾教育的返聘的老太太）没有绰号，因为我们是尊敬你们的！”同事的孩子是我的学生，同事说：“只有你的话，我女儿会放在心上。家长把孩子交给你，就省心了。”我的学生送给我的笔记本扉页上写着：“我为自己感到庆幸，在人生迷茫的时候，有一个高贵的灵魂来到了我的身旁，为我指引方向。老师，我会永远记得您！”

一个男孩子，当我告诉他“天安门自焚”是中共的造假新闻，法轮功不是电视上说的那样，孩子大睁着眼睛，很吃惊。过后，他神情庄重的对我说：“老师，只要是您说的，我都信。”爸爸，这些话，是我教书生涯得到的最大回报。而这些回报，是在一九九九年我开始修炼法轮大法之后。

我的同事夸赞我，我的学生尊敬我。而我，感恩我的师父！感恩法轮大法！没有真、善、忍信仰的力量，在大染缸中，我也只能随波逐流，难以洁身自好。因为逆流而行是需要勇气的。

修炼前，我也对利益看的很重。记得学校开教研会回来可以报出差费，同事们都是尽可能多报几天，车费也尽可能多写，我也一样。修炼后，我不再这样做了，因为我明白，这样做，会损德。

爸爸，您可能还不知道，我曾经因为和两个路边的中学生谈论法轮功而被抓進拘留所。家人不敢告诉您，是怕您上火。这就是为什么他们希望我离开国内的原因。可是，爸爸您知道吗？拘留所的经历，不是我人生的耻辱。我在那里得到的是默默无声的同情，掷地有声的赞叹！

# 迫害法轮功　四川绵阳地区恶报12例

【明慧网二零二一年五月二十二日】（明慧网通讯员四川综合报道）自从中共江泽民集团发动对法轮功的全面迫害后，四川绵阳地区政法部门的人员为了捞取政绩，在中共胁迫下，在名利的驱使下，参与迫害法轮功学员，有的人可能得到了一点黑钱，仕途得到了升迁。可人做了什么，上天都有记载，都得在什么时候偿还。

绵阳梓潼县公安局国保大队胡世然，二零一六年腊月二十九日遭恶报去世。胡世然于二零零五年五月非法抓捕梓潼县法轮功学员夏全富及妻子谢某，并非法关押一个月，后谢某离世。

绵阳市某镇杨松泉，男，五十岁左右，是一个身强力壮的人。他跑去派出所举报：法轮功学员在开庆祝会，还放了鞭炮……。不到一个月，杨松泉就跌了跤，就此而瘫痪，没多久便死掉了。他的儿媳妇又与他人通奸，被他儿子发现，当场杀死了他的妻子，儿子也受到刑事处罚——判刑七年。杨松泉的妻子又双目失明，一家人就剩她带着个小孙子度日。这真是一个令人触目惊心的实实在在的现世现报的真实故事。

警察把六位女性法轮功学员劫持到了哈尔滨第二看守所，把刘文昌非法关押在双城看守所。这次绑架给其中每个法轮功学员的家庭带来了巨大的伤痛，家人找到村干部，要求帮忙营救被非法关押的亲人。然而，最后等来了村书记刘明的电话，无理的让每个人拿五万元现金，去给“疏通”。

二零二一年四月二日，刘文昌、王淑仁等七名法轮功学员，被道里区法院以暗箱操作的手法，用视频的方式所谓“开庭”，仅用了半个小时的时间，就草草收场。十几天后，家属打电话询问，得知王淑仁被非法判六年、罚款两万元；刘文昌三年、罚款一万元；另外五位老人是冯秀梅、孟秋艳、白丽霞、刘淑秋、刘淑英各一年、罚款两千元。

从看守所中得知，刘文昌在双城看守所经常犯癫痫病。刘文昌在炼功前，就有非常严重的癫痫病，犯病时，经常抽过去不省人事，身边不能离人，很多次邻居在路边的土沟中，把犯病抽的不省人事的他救起，送回家。修炼前，他还有严重的肺气肿（哮喘），严重时，上不来气。这些疾病在修炼法轮大法后都不治而愈了。现在，刘文昌被非法关押在看守所恶劣的环境中，亲友都非常担心他的身体状况。
