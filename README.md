### 售票员高喊：“法轮大法好，真善忍好”

文: 河北农村大法弟子 更新: 2022年09月08日

【明慧网二零二二年九月八日】那年九月，我同弟妹赴京给大姑姐做寿，当日下午回家时到车站等车，见一男子愁眉不展站在路旁，我上前搭话，得知是同乡，他妹妹得了不治之症来京买药。当时我想，我是大法弟子，与同乡不期而遇是缘份，这不是师父安排的吗？我便抓紧时间给他讲真相、劝三退，他不搭话。告诉他诚心敬念“法轮大法好，真善忍好”，回家后让妹妹也念，会得到神佛的保佑。他还是一语不发，我认真的讲着。他忽然说：再说我举报你。当时我很镇静，慈悲的面对对方说：老弟呀，你是善良人，不做这样的事，我说的都是真的，大法师父是救人的；共产党、江泽民败坏人类道德，是毁灭人类的；人类败坏将来会有大劫难的。你还年轻，明真相得救，能见证到那一刻。我的善念解体了对方背后的邪恶，他微笑着说：“我记住了。”
这时客车来了。我姐俩上车后，弟妹说：“可把我吓坏了，你真胆大，这是在北京啊，当时我真想躲开……你所说的我也听明白了。”我很欣慰的是两个人同时听了真相，大法弟子救人是最正的事，正念能解体一切邪恶。

上了车，前排有两个空座位，我俩正想落座，售票员呵斥说：“他X的，大老的不在家呆着出来瞎跑，这个位子有人了，起来，起来，一边站着去。”我姐俩站旁边，一站地过去了，不见有人，这两个位子依然空着。弟妹晕车，我让弟妹坐下，我是大法弟子无所谓。尽管售票员满口脏话，我不动心。他歪着脖子看着我，我用善的目光对视他。

我想：这事不是偶然的，是师父安排考验我的心性的。我买票多余的钱，他也不找给我，还满嘴脏话。车里的乘客也都投来各种异样的目光，以为我们会打架。这要是在我不修炼以前，按我的强势就是一场打斗，争个我对你错才能罢休。

又到了一站，上来了三个女士，售票员连骂带数的训斥她们，骂的很难听，满车的人都很紧张。我看不下去，便阻止他这样，就让弟妹站起来让座。其中一女士很客气的说：“您坐着吧，我站着。”推让不得，弟妹就和另一女士坐下，气氛稳定下来了。是我讲真相的机会了。我说：“你们这一来迟了挨一顿训不说，为给你们占座的这位老弟可费心不小哇，差点引起一场矛盾。”

车上人都笑了。我说：“我是学大法的，我师父教我们做好人，救人。人类今后会有大灾难，天灾、瘟疫、人祸不断，是人类自己造成的。我们要做好人，上天才能护佑。法轮大法教人向善，按真善忍的标准做人，心中诚念‘法轮大法好，真善忍好’就能保平安。”

回头我给站着的女士说：“大妹子，你这岁数肯定戴过红领巾，死人血染的戴着它多晦气呀！”她若有所思。我又说：“我们中国是神州，共产党无神论，不让老百姓信神，他们烧香。江泽民迫害法轮功，老天要灭中共了。退出党、团、队才有美好未来。”女士三退了，她答应了用这个方法告诉她的亲人朋友和身边的人。

车马上要到我们下车的站了，我向司机说：“老弟呀，他（售票员）这样强势你没有帮腔真是个好人啊！请您记住诚念：法轮大法好，真善忍好，开车顺利保平安。”司机说：“记住了。”

要下车了，售票员找了钱给我，笑着说：“法轮大法好，真善忍好。”车停下来了，我们下了车，只见售票员对大家喊：“法轮大法好，真善忍好！”上了车，他又冲着车外喊：“法轮大法好！真善忍好！”车上乘客和外面行人都听着他在喊。

我心里被这突然的一幕感动了，是我放下了争斗心，慈悲众生讲真相，大法善的能量清除了世人的污浊，回归了善的本性。我目送客车南去，久久不能平静，真为明真相的生命们感到欣慰。
