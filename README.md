### 法轮功是人类的希望

文: 中国大陆法轮功学员 

【明慧网二零二一年十一月十一日】中共对法轮功的迫害已经持续了二十二年之久，在这二十多年当中，中共集古今中外邪恶之大全对法轮功和法轮功弟子进行残酷的迫害——“名誉上搞臭、经济上截断、肉体上消灭”。中共对秉持“真、善、忍”标准的修炼者进行的迫害，不仅仅是对中国也是对全人类道德体系的破坏和摧毁。
自中共迫害法轮功以来中国大陆整个社会道德急剧下滑，很多中国人的品德已降至历史最低点。中国已经成了物欲横流的社会，人为了钱，为了满足个人的欲望什么坏事都干得出来，暴力、谎言、色情淫乱等各种下流东西处处可见，甚至杀父母、杀妻、杀子，连最基本的人伦底线几乎都不复存在。各种社会问题层出不穷。

在这样的社会里人人都是直接或间接的受害者。人类的出路在哪里？重建人类的道德，恢复传统的价值观已经势在必行。造成人类社会道德全面下滑的根本原因是推行无神论和进化论的中共对中国的统治和操控（详细论述请参阅《共产主义的终极目的》一书）。所以只有解体中共，中国才有希望，只有解体中共人类才有出路，才能走上正统的道路。必须立即停止对修炼真、善、忍的广大法轮功学员的迫害。对法轮功学员长达二十多年惨无人道的迫害，这是人类历史上的耻辱。扶正祛邪，远离中共已经成为摆在每一个人面前的使命与选择。

被中共迫害的法轮功学员到底是怎样的一群人呢？下面是法轮功学员自述的亲身经历，我们从几个层面给予说明。

#### 一、在家庭中

俗话说“清官难断家务事”，幸福的家庭都是一样的，不幸的家庭各有各的矛盾。在如今戾气超重的中国大陆，夫妻关系、婆媳关系是最难处的人际关系，看看法轮功学员是怎样做的？

在《炼功人以德报怨　全家四代受益》中，作者讲述了自己结婚后因遭到公婆的不公平对待和丈夫的打骂，心里充满了怨恨与委屈，她悄悄地存了八百元钱，想着等孩子长大了就和丈夫离婚。

作者学法轮大法后，明白了人受苦的原因，整天乐呵呵的，放下了对公婆和丈夫的怨恨，任劳任怨的干活，从怨恨公婆变为孝敬公婆，在公婆病中尽心尽力的照顾他们，他的善行感化了公婆和丈夫，家庭变得和睦。

在《几近破碎的婚姻走上坦途》中，法轮功学员讲述了自己的亲身经历：自己放弃了大城市的生活，放弃了待遇较高的工作，跟着丈夫回到他的老家。过着没房子、没工作、没钱、举目无亲的日子。婆婆因受中共谎言毒害，怕修炼法轮功的儿媳影响儿子的前途，挑唆儿子与作者离婚。作者的丈夫天天对她发脾气、摔东西，甚至几天不和作者说一句话，在这样以泪洗面的婚姻中，作者无怨无恨，依然善待丈夫，天天煎炒烹炸迎合丈夫口味去做饭。作者持续的善行感化了丈夫，丈夫现在逢人就说：“这些年多亏了她”。

在《婆婆拿我当贴心人》中，作者讲述了自己嫁到了一个公婆打骂儿子、儿媳是家常便饭的人家，作者受尽了公婆的虐待，精神几乎崩溃，也落了一身的病。但作者在修炼法轮大法后，大法师父博大精深的法理驱走了作者心中的阴霾，照亮了作者的心。她明白了作为一个修炼的人要对谁都好，她尽心尽力地照顾年迈的公婆。在公公去世后，她尽心照顾婆婆的衣食住行，婆婆生病时，她为她请医购药，床前床后陪护；给她洗头、洗脚、洗澡；在婆婆大便便不出来时，她多次用手给她一点一点抠出来。让原本的恶婆婆多次被感动得落泪，夸作者好，比她的亲闺女还好。

这样的例子在法轮功学员中不计其数。大法化解了她们心中的怨恨，她们孝敬公婆，善待家人，在家庭中体现出了法轮功学员的风范。

#### 二、在孩子的教育上

在当今的中国，教育孩子是令大多数家长头痛的问题：学校只讲考试分数，不注重学生的品德，整个社会充满了假恶斗，在这样的大环境下很多孩子都变得冷血、暴力、叛逆、不懂感恩。近几年杀母事件更是频发。怎样才能把孩子教育好，怎么样才能培养出懂得感恩的孩子？下面看一下法轮功学员在教育孩子问题上的是如何做的：

在《单亲妈妈与小丫的故事》中作者说：“孩子的成长过程，也是家长的成长过程。一个内心阴暗的家长，很难教育出阳光四溢的孩子；一个唯利是图的家长，很难教育出宽容豁达的下一代。我很感激法轮大法为我教育孩子、带好孩子指明了方向。”在小丫十一岁时，作者成了单亲妈妈，作者经历了撕心裂肺的痛苦，但是，作者并没有让小丫感受到自己内心的各种苦楚，也从来不向她数落她父亲的不是。那段时间，是法轮大法带作者走出了困境。作者说：“放下了怨恨，才能解开捆绑在自身的桎梏；对别人宽容，就会走出狭隘，为自己铺就一条光明的路；善待别人，才能体会到慈悲的博大。”作者通过各种事情教育孩子做一个宽容、善良、为别人着想的孩子。作者的言行影响着孩子，虽然孩子生活在单亲家庭，但孩子阳光活泼，为他人着想，和爸爸关系很好。

在《令婆家自豪的儿媳》中，作者在对孩子的教育过程中，遵循中华传统的道德理念，用正确的价值观引导孩子。平时她经常和孩子一起背师父《洪吟》中的诗词，也经常给他讲做人的道理。在孩子叛逆时期，作者明白要想教育好孩子，就要先改变自己，只有自己变好，才能从根本上解决问题。于是她不再苛求孩子，不管他表现得如何，她都真心地为他好，宽容他的不足，包容他，和风细雨般地循循善诱，讲道理。慢慢地孩子变了，听话了，也能理解妈妈了，同时学习成绩也直线上升，最后考上了理想的大学。

升学宴上，作者在致辞中对孩子说：“孩子，你要懂得感恩，别人对你有一点好处，你都不能忘，都要报答，别人对你不好，你要感谢人家，因为这说明你没做好，他让你看到了自己的不足，你要努力改正。”同时作者也告诉在座的亲朋好友：是因为对真、善、忍的信仰，我才能走到今天，才能把孩子教育好。谢谢大法！感谢师父！

在《明慧小弟子园地》中有很多法轮功小学员因为学了大法戒除了游戏，戒除了不好的习惯，成为了品学兼优的好孩子。这样的例子很多很多，在大法的熔炼中，令家长头痛的孩子教育问题得到了很好的解决。

#### 三、在社会中

现在的中国大陆社会，世风日下，为了钱什么坏事都干，贪污腐败，假货遍地，尔虞我诈，在这样的恶性循环的环境中，很难有人独善其身，我们下面看一下法轮功学员的亲身经历看看他们是怎么样做的。

在《天变蓝了　风变暖了》——一名税官的自述中作者写道：因为曾在酒桌上豪饮，身体每况愈下，加之丈夫的背叛，生命对她来说已经失去了意义。学大法后，她明白了“失与得”的关系，暗暗下决心要做到出污泥而不染，从我做起，从现在做起。于是，有了第一次坦然拒贿；第一次在饭店吃饭自己买单；第一次明白洗浴是要买票的；第一次知道了理发要多少钱；第一次去乘公交车；第一次……有了这许多的第一次之后，她发现天变蓝了，风变暖了，内心不再彷徨，人生有了归宿和目标，由衷地感到：师父真好！

在《经商中修心救人》一文中作者讲到：他修炼前在生意场上挣钱不择手段，修炼后他断掉假货源，诚信经营，从一个奸诈商人转变成了一个正直的商人，经营中为顾客着想，善待员工，和同行以诚相待，对同行提供帮助，真心地希望同行的货销路好。他用自己的行为感动了顾客、进货商、手下员工、不修炼的家人。作者说：“每迈一步，每提高一点，都是师父慈悲保护。是大法纯净了我，是师父塑造了我，我能从十恶毒世的人间破壳而出，毅然走到今天，一切都归于师父和大法！”

在《我用言行证实大法的美好》中作为教师的作者写道：“我用大法法理真、善、忍的标准要求自己。工作中不但教好文化课，生活上我也细心地照顾孩子：孩子有感冒发烧的，我就给孩子烧开水吃药；有肚子疼的，我就用暖宝给孩子暖肚子；冬天给孩子烧热水喝等等。我的做法终于感动了家长们。家长们都知道我是修炼人，从我身上，他们看到了大法的美好，消除了对大法的误解。”作者拒收家长送的钱物。多次把评“优秀教师”的机会让给其他同事。作者的言行让校领导、家长、同事看到了大法弟子是什么样的人，法轮大法好。

这样的例子还有很多，法轮功学员在各个行业中践行着真、善、忍的做人标准，放淡名利，真心地为他人着想。他们的言行，使中共对法轮大法的诬陷谎言不攻自破。

法轮大法的传出如一股清流涤荡着人们内心的污垢，抚平了人们心中的伤痛，使道德升华，人心向善。这样好的功法却遭到了长达二十多年的迫害，这是人类历史上的耻辱，这是对人类良知的践踏，是对正统的价值体系的破坏，使山河变色，人心魔化，只有立即停止对法轮功的迫害，解体中共邪党，人类才会有出路，社会才能有希望。人类才能走向真正美好的未来。
