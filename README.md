### 诚念九字真言　肺癌晚期的老婶绝处逢生

文: 河北大法弟子 

【明慧网二零二二年六月十八日】我是河北省某县商业企业退休女职工。我于一九九七年开始修炼法轮大法，久治不愈的鼻窦炎、头晕等疾病很快就痊愈了，无病一身轻。我丈夫修炼大法后，困扰他多年的十二指肠溃疡、血压高、心律不齐、失眠、肿瘤等疾病在他修大法后不到一年的时间内全部消失，身体非常健康。我们真正感受到了大法的神奇与超常。
我们努力做好“三件事”，履行着自己的使命。二零一八年秋，在我家的南瓜上、铁丝网上盛开了五十多朵圣洁的优昙婆罗花！至今仍开放如新。

我从二零一三年开始面对面讲真相救人，见证了很多世人明白大法真相、“三退”后诚念“法轮大法好，真善忍好”九字真言而得到大法的福泽和大法师父保护的神奇故事。现举一例，我娘家老婶诚念九字真言而绝处逢生的神奇事例。

我娘家老婶今年八十岁，家在农村，没上过学，为人善良。我曾给她讲过大法真相，告诉过她念九字真言，那时她好像不太重视。

老婶从年轻时就患有严重的心脏病，曾多次犯病昏倒。二零二零年十一月，她又被查出患肺癌晚期，癌细胞已扩散到骨头。两种病加在一起，医生也是束手无策。老婶自己也预感不好，坚决要求回家。家人见她病情危急，也告知亲近的人们回家见她最后一面。

二零二零年十一月中旬，我回去看老婶时，她真是到了生命的尽头，呼吸困难，鼻孔插着氧气管儿，一会儿一抢救。怎么抢救呢？当她喘不上气、憋得没办法时，晚辈们多人迅速架着她的胳膊，托起她的身体，后边的人连续快速地拍打她的后背，堂弟对着她耳朵声嘶力竭地不停地大喊：“妈妈！妈妈！”直到她喘上一口气。这期间，又打强心针（可能还有止疼针，因为她还伴有浑身疼痛）。

其他亲近的人们也陆续来到老婶家看望她。她自己也说不行了，嘱咐儿孙们，穿上寿衣后，头要朝哪个方向，在哪儿搭灵棚，吃剩下的药能退就退等一些后事。她还说，昨天已经走了一回了，是孩子们硬把她叫回来的，她还看到了什么什么，说自己过不了今天了。最后她说：“我不想走，舍不得这些孩子们啊！”景象真是凄惨！

这时，我说：“老婶啊，我告诉您一个方法，心念九字真言：法轮大法好，真善忍好。”她说：“什么？你再说一遍。”我又给她说了一遍，并告诉她，法轮大法是佛家上乘大法，按我自己的理解又介绍了真、善、忍三个字的含义。我说：“老婶您这么善良，您诚心敬意的念九字真言，神佛会保佑您的，在心里念也行，您就不停的念。”看得出她是相信的，在生命弥留之际，让她看到了一线希望。大约过了两、三个小时，老婶说：“我哪儿也不难受了，给我洗头洗脚。”她可能还是担心闯不过去，下午我们要回去了，我对弟媳说：“老婶现在状态不错。”弟媳说：“可能是回光返照吧，恐怕不是真好，因病重已有一段时间了。”

第二天，我给老婶的儿子（我堂弟）打电话询问老婶情况，堂弟说老婶从昨天下午开始见好。过了几天，我又电话询问，堂弟说老婶情况越来越好。我告诉他：“要常提醒老婶别忘了念九字真言，你们也跟着念，对她很有帮助。”堂弟说：“知道了，提醒她了，她常念啊！”之后，我又电话了解到，老婶逐渐的越来越好，慢慢的不用吸氧了，不用打针了，自己也能下床大小便了，精神也自然好了。

今年清明节，我们回娘家看她时，我问她：“老婶，告诉您的九字真言还记得吧？”她说：“我常念啊！刚才还念了呢。”看到老婶现在的状况，亲属们都觉的不可思议。在院子外边，我听到老姑和两个堂妹议论说：“再检查一下，这哪像说的癌症呀？真是想不到。”我说：“只要真心相信大法，诚心敬念九字真言，就会有这样的效果。”

我又简单地介绍了国外医学博士对三十六例武汉肺炎患者诚念九字真言而神奇康复这一现象進行科学研究的论文情况，她们都能理解，也很高兴。以前，我也多次给她们讲过大法真相，她们也作了“三退”，这次又给她们每家一份真相资料，并希望她们常念九字真言，都能受益、保平安。

我代表明白大法真相并在大法中受益的世人们、亲友们向慈悲伟大的师尊表示最诚挚的感谢！

师父辛苦了！谢谢师父！
