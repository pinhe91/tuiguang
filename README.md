### 诚心念九字真言　女儿如愿以偿

文: 大陆大法弟子 

【明慧网二零二二年九月十九日】我修炼法轮大法之前，身体很不好，疾病缠身，修炼后身体上的疾病神奇般的不翼而飞，我无病一身轻，修炼后近二十年再没吃过一粒药、没打过一次针，现在健健康康的。感谢师父感谢大法！
我修炼法轮大法，家人明真相也是福报连连的。下面只说说发生在我女儿身上的神奇事。

我女儿是个急性子，十二月份结婚就想要孩子，可过了三个月没有怀孕，她就急了。有一次为了安慰她，我把法轮大法的年历吉祥宝宝送给她一本。第二年五月十二日，我让女儿陪我去参加亲戚的婚礼时，她愁眉苦脸的说：这个月又来月经了，一直怀不上孩子怎么办啊？我说：有一个最好的办法，就是你用心的念诵“法轮大法好、真善忍好”就会有奇迹的。
当时她没有吱声。可是下个月她和丈夫开车来我家，一進院子就惊喜的告诉我说她怀孕了！我说：你一定是回家后念诵“法轮大法好、真善忍好”了，她说：“是的，法轮大法真神奇啊！没想到念诵这九字真管用啊！”

又过了一个多月，女儿来我家，上厕所的时候发现见红了，把她吓坏了，问我怎么办啊？我说：只有大法师父能帮你。她说：“快把大法师父写的《洪吟》书给我看看！”我把书递给她，她一口气把书看完一遍就睡觉了。第二天早上醒来就什么事都没有了，女儿非常高兴。

女儿生孩子的时候难产，折腾了三天也没生出来。我也很着急，就跟她婆婆和她丈夫说：“咱们都帮她念法轮大法好、真善忍好，求大法师父帮帮忙吧！”于是大家就一起念了起来，不一会儿女儿就顺利的生下来一个七斤七两重的女孩，而且母女平安。谢谢师父！感谢大法！

现在小外孙女已经两岁了，聪明伶俐非常可爱。我每次去她家时都教她念“法轮大法好、真善忍好”，她就用小手点自己的脑门说大法好，我唱法轮大法弟子的歌曲，她也跟着唱，真是非常可爱。
