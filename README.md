### 曾陷十二年冤狱　辽宁本溪市张贵生又被构陷到法院

【明慧网二零二二年十一月二十三日】（明慧网通讯员辽宁报道）辽宁省本溪市七旬法轮功学员张贵生先生，二零二二年七月十三日被本溪市公安局明山分局明山派出所警察非法入室绑架，后被劫持到本溪市看守所关押构陷，十一月十八日被构陷本溪市南芬区法院。
张贵生（张桂生）先生，七十四岁，家住本溪市明山区万豪花园，二零二二年七月十三日被本溪市公安局明山分局明山派出所李非凡等警察入室绑架，后被劫持到本溪市看守所迫害。

二零二二年八月十八日，张贵生被本溪市南芬区检察院非法逮捕；本溪市南芬区检察院两次退卷，要公安补侦、构陷。二零二二年十一月十八日，张贵生被本溪市南芬区检察院非法起诉到本溪市南芬区法院。本溪市南芬区检察院公诉科检察官钱文文（雯雯）以张贵生态度坚决，拒不认罪为借口，不允许亲友辩护人阅卷、会见。
此前，张贵生因坚信大法，于二零零二年十二月被本溪市平山区法院枉判重刑十二年，非法关押在辽宁省锦州监狱遭受残酷迫害。以下是张贵生在狱中遭受残酷迫害的部份事实：

二零零三年九月下旬，张贵生坚持炼功，被锦州监狱二大队恶警用胶带将四肢牢牢缠住，嘴也被缠上，由四个犯人架着出工，上千犯人目睹此景，恐怖笼罩全监，大队召开全体大会，逼迫被非法关押的法轮功学员做“检讨”，接着又将张贵生关进小号。这次事件的责任人是二大队恶警大队长张宝志、大队长李跃、分队长李向阳。

在狱中，张贵生还被恶警用“抱凳”折磨，“抱凳”就是：在长一米四左右、宽一米的铁板上固定一个木凳，凳子形状象过去量米用的斗，比斗高，约两尺，倒扣在铁板中间。凳两侧的中央处各有一个铁环，能伸进胳膊。恶警让犯人强迫给法轮功学员戴上脚镣，两腿夹着木凳坐下，将胳膊伸入铁环，扣上手铐。铁环在木凳约一尺处，因此胳膊抬不起来，腰也直不起来，就这么趴着、弯着腰。

![image](https://user-images.githubusercontent.com/79625284/203513100-2595f39d-53d8-4afd-94c6-44fe772a31af.png)

中共酷刑示意图：抱凳

恶警对张贵生实施的迫害手段还包括：蹲狱、整夜不让睡觉、上“大抻”酷刑。张贵生被迫害失眠二十多天、右胳膊抬不起来，腿部被迫害的行走不便，一瘸一瘸的。参与迫害的主要有三个恶人崔建、金伟、张怀东。

张贵生经常遭到毒打，曾被一恶警队长（警号2158336）打的牙齿部份脱落，后来四肢行动迟缓。

二零零七年五月九日，张贵生被恶警关小号迫害，险些丧命。
