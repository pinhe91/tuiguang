### 善待大法一念　天赐幸福平安

文: 中国大陆大法弟子

【明慧网二零二二年九月十一日】我是九六年走進法轮大法中修炼的老年大法学员。大法虽然遭到了中共邪恶集团二十三年的打压，但是千千万万的大法弟子依然坚持不懈的向世人讲述着法轮大法的真相。下面讲一讲两位明真相的世人善待大法得福报的故事。

#### （一）陶厂长一家好运连连

我工作单位的陶厂长（化名）于一九九九年春调到另一家工厂当书记。这年的十月份，我碰到了他，我向他说：“近两三个月，法轮大法受到严重迫害，听说你们厂里收的大法书不少。”他说是收了大法学员的一些书。我说：“法轮大法是佛家上乘修炼大法，大法的书字字句句都是从不同角度教学员做好人的宝书。你们要是毁了这些书就是犯了大罪，而保护好这些书是功德无量的事。”他说：“那我们就不烧不卖也不上交，用报纸把书包好送到二楼文件柜里保存好。”他们也是这样做的。

第二年我因向民众讲大法真相被非法关押在县看守所受迫害时，陶厂长邀本厂一位老干部到看守所找他当兵时的老战友（看守所的教导员）说情。在陶厂长这位老战友的关照下，我就回家了，没有被关到洗脑班继续受迫害。
陶厂长是住在我家附近的老同事。据我所知，他们家近二十年来好运连连。他们老俩口都是七十五岁的人了，他们走在街上总是满面春风，完全都是中青年人的姿态。他儿子和儿媳停薪留职到北京发展也小有成就。偶尔回家，我向他们讲大法真相，他们都是彬彬有礼，用心倾听。他孙子从小学到高中成绩优秀，现在正在北京一所知名大学读书。

陶厂长一家这些年来家运看好，他和他的妻子都知道，这是他保护了大法书，帮助大法弟子减少迫害得到的福报。从他们和我交谈中，可以看出他们觉的自己维护大法和保护大法弟子的做法是对的，并流露出几分明白大法真相的幸运感和自豪感，也做了三退（退出中共党团队）。

#### （二）保护大法书的农民得福报

汪学农（化名）是家住县城附近的农民，为人正直豪爽，我多次向他讲过的大法真相，他都表示认可，并退出了曾经加入的党、团、队。如今他也是六十多岁的人，他夫妇俩也是红光满面，走路生风。前不久，他们到我家来玩。他说：“前几年我媳妇生个儿子，女儿也紧接着生个儿子。亲人好友都问我做么事积了大德，摊上这样的好事，这个问题我好久没想明白。”

我说：“你那些年帮我们保护那么多的大法书，这不是做了一件大好事吗？我们炼功人都说这些书是天书呢。我的一些亲戚都不敢帮忙保管，怕受牵连呢。”

他恍然大悟，意味深长的“呵”了一声，说：“你说的不错，我帮你们保护大法书是积了大德了。怪不得我的石材厂一转让出去，不到二个月，石材厂就出一起事故，一位工人因工伤事故离世。还有我儿子，从省城的重点高校毕业到沿海城市从事软件开发，他四十岁不到年薪就有百万呢。原来大法弟子说的‘善待大法一念，天赐幸福平安’都是真的啊！”

在一九九九年七、八、九月间大法遭到残酷迫害的那些日子里，邪恶人员收缴大法弟子的大法书是他们的迫害手段之一。这时我感觉大法书放在家里确实不大放心，就把大法书送到汪学农家里，请他代我保管。他二话没说就答应了。过了几天又有一个大法弟子送给我十几本大法书，让我替她保管。我又把书送到汪学农家。他说这不要紧，你有书尽管拿来，装在我家的一个空着的大木柜里，保证没问题，你放心就是了。

#### （三）大法弟子的善行能救人

一九九九年三月上旬，我们单位组织五十多岁的女职工去黄山旅游。到半山腰的时候，天气由晴转阴，起了冷风，我们都感觉有些冷。到山顶就更冷了，还有冰雪没有融化，我们带了大衣的人都把大衣穿上了。这时有一位年纪比我大几岁的退休职工，身体单薄，很冷的样子，一看就知道她没带大衣，我就毫不犹豫的脱下大衣给她穿上。

我们的汽车准备返程的时候，发生了年轻职工从车窗翻進去抢座位的事。因为车上原有的固定座位不够，中间过道上摆了些小板凳作为临时座位。我们几个年纪大的被安排在固定座位上。这时带队的副厂长说：“刘月红（化名）不幸生病了，她坐小板凳不行，哪位姐妹让个座位给她坐。”副厂长说完几分钟都没有人让座。这时我站起来说：“小刘，我和你换个座位。”副厂长说：“不行，怎么能要你退休的人让这位呢。”我说，“小刘坐在小板凳上，身不能靠，手不能扶，回家还有三百多公里，她是病人，她难受。我是厂里内退职工，年纪不很大，又是炼功人，没有座位站着也不要紧。”副厂长见没有人让座，也就同意我和小刘换了座位。

时间过了半年，迫害大法的阴风吹進了我们工厂，还是这位副厂长拿着“不修炼的保证书”找我签名。我开始没太注意就签了。他走了不远我就警觉了：那保证书上写了些什么我还没看清，是不是有污蔑大法的话？我立刻快步追上他，把保证书拿来仔细一看，果然是污蔑大法的。我立即跟他说：“这个名我不能签。大法教我们做好人，让千百万人道德回升，身体变好，我还要坚持炼下去，这张保证书我收回去。”副厂长说：“你叫我怎么说呢？你们炼功的人是好人，你是一个最好的人。前不久去黄山旅游的过程中，天气太冷的时候只有你一个人脱下大衣给年老体弱的人穿上，一车几十人只有你一个人给病人让座位。你这样助人为乐，舍己为人的好人值得大家学习。我让你签名也是受人指派的，你不愿签就算了吧。”副厂长的妻子说：“大姐（指我）你是个有信仰的热心快肠的大好人，我们要和你保持交往。你有什么事，我们要帮助你。关照你。”话虽不多，但言为心声，体现了世人对大法弟子的尊重，对法轮大法的虔诚，也是摆放了自己的位置。

这件事后我经常想，黄山之行，我脱了件衣裳给老人穿，让了个座位给病人坐，虽然是很小的善念善行，却能帮助世人了解大法，这对我个人来说其实也是修炼人应有的行为。
