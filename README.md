### 上海老太结束冤狱　原住地、现住地警察仍不放过她

【明慧网二零二二年九月三十日】（明慧网通讯员上海报道）上海78岁的法轮功学员裴珊珍，因发放真相资料，二零一八年底遭中共非法判刑三年半，被非法关押在上海南汇监狱迫害。裴珊珍太太虽于二零二二年六月结束冤刑，但至今仍频遭原住地和现住地的派出所、居委会、社区人员骚扰。
裴珊珍一九四四年十二月二十五日出生，原向群中学（现并入市西中学）退休教师。她原住静安区南阳路183弄小区。她因发放法轮功真相资料，曾多次遭中共法院诬判，二零一八年她再次被关入监狱。她丈夫施金泉由于妻子遭迫害，饱受精神、心理摧残，于二零二零年九月十八日悲愤离世，享年84岁。

遭此变故后，裴珊珍的小儿子将家搬到松江城置路257弄尚樱苑。裴珊珍老人于二零二二年六月二十六日结束冤狱后，又被隔离七天，于七月三日被人送至她小儿子家。
裴珊珍回到亲人身边后，她原住地的南西街道的户籍片警和现住地的松江尚樱苑所属警察，多次上门骚扰她和她的小儿子，逼迫签署文件、拍摄照片。二零二二年九月二十八日晚七点五十分左右，又有两名身穿警服的一男一女敲开裴珊珍住处的门，他们声称是宣传反诈骗的，要求拍摄裴珊珍的人脸照片，还要求签署不知名文件，被裴珊珍的小儿拒绝后关门。此二人没有挨家挨户敲门，而是直接敲裴珊珍租住的房门，明显是冲着她专程前来。

#### 裴珊珍老人多年遭迫害事实简述：

一九九九年七月二十日，中共开始公开迫害大法后，裴珊珍于二零零零年六月进京上访讲大法真相，被警察绑架至上海静安看守所非法拘留七天。

二零零零年七月十三日，裴珊珍因在小区内散发真相资料时被摄像机录像，又被警察抓入静安看守所，非法刑事拘留三十五天后被非法批捕，十一月在静安法院公开开庭，后被释放。

二零零一年三月，裴珊珍因借了一盘李洪志师父讲法录音带给有缘人，又一次被警察绑架至静安看守所。她拒绝放弃信仰，坚持炼功，被狱警铐在铁栅栏上，用毛巾勒住嘴，几次险些背过气去。她绝食反迫害，遭迫害性灌食，吐血不止。她后被劫持到上海女子劳教所非法劳教两年。

二零零五年三月，裴珊珍在小区投真相卡片时遭跟踪的恶人绑架，同年六月二十三日被上海市静安法院非法判刑四年。裴珊珍当庭提出不服理由，审判长芮志民反复大叫：“我们就听党中央的。”

二零一二年六月十四日，裴珊珍因以信的方式向邻居介绍大法真相，第五次被绑架，后被非法判刑四年半，被劫持到青浦女子监狱迫害；于二零一六年十二月十三日结束冤狱。
