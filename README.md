### 山西大同市牛蘭雲被劫持14小時後離奇身亡

【明慧網二零二二年五月二十五日】（明慧網通訊員山西報導）山西省大同市五十九歲的法輪功學員牛蘭雲，二零二二年三月二十八日清晨從其出租屋窗口援繩而下後，被警察劫持，當天晚上八點多鐘，家屬被通知到殯儀館認屍。短短的十四個小時，一個活生生人就這樣的沒了？家屬愕然、悲憤。
牛蘭雲，女，一九六三年出生，原大同市雲崗大廈營銷職員，已退休。她於一九九八年四月開始修煉法輪功，很快久治不癒的產後風濕病症不翼而飛。同時，真善忍的法理讓她心性得到提高，寬厚待人，家庭和睦，孝敬父母，並精心照顧生活不能自理的婆婆。

自從以江澤民為首的中共流氓集團一九九九年七月發動迫害法輪功的運動後，牛蘭雲曾兩次被非法勞教，多次遭受迫害、酷刑折磨、絕食期間被灌食、灌藥，被迫害致生命垂危。

#### 牛蘭雲遭綁架、離家、綁架、死亡的經過

二零二零年，牛蘭雲和幾位法輪功學員被跟蹤的警察綁架。由於疫情關係，警察當天將牛蘭雲釋放，但聲稱以後會再找她。

從二零二一年正月初六開始，大同市公安局國保大隊隊長孫文龍、新華街派出所所長楊浩翔等警察，一直不斷地騷擾、恐嚇牛蘭雲及其家屬，並有再度綁架她的意圖。於是牛蘭雲離開家，在大同西韓嶺恆安小區租房居住。

據悉，一次牛蘭雲上樓的時候，和大同市國保大隊的警察擦肩而過，只是當時沒有被他們認出來。（也許牛蘭雲這時已經感受到大同市國保大隊長孫文龍一夥對她的追蹤。）

二零二二年三月二十八日早晨六點多鐘，有樓下的目擊者看見，牛蘭雲打開窗戶，手拽住一根繩子，從三樓援繩而下，滑到將近一樓時，繩子突然斷了，她摔在地上，但她又站起來，並向目擊者這邊走過來。目擊者對她說幫她打120，牛蘭雲說不用。很快120車來了，牛蘭雲堅決不上車，就在有人拉她上車時，警車開過來了，下來的警察兇狠地把牛蘭雲塞入警車，拉走了。

隨後，警察又綁架了牛蘭雲的房東。打開牛蘭雲的房門，看見盆裏有一塊剛和完的麵團，說明牛蘭雲從窗口援繩而下之前正準備做早飯。（是甚麼緊急情況讓牛蘭雲鋌而走險的逃離？）

當天晚上八點多鐘，警察將牛蘭雲的死訊通知家屬，要家屬直接到大同殯儀館認屍體。家屬愕然兼悲憤，從早上六點鐘事發，在這短短的十四個小時的時間裏，一個活生生人，就這樣的沒了？大同公安局國保大隊警察到底對牛蘭雲做了甚麼？

#### 牛蘭雲曾兩次被非法勞教迫害

二零零零年十一月，牛蘭雲去北京信訪辦上訪，想為大法說句公道話，結果被信訪辦的人通知北京前門派出所的警察將她綁架，前門派出所又通知山西駐京辦事處，第二天，大同「610」人員與大北街派出所警察把她帶回大同，關入大同市第二看守所，獄警曾兩天兩夜不讓她睡覺，三九天強制在院裏剪頭髮，在冷水管沖頭。

二零零一年一月，牛蘭雲被劫持到山西女子勞教所。一次因她堅持煉功，被獄警中隊長陳慧茹拿電棒擊打，打的手腫得像個麵包。她絕食抗議迫害，被強行野蠻灌食，在獄警的慫恿下，五個吸毒犯對她摁腿、摁頭、壓胳膊、用鋼勺撬牙灌食。有一次，惡徒在灌食中下了不明藥物，灌後牛蘭雲感到心裏難受至極，好像要瘋了。一次勞教所召開大會誹謗、誣陷法輪大法師父，牛蘭雲站起來大聲制止，被護衛隊男警察一腳踹在地上，把她關入禁閉室，五個吸毒犯對她拳打腳踢。

二零零五年春天，牛蘭雲因講真相遭人惡告被警察綁架，被非法關押到大同市第二看守所，期間遭野蠻灌食。半月後她被劫持到太原新店勞教所迫害。牛蘭雲絕食抵制迫害，遭到野蠻灌食，灌濃鹽水、灌不明藥物，每天強行輸液。有一次，五個犯人強行對牛蘭雲進行灌食，摁的她喘不過氣。就這樣牛蘭雲被迫害了七十天，體重只有七十斤，送醫檢查說有生命危險，勞教所才通知家人接她回家。三十天後，勞教所人又夥同新華街派出所，再將牛蘭雲綁架，再次對她進行四十天的非人折磨，才讓家人接回家。
