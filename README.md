### 黑龙江依兰县公检法人员遭恶报录

【明慧网二零二二年七月二十七日】（明慧网通讯员黑龙江报道）黑龙江省依兰县的一些公检法人员积极充当中共帮凶，对众多法轮功学员进行绑架、关押构陷、劳教、诬判，造成很多家庭妻离子散、家破人亡。那些积极参与迫害法轮功学员的依兰县公检法人员，后来很多人都不同程度的遭到了恶报。
以下是遭到恶报的部份依兰县公检法人员：

▼韩云杰，原依兰县政保科副科长，曾毒打折磨法轮功学员张敏，致使张敏含冤离世。后来韩云杰突发冠心病暴死。

▼张英铎，原依兰县公安局国保大队长，经常对法轮功学员进行绑架、抄家、谩骂、毒打、骚扰、恐吓、敲诈钱财，连十几岁的小女孩都野蛮毒打，还扬言要把法轮功学员全整死。结果张英铎作恶殃及家人，其十八岁的儿子得鼠疫死亡，半年之后父亲也病死，最后他自己也锒铛入狱，被判刑一年半。

▼杨维兴，依兰县公安局国保大队长，二零一九年十一月绑架七十多岁的法轮功学员周殿文，致使其被非法判刑三年；二零一九年十二月七日伙同七台河市警察欲绑架法轮功学员李艳杰、宫凤强夫妇，迫使李艳杰坠楼身亡。不久，杨维兴于二零二零年春得了脑溢血，断送了前程。

▼刘丹阳，原依兰县国保大队警察，主动参与迫害法轮功学员，致使十几位法轮功学员被非法刑拘、劳教、判刑。在刘丹阳的构陷下，法轮功学员李军被诬判三年。刘丹阳二零一零年五月十四日死于肝癌，时年五十一岁。

▼李柏和，依兰县公安局副局长，积极参与迫害法轮功学员。李柏和二零一九年四月被捕入狱，丢官免职。

▼李树山，原依兰县珠山乡派出所所长，二零零零年二月把十四名法轮功学员绑架到乡政府，用最下流的语言侮辱学员两个多小时，流氓式的语言不堪入耳，并不断叫嚣：“我就骂你们师父了，能怎么的？我怎么没遭报啊？”不久，李树山鬼使神差的在派出所办公室上吊自杀。

▼张宪春，原依兰县东城派出所所长，外号虎张子，多次参与迫害法轮功学员。二零零七年九月二十一日，他怀疑某居民家是法轮功资料点，带人非法搜查，之后还去该居民的亲属家抄家、抓人，把人家吓得住进医院，他还追到医院威胁、恐吓。二零一三年三月二十九日，他带人绑架法轮功学员姜连英，致使姜连英被诬判三年。张宪春后来任依兰县公安局巡警大队大队长。二零一六年八月三十一日，张宪春在押送卖淫女途中，在警车内强奸卖淫女，而后锒铛入狱。

▼魏占全，原依兰县团山子乡派出所副所长。二零零九年八月二十三日晚，魏占全带领亲属周美良开车到百兴村欲绑架一位法轮功学员，绑架未遂，周美良于途中因车祸身亡。魏占全为此赔偿死者家属三十五万元。二零一一年十月二十九日，法轮功学员李慧慧、李欣欣被绑架到公安局，魏占全把李欣欣的手铐在背后，打她嘴巴子，还叫嚣：我就迫害好人，我就打你啦。二零一六年八月上旬，魏占全被调到东城派出所任副所长，还没上任就突发脑溢血，花了很多钱也没保住性命。魏占全下葬的头一天，魏占全的妻子和亲属又出车祸。

▼崔国栋，原依兰县道台桥镇派出所所长，多次到法轮功学员家中骚扰、恐吓、搜查、绑架，曾多次迫害锅炉厂退休工人张可明及其家人，导致张可明含冤惨死。二零一八年十一月，崔国栋因牵扯涉黑案件被开除公职，逮捕入狱。

▼高振伦，依兰县法院法警队长。二零一三年八月二十一日，依兰县法院对法轮功学员非法庭审时，因代理律师不接受法警“搜包儿”的无理要求，高振伦凶相毕露，冲出安检门挥拳要打律师。二零一九年三月，高振伦被省专案组拘捕。

▼张广志，依兰县检察院副检察长，是构陷法轮功学员的幕后主导，一直操纵关押、批捕、追诉法轮功学员。他甚至直接和公诉科长一起当公诉人上庭，和法官串通一气制造一桩桩冤案，把很多法轮功学员诬判入狱。二零一九年四月八日，张广志因牵扯到一起二十年前的涉黑案件，被省专案组拘捕。

▼姜恩有，原依兰县检察院批捕科科长，曾非法批捕法轮功学员陈继忠，致使陈继忠在监狱饱受摧残，其老伴儿因无人照顾冻死在野地里。后姜恩有遭恶报，喝酒摔倒在路旁，脑袋磕在石头上死亡。

▼史锦田，原依兰县法院副院长，多年来积极参与迫害法轮功，多次将法轮功学员判处重刑。二零一零年，史锦田得结肠癌。

▼翟英凯，依兰县检察院原侦查监督科科长；陈玉杰，依兰县检察院原公诉科科长；张安克，依兰县法院原刑庭庭长；这三人多年来非法批捕、枉判法轮功学员数十人。二零一八年十一月，三人因以前充当黑恶势力保护伞事情败露，被纪检监察机关立案审查。

▼李文学，依兰县公安局刑侦大队队长，二零一三年三月二十九日依兰县数十名法轮功学员遭绑架后，李文学带领多名警察对法轮功学员进行刑讯逼供、酷刑折磨，后来有十三名法轮功学员被依兰县法院非法判刑。李文学已遭报被判刑。

#### 依兰县先后四任县领导遭到恶报：

▼赵长满，原依兰县委书记，自二零一二年上任以来，在他的主导下，依兰县公安局于二零一三年三月二十九日在同一时间出动大批警察，对数十名法轮功学员疯狂大抓捕，并进行刑讯逼供、酷刑折磨，后来有十三名法轮功学员被依兰县法院非法判刑，连白发苍苍的老太太都被冤判十三年。赵长满于二零一六年七月一日被逮捕，后被判刑十七年。

▼赵欣，原依兰县委副书记。在二零零二年任职期间对法轮功学员进行疯狂迫害，数十名法轮功学员被抓到看守所，十六人被劳教。其中法轮功学员宋瑞义在看守所遭残酷毒打，备受摧残，不久就含冤离世。赵欣调到通河县任县委书记后，仍继续参与迫害法轮功学员。赵欣于二零一五年十月二十三日被开除公职，后被判刑十余年。

▼王春生，原依兰县委书记。在依兰县任职期间，指挥非法抓捕判刑十余名法轮功学员；他调到哈尔滨任南岗区委书记后，继续参与迫害法轮功学员。二零一七年，王春生因贪腐、强奸等罪被开除公职，逮捕入狱。

▼封殿辉，原依兰县副县长、通河县政法委书记、延寿县委书记。在任期间，积极迫害法轮功学员，致使多名法轮功学员被非法判刑，家破人亡。封殿辉于二零一八年八月，因受贿罪被判有期徒刑十年六个月。
