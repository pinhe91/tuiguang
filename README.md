### 湖南怀化市警察8月29日抄家绑架多名法轮功学员

【明慧网二零二二年九月二十一日】（明慧网通讯员湖南报道）据悉，八月二十九日上午，在怀化市国保大队副大队长张华的带领下，怀化市警察绑架了至少28名法轮功学员，现在大部分已放回了家，只有蒋仲会一人现在被关在看守所。
二零二二年八月二十九日早上六点多钟，怀化市府星路一住宅楼下面来了八个穿便衣的陌生人，其中一个女人。上到四楼一家法轮功学员的家门口，他们喊门，说是修空调的。这时，正好本楼一名住户的儿子从外面回来了，看到了这伙不速之客。这名住户的儿子上到七楼，这伙人喊不开四楼的家门（防盗门），就拿工具撬防盗门，因为他家的门晚上睡觉时上了反锁，他们怎么撬也撬不开，只把防盗门的猫眼撬烂了。撬了一阵门没撬开，就离开了一会儿。法轮功学员也离开了家。

后来，这伙人一直在楼下守着。中午，他们都回去吃中饭了，只留了一人守在楼下。下午，他们开来一辆大警车，在法轮功学员没有人在家的情况下，他们撬烂了防盗门锁，把家中的电脑、打印机、刻录机、电视接收器、打印纸、《九评》书，所有的大法资料，全部抢劫一空。这伙人把这家的防盗门重新换了门锁，同修因此而流离失所了。
二零二二年八月二十九日上午，法轮功学员蒋仲会回到他租住的房子，刚来到门口开门，一伙湖天派出所便衣警察扑上去绑架了他。他们把房间里的打印纸、电脑等抢走。蒋仲会原是市邮政局的一名科级干部，由于坚持修炼大法，被单位非法开除。

这次事件是由怀化市国保大队副大队长张华，带领广州铁路公安处（因为怀化铁路归广州铁路局管）、怀化铁路公安分局的人对怀化市法轮功学员同一时间进行绑架和抄家，到每个法轮功学员家去抄家的警察少则七、八个人一伙，多则二十多人，抄走了大法书和资料。

二十多年来，怀化市国保大队大队长鲁涛、副大队长张华一直是迫害法轮功的急先锋。从二零一九年以来，迫害怀化市法轮功学员的几起案子都是由怀化国保大队送到临近的芷江县法院、芷江县检察院去进行下一步迫害程序。
