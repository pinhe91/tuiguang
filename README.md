### 医院判了她死刑　如今她能爬长城

文: 中国湖北省大法弟子　竹梅 

【明慧网二零二二年九月二十日】二零零五的一天，我正在上班，突然接到妹妹打来的电话，说二姐脑血管堵塞正在医院抢救，叫我赶快过去。我跟领导请了假迅速赶到医院。
一進病房，看到二姐躺在病床上，我就过去俯到她耳边轻声叫她：“你知道我是谁吗？”她说知道。我说：“知道就好。你要记住我告诉你的那句话：诚心敬念‘法轮大法好，真善忍好’。现在只有大法师父能救你。你要不停的念。”她说：“我会的。”

我每天白天上班，晩上就到医院去照顾二姐，也给她送去了大法真相护身符。
一周过去了，二姐的病情没什么好转，姐夫就不想给她治疗了，因家里穷，本来就没有钱交住院费，又听医生说救过来也没什么用，废人一个，他就更不想给二姐治了，说：“还不如不救，就让她这样走了还好点。”我们娘家人坚决不同意。

二姐只有四十多岁，想到她从小就苦，婚后的这个家这么穷，没过过一天好日子。她的两个儿子都没结婚，都在外面打工，就这样放弃吗？

这时二姐夫给两儿子打电话，让他俩赶快回来看一眼。两个外甥打电话给我，我说：“你们现在安心工作，一切听我的。你妈真的不行了我会通知你们的。”话是这么说，当我放下电话，看到二姐放在我家的衣服，我忍不住也哭了。

医院下了病危通知，说要么接回家，要么就转院。我说：“转到武汉去吧。”医生说，人很难坚持到武汉，会在半路上去世。要转武汉，就必须要家属签字，出现这种情况与医院无关。我让姐夫签字，他不签，我说：“那我来签，不会有事的。”

我立刻找朋友联系了救护车，姐夫的熟人在武汉已联系好了医院，我们立刻上路了。车上有我三姐、妹妹和我妻子，还有二姐夫。我对大家说：“咱们都在心里敬念‘法轮大法好，真善忍好’，求大法师父保护我们平安到达。”他们都说好。我又和二姐说：“你心里一定要敬念‘法轮大法好，真善忍好’，大法师父会救你的。”

两个小时的路程，一路上真是提心吊胆。到达省人民医院后，看到病人非常多。尽管已经与医院约好做手术，但也要等两天才能做。留下我妻子照顾二姐，其他人都回来了。

妻子每天多次提醒二姐念“法轮大法好，真善忍好”，二姐也在诚心敬念，不管病房别人怎么想、怎么说，她就尽量出声念。

轮到二姐做手术了。手术从早上做到下午，前后八个小时。手术非常成功！这是她相信大法好，一直不断的念“法轮大法好”，大法给她的福报！出院后只是有一只眼睛的眼皮下坠，遮住眼睛看不见东西了。医生说不知道会不会好。

我告诉二姐：坚持念九字真言，会好的！过了两年，她的眼睛完全好了，只是手脚不太灵活。又过了几年，生活能完全自理了。如今她一直坚持念“九字真言”，她知道她的命是法轮功师父给的。有时她也告诉别人自己的经历，以此证实大法好。

十七年过去了，被医院判了死刑的二姐如今还在。前年和我一起去北京旅游，还爬上了长城。他小儿子说她：“您活的有意思，还爬上了长城。”

我们看到长城上有好多人出现了体力不支或其它不适状态，二姐爬了好多层没有任何不良反应。这是大法在人间创造的又一奇迹，也是她明白大法真相、真心相信大法好得到的福报。

如今二姐家儿孙满堂。她常年跟着小儿子在河北沧州生活，今年还出去打工给别人做饭去了，月入两千多元。

人真心敬重法轮大法就会给自己带来平安与幸福。希望世人能认清中共的邪恶本性，邪党对法轮功的一切报导都是造假诬陷，那是在毁灭人类最本质的东西，真、善、忍是普世价值，是宇宙的法则！如今法轮功洪传世界一百多个国家和地区，得到世界人民的高度赞扬。

望国人千万不要相信邪党对大法和大法弟子的造谣诬陷，相信大法好，才会有美好的未来。
