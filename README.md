### 广东茂名市八旬廖玉英多次受到派出所绑架

【明慧网二零二二年五月十六日】（明慧网通讯员广东报道）二零二二年四月二十一日上午十点左右，茂名市电白区林头镇政府干部、林头派出所多个警察命令大衙居委会人员陈宗志、卢其存等四人带路，有的穿警服，有的没穿，还有几个女人，一行十几个人（门外还有十几人）闯到林头镇大衙社区八十四岁的廖玉英家围捕抢劫。
廖玉英老奶奶不开门，警察用钳子撬开锁，疯狂私闯民宅，在没有任何法律手续的情况下，象强盗一样乱翻，抢走了所有的大法书籍和大法师父的法像，她身上买菜剩下的真相币也被抢走。

警察绑架了廖玉英及在她家中做工的法轮功学员麦伟莲，还有顺便来坐坐的邓玉莲和不修炼的老奶奶。当时的情景犹如土匪下山，让周围邻居咋舌，不可思议。警察把她们四个带走后，还换上了新锁锁住了大门。

到了下午五点左右，居委会一个人又带穿着警服的一个人来抄家，所有家里角落都被翻透，所有与大法有关的物品都洗劫一空，连墙上挂的福字，都撕走，廖玉英炼功穿的黄色衣服，警察都抢走。据邻居们说，这帮人提了好几袋东西和一箱子出来，没有给家属和当事人抄家清单。当时很多常人围观评论：“这帮人真是强盗，一定要控告他们！”

当天晚上，廖玉英和常人老奶奶被放回家。

廖玉英，汉族，一九三八年九月出生。修炼前，她患胃痛二十多年，吃什么东西都很小心，很多东西都不能吃，经常胃痛难忍，备受煎熬。经多方治疗，吃了很多药都无济于事。一九九八年，她听人说，法轮功能治病，她就去试一试。炼功后，开始身体非常舒服，精神抖擞的，每天满心欢喜，胃痛顽症什么时候好的都忘了。廖玉英有缘喜得法轮大法后，二十四年来，她心身得到净化，身体健康，心情开朗，没有上一次医院，没有吃一颗药，亲身感受到无病一身轻。家庭也很和睦，儿孙媳妇都很孝顺，令左右邻居赞赏。

可是，在一九九九年七月二十日，江泽民开始迫害法轮功后，廖玉英遭受了多次的绑架、劳教、关洗脑班，经常受到派出所、居委等人员的骚扰等迫害，让廖玉英和家人家无安宁之日，家人对廖玉英生活安全格外焦心。

二零零零年六月，受恩于法轮功的廖玉英，为了为法轮功说句公道话，到北京上访，告诉政府：法轮大法是正法，被北京公安绑架之后，电白610人员接回茂名后，被非法关押在电白第二看守所十五天。

二零零零年十一月，廖玉英第二次到北京上访，履行公民的合法权利，被警察绑架，被电白610接回来之后，被非法劳教二年。廖玉英被非法关押在电白第二看守所十一个月，送三水劳教所迫害。在劳教所里，警察强迫廖玉英看诬蔑法轮功的电视，廖玉英不看，遭到警察和劳教人员的虐待，身体和心灵受到巨大创伤，血压高到220，生命随时会出现危险。劳教所怕出人命，打电话给她儿子，才把非法关押在劳教所四个月的廖玉英接回家。

二零零三年夏天，廖玉英被警察、居委等人员绑架到茂名市洗脑班，非法关押一周后被放回。

二零零四年，廖玉英在电白观珠讲法轮功被迫害的真相，被不明真相的人举报，被绑架到观珠派出所，从中午关到晚上，亲人去要人才被保释出来。

二零一七年十一月十三日中午12时，茂名市电白区大衙派出所副所长吴波带着四名警察到廖玉英家敲门骚扰，十多分钟后见没人开门，警察们便拿来梯子在拆廖玉英家门口的真相对联，非法拆下后便肆意撕烂，这时廖玉英出来阻止也不停止。当时围观了许多民众，周围邻居纷纷出来指责一群警察欺负一个老太太。一个邻居对警察说：你们为什么又来找她麻烦，她是我们这里最好的人。还有的说：人家家贴对联也没有犯你什么呀！警察便说对联反党。

二零一九年七月九日上午九点多，电白区国保局陆尚辉（当时一便衣指着他，误导大家说：这位是陈主任，就是陈昌兴（610的头目））在没有任何原因的情况下，带领十几个人，其中有几个穿警服，也有些是便衣，气势汹汹地闯进廖玉英老人家里。当时，她家有一位帮她搞卫生的学员麦伟莲与另一位去帮她提水的学员黄英莲。陆尚辉没有出示任何证件，就指使手下楼上楼下四处乱翻，其中一个便衣一脚踹开房门，把廖玉英私人物品：几十本法轮功书籍全部抢走；这时刚好崔芬来探望廖婆婆，他们也顺便把她绑架走。

七月十一号，廖玉英与黄英莲去国保局找陆尚辉要回自己的私人物品，黄英莲刚说：领导，我有点事想找你谈谈……还没等她说完，陆就暴跳如雷，骂骂咧咧地驱赶她们，后来还威胁说：你们若再不离开，我马上去拿手铐过来……

二零二一年八月十二日上午十点多，廖玉英讲真相救人，被电白区那霍镇派出所恶警绑架。下午五点多，林头镇大衙派出所几个恶警去廖玉英家非法搜查，没有任何手续，抢走了大法师父法像、几本旧的《明慧周刊》。晚上九点多，那霍派出所通知廖玉英的家人接她回家。

二零二二年四月二十一日，电白区林头镇政府干部、林头派出所多个警察，一行十几个人又非法闯到廖玉英家抢劫绑架。
