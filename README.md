### 哥哥诚念大法好　绝症无踪影

文: 大陆大法弟子（本人口诉，同修整理） 

【明慧网二零二二年七月二日】我是一九九八年得法的大法弟子，在这二十多年的正法修炼中，经历了很多大法神奇的事。现仅举几例发生在身边亲人身上的事情，见证大法的超常和师恩浩荡。

#### 一、哥哥诚念大法好，绝症无踪影

我哥哥得了淋巴癌，腋下长了三个鹅蛋大小的肿瘤，癌细胞全身扩散，肝、肺、骨关节、血液等到处都是，医院大夫说，他只能活两个月，让家里人准备后事。

我对哥哥说：“只有大法能救你，你诚心敬念‘法轮大法好、真善忍好’吧。你信不信你妹妹说的话？”哥哥说：“我相信你说的话。”然后他就诚心念个不停。用他的话讲：“我天天念，每天只要一清醒我就念，白天走在大街上我就大声念，我怕师父听不见。”几天后，他的病症就消失了，脸色也变的白里透红，亲戚邻居都觉的大法太神奇了。

哥哥的腿不好，我一提醒他，他就赶紧念：“法轮大法好，真善忍好。”马上腿就好了点。有一次他腿肿得很厉害，拄着棍子来我家，我就领他去了医院。医生问他以前得过什么病？陪他同去的侄女说：“十二年前得过淋巴癌晚期。”医生问：“怎么好的？”我说：念“法轮大法好、真善忍好”念好的。医生说：“那怎么不接着念？”哥哥听了马上就开始念。当天他从医院回来就好多了。哥哥回家后，在农村小医院住了两天就全好了。

哥哥经常高兴的说：“我知足了，我们家族里谁也没有我这么长寿的。”

#### 二、信师信法，孙女化险为夷

我家大孙女爽爽六岁那年，有一天从幼儿园接她回家，爽爽走的快，先上了六楼，我刚上了五楼。爽爽爬在六楼的栏杆上喊：“奶奶！奶奶！”我一抬头，就看见孩子从六楼一头栽倒到五楼楼梯台阶上。我当时心没动，大声说：“法轮大法好！真善忍好！咱们有师父管，什么事也没有！”

爽爽哭着自己爬起来。回家后我一检查，孩子身上什么伤也没有，只是肋骨那位置上有一点小红印，因为当时那个部位正好磕在水泥台阶头的棱角上。孩子的爷爷知道后吓坏了，晚上一遍一遍的到孩子的屋里看。我也有些后怕，但转念一想：有师在有法在，没事。

第二天起床一看，孩子真的什么事也没有，又活蹦乱跳的去幼儿园了。

#### 三、弟子念正，师父做主

我儿媳是带着节育环意外怀上小孙女的。到医院检查，医生说：孩子千万不能留下来，因为节育环没掉，将来还不定长到哪里去了，孩子生下来十有八九是个废孩子。叫儿媳做手术。我的心乱了，生气的指责儿媳：都修炼了还能闹这事，这可怎么办？儿媳也没了主意，想第二天去做手术。

我在回家的路上心里翻腾，突然想到：自己和儿媳都是修炼人，这不是杀生吗？孩子不能做掉啊。拿起电话找儿媳，打了好几个电话，儿媳才接到。儿媳听了我的意见挺高兴，这才告诉我：“昨天晚上你儿子还说：这孩子怎么都得留下来，这孩子不是一般的孩子，带着环还能有，说什么都得留下来。”

儿媳怀的孩子刚到八个月就要生了。到了医院，医生还告诉我和儿子：要做好思想准备，这孩子生下来还不定什么样。我当时就否定了：“没有事！我家的孩子一点事都没有。”医生还说：“不管你信什么，你们要做好思想准备。”

结果小孙女生下来什么事都没有，很健康，就是有点小，才四斤六两。可现在长的又高又壮。

像这类事情，还有发生在我自己身上的神奇经历还很多，千言万语道不尽对恩师的感激。唯有不断精進实修，修好自己，抓紧讲真相，多救人，完成史前大愿，回报恩师的慈悲救度。
