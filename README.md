### 正義的鄰居大媽得福報

【明慧網二零二二年四月二十一日】

#### 正義的鄰居大媽得福報

〔中國大陸來稿〕迫害初期的一天，縣「610」人員和公安局警察闖到我家騷擾，當時我沒在家，他們在我家門口打聽我的下落，鄰居大媽走過去說：「你們抓煉法輪功幹啥？煉法輪功的人心眼都好使，都是大好人，不做壞事，你們抓她們幹啥？」警察和「610」的人只好離開了。

二零零九年，市邪黨監獄強制「轉化」法輪功學員，一位大法弟子因為拒絕轉化，被監獄迫害致死，遺體滿身傷痕，監獄卻說是「正常死亡」。於是大法弟子的家人在社會上呼籲民眾伸張正義。鄰居大媽聽說後，一家人都主動簽名，譴責監獄的暴行，伸張正義。

二零一七年，我丈夫因發真相資料遭人惡告被警察綁架，鄰居大媽全家主動在要求放人的徵簽表上簽字，還跟我說：「把我也拉上，去找他們要人。」

鄰居大媽的正義之舉令她這些年福報連連：

鄰居大媽家那些年由於家庭困難，小兒子還有癲癇病，她家兩個兒子都快三十歲了還沒有娶上媳婦，為此鄰居大媽很著急。但是自從她敢為大法弟子說公道話後，兩個兒子先後都娶了媳婦，還生了兩個健康活活潑的孫子。

一天，鄰居大媽和丈夫、二兒子在租的稻田裏扒舊房子，她丈夫在房子裏面，她和二兒子在外面扒，突然房子的牆倒塌了，把她丈夫扣在裏面。房蓋是木構架，牆是泥砌的，上面還有三層塑料布，她和兒子在塑料上來回跑，她喊著丈夫的名字，兒子喊爸爸，但她丈夫沒有一點回應。鄰居大媽突然想起我告訴她誠念九字真言能遇難呈祥，於是她坐下來就開始念誦：「法輪大法好！真善忍好！」求大法師父救她丈夫。她兒子還埋怨說：「不趕緊找我爸爸，你坐那還念上了。」這時她聽到她丈夫在她旁邊的塑料布下「哼」了一聲，她趕緊喊兒子：「過來，你爸在這呢！」她兒子輕易就用手摳破了三層塑料布（平時別說三層，一層也不好捅破的），看到她丈夫被一大塊石頭和大木頭卡在中間，一點也沒傷到身體。鄰居大媽回村後跟我說，感謝李大師保護了她丈夫，全家人都對大法和大法師父感恩不盡。那年的傳統節日，她還讓我幫她給大法師父發賀詞：恭祝李大師節日快樂！謝謝李大師護佑！

#### 哥嫂明白了善惡有報是天理

〔中國大陸來稿〕我和丈夫修煉法輪大法後，經常給哥嫂講關於法輪大法好、三尺頭上有神靈、善惡有報的真實故事。哥嫂都很愛聽，也都認同。迫害開始那些年，警察經常到我村騷擾綁架大法弟子，我哥在街上當著眾人面就說：「煉法輪功的個個都是好人，大街上看到錢都不撿，貪污腐敗幹壞事的都是共產黨員。」

儘管在物慾橫流、拜金思想盛行、好壞分不清的大陸社會環境下，他們遇到的事情也能反思自己。

一天，哥哥開車和幾個朋友到市裏逛百貨大樓，朋友們都在挑褲子，他當時不想買，沒事湊熱鬧，也拽了一條看，看著看著由於人多，就給他擠到外邊去了，他一看沒人看見他，就拿著褲子出了大樓，開車就走了。可是車開不遠，他就把一私家車給刮壞了，給人家修車花了八百多元，而那條褲子才八十多元錢，他明白了善惡有報的天理是真的存在。我對嫂子說：「這是師父在啟悟他的良知呢，人一旦明白真相，知道了天理，再做壞事，就是知法犯法了。」

嫂子過去也是一個很愛財的人。有一年過年去買年貨，她身上帶一百元錢，買回來很多東西，回來後一數錢，除了一百元錢外，還被多找回幾十元。那時她還覺得自己有本事呢，還走哪說哪，我們給她講了「有失有得」的法理後，她也開始約束自己的行為了。

一天，嫂子到我們家說：「你們倆得夸夸我。」原來她那天趕集，一個賣貨人給顧客找錢，從兜裏掏錢時，一捲錢被帶出來掉到嫂子身邊，賣貨人沒察覺。這若在過去，嫂子順手就當作自己的錢拾起來，可這次，她想起了我和丈夫給她講的失與得的關係，她馬上告訴了賣貨的人：你的錢掉了。賣貨的人高興地說：謝謝你。嫂子為自己放下利益之心很高興。

那年十月份，嫂子坐她姑爺開的大車收糧，進村時車開得快，農村的道路很窄，前面還有一輛小轎車，在一個賣店的道路對面停了一輛小轎車，可她們前面的小轎車突然和道邊的車並列停下了，兩輛車把路堵了個嚴嚴實實。嫂子的車剎車已經來不及了，直奔路邊那輛空車撞去，把那空車撞飛貼到賣店的牆上滑落到地上，嫂子的車也翻向賣店的二樓的外樓梯上，坐在副駕駛位子上的嫂子被慣力從車前玻璃窗拋了出來，掉到二樓梯下，外樓梯把車擋住了，車翻了但沒掉下來，要掉下來正好會軋在嫂子身上。她被摔得暈了過去，緩醒了一會，她聽到有一些人叫著她的名字說，過去看看，她可能摔死了，可嫂子慢慢的站了起來，頭髮裏都是玻璃碴子，臉上流血了，是被玻璃劃了一個口子。活動一下哪也不疼。她感覺手裏有東西，她一看是我給她的大法護身符，她當時想這真怪了，明明護身符是放在自己胳膊上的羽絨服的小兜裏了，怎麼在自己手裏攥著呢，她明白了，是大法師父保護了她，救了她。她一到村裏的小賣店裏就說，是法輪大法救了她，還把她姪兒媳婦領我家請護身符。

有一次在高速公路上，我哥開一小運輸車，拉一車牛，嫂子及兩朋友也在車上。上坡時，我哥開足馬力往上開，到坡上時發現停了很多車把路給擋住了，而我哥的車此時剎車失靈了，我哥心想：咋整？挑一個破車撞吧。情急中，嫂子大聲喊「法輪功！法輪功!」就在還有幾米的一瞬間，兩輛小轎車同時起動開走了，我哥的車從這兩輛車的空處開過去了。車上的人都說：「法輪功真好！這真神奇呀！」

#### 她遇到甚麼難事都念九字真言

〔中國大陸來稿〕一天傍晚，我家後院鄰居榮到我家說：「姑，我兒子早上頭疼發燒，上午上市裏醫院檢查沒檢查出病，我們就回來了，白天一天了還行。可這會孩子病情加重了，我們馬上要去市裏醫院，晚上我女兒一個人在家害怕，你上我家和我女兒做個伴兒吧。」我說：「快念法輪大法好，真善忍好，讓孩子也念，全家都念。」我就和她一起去了她家，又對她兒子說：「你念法輪大法好！真善忍好！」她兒子就開始念。她兒子已經三退了，他們當晚去了市裏醫院，醫院醫生診斷是腦炎，還很嚴重。他們全家就不停的念誦「法輪大法好，真善忍好」，她兒子的情況一天比一天好，七天就徹底好了，好的特別快。同病房有患者比他早住院半個月、病情沒他嚴重，還沒出院呢。榮說真是大法保護了她兒子。

一天，榮在家倒玉米棒子，一村婦急急的跑來說：榮，家村邊的那片玉米地正著火呢，已經快燒到你家地了。榮立刻在心裏求大法師父，讓火別燒自家玉米，並一直念「法輪大法好！真善忍好！」結果火燒到她家地邊就不再往前燒了。

從那以後，榮更相信法輪大法好了，她遇到甚麼難事都念九字真言，所以她得福報的事例很多，這裏就不一一說了。

一次，我和榮一同去辦事，她家親戚看到了開玩笑說她：「你跟法輪功混上了啊。」榮很自豪的說：「我們是一夥的。」

在我們村裏，現在老百姓也都在覺醒，知道法輪大法好，法輪功被迫害是冤枉的。很多民眾都認同念「法輪大法好！真善忍好！」也都知道了共產邪黨的偽善，用謊言欺騙，搞獨裁、暴政，都認識到共產邪黨不滅亡，老百姓就沒有好日子過。明白真相的百姓，盼望著法輪功早日平冤、昭雪，環境變了，她們也要修煉法輪功呢!
