### 古今高人指路避疫　逢凶化吉平安有道

文: 小玉 

【明慧網二零二二年九月二十八日】就像人控制不了天下雨一樣，人更控制不了瘟疫的來和去。然而，在歷次疫災中，都留下了瘟疫「繞著走」的奇蹟。

#### 重陽節登高避疫的啟示

九月九日，日月並陽古稱佳節，從漢到魏晉時代已有重九佳節登高的風俗。重九登高的源起和避瘟疫有關，南北朝吳均的《續齊諧記》記載了這個故事。

桓景是東漢汝南人。桓景隨著方士費長房遊學多年學習仙道。有一年晚秋時節費長房警告他說，九月九日汝南將出現瘟疫！費長房教給他避疫的方法：事先準備好茱萸絳囊，九月九日一早讓家中大大小小繫在手臂上，然後出門登高並飲菊花酒，可以祛禍。桓景牢記費長房的警告，到了九月九日大清早，他將準備好的茱萸絳囊給家人繫上，領著家人登高避疫。一家人直到太陽下山才返家，到家中一看，家禽、家畜果然如費長房所言都染疫暴斃，景象真讓人震驚。九月九日登高喝菊花酒，婦女在胳膊上繫茱萸囊辟邪去災的習俗也由此而來。
這個故事更深一層的意思在於，費長房是修道之人，能預知瘟疫的到來；桓景的家人和鄉親們相信了費長房的話才躲過了災禍。古代的人信神，相信有人能預知未來，這時最明智的做法就是聽從善意的提醒，順天而行。

#### 當今渡大劫、保平安的神跡

我是湖北一名法輪功學員。二零二零年一月，疫情在武漢爆發。當時快要過年了，我舅舅去鬧市區購物和探訪親友，回家之後感覺身體不適就到醫院去檢查，確診陽性。但那時醫院裏早已擠滿了染疫者，舅舅只得回家自行隔離。很快舅媽也被傳染上且發展成了重症。

以前，我曾經給舅舅、舅媽講過法輪功真相，舅舅非常接受，還看過《轉法輪》，雖然沒有走入修煉，但一直對大法有正面認識，所以舅舅染疫後一直都是輕症。而舅媽在當時不接受真相，還很排斥。她染疫後很快發展成重症，我知道一定與她對大法的態度有關。

舅媽的狀況很不好，舅舅就帶著她到處去看病。可那時公交車和地鐵都已停運，老兩口打車一個醫院一個醫院的跑，結果是所有的醫院都人滿為患，連掛個號都沒希望。老兩口只好回家。舅媽臥床不起，高燒、呼吸困難、不能進食……舅舅急得像熱鍋上的螞蟻，不停地撥打市長熱線、110、120等電話求助，沒有人管，直到在床上躺了八天之後，才有一家郊區的小醫院接收了舅媽。此時的舅媽身體已經極度虛弱。

拍片顯示她的肺部大部份變白，因為過度緊張和害怕，她又患上了抑鬱症，幾天幾夜不能睡覺，真是雪上加霜！醫院幾次下了病危通知單，並且一再讓舅舅做好心理準備，說舅媽可能挺不過去。

當我知道這些情況的時候，舅媽已經住院半個多月了，舅舅已住進方艙醫院。我在電話中安慰舅舅不要擔心，接著準備在電話中再次給舅媽講真相，希望她能轉變。

電話接通後，傳來了舅媽微弱的聲音，我讓舅媽少說話，保存體力，聽我講。接著我進一步詳細地講述了法輪大法的真相和自己親身受益的體會，講解了為甚麼誠念 「法輪大法好 ，真善忍好」這九字真言會有祛病健身的奇效，並且讓她馬上做「三退」（退出黨、團、隊）。最後，我真誠地對舅媽說：「人的生命和健康是最重要的，您就按照我說的，試一試吧！」

這一次我懷著純淨的心態和真心希望她得救的心，效果非常好，舅媽接受了真相，並且答應我一定誠念九字真言。

此後，我常給舅媽打電話，了解她的情況，舅媽的身體一天比一天好，先是退了燒，接著可以進食了，後來又拔掉了輸氧管，還能圍著病床走幾圈了。一個月後舅媽康復出院了！

回家後，舅媽來電話說，她原來不但不接受我給講真相，還很反感，而這一次她真心接受了，沒想到就過了這一大難！我對舅媽說：「這不怪你，是我沒講清楚。但你能在緊要關頭選擇相信大法，說明你和大法是有緣的。」我一再囑咐舅媽，平時要多念九字真言，舅媽連連說「好好」。

當今社會，除了瘟疫侵擾，還有猝不及防的海嘯、地震、洪災、空難等天災人禍。天垂異象，大劫將至。如何渡大劫、保平安？誠念「法輪大法好，真善忍好」和「三退」就是渡過大劫的法寶。
