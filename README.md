### 北京富豪听《为什么会有人类》 感觉找到了家

文: 碧潭 更新: 2023年03月05日

【明慧网二零二三年三月五日】龙哥请朋友们吃饭。席间有一位是法轮功修炼者阿峰（化名），龙哥好奇，就与阿峰攀谈起来。龙哥问：“最近北京两会，到处风声鹤唳，共产党最怕法轮功，您最近忙什么哪？”阿峰说：“最近在学师父的新经文《为什么会有人类》，您想听听吗？不过原文没带，就给您背诵，行吗？”龙哥很高兴说：“谢谢大法师父赏我一个耳音，您背吧，我听着。”说罢放下手中的筷子，像个乖孩子似的，静静的听。
背诵结束后，阿峰结合现在瘟疫流行告诉龙哥，瘟疫专收邪党份子，还讲了天安门自焚伪案是共产党江氏集团栽赃陷害法轮功。讲到贵州藏字石，龙哥拿出手机当场搜索。看到真相，龙哥很震惊，边看边说：“门票写着‘中国共产党亡'，共产党迫害法轮功二十多年了，您吃了不少苦吧？”

阿峰淡然一笑说：“法轮大法洪传世界，只有共产党不让炼，还残酷迫害法轮功，活摘大法弟子器官……”

龙哥抢话说：“对不起打断一下，江西胡鑫宇就是为给共产党官员延命被活摘了器官，这个我们网上都看到了。”又说：“那个传媒大学毕业的庞勋，也是被共产党警察活活打死的，法轮功学员竟是社会精英，我们冤枉法轮功了。我能做些什么？帮助你们就是帮助我们自己。”

阿峰说：“我们就是希望您知道真相得到平安，记住‘法轮大法好，真善忍好’这个九字真言，危难来时能保命。”龙哥涨红着脸说：“麻烦您教我念几遍，我要记住这救命的真言。”

聊天中，龙哥得知自己与阿峰同龄，但很明显阿峰看上去年轻很多，龙哥就问阿峰：为什么？阿峰坐到沙发上，演示起法轮大法的第五套功法（“神通加持法”）。龙哥目不转睛的看着，然后迫不及待的坐到旁边开始学炼。他说喜欢这套功法，看起来手印漂亮舒展。当搬上腿单盘时，龙哥感到一股热流通便全身。他很兴奋，连说谢谢阿峰。阿峰笑笑说：“谢谢大法师父吧，师尊管您了，给您好东西了，您的悟性真好。”

分手时，龙哥盛情邀请阿峰到自己家去玩儿，说不用预约，因为他觉的自己在人世中走丢了，听闻《为什么会有人类》后，感觉找到自己的家和亲人了。

（责任编辑：文谦）
