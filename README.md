### 因恐惧丧人伦　校长夫妇疯狂砸家、殴打母亲

【明慧网二零二二年六月八日】（明慧网通讯员天津报道）天津市武清区河西务镇法轮功学员武术敏，因坚持法轮大法真善忍信仰，多次遭到迫害。当地中共人员为了逼迫她在放弃信仰的保证书上签字，以威胁工作的流氓手段，挑唆她的儿子、儿媳上门砸家，并对母亲大打出手。
武术敏的儿子身为当地杨村某中学的校长，因为对中共的恐惧，丧失了为人、为子、为师的道德、人伦。

以下是武术敏女士近期遭迫害情况：

二零二零年腊月二十三（二零二零年一月十七日），武术敏因在河西务镇杨村街道泉达路上给民众讲真相、劝三退时遭人恶告，被警察绑架到泉兴路派出所。警察上门非法抄家，抄走一本《转法轮》、三个播放器、十几个卡、一个唱戏机、五十份真相资料、护身符等。当晚十点多，才让她儿子将她接回家。

腊月二十五（二零二零年一月十九日），泉兴路派出所警察打电话让武术敏的儿子和她娘家的人把她带到到派出所，对她实施非法拘留七天，让她签字，遭她拒绝。最后警察就让她儿子签字，她儿子怕失去工作，被迫签字。警察还开了罚单，具体数字待查。

正月初二（二零二零年一月二十六日），武术敏回老家后，当地中共人员不断地给她儿子施加压力。她儿子是杨村一所中学的校长，中共人员就以失去工作、影响前途威胁、恐吓他，导致她儿子几次专门回家逼迫母亲签放弃修炼法轮功的保证书，遭到武术敏拒绝，其子魔性大发，连骂带砸，把母亲家里的玻璃、电动车都砸了，连切刀都不见了。武术敏只好报警，让警察制止其子的闹剧。

但事情还没完，武术敏的儿媳当晚特意回婆家，逼迫婆婆写保证书，遭到武术敏的拒绝，儿媳竟恼羞成怒，出手打婆婆，用小木桌的抽屉打婆婆的脑袋，当时就起了一个大包，又用扫帚打婆婆的肩膀，打得一片青紫，还用椅子往婆婆身上砸，小腿起了一个筋包（半年多才下去）；这个儿媳还不解气，把婆婆里屋的玻璃砸了，穿衣镜也砸了，手机也砸碎了，并且扬言：打死婆婆算她自杀，打残了送精神病院。

当地政府对这种丧失道德人伦的行为毫无反应，但对逼迫一位老人签字却非常“认真”、“尽责”。二零二一年八月三十日，县一级、乡一级的政法委人员在村书记带路下，又来逼武术敏在什么表格上签字，最后也没得逞；他们抄走武术敏的一本《转法轮》和一个播放器，算是交差。九月十二日，乡派出所两个警察又上门问话，也没达到目的，走了。

一级一级的中共官员，就是这样为了自己的官位，无知地、机械地对善良的百姓实施着迫害。因为无神论的毒害，他们忘了善恶有报的天理，谁造下的罪恶，谁终究要还的。
