### 济南八旬老人法庭上质问：按真善忍做人有什么罪？

【明慧网二零二二年七月三十一日】（明慧网通讯员山东报道）七月二十六日，济南市八旬法轮功学员刘春平被章丘法院非法开庭。她说：“按真、善、忍做好人有什么罪？我破坏了哪条法律实施了？”包括主审法官在内的人没一个搭话。刘春平拒绝签字，现已回到家中。
刘春平今年八十二岁，家住在济南市历下区菜市新村。她修炼法轮大法后，按照真、善、忍做好人，身体健康。她深知法轮大法好，为了让百姓也明白法轮大法好，二零二一年十月，时年八十一岁的刘春平老人在东边华龙路的小区内赠送法轮功真相册子，被东风派出所警察绑架，公检法人员捏造罪证，将她构陷到章丘区法院。

在二零二二年七月二十六日上午九点半，刘春平被章丘法院非法开庭，主审法官宣读了他们捏造的所谓“罪证”，说，“刘春平利用×教组织宣传迷信，利用×教组织破坏法律实施罪”（注：中共是真正的邪教）。

刘春平说，“我今年八十二岁了，我没做过一件缺德事，我按真、善、忍做好人有什么罪？我破坏了哪条法律实施了？”他们没有一个搭话的。

她继续说，“人在做、天在看，善恶有报是天理”，等等。有个警察打断她的话说：“在你家翻了八十七本书，对吧？ ”刘春平老人说：“不知道。”法院人员让她签字，她说：“我不签。”主审法官说：“休庭。”

法院安排的律师对刘春平说：“你年龄大了，身体也好，走吧。”

整个过程，除了以上说话的外，没有人再说话。刘春平的两个儿子在场，也没说话。大约半小时完事了，刘春平回家了。

#### 善良老人传真相　被公检法构陷

二零二一年十月，刘春平老人在东边华龙路的小区内赠送法轮功真相册子，被不明真相的老太太用手机拍下，这个老太太拿着三本真相小册子恶意报告给东风派出所。东风派出所警察到刘春平家非法抄家，并绑架了她。刘春平一直给警察讲真相，当天回家。

二零二二年三月二十六日，派出所片警给刘春平送来一张“起诉告知书”，济南历城区检察院对刘春平非法起诉，后又送来三张“审查起诉期限告知书和传唤证”等，起诉期自二零二二年五月十七日至二零二二年六月十七日，告知她六月十七日到章丘检察院去。

二零二二年六月七日，章丘区检察院打电话，叫刘春平去。她的孩子带她去了。一进屋，相关人员就让她坐在带链子锁的椅子上，刘春平说：“我不是犯人，我是好人，我不坐。”相关人员让她坐在沙发上，拿出从她家非法抄的书等让她签字，还给她推荐“律师”让她请。刘春平坚持不请这些所谓“律师”（她知道这些“律师”是协同中共迫害法轮功、做所谓有罪辩护的）。

刘春平给他们讲了自己以前多种病，修炼法轮功后身体健康，二十六年没吃过一粒药，还讲了“四二五”上访和“自焚”真相。刘春平说：“天安门自焚中，王进东的衣服都烧烂了，可头发和塑料瓶却没事……”相关人员说：“法轮功是×教（注：中共是真正的邪教），别炼了。”刘春平问她说：“真、善、忍，邪在什么地方？我要是不炼功，早死了，活不到现在。”相关人员不吱声了。整个过程，刘春平的儿子都在门外，相关人员不让他进屋。

在二零零二年，刘春平因发真相资料，被非法判一年。二零二零年，她在菜市小区发真相册子，被东关派出所绑架抄家，刘春平一直给他们讲真相，当天回家。
