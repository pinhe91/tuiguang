### 曝光辽宁省女子监狱老残队的罪恶

文: 辽宁法轮功学员信淑华 

【明慧网二零二二年六月六日】二零一八年八月十六日，我在街上讲真相救人时，被辽宁省本溪市溪湖区彩屯派出所警察绑架，后被非法判刑三年三个月，被关押在辽宁省女子监狱老残队，十一监区三小队，遭到残酷的迫害。
狱警苏科长、蔡科长、谢队长、李队长指使犯人二十四时夹控折磨法轮功学员，不让盘腿、不让炼功、不让睡觉，还把腿搬下来往铁床上磕。

犯人李慧东、李秀兰不让我吃饭、不让喝水、不让睡觉、不让上厕所，打骂体罚是家常便饭。

二零二零年，狱警指使犯人孙丽芝、范桂芝，用塑料盆碎片来回拉我的脖子，不让我睡觉、不让上厕所，撕扯我的头发，把我的头发薅掉很多，这样折磨我三天三夜。

由于我不配合邪恶“转化”，我还遭酷刑“抻刑”折磨两个多月。

我被绑架时体重一百四十六斤，出狱时瘦到九十来斤，牙齿也脱落了，站也站不起来了，呈危病状态。

据犯人说，小号里装满了法轮功学员，不让躺着、不让睡觉、不让坐椅子，有的被扒光衣服、光着身子，还不断地打骂，每天惨叫声，斯心裂肺，狱警动不动就一群一群地拉出去强行按手印，逼写放弃修炼的“三书”。

听犯人说，几天前犯人毒打一法轮功学员，逼她写“三书”，打完了用棉被裹上后，坐在她身上，结果她被活活憋死了。还说保密不让问。

二零二一年四月初，犯人残酷迫害法轮功学员宋淑娟，宋淑娟被迫害致病危，送医抢救，回来几天就死在号里了。

犯人吴同、李晓芳、孙罗艳、张艳清、田桂菊、衣丙华、李慧东、刘庭丽、范淑珍、孙慧、于淑芹、王凤华、李秀兰、孙丽芝、信红、支照文等，这些打人凶手，心狠毒辣，轮番残害法轮功学员。

二零一八年至二零二一年，监狱里的犯人都在做工芝品。狱警无论是不是法轮功学员都一样对待，不打人不说话，疯狂得像猛兽，每天狂叫翻天，简直不是人样，人性全无呀，被关押的人都非常紧张，恐惧得不行。

狱中伙食极差，狱警食堂将用剩下的菜头、菜尾、菜帮给我们吃，白水煮汤，两勺汤只有两个菜叶，汤全是黑色的，发霉的玉米面儿，做出的饽饽，不熟直掉渣，一个星期三次馒头，硬得打死人，犯人都吃不饱饭。

狱警还在法轮功学员的菜饭里下毒药。上次我被非法关押在这里时，就被狱警灌过毒药，他们叫“废功药”，还分为一至八号，给我灌的是二、四、八号，下毒药的菜饭里都有味道。我曾听到两个狱警的对话，一狱警问：“废功药起什么作用呀？”灌药的狱警说：“用少了浑身无力，用多了失忆，再多就死了。”
