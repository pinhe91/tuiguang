### 父亲诚念“法轮大法好，真善忍好”得福报

文: 中国大陆大法弟子 

【明慧网二零二二年十二月八日】我父亲今年八十九岁，小时候书也念了很多，他说，他也和日本孩子在一起读过书，日本孩子和他们的父母都很善良，经常给他白米饭和西红柿等一些好吃的。那时候父亲根本吃不着白米饭。能在日本人家里吃顿白米饭，就象过年一样特别开心。因为父亲念了十三年书，在我们村算是最有文化的，所以村里人都很尊重他，一到过年都让我父亲给写对联，红白喜事时请他记礼账。
父亲的党文化也很强，又是个无神论者，总说世上没有神佛，我跟他讲真相也不听。我就想，要不给父亲安个新唐人电视看看。于是在二零一二年给父亲安上了新唐人电视。这一看不要紧，他离不开了。弟弟打工回来说：你就看这一个台不行，过年了文艺晚会看不见了。父亲说：那你就再买台电视放你的屋里。父亲特别爱看石涛讲故事，在看新唐人的节目中不断的被清洗党文化无神论观念，终于有一天明白了真相，退出中共邪党一切组织，天天诚念“法轮大法好，真善忍好”得了福报。

父亲平时一个人生活，有一天觉的肚子疼，疼得很厉害。邻居看见老人疼得痛苦的样子，要送他去医院，他说不去，说去医院得花好多钱。邻居说那就找你儿子、姑娘，他也不干，说：他们都很忙。我没有事，我有秘诀。邻居说你有什么秘诀？看你疼的满脸是汗。父亲说我保证不上医院，你们别在这看着我，看着我我更疼。邻居们就都走了。

老人以为是着凉了，就把电褥子插上趴在上面他就开始大声的念“法轮大法好”、“真善忍好”、“李大师好”、救救我。念着念着就睡着了。不知什么时候醒来感觉要上厕所，往尿桶里便尿就听尿桶里噼里啪啦的响。老人觉的奇怪，以前便尿也不是这样的呀！他仔细一看便出一些碎石头，大的、小的都有。

他把石头捡起来用水洗干净给邻居们看、给儿女们看，大家都见证了神奇。孙子从部队回来，他给孙子看，孙子问怎么排出来的，他说我就念“法轮大法好”、“真善忍好”排出来的。

有一次父亲过生日，亲朋好友都来给老人祝寿。要开饭了，老人说先别吃饭，我有几句话要说：今天我能和亲朋好友聚在一起多亏大法师父保护我，让我有个好身体，我才能活到现在，虽然我女儿学大法被关進了监狱，可我女儿身体好，我非常支持她修炼大法。今天在场的亲朋好友请您们都记住“法轮大法好，真善忍好”，我就信大法好。
