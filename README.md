### 诚念大法好　七天扔掉了轮椅、拐杖

【明慧网二零二二年十月二十六日】

#### 诚念大法好　七天扔掉了轮椅、拐杖
〖中国大陆来稿〗我的外甥，二十八岁。一天在高铁工地施工时，一个机器从高处掉下，正好砸到他的脚上，脚被砸扁了。医生说得截肢。外甥的脚、腿疼痛的连胳膊都不能动，连一张纸都拿不了，不能开电灯，打着激素，还疼的天天哭，晚上也不能睡觉。
法轮功学员告诉他念“法轮大法好”，外甥拄着双拐坐在轮椅上，就开始诚念“法轮大法好”，这样他没有象以前那样疼的受不了了。到第七天，奇迹出现了——外甥扔掉了拐杖和轮椅，能自己走路了。外甥和家人非常感谢法轮大法和李洪志师父。

#### 师父救了我女儿
〖中国大陆来稿〗前几日气温骤降，一天夜里十二点左右，我起身给身边的孩子盖被子，顺便摸了一下孩子小手和小脸，一摸吓一跳：怎么这么凉！我又摇摇孩子，让她醒醒起来吃奶。可是摇了几下，孩子不醒，摸了孩子后脖子，也是偏凉，明明盖的被子比较厚，并且大人也在身边。此时不好思维上来了，怎么办？孩子不醒，并且呼出的气息都是凉气。

我叫醒孩子爸爸，商量去医院吧，可外面天气很冷，又担心孩子二次受凉。这时我想到喊：“请师父救救孩子！”并且一直念“法轮大法好，真善忍好”。孩子一下睁眼了，看得出来她气息有些弱，并且无力，我让孩子爸爸给测量体温：36度。商量后不去医院，把孩子用双层被子包裹好，我抱着给她取暖，一边抱着她一边念大法好，求师父救救孩子！我出了很多汗，身体的温度传递给孩子。再给孩子测量体温：36度6。就这样抱着孩子两个多小时，孩子身体热乎起来，也开始吃奶了，我们两人心都踏实下来了。

早晨八点多钟，孩子醒来，象什么事都没发生一样，精神也很饱满。是师父救了我女儿！师父一直守护着我们！

#### 大法太神奇了！
〖中国大陆来稿〗我是一九九七年得法的，得法前脾气暴躁，视钱如命，经常和丈夫吵架，又爱骂街，浑身是病。

我姐姐是学大法的。有一天她到我家劝我学法轮功，我说我脾气不好，法轮功教人做好人我又做不到。姐姐再三劝说：你去试试，不行你再别学。不好意思推托，当晚就去了炼功点。大家正在学法，我一听这是修佛修道的法，从内心深处发了一念：我要一修到底。

从那天起，我就正式走入了修炼大法的门。我再也不骂街了，以前老想改怎么也改不了，刚去炼功点师父就管我了，好象有一种力量加强着自己的意志，浑身的病也都好了。

大法真是太神奇了。有一次吃了晚饭，弟弟给我打电话，说母亲病的很重，叫我和大姐赶快去。外甥骑摩托车带着大姐，我家大女婿骑摩托车带着我和大女儿赶到娘家。母亲躺在炕上正在输液，身体都僵硬了，掐她的人中也一点知觉都没有。医生说就是醒过来也是个植物人。我上到炕上挨着母亲，发自内心的诚念：“法轮大法好，真善忍好。”不停的念。一会儿，母亲睁开了眼睛，瞅着我问：你们怎么来了？我们姐弟六人都在。二姐当时就大声说：“真沾了她们姐俩炼大法的光了。是师父救了我母亲。”当场满屋子的人都见证了大法的神奇。

又隔几年，也是吃晚饭后，弟弟又给我打电话，说：看着母亲不行了，你俩快来吧！我俩去了后，進屋看到母亲心疼的翻来翻去。家人把送老的衣服都拿到跟前了。我赶紧说：“娘，你怎么不叫师父呀？”她说忘了。我说：“你快求师父救你呀，从心里念法轮大法好、真善忍好。”她说“嗯”。不一会儿她就不那么痛了。晚上她说：你俩学法我都听着哪。其实我俩当时没学，是师父在帮她，让她听到法。第二天母亲就没事儿了。师父又救了我母亲的生命。

这样的事太多太多了。弟子无法用语言表达师尊一路的慈悲救度与保护。弟子唯有精進，做好三件事。让师尊放心，谢谢师尊。
