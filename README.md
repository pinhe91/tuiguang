### 念大法好　患脑血栓卧床的公爹能行走自如了

文: 辽宁大法弟子　心平 

【明慧网二零二二年六月二十四日】昨天小组学法后，A同修说有位同修写她婆婆诚念九字真言转危为安的文章在明慧网上发表了。当时我们就迫不及待的打开网页找到文章读了起来。读后特别感慨大法的超常，感慨大法弟子家人的幸运。我也想起了一件至今还令我记忆犹新的一件事儿。
那是二零一五年腊月初七那天，我去小叔子家给公公祝寿，那时公公患脑血栓刚从医院回家，生活不能自理，但思维正常，能与人交流。开席时我说：爸，祝您生日快乐，第一口饭我来喂您，记住诚念“法轮大法好，真善忍好”。还没等我说完，公公就回了我一句：我才不念呢，大洪（我丈夫已过世）都走了。我说大洪不修炼就早走了。公公略有所思。因为此前公公认同大法好，嘱咐过我们注意安全，只是惧怕邪党的迫害。然而当天晚上真真切切发生的事儿让公公发生了彻底的转变。

第二天早上刚刚起床，公公就迫不及待的跟我们说：“这回我可得念了，昨晚地狱小鬼黑白无常来了，到我跟前准备带我走，我就喊小悦、玉兰（都是我妯娌）给我穿衣服，黑白无常让我问你（指我），带我走你同意不？这时我往你睡觉那儿一看，你那边一片通红啊！这下我可相信了。“

通过这件事后，公公天天念“法轮大法好，真善忍好”。有时我给公公打电话，他第一句话先说“法轮大法好”。公公身体康复的很快，不久就能拄棍行走了。现在能自己上下四楼了。

今年端午节我和孩子去看望他，我说您还念吗？他说念，做双手合十的动作念给我看，我真为他高兴，得过脑血栓，现今八十六岁高龄的老人诚念“法轮大法好”后，身体健康。
