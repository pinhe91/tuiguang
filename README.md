### 亲戚躲过郑州大水灾和武汉肺炎感染

——见证“善待大法一念 天赐幸福平安”

文: 中国大陆大法弟子 

【明慧网二零二一年八月二十六日】二零二一年七月二十日，河南省郑州市遭遇特大水灾，大水冲走了人和车辆，淹没了京广路隧道，倒灌地铁，将营运中的列车吞噬……洪水退去，遇难者的遗体横卧街路和地铁站，灾难造成的生命财产损失难以估量。
七月二十六日，我接到郑州亲戚的电话，她激动的说；我遇到的也算个奇迹，也算得福报了。原来，她如约于七月十九日去民政机构办事，去了人家说早了，让第二天七月二十日再去。

七月二十日下午一点多，她要去坐地铁，刚出门，遇上弟媳开的车，弟媳招呼她坐车，她说能坐下吗？她说，还有你一个座位。就这样，只差几分钟的功夫避免了洪水倒灌的灭顶之灾。

这个亲戚住在郑州市内离地铁站不远的地方，冒雨十几分钟就能跑到地铁站去乘地铁。冥冥之中，劫难就这样错过了。

我的亲戚平时很认同“法轮大法好，真善忍好”，家中也有护身符。她说是大法救了她。

她还说，前几天，她和她的一个邻居一块呆过，那个邻居感染了武汉肺炎，那人接触过的很多人，都呈阳性，都隔离了。而她的全家经过核酸检查，都是阴性。

她感激的说：“真是善待大法一念，天赐幸福平安！有大法师父的庇护，危难时，心中会有一种平稳的感觉。”

她说谢谢大法师父的保护！
