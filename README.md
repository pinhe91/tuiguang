### 邪党党员的烦恼

【明慧网二零二二年五月二十九日】

#### 辽宁建平县尹国志被沈阳第一监狱迫害致死

【明慧网二零二二年五月三十日】（明慧网通讯员辽宁报道）辽宁省朝阳市建平县法轮功学员尹国志，被非法判刑十年，二零二二年五月二十二日在沈阳第一监狱被迫害致死。家属在去监狱的路上，狱方告之：他们那有新冠病毒感染者，去了就隔离，家属就没去领尸体。
尹国志的妻子付景华，遭受七年冤狱后，被迫流离失所，于二零一九年十二月三十一日在凄凉的租住房中含冤离世。

尹国志夫妇家住建平县国营八家农场刘家沟村，在一九九九年七月二十日中共迫害法轮功之前开始修炼法轮功，以前患过的多种疾病在修炼法轮功后都好了，从此家庭和睦。

在一九九九年七月中共迫害法轮功后，尹国志和付景华坚持修炼法轮大法，遭到多次骚扰、拘留。二零零九年七月二十八日上午，建平县八家农场乡派出所原所长段伟军、指导员张宝凤、恶警姚云阁突然闯入尹国志家中，进屋就象土匪一样，把家翻得一片狼藉，抢走电脑打印机和法轮大法书等。到中午十二点又打电话叫来八家村村长杜耀军，开车将付景华和她儿子绑架到派出所。孩子后来被放回，尹国志趁机走脱。

付景华被建平法院非法判刑七年，尹国志从此被迫流离在外。那时家中只剩下七、八十岁的老人和一个不懂事的孩子，尹国志的老父亲现卧病在床，靠年迈体弱多病的老母亲照顾。又正值秋收季节，尹国志家里的庄稼只得靠亲戚、邻居出手帮忙收拾。

对这一家人的迫害并未停止，朝阳市政法委对流离失所的尹国志非法下了通缉令，并用五万元钱做诱饵抓捕他。在朝阳市公安局、朝阳政法委监督下，建平县公安局疯狂追捕尹国志，过程中采取蹲坑、监视、株连、诱骗等方式已达到不择手段的地步，建平县的大街小巷到处张贴着抓捕尹国志的告示及照片。二零一一年期间，朝阳市公安局、朝阳政法委人员住入建平、凌源督办此事，为企图寻找尹国志，警察劫持了其家人做人质逼着到处找寻尹国志下落，曾经一天四次去凌钢西区一老年法轮功学员家骚扰。

建平县公安局国保大队队长姜杰和本乡派出所段伟军等多个警察还多次到家中骚扰、恐吓，企图抓捕尹国志，使孩子、父母老人遭到身心双重打击迫害，吓得尹国志的母亲心脏病、高血压复发，于二零一四年离世。孩子在没有父母关心照顾下，一切的生活就可想而知了。付景华的娘家母亲也因思女心切，于二零一三年离世。两位至亲的离世，尹国志、付景华都没能在身旁照顾、为其养老送终。

付景华在二零一五年结束了七年冤狱回到家中，终能在家中照顾80多岁卧病在床、生活不能自理的公公。可在二零一八年十二月十三日，建平县保副大队长张立慧伙同八家农场派出所警察和所长于跃军、张宝凤、刘艳军、于勇等警察等人又再次绑架了付景华，非法拘留十五天。由于这次付景华没有写“保证书”，张立慧伙同派出所所长于跃军等警察又找借口两次闯入付景华的家中骚扰、恐吓她，并威胁说：“你要不写保证书，上次判你七年，这次就给你判刑十六年。”付景华为躲避迫害，只好流离在外，过着有家不能回的艰苦日子。

期间，尹国志的父亲在家没有人照顾，老人在既担心、害怕，又想念儿子、儿媳的情况下，身体、精神承受着巨大压力，等付景华被迫害十五天回家时，就已经卧床不起，生活不能自理了，不长时间就离开人世。

二零一九年九月二十六日，尹国志在朝阳凌源租房家中，因电源失火，被凌源市八间房派出所警察伙同建平县八家派出所警察绑架。被非法关押在建平县看守所，因他不配合迫害，警察曾把尹国志与黑社会人员关在一起，让黑社会人员对尹国志毒打折磨。建平检察院与法院秘密将尹国志无罪冤判十年重刑，直到送往锦州监狱后，建平法院才打电话告知尹国志的家人。

在付景华听到丈夫被绑架后，担心忧虑，自己又居无定所，孤苦伶仃。在极度悲苦与打击下于二零一九年十二月三十一日，在租住房中含冤离世。

二零二二年三月八日，辽宁省沈阳第一监狱两名狱警（不知姓名）伙同建平县国营八家农场八家村干部来到尹国志家，说尹国志在监狱已得晚期肺癌，在医院输氧气，让家属拿钱治病。二零二二年五月二十二日，尹国志在沈阳第一监狱被迫害致死。

本应美满幸福的家庭被中共警察迫害得支离破碎。当地百姓可能都非常清楚这一家人的为人，他们只是一个极普通的善良家庭，并没有做任何伤害他人的事件。可就这样善良的一家人竟被中共警察迫害得如此悲惨。
