### 辽宁省凌海市公安局副局长王景山迫害法轮功的事实

文: 中国大陆法轮功学员 

【明慧网二零二二年六月二十二日】（明慧网通讯员辽宁报道）王景山，曾担任金城造纸厂公安处处长、金城派出所所长、凌海市公安局金城公安分局局长、凌海市公安局副局长。期间，王景山积极追随江泽民流氓集团对法轮功的残酷打压政策，对法轮功学员灭绝人性的迫害。
王景山，男，一九五八年九月出生，辽宁省锦州凌海市公安局副局长，主管迫害法轮功。王景山家住辽宁省凌海市金城镇北街103栋6单元301室。赵英兰，女，王景山的妻子；王可，男，王景山的儿子

在金城地区任金城造纸厂公安处处长、金城派出所所长的时候，有十五名大法弟子被送非法劳教，多人被劫持、拘留、抄家、罚款。王景山对被抓捕的大法弟子辱骂、拳打脚踢、打耳光、电棍电、用绳子捆、勒索钱财、高额罚款（多数不开收据；有的开收据不盖章）。

王景山到凌海市公安局后，游走全市乡镇，带一帮土匪警察任意出入法轮功学员家，劫持、野蛮抄家、绑架、拘押、送劳教所、送监狱。一九九九年到二零一九年，经他手劫持19位法轮功学员，非法拘留80人次、洗脑班7人、开除公职10人、送劳教44人、判刑12人，迫害致疯2人，致残1人，致死3人。

更甚者，法轮功学员的身体不合格监狱拒收的时候，王景山等采取往监狱送礼的方式，也要把法轮功学员送进监狱，使法轮功学员长期遭受各种酷刑折磨与精神摧残，制造多起妻离子散、家破人亡的惨案。

具体迫害事实：

1. 二零零三年二月中旬的一天晚上，法轮功学员王兰芝因挂条幅、告知世人“法轮大法好”，被她工作单位辽宁省锦州市金城造纸股份有限公司公安处（现在的公安分局）处长王景山带人非法抓捕，被判刑长达六年，被关押在沈阳女子监狱。

2. 二零零八年二月二十五日清晨，凌海公安局国保大队长、金城公安分局局长王景山带领一群警察将任桂霞的家团团包围。任桂霞拒绝开门。六点多钟，警察找人开锁，仍未得逞，随后又拿镐刨门，将门刨开，闯入屋内，非法抄走230彩喷一台，电脑一台，手机、存折、大法书籍、师父法像等，室内一片狼藉。并绑架了张国新、任桂霞夫妇，把他们非法关押在凌海市看守所，任桂霞一直绝食反迫害。

3. 二零零九年四月十七日上午六点左右，凌海公安局副局长王景山带人到魏秀英家抄家、绑架。随后多个警察非法闯入魏秀英家强行将她及她的丈夫、女儿绑架，并且非法抄家。警察又以调查情况为由，将她大女儿赵虹绑架，赵虹家里被非法抄家。警察用热水浇魏秀英的头，踢她的心窝，用有水的瓶子打她的右耳，致使她右耳失聪。魏秀英不能行走，并出现心脏衰竭、抽风、高血压等症状，吃喝全吐出来，非常瘦弱，生命极度危险。凌海市公安局副局长王景山拒绝释放魏秀英，并把魏秀英送到抚顺洗脑中心，拒收后又把她送回凌海市看守所，不许她家人接见。

4. 二零零零年四月二十六日，金城造纸集团公安处处长王景山派人将法轮功学员金鑫、韩丽萍、王兰芝、陈小莉从北京接回，把他们非法拘禁在四独身宿舍洗脑，逼写保证书，采手印指纹。非法收路费每人二千元、押金三千元。

5. 二零一一年六月二十九日，法轮功学员金鑫、曹艳华和孙丽英去农村发真相资料，被凌海公安局副局长王景山和两名巡警绑架。金鑫的妻子多次去拘留所要求见人，但里面值班的说领导有话而不让见。

6. 二零零四年七月二十二日，法轮功学员韩丽萍被非法抄家。父母（炼法轮功）被绑架到凌海市拘留所，拘留一天一宿，本人正念走脱。十五日后，金城公安分局预谋二次绑架，韩丽萍一家三口被迫流离失所两年多之久。
