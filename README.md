### 【庆祝513】过去横霸主　今日大法徒

文: 大陆大法弟子（同修帮助整理） 

【明慧网二零二二年五月二十七日】我是农民，今年六十七岁了，我修炼法轮大法二十多年。法轮大法给了我一个好的身体，还去掉了我抽烟和嗜酒如命的坏毛病。再有，就是大法净化我的思想，让我能做一个为别人着想的好人。这些都是我在大法中受益的结果。

今天，我把我修炼这些年中身体的变化和思想的变化说出来，向慈悲伟大的师父和各位同修汇报一下。

#### 病痛折磨　喜得大法

炼功以前，我有老寒腿的毛病，而且很严重，具体症状是：一年要穿九个多月的棉裤，冬天穿厚棉裤，春天穿薄棉裤，到五、六月种地和铲地的时候，也要穿有一小层棉花的棉裤。就这样，夏天也好不了几天，一开始立秋，就要把薄棉裤穿上。那真是：在太阳下晒着，腿好受，上身出汗。在阴凉下，上身凉快，腿难受，一天天不知道怎么活好。

在一九九六年秋天，我堂哥家办喜事，来的亲友很多。我有一个表叔也来了，他对我这样上半身衬衫、下半身棉裤的穿戴很是不解，就问我原因。我就把我病情跟他说了一遍，告诉他这种病很难治。他说：那你炼法轮功吧，法轮功祛病效果很好，现在很多人都在炼，我也在炼。我一看他的气色很好，就说：怎么炼呀？他说：按照书上说的去炼，你到我家去，我给你一本《转法轮》。

一九九六年农历八月二十三，我就去了表叔家，请回了宝书《转法轮》。回家之后，我从头到尾仔细看了一遍。也就是在那天，我下决心修炼法轮大法，按照书里说的去做。

#### 改陋习　家和睦　病体康

提起喝酒，我是冬天一天要喝四顿，夏天要喝三顿，还必须顿顿都喝，一顿不喝都不行。至于抽烟，我早晨起来，睁开眼睛第一件事就是先抽烟。就这两样恶习，妻子非常讨厌，屋里整天乌烟瘴气的。

这还不算，我在家里是绝对的霸主地位。就吃饭来说，哪怕饭盆在我跟前，也要妻子给我盛饭，而且吃多少，我在碗里画一个大体位置，不许多、不许少。吃饺子，要吃刚煮的，刚煮的饺子鼓肚的好吃，反正这一顿饺子，要给我煮几次，直到我吃完为止。吃面条也是，我吃一碗，你得给我煮一碗，我吃下一碗，你再给我下一碗，端上来，一顿饭下来，要煮几次面。为此，妻子也是敢怒不敢言，因为她怕我揍她，或者说她被我打怕了，对我唯命是从。现在回想起来，真是汗颜！

通过学大法，我第一天就戒掉了烟和酒。记得那天吃饭的时候，妻子问我：怎么没喝酒啊？我说：我今天开始炼功，不抽烟，不喝酒了。妻子要给我盛饭，我没用她盛，自己盛的饭。我就开始吃饭，妻子看着我，小声对女儿说：你爸还能不喝酒？！女儿也不信。我对她们说：我炼功，就得做好人哪。从那天开始，我也能吃完饭收拾饭桌，洗碗，干家务活儿，还能做饭了，家里家外活儿，我都多干。

从那开始，我也不打老婆，不骂孩子了，家庭从此和睦了。至于我的老寒腿，那是当年秋天，我给别人帮忙割黄豆，因为着急干活儿，热了也没多想啥，就把衣服裤子（还是薄棉裤）脱了，继续干。到收工穿衣服的时候，才想起来：我怎么把棉裤脱了呢，哎呀，我的腿怎么出汗了？（我的腿以前是不出汗的）大法真的是太神奇了！是师父和大法把折磨我多年的老寒腿治好了！谢谢师父！

妻子一看，大法这么好，她和儿子、女儿也开始炼功了。村里人一看我戒烟、戒酒，而且脾气也改了，身体也好了，全家一天都乐呵呵的，都看到了法轮大法真是好，纷纷来学法轮功。全村一百户人家，最多时，有五、六十人学大法。全村连打扑克的人都玩儿不上了，炼功的这些人都说：我们炼功学法，不玩儿扑克、麻将了。把村风都改变了，法轮大法真好。

#### 做一个诚实的人

修炼大法以前，我有一些不好的行为，比如，干活爱偷懒，说话不诚实，买卖东西斤斤计较，利益心很重。通过学法炼功以后，我改掉了这些不好的行为。大法教我做一个诚实的好人。出去打工，我给别人干活儿，就象给自己家干活儿一样，雇主都说我能干，不偷懒。

二零二一年秋天，我在离家七、八里路的工地上干活儿。第一天，包工头和我还有另外两个人抬水泥板装车。装完车他说：你真能干，身体好，以后你跟我干吧。有一次，也是往车上装水泥板，我一上午就干完了，要是别人一上午干不完。包工头说：你明年跟我干吧，一天我给你三百元。他很高兴，中午请我吃的驴肉馅饺子，还买了饮料给我喝。这个工地有十多个工人干活儿，工钱都是二百元。

包工头知道我不偷懒，给我的时间自由。我的工作就是把水泥板码成垛，计件赚钱。我早晨七点到，中午吃完饭休息三十分钟，下午三点就干完活可以回家，一天工钱四百多元。其他工友都说：这老头儿真能干，不累吗？我说：你们看我累吗？他们说：我们这些人谁也没有能干过你的。我说：我炼法轮功，身体好。师父给我净化身体，别看我六十多岁的年纪，我的身体是年轻人的身体。

他们和包工头都认同法轮大法好，都明白了真相。他们说：你人挺好的。我说：是我的师父让我做一个处处为他人着想的好人。

我的活儿干的好，包工头很满意。工地其他人都走了，他就把我留到最后，把活儿全干完了，他留了我的电话。

#### 市场卖菜的小故事

我从二零一四年开始种菜，卖菜。我从不糊弄人，菜价合理，有不少回头客。买菜的人把挑好的菜上称时，我给的总是只多不少，算账零头不要，买菜的人多数都非常满意。时间一长，有很多人知道我是炼法轮功的，一起在市场里卖菜的人也知道。

一次，一对母女看菜挺好的，就买了几样，付完钱就走，我也没看多少钱。我整理完菜，一查，钱多了好几块。我一看那对母女已经走远了，就大声招呼她们回来，她们以为钱给少了。我说：你们把钱给多了。把钱退给了她们。

有一次，一个人买完菜就走了，我回头一看，地上怎么有钱呢？一看是刚才那个人掉的，我就把那个人招呼回来，把钱给了那个人。周围卖菜的人说：给他干啥？不能给。我说：我们炼功人要做好人，不能要人家的钱，不是自己的不能要。

还有一次，一个从省城来的妇女，要买芹菜根给母亲吃，她老母亲九十多岁了，省城没有，这个市场也没有，她很着急。她来到我的菜摊前，问我有没有芹菜根。我说：家里有，今天没拿来。她说：你明天拿来，多少钱一斤都行，还在这个地方见面。我说：行。第二天她来了，我把芹菜根给了她，问她：够不够？她说：够了。问我多少钱？我说：不要钱，你这是孝敬老人，我帮个忙，这没啥。我又说：我是炼法轮功的，师父让我们做好人，为别人着想。她非常感谢，拎着芹菜根高兴的走了。我也很高兴，因为自己能帮到她。

就这样的事例，还有很多，在此就不一一列举了。回想二十多年的风雨修炼历程，感触最深的是师父的保护，是师父和大法改变了我，让我处处为别人着想，让我做一个真正的好人。

感恩师父！
