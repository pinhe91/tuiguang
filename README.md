### “九字真言”不是一般的神奇

文: 大陆大法弟子

【明慧网二零二二年五月十九日】二零一六年初夏的一天，我接到妹妹的电话，说我侄子突发急病，诊断为“肾脏衰竭并发症”，当地医院拒收。好不容易找熟人去了市医院住了下来，该医院主任医生也说已无回天之力，住院也只是对患者及家属的一种安慰。
我听电话后感到很惊诧，怎么可能呢？从来没有生过病的他，太突然了。侄子只有三十多岁，还这么年轻，他的孩子还在校读书，要真发生这种事，这家就完了。

我急冲冲的赶往医院。那天正好下雨，赶到医院已是下午近两点了，侄子全家人都在那，住院七天了，侄子没进过食，脸色苍白，消瘦无神，医生已通知家属回家养养，大家都在和医生商讨明天出院的事宜。

我看此情景，二话没说，就拿出带去的《天赐洪福》真相小冊子和印有“法轮大法好、真善忍好”的护身符给我侄子，他拿到马上就看了起来。他看明白后问我：只要念“法轮大法好，真善忍好”九个字就行啦？我点了点头。他即刻把护身符放在上衣口袋里，嘴里马上就开始念起“九字真言”。我陪了他一会儿就告辞了，并跟他讲明天出院后来看他。

第二天吃过午饭，我去侄子家看他，他不在，我问弟媳他人呢？弟媳告诉我：“今天我妹妹送午饭后急着回家，他看到饭后就狼吞虎咽起来，他爸看到他这种吃相吓坏了，提醒他说，医生吩咐吃东西要忌口的，不能乱吃。他生气地回答：我已经七天没吃饭了，你还不让我吃！他从医院刚到家，就接了个电话，马上骑摩托车出去了。”我听了太高兴了，奇迹在侄子身上出现了，我的担心放下了。

吃过晚饭，我又去看侄子，刚到他家门口拐弯处，迎面就碰到了他，他手中拿着个方便袋，我问他上哪去，他回答我：“我已好长时间没洗澡了，我要到澡堂去泡泡。”我简直不敢相信，一个被两个医院都判了死刑的人，一夜之间就判若两人，仅仅念了“法轮大法好，真善忍好”九个字，发生了这么巨大的变化，和正常人没什么两样，这在现代医学史上是无法得到解释的。我一天去两次看侄子，和他还攀不上说几句话，可他精神是那么的充沛。

第二天早上，我带上播放器给侄子，叮咛他，这是师父的讲法录音，要反复听。我并叮嘱弟媳，继续帮他念“九字真言”，他们答应后，我就回家了。

约过半个月，我去看侄子的情况，一进他家门，就一眼看到他头发梳的铮亮的，坐在那认真的干活呢。侄子是搞雕塑的。我问弟媳：你平时帮他念“九字真言”了吗？弟媳回答：“我一天给他念两至三百遍呢，他还自己就抽空听师父讲法。”

再过半个月，我又去看望侄子，他又不在家。弟媳说：“他厂里开发新产品，厂长开车把他接走了，要他给新产品开模具。”

看到侄子的神奇变化，他全家人都很感恩师父，都做了“三退”，侄媳妇还帮忙送真相资料，还为身边的人做“三退”。

六年过去了，我侄子虽未走入大法修炼，但是他没忘记大法师父，他按大法要求做人，平平安安到今天，他女儿也已大学毕业了，找了个好工作，一家人幸福美满。

法轮大法的洪大慈悲救了弟弟的一家及所有家人。感恩师父，感恩大法，愿我们每个人心灵深处都由衷地默念：法轮大法好，真善忍好。
