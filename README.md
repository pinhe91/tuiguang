### 辽宁朝阳市王文芳被迫害离世

【明慧网二零二二年十月十日】（明慧网通讯员辽宁报道）辽宁省朝阳市法轮功学员王文芳（王芳）老人，被原朝阳市北塔分局局长王文江非法列为“网逃”追捕，身心受到伤害， 精神恍惚，生活不能自理，于二零二二年九月二日离世，终年65岁。
朝阳市法轮功学员姜翠莲老人，出生于一九五二年，被王文江为首的北塔公安分局以她知道王文芳的行踪、所谓“窝藏罪犯”为由，非法关押在朝阳看守所一年，于二零二一年九月六日回家，回家时不会说话。

#### 修炼法轮功，脱胎换骨

王文芳，出生于一九五七年，一九九六年开始修炼大法。在修炼之前，王文芳在家说一不二的，王文芳一说话，大嗓门，得理不饶人，所以给自己造了一身病，乳腺增生、气管炎、胃炎、咽喉炎、甲亢脖子肿大、子宫下垂、腿痛肿得老粗，胃痛起来让王文芳不能吃饭，手麻颤抖严重，拿不了东西，手指骨变了形都不能伸直握拳，人还经常休克，整夜睡不了觉。打针吃药的，经常上医院。
一九九六年，听人说炼法轮功祛病有奇效，王文芳开始修炼法轮功。神奇的是，也就是四、五天时间，王文芳的病就好了，尤其胃好了后，王文芳一顿能吃三碗饭，走路飘飘的，干活也不累。那种高兴的心情无法表达，没有语言能形容。家人亲眼看到王文芳的变化，丈夫也高兴的说：“变化太大了！变化太大了！”。

从此不论在家、在外，王文芳都按真、善、忍要求自己，做事多为别人考虑。

#### 被迫流离失所

一九九九年七月二十日以江泽民为首的中共邪党开始了对法轮功的迫害，王文芳和成千上万的中国大陆的法轮功学员一样，顶着压力，面对暴力和危险，用各种方式向民众讲大法真相，救度世人。因为法轮功学员都知道，一旦世人听信了中共对法轮功的打压谎言，相信了中共的诬蔑之词“法轮功是×教”，那么这个生命就极其危险，就会在最后淘汰人类的大瘟疫中被淘汰，跟随中共邪党做陪葬。为了救度身边的有缘人，为了朝阳市的父老乡亲能够在即将来临的大瘟疫中能够保命，王文芳做出了巨大的付出。

二零一八年十一月二十一日，王文江所主管的北塔分局警察十多人闯入王文芳家，进房抢到开门钥匙，直接到楼下小房翻东西，抢走几箱真相资料，并把王文芳劫持到北塔分局，后王文芳被非法拘留，因检查身体不合格，被拒收。

为防再次被非法抓捕迫害，王文芳离开了家，结果第二天王文江果然就派北塔分局警察又上门骚扰，用各种手段威逼王文芳儿子找到母亲下落，并把善良老人列为“网逃”非法追捕。从此王文芳居无定所，流离失所。

#### 处境艰难

中共迫害法轮功，有一条龙的程序：公安局非法抓捕，检察院、法院非法判刑，监狱残酷迫害，最后单位给开除公职。从根子上来讲，起步就是错的、是犯罪行为，那么一步一步接下来，也都是错的、是犯罪行为。也就是说，如果没有公安人员的第一步“非法抓捕”，也就没有后面的被非法投进劳教所、监狱而被打死打伤、酷刑折磨。王文江，开始了迫害关键的第一部，启动了迫害王文芳的这个一条龙机制。如果王文江成功地抓到了王文芳，那么就会把王文芳置于悲惨的境地，其结果不可想象。这给王文芳造成了巨大的精神压力。

在以王文江为首的北塔公安分局的非法追捕中，王文芳老人躲进一个被人家闲置废弃的一个小屋里，四壁透风。十一月份，东北很冷，小小的土炉子，只能在做简单的饭菜时生火连取暖。因为环境险恶，王文芳的行踪只有少数同修知道，他们送来了些粮食和蔬菜。每当同修来看她，听见有人敲门，她就十分恐惧，颤抖着声音与同修说话，甚至听见风声、树叶的响声，她也惊恐万状，以为是警察来抓她。她住的小屋及附近没有厕所，大、小便只能便在黑色塑料袋里。

王文芳不能去开工资，不能回家，也不能打工了，没有了生活来源，身无分文，吃饭都已经成了问题。炕边有几包衣服，都是以前同修及亲戚送的旧衣物。屋里灯光昏暗，灯是从房顶吊下来的一个灯泡，她说点太亮的灯怕被别人发现屋里有人。她的精神和身体，一天不如一天，逐渐的在恶化。

#### 姜翠莲被非法抓捕关押一年，回家时不会说话

法轮功学员姜翠莲，女，生于1952年。那时姜翠莲出门在外，家中没人，王文芳给她看过房子，住过一段时间。以王文江为首的朝阳市北塔分局警察查出了王文芳曾在姜翠莲家住过，就开始骚扰姜翠莲老夫妇俩，并以查外来人口为由，到姜翠莲家中各房间找人，没发现什么，才离开了。

二零二零年六月十一日，朝阳市双塔区北塔分局四、五个警察第二次闯到姜翠莲家中，姜翠莲没在家，姜翠莲的老伴（未修炼法轮功）被带到分局，恐吓威逼老人一天一夜，让说出王文芳去向。姜翠莲的老伴并不知道王文芳下落，只知道她经常去的一个小区。警察查到王文芳住过的那个房子，并开始蹲坑监视。

二零二零年六月二十九日，北塔分局蹲坑警察看到此房间亮着灯，就敲门欺骗说，楼下漏水了，骗开了门后，将正在此处的两名法轮功学员绑架，非法关押七天后，放回。

北塔分局警察竟不遗余力的追找一位善良老人。二零二零年九月六日，北塔公安分局两个警察再次闯入姜翠莲家中，把已年近七旬的姜翠莲从家中强行绑架走，北塔公安分局竟以“窝藏罪犯”为由，将姜翠莲老人非法关押在朝阳市看守所。姜翠莲在朝阳看守所被非法关押一年后，二零二一年九月六日回家。

可是回家后的姜翠莲与被非法抓捕关押前的姜翠莲判若两人：她已经和正常人不一样，不会讲话，只会简单的说出几个词，连不成句。

#### 王文芳精神失常、悲惨离世

有一次，一个同修打开反锁的房门，顺院进屋，没见到王文芳，就叫了几声她的名字。她从门后走了出来。因为她听到有人进院，就惊恐地藏到了门后。见是同修，才从门后边出来。见到王文芳处境艰难，这个同修就想把她从这里带走，换个好的环境。当问到王文芳是否同意时，她哭着说：“我也不想在这儿了。”当时正赶上人口普查，社区工作人员挨家挨户的走，这又给王文芳增加了压力，怕被发现。

王文芳后来被同修安置在一个有暖气的楼房里，虽然环境有了好转，可是王文江把她列为“网逃”造成的巨大压力，使她的精神受到了严重的刺激。她总说屋里有动静，大白天把窗帘挂得严严实实。终于，她的精神失常了， 被送回了家。

即使王文芳已经精神失常，身为朝阳市北塔公安分局局长的王文江，还是不肯放过王文芳老人，还是不时地要来几个警察来查看，骚扰。他们经常到王文芳家窗户前窥视，有时给她的儿子打电话，有时也会进屋向王文芳的儿子询问王文芳的情况。这些都给王文芳造成巨大的精神压力，致使她的精神和身体崩溃、垮掉了，在二零二二年九月二日去世。

在中共自从一九九九年七月二十日公开迫害法轮功以来长达23年的时间里，众多的法轮功学员被迫害流离失所，他们过的颠沛流离的生活，所承受的精神压力，是无法想象的。王文芳的遭遇，是这些受难中的法轮功学员的真实写照。

#### 王文江难脱其责

![image](https://user-images.githubusercontent.com/79625284/194835544-39f0fa0e-d728-4f0a-ba40-d4abe6daf95c.png)

王文江，男，汉族，一九七一年一月十二日生，辽宁朝阳市建平人。曾任朝阳市公安局北塔分局局长，双塔区政府副区长人选（兼）。王文江任双塔区副区长的工作分工：负责公安、消防、司法工作。分管光明分局、南塔分局、北塔分局、燕都新区公安分局、司法局等。现已调往辽宁省凌源市任公安局长。王文江手机号码：13842101234、13332362020。

任朝阳市燕北派出所所长、朝阳市南塔社区大队长期间，王文江屡次迫害、勒索法轮功学员；王文江任喀左县公安局政委期间，喀左县警察绑架、恐吓、骚扰当地法轮功学员。
