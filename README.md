### 迫害22年中 我失去了四位亲人

文: 吉林省延边州法轮功学员　崔艳玲 

【明慧网二零二一年七月十九日】法轮大法弘传世界一百多个国家和地区，这高德大法使人心向善，道德回升，身体健康。我的亲人都是因为身体有病走入修炼中来的，他们都在大法中受益匪浅，身体上都出现过奇迹。法轮功祛病健身、升华道德有奇效，这是不争的事实。自一九九九年七月二十日中共恶首江泽民疯狂发动了对法轮功的迫害，至今已持续二十二年，有多少法轮功学员被迫害的家破人亡、妻离子散。
我的亲人他们虽然没有被判刑、劳教，也没有被绑架，可是他们面对中共疯狂地迫害，看到的和经历的是抄家、绑架、跟踪、监听、威胁、恐吓，并顶着巨大压力营救被绑架的亲人、忍受着巨大痛苦也不放弃对真、善、忍信仰。在迫害二十二年中　我失去了四位亲人。

#### 1、大法给了我女儿健康和聪明 邪党迫害使我女儿英年早逝

我女儿崔榕芷从小患有先天性心脏病，四岁时失去了父亲（已故），我们母女相依为命。我每天都担心女儿的身体健康，有什么头疼痛感冒的她都能遇到，我两次带她去沈阳准备手术治疗，给女儿看病的那个老教授钱武杨军医说：孩子太小，长大一点再看看。

![image](https://user-images.githubusercontent.com/79625284/126137103-4bf35785-cdff-4692-b14e-e3489b89acff.png)

女儿崔榕芷

有幸的是，一九九八年我与女儿一起修炼法轮大法，法轮功教人以真、善、忍为准则修心向善、使道德回升，祛病健身有奇效，女儿按真、善、忍要求修自己，做好人，使她身体健康，道德品质提高，修炼二十年，没吃过一片药。是法轮大法给她的健康和智慧。她学习名列前茅，从小学到高中她写的作文都被列为范文，并获得北京、清华、复旦等十所大学主办的“中华杯”第六届全国新概念作文大赛入围奖。中学毕业颁发的是优秀毕业证书。女儿毕业于吉林华桥外国语学院，毕业后在一家公司作翻译，后来她又教英语，不管她在哪工作。她都是为别人着想，受到同事和家长的好评。

一九九九年七月二十日中共江泽民流氓集团开始迫害法轮功后，二零零零年三月我去北京向政府反映法轮功的真实情况，被绑架、关押、罚款，上网通缉等迫害。我被迫流离失所了六年，孩子想跟我通次电话都很难。

二零零一年上小学的女儿放学时看到八家子公安局副局长金升官，刘学彦、段爱军（三十八）等人对我进行了非法抄家，她哭着拽着我不让警察带我走。

二零零六年，女儿在延边二中时，和龙市国保警察两人对我女儿进行骚扰，从课堂上将女儿叫走，强迫恐吓女儿带他们到我的住处，企图绑架我。

上大学时国保赵志奎就去调查过，说女儿写真相信。大学四年级时，女儿突然得了一种怪病，全身抽搐，表面现象就是那种癫痫，医生说癫痫是由脑外伤引起的。而我女儿没有脑外伤，也没有这个病史。可怜的女儿从小到大看到的是绑架、抄家、流离失所、骚扰，恐吓。她是在这种长期的紧张、恐怖的气氛中造成的，一次阿姨来我家敲门的声重了些，阿姨进来了后，女儿身体就出现状况了。

二零一一年十一月十二日，八家子公安局国保大队赵志奎伙同和龙市国保及延边州公安局一行四人利用女儿的手机跟踪、定位。到长春市红旗街居民楼强行撬门非法入室绑架了我，并劫持到延边。女儿马上与单位领导请假，与家人一起回到延边营救我。

二零一六年五月十七日明慧发表了《母亲遭迫害 女儿控告元凶》的文章后，和龙市八家子公安局主管迫害法轮功的局长董立强（57岁）通过家人找我女儿，家人也给女儿施加压力，追问此事，董立强让我的家也参与迫害。女儿的户口和工作单位不在此地，赵志奎、董立强却死追不放，进行骚扰、恐吓，对我女儿正常的学业、工作造成了严重的干扰；对我女儿的成长、身体的健康、心灵造成了极大的伤害；对我女儿的通讯安全、言论自由造成了破坏。

我二零一四年到户口所在地办护照手续（办护照得通过当地公安局），他们迟迟不给办，这次女儿又一次提醒我去办护照，她说：妈咱们出国吧。看着她那无助、可怜的样子，我心里很伤心，女儿吓的不想在这恐怖环境中呆一天，如果我的护照办下来，我们早就出国了，女儿的最大愿望是用她所学的专业施展她的才华。如果赵志奎、张文信不卡着我的护照，后来的事情就不会发生。

二零一八年五月六日乘坐延吉到长春7334号火车，亲属给我和女儿买了两张卧铺票，检票时有工作人员还有两个公安国保警察，其中一人说：不是用自己的身份证买票是犯法。我们重新买了票。延吉车站很小，不到五米远，就有两次安检与搜身，气氛很紧张，女儿怕的都不敢拿出自己的身份证。

我们到家的第二天上午八点多，有人敲门，敲门声音很大。女儿很紧张，我去开门，没有人。当晚十点多，女儿突然从睡梦中惊醒，状态就是全身抽搐， 120急救了近一小时无效。三十二岁的女儿于二零一八年五月七日晚十点三十分离世。

我那好女儿就这样在被骚扰、迫害造成的恐惧和失望中走了。

#### 2、父亲修大法所有的病不治而愈，遭迫害含冤离世

父亲因公负伤患有脑震荡后遗症（经常晕倒）、还有高血压、肝炎、腰椎间盘突出、糖尿病等各种疾病，去了各大城市、各大医院由教授专家坐诊检查，中西药用了很多也不见好（因父亲是离休干部，到哪看病都是公费医疗）。后来他学了气功，自己的病没好，还给别人治病，最后自己都下不了床了。

一九九八年，父亲有幸得了大法。那时他正在卧床不起，弯不下腰走不了路，看师父的讲法录像几分钟，父亲就大声说：哎呀，我身上有法轮在转，哎呀，我全身都在转，全身热。并且马上能下床弯腰穿鞋，走到录像机前看。这法轮功就这么神奇，谁学谁益，是师父在管他了，在给他清理身体了，几天时间全身的病都没有了，所有的病不治而愈。

大法的神奇展现在我父亲身上，全家人都看到了。六十多岁的父亲象年轻一样，走路生风，身体健康，心态祥和，每天早早的就去把炼功场打扫干净，还买了“法轮大法好”的条幅，挂在炼功场上洪扬大法。每天早晨很多人都去炼功。无病一身轻的父亲，更是给国家节省了大量的医药费。

![image](https://user-images.githubusercontent.com/79625284/126137433-019d3b13-782e-4c53-b65f-5ebbb91cd4f2.png)

父亲崔国明

一九九九年七月中共江泽民流氓集团开始疯狂打压法轮功，残酷迫害信仰真、善、忍的法轮功学员。二零零零年后，父亲去银行取500元钱，因亲属家的孩子结婚，要去参加婚礼。他在银行遇到了八家子林业局“610”头目张文信，张文信亮出他警官证，告诉银行的工作人员不让给父亲取钱（注：警察没有这个权力），张文信并没有任何根据说：父亲取钱是要去北京上访。张文信的言行是在制造恐怖，本身就是在执法犯法。父亲钱没取成，身心受到了及大的伤害，情绪很低落。

经过历次运动的父亲，本来就胆小怕事，被张文信这么一恐吓，再加上我被张文信网上通缉，长期流离失所，父亲想见我又怕我被抓，让我离家远走。父亲因此被迫离开了修炼，日日生活在恐惧当中，父亲的压力大，身体状况越来越差。

在二零零七年的夏天，孩子们从外地回来，父亲倚在门边看他的孙女和外孙女吃饭，当时全身的力气都用在那个门上。然而孩子们进来时门没有反锁，门开了，一米八高的父亲整个人都被摔了出去，后脑摔到了水泥地上。当时父亲摔的什么都不知道了，小侄女说：她们给爷爷念好长时间“法轮大法好” 爷爷才醒过来。他摔倒的地方水泥地上有个小坑，坑里有血迹和头发，父亲执意不去医院，说过两天就好了，到了第十八天，不修炼的家人不放心，带着父亲去医院做了检查，医生说危险期已经过了，看了片子后，断定这个人以经是不能说话了。因为没恢复的那个出血点正好压在那个说话的神经上，从医学角度看，根本就不能说话了。而父亲头脑清醒，说话清楚。医生看到后说：奇迹！真是奇迹！

由于父亲不敢学法炼功，身体每况愈下。后来家人叫了120强行送他去医院。抬他的时候，他拽着床不走，把床都拽离位了，其实，父亲心里明白，他所有的病是怎么好的。住院期间，用了很多的药也没有治好父亲的病。二零零九年凌晨，七十七岁的父亲带着恐惧和遗憾含冤离世。

当时由于赵志奎到处抓我，我不敢去医院，在父亲离世的头天晚上去了医院，父亲躺在病床上，好象知道是我去了，冥冥之中他一直说着 一句话：“帮帮我、帮帮我”，当时父亲说话已经不清楚了，我的心里难过极了。好好的一个人被迫害的这样。我跟父亲说：你就念“法轮大法好”，“真、善、忍好”。 父亲迫不及待的一遍接一遍念“法轮大法好，真、善、忍好！”那时父亲多想重新站起来呀。

#### 3、妹妹心脏偷停不治而愈，在迫害中恐惧离世

妹妹患有一种严重的心脏病，心脏偷停，一次在蒸包子的时候出现了这个状况，什么也不知道了，这时正好家里来人了，及时送到了医院，真是太危险的。

妹妹也是在一九九八年学了法轮大法。学法之后她按照真、善、忍的标准做人，并严格要求自己，妹妹是做面食的，她做的饺子、包子很受顾客欢迎。她选食材都是用最好的。她身体轻松，精神愉快，孝敬老人每天乐呵呵的，总是为别人着，全身有使不完的劲儿。她那个心脏偷停病不治而愈，不知道什么时候好的，身体一点都没感觉。

![image](https://user-images.githubusercontent.com/79625284/126137605-c06079e9-558a-43aa-a6d0-f0ef1b642a97.png)

妹妹崔艳杰

一九九九年七月二十日中共打压迫害法轮功后， 我被迫害的流离失所，妹妹来照顾老人和孩子。孩子在延边二中上学，二零一一年八家子公安局国保大队赵志奎伙同和龙市国保警察李某及延边州公安局一行四人利用手机监控跟踪（妹妹和女儿手机被八家子公安局国保队长赵志奎监控）来到长春。他们找到妹妹家的住宅附近，那时母亲正在楼下，赵志奎认识我母亲说是要租房子。母亲回来说：下面有很多人在租房子。刚说完就有人敲门，说是派出所的，他们是跟踪母亲找上来的。

妹妹非常紧张的跟我说：姐可能是针对你来的。妹妹怕我再次被抓，拽着我让我从所住的四楼阳台上跳下去逃走，前阳台我不敢跳，就拽着我让我从后阳台跳，我还是不敢跳。这时外面的门被撬开了，妹妹吓得面无血色。

赵志奎第一个冲进来，妹妹刚要说话，赵志奎先来一个下马威：我知道你是哪的，叫什么名，干什么的。当时我女儿上班不在家，他把女儿在哪个单位、做什么工作，象背课文一样也背了一遍。

妹妹在这种恐怖气氛中一句都没说，这样的事她已经历过一次了。那是在二零零六年五月和龙市国保到延吉延边二中，女儿上学地方进行骚扰，从课堂上将女儿叫走，让给他们带路回家去抓我，女儿吓的拽着老师不走，他们就恐吓女儿，不去就把这件事在学校公开。当时妹妹一个人在家，突然听到有砸门声，妹妹受到吓的心跳喘不上气来，开门看到是警察来抓我，几个房间看个遍。

几次经历我被绑架的过程，妹妹每次都吓得脸色惨白。后来，妹妹出现鼻子出血、牙龈出血。在我流离失所六年期间，所有的家务、老人、孩子都要妹妹一人照顾，我被绑架回家后，妹妹怕我再被骚扰，在房租没到期的情况下就另租了房子，马上收拾搬家。

连累带吓，妹妹的身体状况越来越不好，半年后（二零一二年六月四日），妹妹再也承受不住这种恐怖的迫害而含冤离世，当年54岁。

#### 、不下手术台的母亲修大法获健康、两次目睹迫害、忧心离世

从我记事开始母亲吃饭的时候特别慢，真是细嚼慢咽，就这样还经常被吃下去的饭给噎住，半天也下不去。特别痛苦。什么时候把咽的饭都吐出来才行，喝水都不行。这个现象越来越频繁，怀疑是长了什么东西堵的，后来就去了长春医大医院治疗，检查结果确诊为贲门狭窄，需要手术治疗。两周后到手术那天，我们都在手术室外面等着，不长时间主治医生出来让家属签字：因为麻醉师发现母亲心脏出现早搏，麻醉时有危险。当时我们兄弟姐妹谁也不敢签这个字，最后决定还是保守治疗。我们看到手术的医生、护士一个一个都出来了，就是母亲不出来。过后我们问母亲，你为什么不出来，母亲说：我跟医生讲，我的儿女都来了，又花了很多钱，你们还是给我手术吧，所以我就等着，再一看人都走了、没人理我，我才下了手术台。母亲求医心切。

![image](https://user-images.githubusercontent.com/79625284/126137779-18ff6b2f-57ab-47c8-b5b6-80e32e16c6bb.png)

母亲于秀华

母亲回来后在一九九八年学了法轮大法。学法炼功后，不知不觉一切都正常了，吃饭也不噎了，跟我们吃饭的速度一样快，这法轮大法太神奇了。母亲高兴的说：当年医大医院不给我手术，那时师父就在管我了，这功我一定要好好学、好好炼、听师父的。

一天中午母亲突然出现象脑血栓的症状，话也说不清了，嘴也歪了，胳膊手都不好使了，我要送母亲去医院，母亲说：不用，这不是病，我相信师父说的，是师父在给我净化身体。我睡一会就好了，你上班吧，我上班心里也不踏实，下午三点我就回家看母亲，两个多小时之后母亲一切都正常了，大法的威力真是妙不可言。第二天母亲还去参加了一个婚礼，回来时高兴的跟我们说：我全身轻松，全都好了。她还诙谐说宴席上的美味佳肴她一样都没少吃。

一九九九年七月二十日中共江泽民团伙打压迫害法轮功后，我与母亲家是一个单元，同一层楼。几次被非法抄家时，母亲都看到了，我被张文信上网上通缉，流离在外，母亲在这种恐怖的氛围中，同时又担心我的安全，当时母亲脸色蜡黄，走路腰都直不起来了，大气都不敢出。后来妹妹把母亲接到了延吉。

二零零六年，原八家子公安局国保队长祝吉成带胡胜利等人去延吉家中骚扰。当时母亲说祝吉成的妻子是我女儿小学的语文老师，女儿的作文写的好。那意思是你不要再来骚扰了，你也有女儿。

二零一一年十一月份，赵志奎伙同和龙市国保警察李某及延边州公安局一行四人来长春利用手机监控跟踪母亲找到了妹妹家，撬门破锁，七十多岁母亲被这突如其来事，吓的浑身颤抖，当母亲知道赵志奎是来抓我的并要带我走时，母亲带着哭声质问赵志奎，你要把她带哪去？她做什么坏事了？。 赵志奎没有回答。那时母亲的身体状况就不好。

二零一五年诉江后，当听到诉状都被两高返回到本地时，看的出母亲压力很大，同时又担心我的安全，不让我离开她，一时看不见都找我。母亲带着这种恐惧和担心二零一五年八月离世，终年八十岁。

二十二年被迫害的家破人亡，妻离子散何止我这一家，真是数不胜数。记得有一个大姨长年哮喘，早期学了法轮功之后，整个人都变了，七十几岁的人白里透红，走路象年轻人一样，都让人认不出来了。可是迫害开始之后，你们用她儿女的工作来要挟她，可怜的老人，为了她儿女们，不炼了，没过多久这位老人就离世了。你们这不是在害人吗？
