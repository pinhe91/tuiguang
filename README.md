### 发生在我身边的奇事

文: 山东大法弟子 

【明慧网二零二二年四月二十八日】本人有幸在一九九八年正月初喜得大法。修炼一段时间后就无病一身轻，修到现在也没吃过一粒药。虽说自己不够精進，但一直得到大法和师父的保护，点点滴滴不胜感激。修炼过程中，有几次在危险来临时得到师父的保护，现在把这些写出来，以证实大法的神奇。

#### 念法轮大法好 岳父得救

我岳父今年八十五岁高龄，六年前突发脑梗，经救治后，勉强能够自理。我经常让他和岳母念“法轮大法好，真善忍好”九字真言，这七年来身体恢复的挺好，不但能够自理，还能够照顾岳母做饭。

二零二零年三月十七日下午，我去看望他们时，发现岳父目光呆板，也不和人交谈，我和他说了一会儿话，突然他牙关紧咬，手脚抽搐倒床上。

我和家人都一愣，随后家人便掐他人中穴，我大声说：快念法轮大法好！家人便和我一起念：“法轮大法好，真善忍好。”大约五分钟左右，岳父清醒了，恢复正常，精神头和以前一样，随后家人便把他送往医院。我知道这是大法保护了他，师父救了他，拜谢师父！

#### 修大法避免一场火灾

我在公司管理一个分厂，二零二零年三月十八日上午，我在厂区巡视，闻到一股焦烟味，我顺着烟味去找，从上面车间一直找到地下仓库，在仓库里面发现了满屋的烟雾，但还没找到火源。这个仓库以前是冷库，现在作为常温库在储存原料。我意识到问题的严重性，如果引起火灾，这些材料是很难扑救的。

当我转身出门要叫人时，忽然发现门框底部往外冒烟，虽然没有发现明火，但烟很呛人。我马上关上门，防止空气流通，打电话让人下来救火，不一会儿几名员工快速跑来，用水很快把烟灭掉，事后我查明原因，是两名员工在早晨提原料时在仓库门口抽烟，随手把烟头扔在门框边上引起的，被我发现时已经有半个小时了。

事后员工们谈论起来都很后怕：“这满屋子原料这么长时间都没引起火灾来，这真是神了！”我知道这是师父在保护，替我挡了这一难，我对在场员工讲：你们记住“法轮大法好”吧，这是大法在保护。员工们纷纷点头，说：“法轮大法好！”

#### 念九字真言 孩子奇迹般考上研究生

我孩子从小时师父就给他开天目了，每次发正念时就会看到天龙和天兵天将，并能和他们沟通。随着年龄的增长和学习的加重，逐渐脱离了法，随后师父也把天目给关闭了，但孩子心中也一直装着大法并很支持我修炼。

去年孩子参加研究生考试，由于报的专业比较高端，竞争压力很大，初试成绩刚过线1分，排名在392名，学校只招收300人，这样对于参加复试的他压力很大。在复试前一天晚上，我告诉他不要忘记念九字真言，他说好，记住了。考完试后他说考得不好，很沮丧，准备来年再考一次。

三月十九日下午三点，学校下成绩了，儿子惊喜的告诉我，这次复试成绩太好了，总成绩排名在二百名上，已经够录取资格了，儿子直呼：“这真是奇迹！太不敢相信了！”我说：“这是你诚心念九字真言的结果。”儿子说：“是！是大法在帮我，感谢师父！感谢大法！”

修炼以来，时常都有神奇的事在我身边发生，把它写出来，让世人了解法轮大法好，这是世人的真正护身符，目前这个疫情肆虐，多灾多难的世上，只有明白法轮大法真相才是真正的被救度，只有法轮大法在救人，愿世人早日了解真相，得到大法护佑，愿众生平安！
