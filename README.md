### 永远难忘的记忆

文: 陕西大法弟子　云中 

【明慧网二零二二年五月十一日】一九九四年八月五日，我幸运的参加了师父在哈尔滨冰球馆的传法教功班。虽然过去二十多年了，每每回忆起来，就好象发生在昨天一样，那么的清晰，我是还那么的激动和幸福。
一九九三年八月下旬，我做了一个清楚的梦：我梦见一位四十岁左右的男子，比我还高（我身高1米82），脸色红润，穿着一件天蓝色的上衣，领着我往东北方向走。我们越走越高，山上都是杨树，地上的草很好看。我一边跟着走，一边说话。这位男子一会儿回过头来看看我，冲我笑，也不说话，然后又向前走。一会儿，我就醒了。

一九九四年八月五日，我有幸参加了师父在哈尔滨冰球馆办的传法班。八月六日，师父讲完法休息时，在冰球馆的西北角站着，看着大家。我坐在西侧四排十九号，距离师父约有三、四十米远。突然，师父一下子站在了我的面前，我面对着师父，半米距离都不到。师父的面色红润，很年轻，很英俊，面带慈悲。

我当时感到师父就好象是多年不见的亲人一样。我想，这不是我梦里见到的人吗？！我激动又高兴，眼泪一下子流了下来，我哭了，眼前模糊了。等我再想看清楚师父时，师父还在西北角站着，还是那么远。

师父在讲第二节课时说：我让大家体会体会没病是什么滋味。然后，先从西侧开始，师父让大家站起来，要求动作一定要整齐，然后师父喊：“一、二、三”，师父让大家抬起左腿猛一跺脚。然后，抬起右腿，师父再喊：“一、二、三”，大家再猛一跺脚。跺完脚后，师父让大家活动活动。

当时我真实的感觉到了神清气爽，舒服极了！当一跺脚的一刹那，我看见师父用右手一抓，就把我们身体里不好的东西给抓出去，清理了，师父真是太慈悲了。然后，师父又给东侧的学员清理身体。这是我亲身经历的。四千多人的会场，这激动人心的情景，就可想而知了。

第三天早晨，我身体就有了反应，发低烧，象感冒一样，浑身不舒服。老学员说，这是师父给我净化身体，不是感冒，不用吃药，几天就好了。就是这样难受，在听完课回家的路上，我就象小孩子一样，一会跳到马路沿上，一会跳到马路上，我高兴的蹦啊！跳啊！得法的那个高兴啊，真的无法用语言来形容。那时，我已经五十三岁了，真的是发自内心的喜悦啊！因为我有师父了，我真的找到师父了呀！

师父讲第三节课时，由于拉近了灯光，灯光射到了师父的背上，我看见师父时不时的提提衣领。因为天热，再加上灯光的烘烤，师父的衬衣都粘到了后背上。但师父为了学员们，自己承受着灯光的烘烤，时时处处都为学员们着想，师父真的是太慈悲了。

一九九四年八月十二日，学习班结束了。八月十三日，我住到海林市的亲戚家，准备参加师父在延吉办的传法学习班。这时，我还在发烧，反应还没过去。

一九九四年八月十九日，我们坐汽车到了牡丹江，再坐火车经图们，去参加延吉的学习班。路上我心想，这样发着低烧，影响听课了可不好。

到了牡丹江一下汽车，我的身体一下子正常了，好了。真的是太神奇了。

我们同去参加学习班的还有亲戚家的五个人，其中四个人晕车。我们带的吃的东西很多，还有两个大西瓜。到图们下火车时，东西都吃光了。这时，我们才想起晕车的事。大家觉的很奇怪、很神奇。而且其中有两个人平时不是一般的晕车，很严重。

其实，在我们参加学习班的路上，师父就管我们了。

延吉班一九九四年八月二十七日结束。我们回到关里，同去的亲戚们回到海林市。一路上，没有一个人晕车。他们从海林市打来电话说：“晕车都好了。”当时我们还没有到家。一路上我的心情很激动，我终于得到大法了，我有伟大的师父了！我终于知道了什么叫修炼和怎么修炼了。

回到家，我非常兴奋，就急着告诉别人修炼法轮大法。很快，我们当地就成立了学法小组和炼功点。我们向有缘人洪法，同时放师父的讲法录音和录像，法轮大法在我们当地就洪传开了。

原文 https://www.minghui.org/mh/articles/2022/5/11/永远难忘的记忆-439812.html
