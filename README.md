### 炼法轮功牙髓癌患者得救　坚持信仰遭枉判五年

——河北承德市郭鑫女士被劫持入狱

【明慧网二零二二年二月二十六日】（明慧网通讯员河北报道）河北省承德市双桥区法轮功学员郭鑫于两年前被绑架，后被当地法院非法判刑五年，一度被监视居住。二零二二年二月十六日，郭鑫被劫持到河北省女子监狱关押迫害。

![image](https://user-images.githubusercontent.com/79625284/155837904-4a21cb4e-8d97-48b2-9044-d005e4eb74dc.png)

郭鑫女士

二零一九年一月十一日，双桥区国保大队警察闯入郭鑫家中非法抄家，并将她绑架到看守所，因体检不合格，看守所拒收郭鑫。半年后的二零一九年七月十日，双桥区法院非法庭审郭鑫，承德市检察院检察员刘长缨出庭公诉。郭鑫为自己做了无罪辩护。二零一九年七月二十三日，双桥区法院下发判决书，对郭鑫非法判刑五年，处罚金一万元；郭鑫在家监视居住。

二零二一年十二月十日下午，双桥区国保大队副队长陶怡博带两名国保警察（一男一女）将郭鑫拉去医院强行做核酸检测及检查身体，又拉她去法院补办手续，然后将她非法关押在承德市看守所。二零二二年二月十六日，郭鑫被劫入河北省女子监狱迫害。

郭鑫在法庭上为自己做了无罪辩护，用翔实的事实讲述了自己为什么修炼法轮功。她曾经患有牙髓癌，牙龈上长着小白脓包，整个上牙膛封满了黄油般的脓，疼得她心里都哆嗦，满口的牙全部松动，说话都怕牙掉下来，牙根碎了，门牙向外支出，脸都走形了……医生诊断她活不了两年。

被极度痛苦逼得万般无奈的郭鑫，想到了修炼法轮功。几年下来，牙髓癌症状消退了，口腔的黄脓消退了，萎缩的牙龈长起了，活动的牙齿稳固了……她活下来了。

郭鑫告诉法庭里所有的人：修炼真善忍无罪，我是一个好公民，一个完全合法的人。

#### 下面是郭鑫的辩护词（有部份删减编辑）

我是郭鑫，今年六十一岁，家住承德市又桥区。

几年前，我的牙总流血，用止血牙膏也不管用。后来下颚处长了一个瘤子，越长越大，发展到鸡蛋那么大，发现舌头底下靠着左侧下牙根也长了一个约二至四厘米内瘤子，与外面约五至七厘米的外瘤子内外相应。这个外面的瘤子，很硬，像石头一样稳固不动。经附属医院拍片检查，大夫告诉我说、边缘不清晰，根向里扎，后被家人和大夫告知是牙髓癌，治疗方法是：满口牙都得拔了，腮帮子肉都得提出去。意思是说我活不了两年。

我家里的亲人们不惜倾尽家财为我治疗，想拿钱买回我的命。可是好心人和朋友告诉我说：不能手术，开刀就扩散。并且说我这个瘤子是割了还长的那种，说有一个人是割到第四次的时候就死了。听大夫说，有一个和我一样类似的患者，不到五十岁，牙拔了五颗，下巴骨锯了，做完手术后不到半年，癌细胞就扩散了。到北大口腔医院检查，盼望能被救治，可是大夫说癌细胞已经扩散到眼睛里、脑子里了，没办法再做手术了。那人只能回家靠吃吗啡止痛度日，只能顺着腮帮子上烂的洞往嘴里倒点杏仁露续命，一次吃四片吗啡也止不住痛，忍不住疼得大叫。

我拒绝了那种酷刑般的治疗。唯一的出路我只能继续修炼法轮功。

回到家中，牙越来越痛，牙根疼得炸开花了，像蜂窝状的碎石，掉下来的有象玉米碴子大小，我现在还留有几块，牙龈上长着小白脓包，能挤出象针那么细的脓，连着牙中心都感到特别疼，满口的牙全部活动，疼得吃粥心里都哆嗦，说话气流大了牙都怕掉下来。整个上牙膛封满了像黄油板一样的脓，上牙床变形，导致整个门牙全部错位，整个偏移了中线，牙撑开的缝隙，能塞进手指尖，门牙向外支出四十五度角，嘴闭不上，脸部都走形了……那时候我的家，春夏秋冬二十四小时都是开着窗户，因为室内奇臭无比，有经验的人说是“殃气”。

我儿子怕我死了，失声痛哭，说：“我才这么小，你就让我没妈了？”“你要死了，我可怎么办？”看着我的样子，他每天都是惊恐的眼神，提心吊胆地过日子，还常常做噩梦，梦到我的生命到头了。我的脸色非常难看吓人，蜡黄，没有一点生气。那种身心的压力伴着剧痛、煎熬的每一天、活不成死不了的悲惨，真实的让我感受到生命的脆弱与可怜，这有病也太悲惨了，无法形容。

牙疼得我整个身心、大脑、每个细胞及生命的深处都处在痛苦之中。

有一次最严重的时候，痛得我夜间三点多在床上折腾，睡不了觉，怎么呆着都不能减轻痛苦，每分每秒都在剧痛中难以忍受，我万般无奈，给师父跪下了，心里求师父：救救我！瞬间，只听脑子里一声炸雷，我就什么都不知道了。第二天早晨醒来，好像什么事也没发生过，根本没有牙痛的事，就感到我这辈子从来没有睡过这么香的觉。

从那以后，我的牙痛从最痛的高峰开始回落，减轻到只要我学法炼功，疼痛就能忍得住，静坐止痛效果超过吗啡。我知道是师父的无量慈悲和智慧，为弟子承担了这可怕的病痛，化解了我的罪业，让我从要命的痛苦中解救出来。

师父每天都给我净化身体，早上或深夜，都吐出带臭味的痰或血，还有小瘤子或血粒子，大一点的时候，比高梁米粒大一些，非常圆的小肉瘤子，带着细线一样的根较长，小的时候，根是粗的，象蝌蚪形状，再小的就象血粒子，几年中吐出的数量，可以说成百上千。我说的这些是看的见的，我留有部份照片，可以想象，这么庞大的病灶，我有一百条命也活不过那些长在肉里带根的瘤子。

可是我幸遇大法，幸遇师尊，是师尊帮我把瘤子净化处理了，是师尊把我从无尽痛苦的深渊中救起，我生命延续的每一天，都是任何医疗手段所达不到的。是法轮大法创造了我生命的奇迹，没有大法就没有我生命的今天。

目前我下颚处的大包还在，但用人们的话说：收口了。原来被黄油般的脓封满的上牙膛，变成了鲜嫩的肉，萎缩的牙龈也长出了许多，满口活动的牙有部份稳固了，口腔臭味基本上消失了，我家也没味了，牙也不那么痛了，整个身体向良性方面转化。

我历经了生死的痛苦折磨，大法使我从难以想象的病魔漩涡中解脱出来。我亲身见证大法能祛病健身是真正的科学，不是迷信与唯心，“真、善、忍”会给生命带来希望与美好，如果人们都相信“真、善、忍”做好人，那社会就好了，真正的和谐了，没有犯罪的了，事实证明了这一点。

我儿子被朋友坑骗三十来万，孩子非常生气，想采取报复手段。我告诉孩子，你妈是修大法的，那些炼大法的人被车撞坏的，都不讹人钱，咱们身体啥事没有，就善待、忍让人家吧。我儿子愤恨的心平衡了，我在大法中升华了道德，给社会家庭带来了稳定，我家的这一场经济风波也得以平和解决。

炼法轮功的人没有一例刑事犯罪，没有黄赌毒，偷抢骗，杀盗淫，没有不好。这个事实有目共睹，大法弟子在做好人，做最好的人。天地可鉴，好人何罪之有？相反大法弟子高尚的道德、情操感动着社会以至全世界的人，世界需要“真、善、忍”！

我用微薄的收入，在如此的压力下，印制了小台历，是希望那些象我一样病痛的人，走到生命绝处的人，给他们一线得救的希望，能祛病健身，能脱离苦厄，起死回生，能绝处逢生。

我想说真话：中央电视台播报的天安门自焚是假的，是欺骗全国人民的。法轮功是救人的佛法，是禁止杀生的，更不会自焚和自杀。法轮功是被冤枉的。

我所做的这一切没有任何政治图谋，没有个人所求，只有救人的份，只有为别人好的份。绝不是犯罪，完全是为了别人好，对任何人、任何生命是有利而无害的，是做善事、好事，应该得到社会的认可和表彰。

各级法官请明辨事实，秉承道义、尊重法律、坚持正义与真实，不坑害善良与无辜，不让好人蒙冤受难。

在我国现行法律中也能证明修炼法轮功者无罪：

公安部份别于二零零零年、二零零五年两次发布《关于认定和取缔邪教组织若干问题的通知》（公通字〈2000〉39号、公通字〈2005〉39号）。这两个文件认定的 “邪教组织”中并没有法轮功。

二零一一年三月一日，国家为了依法行政中国新闻出版总署署长柳斌杰发布《新闻版总署令第50号》，公布《新闻出版总署废止第五批规范性文件的决定》，该决定明确废止了一九九九年的两个有关法轮功出版物处理意见的通知。根据这个规定，法轮功书籍都是合法的。

依据《宪法》第三十五条：“中华人民共和国公民有言论、出版、集会、结社、游行、示威的自由”、依据《宪法》第三十六条：“中华人民共和国公民有宗教信仰自由”，公民拥有、阅读和传播法轮功书籍也是合法的。

翻遍中国法律没有一条法律认定法轮功违法，我不构成犯罪。我是一个好公民，一个善良无辜的百姓，一个完全合法的人。

请求：把从我家抄走的私人物品，照片、书籍，退还给我。

辩护人：郭鑫
二零一九年六月二十四日
