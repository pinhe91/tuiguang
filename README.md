### “师父在哪？我能见到师父吗？”

文: 大陆大法弟子 李梅 

【明慧网二零二二年十月十六日】

#### “师父在哪？我能见到师父吗？”

昨天早晨我去早市讲真相救人。看见一个卖毛桃的小伙子，我想给他讲真相，就买了几个毛桃。我说：小伙子，你这么善良，咱们也是缘份。大姨告诉你一个保命的良方。他问：“什么良方？”我说：“记住九字真言：法轮大法好，真、善、忍好，天灾面前能保命。”他说：“行。 ”

我又说：“上学的时候入过团队吗？”他说没有。我说：“那你戴过红领巾吗？”他说戴过。我说：“那我给你起个化名退了吧。因为这个组织从建政以来利用各种政治运动杀害了八千万人。善恶有报是天理，老天爷要惩罚它，你戴过那个东西会跟它遭殃。你心里退出来。用有缘加上你的姓退出这个组织，行吗？”他说：“行。”

我从兜里掏出护身符给他，他伸出双手在那等着。我递给他：“这是福字，是法轮大法护身符戴在身上瘟疫来了保命。”他连连说：谢谢！谢谢！我说：“告诉你的家人，亲朋好友都记住：法轮大法好，真、善、忍好能保命！”他又连声说：谢谢！谢谢！我说：“你要谢就谢大法师父！救人的是大法师父！我只是传大法的福音。”他问：“师父在哪？我能见到师父吗？”我说：“能，一定能。”他说：“太好了。”高兴的要蹦起来了。

#### 二、“再给我两个，我给别人”

接下来，我又碰到一个卖黄瓜、茄子的人，给他讲真相。他说：啥也没入过，我最烦它了。我说：那你就记住“法轮大法好，真、善、忍好”。他说行。我说：再送给你一个法轮大法护身符，瘟疫来了保命。他说：谢谢！接着又说：“大姐，你再给我两个行吗？我给别人。”我又给他两个，并告诉他：你传大法的福音会得到福报。并祝他平安！买卖越来越顺利！

这一次共讲的四人。回去的时候我的小三轮电瓶车欢快的往回跑，我并没有给它加多少电。我知道这是师父对我的鼓励，心里说：谢谢师父！
