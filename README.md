### 大法归正人心　稳定家庭和社会

文: 大陆大法弟子 圆圆  

【明慧网二零二三年二月六日】法轮大法能归正人心，提升道德，福益家庭，福益社会，对任何民族和国家都是有百利而无一害的。愿普天之下所有人都牢记“法轮大法好，真善忍好”，躲过乱世之时的末劫之灾，走向美好的未来！
今天我和大家讲述发生在我身边的两个真实例子。

#### 看完大法书，放弃离婚想法

鸿海（化名）精明能干，有头脑、爱思考。中共迫害法轮功的初期，鸿海去北京，路上发现，乘客上车前必须踏过大法师父的照片，证明自己不是法轮功学员。他就想：李洪志到底是什么人物？共产党掌握着全部国家机器，为什么这么怕法轮功？用这种卑鄙的手段阻止法轮功人员上访，法轮功到底是咋回事？

二零一一年，我遇到他，给他讲了大法真相：法轮功祛病健身有奇效，要求学员按照大法真、善、忍法理做好人，对提升道德非常有好处，通过人传人心传心，法轮功从一九九二年传出到一九九九年，短短七年就有上亿人修炼，江泽民出于对法轮功创始人的妒嫉和对炼功人数众多的恐惧，发动了这场毫无理性的打压和迫害，法轮功是真正的佛法，在打压如此严重的情况下，你只要诚念“法轮大法好，真善忍好”，神佛就能保佑你。鸿海听了真相欣然接受，并恭恭敬敬接过大法护身符，从此他经常念这九字真言。

一个多月后，他高兴的告诉我：“真的有神，你给我的护身符救了我的命。”他给我讲了事情的经过，他开的面包车与大车相撞，车翻了两个个儿，车坏了，他却毫发未损。他问我还有法轮功的东西吗？我送给他大法书，看完后他觉的太好了，相见恨晚，决定要修炼大法。

几天后，鸿海找到我，说他有件难以启齿的事，他从未提过，任何人也不知道，他有位婚外女朋友，他俩相知相爱几年了，女朋友说随时可以离婚，可他还没下狠心，觉的对不起妻子和孩子。他很矛盾，又想修炼法轮大法，又放不下女朋友。我说：法轮大法是万古难遇的修炼大法，错过机缘悔断肠，要想修炼就得改正一切不正的行为，尤其这种不正当的两性关系应该当机立断了结，不能再犹豫。他说怕伤了女朋友的心。我说：你向她说明理由，既了结了你们这段恶缘，又证实了大法，还救了她，她会从内心佩服大法的。

大家都知道这句话“劝赌不劝嫖”，意思是说这种行为很难改，当事人会恨你的，可是鸿海看了大法书，就知道这种行为的严重性，放弃了离婚的想法，妥善的处理了这件事，挽救了两个家庭。现在两个家庭都和和美美，鸿海成了坚定的大法徒，还引导多位有缘人走入大法修炼。

#### 流浪青年明真相回家

一次，我推着孙子在街上走，路边有位妇女卖豆腐，旁边很多人在唠嗑，旁边坐着的一位小伙子突然站起来，向我走来，他说：“大姨，你给我两元钱行吗？”我给了他两元钱，他接过钱，买了一块豆腐，双手捧着，狼吞虎咽的几口就把豆腐吃完了，看样子他很饿了，还没吃饱，他又看着我，我示意他过来，我把兜内的零钱掏给他，大概二十来元，他又买了一块豆腐吃了。

我把小伙子叫过来，问他：“你怎么饿到这种程度？你是哪里人？”他说：“我没有亲妈，也没有亲爸。”我不解的问：“那是咋回事？”他说：“我是现在的父母从筐子里捡回来的，他们对我很不好，经常打骂我，我不想在家呆了，就出来了，身上带的钱早花光了。”我说：“那你回家呀！”他说：“宁可死也不回去。”我说：“孩子，你可不能这样想，看你多可怜哪，现在谁愿意给年轻人一口饭吃呀，认为是好吃懒做、不务正业，才混到这地步的。你父母虽然脾气不好，但毕竟把你养这么大，养育之恩怎能不报，一走了之？没有他们，也许你早就不在人世了。快回家吧，你养父养母一定焦急的盼着你回家。”

他眼睛湿润了，接着说：“我长这么大，从来没人这样跟我说话，你咋这么好？”我说：“我是修炼法轮大法的，是大法师父叫我们在哪里都要做好人，按真、善、忍修自己，说话办事为他人着想，如果不学大法，我也不会对你这样。”我又给他讲了很多真相，他彻底明白了法轮大法好，并退出了曾经加入的团、队组织，一个生命真正得救了。他说：“我没有亲妈，没人疼爱，遇到你就像遇到亲妈，我能叫你一声妈吗？”他哭了，我也落泪了，他甜甜的叫了我一声“妈！”他还说明天就回家，将来有机会也学大法。

每每想起这件事，我就会思绪万千，一个没有亲生父母的孩子，受了这么多的苦，遭了这么多的罪，而我在共产邪党的计划生育政策下，强行堕胎，打掉了两个还在腹中只有几个月的孩子，他们还没出世就被杀害，在另外空间里苦等苦熬，我一定好好修炼，修成圆满，福报两个可怜的孩子。现在中国大陆还有多少孩子，因父母修炼被迫害致死或被非法关押，失去应有的父爱和母爱。

愿善良的中国人和国际上的正义组织及人士，站出来，伸出你们的援手，共同制止这场发生在中国大陆对法轮功善良群体的迫害。
