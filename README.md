### 明真相的警察帮助救人

文: 大陆大法弟子　佛育

【明慧网二零二三年六月五日】二零二一年夏季的一天，我在集市的蔬菜市场上讲真相时，碰到一个受邪党谎言毒害的老头儿，七十岁左右，他不但不听真相，还阻碍别人听真相，说了许多对大法不敬的话，而且还编造了一个谎言，说他们村有一个炼法轮功的叫什么名字，如何不孝什么的。其实他说的那个同修我是知道的，根本不是他所说的那样。我就说：“咱俩一块去见见这个人吧。”这时他心虚了，转过话说：“你和我到派出所去一趟吧。”当时我说：“像你这把年纪的人说谎遭罪图个啥呀，谁不知道我们炼法轮功的人在哪都是在做好人哪。”他红着脸要拉我去派出所。
这时一个穿警服的老警察上前挡住老头儿，说：“你是哪个村的？不要这样。”这位警察说自己是哪哪的，是国家干部，他对老头儿说：“炼法轮功的他们都是好人，他们讲的都是真的，那年我在医院做手术，刀口两年不愈合，流黄水。是一个炼法轮功的告诉我念‘法轮大法好、真善忍好’，说诚心念就能痊愈，我只念了两个星期，肚子上的刀口就长好了。”

警察又说：“咱们都是三里五乡的，你这是在干啥呀？干点正经事吧。”就这样，老头儿的邪气一下子消散了。

这位有良知的警察问我：“你是哪个村的？给我一张真相资料吧。”我说：“你听说过三退保平安吗？”他说：“你给我退了吧。”

旁观的人说：“今天我可真正的看到了法轮功是什么。”在当时正的能量场作用下，有很多人要了真相资料，得救了。

（责任编辑：文谦）
