### 来自上天的洪恩与警告

归元

【正见网2022年11月16日】

中华民族有五千年的辉煌历史，记录并承传了诸多神传文化，因此被誉为神州大地。可是现在的中国人不信神，是因为中共篡政以后，对中国人疯狂洗脑，把无神论、进化论、唯物论、斗争哲学等马列邪教思想强行灌入中国人的大脑中，几十年来的十多次大规模政治运动，不仅铺天盖地的媒体暴力轰炸，还动用坦克、机枪等武器对民众进行残酷的镇压，把百姓真的整怕了，没人再敢向中共要民主、要自由。

在上世纪八十年代兴起的气功热中，人们体验到了一种能够祛病健身的古老身体静修文化，特别是一九九二年五月法轮功传出之后，受到广大气功爱好者的青睐，法轮功超凡脱俗的神奇功效，使这些最先体验法轮功的学员，纷纷转告他们的亲朋好友，亲自去听李洪志师父的讲法，就这样一波一波的向外传，短短七年就传遍了大江南北，上亿人受益。

时任党魁江泽民，看到法轮功发展的迅猛势头极度的恐惧，从一九九六年就利用《光明日报》的评论员文章污蔑法轮功，试图阻止百姓学炼法轮功，结果没能奏效。公安派特务打入法轮功内部跟学员一起学法炼功，结果也是一无所获。又派乔石等老干部带领调查团下基层调查法轮功，结果得出的结论是：法轮功对社会有百利而无一害。然而，这哪是江泽民想要的结果呢？他是想找法轮功的把柄从而取缔法轮功。几计不成之后又派何祚庥捣乱。第一次由何祚庥引起的北京电视台事件，当地法轮功学员去电视台讲真相圆满解决。第二次就是引起万人四二·五上访的天津事件，从此中共江泽民集团对法轮功的迫害从地下转到地上。六月十日，江泽民成立了专门迫害法轮功的“610办公室”，直接听命于江泽民，七月二十日就开始对法轮功大打出手了。为了激起民愤，江泽民还给正法修炼的法轮功扣上了“x教”的帽子。

即使这样，迫害还是难以为继，江泽民、罗干等邪恶集团又精心策划了更恶毒的办法，于二零零一年一月二十三日，导演了天安门自焚惨案嫁祸法轮功，瞬间激起了没有头脑、没有分析能力的中国人对法轮功的仇恨。更加没有人性的残酷迫害开始了。

此后，来自上天的警告频频出现，提醒着善良的人，擦亮双眼，看清真相，不要与邪恶为伍。

#### 藏字蛋蕴天机

中共嫁祸法轮功的天安门自焚伪案播出以后，当年春天，谷雨节气那天，黑龙江省农垦建三江管理局属下的七星农场，刘学顺家中的鹅下了一个带字的蛋，当时老伴并未发现，在她洗蛋准备腌蛋时，突然发现蛋上有字，并且清清楚楚、工工整整的四个大字“神已来到”，且蛋的一端尖部还有一个“王”字。

消息传出，引来当地三江电视台拍片、录影，确认真假。当地电视台甚至找来农大教授监别此蛋，老教授用刀试图刮字，却未能刮动，最后肯定此蛋不是人工所为。随后附近农场就有上千人前来看蛋。 当地官方为了阻止人们再来观看，在电视台播出新闻，谎称“藏字蛋”是假消息，叫人不要相信。

事后一名电视台工作的人员透露，“共产党不信神，所以（此事）不能宣传报导。”

上天有好生之德，总希望迷失的众生能回归神的怀抱，于是，就把真相写在鹅蛋上，一是提醒众生无神论是谎言，赶紧寻找真相，来聆听神的教诲吧。二是警告中共真神来了，别再抱着无神论思想做坏事。可惜当地官员楞把眼睁睁看到的事实，说成是假消息，那些亲眼见到过“藏字蛋”的人知道谁在撒谎。这不就是共产党自断生路了吗？

#### 藏字石示定局

二零零二年六月，在贵州省平塘县掌布乡发现了一块“藏字石”，据专家鉴定该巨石是两亿七千万年前生成，五百年前滚落崩裂的巨石断面内，惊现六个排列整齐的大字“中國共產党亡”。中央电视台曾组织科学家前往考察并做了报道。经地质专家考察一致认为，六个大字乃天然形成，没有人工雕琢的痕迹，堪称旷世奇观。这是不是神给人的启示呢？

“中國共產党亡”几个字一旦被人发现，就说明上天给共产党盖棺定论了，不可再变。加入其组织的党、团、队成员，就是它其中的一份子，如不赶快退出来，到天灭中共时，就会随中共一起被销毁的。这个关系到自身生命存亡的大事，还不用心选择美好未来吗？

#### 优昙婆罗花

中共残酷的迫害，挡不住神迹的发生，两千多年前释迦牟尼佛的预言如期而至。佛经记载，有一种非常奇异的花叫做优昙婆罗花，三千年开一次，当优昙婆罗花开放的时候，转轮圣王即来到了人间。

二零零五年二月中旬，韩国顺天市海龙面须弥山禅院观音佛像上开了十朵优昙婆罗花。须弥山禅院高僧已经郑重告诉世人，人世间真的存在这种佛花，并且就在当世千真万确的开了。此高僧欣然肯定，转轮圣王正在人间正法，他是以转轮支配世界的理想王。这表明历史上佛来世间度人的预言已成事实。

此后正见网上报道了很多很多生长在世间花不可能生长的地方，如玻璃门窗上、铁丝上、西瓜上等等，等待了千万年的机缘就到眼前了，人们盼望已久的神圣之事已经来到了我们的生活中，也许哪个法轮功学员就在给你讲述真相，也许哪个法轮功学员找送给你资料，也许你的车筐、门把上已经给你放好了真相小册子，真心希望你能万分珍惜，这就是你得救的唯一希望。

#### 天鼓鸣 灾难近

中共二十大刚刚结束，中国大陆多地开始发生一系列密集的天鼓鸣事件：10月24日下午6点安徽阜阳巨响一声，当日阜阳市地震局发说明称当地24日晚上无天然地震发生。25日阜阳市政府称，已排除地震引发巨响的可能，具体原因还在调查。

10月25日上午9点上东潍坊巨响两声。

10月25日下午6点河南开封巨响两声，应急管理局官方通报称，当天晚间6时30分左右，开封市区传出巨响。经排查，未接到安全事故报告。

此外，浙江、湖南、内蒙等地网民也表示有巨响。

巨大的声音令人感觉连地面都会振动。新浪网文披露，网民说起初以为是打雷，但开窗户看外面未见下雨。还有民众边拍摄边说：“大家都出来了，都害怕，不知道怎么回事。家里的门窗都‘轰’一下，吓死了。”

河南、安徽巨响均登上大陆百度热搜榜，但官方没有回应。

《史记·天官书》记载：“天鼓，有音如雷非雷，音在地而下及地。其所往者，兵发其下......”。 《天镜》记载：“天鸣主死，百姓哭”。近来我国多地听到不明原因的巨响，这是不是传说中的天鼓鸣呢？朋友们根据当前的时局去悟吧。

#### 血月的警示

近几年来，昭示灾难警示的日食、月食频繁出现，不断警告着党魁悬崖勒马，因中共对人民的无底线残暴，尤其是对修炼真、善、忍的法轮功学员惨无人道的迫害已招天怒。《汉书·天文志》曰:“凡日食修德，月食修刑。”即：发生日食，说明帝王“失德”；发生月食，说明国家刑罚有误，冤情太重。今年十一月八日又逢月全食，且是血月，即是上天的警示。停止迫害法轮功，立刻释放被非法关押的法轮功学员及其他被非法关押的中华同胞，才能避开灾难。

人是神的子民，上天无时无刻不在关注着世间人性道德的变化，当中共走完了它罪恶的一生的时候，人民就应该认清它了。如果你迷失太深，至今还看不清中共真面目，就请看看海外大纪元发表的《九评共产党》、《江泽民其人》及《共产主义的终极目的》等系列文章吧，如果你能够静下心来看，就会一目了然了，你就不会与这样一个旷世独一无二的邪魔为伍了。退党、退团、退队才是你明智的选择。
