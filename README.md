### 小舅诚念“九字真言”得福报

文: 中国大陆大法弟子 

【明慧网二零二二年六月十六日】在小时候的记忆中，小舅个子又高又瘦，总是一副病怏怏的样子，好像一阵风就能把他吹走。小舅今年已经八十五岁了，患有低血压、胆囊炎、前列腺炎等多种疾病，曾做过两次手术。因为我们两家老人的关系，与小舅家一直走的比较近。
二零一一年我有幸步入大法修炼，随着自己不断的学法修心，逐渐明白了众生都是为法而来，都在期盼着得法救度，修好自己救度一方众生是修炼人的责任，觉的应该告诉小舅真相了。

二零一五年的中秋节快到了，心想如果能放假就好了，一来可以去看望老人，二来可以当面和小舅聊一聊，让他明白大法真相。单位居然还真的放假两天，就这样我在八月十四这天买上东西，到乡下去看望小舅。

小舅非常高兴，很热情的招待我。谈话之间很快聊到中共邪党统治下，民不聊生，贪官污吏横行，从中共靠斗争夺权非法建政开始到历次群众运动，杀害了无数的中国人，直到九九年迫害大法修炼者，造成天怒人怨。我讲到了大法是什么，共产党迫害大法的因果和中国人为什么要三退。

小舅知道共产党很坏，也相信善恶有报的道理，很爽快的退出曾经加入过的团队组织。最后我告诉老人要牢记大法好，诚心敬念大法“九字真言”，选择美好未来。为了能够记住，小舅让我给他写出来，并认真的读了起来。我给了他一张大法护身符，只要你每天诚心去念，就会身心健康有福报。小舅笑着说我听你的。从这以后，小舅总是随身携带护身符，身体越来越好了。

后来，如果有时间，我就去看望小舅。记得有一次，我去看小舅，打开院门，从窗户就看到他斜倚在棉被上，隐约听到好像有说话的声音从屋里传出来，我推门進去，和小舅打了声招呼，发现屋里并没有别人啊，就问：小舅，刚才我在院里听到你屋里有人说话，难道是我听错了？小舅笑着从上衣口袋里摸出了大法护身符说，我刚才在念大法护身符上的字，现在都背过来了。

接下来，小舅给我讲了他年前的两件事。一件事是他患了前列腺炎，到市医院检查治疗，躺在椅子上用导尿管排尿，很受罪，挂了半个月的吊瓶也没好。小舅的一个亲戚是乡村医生，给他治也不见效，没治好。回家后，小舅就反复地背诵大法护身符上的字，背着背着不知不觉就好了，也不痛了。从这以后身体一天比一天好起来。康复后的小舅身心健康愉快，心情特别好，就想去看望我大姨，因为我大姨今年都一百周岁了，年前走路不小心跌倒腿摔折了。小舅一人步行三里路来到我姨家，大表兄感到非常惊奇连忙问：您是怎么来的呀？小舅就把自己求医不得，“念九字救命真言”康复的经过告诉了他们，都觉的大法真是太神奇了。
