### 中共警察审问两岁孩子：“你姥姥还炼功吗？”

——内蒙古法轮功学员王艳华屡遭迫害经历

更新: 2022年07月10日

【明慧网二零二二年七月十日】（明慧网通讯员内蒙古报道）内蒙古法轮功学员王艳华女士，因为坚持法轮大法真善忍信仰，屡次遭到中共警察骚扰、拘留。往往一个正常的出行，也会遭到千里之外的警察跟踪、通报、拦截。一次，王艳华与家人在火车站被警察拦截，一警察头目竟将她两岁的外孙单独带入小屋审问：“你姥姥还炼功吗？东西都放哪了？”影视作品中的恶魔，真实地在中华大地出没。
王艳华一九六七年出生，现年五十五岁。她是内蒙古呼伦贝尔市根河市金河镇法轮功学员，目前居住在河北省廊坊地区三河市燕郊镇星河浩月小区的女儿家。以下是王艳华女士屡遭迫害的事实：

#### 三岁女童哭诉：“警察去我们家了，我不敢回家了”

二零二二年七月四日晚八点钟，三河市燕郊镇燕顺路派出所两名戴着口罩的男警察，闯到王艳华女儿的家，进屋就问：“你是王艳华吗？你二零一五年和二零一七年是因为发传单被拘留过吗？”王艳华反问：“你怎么知道的？”警察说是根河市当地派出所给打电话了，并问：“你还炼法轮功吗？”王艳华的丈夫（未修炼法轮功）马上回答：“早就不炼了。”那个警察说：“必须她自己说不炼。”他又问王艳华，王艳华说：“拒绝回答。”“你还和哪些人联系？”一个警察在作笔录，另一个警察在拍照。王艳华说：“你们侵犯人权。”那个警察还在拍照，还照坐在沙发上的小孙女。走时一个警察说：“还有人来给你备案。”王艳华对他们说：“迫害好人有罪。 ”

三周岁的小孙女吓得直哆嗦，问姥姥：“是好警察吗？姥姥你不是说警察保护我们的吗？”王艳华说：“他们很可怜，被利用了。”

第二天上午，好几个警察开着警车又来了，他们敲门，见没人给开门，就叫嚣：“不给开门就撬门。”等了一会，还没人开，警察就把电闸给拉下来了。他们一看家里真是没有人，才把电闸又拉上了。

王艳华的家人受到如此骚扰，三岁的孩子吓得不敢回家，看见谁都哭着说：“警察去我们家了，我不敢回家了。”闻者都很气愤，说：“是哪些人干的，查一查，孩子吓坏了，得追究责任。”

#### 警察审问两岁孩子：“你姥姥还炼功吗？”

二零二一年九月三十日晚二十一点半，王艳华和丈夫领着两岁的外孙，乘坐从金河开往海拉尔的火车。第二天早晨到海拉尔。下火车过出站口刷身份证时，王艳华一行被四、五个海拉尔铁路警察拦截，拉到一个房间里搜查。

警察搜身、翻箱、翻包，女的翻一遍，男的翻一遍，翻到包里存放好多年前的零角钱，大概有三十多元钱。警察一张一张地仔细翻，看到有四张二角钱，上面写有“法轮大法好”，就拿录像机放到桌子上录像，问王艳华：“这钱哪来的？”王艳华说：“你们花钱时，没找回真相币吗？”他们无语，就“嗯”了一声。警察还翻开手机，找里面的所谓证据，问这问那：“微信群都是哪些人？”“你都和哪些人来往？”“戴过什么符吗？”女警把王艳华领到里屋房间，翻她脖子领口，看看脖子上戴没戴护身符。男警又命令女警：“把她的头发打开，看看头发里有没有什么东西藏着。”警察什么东西也没找到。

他们又回原屋，一个穿便衣的，说是领导，再三审问王艳华：“你还炼不炼功？”“你不说，我有办法。”他就把王艳华刚满两周岁的小外孙子单独带进里屋房间，问幼童：“你姥姥还炼功吗？在哪炼啊？东西都放哪了？”孩子吓得不说话，脸色都变了。王艳华一看，对这么小的孩子逼供，她愤怒地说：“他要是被吓着了，我告你们去！”警察这才才罢休。

警察盘查了两个多小时，扣押四张真相币，让王艳华签字后才放他们走，期间还给王艳华录像。

#### 王艳华女士自述遭非法拘留经历

我于一九九九年四月五日喜得高德大法。得法三个月，江氏集团就开始了铺天盖地对法轮功的迫害。

我想：这么好的功法，教人心向善，道德回升，回归净土，咋不让炼了？我于二零零零年去北京讨个公道，坐早晨五点多钟的火车，在火车上过的年，到第二天晚间到达北京，住了一宿，天刚亮就去天安门上访，天安门很多都是便衣警察，到那就有人过来问：你是炼法轮功的吗？我说是，这时又过来三个人一起把我绑架车上，哪有说理的地方呀。一会这车就满了人，送到一个地方，屋里全是炼功的人，一会屋子都装不下了满满的，就送另一个屋子里，有的说一天抓了五千多人。我在北京被扣留了几天后，被本地派出所警察非法关押到根河市看守所，从二零零零年二月十二日到二零零零三月十日非法拘留近一个月。

二零零一年，我和几个同修被绑架，从二零零一年四月十三日到二零零一年七月十一日被非法拘留近三个月。

接下来大约是二零零二年，当地中共人员办“转化”班。本地政府、警察有十多人闯上家门将我强行绑架。记得有一个叫乾兴武的，把我从炕上拖出来，我鞋都没有穿，他们把我倒着扔到车里，就象飞一样的，我心里受到极大的伤害，想起来不寒而栗。我被拉到根河市飞机场的“转化班”，身上还青一块紫一块的。办班期间，政府人员一个星期换一批人，轮流“转化”我们，迫害了我们两个月，他们没有达到目的。

第二年，他们又办班，说是办十天班，应付上级检查。我又被他们强行拉走，被迫害七天。

二零一五年五月十九日，我在楼里发放神韵光盘，被不明真相的人举报，被金河派出所警察绑架到海拉尔，非法拘留十五天。

二零一五年，我向最高检察院、最高法院提交起诉江泽民的诉讼状。二零一五年十一月二十日，我被根河市公安局警察黄良及一个叫“陈姐”的退休人员绑架，非法拘留十天。

二零一七年七月十九日，我在大庆市居住的二哥得了肠癌病，我从金河坐火车去大庆看他，在伊图里河换车，我刚下车走下台阶想去吃饭，就有两个铁路警察追上来，问：“你是王艳华吗？”我说是，他们就把我叫到车站的一个屋里强行翻包，抢走我的两个手机，一本大法书，还给我录像，不让我走，硬是把我拉到派出所非法盘查、审问。我二哥得了癌症住院了，我心里很着急，这里的派出所警察竟不让我走，还给我铐上手铐。一开始他们要铐后面，我极力反抗，后来过来三、四个人把我按在地上，当时挣扎的一片桌椅都弄倒了，也没铐上。后来警察给一当官的打电话，那个人来了说铐前面吧，就这样他硬是把我铐上手铐，把我拉到海拉尔，非法拘留十五天。

#### 不要拿自己的性命给中共当替罪羊

王艳华女士很善良，这些年她本人和家人所遭受的苦难无法言表，警察屡次迫害她，她也没有曝光他们的恶行，想再给他们机会，可是警察还在行恶。

法轮大法是佛家高德大法。迫害佛法，罪大无边。如今瘟疫遍传，民不聊生，上天在敲响警钟，这些警察还不清醒。现在曝光他们的恶行是给他们最后的机会，希望他们不要拿自己的性命给中共当替罪羊。
