### 越南学员：我全家都从修炼大法中受益

文: 越南大法弟子 

【明慧网二零二二年五月七日】我是一名农村妇女，住在越南清化（ThanhHoa）省。因为我的妹妹学了法轮大法，并从中受益，我因此幸运地知道了大法。尽管那时我还没有修炼，但我目睹了妹妹的转变。以前她脾气不好，学了大法后，她变得温和而坦诚，她的几种病也不药而愈，因此我很自然相信，并跟她一起每天学法炼功。
我从妹妹那里得到了《转法轮》，并了解了学法的重要性。我家住在乡下，我的主要工作就是务农，闲暇时也做很多其它事情。

从二零一九年一个炎热的夏季下午起，我开始阅读《转法轮》。神奇的是，每当读《转法轮》时，我就感到周身凉爽。当时气温有39～40摄氏度，即便开了电风扇，我也感到很热，但是当我坐下开始阅读《转法轮》时，就感到周围空气变得很凉爽，很舒服，不再感觉闷热。

我跟着妹妹去了两公里外一位同修家集体学法，去了几天后，妹妹和她女儿回到河内。从那以后，我在家独自学法、炼功，但我知道师父在看护着我。每次炼第二套功法时，我眼泪就止不住地流。那时还没有悟到太多，但我坚持每天炼功学法。在修炼六个月之后，我忽然想起我已经很久没有吃药了，就连草药也没吃过。

过去，我患有胃食道逆流、乳房囊肿、脸颊肌瘤、高血压，有时头痛会发作，每次头痛发作时，我丈夫和孩子们都很担惊受怕，他们找医生来家里帮忙。此外，我还患有颈椎病、腰椎病、风湿病、膝盖常年疼痛。这些疾病在我修炼大法三年后，都不知不觉地消失了，我不再有腹痛、嗳气、消化不良、高血压和口腔疾病；我乳房上的囊肿和脸颊上的肌瘤也自然消散。以前因为这些疾病缠身，我没过上几天安心日子。我现在没病了，感觉很舒服。以前我眼睛痛，看不清字，现在我不用眼镜就能看清书上的小字。

我的脾气也发生了变化，从脾气暴躁、对丈夫和孩子易怒，到现在变成了贤妻良母。丈夫起初很反对我炼功，但是看到我的变化之后，他现在支持我每天学法炼功。我全家人也都从修炼大法中受益。

#### 法轮大法救了我丈夫

二零一九年十二月的一天，我丈夫喝完婚酒回家。他喝醉了，头痛得很厉害，我没有太在意。后来我有事出门，刚出去十五分钟，就看到孩子来叫我马上回家。我赶紧往家跑，看到我丈夫一动不动地躺着，嘴巴歪到一边，就象中风一样。我既害怕，又担心，也很恐慌，我不停地摇着他，喊他的名字。幸运的是，我想起了九字真言：“法轮大法好，真善忍好。”我不停地喊：“师父，救救我丈夫和孩子！”

仅仅几分钟后，我感到丈夫的手指动了，他慢慢苏醒。我让儿子给医生打电话求助。然而医生说他喝醉了，没办法来，但嘱咐我如果情况严重就送他去医院。幸运的是，我丈夫逐渐恢复了意识，所以我不需要送他去医院了。这次奇迹，让我丈夫对大法有了更好的了解。从那以后，每遇到问题，他都念诵师父赐予的九字真言。

#### 我女儿也从大法中受益

二零二一年五月二十一日下午，我接到了女婿的电话，说我大女儿病得很重，被送进了河内的急诊室，她被诊断出胃溃疡、胃窦炎、幽门螺旋杆菌，最危险的是肠胃出血。我赶紧和儿子收拾行囊，带上了《转法轮》和一个装着记忆卡的小手机，用于听师父讲法和炼功音乐。

我赶到医院，看到女儿一动不动地躺着，右手输着液，她已经几天未进食了，一吃东西就吐。她胃肠道出血很严重，皮肤很薄，眼睛下方有黑眼圈。看着她，我感到非常难过，我对她说：“现在你得自救了，这种病按常理需要很长时间才能好，听我说，读《转法轮》吧，如果你有缘份，师父会管你的。”

我女儿点头同意了。当晚，我读《转法轮》给她听，奇迹发生了。隔天一早，她感到不那么累了。她可以吃粥了。她已经几天没吃东西了，当她吃下那碗粥之后，我大大地松了一口气，默默地感谢师父的救度。那晚，我又读《转法轮》给她听，她逐渐地康复了。第二天，女儿催着我回家，她说：“我没事了，你回家吧，我自己读书。”

第三天，女儿打电话告诉我：“妈，我明天就出院了。”我不能相信我的耳朵，就又问了一遍。儿子说：“她好了，大便不再发黑，胃痛消失了，因此医生让她出院。”

以前，我女儿也反对我修炼，因为她听官媒总是抹黑法轮功。现在她目睹了自己母亲的变化和大法的好处，不再反对了。她也总是记着师父的九字真言。

最近，当武汉肺炎到处肆虐的时候，我住的地区也受到了影响。到处都贴上了红牌子“居家治疗中”。我们家总是诚实做人，总是记着九字真言：“法轮大法好，真善忍好。”因此尽管住在当地，我家一直很安全，无人染疫。

在五月十三日师父华诞到来之际，弟子们无限感恩师父。我和全家从心底里表达对师父的深深感激。
