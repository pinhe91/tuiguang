### 大队书记：“我不管，我支持她们！”

文: 山东大法弟子 

【明慧网二零二二年九月五日】我给大家讲两个发生在我身边的真实小故事。

#### （一）村委主任：这才是真心实意的为人民啊！

这是一个发生在邪党残酷迫害法轮大法后的一年夏天。一天傍晚，我们邻村的一位大法弟子，到他自家菜园地去摘菜，准备回家做饭吃。正当路过他村的一座大水库时，他看见有一处坝基正一直往外淌水。由于盛夏，这座大水库的水位很高，水库处于村头的高处，且离村庄又很近，情况非常危急。

这位大法弟子见此情景，也顾不得去菜园地了，不由分说，赶紧跑到村委找到主任向他说明情况，主任一惊，赶紧跟着这位大法弟子去实地察看。看后他也吓出了一身冷汗。

当晚主任马上就指派人员驱动拖拉机、挖土机，连续忙了三天三夜，才把坝基修固好。

险情排除后，主任感动的说：“还是法轮功好啊！这回幸亏某某（大法弟子）发现险情并及时上报。这才是真心实意的为人民啊！”

#### （二）大队书记：我不管，我支持她们！

我村有一位女大法弟子，有一天，她去拾柴火，正当她直起腰歇息时，她发现邻村的苹果园有一处正在冒烟，疑似着火了。转眼间，火越烧越大，她就急忙喊人快来救火。

有邻村的人看见了说：“别去救，离远点！”这位大法弟子想：我是大法弟子，于是她奋不顾身，拿着竹扒猛扑烈火。经过一阵紧张的拼搏，大火渐渐熄灭了。可她拾柴火的工具——竹扒，却被大火给烧秃了头，她的衣襟也烧破了几个洞，手也划破流血了。那一刻她从头到脚都变成了一个“黑”人。她看到大火已经完全熄灭后，才背起柴火，默默的回家了。

后来邻村大队书记找到这位大法弟子，非常感谢她救了他家的苹果园。感谢她做了一件了不起的大好事，因为他家的苹果园一边连着村庄，一边也连着不远处一层层的大山。山上长满了各种树木，如果大火继续蔓延，引发山林火灾，那后果可就太可怕了。

书记说：“谢谢你，大法弟子！李洪志师父好，教育出你这样的好人，法轮大法好！真善忍好！我‘三退’。”

后来这位书记又对别人说：“共产党说人家法轮功不好，人家哪里不好了？好人多还不好吗？以后谁爱学大法就使劲学。他们（大法弟子）爱怎么讲就怎么讲，爱怎么发（真相资料）就怎么发；爱怎么贴（真相不干胶）就怎么贴。我不管，我支持她们！法轮功就是好！以后谁也不许去举报她们！咱们要保护她们！”
