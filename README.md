### 修大法获新生、明真相得福报

文: 湖南大法弟子

【明慧网二零二二年十月七日】一年前，我在路上偶遇一位朋友，招呼过后，我便问候她和她的家人，她说起她女儿觉的很不幸，我问她为什么，她说她女儿是位教师，才二十多岁，却得了肠癌，花了十多万，到大医院去都没治好，医生说这个病没有药治得了。
后来我去她女儿家看望，告诉她诚念“法轮大法好、真善忍好”会得福报，给她讲大法真相以及三退保平安。她女儿很认同大法，叫我帮她退出团和队。我叫她和她女儿记住：法轮大法好，真善忍好。

前不久在路上遇见她们母女俩，她们很高兴，她说她女儿的病好了，又正常上班了！她女儿说：“谢谢你，阿姨！”我笑着说：“你应该谢谢大法师父！”
我是从一九九六年六月开始修炼法轮大法的，至今二十六年了。修炼的这些年来我切身体会到大法的慈悲与伟大，大法使我身心受益，彻底改变了我，我的家人和很多有缘人也有幸蒙受大法的护佑，借此机会把我的亲身经历写出来，让更多的人见证法轮大法的无边法力。

#### 修大法让我获新生、全家受益

我以前在化工厂上班，身体很不好，头痛、风湿病等身体经常不舒服。在我三十八岁那年，在县人民医院检查出胆囊炎，停工住院两个月，期间打针吃药，花了几千还是没好转，医生建议开刀治疗，那时候我工资很低，一个月只有一百来块，因为之前住院已经花了大半的积蓄了，所以当时我没有选择开刀，回家去了。

可是这个胆囊炎让人痛的难受，我吃不好、睡不好，腹部痛、胸口痛，还有其它地方发射性疼痛，加之头痛、风湿，身体状况非常差，同事朋友都看在眼里。

后来经一位同事介绍，于九六年的六月，我有幸参加了当地的法轮大法九天学习班。在那里听大法师父的讲法录像，不收任何费用，同修还义务教功。我学法炼功几天，心里感到法轮大法特别好！就这样我一边听讲法录像一边跟着学功，才几天时间，身体竟出现了奇迹的消业现象，早上刷牙时，从喉咙里咳出来很多脓和血一样的东西，之后整个人变的舒服极了，我切身体会到大法的神奇！

从那以后我的身体状况一天天变好，慢慢的，以前的头痛、风湿病、胆囊炎、反正原本身体上所有大大小小的病通通都不见了。并且除了身体上的变化之外，连精神上都出现奇迹变化，我感觉脑子比以前清晰和灵敏了。后来我知道是我们慈悲伟大的师父在帮我清理身体，把我的身体从里到外都清理了，病根都摘掉了，而且师父还给我开智开慧了。

那时候我真正体会到无病一身轻的感觉，骑自行车真的好像后边有人在推一样，美妙极了。一直到现在我身体状态都非常好，晚辈说我皮肤细嫩，白里透红的，一点不像六十六岁的老人。我也觉的自己的身体状态和精神状态都极好。

自从学了法轮大法以后，我的人生从灰暗转变成幸福。我身体好了，生活上能把家里料理得有条不紊，上班的工作状态也非常好，为厂子带来了更好的经济效益，单位每年都给予我很多奖励。

不仅仅是我自己，我们全家都在大法的福泽中，我的家人个个都工作顺利事业有成。我丈夫主管工作做得很好，年年都被评为“先進工作者”；我儿子在单位工作非常优秀；我女儿当经理，能力卓著，让家人非常自豪。我知道是因为家里人都支持我修大法，所以他们都得到了福报。我们一家人和和睦睦，过着幸福的生活。

我修炼大法不求名、不求利，时时处处都按照大法要求的真、善、忍修自己。有一次我在出去讲真相的路上，看到一个手机掉在地上，我捡起手机，等主人打来电话，我和他约好当天十二点到水电站大门口把手机归还给他，他们来了两个人，见到手机非常激动，他们说：“大姐感谢你！这个手机是我们花了两千块钱刚买不久的新手机”。当时的两千块来说是非常好的手机了，他们说着非要拿一百块钱给我，说要好好感谢我给我买水果。我对他们说：“我是修炼法轮大法的，我们师父要我们按照真、善、忍的标准做好人，做更好的人，所以我不要你的钱！”那两人说：“你们师父真好！”

#### 明真相　得福报

自从一九九九年法轮佛法受到中共迫害以来，为了让更多善良的世人听闻佛法的美好，不再受谎言的蒙蔽，我和千千万万的大法弟子一样，默默的利用自身的时间和条件，比如学做真相期刊、做真相光盘、真相币等，利用各种机会和各种办法让有缘的善良人明白真相，有时候面对面直接用自己的亲身经历告诉世人，让他们了解大法的美好，得到大法的福泽与救度。下面仅举其中几例。

#### （一）七旬老人的感激

有一天，我到一个村子里去讲真相，我看到一个大姐约七十岁左右，她坐在家门口休息，我就过去和这位大姐讲大法的真相，我说：“大姐，您好！您听到有人和您讲过法轮功吗？”她说没有人告诉过她，我就说：“大姐，您今天缘份到了，我今天就是来告诉您法轮大法是一部高德大法，我学法轮功二十多年了，我一身病都好了，没有吃过一粒药、没有打过一次针，身体还特别好，走路一身轻呢！”大姐说：“这么好啊？”我说：“是啊，学法轮功的人是按照真、善、忍的标准做好人的；是处处为别人着想的；遇到问题向内修自己的人。这么伟大的佛法，这么善良的修炼人，中共江泽民却从一九九九年开始迫害法轮大法和大法弟子，编造谎言抹黑法轮功，其实也是迫害中国的老百姓，让老百姓没有机会得到这么好的大法。还让老百姓仇视佛法，对佛法犯罪。所以我来告诉你们真相，你们不能一辈子都上了中共的当啊！”

她很认真的听我讲法轮大法的美好以及大法受到的迫害。我告诉她“三退”保平安，她说她没有戴过红领巾。临走前我送了护身符给她，告诉她上面有九个字：“法轮大法好，真善忍好”，念动这九个字可在危难之时保平安！

等我第二次经过这个村的时候，那个大姐看到我，她一眼就认出我了，拽住我的时候我还没反应过来呢！她很激动的说：“法轮大法真是好啊！你上次给我的护身符救了我一条命呢！”然后她开始向我描述她的神奇经历：

有一天她在山上摘橘子，当时爬在一棵很高的树上，脚下的树枝突然断了，她“扑通”一声从树上掉下去了，正好摔在石头上……当时人都懵了，过了一会儿，思想有意识的时候心里回想起我和她说过的“法轮大法好，真善忍好”。慢慢的她清醒过来了，也没发现伤着哪儿，然后自己慢慢的回家了。她说当时真的不敢想象，那山上一个人都没有，要是她没起来就完了。我说：“大姐相信大法，得福报！是大法师父救了你呀！你一定要记住法轮大法好真善忍好！”她说：“谢谢大法师父！”

一个七旬老人出现这样的意外还能安然无恙，可见诚心相信大法的人都是有神护佑的。

#### （二）悲苦人生的转运

还有一次，我从市区坐班车回家，在车上遇到一个小伙子跟我坐一排，我和他聊了起来，他说他到雷塘下车，他到矿里做事的，要下井工作，我说在那儿上班可辛苦了，他说没办法，家里有老人和两个小孩，小孩要上学，他说自己没好运气、人生很糟糕，家里的第一个老婆出车祸死了，第二个老婆走了，家里老小他一个人负担打理，他觉的人生好苦。

我说：“看小伙子是个有文化的人，大姐告诉你一个最大的好事儿！”他问什么好事？我说：“我是炼法轮功的，法轮大法是教人做好人的高德大法，法轮功祛病健身有奇效，改善了我的身体素质。我以前身体有头痛风湿胆囊炎各种病，是我们的师父把我的身体净化好的。”

他说大姐看起来确实身体很健康，面色白里透红的，为什么法轮功这么好，国家不让炼呢？我告诉他因为江泽民妒嫉法轮功修炼人数众多，编造谎言抹黑法轮功的。按真、善、忍做好人没有错！他很认真的听我讲真相，他问我住哪里，如果他想炼功到哪里炼，我告诉他念九字真言得福报，他说好好！我问他家里有没有影碟机，他说有，于是我送了他真相光盘、小册子、护身符。我让他留下了联系号码，等我打电话给他，他愉快的答应了。他回家看了光盘之后很欣喜，我打电话给他的时候，他问我要了大法书《转法轮》、《九评共产党》，并且很认真的都看了。他说他两天把《转法轮》一本书看完！《九评》光盘都看完了，他说你和我讲的真的是对的，《九评》里面讲得很清楚啊，他很相信大法，还说他也想要跟去讲真相！

三个月后，有一天发生的事情震惊了他，那天轮到他下井，他工作服都已经换好，准备好下井了，很奇怪的是突然得知他被换班了，今天不用下井，他还有点纳闷。结果……那天矿井坍塌，下井的工人全部发生矿难无一幸免……

他曾经是一个觉的自己命运悲苦一生都没有好运气的人，他又是一个听闻佛法真正明白了大法真相的人，唯独这一次，他感觉自己成为了那个最最幸运的人！

在此，我只把其中的几例有缘人的片段回忆、整理出来了。希望通过以上的切身经历，能够让更多可贵的有缘人感受到法轮佛法给众生带来的无量慈悲和无限福泽；感受到生命的希望！我借此机会表达作为大法弟子而感到无比的荣幸，我内心深处每时每刻感激慈悲伟大的师父给予了我最美好的一切！无以报答师尊的慈悲苦度，唯有按照大法的要求，按照真、善、忍的标准，不断提高心性，勇猛精進！
