### 曝光太原市政法委610主要成员张小鹏的罪恶

【明慧网二零二二年八月十日】（明慧网通讯员山西报道）张小鹏，男，52岁，身高1.75-1.80米，国字脸，面黑，复员军人，太原市政法委610主要成员，专事对法轮功学员洗脑迫害、构陷冤狱。
至少从二零一一年起，张小鹏就在太原市政法委610效力，参与山西省610在太原市办的洗脑班——山西新店劳教所管理楼顶层的所谓“山西省法制教育中心”迫害法轮功学员。

从二零二零年四月起，特别是二零二一年中共邪恶610搞“清零”以来，山西省610在全省各地又开始办洗脑班。太原市政法委的张小鹏跳到前台，大搞“清零”骚扰、洗脑，频频对修佛向善的法轮功学员上门骚扰、构陷冤狱。

至今，太原市的很多法轮功学员被骚扰、被跟踪监控、绑架、被构陷冤狱，都有张小鹏在幕后的操纵指使。下面是张小鹏参与迫害的部份事实：

#### 一、洗脑班——法外黑监狱：非法拘禁公民，以肉体和精神摧残，强迫放弃信仰

山西省610在太原市新店劳教所办洗脑班。洗脑班主要负责人是武革慧，骨干成员是张小鹏、任姓科长等人。目前，武革慧已调至外县数年。

他们利用以下非法手段迫害法轮功学员：

1.非法拘禁，劫持进洗脑班；
2.伪善，洗脑班头目常虚伪表白：我们和你们（指法轮功学员）吃住都一样，对你们多好；
3.洗脑攻心，“陪教”是与法轮功学员同单位的人，洗脑班首先对陪教洗脑，从而成为被洗脑班利用的工具，再找些各类所谓迫害“专家”、劳教所监狱恶警、邪悟者，诬蔑大法，巧言利诱；
4.恐吓与迫害，威胁说如果不转化就劳教、判刑，或者胁迫家人，让痛苦恐惧中的家人从侧面动摇大法学员，甚至到几位被绑架到洗脑班的法轮功学员家（以及娘家、或丈母家）非法搜查，图谋迫害；
5.剥夺睡眠，体力透支；
6.劫持走出冤狱的法轮功学员。

#### 二、法轮功学员田云飞被三次送洗脑班遭暴打　妻子被迫害致死

西山煤电集团金城公司职工、法轮功学员田云飞，二零一零年十一月、二零一一年九月底、二零一三年八月，这三次被警察直接绑架到洗脑班，每次强迫洗脑一个月。第三次，在洗脑班，姓李的管理员逼迫田云飞放弃信仰，不让睡觉，罚站，用带铁管的簸箕打田云飞，簸箕的连接处被打断，他又用铁管打田云飞的后脑勺。第二天，田云飞的后背到臀部都是黑紫淤青。

田云飞的妻子张印香给田云飞送衣服，在洗脑班，张印香并没有见到丈夫，就被隔离关了起来，被非法关了大约一个月零五天。夫妻二人根本不让见面，吃饭都是陪教人员把饭打回来吃。在里面，每天播放污蔑法轮大法的录像，不让出去，不让睡觉，整天被一些所谓的“心理学家”、“社会学家”、“大学教授”、狱警、假气功师、“中北大学老师”（曾经学法轮功，后来变成“犹大”的武梅，见附注[1]）围着，强行灌输邪恶的东西。610的人员不报姓名，只称：谭处长、任主管、李狱警等，非常隐秘。

洗脑班给张印香造成了很大的精神打击和伤害。张印香回家后，又多次受不法公安的骚扰恐吓，后来张印香一有动静就害怕，渐渐神智不清，身体出现半身不遂状态，每况愈下，骨瘦如柴，仅剩四、五十斤，二零一八年五月五日清晨5点，张印香含冤离世，终年52岁。

#### 三、参与“清零”迫害，上门骚扰、转化

二零二一年至今，太原市政法委610搞“清零”骚扰，授意各区政法委610、派出所、街道办、社区人员，大范围上门骚扰太原法轮功学员，让写什么“悔过书”等，对法轮功学员及家人造成严重干扰甚至伤害。太原市政法委610主要成员张小鹏，是这场上门骚扰的骨干，他不是站前台，就是在幕后。

“清零”使太原市大多数法轮功学员都受到了不同程度的骚扰迫害。二零二一年三月中旬，在市政法委610授意下，万柏林区的政法委和警察、“犹大”武梅，陆续上门骚扰西山法轮功学员，让写什么“悔过书”等“四书”。被骚扰的有法轮功学员武继平、武美琴、焦认真、王俊青、春香等多人，被骚扰的还有周老太、曹爱秀、孟梅、王秀芝、董润花和孙女欣欣、李燕、李顺琴、秦雅丽、李宏、王志宏、刘荣、张秀琴、季国丽、周华、周勋臻等人。

#### 四、组建监控迫害团伙：摸底、监控、插手司法　构陷冤狱

二零二一年至今两年来，张小鹏把法轮功学员居住地街道社区人员、所在单位领导、居住地派出所社区警察、“犹大”武梅、洗脑班搞心理“转化”的人，组建监控、迫害团伙，对他选定的法轮功学员进行监控、摸底。在法轮功学员家，一坐就是几个小时，声称来“解困”，实则骚扰、摸底，图谋进一步迫害。

张小鹏伪装“说话平和”，会看似不经意的说出几个法轮功学员的名字，问你认不认识。同时背地里，他会利用他的监控迫害团伙对法轮功学员进行监视、跟踪、摄像头监控，收集可供进一步迫害的“证据”。其中，遭构陷的有法轮功学员王志燕、青年中医法轮功学员申国胜、赵晓庆，而李宏、曾晓媛、秦雅丽目前面临冤狱构陷。

二零二一年六月，张小鹏带着“犹大”武梅、搞心理学的专家及社区人员三次上门骚扰，中午和晚上到青年中医法轮功学员申国胜家里，谎说“解困”，说着污蔑法轮大法的邪说，赖着几个小时不走，严重影响其高龄患有癫痫病30多年的老父亲的休息，最后被忍无可忍的老父亲撵走。

几个月之后，二零二一年十二月二十九日，迎泽分局迎泽派出所张皓等几个警察都穿便衣，到申国胜工作的诊所对其实施绑架，去抄家时，只有张皓出示了工作证，其他人都没出示任何证件，什么也没有抄到，便把申国胜绑架到看守所。起因是来自社区人员的举报。仅有的“证据”是一张模糊不清的所谓申国胜近一年前的背影图片。家属被告知是610让办的这案，警察只是听命行事。

二零二二年五月二十六日，张小鹏又带着社区书记等五、六个人，去骚扰法轮功学员申国胜的家属。他去给家属施加压力，干扰家属为申国胜依法伸冤。张小鹏直接插手参与构陷冤狱。

申国胜目前已被非法逮捕、视频开庭。在610的操纵下，法院以疫情为由，非法剥夺家属旁听权。迎泽区检察院的公诉人是员红娟，迎泽区法院主审法官是贺林海。法官和检察官问律师当地的司法局没给他们培训吗？没开过会吗？言语中暗示威胁律师。外地律师开完庭后，表示不惧威胁，是他们不依法办事，把好人抓起来。

#### 五、太原市政法委610增加人员、设备及经费，目的迫害法轮功

太原市政法委610用老百姓的民脂民膏去迫害法轮功。张小鹏所用的迫害机制里面的人员是从公检法到社区网格员，设备之一是“雪亮工程”全覆盖的摄像头网络监控系统。

太原市政法委二零二一年度新增的“全科网格员薪酬市级保障资金”、“全科网格服务管理项目专项经费”、“雪亮工程项目专项运维费”、“雪亮工程网络运营服务费”、“×教人员教育转化专项经费”等几项迫害经费，达到9726.62万元。

太原市政法委610从二零二一年开始为迫害法轮功，新增了超过2100个直接伸到居民社区的黑手（社区网格员）。这些网格员都是大专以上学历，经过考试录用的年轻人，被洗脑利诱后，为中共迫害卖命。他们都成为政法委610迫害佛法修炼人的基础工具。

政法委610张小鹏等人还利用覆盖监控到太原市的大街小巷居民驻地的“雪亮工程”的视频监控系统，对法轮功学员实施监控迫害，要挟社区人员（特别是网格员）、快递、外卖员等恶意举报、作证，把“证人”推到大法的对立面，以达到诬陷法轮功学员的罪恶目的。
