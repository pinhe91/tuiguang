## 长沙张灵革被秘密庭审　当局出动近七十警察

【明慧网二零二一年六月三日】（明慧网通讯员湖南报道）五月的长沙，特别阴冷，长沙地区中共不法人员加剧了对法轮功修炼者的迫害。长沙市法轮功学员张灵革女士，五月八日遭长沙地区浏阳市法院非法秘密审判，除法庭派驻的武警外，当局同时出动了近七十个公安警察。

![image](https://user-images.githubusercontent.com/79625284/120636129-d78b3200-c49f-11eb-851a-fb2ce72af4ce.png)

张灵革

如此大规模地动用公安警察，连参与此次检察院的检察官都惊叹：工作这么多年，从未见就一个人的审判，还是个女的，出动这么多公安的。

据悉，中共当时正在长沙秘密召开所谓的“反邪教大会”（中共是个地地道道的邪教，强迫中华儿女信奉遭西方社会唾弃的马列主义），参加此会的三十名610人员参加了此次所谓“庭审”。

与张灵格同时被绑架迫害的长沙法轮功修炼者有22人，其中15位法轮功学员分别被非法关押在长沙市第一看守所、第二看守所（男）和第四看守所（女），近期得知，构陷他们的所谓“卷宗”再次被递交到浏阳检察院。有公检法人员透露，他们都只是在按程序走个过场，在庭审之前政法委把刑期都定好了。

与以往编造“大案”不同的是，此次检察院采取拆分送检，拆分开庭。六个月的时间，不许律师接见当事人，律师的接见诉求被公安局，派出所，看守所，检察院互相推诿。律师被当地司法局电话约谈威胁。

办案公安与检察院阻止家属请律师，采取恐吓诱骗的方式，要求被关押的修炼者接受法院指派的两位援助律师，致使家属请的辩护律师无法接见介入。

二零二零年十月二十七日、二十八日，长沙公安局动用大量警力，统一行动，非法抓捕李志刚、孟凯、曹志敏、张灵革、章芙蓉、徐丽华等二十多名法轮功学员。其中，长沙市圭塘派出所与浏阳派出所四个警察，于二十七日晚十一点多钟闯入张灵革妈妈家绑架了张灵革。

警察一进屋就野蛮地给张灵革戴上手铐把她反铐起来，同时将张妈妈控制住，叫她坐在床上不要动。然后，抢走她俩的手机，开始非法抄家，把整个房子翻了个遍，抢走了几本大法书籍。大约两点多钟，警察将打着赤脚只穿着拖鞋、反铐着手的张灵革带去她的廉租房搜查，拍照。张灵革的妈妈急忙拿着袜子、鞋子追上第二部车子，鞋子袜子都没让她穿。

张灵革女士，52岁，家住湖南省地质矿产勘查开发局四零二队大院内，她是湖南省湘潭大学历史系的高材生，一九九七年开始修炼法轮功，她更加善良，诚实。如同许许多多法轮功修炼者一样，按照真、善、忍修心向善、提升自己，为年轻的张灵革的生命注入了全新的涵义。

因为坚持修炼法轮功和传播法轮功真、善、忍的真相，张灵革女士多次被绑架至监狱、劳教所、看守所、洗脑班等地，惨遭非人的折磨。二零零二年至二零零八年间，张灵革被非法判刑六年，在看守所关了一年后，转到湖南女子监狱，在那里熬过了漫漫五年时间。六年冤狱回家仅十三个月，张灵革于二零零九年四月二十七日再遭绑架到株洲白马垅劳教所，被非法劳教一年半。

二零一四年三月二十九日，张灵革因长沙市“610”、国安炮制“大案”与言虹、李玄刚、管登洋、姚大华、周德元、柳春霞、赵亚玲七名法轮功学员同时被绑架判刑，张灵革被冤判三年半，再次被非法关押在湖南省女子监狱；还被取消低保。二零一七年出狱后，张灵革多次被绑架到洗脑班、拘留所等地，动不动就关上两天或几天、十几天。

## 昆明市强制戒毒所所长陈波与妻子遭恶报双双被抓

【明慧网二零二一年六月二日】（明慧网通讯员云南报道）陈波，一九六四年出生。二零零三年起，担任昆明市强制戒毒所中共党委书记、所长。二零二零年六月，陈波被查，同年十二月被逮捕，二零二一年一月被起诉到呈贡区法院。陈波的妻子张惠，昆明市强制戒毒所二级主任科员，二零二零年九月被查。目前，陈波夫妻二人均被关押在看守所。
昆明市强制戒毒劳动教养管理所成立于一九九一年十月，位于昆明市王大桥，二零零九年三月，更名为“昆明市劳动教养管理所”，并加挂了“昆明市强制隔离戒毒所”的牌子。二零零九年底，转移至昆明市呈贡区新所。二零零三年，陈波到昆明市强制戒毒劳动教养管理所任所长。自二零零四年开始，先后共有五位女性法轮功学员被非法关押在这里。五位女性法轮功学员均被非法关押在三大队，但是不准互相说话。每个法轮功学员被至少安排两个吸毒犯包夹，二十四小时监视、跟踪迫害。

陈波任职期间，昆明市强制戒毒劳动教养管理所对法轮功学员迫害的部份事实：

### 一、法轮功学员杨小明遭受多种酷刑，眼睛被毒打致失明

杨小明，女，是昆明医学院后勤服务发展中心物业管理科职工。二零零五年二月一日，杨小明正在单位上班，被单位恶人伙同警察绑架，于当日直接送到昆明市强制戒毒劳教所，被非法劳教三年。

在昆明市强制戒毒劳教所三大队被非法关押的三年里，杨小明遭受了多种酷刑：电棍击打、野蛮灌食、踢下身、打嘴巴、暴打、不让睡觉、罚站、罚蹲。暴打使杨小明的眼睛被打残，脚背被打伤。连续多天被罚站，每天站十七、八个小时。

二零零五年四月中旬的一天晚上，三大队副队长刘先敏、吴静、警察周枣兰唆使五个吸毒犯人罗媛惠、张静、王艳红、刘乔丽、赵昆利暴打杨小明。先打嘴巴，然后把她踢倒在地， 将杨小明的头用力向水泥地上砸，杨小明的后脑被砸起一个鸡蛋大的血肿包。不仅如此，这几个人还对杨小明拳打脚踢，把她打的浑身是伤。之后，杨小明的两眼刺痛，睁不开眼。从那以后，她的左眼就什么也看不见了，右眼看东西也模糊不清。后经过医院专家鉴定为一级残疾。

二零零六年五月一日下午，杨小明又遭到吸毒犯人施燕、魏红的毒打，左脚被打的皮开肉绽，流了很多血，还缝了针。杨小明穿的衣服、裤子也被打的破烂不堪。

杨小明曾三次绝食，抗议戒毒劳教所警察及犯人对自己的暴行，遭到野蛮灌浓盐水。集训组长张静多次将杨小明的双手绑在身后，用脚踢她的阴部，使她下身出现血肿，连续五、六个月停月经（杨小明一九七七年出生，被非法劳教时才二十八岁）。警察张红梅用电棍打杨小明的两个手掌，使她的双手肿了一个多月不能动。

二零零九年三月，杨小明从戒毒劳教所回家后，没有了工作，生活全靠近八十岁的工人老父亲微薄的收入维持，还要给她治残、治伤，生活很困难。

### 二、况德英遭多次殴打，被剥夺家属会见、非法加期

况德英，一九六二年出生，个体经营者，家住昆明市江岸小区。二零零四年十一月，况德英回老家云南省昭通市巧家县，向有缘人赠送法轮功真相资料，被巧家公安局抓到巧家新华派出所。五个月后，二零零五年五月，况德英被劫送到昆明市强制戒毒劳教所非法劳教三年。

有一次，况德英看到法轮功学员杨小明被犯人殴打，她站出来制止，就被大队长刘浩拉去大骂一通，还威胁她不要多管闲事，不要管其他法轮功学员的事。之后，她也被犯人殴打，牙齿都被打松了。这次以后，劳教所允许她和家人见面，但不准她说出被戒毒所犯人殴打的事情。

况德英见到家人后，还是说出了她被打的实情。出了接见室，她就喊：“法轮大法好！真善忍好！”护卫队赵姓警察上来就打她的耳光，还一脚把她踹倒在地上，双脚踩在身上掐她的脖子，并给她戴上手铐，口里还骂着脏话。

第二天，况德英就当着众人的面，揭露赵姓警察昨天打她的恶行。这个赵姓警察当众辱骂况德英，还诬陷说是况德英打他。之后，戒毒所再次剥夺了况德英的家属会见，还给她非法加期十五天。

### 三、马玲被戒毒所送到洗脑班迫害，剥夺家属会见

马玲是云南大学图书馆副研究馆员。二零零四年九月二十一日上午，马玲正在图书馆上班，被昆明市五华国保大队练学滕、马斌一行人绑架到五华区看守所。十一月一日，被劫送到昆明市强制戒毒所非法劳教三年。马玲的家属为她行政复议，曾多次到戒毒所要求签署委托书，但是戒毒所百般阻挠。

二零零五年八月二十四日，马玲被戒毒所通知出去“学习”两天，由戒毒所副所长王延忠、三大队队长刘先敏等人将她送去昆明市海埂附近的昆明鑫安会务中心（鑫安洗脑班）进行人身迫害。当时那里已有从云南省女子劳教所、云南省第二劳教所、昆明市强制戒毒劳教所等地方劫持来的数十名法轮功学员。

马玲被安排在三层楼的一个两人间的房子里，强制戒毒所派来一名姓李的警察和马玲住在一起，二十四小时寸步不离，监视她的一举一动。房子里安装了窃听器，墙壁上贴着不准打坐的告示，卫生间的门锁被撬掉。窗子外面白天、黑夜坐着三个男人。白天要求开着窗子，拉上窗帘，晚上要开着窗子，拉开窗帘。窗子外面的男人随时都能看到里面的一举一动。房间的门终日不得打开。

后来有一男、两女自称是省里派来搞调研的人和马玲谈了三天话，问她为什么要炼法轮功，为什么现在还要坚持。谈话的时候，叫戒毒所派来的李姓警察离开房间。

马玲隔壁住着一名男性法轮功学员，马玲每天都能听到隔壁电视里在播放“天安门自焚”伪案录像。有一天中午，院子里有警车的声音，窗子外面的人突然叫把窗帘拉开。一阵急促的脚步声后，就听到隔壁房子里一个男人在大声的宣读对这位男性法轮功学员的逮捕令。之后，就将这位男性法轮功学员带走了。警车开走之后，窗子外面的男人马上又叫把窗帘拉上。

那是一个非常阴森恐怖的地方。马玲在那里呆了二十四天，九月九日又被接回戒毒所。同时，从二零零五年五月，马玲就被戒毒所非法剥夺了家属会见。

每个月的第三个星期日，是昆明市强制戒毒所第三队（女队）的人员与家属的会见日。但是二零零五年五月十五日（星期日），马玲的家属来到强制戒毒所却被告知：已经取消了马玲与家属的正常会见，原因是马玲在里面炼功。对马玲实行单独会见，具体时间、地点等劳教所通知。

真实的情况是，马玲在四月份会见家属时告诉家属戒毒所星期天没有早点，是将头一天警察吃剩的饭菜连同一些碎米线煮在一起，当早点卖给里面的劳教人员。当时在场的三大队队长刘先敏（警号5345024）就打断马玲，不让她继续说，还气急败坏的终止了会见。

此后，马玲被戒毒所非法剥夺会见长达六个月。在家属到戒毒所要求会见时，不仅受到戒毒所门口警察的刁难、推诿，还有自称是马玲的专管警察张红梅的威胁和恐吓。

### 四、张文航被戒毒所送到洗脑班迫害，剥夺家属会见

张文航， 一九七五年出生，云南省质量检验中心职工（后被单位非法开除工职），武汉大学研究生毕业。二零零五年一月，张文航被其单位领导伙同派出所恶警绑架，后被非法劳教三年，被非法关押在昆明市强制戒毒所。在戒毒所，张文航被强迫做奴工。

二零零五年八月，戒毒所将张文航送到昆明海硬附近的鑫安会务中心（鑫安洗脑班）“转化”迫害四十多天。中途还将她转到云南省女子劳教所几天，又把她劫回戒毒所。张文航的家属在去戒毒所会见她时，也多次遭到戒毒所的无理拒绝不让见。

此外，还有一位当时六十多岁的法轮功学员魏家碧， 二零零六年五月被送到昆明市强制戒毒劳教所非法劳教二年。

陈波自二零零三年担任昆明市强制戒毒所所长后，急功近利，通过奴工产品疯狂敛财。每年下达高额的奴工产值，要求全所劳教人员加班加点干活，为戒毒所创收，当时曾扬言每年要创收四百多万。因此，戒毒所的劳工名目之多，劳动强度之大，超出一般人的想象。法轮功学员也被强迫做奴工，还要完成产量。

在一味增加劳动产量的同时，陈波还将戒毒所劳教人员的生活水平降至最低，关闭了原本几百人就餐的餐厅，让劳教人员在露天地上蹲着吃饭，下雨被雨淋，大热天被曝晒。伙食水平极差，星期天从来没有早点，而是将警察头一天吃剩的剩菜残渣加一些米线煮在一起当早点卖给里面的劳教人员，米线里经常掺着鱼刺、鸡骨头等渣滓。没有钱的劳教人员只能饿着肚子。虽然不给吃早点，但是戒毒所在星期天却还要让劳教人员干活。

法轮功学员魏家碧因没有钱买卫生纸，在劳教所两年的时间里，每次如厕后都是用一个塑料瓶子装水清洗。

### 当地政要赞法轮功带给社区美好正能量

西悉尼黑镇是澳洲第三大城镇、纽省人口最多的城镇。每年的游行庆典都非常隆重，自二零零零年起，法轮大法团体已超过二十次应邀参加该庆祝活动，受到主办方和现场观众的喜爱，并多次获金奖。

## 诚念真言心归善　肿瘤脱落排出

文: 中国甘肃大法弟子家属 

【明慧网二零二一年五月三十一日】我的母亲是一名大法弟子，我虽然没有正式修炼，但很相信大法，也非常爱看大法真相资料，尤其是那些传统文化故事，每当师父有新经文发表，我也一篇不落的认真阅读。
二零二零年十二月下旬，这是我今生难以忘怀的一段时光：因为身体的不舒服，把一切都打乱了：先是乏力，每天回家就想睡觉，几天后，连拿拖把的力气都没有了，只好请假在家休息。接下来的几天情况越来越严重——发冷、发烧、咳嗽、恶心、呕吐、头痛，到后来脚肿、呼吸困难、没有食欲、味觉消失，连下床都变的很困难，总感觉有人在把自己摁在床上，胸痛的喘不上气来，更严重的是下半身开始出血，并散发出恶臭……

就这样持续了十来天，我坚持着没有去医院。直到有一天夜里我闻到屋子里到处都是土味，自己感觉快不行了……终于熬到天亮。我赶紧给修炼大法的表姐打了电话，她很快就过来了。一见到表姐，我竟能挣扎着起床坐在凳子上了。

听我讲述了我的情况后，表姐给我认真地讲了她在魔难中正念正行很快闯关的神奇经历，还有生活中她用真、善、忍衡量好与坏及修好自己善待他人的感人种种故事。

表姐微笑着说：“只念法轮大法好，不改好自己这颗心，总这样肯定是不行的！”

我心里一怔，一边听着，一边对照自己，针对有心结的事情尝试着改变自己的观念，尽量用善意理解他人，慢慢的，慢慢的，我感觉轻松了些，心里对自己说：“一定要有正气！”

表姐走后，我想自己做错的事，一定要改正，对他人的伤害也要尽量弥补。便想求师父帮助我，因为我记得师父曾说过：“弟子正念足　师有回天力”[1]。可我不是大法弟子，疑惑师父会不会管我？心里没底，念着念着就睡着了。

第二天早上醒来，感觉好了很多，下身排出来一大块黑紫色血块，剧烈的疼痛随之而来，流血量开始增加。就这样疼一会儿，好一会儿，又持续了几天，我还是一直念着：“法轮大法好！真善忍好！”告诉自己一定要坚强，相信师父一直在帮我。

二零二一年一月十八日那天，持续的疼痛加重了，疼的我大汗淋漓，在床上翻来覆去，并有大出血症状，但我一点也没害怕。女儿不停的用热水给我焐，但一点作用都不起，一直到晚上，情况更加严重，尽管这样，我坚定的守住一念：求师父帮我。母亲在旁边一直鼓励我，说：师父说了，“放下生死你就是神”[2]。

由于当时情况非常严重，家人不放心，就把我送到医院。做了彩超，结果说是有八点五公分的子宫肌瘤，自己掉了下来，只是没有掉出来。我听后就知道师父帮了我，心里非常感谢师父。医生开了点止疼药，让我先回家等着有空床位就去住院。我吃了药，回到家就睡了两小时。醒来之后，一点都不疼了，但出血量还是很大，我心里明白，如果不是师父保护，身体状况绝不会这么快就好转。

第二天抽血化验时针扎進去后血却抽不出来，其它检查做完后，医生说子宫还是有问题，建议切除子宫。但是我不想切除，不想面对残缺衰老的后半生，表姐鼓励我说：“大法无所不能，但是到底怎么做得自己说了算。”

我相信我一定会好，就想请师父赐我一个完整的身体，但因为心里不稳，又和表姐通了很长时间的电话。

当天住院。晚上我做了个梦：一群鬼抓着我说“找的就是你！”第二天醒来，我就想要出院，家人问我：“出院你就自己能好吗？”我说：“能！大法无所不能，我一定会好！”丈夫和哥哥不信，医生也继续找我谈话，而我很坚定，决定不做手术了。如果做不了手术，我就出院。检查结果是：血压特别高，手术不能做。我更坚信这是师父在帮我。但是医生和家人都不同意我出院，说要采取治疗手段，给了我一片降压药。奇怪的是吃了降压药血压反而更高！医生也很奇怪，一会進来测一下，让我躺在床上，不能动。第二天又让我做检查，甚至还让我去重症监护室，让那里的专家会诊。我心里念着九字真言，求师父帮我：我要出院。

专家又测了血压，做了检查，血压高居不下，这样谁也不敢做手术，

我和表姐再次通电话，她又一次给了我信心。于是我心里默默的对自己的身体说：“我身体的每一部份，请你们都记住：法轮大法好，真善忍好，师父好！希望你们都得到大法的救度！”心里一遍一遍的念，念着念着又睡着了。

晚上，我又做了一个梦：一只白色老鼠想来我这里，三次都被我赶走了。一会儿一个像强盗一样的人拿了一把枪，抓着我的手朝天放了三枪，非常响，枪口却喷着爆米花。

第三天早上四点多我起来去上厕所。感觉到掉下来两块东西，也听到“嗵，嗵”两声，我一下明白：肌瘤全部掉出来了！我那个高兴呀，再次感谢师父！

第四天早上，出血量少了，几乎没有了，我决定出院。家人问我：出去之后自己能调整好吗？我说：“能！”就这样在医院只住了四天，没做任何治疗，瘤子却消失了！

我出院了，感觉好轻松！

出院之后，为了让丈夫相信这是真的，我在别的医院又做了一次彩超复查，结果是子宫完全正常！家人和我都很高兴。

这是我近期的一段经历，写出来与大家交流。虽然这一段路走的很艰难，但是心中一直守着一念：大法师父知道我，大法师父会管我！

在此再次衷心感谢师父的保护！感谢大法弟子的真诚帮助和鼓励！同时向师父和大法弟子们表达我对师父的真诚感恩！感谢师父的慈悲救度！愿所有大法弟子精進、圆满！愿世上所有善良的人们都能得到大法的救度！

谢谢大家！
