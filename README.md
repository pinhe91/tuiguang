### 邪党党员的烦恼

【明慧网二零二二年五月二十九日】

#### 邪党党员的烦恼

〖中国大陆来稿〗这些年因为工作及讲真相，我接触过不少中共邪党的党员，他们中有一部份人十分烦恼，烦恼的共同问题是，邪党只进不出，太邪恶。

他们中职业不同、身份不同，但是在亲身经历中共的邪恶后，都深感身为邪党的一份子，感到十分懊悔，尤其是邪党要求每天学习后，他们更加厌恶，面对每天的邪党洗脑教育，他们都想退出邪党。虽然成为邪党的一份子能得到好处，但是和每天令人恶心的洗脑学习相比，他们宁愿不要这个邪党的身份也不愿意再被洗脑。

他们中有的说为了退出这个党员，找组织说了好几次都不行，最后他们明白这是一个放进不放出的机构。他们说没有加入邪党不知道邪党有多邪恶，加入后慢慢地才亲身体会到，邪党说一套做一套，完全不为百姓着想，完全是反人性的，中共是中国一切不安定因素的根源。他们后悔之余，告诫自己的亲人千万不要入党。

中共少先队员、团员按照中共的说法到了年龄不再是了，也就不再骚扰（其实必须退出才有效，因为曾经举拳头发誓过，只有公开退出才能真正抹去兽印），而中共党员却是不断地被骚扰，各种学习、各种洗脑、各种答题等等，还得交党费，不学习不行等等，终身骚扰。很多人因为受不了骚扰想要退出，但是均不被允许。邪党组织称：退出的一定是思想有问题，只能被开除，不准自动退出。真是邪恶至极。通过接触我发现，除自愿和邪党为伍的外，很多人并不是真心加入邪党的，而是在单位被动或者是因为优秀表现被邪党强行冠上的邪党党员身份。

邪党生日快到了，他们中那些十分烦恼的人说，希望世人都能早日看清邪党，希望邪党早日解体，他们就可以不再受邪党的控制，不再被邪党强制洗脑。

#### 刑警队长也三退

〖中国大陆来稿〗在打真相电话中，一次接听电话的男士声称自己是刑侦大队队长，态度极其恶劣，威胁能抓到我把我绳之以法。满口说的都是被邪党洗脑后的标准内容：什么共产党发工资了、发退休金了、什么谁也打不倒共产党了。他无知地对法轮功充满仇视。

我开始问他：共产党有钱吗？它创造财富吗？并告诉他：老百姓所有的工资、福利是人们自己创造的财富，是我们应该获得的，不是共产党的恩赐。自邪党十八大以来有392位省部级高官落马，国家等于被一群罪犯管理着，这个国家能好吗？老百姓的日子能好吗？

这时他也跟着声讨起这些贪官的罪恶，我抓住这个机会肯定了他是一个有正义感的人。接着我讲：如果你回家问老一辈的人，都相信神佛菩萨保平安，相信善恶有报，但现在的中国人都不信了。让中华传统文化没有了承传，就是共产党从小就教育我们无神论、进化论的结果。人不信善恶有报，什么坏事都敢干，什么谎话都敢说，这样的社会能好吗？

他说自己从不贪污，从不干缺德的事。我接着肯定了他坚守做人的底线会获得福报的。我说：用善恶有报的法则推断你的孩子们一定都是非常好的，他很惊讶问：你咋知道的？我说：听你说话就能判断出来呀。这时他已经完全改变了态度，说他家三个孩子都很好，说他们都可以入党但他叫孩子不入，并说：“我其实早就看透了共产党，我是不让孩子们沾这个党的边的。”

接着我问：您对着那个党旗宣誓：为共产主义奋斗终身。这句话你不一定知道是什么意思，就是把命交给共产主义。你想想当初你加入的时候，无非是要有个好前程，但为了这个把命交出去你会干吗？收回您的命吧，退出这个邪恶的组织，求神佛保平安。常念法轮大法好就能保佑自己平安度过劫难。

话说到此，他竟欣然同意了退党。

对话持续了二十分钟，以上陈述也只是一部份内容。能把这个人救下来很欣慰。看似很顽固的体制内的人，心中早已对这个邪恶有了清醒的认识，但自动脱离邪党对他来说也很难。感恩师父能有机缘让我救了他，这次经历更增加了我救人的信心。
