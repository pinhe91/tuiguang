### 想举报的人现世现报、逃之夭夭

文: 广东大法弟子

【明慧网二零二三年四月十日】六十多岁的我心里惦记着老家的乡亲们，总想回去多救人，三月份终于回去一次。这天我正在衔上跟两个有缘人讲着真相，突然不知从哪里走过来一个自称是退伍军人的中年男子，表情凶巴巴的瞪着眼睛看着我问：你在说什么，你在讲什么，你在宣传法轮功吗？你敢在这里宣传法轮功？
我说：是啊！法轮功这么好，祛病健身，教人向善做好人，为什么不宣传？那人说：“我是一个退伍军人，你再敢宣传法轮功，我打电话报警把你抓走。”

我一看来了一个不讲理的人，当时也没有害怕，心里不慌不忙，知道师父就在身边，就小声说着：师父啊！这个人在诽谤大法，干扰我救人呐！法轮大法好、真善忍好！请师父帮助弟子。这样说着说着说了几遍。

那个退伍军人突然开始难受大声咳嗽起来，而且本来站在那里质问我的他咳得眼泪都出来了，站也站不住，蹲在原地不停的咳。他抬头用惊异的眼光看着我说：“你在念什么？你在说什么？”我说我在念法轮大法好！真善忍好！那个人话也不敢讲了，蹲在地上一直不停的咳着咳着，一会儿飞快的跑掉了。

等这个人走了之后，在旁边听我讲真相的两个有缘人也问我：阿姨刚才你念什么？为什么那个人会突然难受咳嗽起来呀？我说：我念法轮大法好、真善忍好。

我继续说：人在做，天在看。现在这时期只有相信法轮大法，退出共产邪党组织的人才能得到神佛的保护，才能躲过天灾、瘟疫保平安！只有大法才能救度所有的世人。那些诽谤大法、不相信善恶有报、迫害大法弟子的坏人都会遭到报应，都将被历史淘汰。只有明白大法真相，退出邪党组织，人才能有未来。

那两个有缘人听明白真相，又亲眼见证了要举报我的坏人遭现报的情景，都开心的退出曾经加入过的邪党组织，选择了美好的未来。

这次回老家还发现一个问题，我去年回来讲过真相、做过三退的村庄在疫情开放期间没有死一个人，而邻村不听真相、不做三退又说要举报我的那个村死了好几个人，有一家就死了两个人。真心希望所有的人赶快醒悟过来，快找真相、快听真相、快做三退，了解真相才是得救的唯一希望！

#### 患新冠白肺重症不起　念九字真言十天康复

〖中国大陆来稿〗二零二二年十一月底，中国大陆疫情封控完全解除，全国应阳尽阳，我们这里也全面沦陷，大小医院，人满为患。

耀新（化名）是县医院的职工，在疫情中也未能幸免，而且很快发展成重症白肺，住進了县医院重症监护室。耀新的父母是退休老师，年近七旬，两人每天轮流照看儿子。不久，耀新的母亲也被感染上了瘟疫，很快就去世了。耀新父亲的学生留真（化名）帮老师为师母办了丧事。

之后，留真在电话聊天中把其老师一家的遭遇告诉了一位修炼法轮功的同事，提及师弟耀新在医院重症监护室已经躺了一、两个月了，尚未脱离危险。这位同事就在电话中告诉他：诚念“法轮大法好，真善忍好”这九字真言可消灾消难，逢凶化吉。

留真将同事所说转达给师弟耀新。耀新坚持诚心念诵“法轮大法好，真善忍好”。五天后，耀新从重症监护室转至普通病房；又五天后，他康复出院。

看到耀新劫后余生，留真感慨万分：诚念九字真言真能逢凶化吉、遇难呈祥呀！

#### 九旬老翁信大法　几次病危复安康

〖中国大陆来稿〗我公公今年九十四岁了，年轻时得过肺结核，有一侧肺叶被压死了，呼吸困难，上班时，是铆工，打铁，耳朵也震聋了。但是，老人很相信大法，把加入的邪党党员组织退出来了。我告诉他念“法轮大法好，真善忍好”，他很认可。公公九十岁那年，我给他用耳机听师父讲法，他天天非常认真的听。

二零一八年年前的一天，老人突然病了，小姑子带他去看病，确诊为老年小脑萎缩，住了几天医院，治疗效果也不好，医生说保持这样就不错了。我们知道后，就和小姑子说：老人也不爱住院，就出院吧。老人出院后，我们让他坚持听师父的讲法，有空就念“法轮大法好，真善忍好”。老人很快就好了，他很高兴的说：“这是师父救了我，要不是师父，我今年就过不去了。”

老人九十一岁那年，又病了两次，病得很重，最后一次连吞咽功能都消失了，医院下了病危通知书，让做好思想准备。我们就一直在他耳边念“法轮大法好，真善忍好”，并告诉他自己在心里也念。很快老人就好了，出院了。小姑子见证了大法的神奇，她感慨的说：“我爸这场病好了以后，现在也不喘了，也能上来气了，耳朵也不聋了，和他说话都能听到了，和正常人一样了。”

是啊，九十多岁的人了，得了这么重的病，而且医院还下了病危通知书，如果老人不相信大法，很难活过来。是大法救了老人。
