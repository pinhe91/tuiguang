### 多位亲人念“法轮大法好”都得福报

文: 湖北大法弟子 香莲 

【明慧网二零二二年四月十五日】下面我举几个发生在我亲人身上的实例，他们通过诚心念诵九字真言“法轮大法好，真善忍好”，身体很快达到了康复，创造了一个个神奇。

#### 一、法轮大法救了我爸的命

二零一七年三四月份的一天晚上八点左右，父亲突然间心脏狂跳，很难受，流汗象下雨一样，衣服全湿透了，叫我妈赶快锤他的背，我妈见状吓得直哆嗦，两腿发软，赶快去找村里的一个叔叔（我弟弟他们都在城里），叔叔见状赶快送医院，医院不收，急忙打电话找医院的一个熟人。叔叔说，人已经不行了，赶快抢救吧。做了检查，打了针就睡了。第二天一大早，叔叔打电话来跟我弟弟说，你爸现在在医院，昨晚送来的，医院做了检查，很严重，这里治不了，赶快送武汉。

到了亚洲心脏医院，医生看了头天的检查结果，心颤360，全身器官大部分都不正常，全身大部分血管都不正常了。我爸那时74岁，亚洲心脏医院也不收，说没有床位，赶快找别的医院。又找到附近的一个三甲医院，医生看了检查结果也不收。这时我找到了一位在医院上班的大法弟子，正好他是在重症上班，他说去问问他们主任收不收。结果主任说，开个条子就可以住進来。过了几天，顺利的做完手术，搭了三个支架就回家了。

父亲手术后回家，没有好好休息，家里旧房换房子，弟弟工作很忙，弟媳生意也忙，房子的事情完全都是我父母亲照看，加上这期间邻居总是找茬，忍让也不行，天天闹，搞得父亲心力交瘁，身心疲惫。毕竟是70多岁的人了，吃了一辈子的苦，也落下不少的病。邻居这一闹，病又发了，天天喝药，把胃也喝坏了，饭也吃不進。我打电话，父亲说，又不行了。我马上打电话给我弟赶快回去看看。

果不其然，赶快又送医院，又不收。弟弟准备放弃治疗。我和丈夫说，赶快到武汉来，只要有一线希望，就不能放弃。丈夫说，我现在就去急诊挂号，你们赶快来。经检查，医生说血管这边也堵了，又要搭三个支架。弟弟问我搭不搭？我说坚决不搭，年龄大了受不了。弟弟说，你安排吧。他把卡给了我，就回去了。

我爸跟我说，他不想死，还想多活几年。我说反正你的病也治不了，走吧，就在我这住些时。晚上我就叫我爸跟我一起念“法轮大法好，真善忍好”，我爸说，说话的力气都没有。我说不要紧，我大声念，你就跟着我念。

念着念着，我爸的声音就慢慢变大了些，整整念了半个小时。我爸说好累呀，我说赶快休息吧！第二天我就放真相视频给他看，晚上又和我一起大声念“法轮大法好，真善忍好”。第三天又这样。

到了第四天，我叫我爸吃早饭，他说他今天早上起得很早，整整念了一个小时。吃完早饭，我放师父广州讲法给他看，他看得很认真，看着看着就笑起来了。

法轮大法救了我的父亲。父亲的精神面貌一天比一天好，把师父讲法九讲都看完了。弟弟来接我爸回家，我爸告诉我弟弟，说他的脚好了，一辈子的风湿好了，心脏好了，吃饭也正常了，腰也好了。

村里的人跟我妈说，看来这次他爸真的是不行了。等我爸回家，看到的人都感到震惊，怎么不像有病的人那样啊？我说是法轮大法救了我爸的命。

#### 二、 妹妹得福报的故事

我妹妹以前在棉纺厂工作，三班倒，结婚之后就明显感觉身体差了，由于单位效益不行，买断就回家了，他们家三叔就把他的一个干菜店给了他们。码头很好，所以生意也很好，经济宽裕了，就想生第二胎，可是一怀就掉。

那时我被迫害在监狱里，回来之后，妹妹把这事告诉了我，说她又怀上了，医生说又难保。我叫妹妹赶快念“法轮大法好，真善忍好”。她每天就躺在床上，很快就好了，没打针也没吃药。现在孩子都15岁了，身体很壮、很乖，学习成绩也好。妹妹在菜市场卖干菜，只要有人提到法轮功，她和女儿都敢公开站出来说法轮大法好。

还有一件事，就是二零一七年腊月二十六，年底结账，发现有张账单不见了，怎么找也找不到。就在这时，妹妹想起了我告诉他的危难来时诚心念九字真言，就在房间里大声喊：“李师父，请帮帮忙，法轮大法好，真善忍好。”话音刚落，姑姐从外面走进来说：“找到了，找到了。”在垃圾堆旁边找到了，可能是打扫卫生不经意扫出去了，这个账单是3600元，看到帐单一家人高兴了。到了大年初一一大早，妹妹就双手合十，对着天喊：“李师父，新年好，谢谢李师父。”这是初三拜年，妹妹当着全家的面告诉我的。

#### 三、 婆婆念“法轮大法好”现在90多岁了

婆婆有一次把腰闪了，当时80多岁的她躺在床上，翻身都很难，吃饭都要我把她扶起来喂，婆婆个头比我大，扶起来就比较困难，我不好得力，她也疼痛难受。这真是：无论是谁，曾经名望多崇高，财产多富裕，躺在病床上谁也替代不了，都要自己承受。健康是多么的重要啊。

看着婆婆难受，我心生慈悲，告诉婆婆赶快念“法轮大法好，真善忍好”，婆婆就天天念，一个星期就能够自理了，半个月就好了。伤筋动骨100天，80多岁的老人，半个月就好了，真是奇迹。

有一天晚上十点钟左右，我从朋友家回来，一进门婆婆告诉我说刚才有个东西把她压得很难受，喊我也喊不应，她说昨天也是这样。我就告诉她，你不要喊我，你就喊：“李洪志师父救我。”第三天早上，她把我的大法书从床底下全部都拿到客厅来，说昨天晚上又象前两天一样，我一喊师父就好了，肯定是不应该把这些书藏在床底下。后来我又被迫害一次，婆婆都把我的书保存的很好。

婆婆关键时刻都知道求李大师。婆婆现在90多岁了，身体很好，真是明白真相得福报，善待大法得福报，支持大法得福报。

#### 四、 活不起来的人，现在当小领导了

一九九六年刚得法不久，单位放假，我就和丈夫、小孩、婆婆一起回农村老家一趟，丈夫看到村里的荷花（化名）三十岁不到，象婆婆一样，说你怎么变成这样。她说身体不好，我丈夫说快到我家去，我媳妇回来了，叫她教你炼功。她说：真的？丈夫说她会教你的 ，去吧。她就真的来了。她就讲述了她所有的遭遇、痛苦，寻了三次死都没死成，丈夫经常打她，成了家常便饭，把她腿打坏了，去医院做手术，结果针头断在里面了，要做第二次手术取针头，那年代医疗就那个状态。她说，度日如年哪，搞了一身病，而且治也治不好，未老先衰。我说我教你炼功吧。

教她炼功的时候，她身上那种药味往外排，好难闻哪，满屋都是药味，她很认真，很高兴，那时我28岁，村里和我年龄差不多的四个年轻的媳妇和一个婆婆，看到她在炼，都跟着炼，炼功之后，都觉得非常好，干活也不累，都感到很神奇和超常。那时我只有《转法轮》，所以她们就听我读法。

一天， 荷花跟我讲，说她早上看到师父法身（穿着是西服），来到她床前，用手在她前面抓了一下，不知抓了个什么东西走了。她觉得很不舒服，随后就吐了一些乌黑黑的血，吐完之后就轻松了。炼了几天功，她身体所有的病都没有了。村里人见证了神奇。由于我只放了几天假，就又回单位去了。荷花他们两个就到深圳去打工了，有一个到别的地方去了，还有一个和婆婆在村里，从此以后，大法在她们心里都扎下了根。

前几天我回老家过年，见到她们，对大法都很相信，身体健康，家人也受益了。荷花讲述了他在深圳厂里的一件事，有一次厂里无名的发了一场火，都不知道是怎么回事，荷花跟另外一个叫喜喜（化名）说，走，我们跟领导讲讲，肯定是贴在门上的那个说法轮功不好的话导致的，叫领导赶快拿下来。荷花在厂里还是个小领导呢，荷花和喜喜在厂里做的都很好，深受领导的信任，他们说法轮功是叫人做好人的，不是电视上说的那样，领导就拿下来了。

我的全家，还有亲戚朋友、街坊邻居，很多都相信法轮大法好，得到了大法恩泽。说心里话，无论是炼法轮功的人，还是明白真相退出中共党团队组织的人，诚心敬念“法轮大法好，真善忍好”，就会天赐洪福，遇难呈祥，就会拥有美好的明天。
