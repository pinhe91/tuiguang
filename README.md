### 小伙子大喊：“我活过来了！”

文: 中国大陆大法弟子

【明慧网二零二二年十一月三十日】一天，和同修相约去讲真相。转过路口，远远就看到同修和五、六个人围在一起。我紧走了几步过去后，发现中间有一位小伙子骑着电动车，脸色发青、嘴唇发紫，身体僵硬的坐在电车上。周围人问他哪不舒服？他不停的说：“全身动不了、发麻，用手拍着心脏部位说难受，打120、打120、快打120。”
同修看到这种情况，就嘱咐身边的一位女士帮忙打120电话，然后挤到电车跟前，迟疑了一下后，就告诉小伙子念：“法轮大法好，真善忍好。”面对生死关头，只有大法能救你，大法无所不能。然后，同修念一句“法轮大法好”，小伙子跟着念一句；同修念一句“真善忍好”，小伙子跟着念一句。念的声音一声比一声高，念了六、七遍以后，小伙子突然大声喊道：“我活过来了！我活过来了！”

同修看到这个情景，随即告诉小伙子快谢谢法轮大法师父，小伙子也激动地说：“谢谢法轮大法师父！”
这时，小伙子的脸色慢慢的缓过来了，我赶紧问小伙子入过党、团、队没有？小伙子说只入过少先队，我说，那就把发过的誓作废了，从心里退出少先队，小伙子同意了。我又告诉小伙子要一遍一遍的念“法轮大法好，真善忍好”。不一会儿，120车到了，小伙子被大伙扶到车上，拉走了。

这时，我和同修才回过神来，打量了一下打电话的那位女士，才发现她穿着得体、举止端庄，显得很有修养，我们很自然的攀谈起来。

我们在路边聊了一会儿，给她讲了法轮功的基本真相、三退的重要性。后来了解到，她是信佛的，她也想帮助那个小伙子，以前许多人给她讲过法轮功的真相，她总是不相信。今天遇到这件事情，她很感触，并且同意了三退。分手的时候，她说，不知为什么特别想和你们聊聊。大家说，有缘我们再见吧。

我和同修又去讲真相了，路上久久不能平静，心想，师父真是伟大啊，利用各种形式救度着世人、成就着大法弟子。
