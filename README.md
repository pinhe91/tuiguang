### 狱警明白真相得福报

文: 中国大陆大法弟子 

【明慧网二零二二年八月二十七日】二零一一年，我讲真相被绑架，后关入监狱。有一个姓徐的狱警，是专门管法轮功学员的。有一天，她把我叫去谈话。我当时也没有害怕，很平和的跟她讲大法的真相。
我说：“我是因修大法救人被绑架来的，我们修炼人要讲真话，我对你也得讲真话，说真话是救人的，说假话是害人的。大法是教人向善的，让人做好人，做更好的人。师父要求我们要做到打不还手，骂不还口，要设身处地的为别人好。现在世界上有一百多个国家都让炼法轮功，就是中国搞镇压。大法要不好，能有这么多人炼吗？法轮功对祛病健身有奇效，我以前患多种病，炼法轮功后，再没有吃一粒药，全都好了。你看我现在的身体多好。”

她说：“你讲的这些，别人也跟我讲过，你们那本《转法轮》我也看了多遍了。”我说：“谁看谁受益，看的越多，受益越多。”她说：“你说的这些，我凭什么要相信呢？”
我说：“凭我的精神状态，凭我的为人处事，凭我对自己的严格要求。”她说：“我不看这些。”我说：”那你看什么呢？“她说：”我就想要个孩子，老也怀不上，我要能怀孕了，我就相信你说的这些。“

当时，我扑哧一笑，她问我笑什么，我说我笑你很善良啊。她说：“善良什么呀，你们都给我上恶人榜了，你还说我善良。”我说：“你可以弥补啊。你可以在你的责任范围之内，给我们一个宽松的环境，让我们能学到法，这就是行善事，这就是在弥补，这是功德无量的大善事。”她说：“我早就知道你学法，你还抄书呢。”我说：“那你就睁一只眼，闭一只眼，装不知道就行了，你这就是在弥补啊。”

通过讲真相，那位狱警对法轮功学员的态度有所转变，没过多长时间，她就怀孕了，生了一个男孩。
