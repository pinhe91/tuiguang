### 参与迫害十余年　内蒙古恶警冯小虎遭恶报

【明慧网二零二二年七月二十二日】（明慧网通讯员内蒙古报道）内蒙古赤峰市公安局元宝山区分局看守所副所长冯小虎，二零二二年七月八日左右因酒驾发生车祸，他不顾伤者逃离肇事现场，结果被外地警察抓捕。冯小虎现面临被开除公职，甚至被判刑入狱。而此前，冯小虎也遇家庭破裂，妻子与他离婚，并带走全部存款积蓄。
冯小虎为何落得如此下场？有道是：朗朗乾坤皎皎月，天网恢恢放过谁。

冯小虎，男，一九七零年十月月十九日出生，原元宝山区国保大队警察，后调元宝山区公安分局特巡警大队任大队长，再后来调入元宝山区看守所任副所长。冯小虎参与迫害法轮功学员长达十余年，他积极参与第一线的迫害行动：直接绑架、殴打、酷刑折磨，并敲诈、勒索、抢劫法轮功学员巨额钱财等。

![image](https://user-images.githubusercontent.com/79625284/180411522-ba695742-4f9e-4f71-b477-cc66d59a6a01.png)

冯小虎

冯小虎恶行累累，以下仅记录几宗：

▼毒打法轮功学员高文江：二零零八年九月，元宝山区法轮功学员高文江被警察绑架到元宝山区公安分局，张志刚、冯小虎等警察轮番上阵对高文江刑讯逼供。期间，张志刚揪住高文江的头发，无数次的左右扇他耳光，用胶皮钢丝棒（钢丝棒的外部是用类似汽车轮胎一样的胶皮，里边是钢的）抽打他的腰部、臀部，酷刑折磨长达数小时。张志刚累得打不动了，冯小虎接着上阵，他飞脚踢在高文江的脸上，开始左右打耳光，不止几十个耳光。冯小虎打累了，再换其他的警察上来打。

▼迫害法轮功杨桂芝：优秀教师杨桂芝在翁牛特旗被绑架后，冯小虎入室抢劫，抢走杨桂芝的电脑等私人物品，并绑架杨桂芝未修炼的女儿；他在法院对杨桂芝的诬判案中作伪证。杨桂芝后遭冤刑四年半，被非法开除，后含冤离世。

▼迫害法轮功学员杨翠玲：元宝山区山前镇法轮功学员杨翠玲老人，二零一一年六月三日被喀喇沁旗公安局国保大队刘振庭等绑架，冯小虎伙同喀喇沁旗警察主动参与迫害。杨翠玲老人被非法判刑三年。

▼二零零九年春夏之交，遭非法通缉的元宝山区法轮功学员李丹在吉林被冯小虎等警察绑架，后被非法判刑七年半。李丹被绑架时，冯小虎非常得意，好象立了什么大功一样。李丹问他叫什么名字时，他报了姓名，还嚣张反问：怎么的？要给上恶人榜吗？（冯小虎没有想到恶报应在今天）

▼一次，冯小虎为了得到真相资料的来源处，疯狂毒打一位姓罗年轻男性法轮功学员，揪掉他很多头发，还扒光其衣服电击生殖器，并戏耍侮辱该法轮功学员。

▼抢劫、敲诈、勒索：冯小虎绑架法轮功学员后，除了实施暴力，还千方百计勒索钱财，诱骗法轮功学员家人拿钱来赎人，动辄万元。二零一零年九月中旬，冯小虎闯进法轮功学员赵桂春的家，抢走好几万元的现金。在二零零八年的一次绑架案中，他抢走法轮功学员赵淑兰手提包里的几千元，人家要追回这笔钱时，冯小虎大言不惭地说：“都喝酒花尽了。”

#### 天日昭昭 善恶必报

善恶到头终有报，只争来早与来迟。在中共迫害法轮功的二十三年中，中共各级公安官员、警察，听命于中共江氏集团的迫害指令，残害当地法轮功修炼者，造下罪业空前，所以他们恶报连连，或暴病身亡、或撤职开除、或落马入狱，或殃及家人……冯小虎就是其中一例，他与邪恶为伍，最终大祸临头。

希望那些至今仍执迷不悟的公检法人员及政府官员，快抓住大淘汰来临前仅剩的一点时间，立即停止参与迫害法轮功，给自己选择最后的出路，否则生命将永远被淘汰。
